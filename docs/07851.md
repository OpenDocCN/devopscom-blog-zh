# 新的研究揭示了优化策略对于选择、支持和维护开源软件的重要性

> 原文：<https://devops.com/new-study-reveals-importance-of-optimized-strategy-for-the-selection-support-and-maintenance-of-open-source-software/>

拥有优化的 OSS 策略的组织有 82%的可能说他们公司的 OSS 计划超出了预期

明尼阿波利斯，2021 年 5 月 11 日/美通社/——Perforce Software，一家为在开发生命周期中需要生产力、可见性和规模的企业团队提供解决方案的公司，今天宣布了由 Forrester Consulting 进行的[委托研究的结果，以及即将举行的](https://c212.net/c/link/?t=0&l=en&o=3157692-1&h=2837284733&u=http%3A%2F%2Fwww.openlogic.com%2Fresources%2Fforrester-consulting-study-open-source-strategy&a=commissioned+study+conducted+by+Forrester+Consulting)[网络研讨会](https://c212.net/c/link/?t=0&l=en&o=3157692-1&h=4104552640&u=https%3A%2F%2Fwww.openlogic.com%2Fresources%2Fevents%2Fwhy-oss-strategy-is-key&a=webinar)对结果进行分析。

该研究基于对企业组织决策者的调查，显示 91%采用开源的公司达到或超过了商业预期。在实施 OSS 的团队中，那些拥有优化的 OSS 策略的团队有 82%的可能性超出预期。

然而，即使对于有开源软件支持策略的公司，60%的公司承认有一个次优的策略。

“这项研究支持了 OpenLogic 的一个长期观点，”Perforce Software 的首席技术官兼 OpenLogic 创始人 Rod Cope 说。“作为一个企业组织，使用开源软件取得成功需要一个完整的战略——尤其是当它应用于支持等关键领域时。”

对于企业公司来说，风险是很高的，正如业务关键型 IT 基础设施的所有领域越来越多地转向开源所表明的那样。Forrester 的研究发现，97%的受访者打算将他们的 IT 基础设施的至少一个方面迁移到 OSS。

这种转变的背后是它与积极的业务成果之间的联系。Forrester 的研究表明，采用 OSS 的公司体验到了更大的灵活性、更快的创新、更好的员工满意度和更低的运营开销。

“开源软件是现代 IT 系统和基础设施的重要组成部分，”Perforce Software 的首席产品官 Tim Russell 说。“这并不局限于传统的投资回报率指标——这项研究表明，开源软件的采用、员工保留率和客户成效之间存在强有力的积极联系。”
