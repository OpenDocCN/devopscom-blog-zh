# EP 10:可观察性和持续改进

> 原文：<https://devops.com/devops-unbound-ep-10-observability-continuous-improvement/>

[企业级的可观察性](https://devops.com/?s=observability)带来了额外的应用生命周期管理需求。成功需要知道哪些变化导致了改进，哪些没有产生积极的结果，以及这些信息如何通过反馈循环返回到 DevOps 团队。在《devo PS unbounded》的这一集中，Alan Shimel 和 Mitch Ashley 与 Tricentis 全球发行管理高级总监 Kurt Chase、Charity Majors、Honeycomb 的联合创始人以及史密森尼的高级应用经理 Ravyn Manuel 讨论了可观察性和持续改进。下面是视频，后面是对话的文字记录。
