# 游戏开发者不应该忽视 Python 的潜力

> 原文：<https://devops.com/game-developers-shouldnt-overlook-pythons-potential/>

Python 是一种面向对象的通用高级编程语言，由吉多·范·罗苏姆于 1991 年开发。自开发以来，Python 已经成为世界上最流行的编程语言之一。它经常在流行度调查中排名很高——例如，它在编程语言流行度指数中排名第一，在 TIOBE 指数中排名第二。

Python 赢得了广泛使用的工具的赞誉，因为它适用于多种目的和各种项目——从数据分析和可视化到人工智能、语言开发、设计和 web 开发。

也就是说，Python 在视频游戏开发中的应用并不出名。但是这是否意味着游戏开发者应该完全忽视 Python 呢？一点也不。

## 熟悉会产生生产力

由于这种语言的简单性和编码速度，Python 是构建原型的绝佳选择。工作产品立即可见，并有可能迅速向潜在投资者交付一个可玩的项目。

因此，Python 也是开发者开始游戏创作之旅的理想语言，无论用户是已经精通还是刚刚入门。

由于 Python 对新人友好的语法，开发者可以专注于游戏编程的基础，而不是语言本身的复杂性。

这些效果很快就会显现出来，使初学者能够积极参与到创建项目的过程中。另一方面，高级开发人员会欣赏这样一个事实，即他们可以使用他们熟悉的日常工作工具来创建一些可玩的东西，并且编码的过程将一如既往地非常愉快和令人满意。

## 预先存在的 Python 工具和库

Python 程序员也可以利用专门为他们编写的工具。

支持 [Python](https://devops.com/?s=Python) 游戏开发的最流行的库是 PyGame。这个引擎可以在 LGPL 下获得，允许用户构建完全商业化的项目。此外，该库的设计方式允许用户轻松使用多个处理器内核。主要的函数都是用 C 和汇编语言编写的，因此用户的项目将比只用 Python 编写快 10 到 20 倍。这个工具非常容易学习。它的创建者也准备了大量的文档来帮助指导用户完成这个过程。

也有可用的 3D 渲染工具，如 Ursina 或 Panda3D。当然，这些并不是为 Python 用户设计的唯一技术。事实上，有很多工具和引擎可供选择，比如 PyOgre、PyKyra 和 Ren'Py，都值得一试。

最后，还应该提到 Kiva 图书馆。它可能不是为游戏而设计的，但它非常适合游戏。Kiva 是一个多平台工具，用于快速创建窗口应用程序。

## 用于游戏开发的 python

如果你用谷歌搜索“我应该用哪种语言进行游戏开发？”您看到的第一个结果可能是避免使用 Python 的建议。然而，这不应该成为一种威慑；可以做到！

过去，Python 在游戏开发中的使用受到了批评，主要集中在 Python 不是为计算性能而设计的，而是为编码性能而设计的。这是有道理的。

但是，如果用户、程序员或游戏开发者已经熟悉了编程工具，并利用了 预先存在的工具和库，那么 Python 在游戏开发方面有着明确的前景，无论是现在还是将来。

如果你想证明用 Python 编写的游戏可以在国际上获得成功，我推荐[EVE Online](https://www.eveonline.com/)——这是一个每天吸引超过 20，000 名玩家的游戏！
