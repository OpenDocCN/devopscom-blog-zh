# Morpheus v5.0 通过基于角色的云管理/扩展的 FinOps 报告为新客户群带来了自助式 IT

> 原文：<https://devops.com/morpheus-v5-0-brings-self-service-it-to-new-customer-segments-with-persona-based-cloud-management-and-expanded-finops-reporting-2/>

科罗拉多州格林伍德村—2020 年 10 月 6 日—混合云应用程序编排公司 Morpheus Data 发布了其下一个主要版本的[混合云管理](https://morpheusdata.com/hybrid-cloud-management/)平台，该平台扩展了对自助式 IT 的访问，简化了跨云成本管理，并加快了与企业环境的集成。

在 Gartner 的云计算管理平台魔力象限中，Morpheus 被公认为领导者，在三个关键用例中获得最高的关键能力分数。在 5.0 版本中，Morpheus 扩展了这一领先优势，并为云管理、容器编排和基础设施自动化的融合奠定了基础。此版本的亮点包括:

*   基于角色的“一切即服务”(XaaS)界面，为非技术用户提供简单的自助服务 IT，并为 Morpheus 合作伙伴拓展机会。
*   更新的成本分析和报告引擎，包括发票数据、云节省详情和云迁移规划，为 FinOps 提供更多可见性。
*   扩展的无代码集成，包括新的“裸机即服务”选项和最新 ServiceNow 版本的认证，以加快价值实现。

根据 IDC 的数据，84%的企业计划在 2020 年将一些工作负载从公共云迁移或迁移到内部私有云、托管私有云以及非云环境。他们的研究还指出，74%的大型组织表示，通用跨云管理控制平面对于帮助扩展他们的运营非常重要。[二]

这些数据点是 Morpheus Data 在 2020 年看到的增长的领先指标，尽管新冠肺炎对全球经济产生了影响。Morpheus 在 2020 年同比增长超过 70%，新客户和合作伙伴增长超过 100%。为了帮助更多公司简化混合云管理，Morpheus 宣布推出一款面向中型企业的低成本“Essentials”软件包。

*“混合云管理和基础设施自动化市场正在迅速融合，”*Morpheus Data 首席营销官 Brad Parks 表示。*“我们的愿景是通过一个统一的平台来简化应用流程编排，为从边缘到数据中心再到公共云的所有类型的用户提供自助式 IT 服务。”*

**为“一切即服务”(XaaS)启用一键式购物车**

云计算被定义为通过自助服务门户按需、灵活地访问资源的能力。随着混合云的大规模增长和采用，这种自助式 IT 期望已经超出了开发人员的范畴。科学家、营销团队和其他最终用户群体也希望按需访问他们的应用服务。

在 5.0 版本中，Morpheus 通过新的界面选项扩展了其屡获殊荣的平台，旨在为寻求购物车式应用配置方法的技术含量较低的用户群体提供自助式 IT。单个统一平台第一次可以满足所有用户角色的企业自助服务 IT 需求，从简化的“简单按钮”到强大的基础架构即代码蓝图，以及介于两者之间的一切，从而将应用程序配置速度提高 150 倍或更多。

管理员和 IT 运营人员可以使用强大的向导驱动界面快速创建目录项目，该界面包括定制的选项列表和规格，最大限度地减少甚至消除了最终用户的输入需求。新的基于角色的界面选项适合现有的基于角色的 Morpheus 访问引擎，包括集成到身份管理提供商，如 Active Directory、Okta、SAML 等。这意味着客户可以轻松地将自助服务扩展到新的用户社区，而无需任何定制编程。

**为财务运营提供管理云成本和提高效率的可见性**

分析和成本管理继续在混合云管理需求中发挥重要作用，这在 2020 年变得更加重要，因为财务团队会考虑每一项 IT 采购，以节省成本和提高投资回报(ROI)。这些团队面临着在多个公共云成本计算工具中导航的挑战，并且缺乏来自内部私有云的可比信息。

在 5.0 版本中，Morpheus 是唯一能够为公共云和私有云利用率及成本数据提供单点集成的产品。新的发票界面展示了财务团队以前无法获得的大量数据。客户可以查看来自公共云提供商(如 Amazon Web Services (AWS)和 Microsoft Azure plus)的高度精细的成本数据，并从内部虚拟机管理程序(如 VMware 和 Nutanix)聚合相同级别的信息。当与 Morpheus 策略引擎和基于角色的访问控制结合使用时，客户可以创建和导出发票视图以进行交叉计费，并显示或将自定义视图导出到其他工具中以进行更深入的分析。

Morpheus 报告引擎也进行了全面的改进，增加了新的界面、报告类型和灵活的报告调度功能，以管理时间点或重复报告。针对成本和使用情况的新报告类型包括 AWS 预订覆盖范围和节约计划视图，以及用于比较总体最佳部署地点的迁移规划报告。正确利用这些报告可以帮助组织减少 70%或更多的云成本[iii]。

**扩展的无代码集成支持“裸机即服务”**

企业一直处于现代化之旅，这意味着在任何给定时间，他们都有应用程序运行在裸机服务器、虚拟化技术、Kubernetes 等容器平台上，并利用云原生 PaaS 服务。一个主要的挑战是以一致的方式连接这些异构平台，并避免供应商锁定。

Morpheus 是一个 100%不可知的混合云应用编排平台。这意味着，在管理与硬件 OEM、虚拟机管理程序公司和公共云超大规模提供商的供应商关系时，它可以为组织提供战略优势。

Morpheus 的 5.0 版本增加了 Canonical MaaS 作为支持的云类型，这立即增加了数十个新的裸机硬件提供商，作为边缘或企业数据中心云的支持选项。这为单个自动化平台内支持的业内最大的私有云和公共云阵列增添了支持

除了裸机云更新，该版本还增强了现有集成中的功能，包括 VMware vCloud Director (vCD)和 NSX-T、Microsoft System Center Virtual Machine Manager(scv mm)、Microsoft Azure 和 Veeam。
