亚马逊网络服务公司(Amazon Web Services)的企业策略师马克·施瓦茨(Mark Schwartz)撰写的《官僚主义的(微妙)艺术》展示了他作为政府官员时的幽默、历史和现实生活中的例子。施瓦茨提出了一个独一无二的剧本，向技术和商业领袖展示如何将官僚主义作为一种超级力量来运用，并同时打破它。

“我学到的是，官僚主义的挫折可以通过类似于我们在 DevOps 和数字化转型中使用的技术来解决。”
