# KubeCon: SysEleven 推出 MetaKube 运营商

> 原文：<https://devops.com/kubecon-syseleven-unveils-metakube-operator/>

*–客户对数字主权安全云运营的需求大幅增加* *–收入连续四年翻番，第三大区域是 live* *–更加关注 Kubernetes 作为增长驱动力*

**柏林/巴伦西亚，2022 年 5 月 16 日** **—** 由于收入连续四年翻番，云服务提供商 SysEleven 已经成为德国最重要的 Kubernetes 提供商之一。随着“MetaKube Operator”今天在 KubeCon 推出，SysEleven 通过其完全管理的 Kubernetes 产品将其产品组合提升到一个新的水平。当组织希望使用 DSGVO 兼容的云基础设施而不受供应商限制时，SysEleven 脱颖而出:“公司和公共机构正在积极寻找 AWS 和谷歌云的替代品，并越来越多地要求基于开源的解决方案，”SysEleven 首席执行官 Marc Korthaus 说。

**回顾与展望:Kubernetes 快速成长**

作为一家崭露头角的公司，SysEleven 已经在竞争激烈的云市场上确立了自己作为中型解决方案提供商的地位，并认为自己已经为未来专注于 Kubernetes 解决方案做好了准备。“所有迹象都表明增长在加速”，创始人兼首席执行官 Marc Korthaus 这样描述这种情况:“我们已经委托在法兰克福建立第三个数据中心，任命 Andreas Hermann 为我们的第三任董事总经理，并大幅增加了我们的员工人数。”

**数据感知组织的灯塔**

对安全敏感的公司和公共机构需要数据保护和主权的确定性，以实现其 IT 基础设施的现代化。SysEleven 首席执行官 Korthaus 指出:“云法案之后会发生什么仍然不明朗。“因此，需要新的策略来定义哪些数据可以在哪些基础架构中使用。作为符合 DSGVO 的提供商，我们的客户摆脱了这种法律不确定性。”

KubeCon 的 sys eleven

SysEleven 将于 5 月 16 日至 20 日在西班牙巴伦西亚 KubeCon 的 **展位 S62** 提供关于 MetaKube 运营商及其其他产品的详细信息。

**关于 MetaKube 运算符**

一年前，SysEleven 在 KubeCon 推出了 MetaKube Accelerator，这是一款半自动 Kubernetes 解决方案，面向那些希望使用经过测试的组件和基本应用程序的客户。他们在此基础上构建自己的云原生软件项目。有了 MetaKube 操作符，SysEleven 就接管了完整的操作。

SysEleven 创始人兼首席执行官 Marc Korthaus 表示:“借助 MetaKube Operator，我们显著简化了 Kubernetes 上云原生应用的设置和管理。“这是我们对客户要求的回应，客户希望将符合 DSGVO 且可扩展的 Kubernetes 平台的操作交给有经验的人。”

参与 MetaKube Operator 的封闭测试，有意者请直接在此联系 sys eleven:[https://www . sys eleven . de/en/products-services/meta kube-Operator/](https://www.syseleven.de/en/products-services/metakube-operator/)

**关于 SysEleven MetaKube**

该框架的技术基础是 MetaKube Core，这是一种符合 DSGVO 标准的托管 Kubernetes 产品，客户无需提供基础设施即可运行。MetaKube Accelerator 以此为基础，提供了一个集成环境，应用程序开发人员可以在最终版本中从策划的软件模块组装完整的设置。

MetaKube Operator 特别适合为客户提供云原生应用的公司。需要为客户的在线业务快速实现许多高性能项目的机构也可以从 MetaKube Accelerator 中受益。你可以从 SysEleven 这里找到更多关于 MetaKube 框架的细节:[https://www.syseleven.de/en/products-services/kubernetes/](https://www.syseleven.de/en/products-services/kubernetes/)

**关于 SysEleven**

SysEleven 是一家领先的云和 Kubernetes 服务提供商，专注于分布式和关键任务系统的运营以及工作负载的快速扩展。该公司支持希望完成向云原生环境转型并重视速度、灵活性和自主性的组织。服务组合由全面的服务组成，从概念咨询、测试环境设置、培训和全面管理的运营开始。垂直技术堆栈的核心是“MetaKube”，这是一个托管的 Kubernetes 产品，使组织能够开发和操作基于 web 的软件产品。SysEleven 非常重视数据保护和数字主权。因此，所有解决方案都由 ISO27001 认证的德国数据中心独家托管，使客户和合作伙伴能够遵守德国数据保护法规(DSGVO)。服务范围包括托管云服务、OpenStack 公共云、托管 Kubernetes 和各种网络服务。

总部位于柏林的 SysEleven GmbH 由 Marc Korthaus 于 2007 年创立，他和 Jens Ihlenfeld 仍然是董事总经理，拥有 100 多名员工。其客户包括 Schiesser、Intersport、DFS Aviation Services、Babymarkt、FTAPI、Staffbase、Senatsverwaltung Berlin、brand eins 和 Golem.de 等，更多信息可登陆[【https://www.syseleven.de/en/】](https://www.syseleven.de/en/)。
