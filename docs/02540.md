# Tricentis 被评为 2019 年 3 月 Gartner Peer Insights 软件测试自动化客户选择

**Gartner Peer Insights 客户选择区分基于积极的客户评论和评级**

云的头号连续测试平台 Tricentis 今天宣布，它被评为 2019 年 3 月 Gartner Peer Insights 软件测试自动化客户选择。除了被评为客户之选，Tricentis 最近还连续第四年在 [2018 年软件测试自动化魔力象限](https://www.tricentis.com/resources/gartner-magic-quadrant-software-test-automation/)中被评为领导者，并在 [2018 年 Gartner 软件测试自动化关键能力报告](https://www.tricentis.com/resources/gartner-critical-capabilities-for-software-test-automation/)中就*企业端到端*测试*和渐进式 Web 应用/响应式 Web/本机应用*用例获得最高分。

Gartner Peer Insights 客户的选择是基于经验证的最终用户专业人员提交的供应商认可。Gartner 在识别客户满意度高的供应商时遵循严格的标准。

Tricentis 首席执行官 Sandeep Johri 表示:“超过 1600 家全球组织采用了 Tricentis 连续测试平台来加速数字化转型。“在一个由测试*工具*主导的市场中，我们实现客户成功的方法专注于测试*转换*。Tricentis 强劲增长的原因是，我们与客户合作，转变他们的测试流程，而不仅仅是向他们销售单点工具。这些领先的企业现在一直通过软件提供与众不同的客户/用户体验，而他们不太先进的竞争对手却还在苟延残喘。"

Gartner Peer Insights 的大量客户评论列举了 Tricentis 基于模型的创新测试自动化如何帮助他们快速转变敏捷和开发运维的企业测试流程。以下是一些客户的看法:
