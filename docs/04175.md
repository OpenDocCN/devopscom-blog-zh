# Automic 工作负载自动化实现了与 Oracle JD Edwards、Oracle 电子商务套件和 Oracle PeopleSoft 的 Oracle 验证集成

> 原文:[https://devo PS . com/automic-workload-automation-achieves-Oracle-validated-integration-Oracle-JD-Edwards-Oracle-e-business-suite-Oracle-PeopleSoft/](https://devops.com/automic-workload-automation-achieves-oracle-validated-integration-oracles-jd-edwards-oracle-e-business-suite-oracles-peoplesoft/)

### *新的集成使甲骨文客户能够通过自动化来支持数字化转型计划*

**纽约州纽约市——2017 年 7 月 18 日——**[automicsoftware](https://automic.com/)，CA Technologies (NASDAQ: CA)旗下业务自动化软件的领导者和[甲骨文合作伙伴网络](http://www.oracle.com/partners/index.html) (OPN)的白金级会员，今天宣布实现了[automicworkload Automation(AWA)](https://automic.com/products/automic-workload-automation)与甲骨文 JD Edwards、甲骨文电子商务套件和甲骨文 PeopleSoft 的甲骨文验证集成。

Automic CMO 的 Chris Boorman 表示:“Automic 与这些 Oracle 应用程序的集成为当今企业所依赖的一些最关键、最复杂的流程带来了现代业务自动化。“我们很高兴实现了与 Oracle JD Edwards、Oracle 电子商务套件和 Oracle PeopleSoft 应用程序的 Oracle 验证集成。”

为了实现 Oracle 验证集成，Oracle 合作伙伴需要满足一系列严格的要求，这些要求基于客户的需求和优先事项。Automic 的集成为 Oracle 客户提供了现代业务自动化，旨在减少时间、提高能力和推动数字化转型计划。 [AWA](https://automic.com/products/automic-workload-automation) 集成的主要优势包括:

### *甲骨文的 JD Edwards EnterpriseOne:*

*   缩短传统批处理窗口的长度
*   降低版本维护开销
*   轻松跟踪子流程
*   向用户快速分发报告内容
*   JD Edwards EnterpriseOne 和其他应用程序之间的自动化处理
*   集中监控和审计

### *甲骨文电子商务套件:*

*   导入并发程序、请求集和参数
*   请求执行作业
*   在执行过程中反映并发管理器的任务状态
*   终止在并发管理器中运行的工作量
*   将并发管理器作业截取为原子作业，以便以后发布
*   查看并发请求输出的内容并对其做出反应

### *甲骨文的 PeopleSoft:*

*   使 PeopleSoft 内部以及与其他业务应用程序的数据流自动化
*   减少应用程序孤岛，以获得关键业务流程的可见性
*   提供高级队列和吞吐量管理
*   自动化错误检测和补救
*   通过在错误发生时立即通知用户来增强业务流程

Oracle 全球 ISV、OEM 和 Java 业务开发副总裁大卫·希克斯表示:“实现 Oracle 经验证的集成让我们的客户相信，Automic Workload Automation 与 Oracle JD Edwards、Oracle 电子商务套件和 Oracle PeopleSoft 的集成在功能上是可靠的，并且经过了测试。对于在内部、云中或同时在内部和云中部署的解决方案，Oracle 验证集成应用了严格的技术审查和测试流程，有助于降低部署风险并改善合作伙伴集成产品的用户体验

Automic 业务自动化为企业提供了应对当今不断发展的技术环境所需的可见性、敏捷性、速度和可扩展性。这款同类最佳的业务自动化产品集中管理多个应用程序和支持基础架构中所有客户工作负载的执行。要了解更多关于 Automic 产品的信息，请访问:[https://automic.com/products](https://automic.com/products)

### 关于甲骨文合作伙伴网络

甲骨文合作伙伴网络(OPN)是甲骨文的合作伙伴计划，为合作伙伴提供开发、销售和实施甲骨文解决方案的差异化优势。OPN 提供资源来培训和支持 Oracle 产品和解决方案的专业知识，并认识到 Oracle 不断增长的产品组合、合作伙伴基础和业务机会。OPN 最新增强的关键是合作伙伴能够因其在 Oracle 云上的投资而得到认可和奖励。与 Oracle 合作的合作伙伴将能够通过 OPN 云计划展示他们的 Oracle 云专业知识和在客户中取得的成功。该创新计划是对现有 OPN 计划级别的补充，为与 Oracle 云合作的合作伙伴提供多层次的认可和累进收益。要了解更多信息，请访问:[http://www.oracle.com/partners](http://www.oracle.com/partners)。
