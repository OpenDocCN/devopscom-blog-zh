# Instana Pipeline Feedback 集成了自动化 APM 和 CI/CD 应用程序版本

> 原文：<https://devops.com/instana-pipeline-feedback-integrates-automated-apm-with-ci-cd-application-releases/>

*开发人员和开发人员获得关于新版本性能的即时反馈*

德国索林根和加州圣马特奥，2019 年 8 月 15 日讯——针对动态微服务应用的自动化应用性能管理(APM)解决方案的领先提供商 Instana 宣布推出应用发布跟踪和反馈功能，这使其成为首个包含自动发布跟踪的 APM 解决方案。

新功能是 Instana Pipeline Feedback 的一部分，Instana Pipeline Feedback 是一种应用交付管道性能跟踪和分析解决方案，与 Instana 的自动化 APM 解决方案直接集成。除了新的渠道反馈系统之外，今天的发布还涵盖了具体的新功能:

*   应用程序发布的自动跟踪和性能反馈
*   与 Jenkins 集成，自动识别新版本

Instana 联合创始人兼首席运营官 Pete Abrams 表示:“随着应用团队努力提高部署流程的速度，他们需要在发生变化时立即获得反馈。Instana Pipeline 反馈跟踪并隔离应用程序发布性能，在几秒钟内通知开发人员和开发人员问题

Instana 的自动化 APM 解决方案包含管道反馈，可映射、可视化和管理微服务应用，提供应用团队提供高质量应用所需的可操作数据。新的 CI/CD 流程管理功能免费提供。

Instana 的自动化应用程序性能监控(APM)解决方案可发现应用程序服务组件和应用程序基础架构，包括 Jenkins、Kubernetes 和 Docker。该工具自动为应用技术堆栈的每个部分部署监控传感器，并跟踪所有应用请求，无需任何人工配置，甚至无需重启应用。该解决方案实时检测应用环境中的变化，调整自己的模型，并在几秒钟内可视化变化和对性能的影响。

对于部署团队来说，完整的系统是了解新软件部署质量的强大工具。每当部署正式版本时，Jenkins 都会通知 Instana Pipeline 反馈，该反馈将隔离每个新服务或代码，提供 Instana 性能报告，并提供服务和应用程序性能及运行状况与之前版本的比较。

像其他 Instana 功能一样，管道反馈是自动的，并在几秒钟内提供可操作的数据，为开发人员和开发人员提供他们需要的即时反馈，以了解新版本是否按预期执行。新功能将在 JenkinsWorld / DevOps 世界大会上展示。在 instana.com 了解更多。
