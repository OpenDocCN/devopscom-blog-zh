# OpenText 打造软件强国——以 60 亿美元收购 Micro Focus

> 原文：<https://devops.com/opentext-to-create-software-powerhouse-acquires-micro-focus-for-6b/>

OpenText 本周宣布将以 60 亿美元收购 Micro Focus。此次收购是重振广泛的信息管理、[应用开发](https://devops.com/?s=appliation+development)以及测试、集成和网络安全平台组合的努力的一部分，旨在加快这些平台向 OpenText 托管私有云的过渡速度。

此次收购预计将于 2023 年第一季度完成，合并后的公司预计将在 Micro Focus 之前承诺削减的 3 亿美元的基础上再削减 1 亿美元的成本，作为恢复盈利和增长的持续努力的一部分。

OpenText 首席执行官 Mark J. Barrenechea 告诉分析师，最终目标是让组织能够使用一套驻留在私有云上的以应用编程接口(API)为中心的产品来加速他们的[数字业务转型](https://digitalcxo.com/)计划。

他说，数字化转型代表着计算机历史上最大的机遇，但 Barrenechea 也承认，许多公司仍在努力实现这种转型。“一定有更好的方法来实现数字化转型，”他说。

Micro Focus 产品组合包括机器人流程自动化(RPA)产品、最近推出的 IT 服务管理(ITSM)产品(称为服务自动化管理 X)以及 Vertica 数据库和 LoadRunner 测试工具。还有一系列网络安全平台，包括 ArcSight、Fortify、IDOL Zenworks 和 Sentinel，以及遗留的 COBOL 应用程序开发工具，甚至 Novell 的残余部分。

实际上，在之前收购了 Documentum、Carbonite、Hummingbird 和 Zix 之后，OpenText 现在拥有了业内最广泛的软件产品组合之一。“策略似乎是简单地收集棋盘上的所有棋子，”Techstrong Research 的首席分析师 Mitch Ashley 说，该公司是 Techstrong Group 的一个分支，也是 DevOps.com 的母公司。

Barrenechea 说，尽管 Micro Focus 的投资组合支离破碎，但 OpenText 无意出售其中任何一种产品。展望未来，该战略不仅要加速向云的过渡，还要扩大应用许可更新服务，他补充道。Barrenechea 指出，OpenText 还看到了将微焦点产品组合的吸引力扩展到中小型企业(SMB)市场的机会。

总而言之，一旦交易完成，将需要六个季度的时间来使微焦点的商业模式与 OpenText 目前使用的模式保持一致，Barrenechea 表示，这将导致交易完成后八个季度开始增长。

毫无疑问，一旦交易完成，依赖 Micro Focus 的 IT 组织将密切关注投资组合中的持续投资水平，以寻找一个平台被另一个平台取代的任何迹象。同样可能的是，许多客户看到那些平台的管理工作由一家财务基础更稳固的公司来承担，会松一口气。

然而，不管 OpenText 产品组合在 2023 年如何发展，有一点是肯定的，IT 组织不缺乏备选方案来考虑。
