# Cider Security 筹集了 3800 万美元，将首个 AppSec 操作系统推向市场

> 原文:[https://devo PS . com/cider-security-raises-3800 万美元将首个应用安全操作系统推向市场/](https://devops.com/cider-security-raises-38-million-to-bring-the-first-appsec-operating-system-to-market/)

***以色列特拉维夫，2022 年 3 月 7 日**–[苹果酒](https://cidersecurity.io/) [安全](https://cidersecurity.io/)，世界上第一个 AppSec 操作系统，今天秘密浮出水面，宣布它已经筹集了 3800 万美元的资金，最近的 A 轮由[老虎全球管理](https://www.tigerglobal.com/)领投。加入这轮融资的是来自 Cider Security 种子融资的主要投资者 Glilot Capital Partners，以及 Glilot 早期增长基金 Glilot+的参与。*

 *当今工程生态系统的多样性、快速性和动态性带来了一系列新的安全挑战和缺口，这些挑战和缺口不仅减缓了工程进度，还不断被黑客利用。

Cider Security 的平台提供了整个工程生态系统的统一视图，使安全团队能够深入了解和全面了解每个工程环境特有的技术、系统和流程。为安全团队提供了一套高度定制的控制和解决方案，以优化安全性，并在整个 CI/CD 管道(从代码到部署)中实现全面的弹性。Cider Security 旨在消除安全团队和工程团队之间的摩擦，允许在几分钟内实施 AppSec 计划，使安全民主化，并允许 AppSec 成为一种商品，可供所有垂直行业、规模和成熟度级别的组织使用。

“我和我的联合创始人丹尼尔经历了安全工程师今天面临的许多挑战，所以我们开始建立一种新的方法，”Cider Security 的首席执行官兼联合创始人盖伊·弗莱彻特(Guy Flechter)说。“通过消除实施保护的障碍——实质上是使 AppSec 团队的安全性民主化——我们正在改变许多人认为一成不变的流程。用户的兴奋程度完全超出了我们的预期，我们迫不及待地想改变各地公司的安全和工程团队的运营方式。”

这笔资金将用于扩大 T2 在以色列的研发业务，并在全球开设新的办事处，以支持不断增长的需求。该公司已经与数十家全球公司合作，还拥有来自 Atlassian、Databricks、网飞、Rapid7、Built Technologies、lemma、Rapyd 和其他公司的安全领导者，作为其顾问委员会和投资者的一部分。

Tiger Global Management 的合伙人约翰·柯蒂斯(John Curtius)说:“苹果酒安全市场需求巨大。“公司使用的方法没有跟上威胁形势的步伐，造成了严重的漏洞，影响了全球的业务，而 Cider Security 即将成为 CI/CD 安全的行业标准。即使在这样一个早期阶段，他们已经达到了令人印象深刻的里程碑，并收到了令人难以置信的市场反馈。我们期待着与他们一起开始旅程。”

Glilot 创始管理合伙人 Arik Kleinstein 表示:“作为首个 AppSec 操作系统，Cider Security 有望成为市场上领先的 DevSecOps 解决方案。“该解决方案在短时间内为其客户带来了巨大的影响，提供了一种考虑到开发现实的新的应用程序安全方法。”

**关于苹果酒的安全性**

Cider Security 是首款 AppSec 操作系统，为安全和工程团队提供了一种单一、一致的方法，通过单一、统一的平台来协调和实施端到端 CI/CD 安全。从代码到部署，该公司对工程流程和系统的安全性采取整体方法。它建立了工程环境的全面技术 DNA，为安全团队提供了优化 AppSec 和实现全面弹性所需的透明度和可见性。Cider Security 由网络安全行业资深人士 Guy Flechter 和 Daniel Krivelevich 于 2020 年底成立，其使命是解决 CISOs 和安全工程师当今面临的最常见挑战。欲了解更多信息，请访问。*
