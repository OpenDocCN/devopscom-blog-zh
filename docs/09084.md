# IBM Automation Insights:提高可观察性和应用程序性能

> 原文:[https://devo PS . com/IBM-automation-insights-improving-observability-and-app-performance/](https://devops.com/ibm-automation-insights-improving-observability-and-app-performance/)

可观察性提供了对复杂系统内部状态的深入了解，以便快速识别和排除性能问题，从而满足客户期望和业务需求。可观察性支持更好的应用程序性能监控(APM)。那么，DevOps 团队如何获得对现代应用程序的完全可观察性呢？

2 月 23 日，来自 IBM 和 GitLab 的专家将齐聚[IBM Automation Insights](https://www.techstrongevents.com/ibm-automation-insights)虚拟活动，讨论如何跨平台有效地观察、管理和维护应用，以及如何提高应用性能。

虚拟活动旨在创建互动会议，由 DevOps 专家主持，他们将深入探讨最新的创新，进行演示并分享应对挑战的方法。

[IBM Automation Insights](https://www.techstrongevents.com/ibm-automation-insights)特色演讲嘉宾阵容由杰出的工程师、IT 领袖和高管组成，包括:

*   IBM devo PS 业务主管 Cosmo Schillaci
*   IBM 美洲开发部主管 Jennifer Velasquez
*   IBM 产品经理 Laurel Dickson-Bull
*   Kurt Dusek，GitLab 高级解决方案架构师
*   IBM 软件工程师杰米·科尔曼

与会者将了解 Instana 和 GitLab 如何帮助 DevOps 团队加速 CI/CD 管道，以及如何在本地使用云原生 Java 应用和微服务、使用容器以及为 OpenShift/Kubernetes 轻松构建应用。

将会有一个跨平台的小组讨论，与会者将有机会与演讲者互动，就提出的任何策略提出问题，并讨论几个主题，包括网络安全、可观察性优化、漏洞工具扩散等。

欲了解更多信息并进行注册，请访问 [IBM 自动化洞察网站](https://www.techstrongevents.com/ibm-automation-insights) 。
