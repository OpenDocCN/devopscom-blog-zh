# Sumo Logic 为亚马逊网络服务推出强大的全球威胁基准测试服务

> 原文:[https://devo PS . com/sumo-logic-launchs-powerful-global-threat-benchmark-service-for-Amazon-we b-services/](https://devops.com/sumo-logic-launches-powerful-global-threat-benchmarking-service-for-amazon-web-services/)

*面向 Amazon GuardDuty 的全新全球情报服务，提供几乎实时的可操作见解，以对 AWS 工作负载和数据的安全状况进行基准测试、优先排序和优化*

**加利福尼亚州雷德伍德城——2019 年 6 月 25 日**—[Sumo Logic](https://www.sumologic.com/)，领先的云原生机器数据分析平台，提供[持续情报](https://www.sumologic.com/company/about-us/)，今天在 [AWS re:Inforce 2019](https://reinforce.awsevents.com/) 上宣布，为亚马逊 GuardDuty 提供 Sumo Logic 全球情报服务，以提供几乎实时的可操作见解，允许客户将自己与亚马逊网络服务(AWS)云基础设施的其他采用者进行基准对比，以进一步加强云安全态势，改进威胁检测，并增强法规合规性 Sumo Logic 将于本周在波士顿举行的 AWS re:Inforce 2019 上展示这项新服务，展位号为 714。

Sumo Logic 产品和战略创始副总裁 Bruno Kurtic 表示:“在当今高度互联的世界中，组织正面临着快速发展的业务需求，这些需求要求他们彻底反思构成其业务支柱的架构、IT 环境和应用堆栈。“作为一个多租户、云原生机器数据分析平台，Sumo Logic 具有独特的定位，不仅可以提供丰富的背景信息，帮助我们了解客户的应用和基础架构堆栈如何运行和表现，还可以了解其行为和表现与同类最佳组织有何不同。这种级别的洞察力有助于客户识别影响应用程序性能和可用性的异常模式、异常事件和关键性能和风险指标，以及可能对组织的环境(甚至客户)产生负面影响的关键威胁。"

用于 Amazon GuardDuty 的 Sumo Logic 全球情报服务分析由 Amazon GuardDuty 服务检测到的全球活跃威胁，以提供关于组织的整体威胁特征与行业同行有何不同的重要见解和背景，并识别组织环境中的罕见或关键威胁。例如，Sumo Logic 可以检测并突出显示很少在全球范围内发生但在客户环境中活跃运行的威胁，如试图与 XorDDos 恶意软件通信或其他可能隐藏在大量更常见事件下的威胁。

ThoughtWorks 的高级安全分析师 Philip Duldig 表示:“作为一家全球咨询公司，我们的组织每天都会面临成百上千的潜在安全威胁和事件，因此不仅要跟踪这些事件，还要确定处理这些事件的优先级，这是一项挑战。“作为 Sumo Logic 为 Amazon GuardDuty 提供的全球情报服务的早期采用者，我们体验到的最大价值是能够获得可操作的见解，以便对 AWS 工作负载中罕见或不频繁的安全事件进行优先排序和基准测试，从而优化我们的安全状况。我还喜欢将全球基准数据与我的本地数据进行比较，看看我们的进展如何。”

Sumo Logic 全球情报服务强调客户的总体威胁特征与前瞻性领导者之间的关键差异，这可能是由于配置错误、主动入侵或只是缺乏对特定攻击媒介的了解或关注。凭借对安全计划执行情况的强大数据洞察力，AWS 客户可以直接在 Sumo Logic 平台中轻松识别异常活动，并立即采取措施解决这些安全问题，以免影响其业务。

亚马逊网络服务公司安全服务副总裁 Dan Plastina 表示:“随着 DevSecOps 的兴起和不断扩大的全球威胁形势，组织持续监控其环境和工作负载中的恶意活动比以往任何时候都更加重要。Sumo Logic 平台可以补充亚马逊 GuardDuty 现有的威胁洞察，使安全团队更容易检测威胁并改善其云安全态势。”

使用 Sumo Logic Global Intelligence Service for Amazon guard duty，客户可以深入了解他们的威胁媒介，以便:

*   基准测试:提供对各种威胁媒介和攻击的额外见解，以基准测试这些攻击的正常情况，有多少用户见过类似的攻击，以及在 AWS 用户中最常见和最不常见的是什么

*   区分优先级:根据罕见事件确定安全事件的正确优先级

*   优化:通过对特定警报和安全最佳实践进行分类和优先排序，采取措施改善安全状况

Sumo Logic 全球情报服务是一项运营和安全基准测试服务，利用机器学习和统计分析来揭示全球关键绩效指标(KPI)和关键风险指标(kri)。这些全球关键绩效指标和 KRI 基准允许组织根据世界领先的新技术、现代架构和云基础架构采用者来衡量他们的技术选择、性能和行为。

“我们开发了 Sumo Logic 全球情报服务基准，以帮助我们的客户加快采用云服务和新应用程序组件等现代技术，推动他们的数字计划，并使他们在竞争中脱颖而出，”Kurtic 说。“这不仅为我们的客户树立了一个标准，也为如何改进他们当前的流程和计划以在数字时代保持竞争力提供了清晰的视线。这就是为什么我们致力于扩大全球情报服务的广度和深度，并期待为我们客户的开发、运营和安全管理生命周期的每个部分提供基准洞察。"
