# 集装箱化时代即将到来！Gaia 通过三种方式帮助企业创新和转型:培训课程、集装箱安全和专业服务

> 原文:[https://devo PS . com/the-era-of-containing-is-coming-Gaia-helps-enterprises-innovation-and-transform-in-three-way-training-courses-container-security-and-professional-services/](https://devops.com/the-era-of-containerization-is-coming-gaia-helps-enterprises-innovate-and-transform-in-three-ways-training-courses-container-security-and-professional-services/)

Gartner 对容器技术在行业中的影响持乐观态度，预计到 2022 年，超过 75%的全球组织将在生产中运行容器化应用程序，而目前这一比例不到 30%。随着集装箱使用量的增加，企业对集装箱管理的需求也在增加。全球集装箱管理收入将强劲增长，2024 年将达到 9.44 亿美元。因此，Gaia 不仅加强了员工提供 Kubernetes 咨询、部署服务的技能，并获得了提供 Kubernetes 官方中文课程的认证，而且还引入了容器工具，如 CI/CD 平台 Gitlab 和漏洞扫描服务，可以帮助组织逐步构建容器化的架构，并确保各项业务的运营高效工作。盖亚创始人兼首席执行官吴冰-君表示，“为了帮助组织保持领先地位，盖亚不仅投入巨资培养认证讲师，还致力于通过培训课程培养更多人才。”

为了减少对虚拟机的依赖，以更经济的方式保持系统资源的有效利用，并快速响应不断变化的市场，容器技术在过去几年中引起了市场的关注。AWS、微软 Azure、谷歌云等云计算平台积极拥抱容器技术。许多组织也开始部署容器。作为企业创建和部署云服务的最佳合作伙伴，Gaia 不仅积极拥抱容器技术，如在容器化环境中开发官方网站、客户服务、会计管理和其他系统，而且在提供容器部署和咨询服务以及与 Linux 基金会下属的云原生计算基金会(CNCF)谈判合作方面积累了大量经验，以提供授权的 Kubernetes 培训课程，并帮助学生准备像 CKA(认证 Kubernetes 管理员)和 CKAD(认证 Kubernetes 应用程序开发员)这样的证书。

“Gaia 是台湾首家 CNCF 认证的混合云提供商，同时也是拥有最多认证教师的机构。仅去年(2021 年)，我们就有五名员工成为了 CNCF 认证讲师(大中华区只有 26 名认证讲师)。今年(2022 年)，我们将把教官人数扩充至 7 人。”

Gaia 的 R & D 主管侯麟-年解释说，Gaia 从教育服务开始有两个原因:首先，容器技术的快速发展，其次，容器管理涉及各种技术技能，如开发、基础架构架构师、微服务管理以及解耦设计技能等。因此，培养人才并不容易，世界正在经历持续的人才短缺。

为了缓解人才短缺的困境，盖亚与 CNCF 合作，提供中文(官方授权)培训课程，并通过在开源社区写文章、参加技术论坛等方式，积极与市场分享实践经验。Gaia 还计划制作教程视频，帮助 IT 工程师、软件工程师、应届毕业生或转行的人更快地成为容器技术大师。

“在开展培训课程后的 3 个月内，我们已成功协助运输和半导体机构共培养了 40 名集装箱技术专业人才，并帮助 30 多名学员成功获得了执照。我们还实现了约 98%的高客户满意度。 盖亚将继续提供更多的 CKA 和 CKAD 课程，以应对市场需求。此外，我们愿意与教育机构合作，培养更多的人才，”吴冰君说。

只提供 Kubernetes 培训课程是不够的。Gaia 针对实际需求实现平台服务:Gitlab 和漏洞评估服务。侯麟-年表示，Gitlab 是一个一站式 DevOps 平台，不仅可以帮助企业同时管理和跟踪集装箱化项目的进度，还具有强大的 CI/CD 功能。Gitlab 可以轻松连接 GitHub、Bitbucket 等服务。

“至于漏洞评估服务，我们可以在开发容器化应用程序时进行，这可以在第一时间为我们的客户降低安全风险。我们已经帮助许多金融和制造业客户引入了这项服务。”

除了计划培训课程和平台服务，盖亚还根据客户要求提供定制服务，例如通过概念验证(PoC)方法帮助企业部署容器化应用。吴冰-君表示:“我们的目标是为客户提供一站式服务，从而降低客户部署容器化应用的门槛，并享受云就绪基础设施的最大好处。随着集装箱服务的普及，我们将进一步把业务扩展到托管服务。”

展望未来，Gaia 将继续加强现有的容器化产品和服务，开发更多样化的 DevOps 和 DevSecOps 服务，并通过扩展服务产品来应对容器化趋势——基础设施即代码、自动化部署、可扩展扩展、云到云集成——以帮助客户通过容器技术创建与时俱进的云就绪系统环境，从而推动业务绩效。
