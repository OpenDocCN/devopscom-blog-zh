# GitLab 以 1 亿美元的融资加入独角兽俱乐部

> 原文:[https://devo PS . com/git lab-joins-unicorn-club-with-100m-raise/](https://devops.com/gitlab-joins-unicorn-club-with-100m-raise/)

[![GitLab](../Images/0e84de4aa2c4bfd85b74ce037acf5388.png)](https://devops.com/wp-content/uploads/2018/09/gitlab.jpeg) 在[宣布](https://about.gitlab.com/2018/09/19/announcing-100m-series-d-funding/)获得由亚由美资本领投的 1 亿美元融资后，GitLab 成为独角兽俱乐部的最新成员。最近一轮 D 轮融资使 GitLab 在 3 年内的总融资额达到 1 . 45 亿美元。据该公司称，这也提高了他们的市场价值超过神奇的 1B 马克。

GitLab 在最近一轮融资中的成功可以归功于水涨船高。微软以 75 亿美元收购 GitHub 继续在软件开发工具领域掀起波澜。但是尽管他们以自己的名义分享 Git，GitLab 并不仅仅是另一个 Git 或 GitHub 的模仿者。不，GitLab 有更远大的使命。他们实际上希望成为完整的端到端开发运维解决方案的一站式商店。使用 GitLab，您可以替换 6 到 9 个不同的 DevOps 相关工具。或者至少这是一个愿景。

我一直认为这是一个艰巨的任务，需要一种不同的方式来构建软件才能成功。GitLab 的首席执行官 Sid Sijbrandij 认为他已经找到了这种新方法。使用开源软件，GitLab 拥有一大批软件贡献者，他们每天都向 GitLab 贡献他们的软件。通过每月发布的版本，你可以立刻看到这一理念在各方面不断进步的结果。

虽然 Sid 意识到 GitLab 的每个组件或领域可能都不是当今最好的，但他认为它会是最好的。那么，今天 1 亿美元能让你认识到这一愿景吗？嗯，你可以从那匹~~马的~~独角兽的嘴里听到。我和席德坐下来讨论加薪、GitLab 的未来以及整个市场。

在之前，我有幸采访了希德。他对市场现状和发展方向有一些非常好的想法。以下是我们的对话，欣赏！
