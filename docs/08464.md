# Sendachi 启动了 3000 万美元的初始资金承诺，以帮助全球企业利用现代技术和方法的变革力量

> 原文:[https://devo PS . com/senda chi-launches-with-3000 万初始资金-承诺帮助全球企业-利用现代技术和方法的变革力量/](https://devops.com/sendachi-launches-with-30m-initial-funding-commitment-to-help-enterprises-worldwide-harness-the-transformative-power-of-modern-technologies-and-methodologies/)

*新公司联合美国和英国的专家，为企业 IT 项目团队提供实践指导，以实现 DevOps 和持续交付的自给自足。*

**西雅图–2016 年 1 月 21 日**–senda chi 是一家技术服务公司，旨在促进企业 IT 部门内充满活力的现代软件开发文化，该公司正式启动，风险投资家哥伦比亚资本(Columbia Capital)承诺投资 3000 万美元。Sendachi 是由志同道合的 IT 服务公司 Clutch(总部位于西雅图)和 Contino(总部位于伦敦)合并而成。森达奇将由前离合器首席执行官史蒂文·安德森领导。

Sendachi 正在解决当今企业 IT 部门面临的最棘手的问题之一。具体来说:如何加速新服务和服务增强的部署，以获得让企业家和以客户为中心的方法蓬勃发展的敏捷性和速度。在企业中实现这一目标并不容易，在企业中，内部开发团队被遗留应用程序的现有责任和几十年来一层一层增长的技术堆栈所拖累。

许多世界上最大的品牌已经开始将软件视为一种战略差异，但创造下一波数字产品和服务需要对云和容器、微服务架构等技术以及敏捷和 DevOps 等方法的经验。最具颠覆性的互联网时代的公司已经展示了如何利用这些技术来实现指数增长和确保竞争优势。老牌企业想要效仿，但需要手把手的指导和技术支持才能实现。

这就是仙达智擅长的地方。它的专家团队与客户一起工作，指导他们，因此内部员工可以以实际的方式体验现代系统和方法，并学习复制这些战略和战术。“我们最大的优势，”安德森说。“我们不以抽象的方式培训内部团队，我们与他们一起参与，通过执行他们的实际工作向他们展示如何做。这不是学术性的。绝对实用。我们向他们展示如何使用新的堆栈技术和不同的工作方式来实现加速和质量。”

451 Research 的云管理和容器研究经理 Jay Lyman 表示:“当今的企业 IT 环境充满了以敏捷性和效率为中心的新技术和方法，可以帮助组织应对时间、规模、业务、竞争和其他挑战。“然而，现实情况是，组织必须在现有的基础架构、流程和人员的基础上进行有效转型，这意味着利用创新软件和密集支持来启动和构建成功的开发运维计划、应用和服务。”

Sendachi 专注于高端市场的客户和跨行业的大型企业。其员工在 DevOps 和持续集成/持续交付方面拥有丰富的经验，曾在美国和欧洲的财富 50 强公司工作过。森达奇的全球总部设在西雅图，欧洲总部设在伦敦。Contino 联合创始人 Benjamin Wootton 和 Matt Farmer 将继续留在新公司，Wootton 担任欧洲技术副总裁，Farmer 担任欧洲总经理，前 Clutch 负责人 Jerome Gagner 被任命为美洲技术副总裁，前 CA Technologies 全球数字化转型负责人 Justin Vaughan-Brown 被任命为营销副总裁。该公司将在接下来的一年里在两大洲大举招聘。

“从银行业到媒体甚至出租车，各行各业都感受到了基于软件的颠覆带来的威胁或痛苦。伍顿评论道:“企业必须越来越大胆地押注，才能保持领先地位。”“在接下来的两年里，我们将会看到更多的重组、更多的内包、更多的平台再造和更多的架构再造，这些都是我们在过去十年中看到的，因为各公司都在争先恐后地从软件遗产转向软件作为竞争优势。随着大西洋两岸团队的不断壮大，我们期待为更多组织提供技能、工具和文化，以实现这一优势。”

Columbia Capital 的合伙人 Jason Booma 表示:“Contino 和 Clutch 都在证明，他们的企业客户可以从利用新技术和新方法完成低风险项目，逐步实现战略性的企业范围转型。“作为 Sendachi，该公司将继续其在应用程序外包市场的不同做法，其中一个关键的客户利益是提高技术开发能力，而不仅仅是一个伟大的应用程序。我们很高兴能与 Steven 及其团队合作，共同完成这项客户使命。”

**关于森达奇**
森达奇是一家专注于开发运维、持续集成/持续交付和转型项目的技术和服务公司。该公司的快速原型开发和 DevOps 加速服务有助于组织加快高质量新的和重新设计的应用程序的上市时间。从战略和运营到文化和技术，森达奇帮助商业和技术领导者发现并抓住增长和盈利的机会。Sendachi 为所有 DevOps 和敏捷技术提供培训、开发、部署和优化服务，包括应用生命周期管理(ALM)、现代开发和持续交付工具、微服务架构、容器化、安全、分析、测试和云基础设施平台。在 www.sendachi.com 了解更多。
