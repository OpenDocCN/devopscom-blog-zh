# 在 Gartner 的 2019 年应用发布协调关键能力中，CloudBees(电子云)流在所有三个用例中得分最高

> 原文:[https://devo PS . com/cloud bees-electric-cloud-flow-scores-high-in-all-three-use-cases-in-Gartner-2019-critical-capabilities-for-application-release-orchestration/](https://devops.com/cloudbees-electric-cloud-flow-scores-highest-in-all-three-use-cases-in-gartners-2019-critical-capabilities-for-application-release-orchestration/)

*Gartner 对应用程序发布协调(ARO)产品的评估基于围绕发布协调的使用案例；管道和环境管理；和部署自动化*

<time datetime="2019-10-29T15:30:00Z">October 29, 2019 11:30 AM Eastern Daylight Time</time>

加利福尼亚州圣何塞([美国商业资讯](https://www.businesswire.com/))–[推动持续经济发展的企业开发运维领导者 CloudBees](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.cloudbees.com&esheet=52118651&newsitemid=20191029005756&lan=en-US&anchor=CloudBees&index=1&md5=3536eee8e4d56b8fd833a465d592eed8) ，今天宣布 CloudBees(电子云*)在 Gartner 的 2019 年应用发布协调关键能力(ARO) ¹ 中定义的所有三个用例中获得最高分。Cloud bees Flow 9.0(Electric Cloud Electric Flow)连续第二年在每个使用案例中排名最高，其中两个使用案例的评估与去年同期相比有所增加。

Gartner 的流程以五分制评估产品。CloudBees(电云)在 2019 年应用发布协调关键能力(ARO)的所有用例中获得了 Gartner 的最高分:发布经理(4.9/5)、应用/产品负责人(4.9/5)、开发运维/应用运营团队成员(4.9/5)。

“ARO 工具使开发运维团队能够自动化应用部署、管理持续集成/持续交付渠道并协调发布工作流，”丹尼尔·贝茨、克里斯·桑德斯、克里斯托弗·利特尔和哈桑·安纳西里在 *Gartner 应用发布协调的关键能力中指出。*“I&O 领导者必须评估这些工具的能力，以改进应用交付节奏并支持数字业务计划。”

CloudBees Flow 是业内首个为企业级 DevOps 打造的统一应用资源编排平台。CloudBees Flow 通过自动化和协调软件发布、管道和部署，并利用分析和洞察力来衡量、跟踪和改善结果，从而帮助提高 IT 效率。

这一愿景和当前的产品领先地位，以及显著的企业客户成功，使 CloudBees 成为 ARO 企业事实上的标准。市场领导者选择 CloudBees Flow 是因为它提供了:

*   一种实施和共享安全、可重复和适应性强的管道的方式，用于交付大型机、单片和微服务应用
*   对企业版本运行状况和状态的前所未有、可操作的洞察，无需手动收集和分析数据
*   该解决方案将随着他们在整个组织中推广开发运维流程而可靠地扩展

“在当今竞争激烈的商业环境中，每个人都有一个共同的目标:更快、更安全地交付更好的软件。CloudBees 首席执行官兼联合创始人 Sacha Labourey 表示:“CloudBees 致力于帮助我们的客户拥有一个软件交付管理系统，帮助他们连接和衡量所有端到端活动，实现他们成为软件第一公司的愿景。“我们的关注得到了回报:CloudBees 再次在 Gartner 2019 年应用发布协调关键能力的所有用例中获得最高分，并在 Gartner 2019 年自动化发布协调魔力象限中四次成为领导者。”

**关于 CloudBees**

CloudBees 通过提供世界上第一个端到端连续软件交付管理系统(SDM)为连续经济提供动力。对于数百万推动各种规模企业创新的开发人员和产品团队来说，SDM 建立在持续集成(CI)和持续交付(CD)的基础上，使软件交付组织内部和周围的所有功能和团队能够最好地合作，以扩大价值创造。

CloudBees 是 CI、CD 和应用程序发布流程编排(ARO)的核心，建立在其产品的商业成功及其开源领先地位之上。CloudBees 是[詹金斯](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fjenkins.io%2F&esheet=52118651&newsitemid=20191029005756&lan=en-US&anchor=Jenkins&index=5&md5=b6365e42b2685f923f2bec0345ffb53f)和[詹金斯 X](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fjenkins-x.io%2F&esheet=52118651&newsitemid=20191029005756&lan=en-US&anchor=Jenkins+X&index=6&md5=5d58d409a17c6b55872a7cfb8b4c5643) 的最大贡献者，也是[持续交付基金会](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fcd.foundation%2F&esheet=52118651&newsitemid=20191029005756&lan=en-US&anchor=Continuous+Delivery+Foundation&index=7&md5=05ea8f754d914cc86c56f5d2cbe64e07) (CDF)的创始成员。从全栈开发人员实践 NoOps 的创业公司到大型财富 100 强公司，CloudBees 使所有软件驱动的组织能够在正确的时间智能地部署正确的功能。

超过 3，500 个世界知名品牌和超过 50%的财富 500 强企业依赖 CloudBees，因为它能够在任何开发环境中跨任何云工作，并平衡公司治理和控制与开发人员的灵活性和自由度。CloudBees 是世界领先的 DevOps 专家的家园，帮助数千家公司利用“持续一切”的力量，将他们置于从伟大的想法到伟大的软件，再到伟大的商业价值的最快路径上。
