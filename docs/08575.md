# 你的技术和你的基础设施一样好

> 原文：<https://devops.com/your-tech-is-only-as-good-as-your-infrastructure/>

几周前，我的一个邻居在他的院子里挖了一条沟，撞上了一条光纤互联网线路，使我的互联网瘫痪了八个小时。我就在那里，准备开始工作，却和同事们隔绝了一整天。

那次经历凸显了良好的[数字基础设施](https://devops.com/?s=infrastructure)的重要性——无论是在办公室还是在家里——尤其是在疫情已经引入了混合工作模式的情况下。当疫情来袭时，灵活且拥有强大数字基础设施(包括流程)的组织能够更好地快速转移到远程工作。那些数字技术不成熟的公司被迫适应。

18 个月后，混合模式显然将继续存在，这使得组织确保其员工拥有正确的技术工具来有效工作变得更加重要，而这一切都始于强大的数字基础设施。

## 仅仅因为它在屏幕上并不意味着它是数字的

让我们从数字的定义开始。我最近与一位企业主进行了一次对话，他自豪地宣称，“我们现在是数字化的了。”实际上，他所做的只是在 iPad 上放了一个 PDF 表格。一旦他的客户填写了 PDF，他仍然打印出来，放在一个文件夹里。这就是我所说的不成熟的数字化愿景，这是围绕数字化转型概念经常存在的模糊性的一个极端例子。

这并不是说纸是数字流程的诅咒；在某些行业(例如医疗保健)，我们需要纸张作为备份。例如，如果我是你床边的医生，停电和/或 Wi-Fi 中断，我无法访问你的数字记录，我需要能够查看你的病历并为你治疗。所以有对纸的需求。但是，当考虑混合或远程工作环境时，目标应该是将流程在线移动。

## **数字基础设施是敏捷性的关键**

如前所述，敏捷企业能够更好地适应变化，比如转向混合工作模式。敏捷仅仅意味着快速、轻松地行动的能力，快速行动并以流畅的方式做出决策的能力。要做到这一点，我们必须删除不必要的步骤，以便我们能够专注于最终目标和流程的重要部分。

这就是强大(可靠)的数字基础设施的用武之地。当你想到马斯洛的需求层次理论时，你会想到数字基础设施的基础是电力和互联网，然后你可以在此基础上进行建设。我需要计算机、硬件和网络，还需要软件平台和最终用户界面。

但是我们超越了自己。让我们回到基础。一旦我们有了可靠的电力和互联网，我们需要寻找其他可以让远程工作人员受益的技术，比如为所有这些变焦会议升级麦克风和摄像头。

这些升级不应该是组织的事后想法；他们是员工混合体验的中心，也是成功的另一个关键。让员工带着笔记本电脑和显示器回家已经不够了。雇主必须关注人体工程学(站立式办公桌、好椅子)和员工在家高效工作所需的硬件。如果雇主不直接提供，他们可能会考虑为员工提供津贴，让他们自己进行升级。

(组织甚至可以考虑向居住在农村地区的工人提供发电机。我家里有一个。一旦 [卫星互联网](https://www.starlink.com/) 服务来到我的城镇，我将拥有一个更加可靠的数字基础设施。)

现在，我们可以开始在基础设施上进行建设。从音频和视频通信开始(缩放、团队、拨号盘)。然后添加聊天(Slack，Google Chat)和一个好的客户关系管理(CRM)系统(Salesforce，HubSpot)。您的数据存储在哪里？拥有一个优秀的 CRM 系统至关重要。

## **用数字以人为本**

组织忽视数字基础设施的重要性，后果自负。不仅员工在一家公司呆上三四十年的日子已经一去不复返，而且我们现在正处于大辞职的[](https://trainingindustry.com/articles/onboarding/covid-shuffled-the-workforce-deck-heres-how-to-come-out-aces/)中，工人们正成群结队地离开他们的岗位。因此，组织有责任倾听员工的心声，并给予他们在混合环境中取得成功所需的一切。

我们谈论的不仅仅是技术。组织需要让员工参与进来，以真正了解组织的脉搏，因为我们都知道，组织的文化和氛围可能会起起落落。对于组织来说，真正倾听员工的要求并保持灵活性是至关重要的，这样他们才能做出反应并适应不断变化的时代。

如果他们不这样做，互联网电缆破裂可能是他们最不担心的。
