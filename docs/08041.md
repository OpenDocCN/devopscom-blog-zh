# Atlan 在由 Insight Partners 牵头的 A 轮融资中筹集了 1600 万美元，用于为数据团队建立一个协作中心

> 原文:[https://devo PS . com/atlan-raises-1600 万次 a 轮融资-由 insight-partners-to-build-a-collaboration-hub-for-data-teams/](https://devops.com/atlan-raises-16m-series-a-round-led-by-insight-partners-to-build-a-collaboration-hub-for-data-teams/)

*在过去 6 个月增长了 16 倍之后，Atlan 计划利用融资来加速产品开发并投资于上市*

**纽约州纽约市，2021 年 5 月 25 日—** Atlan 宣布已完成由全球风险投资和私募股权公司 Insight Partners 牵头的 1600 万美元 A 轮融资。一些备受瞩目的天使投资人也参与了该轮投资，包括雪花公司前首席执行官鲍勃·穆利亚(Bob Muglia)，Stitch 创始人鲍勃·摩尔(Bob Moore)和杰克·斯坦(Jake Stein)，以及 Safegraph 和 Liveramp 的创始人奥伦·霍夫曼(Auren Hoffman)。现有投资者 Sequoia Surge 和 Waterbridge Ventures 也参与了这轮投资。

作为投资的一部分，Insight 董事总经理 Teddie Wardi 将加入 Atlan 的董事会。“随着现代数据栈成为主流，一个协作层作为各种工具的粘合剂，如雪花、数据块、BI 工具和 AI/ML 应用程序，正在成为一种必要，”Wardi 说。“Atlan 为现代数据堆栈带来了真正的协作方法，就像 Figma 为设计团队或 Github 为工程团队所做的那样。我们非常高兴地欢迎 Atlan 加入 Insight，并为团队提供运营、上市和产品开发方面的建议。”

Atlan 最初是 Data for good company，SocialCops 的一个内部项目，孵化了 200 多个数据项目，包括总理本人使用的印度国家数据平台。自从摆脱隐身状态以来，该公司发展迅速，在过去两个季度中增长了 16 倍。Atlan 目前被联合利华和 Scripps Health 等大型企业的团队以及 Postman 等高增长初创公司使用。

Atlan 的数据协作平台在整个现代数据堆栈中创建了统一的发现和协作体验。Atlan 的联合创始人 Prukalpa Sankar 说:“今天，数据资产不仅仅是表格，还有代码、模型、BI 仪表板和管道。“在 Atlan，我们正在用数据重新想象人类的体验——为什么数据资产不能像在 Google Docs 上共享链接那样容易地共享，或者如果 Google Analytics 可以告诉你网站上的使用情况，为什么我们不能对我们的数据做同样的事情？”

“当我们推出我们的现代数据平台时，我们一直在寻找一种产品，使我们的数据更容易民主化，并且不太依赖某个中心人员一次性回答每个分析师的问题。市场上的传统解决方案是针对 IT 或单个数据管理团队拥有数据的传统系统和方法量身定制的。Atlan 是唯一一个为协作、自下而上的方法而构建的解决方案。Atlan 与我们的现代分析堆栈(如 Snowflake 和 Tableau)进行了本机集成，非常容易设置——我们在第一天就让所有数据源流动起来。”Techstyle 的数据和分析副总裁丹妮尔·波格林·拉冈说，tech style 是世界上最大的会员制时装公司之一，拥有多元化的品牌组合，包括 Fabletics、Savage X Fenty、JustFab、FabKids 和 ShoeDazzle。

Atlan 最近被 DataOps 评为 Gartner 2020 年最佳供应商，是全球排名前三的公司之一。

“我们很高兴与 Insight 合作，因为他们在支持 SaaS 公司扩大规模方面的重要专业知识将有助于加快我们的增长，”Atlan 的联合创始人 Varun Banka 说。Atlan 计划利用这笔资金扩大其目前由 40 人组成的营销、销售和客户成功团队，并加快其走向市场的行动。

**关于亚特兰蒂斯**

Atlan 是一个现代化的数据协作工作空间(就像工程领域的 Github 或设计领域的 Figma)。通过充当从表格和仪表板到模型和代码的数据资产的虚拟中心，Atlan 使团队能够为他们的所有数据资产创建单一的真实来源，并通过与 Slack 等工具、雪花和红移等数据仓库、Looker、Sisense 和 Tableau 等 BI 工具、数据科学工具等的深度集成，在现代数据堆栈中进行协作。

作为该领域的先驱，Atlan 被 Gartner 评为 DataOps 领域的优秀供应商，是全球三大公司之一。
