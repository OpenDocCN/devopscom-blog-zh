# 是德科技收购茄子

> 原文:[https://devops.com/keysight-technologies-acquires-eggplant/](https://devops.com/keysight-technologies-acquires-eggplant/)

***联合两家互补的公司，打造智能软件测试自动化市场的创新力量***

是德科技(NYSE: KEYS)是一家领先的技术公司，帮助企业、服务提供商和政府加速创新，以连接和保护世界，数字自动化智能专家[茄子](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.eggplantsoftware.com%2F&esheet=52241444&newsitemid=20200625005633&lan=en-US&anchor=Eggplant&index=1&md5=1baa2f325d76b4f1d74ad9cb092e0341)今天宣布，是德科技已完成从凯雷集团收购茄子。

茄子是行业领先的软件测试自动化平台提供商，使用人工智能(AI)和分析来自动化测试创建和测试执行。茄子的数字自动化智能平台可以在任何层的任何设备、操作系统或浏览器上测试任何技术，从用户界面(UI)到应用程序编程接口(API)再到数据库。

随着今天的数字革命，软件内容和价值呈指数级增长。数百万行代码使智能和互联产品成为可能。企业必须加快数字化转型，以保持竞争力和相关性，同时继续支持传统平台。数字产品所需的测试规模和广度不断扩大，这增加了产品的复杂性，从而缩短了上市时间。通过使用人工智能和分析来自动化测试，茄子提高了软件应用程序的开发速度和质量，推动了竞争优势和差异化。该公司的客户跨越广泛的部门重叠是德科技现有的客户基础，同时扩大软件测试机会到新的终端市场。

是德科技和茄子的结合将两家互补的公司聚集在一起，在自动化软件测试市场创造了一股跨越物理层和协议层并进入应用层的创新力量。此次收购实现了两家公司之间测量技术的双向利用，从而在扩展的产品中增加了解决方案的差异化。

“作为第 1-7 层设计和测试领域公认的领导者和值得信赖的顾问，是德科技很高兴为软件应用层增加茄子的测试功能，这符合我们的战略，以发展我们的首个以软件为中心的解决方案，”是德科技董事长兼首席执行官 Ron Nersesian 说。“我们非常高兴地欢迎茄子团队加入是德科技大家庭，并期待在快速增长的智能软件测试市场中与差异化的软件即服务技术合作。”

茄子公司首席执行官约翰·贝茨博士表示:“与是德科技的合作使茄子公司能够扩展我们的智能自动化平台，并覆盖全球更多的组织。”。“我们拥有加速创新的共同愿景，我们将携手帮助客户踏上数字化转型之旅。我们为我们通过员工和合作伙伴的贡献所取得的成就感到自豪，我们对下一个篇章感到兴奋。"

这笔交易价值 3.3 亿美元。茄子的 2019 年收入为 3800 万美元，其首席执行官约翰·贝茨将加入是德科技的领导团队，向是德科技电子工业解决方案集团总裁 Soon-Chai Gooi 报告。
