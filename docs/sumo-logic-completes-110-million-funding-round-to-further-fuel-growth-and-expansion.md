# Sumo Logic 完成 1.1 亿美元融资，进一步推动增长和扩张

> 原文：<https://devops.com/sumo-logic-completes-110-million-funding-round-to-further-fuel-growth-and-expansion/>

*Battery Ventures、Tiger Global Management 和 Franklin Templeton* *签约* *重回 Sumo Logic 业界领先的持续智能平台*

**加利福尼亚州雷德伍德城——2019 年 5 月 8 日——**[Sumo Logic](https://www.sumologic.com/)，领先的云原生机器数据分析平台，提供[持续智能](https://www.sumologic.com/company/continuous-intelligence/)，今天宣布获得由 Battery Ventures 领投的 1.1 亿美元融资，新投资者包括 Tiger Global Management 和 Franklin Templeton，现有投资者也有参与。Sumo Logic 目前的估值超过 10 亿美元，2019 财年的收入超过 1 亿美元，全球有超过 2，000 家客户利用其服务，因此它牢牢地定位为云时代首选的 SaaS 机器数据分析领导者。

Sumo Logic 总裁兼首席执行官 Ramin Sayar 表示:“Sumo Logic 的增长是由所有行业和所有规模的公司向数字业务和云应用的转变推动的。“我们已经证明，我们不仅是云原生公司的首选平台，也是企业公司及其云迁移计划的首选平台。在我们加快品类领先地位的过程中，拥有如此强大的领先投资者和生态系统合作伙伴真是太好了。”

这笔 1.1 亿美元的资金使迄今为止的资金总额达到 3.45 亿美元。这些投资将继续推动 Sumo Logic 的业务，包括工程、销售和全球运营，重点是扩展其服务的平台分析能力，以进一步满足现代企业的运营、安全和业务需求，因为它们快速采用新的多云和混合云基础设施、架构、工具和流程。

Battery Ventures 普通合伙人 Dharmesh Thakker 表示:“我们跟踪 Sumo Logic 团队已经有一段时间了，我们很钦佩该公司对巨大的云原生机会以及新的现代应用架构的兴起的早期理解。“该公司快速增长的业务与 Battery 关于“OpenCloud”的更大主题相吻合，open cloud 是指开源和云原生技术，与新的云分发模式合作，而不是对抗新的云分发模式，我们很高兴与该团队合作，以抓住未来的重大机遇。”

Franklin Templeton 副总裁 Jonathan Curtis 表示:“我们很高兴在这个全行业向云计算转型的重要时刻加入 Sumo Logic。“作为唯一一个专为应对数字业务挑战而构建的云原生 DevSecOps 平台，我们相信 Sumo Logic 已做好充分准备，可以占领其市场的重要份额。”

自 2017 年 6 月的上一轮融资以来，Sumo Logic 的业务规模显著扩大，这得益于其专注的多年增长战略，包括:

***快速的收入增长和客户获取***-该公司在全球范围内的客户数量增加到 2000 多家，代表了所有主要的垂直行业和公司规模，从诞生于云的领导者，如 Airbnb、Pinterest、Pokemon Co. International、Samsung SmartThings 和 Zuora，到数字化转型企业，如 ABinBev、Adobe、阿拉斯加航空公司、BBC、Genesys、Hearst Media、Infor、Levi's、Marriott、Pitney Bowes 和 USA TODAY NETWORK。

***平台创新和公司扩张***–Sumo Logic 服务是世界上最复杂的多租户、云原生分析平台之一。该公司继续推动平台创新，以帮助满足客户不断变化的需求。几个值得注意的亮点:

*   推出测试版[云 SIEM 解决方案](https://www.sumologic.com/press-release/platform-enhancements/)以实现传统 IT 和包括 DevSecOps 在内的新云模式的安全分析民主化
*   通过[全球情报服务](https://www.sumologic.com/press-release/platform-enhancements/)引入了新的分析洞察类别，旨在将机器学习和基准数据扩展到整个企业的新团队和用例
*   差异化的平台增强功能，可更好地监控和排查基于分布式微服务的架构，如 Docker、Kubernetes 和[亚马逊 EKS](https://www.sumologic.com/application/aws-eks/) 等，以推动更好的客户体验
*   将员工人数扩大到 500 多人，并扩大了在欧洲、日本和整个亚太地区(APAC)的全球足迹，以满足对云原生机器数据分析不断增长的需求

***关键执行领导的加入——在过去的 12 个月里，Sumo Logic 迎来了一批新的领导加入团队。其中包括该公司的第一个独立董事会成员兰迪·戈特弗里德、查克·罗贝尔和梭子鱼首席执行官 BJ·詹金斯。此外，Sumo Logic 还任命 Sydney Carey 为首席财务官，Kiki Haar 为总法律顾问。***

******行业认知度***–Sumo Logic 继续获得重要的行业认可，包括: [2019 年最佳工作场所和财富杂志评选的最佳技术工作场所](https://www.sumologic.com/press-release/best-workplaces-in-technology-great-place-to-work-fortune/)、[福布斯云 100 强榜单](https://www.sumologic.com/press-release/forbes-cloud-100-2018/)、 [JMP 证券 2018 年热门 100 强榜单](https://www.sumologic.com/press-release/jmp-securities-hot-100-list/)、wealth front 2019 Career launch Company、CRN 20 大最酷云安全供应商和商业分析公司等等。***
