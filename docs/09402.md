# HCL Technologies 推出 Kubernetes 迁移平台，加速应用程序现代化

> 原文:[https://devo PS . com/HCl-technologies-launchs-kubernetes-migration-platform-to-accelerate-application-moderation/](https://devops.com/hcl-technologies-launches-kubernetes-migration-platform-to-accelerate-application-modernization/)

*自动化平台支持从云铸造环境进行可重复、可靠且经济高效的过渡*

**西班牙巴伦西亚和印度诺伊达–2022 年 5 月 16 日**–全球领先的技术公司 HCL Technologies (HCL)推出了其 Kubernetes 迁移平台(KMP)，以帮助组织加快应用程序现代化。HCL KMP 是一个自动化解决方案，帮助组织将工作负载从传统的内部部署和云铸造环境迁移到基于 Kubernetes 的现代平台。KMP 的可重复、可靠且经济高效的方法使组织能够降低风险，完成迁移的速度比手动方法快 120 倍。

此外，HCL KMP 使组织能够毫不费力地在基于不同 Kubernetes 平台的混合、多云环境之间迁移工作负载，包括:谷歌 GKE、微软 Azure AKS、亚马逊 EKS、VMWare TKGI、Rancher 和 Red Hat OpenShift。该平台为组织提供了更多的选择自由，确保它们运行在最具成本效益的平台上，并有助于避免供应商锁定。

HCL Technologies 首席技术官兼生态系统负责人 Kalyan Kumar 表示:“企业越来越关注在云环境中维护传统云铸造环境的运营成本。“Cloud Foundry 有限的现代化能力使得在不投入过多时间和资源的情况下很难迁移到 Kubernetes。KMP 为我们的客户提供了一个快速、全自动、无障碍的通往任何基于 Kubernetes 平台的旅程，从而搭建了一座通向未来的桥梁。”
