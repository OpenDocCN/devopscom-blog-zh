所以我们总是有一些 CD 功能，但是随着时间的推移，我们使用 CD 的客户数量或客户的百分比或细分数量都有了相当大的增长，我认为这部分是我们提供的工具种类的影响，部分是 CD 含义的转变，部分是市场的增长和人们做这些的舒适程度，对吗？当然，CD 总是充满挑战，因为有些人在谈论连续交付，有些人在谈论连续部署，但是，特别是在连续部署方面，还必须考虑其他因素，比如标记、金丝雀，以及更好地控制实际投入生产意味着什么。舒适度和舒适度的提高也让更多的人使用我们的 CD 平台。
