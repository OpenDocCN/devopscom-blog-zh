# Zenlayer 在亚马逊网络服务合作伙伴网络中获得了标准技术合作伙伴资格

> 原文:[https://devo PS . com/Zen layer-achieves-standard-technology-partner-status-in-the-Amazon-web-services-partner-network/](https://devops.com/zenlayer-achieves-standard-technology-partner-status-in-the-amazon-web-services-partner-network/)

Zenlayer 的 SD-WAN 和云连接服务提供了私有的低延迟连接

洛杉矶，2018 年 10 月 2 日–全球软件定义的网络和服务提供商 Zenlayer 今天宣布，它已在亚马逊网络服务(AWS)合作伙伴网络(APN)中获得标准技术合作伙伴地位。这一殊荣表明 Zenlayer 在帮助企业客户构建直接连接到 AWS 的混合云解决方案方面提供了业经验证的专业知识、技术熟练程度和成功经验。

作为 APN 标准技术合作伙伴，Zenlayer 获得了 AWS 工具、培训和支持，以更好地根据客户的 AWS 要求定制其云连接服务。这反过来又允许 Zenlayer 的 Zenlink SD-WAN 和 Cloud Connect 解决方案与 AWS 无缝协作。此外，Zenlayer 现在可以扩展其未来云网络集成的选项，并将其云连接服务的可用性扩展到 AWS 客户。

Zenlink 允许各垂直行业的企业建立从其场所到 AWS 的专用网络连接，连接数据中心和办公室。它的一些好处包括能够更好地监控网络质量，并查看哪些应用程序消耗了最多的带宽。此外，企业将能够安全可靠地在云中处理其工作负载，同时降低网络成本并提高带宽吞吐量。Zenlayer 连接到七个 AWS 地区，包括弗吉尼亚州阿什伯恩；德克萨斯州达拉斯/沃斯堡；加利福尼亚州圣何塞；加利福尼亚州洛杉矶；日本东京；新加坡；德国法兰克福，中国和巴西也将紧随其后。

“我们很荣幸加入 AWS 合作伙伴网络，为使用 AWS 的企业提供云连接解决方案，”Zenlayer 首席执行官兼创始人 Joe Zhu 表示。“AWS 以其强大的按需云计算服务而闻名，被认可为 APN 标准技术合作伙伴反映了我们对满足客户不断发展的云需求的坚定承诺。我们期待着发展我们的关系，以帮助更多企业实现云的全部优势。”

**关于 Zenlayer**

Zenlayer 是一家软件定义的网络和服务提供商，总部位于洛杉矶和上海，在新加坡、香港、北京和深圳设有办事处。企业利用 Zenlayer 的软件定义平台即时实现全球连接并提供卓越的用户体验。Zenlayer 在六大洲的 90 多个数据中心提供按需裸机云、SD-WAN 云连接、边缘计算和主机托管。
