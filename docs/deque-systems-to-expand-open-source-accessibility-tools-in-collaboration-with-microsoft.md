# Deque Systems 将与微软合作扩展开源辅助工具

> 原文：<https://devops.com/deque-systems-to-expand-open-source-accessibility-tools-in-collaboration-with-microsoft/>

Deque 的 axe 可访问性引擎得到了微软的支持，增加了新的 Windows 规则，并发布了 Deque 的开源 Android 和 iOS 规则

March 12, 2019 12:00 PM Eastern Daylight Time

加利福尼亚州阿纳海姆([商业资讯](https://www.businesswire.com/))——专注于数字平等的领先辅助功能软件公司 Deque Systems 在 [CSUN 2019](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.csun.edu%2Fcod%2Fconference%2F2019%2Fsessions%2Findex.php%2Fpublic%2Fwebsite_pages%2Fview%2F1&esheet=51953342&newsitemid=20190312005303&lan=en-US&anchor=CSUN+2019&index=1&md5=ed3b87442c4620b1d976fde2f56a5c8b) 上宣布，它正在与微软合作，为 [axe 辅助功能规则引擎](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.deque.com%2Faxe%2F&esheet=51953342&newsitemid=20190312005303&lan=en-US&anchor=axe+accessibility+rules+engine&index=2&md5=cc08224f8e688b5e373ee22c2e29931d)添加 Windows 平台支持。

Axe 拥有世界上部署最广泛的 Web 可访问性规则引擎，目前已部署到 2500 万台设备上。微软对 axe 的新的和重要的 Windows 贡献扩展了这个被广泛采用的事实上的 Web accessibility ruleset 标准。

现在，跨 Web、Windows、Android 和 iOS 平台的开发人员都可以使用一种通用、统一的方法测试他们开发的代码，使他们能够防止关键问题影响世界各地的残疾人。

“为了这一刻，我们已经工作了 20 年，”德克系统公司首席执行官 Preety Kumar 评论道。“Deque 的核心使命是让可访问性成为世界上每个开发和设计团队的日常责任。这一合作是实现这一目标的重要一步。”

微软开发人员服务部总经理 Keith Ballinger 表示:“微软和 Deque 对更具包容性的数字未来有着共同的愿景。我们很高兴扩大与 Deque 的合作，因为这是我们设计、制造和推出更易访问的产品之旅的重要组成部分。”

除了 axe 的贡献之外，微软还宣布了[Accessibility Insights](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Faka.ms%2FAA4gs43&esheet=51953342&newsitemid=20190312005303&lan=en-US&anchor=open+sourcing+of+Accessibility+Insights&index=3&md5=96e6cdd44885b58c5a07b534303c6c8a)的开源，这是一套让开发者能够构建更具包容性的解决方案的工具。

我们邀请世界各地的开发人员通过将这些开源工具集成到他们的内部开发循环中来改变易访问性。齐心协力，我们就能实现数字平等，让每个人都能进入数字世界。

要了解更多信息，请访问 https://www.deque.com/axe/。
