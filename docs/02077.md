# 2019 DEVOPS WORLD |詹金斯世界奖获奖名单揭晓

> 原文:[https://devo PS . com/2019-devo PS-world-Jenkins-world-award-winders-announced/](https://devops.com/2019-devops-world-jenkins-world-award-winners-announced/)

*15 个组织和个人因出色的 DevOps 成就、开源贡献和渠道合作伙伴表现而获奖*

**DEVOPS WORLD | JENKINS WORLD，SAN FRANCISCO****–2019 年 8 月 15 日—**[cloud bees](https://www.cloudbees.com)，为持续经济提供动力的企业 DEVOPS 领导者，今天宣布了[2019 devo PS WORLD | JENKINS WORLD Awards](https://www.cloudbees.com/devops-world/san-francisco/awards)的获奖者。该奖项表彰三个方面的成就:詹金斯社区奖，表彰那些对詹金斯项目做出重大贡献的人；CloudBees 创新奖，表彰在 DevOps 转型中取得优异成绩的组织；CloudBees 渠道合作伙伴奖，旨在表彰在创收和帮助组织实现 DevOps 转型方面表现出色的合作伙伴。

CloudBees 创新奖获得者包括各种规模的组织，从小型风险投资创业公司到全球科技巨头。詹金斯社区奖授予了四位对詹金斯社区做出贡献的个人。最后，五家渠道合作伙伴获得了行业卓越奖。

**詹金斯社区奖**

詹金斯社区奖获奖者是由詹金斯创始人兼 CloudBees 首席科学家 Kohsuke Kawaguchi 和詹金斯大使组成的小组选出的。詹金斯大使每年提名一次，他们是有影响力的詹金斯爱好者，热衷于与他人分享他们的詹金斯技术专长。

“2019 年詹金斯社区奖表彰了社区成员的努力，他们在几个关键领域为推进詹金斯项目做出了重大贡献，”川口说。"今年提交的作品质量比以往任何时候都要好，各方面都是如此。"

2019 年詹金斯社区奖获奖者:

*   2019 年最有价值詹金斯贡献者——Oliver gonda
*   2019 年詹金斯安全 MVP——维克多·加斯达格
*   2019 最有价值詹金斯倡导者——赵晓洁
*   2019 年最具创新性的 Jenkins X 实施–Vincent Behar

**CLOUDBEES 创新奖**

CloudBees 创新奖获奖者由 CloudBees 首席执行官兼联合创始人 Sacha Labourey 组成的评委团选出；艾伦·希梅尔，*DevOps.com*主编兼创始人；以及谷歌云产品管理总监 Aparna Sinha。

“我们很高兴有机会庆祝 CloudBees 客户定期展示的杰出成就，”CloudBees 首席执行官兼联合创始人 Sacha Labourey 说。“2019 年 DevOps World | Jenkins World Award 获奖者是 DevOps 行业杰出领导力的五个例子。”

2019 年 CloudBees 创新奖获奖者:

*   DevOps 可扩展性成就–AIG
*   DevOps 可扩展性成就–埃森哲
*   DevOps 自动化卓越——美国运通全球商务旅行

 **关于 CloudBees**

CloudBees 通过提供世界上第一个端到端连续软件交付管理系统(SDM)为连续经济提供动力。对于数百万推动各种规模企业创新的开发人员和产品团队来说，SDM 建立在持续集成(CI)和持续交付(CD)的基础上，使软件交付组织内部和周围的所有功能和团队能够最好地合作，以扩大价值创造。

CloudBees 是 CI、CD 和应用程序发布流程编排(ARO)的核心，建立在其产品的商业成功及其开源领先地位之上。CloudBees 是[詹金斯](https://jenkins.io/)和[詹金斯 X](https://jenkins-x.io/) 的最大贡献者，也是[持续交付基金会](https://cd.foundation/) (CDF)的创始成员。从全栈开发人员实践 NoOps 的创业公司到大型财富 100 强公司，CloudBees 使所有软件驱动的组织能够在正确的时间智能地部署正确的功能。

超过 3，500 个世界知名品牌和超过 50%的财富 500 强企业投资 CloudBees，因为它能够跨任何云、在任何开发环境中工作，并平衡公司治理和控制与开发人员的灵活性和自由度。CloudBees 是世界领先的 DevOps 专家的家园，帮助数千家公司利用“持续一切”的力量，将他们置于从伟大的想法到伟大的软件，再到伟大的商业价值的最快路径上。

CloudBees 由 Matrix Partners、Lightspeed Venture Partners、威瑞森风险投资公司、Delta-v Capital、Golub Capital 和 unregular Ventures 支持，于 2010 年由前 JBoss CTO Sacha Labourey 和一个由持续集成、持续交付和 DevOps 专业人士组成的精英团队创立。在 [Twitter](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ftwitter.com%2FCloudBees&esheet=51911854&newsitemid=20181211005061&lan=en-US&anchor=Twitter&index=7&md5=cbb3ff10b2efbe78aa6dc1cb8917c822) 、[脸书](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.facebook.com%2FCloudBees&esheet=51911854&newsitemid=20181211005061&lan=en-US&anchor=Facebook&index=8&md5=dc4c3d999fa66ccfa43dab58dd19fce2)和 [LinkedIn](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.linkedin.com%2Fcompany%2Fcloudbees&esheet=51911854&newsitemid=20181211005061&lan=en-US&anchor=LinkedIn&index=9&md5=c6fdd30c619f62cfd747e8a2d20db634) 上关注 CloudBees。
