# Clearlake Capital 支持的 Perforce Software 收购 Perfecto Mobil

> 原文:[https://devo PS . com/clear lake-capital-backed-perforce-software-to-acquire-perfecto-Mobil/](https://devops.com/clearlake-capital-backed-perforce-software-to-acquire-perfecto-mobil/)

增强的 Perforce 解决方案组合包括基于云的大规模连续自动化测试，适用于企业移动和 web 应用

加利福尼亚州明尼阿波利斯和圣莫尼卡 2018 年 10 月 8 日电/美通社/ —由 Clearlake Capital Group，L.P .(及其附属公司“Clearlake”)支持的企业级 DevOps 软件解决方案全球提供商 Perforce Software(“Perforce”)今天宣布，已达成最终协议，收购基于云的自动化移动和 web 应用程序测试软件解决方案的市场领导者 Perfecto Mobile(“Perfecto”)。此次收购扩大了 Perforce 的软件产品组合，为企业 DevOps 团队提供了额外的功能，以实现跨 web、移动和物联网应用的大规模连续测试。此次收购预计将于今年完成。

Perfecto 是行业先锋，通过其全球可访问的持续质量实验室提供唯一真正的企业级移动和网络自动化测试软件平台。该公司是需要加速数字化转型并为客户提供最佳在线移动和网络体验的大型组织的首选提供商。银行、零售、电信和保险领域的领先企业依靠 Perfecto 强大的自动化测试软件更快地提供优质体验，并在竞争激烈的市场中留住用户。

Perforce 首席执行官 Mark Ties 表示:“企业继续需要更多自动化来扩展和加速应用交付。“Perfecto 为我们的 DevOps 功能增加了一个关键组件，使 Perforce 能够为重视可扩展性和安全性的大型企业客户提供更多价值。”

Perforce 继续向其软件产品组合添加功能，以独特的方式满足技术开发团队的需求，这些团队面临多方面的规模挑战，但仍必须快速交付产品。对 Perfecto 的收购将是 Perforce 在过去两年中的第五次收购。

Clearlake 的 Prashant Mehrotra 和 Paul Huber 补充说:“通过这次战略性收购，Perforce 能够充分利用企业开发运维团队持续测试需求的重要市场趋势。“我们很高兴能继续支持 Mark 和 Perforce 管理团队，因为他们推动了行业整合，加速了有机增长。”

“Perfecto 是企业客户首选的移动和网络测试软件平台。数字化转型正在推动 DevOps 及其活动(如持续集成、测试自动化和持续交付)的加速采用，以更快地在线捕获和吸引客户。Perfecto 联合创始人兼首席执行官 Eran Yaniv 表示:“我们的持续质量实验室能够帮助企业消除瓶颈，快速验证和交付吸引客户的移动和网络应用。“我们很高兴加入 Perforce 团队，并相信这种合作关系将为我们提供更多资源，以利用持续测试市场机会。”
