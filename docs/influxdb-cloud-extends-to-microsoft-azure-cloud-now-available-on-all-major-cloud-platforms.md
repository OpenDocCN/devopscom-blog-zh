# InfluxDB Cloud 扩展到 Microsoft Azure Cloud —现在在所有主要的云平台上都可用

> 原文：<https://devops.com/influxdb-cloud-extends-to-microsoft-azure-cloud-now-available-on-all-major-cloud-platforms/>

*InfluxData 希望通过战略产品集成，利用 PTC 加速物联网应用开发*

旧金山——2020 年 7 月 22 日——[时间序列数据库 InfluxDB 的创建者 InfluxData](https://www.influxdata.com/) 今天宣布，InfluxDB 云现已在微软 Azure 上可用，进一步兑现了该公司增加开发者可访问性的承诺。随着这一宣布，InfluxDB Cloud 现已在所有三大云平台上上线——微软 Azure、谷歌云和亚马逊网络服务。此外，InfluxDB 将成为 PTC 市场领先的 ThingWorx 工业物联网(IIoT)平台的时序引擎。

“InfluxDB 云使开发人员能够从他们的数据中获得新的见解，并通过基于使用的定价，确保他们永远不必担心过度配置的额外成本或在需要时应用程序不可用的风险，”InfluxData 首席执行官 Evan Kaplan 解释道。“微软 Azure 是主流云平台之一，它的加入进一步扩展了我们云产品的全球覆盖范围。除了与 PTC 的战略联盟，我们还创建了一个全面的市场，为开发人员提供最快的成果。借助 InfluxDB Cloud，开发人员可以在几分钟内启动并运行，并根据需要在他们选择的云上轻松扩展他们的物联网或开发运维项目。”

**InfluxDB Cloud 现已在所有主要云平台上可用**

最近向远程工作的突然转变加速了向云的迁移，企业领导者越来越多地寻求云服务来支持数字化转型。这发生在许多公司——意识到供应商锁定并热衷于抽象他们的应用程序以便他们可以跨云移动——倾向于多云战略的时候。现在，InfluxDB Cloud 可以在所有主要的云平台上使用，它是作为云原生的，在 Kubernetes 上作为有状态的应用程序运行。这使得它成为第一个也是唯一一个无服务器的时间序列数据平台，具有灵活透明的基于使用的定价和免费层，以扩大对所有开发人员的访问。

“当今公司的两个主要增长领域是时序数据库和云部署。在微软 Azure 上部署 InfluxDB 云为客户开发和运行他们的应用程序提供了一个理想的环境，”微软全球战略联盟负责人 Ayelet Steinitz 表示。“组织可以加快应用程序的部署，并从统一计费和支持以及与关键 Microsoft Azure 技术的集成中受益。”

**InfluxData & PTC 致力于加速 IIoT 解决方案的开发**

InfluxData 与 PTC 的合作旨在帮助公司加速 IIoT 解决方案的开发。InfluxDB Cloud 将由 PTC 直接销售、交付和支持，用于在 PTC 的云环境中托管的 ThingWorx 的部署。PTC 将其 ThingWorx 平台与微软 Azure 深度集成，后者是其首选的工业领域云平台。

PTC 公司 ThingWorx 产品管理副总裁克里斯·鲍德温(Chris Baldwin)表示:“我们相信，开发和部署的速度对于推出新的 IIoT 解决方案至关重要。“通过将 InfluxDB 嵌入 ThingWorx，InfluxData 和 PTC 旨在简化和加速开发人员在其 IIoT 部署的多个领域中收集、处理、存储和分析更大时间序列数据工作负载的能力。"

将 InfluxDB 嵌入 ThingWorx 将使客户能够从数万个联网设备和传感器中生成数据洞察，每个设备和传感器都有大量测量结果，每秒钟生成数十万次写入。
