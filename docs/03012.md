# DevOps Institute 完成了“2019 技能提升:企业 DevOps 技能报告”调查

> 原文：<https://devops.com/the-devops-institute-fields-the-2019-upskilling-enterprise-devops-skills-report-survey/>

*首次在全球企业 IT 社区中开展的基线和跟踪开发运维技能的调查获得了广泛的行业支持*

佛罗里达州 BOCA 拉顿，2018 年 10 月 22 日–DevOps 专业人士协会 DevOps Institute (DOI)今天宣布与创始白金赞助商 Electric Cloud 和创始金牌赞助商 CloudBees 合作推出“[2019 提升技能:企业 devo PS 技能报告](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=5eXH0qSKdBRphS6WO4YlQwwSTi8wtQOqejFjZKiddbCzZ5Bbl4XDYaFzCBlKjmMmr%2BznpL6dRwXOn3Kvl6joPdh8%2F7nVO5VYzPY23rRW7TWIJztIIae9Ew%3D%3D&G=0&R=https%3A%2F%2Fwww.research.net%2Fr%2FK9WD2XC&I=20181022130201.00000003198a%40mail6-43-usnbn1&X=MHwxMDQ2NzU4OjViY2NiNDEwM2RjYjc1OTNjYjY5OTRjYzs%3D&S=mcw3ykbX89HvasFaCOwLBS25CRHBn1sWKigaeSwIKg8)”社区调查。该调查和由此产生的报告旨在帮助公司更好地了解全球 IT 社区开发技能和交叉技能的当前和未来状态。使用数据科学结合定性分析，DOI 发布该报告的目标是提供切实的见解，说明 IT 从业者、经理、领导者和业务利益相关者为什么以及在哪里应该集中精力发展个人和组织技能组合。Forrester 前副总裁兼研究总监 Eveline Oehrlich 将作为该报告的首席研究分析师，带来她的数据科学专业知识。

“ [2019 技能提升:企业开发技能报告](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=5eXH0qSKdBRphS6WO4YlQwwSTi8wtQOqejFjZKiddbCzZ5Bbl4XDYaFzCBlKjmMmr%2BznpL6dRwXOn3Kvl6joPdh8%2F7nVO5VYzPY23rRW7TWIJztIIae9Ew%3D%3D&G=0&R=https%3A%2F%2Fwww.research.net%2Fr%2FK9WD2XC&I=20181022130201.00000003198a%40mail6-43-usnbn1&X=MHwxMDQ2NzU4OjViY2NiNDEwM2RjYjc1OTNjYjY5OTRjYzs%3D&S=mcw3ykbX89HvasFaCOwLBS25CRHBn1sWKigaeSwIKg8)”调查截止到 2018 年 12 月 7 日，相应的报告将于 2019 年初发布。除了为全球 DevOps 社区做出贡献之外，任何拥有至少 20 名调查受访者的组织都将在报告发布时收到其技能概况的快照。DOI 很高兴能够验证整个全球 IT 社区的 DevOps 技能培养和跨技能培养需求，同时也支持各个组织，了解他们目前的状况和未来的发展方向。

DevOps Institute 首席执行官 Jayne Groll 表示:“T 是转型的形状，数字化转型需要 IT 组织所有级别的跨职能技能。“然而，从深度熟练的专家到跨职能的 T 型从业者的转变，让许多 IT 人员困惑于他们应该获得和雇佣什么技能以及何时获得和雇佣。希望这项研究将提供一些数据支持的清晰度。”

“DevOps 运动是历史上最大的业务转型之一的核心，使组织能够以前所未有的速度进行创新，”Electric Cloud 首席执行官 Carmine Napolitano 解释说。“如果团队无法获得或培养正确的技能类型，这种转变将永远无法实现其潜力。当 DevOps Institute 找到我们来赞助这项旨在确定这些技能以便我们的行业能够解决这些差距的研究时，我们抓住了参与的机会。我们相信，这项研究将创造一条数据驱动的途径，来开发一个新的高技能人才库，能够驱动而不是反应这些市场力量。这一举措延续了电云通过与 Gene Kim 和 IT Revolution 共同创办 DevOps 企业峰会及其 c9d9 教育播客系列来支持 DevOps 社区的悠久历史。”

“当我与全球企业的 DevOps 领导者和团队交谈时，有一件事是明确的:DevOps 之旅是一场涉及流程和技术的真正变革。然而，最重要的资产是拥有合适技能的人，以应对任何 DevOps 转型的多方面挑战，”CloudBees 首席执行官兼联合创始人 Sacha Labourey 说。“这些技能不一定是五到十年前确保成功的技能。我们渴望与 DOI 合作开展这项研究，以发现组织领导其最重要的战略创新和增长计划所需的微妙技能。作为 DevOps 行业和 Jenkins 社区的领导者，我们期待支持这项调查，并帮助企业成为通过软件提供商业价值的一流企业。”

**关于 DevOps 学院**
DevOps 学院(DOI)是 devo PS 专业人士的*协会，鼓励将持续学习作为一种生活方式。会员可以在全球范围内获得最具创新性、启发性和变革性的 DevOps 网络和职业发展机会。*
