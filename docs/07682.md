自 2019 年测试自动化大学成立以来，Applitools 已经收到了对额外资源的请求，以帮助人们快速找到常见日常问题的答案。现在，有了自动化食谱，就有了在一个地方提供少量内容的地方，而不需要在冗长的视频教程、在线论坛和问答线程的无尽网络中进行筛选，而只是以支离破碎的代码示例结束。

Fayock 说:“根据社区的反馈，我们的自动化指南的目标是帮助工程师用可视化编码方法获得常见问题的答案。

自动化食谱的发布包括 12 个视频，展示了两个最著名的 web 应用功能测试自动化框架，Selenium WebDriver 和 Cypress。每个框架总共有六个视频开始。我们将定期添加新视频，并计划在未来推出更多框架。视频长度从 3-10 分钟不等。
