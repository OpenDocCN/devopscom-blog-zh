在本次 DevOps 聊天中，我们采访了 GitLab 首席执行官 Sid Sijbrandij。Sid 作为联合创始人带领 GitLab 到达了 DevOps 市场的顶峰。加入 Sid 和我的是 Ashish Kuthiala，他最近加入了 GitLab。GitLabs 对 DevOps 工具链的愿景的[完整性以及该公司想要提供的东西一直给我留下深刻印象。Sid 对此的设想是他称之为并发 DevOps。在我们的聊天中，以及在接下来的几个月中，您将会听到更多关于并发 DevOps 的内容。](https://devops.com/gitlab-acquires-gemnasium-to-advance-devsecops/)
