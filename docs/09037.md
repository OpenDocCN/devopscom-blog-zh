# 云数据库需求刺激 ScyllaDB 增长 198%

> 原文：<https://devops.com/cloud-database-demand-spurs-198-growth-for-scylladb/>

*ScyllaDB 是数据密集型应用程序的首选数据库，这些应用程序需要在极端规模下保持一致的高性能*

加利福尼亚州帕洛阿尔托，2022 年 1 月 25 日— ScyllaDB 是一家为需要高性能和低延迟的数据密集型应用提供数据库服务的公司，该公司今天宣布其净新增收入同比增长 106%。这一增长大部分来自 [Scylla Cloud，该公司的数据库即服务(DBaaS)](https://www.scylladb.com/product/scylla-cloud/) ，比 2021 年增长了 198%。

令人印象深刻的 2021 年结果是对越来越多面临快速数据增长挑战的改变游戏规则的组织的直接回应。ScyllaDB 数据库提供开源、企业和云选项，具有深度架构改进，能够以极高的规模和效率实现一致的高性能。对于那些经历了第一代 NoSQL 数据库(如 Apache Cassandra、MongoDB)的限制，并需要一种完全不同的方法来驱动其应用程序的核心工程师来说，它越来越成为首选数据库。

值得注意的 2021 年里程碑包括:

*   欢迎社交媒体、食品配送、旅行规划、运动跟踪、电信、广告技术和网络安全领域的行业领袖成为新客户
*   除 AWS 外，在 GCP 发布 Scylla Cloud
*   在 AWS 市场上提供 Scylla 云和 Scylla Enterprise
*   推出业界首个在客户自己的云帐户中运行的完全托管的 NoSQL 数据库即服务(BYOA)
*   在 [锡拉大学](https://university.scylladb.com/) 取得创纪录的注册人数和参与度，实现了认证同比增长 165%
*   完成广泛的 [基准测试](https://www.scylladb.com/product/benchmarks/)vs cocroach db 和 Apache Cassandra 4.0
*   表彰 Zillow、GE Healthcare、Disney+ Hotstar 和 Fanatics 等用户的成就，颁发 ScyllaDB 用户奖
*   发布 [P99CONF](https://www.p99conf.io/) ，一个高度互动的虚拟会议吸引了痴迷于 P99 百分位和高性能、低延迟应用的 5K 开发者

ScyllaDB 首席执行官兼联合创始人 Dor Laor 解释说:“COVID 的一个有趣的业务影响是构建和加速数据密集型应用程序的需求大幅增长。“随着 B2B 和 B2C 对在线服务的期望激增，组织变得数字化高效的需求也显著增加。这推动了我们所有产品的大幅增长。对于我们的组织来说，2021 年最显著的影响之一是来自行业颠覆者的需求增加:利用数据改变游戏的组织(例如，Discord、Disney+ Hotstar、Expedia、FireEye 和 Zillow)。他们选择 ScyllaDB 是因为我们能够以超低延迟提供一致的高性能。”

**关于 ScyllaDB**

ScyllaDB 是用于需要高性能和低延迟的数据密集型应用的数据库。它使团队能够利用现代基础设施不断增长的计算能力，消除随着数据增长而扩展的障碍。与任何其他数据库不同，ScyllaDB 采用深度架构改进构建，以极低的成本实现卓越的最终用户体验。超过 400 家改变游戏规则的公司，如 Disney+ Hotstar、Expedia、FireEye、Discord、Crypto.com、Zillow、Starbucks、Comcast 和 Samsung，使用 ScyllaDB 来应对最棘手的数据库挑战。ScyllaDB 是免费的开源软件，是一个完全受支持的企业产品，也是多个云提供商的完全托管服务。详情:[](https://www.scylladb.com/)
