# 大使实验室宣布推出新的大使云团队版

> 原文:[https://devo PS . com/ambassador-labs-announces-new-ambassador-cloud-team-edition/](https://devops.com/ambassador-labs-announces-new-ambassador-cloud-team-edition/)

*利用 Kubernetes* 的新功能和定价模式来释放整个团队的开发人员生产力

**KubeCon + CloudNativeCon EU，BOSTON—2022 年 5 月 16 日** **—** [大使实验室](http://www.getambassador.io/?utm_campaign=c2-22&utm_content=pr) ，云原生开发者体验领袖，今日公布了 [大使云](https://www.getambassador.io/products/ambassador-cloud/?utm_campaign=c2-22&utm_content=pr) 最新发布。大使云建立在领先的开源云本地计算基金会(CNCF)项目之上，包括使者入口、网真和 Argo，它包括新的功能，可以提高开发人员团队的工作效率，并可以自由地利用他们已经用于 Kubernetes 整个应用生命周期的技术和工具。新功能包括 Docker 远程呈现扩展，可提高分布式团队开发人员的工作效率和协作能力，能够以前所未有的速度编码、调试和部署应用程序。此外，新的定价和打包模式使得整个团队在大规模编码、测试、交付和运行云原生应用时，比以往任何时候都更容易加速 Kubernetes 的采用。

为了加强 Ambassador Labs 对 Kubernetes 开发人员教育的承诺，该公司还推出了一项新的免费技术午餐和学习活动，以帮助团队使用成熟的架构模式和可信的开源工具来采用 Kubernetes。被命名为“ [”的黄金路径披萨派对](https://www.getambassador.io/the-golden-path-pizza-party/) ”，这是面向全球组织的最新计划，旨在解决云本地团队在绘制其“黄金路径”时面临的特定技术挑战。要了解更多信息并请求课程，请点击 [此处](https://www.getambassador.io/the-golden-path-pizza-party/) 。

**关于大使实验室**

Ambassador Labs 是云原生开发人员体验的领导者，使开发人员能够比以往更快、更轻松地编写、发布和运行应用。作为顶级云本地计算基金会(CNCF)开源项目(包括使者入口和网真)的制造商，Ambassador Labs 为 Kubernetes 提供了一个开发人员控制平面，该平面为全球的开发人员和组织(包括微软、PTC、NVidia 和 Ticketmaster)集成了开发、部署和生产基础设施。大使实验室得到了包括 Insight Partners 和 Matrix Partners 在内的顶级投资者的支持。在[www . getambassador . io](http://www.getambassador.io/?utm_campaign=c2-22&utm_content=pr)了解更多，免费上手。
