> 原文:[https://devo PS . com/smart bear-is-only-vendor-sconded-in-top-4-product-scores-for-all-5-use-cases-in-Gartner-critical-capabilities-for-software-test-automation/](https://devops.com/smartbear-is-only-vendor-scored-in-top-4-product-scores-for-all-5-use-cases-in-gartners-critical-capabilities-for-software-test-automation/)

SmartBear Recognized in Capabilities in Continuous Testing, Open Source-Based Testing Acceleration, Progressive Web Apps/Responsive Web/Native Apps, API/Web Service Testing, and End-to-End Testing Use Cases

马萨诸塞州萨默维尔。—2018 年 3 月 26 日— [团队软件质量工具的领导者 SmartBear](https://smartbear.com/) ，是 Gartner 2018 年[软件测试自动化关键能力](https://www.gartner.com/doc/3852266/critical-capabilities-software-test-automation)[【1】](https://smartbear.com/news/news-releases/smartbear-is-only-vendor-scored-in-top-4-product-s/#_ftn1)中唯一一家在[所有五个用例](https://www.gartner.com/document/3852266)的产品得分中排名前四的厂商。[全球领先的信息技术研究和咨询公司 Gartner](https://www.gartner.com/technology/home.jsp) 分析了持续测试、基于开源的测试加速、渐进式 Web 应用/本地应用/响应式 Web、API/Web 服务测试和端到端测试等用例领域的 10 家厂商。SmartBear 相信，这是对 SmartBear 长期承诺的认可，smart bear 致力于提供测试自动化工具，这些工具是健壮的，并且易于由具有任何水平的测试经验、技术专业知识的软件质量团队在任何平台上尝试、实现和使用。

[TestComplete](https://smartbear.com/product/testcomplete/overview/) 让奥莱利轻松地将数千次手动测试转换为自动化测试，无缝集成到我们的测试生态系统中，[奥莱利汽车零部件](https://corporate.oreillyauto.com/corporate/)的 QA 自动化主管[卡森·安德伍德](https://www.linkedin.com/in/cunderw/)说。“TestComplete 允许我的团队扩展我们的自动化工作，并确保我们向所有客户发布高质量的产品，同时大大缩短我们的测试周期。”

SmartBear 与以下其他提供商一起被列入关键能力报告:CA Technologies、Tricentis、Microsoft、IBM、TestPlant、Parasoft、Micro Focus“统一功能测试、Micro Focus”(Silk Test)和 Ranorex。除了产品内的优势之外，与 SmartBear 合作还带来了其他优势，如指定的客户成功代表、进入[活跃社区](https://community.smartbear.com/)并在帖子发布后 60 分钟内回复，以及免费培训和支持。

SmartBear 产品、测试和开发副总裁[瑞恩·洛伊德](https://www.linkedin.com/in/ryanlloyd/)表示:“SmartBear 产品使桌面、web、移动、API、物联网等领域的 UI 和 API 测试自动化变得快速而简单。“我们活跃的社区、卓越的客户支持和入职体验让团队能够快速启动和运行。总之，我们的产品组合和客户服务使客户能够在一个交付高质量软件的压力越来越大的环境中更快地实现价值。”

SmartBear 继续为客户带来巨大的成功，并扩展了软件测试自动化工具的产品组合。
