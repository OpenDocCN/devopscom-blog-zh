# Domo 加入雪花的雪地公园加速计划

> 原文:[https://devo PS . com/domo-joins-snow flags-snow park-accelerated-program/](https://devops.com/domo-joins-snowflakes-snowpark-accelerated-program/)

*Domo 宣布与 Snowpark 进行新的集成，使开发人员、数据工程师和数据科学家能够更轻松地在整个企业中构建和扩展定制的数据驱动解决方案*

**犹他州 Silicon Slopes，2021 年 6 月 10 日—**Domo 商业云提供商 Domo (Nasdaq: DOMO)今天宣布支持 Snowpark，这是数据云公司 T2 雪花的新开发人员体验，旨在提供联合解决方案，使开发人员、数据工程师和数据科学家能够更轻松地构建和扩展整个企业的定制解决方案。作为此次发布的一部分，Domo 引入了与 Snowpark 的新集成，Snowpark 是雪花的新开发人员体验，可以更容易地使用 Scala 和 Java 等其他语言在 Domo 的现代 BI 平台内构建定制解决方案。此外，Domo 宣布支持雪花的 Java/Scala 用户定义函数(UDF)，客户可以在 Domo 平台上运行代码编辑器接口来创建和编辑他们的 UDF。

借助 Snowpark 和 Java/Scala UDF，Snowflake 和 Domo 正在利用定制代码的力量来解决最复杂的业务问题，使共同客户能够从他们的数据中获取更多价值，并为组织内外的任何人提供强大且可操作的数据洞察。

“Domo 和雪花的共同使命是为客户提供从海量数据中获取更多价值所需的灵活性、可扩展性和安全性，无论这些数据位于何处。作为 Snowpark 加速计划的一部分，Domo 将提供一个集成和定制的界面，在 Domo 平台上运行 Snowpark 和 Java/Scala UDF，使用户能够使用更多工具来进一步管理他们的整个数据生态系统，并将数据应用于整个企业，”Domo 首席技术官 Daren Thayne 说。

“我们最新的 Snowpark 公告延续了雪花移动世界数据的使命，让更多用户能够发现强大的数据洞察力，”全球合作伙伴和联盟的雪花 SVP 说。“我们与 Domo 的持续合作伙伴关系通过提供丰富的见解扩展了大量数据的价值，这些见解可以推动任何企业做出更好的决策和行动。”

最近，Domo 宣布 [Domo for Snowflake](https://www.domo.com/news/press/domo-announces-new-native-snowflake-integration) ，这是一个丰富的原生集成，将现代 BI 直接置于 Snowflake 平台之上，允许客户将分析工作负载的数据直接保存在 Snowflake 数据云中。客户可以直接读写雪花，创建数据库，并根据他们建立的模式管理对数据库的访问，同时使用 Domo 使数据对企业中的任何人都更容易访问和操作。Domo 的现代 BI 平台不仅确保了数据的可访问性，还确保了数据的妥善管理，让 BI 和数据专业人员有信心利用智能应用和 Domo Everywhere 等 Domo 解决方案在组织内外释放数据。Domo 还是雪花合作伙伴连接计划的首要合作伙伴。
