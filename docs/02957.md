# 茄子加强其人工智能驱动的数字自动化智能套件

> 原文:[https://devo PS . com/茄子-加强-its-ai-powered-digital-automation-intelligence-suite/](https://devops.com/eggplant-strengthens-its-ai-powered-digital-automation-intelligence-suite/)

*深化人工智能能力，实现复杂数字产品的自动化和问题诊断；增加自动化可用性测试的能力*

客户体验优化专家茄子今天宣布了其数字自动化智能(DAI)套件的最新增强功能。这些新功能通过协助复杂系统的建模和自动化、诊断测试中发现的问题以及自动化可用性测试，进一步扩展了智能自动化水平和对客户体验的洞察。

DAI 是一种人工智能驱动的方法，提供持续的智能测试自动化，使组织能够测试端到端的客户体验，并在发布新产品版本之前预测用户影响。通过扩展自动化和可见性，DAI 使组织能够真正优化数字体验。

**数字自动化智能套件的新增功能包括:**

由人工智能驱动的测试建模和自动化测试创建产品茄子 AI ，现在能够提供智能报告，允许 DevOps 团队了解为什么某个发布会有问题，而不仅仅是接收失败通知。这些智能见解基于茄子的 bug 搜索算法，并使用这些算法快速识别导致失败的根本原因模式。茄子人工智能还被扩展到允许通过子模型、模型版本和其他协作功能测试大型企业软件系统。子模型允许一个大的、复杂的系统模型被分解，以便于使用和管理。

市场领先的自动化引擎茄子 Functional 现在将为组织提供分析软件可用性的能力，而无需人工反馈。茄子可以发现会引起用户不满的问题，这些问题可能会导致更差的业务成果。这些可能包括难以阅读的文本、错误的颜色选择、不正确的渲染或模糊的图像。

该套件的其他新增功能包括增强的集成和协作功能，以支持大型企业团队。

**引言和评论**

“只测试功能，而不是数字体验和由此产生的业务成果的时代已经过去。组织必须加快交付速度，同时确保新数字产品和服务的质量和有效性。数字自动化智能套件允许 DevOps 团队转变他们的测试方法，提供令人愉悦的数字体验。”

安东尼·爱德华兹，首席运营官茄子
