# DevOps 领导力系列:Gov Does DevOps(第 2 部分)

> 原文：<https://devops.com/devops-leadership-series-gov-does-devops-part-2/>

### 

在 DC DevOpsDays 的第二天，我有机会结识了几位行业思想领袖。首先，[约翰·威利斯](https://www.linkedin.com/profile/view?id=15628334&authType=NAME_SEARCH&authToken=nF2E&locale=en_US&trk=tyah&trkInfo=clickedVertical%3Amynetwork%2Cidx%3A1-1-1%2CtarId%3A1434386591821%2Ctas%3AJohn%20Willis)，(@botchagalupe) DevOps Days 核心组织者，即将出版的《DevOps 食谱》的合著者。

在 DevOps 领导力系列的这一集中，John 重点介绍了他对 DevOps 的第一次介绍以及他所经历的旅程，以及 DevOps 的现状中哪些让他感到兴奋。他说 DevOps 真的到处都在发生。如今，大型组织和政府机构的首席信息官们正在各个阶段试验 DevOps。他提到了不一定拥有“积极 IT”的传统公司，以及他们现在如何希望分享他们的 DevOps 经验。

[https://www.youtube.com/embed/jU1O2bCU_hc?feature=oembed](https://www.youtube.com/embed/jU1O2bCU_hc?feature=oembed)

在那之后，我有机会和 OmniTI 的副总裁 Leon Fayer 聊天，在他的 DevOps Days DC 演讲之后。Leon 举例说明了他在 DevOpsDays 会议上发现的最有趣的事情。他指出，许多人已经开始关注货币策略，以及如何以 DevOps 的方式从上到下进行监控。莱昂很高兴地看到，人们现在对如何更准确地生成和收集数据以解决他们的问题更感兴趣。
