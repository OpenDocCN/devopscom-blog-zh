# Rackspace Technology 宣布将 Moogsoft AIOps 平台集成到 Rackspace Fabric 中

> 原文:[https://devo PS . com/rackspace-technology-announces-moog soft-aiops-platform-integration-into-rackspace-fabric/](https://devops.com/rackspace-technology-announces-moogsoft-aiops-platform-integration-into-rackspace-fabric/)

*用 AIOps* *增强 Rackspace Technology 的综合多云平台*

 ***圣安东尼奥—****2020 年 8 月 20 日**—Rackspace Technology(纳斯达克股票代码:RXT)今天宣布，Rackspace Fabric 的最新版本现在包括 Moogsoft AIOps 平台，该平台将人工智能(AI)和机器学习(ML)应用于日志、指标、跟踪和警报数据，以帮助更快、更有效地解决 IT 事件。与 Rackspace Fabric 的集成意味着 Rackspace 客户将通过利用机器学习来解决 IT 问题，从而增加正常运行时间并减少事故警报。

Rackspace Fabric 是一个全面的平台，为跨多云环境的管理任务提供了统一的方法。许多公司都在努力管理复杂的云环境，它们的身份认证系统相互脱节，支持不可靠，管理能力也各不相同。Rackspace Fabric 通过跨这些技术提供一致的管理和支持功能，简化了对 Amazon Web Services (AWS)、Google Cloud Platform、Microsoft Azure 和 VMware 环境的管理。

Rackspace Fabric 提供通用的计费、治理、支持、监控、事件和变更管理服务，同时仍然包含每个云的独特功能。Rackspace Fabric 每月处理超过 10 亿次自动化操作，为消费和管理来自多个提供商的云资源提供了更快、更一致的方法，使客户能够更快地实现云的转型能力。

Rackspace Technology 首席技术官托尔加·塔尔汉表示:“Rackspace Fabric 采用了我们过去 20 年来在运营任务关键型应用程序方面学到的一切，将其与最新的云技术相结合，并将其编织到一个平台——一个 Fabric——我们将其作为服务交付的基础。“Moogsoft AIOps 的集成融合了全面的 AI 和 ML 功能，实现了跨服务的可见性，以增强事件响应并增加我们客户云环境的正常运行时间。随着这些云环境变得越来越具有自我修复能力，创新的步伐也在加快，客户可以腾出更多资源来更快地为终端用户构建和交付产品。"

“随着组织将新服务部署到多云环境中，数字化转型带来了前所未有的规模和复杂性，这需要人工智能和人工智能的帮助，以补充 IT 运营的创新能力，”Moogsoft 创始人兼首席执行官 Phil Tee 说。这正是 Rackspace Technology 通过将 Moogsoft AIOps 集成到 Rackspace Fabric 中所做的事情。借助 AIOps，Rackspace Technology 的客户可以将更多的时间集中在开发推动当今业务发展的数字服务上，而不是解决这些问题。"
