# Datadobi 宣布支持 Microsoft Azure Blob 存储上的文件数据保护

> 原文：<https://devops.com/datadobi-announces-support-for-file-data-protection-on-microsoft-azure-blob-storage/>

Microsoft Azure 全球客户现在可以获得 DobiProtect 的支持，以将存储成本保持在最低水平，快速恢复数据，并保护业务关键型非结构化数据资产

比利时鲁汶——2021 年 3 月 16 日—[全球非结构化数据管理软件的领导者 Datadobi](http://www.datadobi.com/) 今天宣布，它在 DobiProtect 5.11 中增加了对 Azure Blob 存储的支持，使客户能够保护其数据从任何 NAS 或云文件系统到微软基于云的对象存储解决方案。随着非结构化数据持续快速增长，Datadobi 允许企业通过简单、快速且经济高效的解决方案来防范网络威胁、勒索软件、意外删除和软件漏洞。

Azure Blob storage 是一种可大规模扩展的安全对象存储服务，适用于数据湖、归档、云原生工作负载、分析和机器学习。使用 Datadobi，组织现在可以在 Azure Blob 存储中维护其文件数据的保护副本。

Datadobi 的 Carl D 'Halluin 评论道:“我们看到客户正在大规模转移到云中并保护数据，微软 Azure Blob 存储显然是一个领先且值得信赖的云解决方案。随着 Datadobi 解决方案的加入，Azure 客户可以在云中托管和保护他们的非结构化数据，优化存储投资，并保持百分之百的数据准确性和完整性。”

“我们很高兴欢迎 Datadobi 加入不断发展的微软 Azure 生态系统，”Azure Storage 集团产品经理 Vamshi Kommineni 表示。“Azure Blob 存储为非结构化数据提供了强大、安全且经济高效的解决方案。这一增加将满足我们的客户对大规模非结构化数据迁移到公共云的不断增长的需求，并在威胁和漏洞泛滥的时代提供所需的保护。”
