# 思杰和谷歌云简化了向灵活工作的转变

> 原文:[https://devo PS . com/Citrix-and-Google-cloud-simplify-shift-to-flexible-work/](https://devops.com/citrix-and-google-cloud-simplify-shift-to-flexible-work/)

Google Cloud Marketplace 上新推出的 Citrix DaaS 解决方案在从采购到部署的过渡过程中节省了成本和时间。

**佛罗里达州劳德代尔堡，2022 年 5 月 4 日—** 去年， [Citrix Systems，Inc.](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.citrix.com%2F&esheet=52707039&newsitemid=20220504005171&lan=en-US&anchor=Citrix+Systems%2C+Inc.&index=1&md5=68b59ca10da2ca090180ee09ef029503) (纳斯达克股票代码:CTXS)和谷歌云[宣布](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.citrix.com%2Fnews%2Fannouncements%2Foct-2021%2Fcitrix-google-cloud-team-deliver-future-hybrid-work.html&esheet=52707039&newsitemid=20220504005171&lan=en-US&anchor=announced&index=2&md5=66ff84894f6a32996e36428250c3be73)他们致力于通过桌面即服务(DaaS)解决方案帮助客户加速和接受混合工作模式。他们正在兑现承诺。这两家公司今天宣布，两款新的思杰 DaaS 解决方案可在[谷歌云市场](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fconsole.cloud.google.com%2Fmarketplace%2Fproduct%2Fcitrix-public%2Fcitrix-daas-for-google-cloud.endpoints.citrix-master-project.cloud.goog&esheet=52707039&newsitemid=20220504005171&lan=en-US&anchor=Google+Cloud+Marketplace&index=3&md5=6ebd023cd34bbc72f31fa62ce47707bc)购买，这使得拥有思杰 VDI 解决方案的组织比以往任何时候都更容易过渡到基于谷歌云的思杰 DaaS。

时机再好不过了。

Citrix 产品营销副总裁 Carisa Stringer 表示:“组织正在快速采用混合工作，IT 需要一种高效、经济、安全且灵活的方式来实现这一目标。

***简化采购***

Google Cloud Marketplace 使公司能够轻松快速地购买和部署市场领先的数字工作空间解决方案，这些解决方案旨在实现安全的远程工作，包括专为 Google Cloud 打造的思杰 DaaS 解决方案:

*   **适用于 Google Cloud 的思杰 DaaS 标准版**–专为刚接触 DaaS 或 VDI、需要经济实惠且全面的混合工作解决方案来快速、轻松、安全地交付虚拟应用和桌面的组织或部门而设计。
*   **Citrix DaaS Premium for Google Cloud**–Premium edition 是拥有现有 VDI 投资并需要实现部署现代化的组织的理想选择，它为 IT 管理员提供了快速迁移和配置工具、简化的 Google 云和/或内部资源管理、高级安全功能以及卓越的监控和云优化。

通过 Google Cloud Marketplace 购买的 Citrix DaaS 支出计入客户的 Google Cloud 承诺，并包含在他们的 Google Cloud 发票中，从而将交易简化为一张账单。

***启用工作的未来***

Citrix Systems Integrator 和 Google Cloud 合作伙伴 Wipro 期待使用这些解决方案来帮助推进未来的工作。

Wipro VirtuaDesk 云计算和基础设施服务总经理兼全球负责人 Seshu Venkata 表示:“Wipro 非常高兴并渴望通过为混合工作部署真正的混合云解决方案来定位 Citrix 和 Google Cloud 的 DaaS 解决方案。

***飞驰到云端***

除了这些新产品和简化的交易流程之外，Citrix 和 Google Cloud 还在继续开发工具和资源，以使客户从内部 VDI 部署过渡到使用 Google Cloud 的 DaaS 比以往任何时候都更容易，包括:

*   **[Google Cloud](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fdocs.citrix.com%2Fen-us%2Fcitrix-virtual-apps-desktops-service%2Fmigrate-workloads.html&esheet=52707039&newsitemid=20220504005171&lan=en-US&anchor=Image+Portability+Service+for+Google+Cloud&index=4&md5=5402445dda53e14d1f29c65a367cf0a7)**的映像可移植性服务–从现有的内部 PVS/MCS 供应的 Citrix 环境中自动部署主映像，目标是托管在 Google Cloud 中的 Citrix DaaS 资源位置，以减少迁移到云时的重复工作，以及用于容量管理、改进的灾难恢复和业务连续性计划以及简化的映像管理的突发到云功能。

Stringer 说:“向云过渡是 IT 部门的重中之重，需要进行大量仔细的规划和考虑。“与谷歌一起，我们致力于提供解决方案和资源，以简化和加快流程并交付成果。”
