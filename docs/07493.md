# Core Scientific 获得 AWS 高性能计算资格认证

> 原文:[https://devo PS . com/core-scientific-achieves-AWS-high-performance-computing-competency-status/](https://devops.com/core-scientific-achieves-aws-high-performance-computing-competency-status/)

华盛顿贝尔维尤。，2021 年 3 月 1 日— [由微软前首席运营官首席执行官凯文·特纳领导的人工智能和区块链的领先基础设施和软件解决方案提供商 Core Scientific](https://c212.net/c/link/?t=0&l=en&o=3082067-1&h=434345138&u=https%3A%2F%2Fwww.corescientific.com%2F&a=Core+Scientific) 今天宣布，它已获得[亚马逊网络服务(AWS)高性能计算(HPC)资格认证](https://c212.net/c/link/?t=0&l=en&o=3082067-1&h=3248450757&u=https%3A%2F%2Faws.amazon.com%2Fhpc%2Fpartner-solutions%2F&a=Amazon+Web+Services+(AWS)+High+Performance+Computing+(HPC)+Competency)。这一称号表明 Core Scientific 在帮助客户使用 AWS 的弹性和可扩展云基础架构优化其 HPC 工作负载以提高性能和效率方面拥有丰富的经验。

AWS 推出了 AWS HPC 能力认证，以帮助客户轻松自信地与高度专业化的 AWS 合作伙伴合作，这些合作伙伴在基因组学、计算化学、金融风险建模、计算机辅助工程以及深度学习和自动驾驶等新兴应用领域提供技术产品，以实现更快的创新、更好的敏捷性和灵活性。AWS HPC 能力承担了识别和验证行业领导者的重任，这些领导者拥有业经验证的客户成功和技术熟练程度。

获得 AWS HPC 能力认证使 Core Scientific 成为在以下一个或多个领域拥有深厚专业知识的 AWS 合作伙伴:应用程序、HPC 管理和基础技术。

“Core Scientific 很荣幸成为首批获得 AWS HPC 资格认证的 AWS 合作伙伴之一，”Core Scientific 人工智能首席产品官 Ian Ferreira 表示。“我们的客户来自一系列重要的、改变游戏规则的行业和背景。通过将核心 Scientific Plexus HPC-Ops 软件堆栈与 AWS HPC 基础设施相结合，客户可以无缝运行非常大的并行工作负载。”

从初创企业到全球企业，AWS 都在提供可扩展、灵活且经济高效的解决方案。为了支持这些解决方案的无缝集成和部署，AWS 建立了 AWS 能力计划，以帮助客户确定具有丰富行业经验和专业知识的咨询和技术 AWS 合作伙伴。

Core Scientific 的 Plexus 是一个混合云平台，将 AWS 与客户可能拥有的现有内部资源相结合。凭借单一管理平台，Plexus 提供了跨基础设施的一致工作流，减少了重新培训最终用户的需求。

Plexus for HPC-ops 控制平面提供与 AWS 计算服务的交钥匙集成。这使客户能够管理、协调和运行由增强的 AWS 计算基础架构支持的 HPC 工作负载。Core Scientific 的细粒度调度程序支持在粗粒度计算上运行令人尴尬的并行工作负载，从而最大限度地提高客户投资回报率。
