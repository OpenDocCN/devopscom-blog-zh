# 加速数字化转型，加速数据转型

> 原文：<https://devops.com/to-accelerate-digital-transformation-accelerate-data-transformation/>

没有成功的数据转换，就不可能有成功的数字转换。毕竟，是关于一个组织的客户、员工、基础设施、市场动态、供应链、商业技术系统等的数据——以及分析这些数据的能力——使得[数字化转型](https://devops.com/?s=digital%20transformation)成为可能。

然而，去年 Gartner 说“信息作为一种资产”仍处于初级阶段。“数据和分析是组织数字化和转型工作的关键催化剂。然而，今天，只有不到 50%的记录在案的企业战略提到数据和分析是实现企业价值的基本组成部分，”该分析公司在其报告中写道，“为什么数据和分析是数字化转型的关键

此外，Gartner 预测变革即将到来，而且速度很快:Gartner 表示，在两年内，90%的企业战略将明确地将信息列为关键的企业资产，将分析列为基本能力。

要实现这一目标，并在数据和数据分析方面取得成功，必须面对巨大的挑战。

根据 IDG(由云数据仓库的云数据转换提供商 Matillion 赞助)发布的一项调查，挑战是严峻的。该调查对至少拥有 1000 名员工的北美组织的 200 多名 IT、数据科学和数据工程专业人员进行了调查。

企业面临的挑战是什么？根据调查结果，47%的人表示数据控制问题是数据分析项目的最大挑战。相比之下，指出的其他主要挑战是缺乏可扩展、可靠的技术平台来处理大型数据集(45%)，有太多的手动流程(38%)，以及清理和准备数据的挑战(36%)。此外，40%的受访者表示，在与业务用户协作时，缺乏对数据孤岛的可见性和控制是一项挑战。

因此，了解到收集和准备用于有用分析的数据平均需要一周时间，并且花费在数据分析计划上的所有时间中有 45%被数据准备所消耗，就不足为奇了。大约 60%的人表示，他们在准备数据进行分析时缺乏所需的可扩展性和灵活性。

当谈到寻求加快数据分析改进方法的公司数量时，几乎是一致的。

那么，企业如何加快努力改进数据分析，以更好地推动数字化转型呢？受访者表示，数据便携性(57%)、入职便利性(57%)和成本效益(52%)是分析平台的首要特征，可以帮助他们克服当前的障碍。接受调查的 IT 专业人员更喜欢用户友好(50%)和更容易连接到数据源(50%)作为首要功能；然而，数据工程师、数据科学家、数据架构师和其他人认为价值实现时间(57%)和提供自助服务的能力(51%)是基本能力。

相当一部分被调查者表示，他们已经开始采用他们需要的技术和流程来提高他们分析数据的能力，以便快速做出更好的决策。根据调查结果，38%已经在使用云数据仓库。在未来，43%的受访者表示他们希望将所有数据放在云中，57%的受访者计划同时使用云和内部数据策略。

那些更快掌握数据分析的公司可能享有先发优势。“一家公司在新兴数字经济中的竞争能力将需要更快的节奏和前瞻性的决策，”Gartner 著名副总裁兼分析师 Douglas Laney 说。“数据和分析领导者需要在企业战略规划中确立自己的地位，以确保数据和分析能力融入最高级别的面向公众的企业计划中。”
