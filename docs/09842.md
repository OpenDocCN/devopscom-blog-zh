一个新的前沿:深度可观测性

今天的版本引入了一系列重大改进，包括:

● **下一代集装箱网络可见性—**giga vue 现在能够采集集装箱流量，提供业内首个通用集装箱 tap (UCT)解决方案，可部署到任何集装箱网络接口(CNI)和任何集装箱编排，由增强型 Berkeley 数据包过滤器(eBPF)提供支持，以最小的开销实现网络级可见性。无论基于容器的微服务的短暂性或规模如何，安全性和性能的深度可观察性现在都可以跟随工作负载。

● **应用元数据集成框架——**giga mon Hawk 应用元数据引擎使用 JavaScript object notation (JSON)和 Kafka 直接与领先的可观察性平台集成，包括 [Dynatrace](https://streaklinks.com/BPD-y-KICApi5abHOgEGCl7j/https%3A%2F%2Fwww.dynatrace.com%2F) 、 [New Relic](https://streaklinks.com/BPD-y-G9Pa3RldSoLAW5OYLB/https%3A%2F%2Fwww.gigamon.com%2Fcontent%2Fdam%2Fresource-library%2Fenglish%2Ftechnology-partner-solution-brief%2Fjs-gigamon-newrelic.pdf) 和 [Sumo Logic](https://streaklinks.com/BPD-y-GGxjbPh2lX7wOKxqve/https%3A%2F%2Fwww.gigamon.com%2Fcontent%2Fdam%2Fresource-library%2Fenglish%2Ftechnology-partner-solution-brief%2Fjs-sumo-logic-and-gigamon-deep-observability-pipeine.pdf) 。这种集成使现有的可观察性工具能够执行新的安全功能，例如识别流氓服务、活动和非法加密挖掘。

● **云级网络遥测处理—**作为 Gigamon Hawk 的底层组件，新的 GigaVUE-HC1 Plus visibility 设备与上一代产品相比，在物理占地面积和功耗需求减半的情况下，提供了两倍的管道处理性能。这种专门构建的设备可以通过聚合来自物理、虚拟和/或容器工作负载的遥测数据，包括来自公共云工作负载的遥测数据，高效、经济地处理网络遥测数据。
