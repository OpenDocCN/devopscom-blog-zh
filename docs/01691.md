# 网络研讨会:在混合云中使用后端大型机服务安全地构建、测试和部署移动应用

> 原文:[https://devo PS . com/build-test-and-deploy-mobile-apps-with-back-end-mainframe-services-securely-in-a-hybrid-cloud/](https://devops.com/build-test-and-deploy-mobile-apps-with-back-end-mainframe-services-securely-in-a-hybrid-cloud/)

您可能已经对混合云及其优势有了很好的理解。但是，如何将过去三十年对后端基础架构投资的价值更快地传递到最终用户手中呢？

在这个网络广播中，我们将展示如何利用最新的 IBM 技术，快速、轻松地将后端服务集成到运行在 IBM Bluemix 上的移动应用程序中。我们还将重点介绍 IBM DevOps Services (IDS)和 IBM Bluemix 支持端到端移动开发生命周期的最新功能。

加入 IBM 业务合作伙伴 PointSource 的执行移动策略师 Kristal York 和 CLM 和 DevOps 的杰出工程师兼首席架构师 Rosalind Radcliffe，他们将带您深入了解 IBM DevOps 服务和 Bluemix，这是一个开放标准的基于云的平台，用于构建、管理和运行与您的后端服务安全连接的移动应用。

# **关于小组成员:**

**Kristal York，IBM 业务合作伙伴 PointSource 的执行移动策略师**

Kristal 于 2012 年加入 PointSource，领导过金融、零售和保险垂直行业的团队。她擅长营造协作环境，并带领他人解决复杂的业务问题。Kristal 之前作为业务分析师的经验有助于作为移动战略家定义和分析我们客户的移动时刻。她将这些移动时刻转化为跨组织业务流程和系统的移动战略。移动需要一种迭代的方法，因此 Kristal 帮助分析战略规划、可用性、研究以及持续更新。她利用自己的商业分析技能来帮助定义应用程序推出后的采用率和情绪分析。此前，Kristal 在 Cartus 工作了 12 年，最近担任高级业务分析师、业务分析和移动产品负责人。

[![Rosalind](../Images/4fe7a0d2024624a8eecde612f9297a4d.png)](https://devops.com/wp-content/uploads/2014/11/Rosalind.jpg)

Rosalind 是 IBM 公司的杰出工程师，也是 CLM 和 DevOps 的首席架构师。她负责推动多平台架构的开发运维；这包括 z 系统和动力系统。
