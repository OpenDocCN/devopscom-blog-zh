**关于书**

*面向现代企业的开发运维* 许多组织在传统 it 基础架构的现代化方面都面临着严峻的挑战。大多数都是通过从传统或遗留制造中吸取经验教训而发展起来的，例如创建一个生产流程，将重点放在过程而不是执行任务的人，并允许组织将人视为资源，以努力实现高质量的结果。但这些实践和想法在现代 IT 中是失败的，现代 IT 需要协作和创造性来实现高绩效、高质量的成功。使用敏捷、精益和 DevOps 的原则，以及来自企业世界的第一手例子，Hering 解决了传统组织在转变为现代 IT 部门时面临的不同挑战。
