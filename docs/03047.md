# CollabNet VersionOne 和 Parasoft 合作为企业开发人员带来自动化测试

加利福尼亚州亚特兰大和蒙罗维亚——2018 年 10 月 10 日——[企业价值流管理、敏捷管理、Git 版本控制和 DevOps 领域的全球领导者 CollabNet VersionOne](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=5eXH0qSKdBRphS6WO4YlQwwSTi8wtQOqejFjZKiddbCzZ5Bbl4XDYf7Fp0Zg3PbWOCPTEv55ZYFE9Jx6Ambva5ncXj4e0rGr5blmZATGLIWwbaI4HDZb9g%3D%3D&G=0&R=http%3A%2F%2Fwww.collabnet.com%2F&I=20181010194639.00000001da40%40mail6-43-usnbn1&X=MHwxMDQ2NzU4OjViYmU1NzE0YmQ3OTdkYzNiZjc2YWE3NDs%3D&S=Xv7mdt01fzR5RYxymWeDdLPMgb_PdSSCtorBrgX2TbA) 与自动化软件测试和持续质量领域的全球领导者 [Parasoft](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=5eXH0qSKdBRphS6WO4YlQwwSTi8wtQOqejFjZKiddbCzZ5Bbl4XDYf7Fp0Zg3PbWOCPTEv55ZYFE9Jx6Ambva5ncXj4e0rGr5blmZATGLIWwbaI4HDZb9g%3D%3D&G=0&R=http%3A%2F%2Fwww.parasoft.com%2F&I=20181010194639.00000001da40%40mail6-43-usnbn1&X=MHwxMDQ2NzU4OjViYmU1NzE0YmQ3OTdkYzNiZjc2YWE3NDs%3D&S=Sri_195oK6o_d69BUkxPYBGtDzRcUNPx7vl5tXLFkYg) 宣布建立新的合作伙伴关系。CollabNet VersionOne 将把 Parasoft 的自动化测试套件集成到其价值流管理(VSM)产品中。

开发人员可以一天花几个小时手动测试软件。这种孤立的手动方法与扩展敏捷和开发运维，以及编排和自动化工作流以建立价值流的组织目标相冲突。这个新的联盟通过将自动化测试实践集成到他们的价值流中来解决这个问题，以支持持续的集成和交付。

最初的集成将把 CollabNet VersionOne Continuum(一个 DevOps 发布管理和性能优化产品)与 Parasoft 的自动化软件测试工具(包括其复杂的分析)结合在一起。Continuum 将为用户提供一个单一面板视图，并控制他们的自动化测试阶段和结果数据，它将把测试阶段与开发工作流的其余部分联系起来。

Parasoft 首席执行官 Elizabeth Kolawa 表示:“自动化软件测试对于我们的客户成功实施其最具战略性的开发计划至关重要，我们专注于帮助组织有效地扩展其测试实践，以快速交付高质量的软件，特别是在机器学习和人工智能领域的最新创新。”。“通过将 Parasoft 工具套件的深度代码分析和广泛的测试实践与 CollabNet VersionOne DevOps 解决方案相集成，我们能够为软件开发社区提供对其价值流质量的完整可见性，以便放心发布。”

CollabNet VersionOne 首席执行官兼总裁 Flint Brenton 表示:“这一联盟反映了我们对敏捷社区和 DevOps 创新的承诺。“通过这种集成，团队将有一个单一的界面来执行自动化测试，查看测试过程和结果数据，快速揭示质量差距、安全问题和其他错误和风险，同时允许他们看到他们的测试阶段与其他上游和下游开发过程的关系。”

Continuum 的可视化界面使用户很容易理解开发过程中发生的进展。随着 Parasoft 测试工具的集成，Continuum 用户将能够针对正在测试的应用程序执行测试(包括单元测试、代码质量和覆盖范围分析、API 和功能测试以及性能和负载测试)，然后在 Continuum 界面中访问这些工具的分析和结果，因此整个团队都可以看到测试实践对工作流的影响。

“我们帮助客户使软件开发和交付成为一种透明的、协调良好的实践，并始终与业务目标保持一致。添加测试自动化功能支持我们的愿景，即消除大规模采用敏捷、扩展开发运维以及建立企业价值流的障碍。Brenton 说:“我们很高兴与 Parasoft 建立合作伙伴关系，因为他们与我们有着共同的目标，即改进软件开发实践。

## 关于 Parasoft

Parasoft 提供了创新的工具，可以自动执行耗时的测试任务，并为管理层提供智能分析，从而专注于重要的事情。Parasoft 的技术通过集成静态和运行时分析，减少了交付安全、可靠和合规软件的时间、精力和成本；单元、功能和 API 测试；和服务虚拟化。Parasoft 支持软件组织在嵌入式、企业和物联网市场开发和部署应用。借助开发人员测试工具、经理报告/分析和高管仪表板，Parasoft 使组织能够在当今最具战略性的开发计划中取得成功，这些计划包括敏捷、持续测试、开发运维以及安全性。
