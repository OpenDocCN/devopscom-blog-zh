# DevOps 学院启动分层全球教育合作伙伴计划

> 原文：<https://devops.com/devops-institute-launches-tiered-global-education-partner-program/>

*培训合作伙伴支持技能、知识、想法和学习(SKIL)框架，以促进“开发人员”和全球数百万 IT 专业人员的发展*

**佛罗里达州 BOCA 拉顿，2020 年 2 月 6 日—**[](https://devopsinstitute.com/)**DevOps Institute 是一家全球会员制协会，旨在推进 devo PS 的人文元素，今天宣布了其新修订的全球教育合作伙伴计划。DevOps Institute 的全球教育合作伙伴按层级(注册、高级和精英)组织，优化认证课程和计划，向学生传授企业 IT、DevOps、敏捷、安全、测试、持续交付和站点可靠性工程(SRE)的最新模式和实践。每个课程和计划都为 IT 专业人员通过 DevOps Institute 的相应认证考试做准备。有关 DevOps Institute 提供的 IT 行业认证的完整列表，请访问:([https://devopsinstitute.com/certifications/](https://devopsinstitute.com/certifications/))。**

 **目前，DevOps Institute 的全球教育合作伙伴计划包括代表全球 70 多个国家的 200 个合作伙伴组织。这包括咨询和专业服务机构、软件和基础设施提供商、培训公司和系统集成商。

DevOps Institute 为其合作伙伴配备了超过 180 小时的学习材料，这是当今市场上最可用、最相关和最受认可的培训和认证。通过在当地提供培训，全球注册教育合作伙伴支持 DevOps 学院的使命，以促进 DevOps 的人的因素。

**更多关于 DevOps Institute 全球教育合作伙伴网络的信息，请访问:(**[**)https://devo psinstitute . com/partners/find-a-Education-Partner/**](https://devopsinstitute.com/partners/find-a-education-partner/)**)。**

**新的全球教育合作伙伴计划将合作伙伴分为三个等级:**

**注册合作伙伴**是那些提供 DevOps 学院培训和认证组合的组织。

**卓越合作伙伴**接受了 DevOps Institute 的课程和认证组合，并为服务提供方向提供了宝贵的反馈，进一步提高了各自地区对 DevOps 的认识。

**Elite Partners** 通过技能、知识、想法和学习(SKIL)围绕 DevOps 进一步教育市场，是 DevOps 运动的真正倡导者。精英合作伙伴包括:

| 澳大利亚 DDLS | 澳大利亚 |
| ALC 培训 | 澳大利亚 |
| 超前技术 | 法国/加拿大 |
| 最大端口 | 德国 |
| NTUC 学习中心 | 新加坡 |
| 质量保证 | 联合王国 |
| 新视野 | 美国 |
| ITSM 学院 | 美国 |

**欲了解更多，成为 DevOps 学院全球教育合作伙伴，请访问:(**[**)https://devo psinstitute . com/partners/become-an-Education-Partner/**](https://devopsinstitute.com/partners/become-an-education-partner/)**)。**

DevOps Institute 首席执行官 Jayne Groll 表示:“我们很高兴宣布我们扩大的全球教育合作伙伴计划，并特别表彰我们的精英合作伙伴，他们通过 DevOps Institute 的 SKIL 框架完全支持 DevOps 人类的进步。“我们致力于支持和帮助我们的合作伙伴提高全球 IT 专业人员在开发运维、敏捷和 SRE 领域的意识和技能。在接下来的几个月中，我们将宣布更多的本地化工作，以支持更广泛的企业 IT 市场。”

“QA 很高兴成为 DevOps 研究所的精英合作伙伴，并支持英国和其他地区的市场，以推动 DevOps 的成功并提高组织效率，”QA 首席营销官 James Frost 说。

**关于 DevOps 学院**

DevOps 学院致力于推动 DevOps 成功的人文因素。作为一个全球性的会员制协会，DevOps Institute 是连接 IT 从业者、教育合作伙伴、顾问、人才获取和业务高管的首选学习中心，旨在为支持数字化转型和新 IT 铺平道路。

我们基于以人为本的 SKIL 框架，包括技能、知识、想法和学习，在 DevOps 社区内帮助促进职业发展并支持新兴实践。我们的所有工作，包括认证、研究、活动和持续学习计划，都专注于提供“人类知识”,以实现 IT 现代化并使 DevOps 取得成功。
