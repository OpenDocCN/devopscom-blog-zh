# Akamai 推出基于 Salesforce 疫苗云的新冠肺炎疫苗管理解决方案

> 原文:[https://devo PS . com/akamai-launches-新冠肺炎-疫苗-管理-解决方案-构建-销售力量-疫苗-云/](https://devops.com/akamai-launches-covid-19-vaccine-management-solution-built-on-salesforce-vaccine-cloud/)

面向 Salesforce Vaccine Cloud 的 Vaccine Edge 使系统能够有效、快速地扩展，允许大量用户同时注册在线预约

马萨诸塞州剑桥| 2021 年 3 月 30 日— [Akamai 技术公司](https://www.akamai.com/)。(纳斯达克股票代码:AKAM)，一个保护和提供数字体验的可信解决方案，今天宣布它已经为 Salesforce Vaccine Cloud 开发了 Akamai Vaccine Edge(“[Vaccine Edge](https://www.akamai.com/us/en/campaign/introducing-akamai-vaccine-edge.jsp)”)，以帮助政府机构，医疗保健组织，企业，非营利组织和教育机构扩大其新冠肺炎疫苗的在线分销。作为 Salesforce 最近推出的[疫苗云](https://www.salesforce.com/news/press-releases/2021/01/27/vaccine-cloud-salesforce-update/)的补充，Vaccine Edge 提供了一种快速、灵活的方法来管理、交付和实施集中式疫苗计划，当网站因高需求而处于压力之下时，它有助于增强注册和注册流程。

Vaccine Edge 旨在通过加快页面加载速度来支持增强的客户体验，无论同时注册的用户数量有多少。当负载过高时，Vaccine Edge 会将用户转移到等候室以保持站点可用，这样用户就可以保持他们在队列中的位置。Vaccine Edge 还构建了拒绝服务(" [DDoS](https://www.akamai.com/us/en/resources/ddos-mitigation.jsp) ")保护，以帮助监控和阻止试图通过有针对性的请求淹没网站的违规者。

Akamai Technologies 全球服务和支持执行副总裁兼首席信息官 Mani Sundaram 表示:“公民需要关于疫苗接种资格和预约物流的准确、最新信息。“疫苗接种注册网站必须实现这一切，并处理流量的突然激增。Vaccine Edge 帮助我们的客户提供可靠的用户体验，而不会给他们的 IT 团队带来更多复杂性。”

Akamai 的疫苗优势已被服务于大量人群的站点成功部署，并已为市民提供快速获得新冠肺炎疫苗预约的途径。该解决方案目前可用于[美国和加拿大]州/省和地方政府，以及下一阶段的疫苗分销商，如主要零售商、药店、医疗诊所和医院。

“对于大多数疫苗接种者来说，保护自己的第一步是安排在线预约，”Salesforce Vaccine Cloud 高级副总裁兼总经理 Meredith Flynn-Ripley 说。“Akamai 的疫苗边缘解决方案可以帮助患者获得与 Salesforce 集成的高效、有效和公平的体验。”

Vaccine Edge 是 Akamai 新冠肺炎在美国和加拿大的几个项目之一。该公司已经通过性能优化和虚拟候诊室帮助冠状病毒援助、救济和经济安全(CARES)法案的受益者。

更多信息，请访问 [Akamai Vaccine Edge](https://www.akamai.com/us/en/campaign/introducing-akamai-vaccine-edge.jsp) 。
