# Instana 为微服务 APM 解决方案增加了对 GraphQL 查询和性能的深度检查

> 原文：<https://devops.com/instana-adds-deep-inspection-of-graphql-queries-and-performance-to-microservice-apm-solution/>

*公司是首家提供完整 GraphQL 系统和应用性能监控查询详情的 APM 提供商*

**2019 年 10 月 8 日，加利福尼亚州圣何塞，API World 大会**–针对动态微服务应用的[自动化应用性能管理(APM)解决方案](https://instana.com/application-management)的领先提供商 Instana 今天宣布扩展使用 GraphQL 作为其基础设施一部分的应用的监控和跟踪功能，包括对 GraphQL 查询的深度检查。这个最新的扩展使 Instana 成为第一个使用独特的数据查询引擎来监控 GraphQL 和应用程序的 APM 提供者。

“由于应用和微服务平台技术在不断发展，应用管理解决方案跟上发展步伐至关重要，”Instana 的开发人员 Chris Engelbert 说道。通过监控和跟踪应用程序以外的 GraphQL 性能，Instana 提供了优化 GraphQL 查询性能所需的准确数据

Instana 捕获 GraphQL HTTP 流量，并以纯文本形式显示正在执行的查询。然而，Instana 更进一步，解析实际的 GraphQL 查询——使 Instana 成为唯一一个作为自动化解决方案的一部分提供完整全面的 GraphQL 监控的 APM 解决方案。

Instana 的自动化应用性能监控(APM)解决方案可发现所有应用服务组件和应用基础设施，包括 GraphQL、Jenkins、Kubernetes 和 Docker。Instana 自动为应用技术堆栈的每个部分部署监控传感器，并跟踪所有应用请求，无需任何人工配置，甚至无需重启应用。该解决方案实时检测应用环境中的变化，调整自己的模型，并在几秒钟内可视化变化和对性能的影响。

Instana 的营销总监克里斯·法雷尔(Chris Farrell)说:“最终，应用程序提供商必须创造尽可能好的用户体验。“通过提供对更多组件的监控和深度检查，Instana 的全面自动化监控提供了应用程序利益相关方优化应用程序性能所需的精确数据，尤其是对于 GraphQL 等平台。”

Instana 的 GraphQL 监控和跟踪是 Instana 更大的动态应用和基础设施监控的一部分，包括对从 Java 应用代码到专业微服务查询平台的一切支持。该公司将在圣何塞的 API World(207 号展位)上展示他们的解决方案，包括新功能。在 Instana.com 了解更多信息
