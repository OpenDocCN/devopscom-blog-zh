这本书是由 CA Technologies 的持续测试业务部门总经理 Jeff Schaeffer 撰写的；CA Technologies 持续测试 CTO 亚历克斯·马丁(前 Rally Software CTO)；CA Technologies 的 Gobal 营销副总裁 Aruna Ravichandran 是《数字领导者的 DevOps》一书的作者。除了这本书，还有一本关于持续测试的配套电子书，以及其他相关内容。
