# CloudBees 宣布新的技术联盟合作伙伴计划

> 原文:[https://devo PS . com/cloud bees-announces-new-technical-alliance-partner-program/](https://devops.com/cloudbees-announces-new-technical-alliance-partner-program/)

加利福尼亚州圣何塞([美国商业资讯](https://www.businesswire.com/))—[推动持续经济发展的企业开发运维领导者 CloudBees](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.cloudbees.com&esheet=52108297&newsitemid=20191010005087&lan=en-US&anchor=CloudBees&index=1&md5=7c369924735d182a6af982ef9736f537) 今天推出了一项新的合作伙伴计划，该计划旨在增强 ISV 合作伙伴与 CloudBees 产品和全球 Jenkins 社区保持一致的能力。[技术联盟合作伙伴计划(TAPP)](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.cloudbees.com%2Ftapp&esheet=52108297&newsitemid=20191010005087&lan=en-US&anchor=Technical+Alliance+Partner+Program+%28TAPP%29&index=2&md5=e8931c5f5ac98dec0d921ef70b5bdcab) 将通过新的计划和更紧密的联系为 CloudBees 的合作伙伴生态系统注入活力，帮助 ISV 创造新的收入来源并进一步发展其 DevOps 业务。

APP 包括 27 个发布合作伙伴，并引入了三个新的上市层级，以更好地把握不断发展的 DevOps 市场中的增长机遇。这些新等级的成员------------------------------------------------------------------注册、高级和高级-------------------可以利用与 CloudBees 的联合走向市场活动，参与 CloudBees 的专家服务，并访问扩展的销售和营销资源，从而为发展其 DevOps 业务创造资产。

“在 CloudBees，我们坚信同类最佳是我们客户的终极模式，”CloudBees 的联合创始人兼首席战略官 Francois Dechery 说。“这关乎选择、深度和快速创新。这就是为什么我们将 CloudBees 技术联盟合作伙伴计划(TAPP)视为一项战略计划，旨在为我们的客户带来最好的 DevOps 生态系统。”

TAPP 计划通过增加可用于满足开发运维需求的工具和服务的范围和广度，为 CloudBees 客户创造价值。通过与 CloudBees 建立更深入的技术集成，合作伙伴扩展了生态系统的能力，以提供可扩展的、安全的和专门构建的解决方案来解决 DevOps 管道中的挑战。

******关于 CloudBees** CloudBees 通过提供世界上第一个端到端连续软件交付管理系统(SDM)为连续经济提供动力。对于数百万推动各种规模企业创新的开发人员和产品团队来说，SDM 建立在持续集成(CI)和持续交付(CD)的基础上，使软件交付组织内部和周围的所有功能和团队能够最好地合作，以扩大价值创造。****

****CloudBees 是 CI、CD 和应用程序发布流程编排(ARO)的核心，建立在其产品的商业成功及其开源领先地位之上。CloudBees 是[詹金斯](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fjenkins.io%2F&esheet=52108297&newsitemid=20191010005087&lan=en-US&anchor=Jenkins&index=36&md5=b464c6957d4f2dfb48f827df68854a09)和[詹金斯 X](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fjenkins-x.io%2F&esheet=52108297&newsitemid=20191010005087&lan=en-US&anchor=Jenkins+X&index=37&md5=3b014f10b0a00b1329eaf55b44cd9a6f) 的最大贡献者，也是[持续交付基金会](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fcd.foundation%2F&esheet=52108297&newsitemid=20191010005087&lan=en-US&anchor=Continuous+Delivery+Foundation&index=38&md5=140059f0b9e44c7bac0b1fb9408ed82d) (CDF)的创始成员。从全栈开发人员实践 NoOps 的创业公司到大型财富 100 强公司，CloudBees 使所有软件驱动的组织能够在正确的时间智能地部署正确的功能。****

****超过 3，500 个世界知名品牌和超过 50%的财富 500 强企业依赖 CloudBees，因为它能够在任何开发环境中跨任何云工作，并平衡公司治理和控制与开发人员的灵活性和自由度。CloudBees 是世界领先的 DevOps 专家的家园，帮助数千家公司利用“持续一切”的力量，将他们置于从伟大的想法到伟大的软件，再到伟大的商业价值的最快路径上。****
