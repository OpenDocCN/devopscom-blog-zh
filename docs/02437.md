# DevOps 市场整合成为现实:CloudBees 收购电子云

> 原文:[https://devo PS . com/devo PS-market-consolidation-gets-real-cloud bees-acquisites-electric-cloud/](https://devops.com/devops-market-consolidation-gets-real-cloudbees-acquires-electric-cloud/)

您可能已经看到了今天的新闻:CloudBees 收购了 Electric Cloud，这一举动标志着 DevOps 领域真正的市场整合。我有机会与 CloudBees 首席执行官 Sacha Labourey 讨论了收购事宜。我们讨论了交易的动机、合并后的实体未来可能会是什么样子等等。我们对话的视频在本文底部。

在我看来，这次收购代表了 DevOps 工具供应商市场的真正整合。我这样做的理由如下:

1.  到目前为止，DevOps 领域的许多收购都是由 CA、HP 或 IBM 等大型成熟上市公司进行的。通常，这类交易与其说是整合，不如说是大公司利用 M&A 的活动代替研发。CloudBees 不是大型供应商——这不是穿着收购外衣的 R&D。
2.  电云不是典型的寻找退出机会的风险投资支持的初创公司。该公司已存在多年。它拥有大量的企业客户和经过企业测试的产品。
3.  随着 DevOps 供应商市场最近的变化，包括 CA(现为 Broadcom)、HP(E) now Micro Focus 和 IBM 专注于 Red Hat，企业领导层出现了真空。这两家大规模供应商的合并确实是一次整合行动。

祝你今后好运，并祝贺我们在 CloudBees 和 Electric Cloud 的朋友们。看看这笔交易会给 DevOps 市场的其他部分带来什么影响，这将是一件有趣的事情。
