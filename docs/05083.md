# apacheⓡ软件基金会欢迎 CloudBees 成为其最新的白金级定向赞助商

> 原文：<https://devops.com/the-apache%e2%93%a1-software-foundation-welcomes-cloudbees-as-its-newest-targeted-sponsor-at-the-platinum-level/>

***捐赠持续交付软件服务有助于支持 Apache 基础设施、项目和社区***

**马萨诸塞州 Wakefield 和 DevOps World |葡萄牙里斯本 Jenkins World——2019 年 12 月 4 日—**Apache^(软件基金会(ASF)是 350 多个开源项目和计划的志愿者开发者、管理者和孵化器，今天宣布 CloudBees 成为 ASF 白金级目标赞助商。)

“我们很高兴地欢迎 CloudBees 成为目标白金赞助商，”ASF 筹款副总裁丹尼尔·鲁杰里说。“ASF 赞助商帮助抵消我们的日常运营开支，从会计到基础设施，从法律到营销等等。定向赞助提供旨在支持特定 ASF 操作以及指定的 Apache 项目及其社区的活动和计划的捐助。在 Apache 基础设施的诸多好处中，定向捐赠包括开发/协作工具、协同定位空间、云服务、监控系统等等。由于 CloudBees 的支持，我们很高兴能够将持续交付软件服务添加到该列表中。”

“CloudBees 是开源的大力提倡者。作为许多开源项目的主要贡献者，包括 Jenkins 和 Jenkins X，我们看到了组织每天从使用开源中获得的价值。我们对 Apache Software Foundation 的贡献将使它能够继续其开发开源项目的使命，并支持开源在全世界的使用。我们很荣幸成为目标赞助商。"

ASF 基础设施团队保持基金会的全球服务全天候运行，几乎 100%正常运行，每个项目的成本不到 5，000 美元。在 http://status.apache.org/可以得到反映超过 700 万周检查和跨越 2，059 个列表的项目邮件量的性能统计

Ruggeri 补充说:“每天，数十亿的 Apache 项目用户受益于 ASF 基础设施提供的许多服务。“我们赞助商的慷慨帮助加强了对 350 多个 Apache 项目及其社区的支持。我们期待着扩大有针对性的赞助计划，以满足日益增长的对 Apache 项目的需求，这些项目建立在我们为公众提供软件的使命之上。"

CloudBees 加入了以下目标赞助商:

*   白金——DLA Piper、JetBrains、微软、OSU 开源实验室、Sonatype 和威瑞森媒体；
*   黄金——Atlassian、CrytpoFund、Datadog、PhoenixNAP 和 Quenda
*   银牌——亚马逊网络服务、Hotwax 系统和 Rackspace
*   铜牌——bin tray、美国教育网络、Google、Hopsie、No-IP、PagerDuty、百富勤计算机顾问公司、Sonic.net、SURFnet 和 Virtru。

要成为 ASF 赞助商，请访问 http://apache.org/foundation/sponsorship.html

**关于 CloudBees**

CloudBees 通过提供世界上第一个端到端连续软件交付管理系统(SDM)为连续经济提供动力。对于数百万推动各种规模企业创新的开发人员和产品团队来说，SDM 建立在持续集成(CI)和持续交付(CD)的基础上，使软件交付组织内部和周围的所有功能和团队能够最好地合作，以扩大价值创造。CloudBees 是 CI、CD 和应用程序发布流程编排(ARO)的核心，建立在其产品的商业成功及其开源领先地位之上。CloudBees 是 [詹金斯](https://jenkins.io) 和 [詹金斯 X](https://jenkins-x.io) 的最大贡献者，也是 [连续交付基金会](https://cd.foundation) (CDF)的创始成员。从全栈开发人员实践 NoOps 的创业公司到大型财富 100 强公司，CloudBees 使所有软件驱动的组织能够在正确的时间智能地部署正确的功能。

超过 3，500 个世界知名品牌和超过 50%的财富 500 强企业依赖 CloudBees，因为它能够在任何开发环境中跨任何云工作，并平衡公司治理和控制与开发人员的灵活性和自由度。CloudBees 是世界领先的 DevOps 专家的家园，帮助数千家公司利用“持续一切”的力量，将他们置于从伟大的想法到伟大的软件，再到伟大的商业价值的最快路径上。

CloudBees 由 Matrix Partners、Lightspeed Venture Partners、威瑞森风险投资公司、Delta-v Capital、Golub Capital 和 unregular Ventures 支持，于 2010 年由前 JBoss CTO Sacha Labourey 和一个由持续集成、持续交付和 DevOps 专业人士组成的精英团队创立。关注 cloud bees on[Twitter](https://twitter.com/CloudBees)[脸书](https://www.facebook.com/CloudBees) 和 [LinkedIn](https://www.linkedin.com/company/cloudbees/) 。

**关于 Apache 软件基金会(ASF)**

成立于 1999 年的全志愿者基金会监管着 350 多个领先的开源项目，包括 Apache HTTP Server——世界上最流行的 Web 服务器软件。通过 ASF 被称为“Apache 方式”的精英管理流程，六大洲的 760 多名个人会员和 7，400 名委托人成功地合作开发了免费提供的企业级软件，使全球数百万用户受益:数以千计的软件解决方案在 Apache 许可下分发；社区积极参与 ASF 邮件列表、指导计划和 ApacheCon，即基金会的官方用户会议、培训和展览。ASF 是一个美国 501(c)(3)慈善组织，由个人捐款和公司赞助商资助，包括 Aetna、Anonymous、ARM、彭博、Budget Direct、Capital One、Cerner、Cloudera、Comcast、脸书、谷歌、Hortonworks、华为、IBM、Indeed、浪潮、LeaseWeb、微软、Oath、ODPi、菠萝基金、Pivotal、Private Internet Access、Red Hat、Target 和 Union Investment。欲了解更多信息，请访问 http://www.apache.org/和 https://twitter.com/TheASF
