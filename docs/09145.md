*联合解决方案提供了对跨云和内部的网络安全资产管理和网络风险的独特见解*

波士顿，马，2022 年 3 月 9 日 — [Noetic Cyber](https://c212.net/c/link/?t=0&l=en&o=3466590-1&h=954060142&u=https%3A%2F%2Fnoeticcyber.com%2F&a=Noetic+Cyber) ，一个基于云的持续网络资产管理和控制平台，今天宣布与[Rumble Network Discovery](https://c212.net/c/link/?t=0&l=en&o=3466590-1&h=3225513408&u=https%3A%2F%2Fwww.rumble.run%2F&a=Rumble+Network+Discovery)集成，后者是一个针对 IT 和 OT 环境的网络发现和资产清点解决方案。这种合作关系为安全团队提供了跨内部网络和云的组织中所有资产的完整视图，揭示了他们所有资产的当前安全状态，并映射了他们之间的关键网络关系。

网络资产发现和清点是任何网络安全计划的重要组成部分。随着新资产和设备的上线，组织需要采用能够为当今数字基础设施不断变化的本质提供更大背景的解决方案。Noetic 与 Rumble 的合作提供了端到端的网络安全资产管理解决方案，为安全团队提供了完整的资产可见性和自动化，以解决他们发现的问题。

Rumble 是一种资产清点和网络可见性解决方案，可以精确识别网络上受管的*和不受管的*设备。由以 Metasploit 的创造者而闻名的 HD Moore 共同创立的 Rumble 可以准确地对资产进行指纹识别，而不需要凭据、代理或 SPAN 端口。

“网络风险管理基于可靠的资产库存，大多数公司都很难发现不受管理的流氓设备，”Rumble 首席执行官兼联合创始人克里斯·基尔希说。“Noetic 与 Rumble 的集成确保网络风险不仅考虑到受管设备，还考虑到通常风险系数更高的不受管设备。”

Noetic 强大的自动化和工作流引擎使安全团队能够采取纠正措施。通过利用 Rumble 未经认证的主动发现功能，Noetic 可以访问高保真数据，并与其他来源相结合，帮助安全团队生成其所有资产的网络关系图，根据网络风险确定工作负载的优先级，纠正技术偏差并弥补安全覆盖缺口。

主要集成特性包括:

*   **快速实现价值**–no etic 和 Rumble 连接器附带预配置的工作流程、功能和查询，有助于加快使用速度。
*   **提高整个环境的可见性**–Rumble 独特的扫描功能可发现未受管理的设备，包括孤立资产和物联网/OT。Noetic 平台利用这些见解，以及其他安全和 IT 管理工具，来发现跨云和内部部署的策略漂移和安全覆盖缺口。
*   **一次发现问题，持续补救**–no etic 的全面自动化工作流引擎使持续的资产清点和补救变得简单，并减少了资源消耗。
*   **优化现有的 IT 和安全工具**–通过 Rumble 和 Noetic 集成发现的更新的元数据可以输入到相关的安全和 IT 管理工具中，以提高效率并推动现有投资的价值。

“我们相信，与 Rumble 的合作将为我们的共同客户提供他们在当今威胁环境下所需的完整网络资产覆盖，”Noetic Cyber 的联合创始人兼首席执行官 Paul Ayers 说。“通过将主动网络扫描与 Noetic 平台中提供的相关洞察相结合，我们能够提供跨组织的网络安全资产管理总体状态的高度准确的信息。”
