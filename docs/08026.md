# Perforce 记录了游戏和制作工作室适应新工作方式的增长

> 原文：<https://devops.com/perforce-documents-record-growth-as-gaming-and-production-studios-adapt-to-new-ways-of-working/>

*2020 年的强劲增长将持续到 2021 年，因为游戏和虚拟制作工作室通过高效、日益分散的工作和协作技术来满足日益增长的客户需求，这些技术现在是新常态的一部分。*

**明尼苏达州明尼阿波利斯，2021 年 6 月 7 日—**[Perforce Software](https://c212.net/c/link/?t=0&l=en&o=3186351-1&h=2628175483&u=http%3A%2F%2Fwww.perforce.com%2F&a=Perforce+Software)，(“Perforce”)是一家为在开发生命周期中需要生产力、可见性和规模的企业团队提供 DevOps 解决方案的供应商，该公司宣布，其 Helix 品牌的资产管理业务在游戏和虚拟生产垂直领域继续保持创纪录的增长，因为各团队提高了产量以满足不断增长的客户需求。

在疫情期间，传统的电影和电视工作室无法正常运作，导致许多重大制作的延迟。许多项目转向虚拟制作方法，以确保项目按时完成，并为流媒体服务等高需求渠道提供新内容。与此同时， [55%](https://c212.net/c/link/?t=0&l=en&o=3186351-1&h=2645515363&u=https%3A%2F%2Fwww.theverge.com%2F2021%2F1%2F6%2F22215786%2Fvideo-games-covid-19-animal-crossing-among-us&a=55%25) 的人在封锁期间转向视频游戏娱乐。这导致对解决方案的需求增加，这些解决方案可以支持安全性和可扩展性方面的协作，以满足客户需求。

领先的项目和工作室使用 Helix Core 为任何规模的项目提供安全的资产管理和团队协作，2020 年其虚拟制作和游戏垂直业务增长了 50%。疫情的扩张使得 Helix Core 在全球的客户数量超过了 4000 家。

“我们很高兴与 X 特遣队的 Perforce 合作。我们的团队使用螺旋核心，因为它只是工作。规模或安全性从来没有任何问题，我们需要确保我们可以依赖构成我们生产支柱的系统。此外，从技术用户到设计师和艺术家，每个人都可以轻松使用该工具。Perforce 是我们的一个很好的合作伙伴，”哈龙娱乐公司说。

“自 1995 年以来，Helix Core 一直是游戏和虚拟制作工作室事实上的选择。我们很高兴在这些行业快速发展的过程中与他们合作，并携手合作，确保螺旋核心和螺旋平台不断发展，以满足他们的需求。我们致力于客户的成功，Perforce Software 首席技术官" Brad Hart 。
