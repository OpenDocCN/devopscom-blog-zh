# 云安全:具有神游功能的自我修复基础设施

> 原文：<https://devops.com/cloud-security-self-healing-infrastructure-with-fugue/>

错误配置是云中数据泄露的首要原因。Fugue 有助于消除数据泄露，因为它通过提供基线漂移检测和自动化补救来修复任何漂移。神游是关于云安全的。它可以识别安全风险和违规行为，并确保它们不会再次发生。

在这次采访中， [Fugue](https://www.fugue.co/) 的联合创始人兼首席技术官 Josh Stella 解释了他的公司如何提供开发人员友好的工具，告诉你是否“违反了规则以及如何修复它”，为所有 DevSecOps 团队提供可见性和控制。有了自我修复的云基础设施，Fugue 可以纠正错误配置，并在整个配置上工作，而无需编写代码或自动化脚本。
