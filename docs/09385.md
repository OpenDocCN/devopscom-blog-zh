# CodeZero 发布了 Surf，这是一个新的开发工具，用于在生产前的 Kubernetes 环境中进行观察

> 原文:[https://devo PS . com/code zero-launches-surf-a-new-developer-tool-for-observability-in-pre-production-kubernetes-environments/](https://devops.com/codezero-launches-surf-a-new-developer-tool-for-observability-in-pre-production-kubernetes-environments/)

**温哥华，2022 年 5 月 16 日—**Kubernetes 软件开发生命周期(SDLC)现代开发工具的领导者 CodeZero 宣布正式推出 Surf [https://k8s.surf，](https://k8s.surf)这是一款开发工具，可为预生产 Kubernetes 环境提供实时、协作、丰富的查询功能。Surf 允许开发人员和团队揭开 Kubernetes 工件的神秘面纱，提供强大的故障排除功能并改进协作。

CodeZero 的产品运营副总裁劳里·加夫尼(Lawrie Gaffney)说:“在 Kubernetes 的可观察性方面，开发者一直被忽视。“他们必须混合搭配为开发运维、运营和站点可靠性工程师设计的多种工具，关联数据，然后尝试理解这些数据。有了 Surf，开发人员就有了一个强大的查询工具，只显示他们想看的内容，提供实时状态，他们可以迅速与同事分享他们正在看的内容。

Surf 在 [https://k8s.surf](https://k8s.surf) 对任何人都是免费使用的，源代码可供社区添加和分享自己的想法。功能包括:

**丰富、简单的查询**

Surf 提供了一个开发者优先的界面，结合了丰富而简单的查询和可理解的结果。查询界面支持任意字段、通配符搜索和简单连接词。

**热门结果**

随着其他因素影响环境，开发人员可以实时看到结果的变化。颜色编码可以很容易地看到警告或错误。

**可扩展 UI**

开发者可以在浏览器应用中使用键盘优先导航。插件将允许用户查看资源的具体细节，如 pod 日志和节点健康状况。开发人员还可以基于结果集使用其他 CodeZero 操作，如 Teleport 和 Intercept。

**协作**

用户只需将书签粘贴到 Slack 或电子邮件中，就可以与同事分享查询和实时结果。

**关于 CodeZero 公司**

CodeZero 是 Kubernetes 软件开发生命周期(SDLC)的现代开发工具的领导者，它支持开发团队之间的无缝实时协作。因为 CodeZero 没有本地依赖性，并集成到任何现有的环境中——移动流量而不是容器是即时的，节省了开发人员的时间，同时提供了更好的整体体验。使用 CodeZero 的组织消除了环境漂移、可伸缩性挑战和过度工具。
