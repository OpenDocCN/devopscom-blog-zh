# 安全编码冒险公园虚拟峰会:比以往更狂野、更快速、更有远见

> 原文：<https://devops.com/secure-coding-adventure-park-virtual-summit-wilder-faster-and-more-visionary-than-ever/>

DevSecOps 有助于从生产开始就保护代码，而不是在以后的阶段修补和调试代码。它将开发、运营和安全团队聚集在一起，使他们能够更快地发布安全软件。在 9 月 29 日举行的[安全编码冒险公园虚拟峰会](https://www.mediaopsevents.com/securecoding2021)上，行业领先的 AppSec 和 DevSecOps 从业者、分析师和远见者将齐聚一堂，分享他们在安全过山车上最疯狂的经历，并讨论您需要了解的关于从头构建安全代码的一切。

安全编码冒险公园虚拟峰会的议程包含一系列主题，包括 AppSec、DevSecOps、开源安全、CI/CD 等。

来自 AWS、Secure Code Warrior、GitLab、Sysdig、WhiteSource 和许多其他知名 IT 公司的安全专家将讨论:

*   如何从代码到生产防止供应链攻击
*   为什么开发人员在应用程序安全性方面苦苦挣扎
*   AppSec 测试在保护软件供应链中的作用
*   如何保护您的开源项目并创建一个安全的 CI/CD 管道
*   远程工作如何影响 DevSecOps 的采用
*   如何通过自动化您的 CI 渠道来实现简单性、可见性和控制，从而最好地实现现代软件安全性
