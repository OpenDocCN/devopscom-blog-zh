# 在 Gartner 的新应用发布自动化幻方图中，电云在愿景和执行方面排名最高

> 原文：<https://devops.com/electric-cloud-positioned-highest-vision-execution-gartners-new-application-release-automation-magic-quadrant/>

加利福尼亚州圣何塞，2016 年 8 月 3 日——devo PS 发布自动化领域的领导者 Electric Cloud([http://electric-cloud.com](http://electric-cloud.com/))今天宣布，它已被 Gartner，Inc .评为其 2016 年应用发布自动化魔力象限的领导者。*在其 ARA research 的魔力象限(MQ)中，Gartner 分析师采访了客户，并评估了公司的“执行能力”和“愿景的完整性”，其中电子云在最上面和最右边。在([http://electric-cloud . com/resources/white papers/Gartner-magic-quadrant-application-release-automation/](http://electric-cloud.com/resources/whitepapers/gartner-magic-quadrant-application-release-automation/))下载报告全文。

Electric Cloud 首席执行官斯蒂夫·布罗狄表示:“我们很荣幸被评为 Gartner 首次应用发布自动化魔力象限的领导者。“我们认为，Gartner 选择发布 ARA MQ 的事实证明了这一不断增长的市场的重要性，并相信我们在 ARA MQ 中的地位证明了我们有能力帮助全球各种规模的组织更快地交付更好的软件。”

Gartner 备受追捧的幻方图聚焦于三项关键能力，这三项能力是在应用程序发布自动化行业研究中成为一名合格的竞争者所必需的。它们是:1)自动化 2)环境建模和 3)发布协调。

科林·弗莱彻和大卫·保罗·威廉姆斯在 2016 年 7 月的一份 Gartner 研究报告中指出:“IT 组织很少在整个生命周期内一致地管理应用程序发布，由运营、开发或联合开发运维团队领导个人努力。”。*“ARA 工具提供了自动化、环境建模和发布协调功能的组合，以同时提高应用发布的质量和速度。这些工具是实现 DevOps 目标的关键部分，即通过大量快速的小版本实现持续交付。”

Electric Cloud 的 ElectricFlow 是业界第一个也是唯一一个端到端 DevOps 发布自动化平台，为高级部署策略(如蓝/绿、金丝雀和滚动部署)提供本机、按钮式支持。ElectricFlow 使 IT 能够频繁、可靠地大规模部署任务关键型应用，并避免对其最终用户的服务中断 *(* [*参见 2016 年 7 月 19 日发布的新闻稿“Electric Cloud 宣布发布业界首个端到端 devo PS”*](http://electric-cloud.com/company/news/press-releases/item/electric-cloud-announces-industrys-first-end-end/)*)。*

Brodie 补充道:“DevOps 既关乎技术，也关乎文化。”“为了帮助解决我们的客户面临的文化挑战，我们推出了名为[连续讨论(#c9d9)](http://electric-cloud.com/lp/continuous-discussions/) 的双月社区视频小组系列，并与 Gene Kim 和 IT Revolution 共同创办了两次售罄的 [DevOps 企业峰会](http://events.itrevolution.com/)。连续两年来，这些计划为大型组织中的领导者和从业者提供了有利于分享经验、讨论最佳实践和帮助彼此学习和成长的环境。这些价值也反映了我们的愿景和我们通过 ElectricFlow 提供的东西，即帮助大型企业弥合开发和运营之间的差距。”
