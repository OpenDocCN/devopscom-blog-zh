# Jelastic 借助 SaveInCloud 推进云托管联盟

> 原文：<https://devops.com/jelastic-boosts-cloud-hosting-union-saveincloud/>

加州 PALO 阿尔托(PRWEB)2016 年 8 月 25 日

支持 Docker 容器的领先 DevOps 云平台 Jelastic 宣布 SaveInCloud 成为他们来自巴拉圭的新合作伙伴。

该联盟旨在为拉丁美洲的开发者提供一个直观、经济且灵活的平台。从现在开始， [SaveInCloud](http://www.prweb.net/Redirect.aspx?id=aHR0cHM6Ly9qZWxhc3RpYy5jbG91ZC9kZXRhaWxzL3NhdmVpbmNsb3Vk) 客户可以设置自己的 Java、PHP、Ruby、Node.js、。NET 或 Python 环境，节省高达 70%，只需为正在使用的资源付费。

“通过与 SaveInCloud 合作，Jelastic 为 DevOps 提供了一个强大的平台，具有一系列广泛的功能。SaveInCloud 完美地补充了 Jelastic Cloud Union，使客户能够体验高可用性、强大的性能、完全自动化和直观的 UI，”Jelastic 首席执行官 Ruslan Synytsky 评论道。

SaveInCloud 的母公司 IP 公司位于拉丁美洲，自 1998 年起提供互联网服务。该公司在拉丁美洲和加拿大拥有 3 个高效的数据中心，托管超过 10k 个域。

“这种合作关系有助于我们为 LATAM DevOps 提供渐进式托管服务，由于本地可用的数据服务器、熟练的技术支持和高水平的性能，实现了低延迟。我们希望为本地开发者提供最终的平台即服务。在不久的将来，我们还计划在更多的拉美国家提供 SaveInCloud。

SaveInCloud 平台提供的主要优势:

*   LATAM 用户的低延迟
*   基础设施中的 100%固态硬盘
*   戴尔服务器
*   。NET 和 MSSQL 节点支持
*   西班牙语/葡萄牙语本地化和技术支持

客户有机会[免费试用由 Jelastic 提供支持的 SaveInCloud](http://www.prweb.net/Redirect.aspx?id=aHR0cDovL3d3dy5zYXZlaW5jbG91ZC5jb20v) 主机，享受全自动 DevOps 平台和先进的容器编排带来的好处。

关于 Jelastic
Jelastic 是一款面向企业的 DevOps PaaS 和 CaaS，可以部署在裸机硬件或任何 IaaS 上。该平台为 Java、PHP、Ruby、Node.js、Python 和。NET 和使用定制 Docker 容器的能力。Jelastic 提供灵活的部署模式，无需对专有 API 进行编码，为无状态和有状态应用程序、协作、访问控制、监控、备份和灾难恢复、内置计费和业务分析工具提供灵活的自动扩展，同时通过高密度和硬件利用率降低总拥有成本。欲了解更多信息，请访问我们在[http://www.jelastic.com](http://www.prweb.net/Redirect.aspx?id=aHR0cHM6Ly9qZWxhc3RpYy5jb20v)

关于 SaveInCloud
SaveInCloud 是一家专注于拉丁美洲云解决方案的托管提供商公司。
该公司总部位于巴拉圭亚松森，拥有先进的基础设施，能够以西班牙语、葡萄牙语或英语为所有客户提供高质量的服务和技术支持，并确保卓越的效率和安全性。SaveInCloud 基础设施位于 LATAM 数据中心，为所有系统(从空调到网络)提供最高的冗余，有助于满足公司客户的最高需求。http://saveincloud.com
