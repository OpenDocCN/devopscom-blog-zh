# 可持续发展不再仅仅是云计算 IT 的时髦词汇

> 原文：<https://devops.com/sustainability-no-longer-just-a-buzzword-for-cloud-it/>

最近一项关于 it 可持续性的 CloudBolt 调查发现，可持续性不再仅仅是一个时髦的词——市场需要真正的改变，似乎环境问题最终正在影响大型 IT 投资。例如，67%的受访者表示，云提供商的可持续发展计划在决定是否与其合作时非常重要。相当一部分组织甚至愿意支付额外费用来支持绿色计划！

下面，我将回顾这份报告，找出它的要点。我们将看到可持续发展如何指导云 IT 中的实际行动，并简要比较市场如何看待三大云提供商(AWS、GCP 和 Azure)当前的绿色计划。

## 可持续发展获得现实世界的关注

2021 年 6 月和 7 月，CloudBolt 在其 Pulse 平台上对 256 名全球首席执行官和技术领导者进行了调查，发现 76%的受访者现在认为气候变化是一种生存威胁。不仅是 IT 购买者考虑到了计算和成本，而且一种可持续发展的良知也正在影响着高管们去引导彻底的变革。事实上，79%的受访者表示，他们的 IT 部门有望帮助他们的公司实现由最高管理层董事会推动的特定可持续发展计划。

有趣的是，其他研究发现，可持续发展承诺会带来切实的业务成果。例如，由牛津大学和 Arabesque 进行的 88%的研究表明，扎实的环境、社会和治理(ESG)实践可以提高企业的运营绩效和利润。这项研究还指出，80%的时候，公司的股票价格受到可持续发展实践的积极影响。

## 气候良知引导云选择

现在，大多数组织在选择是否与云供应商开展业务时，都会考虑该供应商的可持续发展计划。整整 78%的受访者认为，他们做出的公共/私有云选择会对环境产生重大影响，无论是好是坏。新的环境云承诺不仅试图最大限度地减少数据中心的碳排放，还考虑了生态影响，旨在消除公司的历史碳足迹。

云消费者也希望为解决这些绿色计划做出贡献。令人印象深刻的是，79%的受访者表示，他们愿意支付额外费用与将可持续发展融入其业务的供应商合作。41%的公司愿意支付当前账单 11 %至 15%的溢价。这表明，可持续发展现在是市场中的一个重要因素，公司愿意付出代价来提高自己的环境地位。

尽管这种承诺无疑在增长，但该报告确实发现，在比较云提供商时，安全性、成本和敏捷性等运营要求仍明显高于可持续性。尽管 IT 专业人员认识到气候变化是一个生存威胁，但他们仍然有日常工作，必须保持运营。

## AWS:绿色云之王

略多于一半(51%)的受访者认为，全行业的绿色能源承诺是真实的，并且在所有云提供商中不断增长。然而，当比较前三名——亚马逊网络服务(AWS)、Azure 和谷歌时，AWS 似乎正在成为 IT 可持续发展的领导者；41%的受访者认为 AWS 在解决绿色倡议方面做得最多。

AWS 有望在 2025 年前为其全部运营提供 100%的可再生能源。随着 2021 年 4 月在[投资太阳能工厂，亚马逊成为欧洲最大的可再生能源企业买家。](https://press.aboutamazon.com/news-releases/news-release-details/amazon-becomes-europes-largest-corporate-buyer-renewable-energy)[在全球范围内](https://sustainability.aboutamazon.com/about/around-the-globe)，AWS 投资了现场太阳能、太阳能农场和风力农场，为其数据中心和全食超市部门提供电力。

值得称赞的是，微软和谷歌也致力于影响其云足迹的气候承诺。除了进一步投资改善水、废物处理和周围的当地生态系统之外，Azure 还支持一项类似的倡议，即到 2025 年实现 100%的可再生能源。谷歌云的[承诺](https://cloud.google.com/sustainability)是到 2030 年 24 小时依靠无碳能源运行，该公司自 2017 年以来已经实现了每年 100%可再生能源。

AWS 不仅在解决其环境影响方面做得最多，而且作为云领导者，它还保持了最大的市场吸引力。大多数受访者(62%)表示 AWS 正在赢得云战争，只有 31%的人表示 Azure，只有 7%的人表示 GCP。

## 最后的想法

令人振奋的是，ESG 标准正在对公司的云计算忠诚度产生显著影响。市场现在需要真正的改变，并愿意为更道德和可持续的云托管付出代价。

但是选择云提供商只是减少 IT 影响的一个方面。正如我在之前报道的那样，有许多方法可以抑制其气候影响。这些措施包括优化整体基础设施、关闭僵尸虚拟机、避免过度配置资源、提高性能可见性、重构代码和其他措施。最棒的是，这些计划中的大多数都有降低业务成本的额外好处。
