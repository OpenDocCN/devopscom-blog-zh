# 茄子通过深度学习增强其数字自动化智能平台

> 原文:[https://devo PS . com/guillenge-enhances-its-digital-automation-intelligence-platform-with-deep-learning/](https://devops.com/eggplant-enhances-its-digital-automation-intelligence-platform-with-deep-learning/)

进一步简化和加速测试自动化，并提供影响分析以检测风险

伦敦和博尔德，2020 年 5 月 28 日] [茄子，](https://www.eggplantsoftware.com/)智能自动化专家，今天宣布更新其数字自动化智能(DAI)平台。随着冠状病毒限制和转向远程工作导致的数字需求增加，这些新功能通过智能、人工智能驱动的自动化为组织提供了更快的上市时间。通过简化和统一企业规模部署的测试，以跨组织自动化整个测试生命周期，茄子确保软件和应用程序的性能符合用户的期望。

随着对数字化的依赖日益增加，对企业来说，扩展和优化开发运维以提供无摩擦的高质量数字化体验变得前所未有的重要。茄子的 DAI 平台支持持续的智能自动化，从用户的角度进行测试，而不仅仅是代码合规性。新增加的内容有助于通过进一步简化和提高自动化和洞察力的水平来加速软件交付的步伐，同时为测试团队带来效率的提高。

茄子的 DAI 平台新增内容包括:

*   自动测试生成的深度学习模型:通过智能应用分析，这种能力可以跨任何平台自动识别和分类 UI 元素，包括销售点和移动设备。此外，它自动生成即时运行自动化测试所需的所有自动化资产。

*   业务影响预测器:该功能使用强大的分析来驱动可视化，向用户显示常见应用程序故障模式的重叠，这些模式也被客户大量使用。这些信息可以用来自动优化测试和通知发布决策。

*   用于大规模测试执行的中央资产存储库:这一功能使得在多台测试机器上部署茄子的企业级测试功能变得非常容易，并且通过一个单一的现代界面进一步简化了测试过程。
