# Fluree 开源了它的整个 Web3 数据平台

> 原文：<https://devops.com/fluree-open-sources-its-entire-web3-data-platform/>

*   受区块链保护的语义图数据库为开源用户(包括开发人员、教育机构、初创公司和企业)提供以数据为中心的应用

*   开发人员社区现在可以广泛访问 Fluree 代码和库，利用并促进安全数据平台

*   Fluree 完全基于标准构建，可以跨任何云或内部环境进行部署。与行业标准 SPARQL、GraphQL 以及 Javascript、NodeJS 和 React 库的本机集成使 Fluree 易于任何现代开发人员使用。

北卡罗来纳州 WINSTON-SALEM—2020 年 10 月 20 日— [安全数据管理领域的市场领导者 Fluree](https://flur.ee/) 将在 AGPL 开源许可下发布其核心源代码。开发人员现在可以从 Github 上的 Fluree 中获得并为其做出贡献，进而构建一个新的互联网生态系统，促进以数据为中心的安全性、可追溯性和全球互操作性。

“通过开源我们的技术，我们拒绝了以专有格式锁定数据的现状，而是坚定了我们为现代数据管理问题构建最佳开源解决方案的承诺，”Fluree 联合首席执行官 Brain Platz 表示。“我们正在为企业提供一座桥梁，从供应商锁定走向完全数据所有权、可移植性和互操作性的未来。”

出现在 Web3.0 时代的不断增长的社区可以使用 Fluree 来创建开源的、基于 W3C 标准的应用程序。Fluree 的主要用例包括可验证的凭证、分散的标识符、主数据管理、开放数据平台以及与公共区块链的集成。这些应用对于未来可信、安全的 Web3.0 至关重要。公益公司 Fluree(PBC)认识到专有软件在生态系统和创新方面的障碍，为了所有人的利益，正在开放其经过企业验证的平台。

“作为一家公司和技术，Fluree 认识到基于社区的创新的重要性，”Fluree 的联合首席执行官 Flip Filipowski 说。“我们一直信奉开源思维，并一直计划在合适的时候加入 Fluree。我们欢迎各行各业的创新者加入 Fluree 社区，与我们共同建设。”

Fluree 的开源代码现在可以在以下链接查看和下载:[https://github.com/fluree/](https://github.com/fluree/)。

Fluree 可在:[http://flur.ee/getstarted](http://flur.ee/getstarted)下载/DbaaS。
