# Reactive Foundation 启动以支持软件架构的下一阶段

> 原文:[https://devo PS . com/the-reactive-foundation-launchs-to-support-next-phase-of-software-architecture/](https://devops.com/the-reactive-foundation-launches-to-support-next-phase-of-software-architecture/)

**架构**

*阿里巴巴、Lightbend、Netifi 和 Pivotal 建立了一个新的中立开源基金会，以加快反应式编程规范和软件的可用性*

旧金山，2019 年 9 月 10 日——通过开源实现大规模创新的非营利组织 Linux 基金会今天宣布推出 Reactive Foundation，这是一个由领导者组成的社区，旨在加速构建下一代网络应用的技术。该基金会由阿里巴巴、Lightbend、Netifi 和 Pivotal 作为初始成员组成，包括成功的开源规范 RSocket 和编程语言实现。

反应式编程的目标是构建保持一致用户体验的应用，而不管网络流量、基础设施性能和不同的最终用户设备(计算机、平板电脑、智能手机)。反应式编程使用消息驱动的方法来实现当今联网的云原生应用程序所需的弹性、可伸缩性和响应能力，而不依赖于其底层基础架构。

Reactive Foundation 为支持开源反应式编程项目建立了一个正式的开放治理模型和中立的生态系统。

“我在网飞工作期间，从我们开始研发 RSocket 开始，我们的目的就是要有一个鼓励广泛采用的开放系统，这对于网络技术来说是必不可少的。Netifi 和基金会社区主席、联合创始人兼首席技术官瑞兰德·德格南说:“我们很高兴能在 Linux 基金会得到业界领袖和颠覆者的支持，我们也很高兴能在支持反应式编程方面取得进展。

Linux 基金会战略项目副总裁 Michael Dolan 表示:“随着云原生计算和现代应用程序开发实践的兴起，反应式编程解决了信息流方面的挑战，对于采用这种编程至关重要。“有了 Reactive Foundation，业界现在有了一个中立的家来支持支持 Reactive 编程的开源项目。”

RSocket 是一种开源协议，建立在反应流的基础上，提供网络上的应用流控制，以防止中断并提高应用的弹性。它旨在支持反应式编程和当今基于微服务的云原生应用，作为传统 HTTP 的高性能替代。RSocket 允许使用单个连接，消息通过该连接作为数据流传递。它支持跨不同传输连接的长期流，这对于网络连接频繁断开、切换和重新连接的移动设备到服务器的通信特别有用。

**附加引号:**

“RSocket 旨在在微服务和物联网设备时代大放异彩，”阿里巴巴开发者权益倡导者石安迪表示。“我们相信，建立在 RSocket 协议和反应流之上的项目将会颠覆微服务架构的格局。反应基金会是这些激动人心的项目的中心。”

Netifi 首席执行官 Arsalan Farooq 表示:“RSocket 应用协议是 HTTP 的重要替代品，可为联网的云原生应用和微服务通信提供弹性和高性能。“反应式基金会将努力围绕 RSocket 和反应式编程建立一个更广泛的开源社区。”

Pivotal 项目反应堆负责人夏羽·马尔蒂尼说:“许多观察家，从行业分析师到决策者和开发者，都认识到了设计反应式软件的积极影响。“经过十多年的创新，反应式生态系统正在成为主流，Project Reactor、Spring Boot 和 Spring Framework 加速了它的采用。Reactive Foundation 利用 RSocket 这样的定义性项目来巩固这一地位，可谓恰逢其时。通过重新思考我们设计分布式系统的方式，并使用正确的协议来协调它们，我们可以一起构建超高效、可扩展的分布式系统。反应基金会邀请所有准备好踏上这一变革之旅的人。”

Lightbend 首席技术官兼联合创始人 Jonas Bonér 表示:“2013 年，当我与他人共同撰写 [Reactive Manifesto](http://www.reactivemanifesto.org/) 时，我们预计，在构建能够对现代服务需求做出反应的系统方面，行业将会发生重大转变。自那以来，已有超过 26，000 人签署了这份宣言。“Reactive Foundation 是一个重要的里程碑，它将技术行业领导者聚集在一起，采用消息驱动的架构来支持云原生应用程序，这些应用程序构建得强大、有弹性、灵活，并且在编写时考虑了现代硬件、虚拟化、富 web 客户端和移动客户端。”

关于 Linux 基金会

Linux 基金会是世界顶级开发者和公司的首选组织，旨在构建加速开放技术开发和行业采用的生态系统。它与全球开源社区一起，通过创造历史上最大的共享技术投资来解决最困难的技术问题。Linux 基金会成立于 2000 年，如今提供工具、培训和活动来扩展任何开源项目，这些共同带来了任何一家公司都无法实现的经济影响。更多信息可在[www.linuxfoundation.org](https://www.linuxfoundation.org/)找到。
