# DevOps Institute 推出继续教育计划，以促进 IT 专业人员和组织的终身学习

> 原文：<https://devops.com/devops-institute-launches-continuing-education-program-to-foster-lifelong-learning-for-it-professionals-and-organizations/>

*新计划通过继续教育部门更新认证，帮助 DevOps Institute 学习社区成员掌握职业发展所需的最新技能*

**佛罗里达州 Lehigh Acres，2022 年 1 月 27 日—** DevOps Institute 是一家专业成员协会，也是推动 DevOps 人性化元素的认证机构，今天宣布了其新的继续教育计划。DevOps Institute 的目标是培养所有 DevOps 人不断学习的精神，推动他们的职业发展，帮助他们的组织走向成熟。继续教育计划为认证成员提供保持相关性、优化新兴趋势和实现其职业目标所需的技能、知识、想法和学习(SKIL)。

要保留 DevOps Institute 认证，个人需要通过获得继续教育单元(ceu)每两年更新一次认证。他们还必须是一个信誉良好的学习社区成员。了解更多关于继续教育计划的信息:[devopsinstitute.com/continuing-education](http://devopsinstitute.com/continuing-education)

“在今天的工作环境中，终身学习对所有人来说都是必不可少的，尤其是那些正在经历 DevOps 旅程的 IT 人员，”DevOps Institute 首席执行官 Jayne Groll 说。“继续教育计划提供广泛的活动和机会选择，重点关注学习、技能发展、跟上现代实践以及数字时代的职业发展。”

继续教育计划通过以下方式使 DevOps 专业人员和 IT 组织受益:

**对个人的好处:**

*   通过继续教育学分为认证提供更大的价值
*   允许个人利用当前最相关的 DevOps 技能、知识、想法和学习来保持竞争力
*   支持持续的技能提升，带来发展机会
*   向当前和潜在雇主展示对职业发展的承诺
*   提升简历、个人品牌和市场竞争力，在雇主和客户面前脱颖而出
*   提高工作生产率和效率

**对组织的益处:**

*   增强员工招聘和保留–在竞争激烈的招聘市场中，DevOps 专业人员希望了解他们的组织是否愿意支持和投资于他们的职业发展
*   提供激情和目标——培养持续学习的文化，让工作有意义和目标
*   通过学习新的或混合的开发运维技能，协助交叉培训和覆盖
*   提高团队生产力和效率
*   降低成本–通过减少停机时间、减少错误和人员流动

**继续教育学分如何运作**

所有 DevOps 学院认证在两年后失效。要更新 DevOps Institute 认证，个人和组织必须升级为 DevOps Institute 学习社区成员，并在认证到期日前通过合格的学习活动获得所需数量的继续教育单元(ceu)。

有关继续教育学分更新认证的更多信息，请访问[常见问题页面](https://www.devopsinstitute.com/continuing-education/)。

DevOps Institute 学习和认证总监 Rinku Sachdeva 指出:“随着对高技能 IT 员工的需求持续增长，我们必须为人们提供必要的学习渠道，以全面扩展他们所需的技能组合。“继续教育计划不仅是 IT 行业许多人的共同价值观，而且也反映了不断发展的主题和就业要求的多样性。作为 DevOps 人类的认证机构，我们致力于为任何人提供在职业生涯中取得成功的课程和机会。”

无论 IT 专业人士是想推进他们的职业发展，开始一个新的领域，还是追求一种新的激情，DevOps Institute 都有各种各样的[教育经历](https://www.devopsinstitute.com/events/?utm_campaign=Brand%20Awareness%202022&utm_source=January-Continuing-Education-Announcement-Press-Release)和[认证服务](https://www.devopsinstitute.com/certifications/?utm_campaign=Brand%20Awareness%202022&utm_source=January-Continuing-Education-Announcement-Press-Release)，这将帮助成员向雇主扩展他们的价值。

**关于 devo PS Institute**
devo PS Institute 是专业的会员协会和认证机构。DevOps 研究所的任务是促进 DevOps 的人文因素。这是一个由 It 从业者、顾问、人才收购和高管组成的开放且不断发展的专业社区的统一力量，有助于为支持数字化转型和新 IT 铺平道路。
