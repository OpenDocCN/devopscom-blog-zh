# Micro Focus 在最新的部署自动化版本中提升了 DevOps 的功能

> 原文：<https://devops.com/micro-focus-advances-capabilities-for-devops-in-latest-deployment-automation-release/>

更新支持发布团队和开发运维转型项目在企业范围内的快速采用

**加州圣克拉拉–2019 年 9 月 30 日** [微焦点](https://www.microfocus.com/en-us/home)(LSE:MCRO；纽约证券交易所:MFGP)今天宣布[部署自动化](https://www.microfocus.com/products/deployment-automation/overview) 6.3 正式上市，为其发布协调解决方案集提供了新的部署改进。新增强的 Micro Focus 部署自动化产品为正在拥抱 [DevOps](https://www.microfocus.com/trend/enterprise-devops) 运动的 IT 团队以及那些积极寻求提高其软件应用发布节奏的团队提供了必要的工具。

Micro Focus 负责应用程序生命周期管理的产品管理副总裁 Tal Levi Joseph 表示:“由于 DevOps 已成为 IT 组织交付关键业务软件应用程序的主要驱动力，应用程序增强的速度和频率将直接影响企业响应客户请求、竞争动向和不断变化的市场条件的速度。借助最新版本的部署自动化，团队可以更轻松地在所有交付环境中协调复杂的部署，同时还能削减成本和时间

**微焦点部署自动化 6.3 新特性和优势:**

*   简化、直观的导航帮助新手用户在更短的时间内变得高效
*   通过最少的培训，在发布团队中快速实现企业范围的采用，减少了自动化应用程序发布的时间
*   更新的用户界面与其他微焦点应用交付管理解决方案具有一致的外观和感觉

*   支持 Sonatype 的 Nexus Repository Manager 的新插件，提供无缝集成和单一事实来源，确保部署的组件与预生产环境中测试的组件相同

最新版本的 Micro Focus 部署自动化技术使公司能够轻松快速地在物理、虚拟化和云环境中自动部署应用程序版本。

micro Focus Deployment Automation 6.3 现已面向客户推出，并将于 10 月 28 日^日在拉斯维加斯举行的 DevOps 企业峰会的 400 号展位进行演示。
