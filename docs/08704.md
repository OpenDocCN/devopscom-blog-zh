# Jenkins-cloud bees 在整个企业中实现 CI 和 CD

> 原文：<https://devops.com/ci-and-cd-across-enterprise-jenkins/>

我们很高兴宣布与 CloudBees 合作举办的下一次 DevOps.com 网络研讨会:

## 【Jenkins 的 CI 和 CD 在整个企业中的应用

由于持续交付和开发运维，更快地为企业提供价值是 IT 组织的新口号。在本次网络研讨会中，CloudBees 将讨论最受欢迎的开源持续集成工具 Jenkins 如何让 DevOps 团队实现持续交付。

您将学习如何:

*   利用新的工作流功能协调连续的交付渠道，
*   使用 CloudBees 的 Jenkins Operations Center 在您的组织中横向扩展 Jenkins，
*   用 Jenkins 和 Puppet 和 Chef 实现端到端的可追溯性。
