# Veeam 的 Kasten 宣布推出新的 Kasten K10 V5.0，具有增强的 Kubernetes 安全性、简化的 CI/CD 管道和生态系统进步

> 原文:[https://devo PS . com/kasten-by-veeam-announcements-new-kasten-k10-V5-0-with-enhanced-kubernetes-security-streamlined-ci-CD-pipelines-and-ecosystem-advances/](https://devops.com/kasten-by-veeam-announces-new-kasten-k10-v5-0-with-enhanced-kubernetes-security-streamlined-ci-cd-pipelines-and-ecosystem-advancements/)

*新的特性和功能全面增强了 Kubernetes 的安全性，* *有助于将左移安全性应用于数据保护，并有助于持续的生态系统扩展*

**西班牙巴伦西亚；kube con+CloudNativeCon Europe 2022—2022 年 5 月 16 日—**[Kasten by Veeam](https://www.kasten.io/)，Kubernetes 备份和灾难恢复的市场领导者，今天宣布推出全新的 Kasten by Veeam K10 V5.0 Kubernetes 数据管理平台。这一最新版本专为 Kubernetes 打造，专注于提供全面的风险管理战略、简化的 CI/CD 渠道和新的生态系统发展细节，以优化 Kubernetes 投资并降低其风险。这些进步都旨在帮助不断扩大的 Kubernetes 社区在云原生应用程序中更高效、更安全地工作。

“由于勒索软件攻击仍然是组织最关心的问题，帮助持续识别风险、保护数据、检测攻击并为云原生应用程序提供灾难恢复的全面风险管理战略至关重要，”Veeam Kasten 产品和合作伙伴关系副总裁 Gaurav Rishi 表示。“我们的最新版本仍然专注于帮助企业最大限度地减少数据和云应用程序攻击造成的财务影响。同时，我们还支持开发人员对备份解决方案的需求，这些解决方案不仅能与他们使用的 CI/CD 工具集成，还能为在部署应用程序时自动检测和保护应用程序铺平道路。”

对许多组织来说，备份 Kubernetes 应用程序，同时适应敏捷、左移开发和分布式部署环境仍然是一个挑战。基于设备的方法不能正确解释 Kubernetes 的独特特征，并且复制不能防止基础设施故障、数据损坏、数据丢失或勒索软件。部署云原生解决方案可确保安全可靠的备份，而不会中断工作流或增加会抑制创新的复杂性。

Kasten K10 v 5.0 的主要新特性包括:

*   **安全无处不在:** 增强的平台加固，包括 KMS 与 AWS KMS 和 HashiCorp Vault 的集成，再加上 UI 仪表盘中显示的 Kubernetes-native RBAC 对象的易用性，与 AWS S3 或 S3 兼容的存储支持 S3 对象锁的勒索软件攻击检测，数据保护策略护栏，以及对具有不变性的 Veeam 加固的 Linux 存储库的支持，提供全面的勒索软件保护。
*   **Shift-left 应用于数据保护:** 新推出的亚马逊 EKS 蓝图完全集成的插件，具有全生命周期功能的三级认证 Red Hat OpenShift 操作员，用于 MS SQL 和 PostgreSQL 操作员的新内置 Kanister 蓝图，以及用于操作的简化 UX，具有更直观和简化的工作流的新蓝图编辑器，以及用于关键指标的新报告生成器。
*   **扩展的生态系统:** 实现与行业领先技术和平台的集成，确保客户可以在整个生态系统中的不断扩展的解决方案中自由选择，包括对 Red Hat Marketplace、SUSE Rancher Marketplace、VMWare vSphere with Tanzu 和 Falco 的支持。

“云原生应用正以前所未有的速度发展，借助 DevOps 和 shift-left 原则，Kubernetes 中的应用部署速度更快，可扩展性更强。然而，与此同时，备份 Kubernetes 应用程序仍然是一个持续的挑战，”Veeam 的首席技术官 Danny Allan 说。“这一最新版本为 Kubernetes 数据和应用程序提供了安全的备份和恢复，同时消除了 Kubernetes 在企业中部署和操作的复杂性。随着越来越多的 Veeam 客户寻求利用 Kubernetes，Kasten K10 V5.0 的易用性对于支持其新环境中的数据保护和备份至关重要。"

“勒索软件攻击的惊人速度和云原生应用程序开发使用的增加意味着云原生应用程序成为更严重的目标只是时间问题，”IDC 存储和计算研究经理 于尚民说。“现在，企业比以往任何时候都更需要一种备份解决方案，能够灵活地跨多种云 ，并且能够在潜在攻击发生之前主动提供对其的可见性。通过其新的 K10 V5.0 版本，Kasten by Veeam 将自己定位为寻求针对 Kubernetes 环境的全面、端到端勒索软件保护的公司的可靠选择。”
