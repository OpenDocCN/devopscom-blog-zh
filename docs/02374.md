# APN 高级层合作伙伴 Tricentis 将在 SAP Sapphire 上展示 AWS 云负载测试

> 原文:[https://devo PS . com/APN-advanced-tier-partner-trice ntis-to-present-on-AWS-cloud-load-testing-at-sap-sapphire/](https://devops.com/apn-advanced-tier-partner-tricentis-to-present-on-aws-cloud-load-testing-at-sap-sapphire/)

云的头号持续测试平台 Tricentis 今天宣布，他们将于 5 月 8 日(星期三)下午 5:30 在 SAP SAPPHIRE NOW 的 AWS 展台上展示他们的云负载测试解决方案。这一战略性新产品现已在 [AWS 解决方案空间推出。](https://aws.amazon.com/solutionspace/solutions/tricentis-testing-automation/)

“由 S/4HANA 和费奥里 2.0 支持的现代 SAP 应用程序涉及许多高度分布式的组件，其中任何一个组件的性能问题都可能对整个应用程序产生连锁反应，”Tricentis 战略计划的 Kevin Dunne 解释道。“Tricentis 允许参与质量流程的任何人从他们的桌面开始负载和性能测试。用于功能测试的相同的业务可读测试用例定义可以用于负载和性能测试。”

基于云的费奥里负载测试只是 Tricentis 行业领先的 AWS SAP 测试产品的一部分。Tricentis Tosca 是公认的 SAP[# 1 连续测试平台](https://www.tricentis.com/resources/gartner-critical-capabilities-for-software-test-automation/)，帮助领先企业实现 SAP 数字化转型计划测试的现代化。凭借市场上最先进的 SAP 业务流程测试自动化技术，Tricentis Tosca 可帮助企业轻松揭示 SAP 内外的核心端到端业务交易(API、web UIs、Java、移动接口、大型机、数据仓库等)的变化影响。

该解决方案的功能包括:
**针对 SAP 的自动化端到端测试:**自动测试您的 SAP 实施及其他。跨所有连接的应用程序+API 测试业务流程

*   **在 AWS 上验证 SAP 工作负载:**通过重新验证 SAP 业务流程来防止业务中断——衡量每个关键路径的性能影响
*   **针对 SAP 费奥里的负载测试:**轻松构建和运行专门为支持 SAP 费奥里而调整的负载测试
*   **面向业务用户的人工智能测试:** Tricentis Tosca 基于模型的测试自动化方法使业务用户能够快速创建弹性、可重用的测试资产。
