# SmartBear 宣布被 Francisco Partners 收购

> 原文:[https://devo PS . com/smart bear-announces-acquisition-Francisco-partners/](https://devops.com/smartbear-announces-acquisition-francisco-partners/)

由领先的全球技术投资公司投资，支持公司加速发展

马萨诸塞州萨默维尔。–2017 年 5 月 3 日–团队软件质量工具的领导者 SmartBear Software 今天宣布，全球领先的专注于技术的私募股权公司 Francisco Partners 收购了该公司的多数股权，旨在支持该公司的加速增长。交易条款没有公布。

SmartBear 利润丰厚，五年多来一直以两位数的速度增长。Francisco Partners 的投资将有助于支持该公司的增长，该公司的增长是由管理应用程序生命周期的解决方案需求和快速扩张的 API 经济推动的。Francisco Partners 通过提供 M&A 的专业知识、市场指导和资本，成功地加速了科技公司的发展。

随着软件渗透到经济的各个领域，对帮助开发人员构建和测试新应用程序的易用工具的需求不断增长。这些应用程序通过 API 的互联性也迫使各种各样的企业重新思考他们如何与客户互动，如何最好地为客户服务。此外，不断扩展的应用程序和设备组合使得软件质量变得更加复杂。

SmartBear 是帮助软件开发人员和测试人员使用包括 TestComplete、QAComplete、Collaborator、AlertSite 和 CrossBrowserTesting 在内的产品创建最高质量的软件应用程序不可或缺的一部分。SmartBear 的产品组合还包括一些用于 API 设计和测试的最大的开源和商业产品，包括 SoapUI 和 Swagger。每四秒钟就有一个新的 SmartBear 产品试用版。

Francisco Partners 的收购标志着 SmartBear 的主要投资者从 2007 年的原始投资者 Insight Venture Partners 转变为主要投资者。在 Insight 的投资期间，该公司从一家 U/I 测试的单一产品公司成长为拥有多条产品线和 20，000 多名客户的团队的软件质量工具行业领导者。

“我们很高兴与旧金山合作伙伴合作，”SmartBear 首席执行官贾斯汀·蒂格说。“他们在成长型公司方面的记录是无与伦比的，他们非常适合 SmartBear 的下一阶段扩张。”

Francisco Partners 的负责人 Brian Decker 表示:“SmartBear 已经建立了一个强大的产品组合，可以满足开发组织的关键软件测试管理和自动化需求。“该公司凭借其旗舰开源产品 SoapUI 和 Swagger，在快速增长的 API 生命周期管理领域占据了令人印象深刻的领导地位。我们很高兴与 Justin 和 SmartBear 管理团队合作，因为他们继续推动业务的巨大增长。”

“我们很高兴与 SmartBear 合作，因为该公司开始了下一阶段的增长，”Francisco Partners 的合伙人兼首席投资官 David Golob 说。“该公司站在软件行业两个强大趋势的交汇点上:API 经济’的到来和企业中基层团队和个人购买软件的趋势。”

智能熊软件是由 Evercore 和威尔基法尔&加拉格尔 LLP。弗朗西斯科合伙人公司由保罗·黑斯廷斯·LLP 担任顾问。
