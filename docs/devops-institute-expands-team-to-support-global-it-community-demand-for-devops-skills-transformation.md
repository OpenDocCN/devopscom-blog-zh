# DevOps Institute 扩大团队以支持全球 IT 社区对 DevOps 技能转换的需求

> 原文：<https://devops.com/devops-institute-expands-team-to-support-global-it-community-demand-for-devops-skills-transformation/>

*作为一个会员制协会，DevOps Institute 正在统一一个由 IT 专业人员、教育合作伙伴、顾问、人才获取和业务主管组成的开放式持续学习社区*

佛罗里达州 BOCA 拉顿，2019 年 4 月 8 日——DevOps Institute([https://devopsinstitute.com/](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=AqX%2Fyxxn%2FCsKfNEzXNs%2BvxKe7ZZW379%2BIapVVCHkcj06tGRioNXHyZ17WFc6l3T%2BZnZpZbrtxCRDd3DQK72vqIcB%2FERNXNKxQCSuTDCcyrXHBmAUA0ertA%3D%3D&G=0&R=https%3A%2F%2Fdevopsinstitute.com%2F&I=20190408130019.0000004c59e5%40mail6-60-usnbn1&X=MHwxMDQ2NzU4OjVjYTdiZjY5MmQ3OGYyZmM0ZWFmODVlODs%3D&S=Hm5U-2VpSWP-nQGHwHUyjETXKtynzhd8W-HNloUbLwM))是一个促进 devo PS 人性化元素的会员制协会，今天宣布随着几名新成员的加入和晋升，它将继续发展。DevOps Institute 拥有 200 多家全球注册教育合作伙伴、成千上万名认证专业人员和迅速扩大的全球成员社区，并扩大了其团队，以进一步支持其 SKIL 技能、知识、理念和学习框架。

DevOps Institute 致力于通过为 IT 从业者和 DevOps 社区提供推进其职业发展和支持其组织转型所需的整体框架来推进 DevOps 成功的人力要素。从认证和研究到现场活动和持续学习计划，DevOps 学院 100%专注于提供“专业知识”以使 DevOps 成为可能。

DevOps Institute 首席执行官 Jayne Groll 表示:“我们最近的技能提升研究结果概述了对 T 型专业人员*和席卷全球企业 IT 社区的新技能组合的日益增长的需求。“为了支持我们的成员和合作伙伴，发展我们自己的 SKIL 框架，我们用忠诚和充满激情的团队成员加强了我们的组织。我们相信充满激情、技术娴熟的员工是 DevOps 成功的关键因素。”

除了现在支持欧洲和拉丁美洲市场的社区倡导者团队，DevOps Institute 的其他团队成员还包括:

*   Rinku Sachdeva:认证和学习项目经理

Sachdeva 总部位于多伦多，在管理和营销企业认证和学习项目方面拥有丰富的经验，尤其是在 IT 社区的产品方面。她将在为合作伙伴和社区建立和提供持续学习项目方面发挥不可估量的作用。

*   阿琳·霍夫曼:全球营销总监

Hoffman 以南佛罗里达为基地，在多个 B2B 垂直市场带来了广泛的营销专业知识，包括内容管理、数字创作、品牌信息和营销自动化。她将确保行业了解并获得不断增长的 SKIL 框架知识。

*   肖恩·布什:活动管理专家

Bush 毕业于佛罗里达州立大学，获得了市场营销和企业家学位，在南佛罗里达工作，从一家翻译软件供应商来到 DevOps Institute。在未来的几年里，他将在 DevOps Institute 的成长事件计划中发挥积极作用。

此外，DevOps Institute 很高兴宣布以下促销活动:

*   Dheeraj Nayal 被提升为全球社区大使
*   Marc Halcrow 被提升为开发和运营总监
*   泰勒·普勒里被提升为数字营销和社交媒体经理
*   Alice Acedia 被提升为客户经理，为现有合作伙伴提供支持

**关于 DevOps 学院**

DevOps Institute 是一个会员协会，致力于推动 DevOps 成功的人文因素。我们相信 DevOps 最有希望帮助企业将整个 IT 范式转变为包含组织所有方面的竞争优势。我们的使命是基于技能、知识、理念和学习(SKIL)的框架，从业者需要这些来推进 DevOps 和他们的职业生涯。它的所有工作，包括认证、研究、活动和持续学习计划，都专注于提供使开发运维成为可能的“知识”。
