# SUSE CaaS 平台的 SUSE Ready 认证现可用于合作伙伴的容器化应用

> 原文：<https://devops.com/suse-ready-certification-for-suse-caas-platform-now-available-for-partners-containerized-applications/>

企业客户可以放心地部署在 SUSE 的容器管理解决方案上测试和支持的 ISV 容器化产品

丹麦哥本哈根，2018 年 5 月 2 日/美通社/—**(kube con+CloudNativeCon)**SUSE^(今日推出针对 SUSE CaaS 平台的 SUSE Ready 认证，扩展其备受推崇的独立软件供应商(ISV)应用认证计划。使用开源 docker 容器格式构建容器的 ISV 现在可以在 [SUSE CaaS Platform](https://www.suse.com/products/caas-platform/) 上认证他们的容器化应用，这是一种企业级容器管理解决方案，使 IT 和 DevOps 专业人员能够更轻松地部署、管理和扩展基于容器的应用和服务。SUSE Ready 为客户验证合作伙伴解决方案是否在 SUSE 软件上得到测试和支持，因此他们可以放心地使用最能满足其业务需求的兼容和受支持的第三方解决方案，而不会局限于单一供应商。)

“SUSE Ready 认证计划受到了合作伙伴和客户的好评，”SUSE 的 ISV 合作伙伴计划主管弗兰克·减压阀说。“ISV 社区很高兴，因为它允许他们使用现有的测试方法来确保与 SUSE 产品的兼容性。客户希望相信他们的解决方案会得到应用专家(创建解决方案的合作伙伴)的支持。SUSE Ready 认证在过去一年中大幅增长，将 SUSE Ready 扩展到 SUSE CaaS 平台将进一步扩展生态系统，并为更多合作伙伴和客户创造价值。”

容器加速了应用程序交付，促进了应用程序的可移植性，并支持一类新的高性能云原生应用程序。根据最近对 SUSE ISV 生态系统的调查，随着容器使用的增长，ISV 越来越多地为他们的应用程序提供容器部署选项。SUSE CaaS Platform 使用开源 Kubernetes 为企业提供大规模的生产级容器编排，满足 ISV 对平台的需求，以便在客户位置管理他们的容器化软件。

SUSE Ready 是一项认证计划，允许 ISV 使用 SUSE 产品测试其应用程序，以确保互操作性。ISV 执行他们在 SUSE 环境中支持其解决方案所需的测试。SUSE Ready 应用程序列在 [SUSE 合作伙伴软件目录](https://www.suse.com/susePSC/home)中，这是客户寻找支持 SUSE 软件的解决方案的主要来源。这个可搜索的合作伙伴解决方案数据库允许 ISV 展示已经过 SUSE Linux Enterprise、SUSE OpenStack Cloud 和 SUSE Enterprise Storage 测试和认证的应用。随着 SUSE [CaaS Platform](https://whatis.suse.com/definition/containers-service-caas/) 加入目录，ISV 现在有了一个集中展示其所有 SUSE Ready 应用程序和部署选项的地方，无论是物理、虚拟、云还是容器化的。

许多提供为容器部署增值的应用的 ISV 已经通过了 SUSE CaaS 平台的 SUSE Ready 认证，包括 SAP、Micro Focus、Aqua Security、Avi Networks、Datadog、Dynatrace、JFrog、Minio、NeuVector、Pachyderm、Tymlez、Univa 和 XebiaLabs。

Micro Focus 产品管理副总裁 Rahul Tripathi 表示:“随着我们继续应用程序的容器交付，SUSE CaaS 平台的 SUSE Ready 认证将非常重要，以确保客户能够将 Micro Focus 容器化应用程序部署到可信环境中。将我们的容器化应用程序与我们现有的目录条目一起列在一个集中的位置，这不仅有利于向我们的客户展示他们的各种部署选项，而且有利于利用我们的解决方案来管理其他 SUSE Ready 应用程序。”

neu vector 的联合创始人兼首席执行官黄飞说:“客户希望能够放心地部署 Kubernetes 应用程序，完全安全地投入生产。SUSE CaaS 平台的 SUSE Ready 认证提供了信心，即 NeuVector 多向量集装箱防火墙已在 SUSE CaaS 平台上经过全面测试，使他们能够保护东西集装箱流量，监控集装箱的可疑活动，并从单一安全解决方案中扫描漏洞。”

ISV 可以通过加入 SUSE 合作伙伴计划轻松开始 SUSE CaaS 平台的 SUSE Ready 认证，该计划还为 SUSE ISV 合作伙伴提供合作伙伴下载、支持、培训和联合营销机会。ISV 可以与 SUSE 合作，为合作伙伴在 SUSE CaaS 平台测试环境中部署他们的容器化应用程序。欲了解更多信息，请访问[www.suse.com/partners/isv/caas-platform](https://www.suse.com/partners/isv/caas-platform)。
