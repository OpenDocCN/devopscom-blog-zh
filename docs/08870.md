# FinConDX:在金融科技中利用 DevOps

> 原文:[https://devops.com/fincondx-leveraging-devops-in-fintech/](https://devops.com/fincondx-leveraging-devops-in-fintech/)

随着消费者越来越多地转向在线购物和非接触式支付，金融科技行业在过去几年中经历了显著增长。金融服务组织正在采用 DevOps 实践来加快软件发布速度、提高质量并满足客户期望。12 月 9 日，金融服务和技术领袖将在 [FinConDX 虚拟活动](https://www.techstrongevents.com/FinConDX)上齐聚一堂，讨论最新和最有效的 DevOps、DevSecOps 和数字化转型实践以及在金融科技行业蓬勃发展所需的工具。

[FinConDX](https://www.techstrongevents.com/FinConDX) 的议程包含多个主题，包括持续安全性和持续合规性最佳实践、价值流管理、高度监管行业中的安全开发、金融服务中的源代码安全性、威胁检测、事件响应、软件供应链攻击、云安全、新兴技术和现代化工具。
