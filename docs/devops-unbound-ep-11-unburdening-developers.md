# EP 11:减轻开发人员的负担

> 原文：<https://devops.com/devops-unbound-ep-11-unburdening-developers/>

每个应用程序开发需求似乎最终都要由开发者来满足。构建安全性。向左移动测试。以云的速度和规模为用户提供更酷、更个性化的体验。当然，要做得更快。我们如何在不累垮开发人员的情况下适应不断增长的期望？其他团队成员可以做些什么来减轻负担，让开发人员专注于他们成长所需的创造性工作？在《devo PS unbounded》的这一集中，主持人艾伦·希梅尔和米切尔·阿什利与来自 Tricentis 的亚当·卡尔西、来自 DeployHub 的特雷西·拉冈和来自 GitHub 的贾斯汀·休金斯一起讨论减轻开发者负担的问题。下面是视频，后面是文字记录。
