# CData 的数据需求和挑战

> 原文：<https://devops.com/data-needs-and-challenges-with-cdata/>

企业和应用程序需要的数据无处不在，并通过大量的技术接口进行访问，包括 API、数据库查询、ODBC 等数据层、云服务和 SaaS 应用程序(仅举几例)。

在本期 TechStrong 电视节目中，CData 的技术宣传员 Jerod Johnson 与 Mitch Ashley 一起讨论了满足数据需求所面临的挑战，这些数据需求是应用支持数字化和转型计划所必需的。
