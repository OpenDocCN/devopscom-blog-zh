# CA 技术公司完成对 Veracode 的收购

> 原文：<https://devops.com/ca-technologies-completes-acquisition-veracode/>

纽约—(商业资讯)—CA Technologies(NASDAQ:CA)今天宣布已完成对 Veracode 的收购，vera code 是软件开发生命周期中保护 web、移动和第三方应用程序的领先公司。此次收购确立了 CA Technologies 作为安全 DevOps 市场领导者的地位，将其安全业务与其广泛的 DevOps 产品组合相结合，并增加了其不断增长的 SaaS 业务。

随着交易的完成，前首席执行官 Bob Brennan 现在是公司产品开发组织 Veracode 业务的总经理，向 CA 总裁兼首席产品官 Ayman Sayed 报告。

根据协议条款，该交易的现金价值约为 6.14 亿美元。

CA 将在 5 月份报告第四季度收益时提供关于收购对指南的影响的详细信息。
