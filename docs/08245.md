# Hazelcast 推出实时智能应用平台

> 原文:[https://devo PS . com/hazel cast-unveils-real-time-intelligent-applications-platform/](https://devops.com/hazelcast-unveils-real-time-intelligent-applications-platform/)

*Hazelcast 平台为实时应用和分析统一了流事件和静态数据*

**加利福尼亚州圣马特奥，2021 年 7 月 14 日**–[实时智能应用平台 Hazelcast](https://hazelcast.com/) 今天宣布推出全新的 Hazelcast 平台，这款企业软件能够作为交易、运营和分析工作负载的单一平台。Hazelcast 平台将实时流处理引擎的功能与内存计算相结合，提供了一个简化的架构，具有高性能、可伸缩性和可靠性。

根据最近对 600 名 IT 决策者的全球调查，近五分之四的专业人士表示，他们在合并实时流数据和静态数据时面临一定程度的困难。借助 Hazelcast 平台，企业可以将事件流与传统数据源(如数据库和数据湖)的上下文统一起来，从而加快收入增长、显著降低风险或提高成本效率。

Forrester1 表示，“传统数据平台无法满足新的业务需求，这些需求要求实时数据、性能、规模、集成数据和安全性的完美结合。不同的技术堆栈会影响向各种应用程序、运营系统和分析提供及时、集成的数据。最大的优势是能够在单个数据库中执行所有这些工作负载(事务性、操作性和分析性)，利用内存、多模式、分布式和云架构方面的创新。”

从电子商务到金融服务再到制造业，受益于 Hazelcast 平台的使用案例数不胜数。无论是交易监控还是为客户量身定制的产品，企业都可以利用 Hazelcast 平台来组合不同的数据系统，并在数据生成时持续处理数据，并与历史数据相统一。

Hazelcast 首席执行官 Kelly Herrell 表示:“虽然数据仍然是企业最有价值的资源，但只有当他们能够及时获得可操作的见解时，它才是有用的。“Hazelcast 平台代表着在创建实时、智能应用程序方面向前迈出的一大步，这些应用程序可以帮助企业获取他们可能会错过的价值。”

与 Hazelcast 平台的高性能和可靠性相辅相成的是它简化应用程序开发和维护的能力。凭借其执行事务性、分析性和操作性工作负载的能力，Hazelcast 平台可作为应用程序访问的单一数据层和接入点。由于 Hazelcast 强大的连接器库，这一优势成为可能。从 MongoDB 到 Redis Labs 再到 Snowflake，Hazelcast 开发和维护每个连接器以确保高性能、可靠性和安全性，因此开发人员只需针对 Hazelcast 构建应用程序。

Hazelcast 平台目前处于测试阶段，预计将于 2021 年 8 月正式推出。作为该公司与 IBM[合作](https://hazelcast.com/press-release/hazelcast-to-provide-additional-capabilities-to-ibm-cloud-pak-for-multicloud-management/)的一部分，Hazelcast 平台的功能也将通过 IBM Cloud Paks 的 [Hazelcast 平台提供，这是一个运行在 Red Hat OpenShift 上的集成软件产品。](https://www.ibm.com/cloud/hazelcast)

欲了解更多关于 Hazelcast 平台的信息，请访问:[https://hazelcast.com/products/hazelcast-platform/](https://hazelcast.com/products/hazelcast-platform/)
