# 请击鼓！2021 年 DevOps 十二大奖的获奖者是…

> 原文：<https://devops.com/drumroll-please-the-winners-of-the-2021-devops-dozen%c2%b2-awards-are/>

我们非常自豪地宣布 2021 年 DevOps 十二大奖的获奖者。在过去的七年里， [DevOps 十几个奖项](https://devopsdozen.com/) 已经认可了 DevOps 社区的明星。这些年来，我们扩大了奖项的范围，以表彰十几个 DevOps 工具和服务以及十几个 DevOps 社区最优秀和最聪明的人。

每年都很难在每个类别中选出一个获胜者。有这么多有价值的竞争者，甚至将每个类别削减到六七个决赛选手都很困难。

一旦每个类别的决赛选手被选中，事情就变得更具挑战性了。我的评委同事们( [【米切尔·阿什利](https://www.linkedin.com/in/mitchellashley)[杰恩·格罗](https://www.linkedin.com/in/jaynegroll)[海伦·比尔](https://uk.linkedin.com/in/helenjbeal) 和 [约翰·威利斯](https://www.linkedin.com/in/johnwillisatlanta) )和我试着看看大众投票并以此来指导我们。但我们也非常熟悉这个领域，并试图让这些奖项不仅仅是一场人气竞赛。

我知道这听起来像是老生常谈，但这确实是真的——今年 DevOps 十几个奖项的每一位决赛选手都应该是他们所在类别的获奖者。祝贺他们所有人。你可以在这里 看到完整的入围名单 [。](https://devopsdozen.com/the-2020-devops-dozen-awards-finalists/community-awards-nominations-closed/devops-dozen-2021-community-awards-finalists/)

然而，正如他们在电影《汉兰达》中所说，“只能有一个。”事不宜迟，我很高兴宣布 2021 年 DevOps 十几个奖项的获奖者:
