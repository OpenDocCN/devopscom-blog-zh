# DevOps 聊天:与 nClouds 首席执行官 JT·吉里讨论 DevOps 和云计算之路

> 原文:[https://devo PS . com/devo PS-chat-devo PS-vehicle-road-cloud-JT-giri-nclouds/](https://devops.com/devops-chat-devops-vehicle-road-cloud-jt-giri-nclouds/)

对于我们 IT 行业的许多人来说，迁移到云已经是旧闻了。甚至 DevOps 如何在通往云的道路上提供帮助的概念也不是一个新颖的想法。然而，我们需要记住，虽然许多组织都有一些基于云的基础架构，但真正向云迁移仍在我们前面。出于同样的原因，虽然 DevOps 比以往任何时候都更受欢迎，但完全采用 DevOps 的组织的百分比仍然很低。

我与 nClouds 联合创始人兼首席执行官 JT·吉里的谈话深入探讨了这一事实，以及像 nClouds 这样的公司如何帮助大大小小的组织实现这一转变。像往常一样，下面是我们对话的音频流，然后是我们聊天的文字记录。
