# DevOps 学院宣布推出新的 DevOps 领导者(DOL)认证课程

> 原文:[https://devo PS . com/devo PS-institute-announces-launch-new-devo PS-leader-dol-certification-course/](https://devops.com/devops-institute-announces-launch-new-devops-leader-dol-certification-course/)

**成功通过** **新 DevOps 从业者** **基于角色的课程的考生将获得 *DevOps 领导者(DOL)* 认证**

佛罗里达州 BOCA 拉顿——2017 年 7 月 13 日——DevOps 学院(DOI)，一个围绕新兴 devo PS 实践、培训和认证的持续学习社区，自豪地宣布推出一项新的认证: *DevOps Leader (DOL)* 。这一独特的课程提供了现代 IT 领导者在其组织内参与 DevOps 文化转型所需的见解、技术和创新方法。DevOps Leader 是 DevOps Institute 的 DevOps 从业者认证系列中的最新认证，将由 DOI 的注册教育合作伙伴全球渠道提供。

DevOps 领导者课程由 Mustafa kapa dia([@ MKapadiaTweets](https://twitter.com/MKapadiaTweets))撰写，他是一位备受尊敬的 DevOps 服务线领导者和数字化转型专家。卡帕迪亚围绕理想的 DevOps 领导者设计了课程——一个帮助设计、影响、实施或激励文化转型的战术或战略个人，文化转型被证明是采用 DevOps 的关键成功因素。最成功的 DevOps 领导者了解文化变化的人类动力，并配备了实践、方法和工具来吸引 DevOps 范围内的人们。Puppet Labs 和 DORA 最近发布的 2017 年 DevOps 报告中也提到了对变革型领导者的需求，本课程将对此进行阐述。

“通过 DevOps，组织可以更快地将软件推向市场，质量更好，缺陷更少，风险更低，交付周期更短，”Kapadia 说。“对于任何参与企业数字化战略转型的 IT 专业人员来说，本课程都是必不可少的。"

IT 团队领导、经理、主管、业务利益相关者、实践者和变革推动者、工具提供商和系统集成商以及 DevOps 顾问将会发现本课程特别有帮助，因为他们探讨了关键的 DevOps 领导力主题，包括:

*   开发运营和上市时间
*   业务和 IT 视角
*   开发运维 IT 与传统 IT 的主要区别
*   规划和组织
*   绩效和文化
*   测量差异
*   成为 DevOps 组织–设计原则
    *   关注重要的事情
    *   组织工作
    *   分享信息
    *   韵律学
*   价值流图
*   小队模型
    *   小组
    *   部落
    *   章
    *   行会
*   管理文化变革
    *   文化类型
    *   变化的类型
    *   管理变革
*   流行工具和关键实践

“DevOps 领导者课程体现了成为一名有效的 DevOps 领导者的真正意义，”DevOps 研究所首席执行官 Jayne Groll 说。“这不是你典型的领导管理课。内容代表了为真实组织提供真实价值的最新和新兴实践。参加 DOL 后，候选人将能够在其组织内领导创新的 DevOps 转型，这将使他们能够更快、更频繁地交付高质量的软件。”

欲了解更多信息，请访问[http://devopsinstitute.com/courses/#DOL](http://devopsinstitute.com/courses/#DOL)。

**关于课程作者**

Mustafa Kapadia 是 IBM 数字转型 DevOps 实践的服务线负责人，这是一项业务咨询实践，专注于帮助大型企业转变其软件和应用交付。他是开发运维、云、移动、分析、应用开发、外包和离岸领域的专家。在加入 IBM 之前，Mustafa 是德勤战略与运营部的管理顾问。他住在旧金山湾区，是一个狂热的博主(如果时间允许的话)，一个演说家，也是初创公司的顾问。

**关于 DevOps 学院** DevOps 学院是围绕新兴 DevOps 实践的全球学习社区，将企业级 devo PS 培训和认证引入 IT 市场。DOI 与来自董事会、战略考试合作伙伴 PEOPLECERT 和 DevOps Express 供应商联盟的数字影响者合作，为相关、最新和可持续的 DevOps 课程内容和认证设定了质量标准。
