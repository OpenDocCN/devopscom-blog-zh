# WaveMaker 开始交付面向移动性的“无编码”应用

> 原文：<https://devops.com/wavemaker-jump-starts-delivery-no-coding-apps-mobility/>

*专业和非专业开发人员都可以将可视化、无编码的 RAD(快速应用程序开发)与增强的 WaveMaker Professional 服务相结合，在几天内实现移动应用*

**加州山景城2016 年 6 月 1 日–**[领先的企业快速应用程序开发平台软件公司](http://www.wavemaker.com/)今天宣布推出最全面的移动应用程序快速开发专业服务。有了这些新服务，专业和非专业开发人员都可以加快新应用的部署，让他们的公司提高竞争地位，加快收入增长，并减轻他们负担过重的 IT 部门。

 WaveMaker 解决了企业面临的日益严峻的挑战，即降低应用程序开发的复杂性，尤其是必须为各种平台或操作系统并行开发的应用程序。例如，WaveMaker professional 服务解决了在 Android、iOS 和 Windows 等平台上对应用程序的移动访问不断增长的需求。使用 web 技术对现有应用的前端进行现代化改造，以增加移动功能，这比为每个移动平台构建原生应用更高效、更具成本效益。

在当今市场中，三个因素对于缩短应用交付时间尤为重要:

*   企业需要敏捷开发的最佳实践方法，以减少开发时间并确保核心应用组件的可重用性
*   开发人员希望在应用开发中使用微服务架构，使用单独的预测试微服务来简化应用设计、测试和上线
*   开发人员需要新的工具，通过自动创建 API(应用程序编程接口)来实现更快的服务集成

新服务旨在帮助企业推进开发项目，利用快速应用开发的内在优势。RAD 的核心是快速原型开发，它允许开发人员在几天内创建“真实外观”的应用程序，而不会减慢整个开发生命周期。在传统的服务交付中，原型是侵入性的，并且转移了实际软件开发的开发工作。借助 RAD，原型可以快速展示应用的核心功能，而无需漫长的审批和预算过程。

WaveMaker 新的和扩展的专业服务提供完整的交钥匙应用程序开发服务。服务包括:

*   **开发服务**为企业客户提供定制应用的设计、开发和托管。
*   **快速启动** **服务**是短期咨询项目，在交付生命周期的任何阶段，在 WaveMaker 专家的帮助下快速启动应用程序开发。
*   **培训** **服务**提供灵活的选项，帮助客户快速了解 WaveMaker RAD。

“跨一系列平台的移动设备正以前所未有的速度入侵企业，”WaveMaker 首席执行官 Vijay Pullur 说。“推动用户在这些设备上工作效率的应用程序绝大多数都是创收应用程序，比如电子商务所需的应用程序。这给企业带来了一个挑战，即以极快的速度定义、开发和部署新应用。对于许多企业来说，唯一可行的选择是外包开发，这样做成本很高，而且失去了对质量的直接控制。WaveMaker 的无编码 RAD 平台，加上全面的专业服务，使内部开发人员能够掌控自己的命运。”
