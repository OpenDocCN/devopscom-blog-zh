# Trustero 通过了 SOC 2 Type 1 认证，实现了合规性即服务

> 原文:[https://devo PS . com/trustero-completes-SOC-2-type-1-认证其合规性即服务/](https://devops.com/trustero-completes-soc-2-type-1-certification-for-its-compliance-as-a-service/)

**加利福尼亚州帕洛阿尔托，2022 年 3 月 22 日**–新兴企业合规即服务的创新者 Trustero 宣布，其合规平台已完成系统和组织控制(SOC 2)1 类审计。

由美国注册会计师协会(AICPA)制定的公认审计标准 SOC 2 合规性确认 Trustero 的控制和流程符合 AICPA 的信托服务标准，并通过严格和独立的审计进行验证。

Trustero 合规即服务(CaaS)平台通过漏洞评估、安全风险分析和其他措施来确保所有业务流程和系统保持完全合规，从而建立和管理法规合规性。

Trustero 首席执行官兼创始人 Phil 刘表示:“我们很高兴通过了 SOC 2 Type 1 考试，这让客户和潜在客户对我们的合规性即服务解决方案充满了信心。“ 采取这一额外措施表明，我们采取了严格的 措施来确保客户数据的安全性和保密性，并强调了我们对继续投资安全控制的 承诺。”
