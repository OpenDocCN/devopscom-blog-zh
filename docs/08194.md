# 不再维持现状:通过企业家周刊尼采的颠覆与重塑

> 原文:[https://devo PS . com/no-more-status-quot -- disruption-and-reinvention-through-the-entrepreneurs-weekly-Nietzsche/](https://devops.com/no-more-status-quo-disruption-and-reinvention-through-the-entrepreneurs-weekly-nietzsche/)

每天我们都在重塑自己。无论我们是通过学习重新发明知识，还是通过开发重新发明技术，没有什么是停滞不前的，而颠覆性可以确保这一点。

在 [TechStrong 电视节目](https://digitalanarchist.com/videos/featured-guests/brad-feld-tstv)中，MediaOps 联合创始人兼总编辑 Alan Shimel 与 Foundry Group 现任董事总经理、《企业家周刊》 【尼采:颠覆者之书】 的作者 Brad Feld 坐在一起，讨论这种重塑以及颠覆者的重要性。

如果你不熟悉 Brad Feld，他是一位著名的风险投资家和作家，作为一名作家和资本市场公司 TechStars 的创始人，他涉足了创业社区。他与布拉德的第一家公司 Feld Technologies 的前合伙人戴夫·吉尔克(Dave Jilk)合作。

循环回到单词“，”这是一个布拉德希望你坚持下去。他说，“任何参与成为一个行业颠覆者的人，无论你是一家公司的创始人、领导者还是员工，我们都认为这适用于你正在做的事情。”

由于专注于颠覆者，布拉德和戴夫选择专注于历史上最具颠覆性的哲学家之一:弗里德里希·尼采是再合适不过的了。正如布拉德所说，“在 19 世纪 60 年代到 90 年代，当他写他的作品的主体时，尼采对现状——对一切事物——都有很大的破坏性。”像尼采一样，如果你是一名企业家或者在今天的技术社区，你就是在挑战一致性和重塑世界。

虽然这本书是为企业家而写的，但你并不一定要成为企业家才能享受这本书。“创业思维在生活的各个方面对每个人都有价值。费尔德说:“这并不意味着你必须成为一名企业家，但企业家的这种思维方式很容易理解。”。如果你是一个破坏者，或者只是喜欢学习，你会从这本书里学到很多。

如果你担心理解不了哲学部分，不用担心。" 《创业家周刊·尼采:给颠覆者的一本书》 难以置信的容易获得。布莱德引用了尼采的一小段话(一到三句),并将其翻译成当代英语。在每一句引语后都有一篇短文，意在“引发思考，而不是告诉你答案。”正如布拉德所说，“这本书的一个元目标不是告诉人们该做什么。是为了激发思考。”

作为大多数引语和文章的额外收获，实际上有一个企业家(一些著名的，一些不那么著名的)写的叙述，讨论他们的经历和当布拉德和戴夫给他们引语时他们的想法。

这本书有 52 个不同的主题，不是用来狼吞虎咽的。虽然你可以，但当你涉足你感兴趣的领域时，你会从中获得更多。Feld 和 Jilk 的希望只是让人们思考:“我们试图帮助人们围绕这些想法激发思考。想必这些话题中的任何一个都是你可以深思熟虑的。”

《创业家周刊尼采》 不是不可触及的学术作品，而是人类经验的相关反映。谈到在书中包含自己经历的重要性，布拉德强调，“我不是为了谋生而写书。我以风险投资为生。我写书是因为我爱写作，我爱阅读，我爱思考。我写作的动机是我自己经验学习的一部分。”

无论你是选择每周还是每隔几周读一章，布拉德和艾伦都建议你独立阅读每一章。想一想，内化后再继续。如果你能重读章节，那就更好了。“今天，当我做这些演讲时，我发现自己在做的是再次阅读这些引语，并在引语中发现新的东西，以及它们让我思考的东西，”布拉德说。“再说一次，不是试图告诉我引用的绝对真理，而是它刺激的想法。”
