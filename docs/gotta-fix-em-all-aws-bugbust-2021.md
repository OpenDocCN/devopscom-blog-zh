# 必须解决所有问题:AWS BugBust 2021

> 原文：<https://devops.com/gotta-fix-em-all-aws-bugbust-2021/>

软件错误令人烦恼。代码审查和错误查找并不是开发人员工作描述中特别有趣的部分。查找 bug 既繁琐又费时，而且对眼睛不好。bug 不仅会破坏你的代码，还会破坏你的生活。但是如果打击虫子能让你获得比骄傲更大的回报呢？这就是 AWS BugBust 出现的原因。

AWS BugBust 是世界上第一个面向 Java 和 Python 开发者的全球竞赛。比赛有两个总体目标:

1.  修复一百万个错误
2.  在全球范围内减少超过 1 亿美元的技术债务

代码审查和监控是常见的做法；一些组织甚至举办 bug bashes，成员们聚在一起揭示 bug 并修复它们。然而，修复代码仍然是一个艰巨的过程，还有更多的事情可以做，以使打击 bug 的努力变得有趣和快速。AWS BugBust 通过添加游戏化的积分/竞赛/奖励组件来实现这两者！

## AWS BugBust 是如何工作的？

你修复的错误越多，你获得的积分就越多，你在排行榜上的排名就越高。使用亚马逊的 CodeGuru ，开发人员将竞相修复最多的 bug(由 CodeGuru Reviewer 确定)和最昂贵的代码行(由 CodeGuru Profiler 突出显示)。Amazon CodeGuru 通过自动化过程减轻了代码审查和错误查找的痛苦。使用机器学习，亚马逊 CodeGuru 将快速扫描代码，识别错误并提供修复建议。

以下是关于比赛的更多信息:

## **您如何赢取积分？**

AWS BugBust 使用 Amazon CodeGuru Reviewer 来分析应用程序的代码，以识别 bug，并使用 CodeGuru Profiler 来解决性能问题。挑战仪表板上提供了错误和分析组，参与者可以对其进行声明、修复和评分。根据错误的严重程度(低、中和高)来计分

为了得分，参与者在他们的存储库中进行代码变更，创建代码评审请求，并触发代码构建。CodeGuru Reviewer 将作为构建过程的一部分分析代码更改，并通知更改的代码是否解决了 bug。

修复的错误总数和节省的费用将存储在 **AWS 全局 BugJar 中。**两项指标的总和将显示在全球排行榜上。

参与者将在三个关键里程碑获得数字徽章和定制礼品:修复 100、2000、10000 个错误。赃物包括别针、t 恤衫、BugJars、背包和帽衫/夹克。

## AWS 漏洞会在什么时候出现？

比赛将于 6 月 24 日星期四开始。你可以通过为你的团队设置一个 AWS BugBust 来开始。

## 终极大奖是什么？

截至 2021 年 9 月 30 日，全球排行榜排名前几位的开发者将获得一次免费的 re:Invent 2021 之旅。AWS 还将通过他们的私人活动为全球 BugJar 贡献最多 bug 的前 10 家企业颁奖，并为他们的顶级 bug 破坏开发者提供两张 re:Invent 2021 的门票。如果你是一名 Java/Python 开发人员，这个比赛就像是一边做你的工作，一边享受游戏般的氛围！

## 以下是一些重要的规则:

1.  需要有一个 AWS 帐户**才能参加挑战**(参与者可以在挑战开始时或比赛期间的任何时候创建一个)。
2.  拥有 Java 和/或 Python 代码库并存储在受支持的 GitHub、GitHub Enterprise、Bitbucket 和 AWS CodeCommit 存储库中的客户能够参与 AWS BugBust。
3.  总的来说，挑战注册和控制台体验在桌面上要好得多。控制台中的链接将指向在桌面上更容易访问的代码库。将考虑移动位置的认知度，但应优先考虑桌面位置。

AWS BugBust 不是你奶奶的 bug 狩猎。您现在拥有机器学习能力，包括自动为您找到错误并向您呈现错误列表的能力——甚至在挑战开始之前。从 6 月 24 日开始，你可以让艰巨的检查过程变得有趣，赢得一些奖励，成为消灭 bug 并找到它们的英雄！
