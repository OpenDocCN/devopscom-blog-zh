# DevOps Institute 启动“2020 提升技能:企业 DevOps 技能报告”调查

> 原文:[https://devo PS . com/devo PS-institute-opens-2020-up skilling-enterprise-devo PS-skills-report-survey/](https://devops.com/devops-institute-opens-2020-upskilling-enterprise-devops-skills-report-survey/)

第二次年度调查和结果报告旨在帮助公司更好地了解全球 IT 社区开发技能和交叉技能的当前和未来状态。

![](../Images/1b8c60bb9f44df4de6dc3e3c9b7591b8.png)

佛罗里达州博卡拉顿，2019 年 8 月 20 日-[DevOps Institute](https://devopsinstitute.com/)，一个促进 DevOps 人力要素的全球会员制协会，今天宣布启动 2020 年技能提升:企业 devo PS 技能报告社区调查。与金牌赞助商 CloudBees 和社区赞助商持续交付(CD)基金会合作，2020 年技能提升报告将基于 2019 年基准报告中的 1，600 多份调查反馈，衡量 DevOps 技能集开发的趋势和进展。

“2019 年，我们发现从领导者到个人贡献者的所有角色都同意自动化技能、流程技能和软技能是 DevOps 人类最重要的必备技能类别，”Forrester 前副总裁兼研究总监、现任 DevOps Institute 首席研究分析师 Eveline Oehrlich 说。“最重要的必备过程技能是:熟悉软件开发生命周期；理解工艺流程和分析；敏捷；具有测试驱动部署概念和方法以及系统思维的经验。前三大必备软技能包括协作与合作、解决问题、人际交往技能、分享和知识转移。第二轮调查将包括相同的技能类别，但我们增加了几个更高级别的技能问题。”

点击[此处](https://hubs.ly/H0klbNN0)参与“2020 年技能提升:企业开发技能报告”调查。从开发运维人员和 IT 领导到业务和人力资源团队成员，以及服务提供商团队成员，每个角色都应该在调查中发表意见。

DevOps Institute 首席执行官杰恩·格罗尔(Jayne Groll)表示:“这份技能提升报告旨在通过识别导致 DevOps 成功的想法和学习的技能和知识，帮助增强和推动人类转型。“DevOps Institute 期待在 SKIL 框架(技能、知识、想法、学习)和 2019 年基准报告的成功基础上，从逐年的角度提供关于 DevOps 技能组合趋势的新的和更新的见解。”

2019 年的技能提升报告提供了一个衡量未来模式和趋势的基准，通过确定哪些技能被认为是跨角色、区域和视角的“必备”、“最好拥有”和“不重要”。2019 年的技能提升报告还表明，流程、自动化和软/人力技能类别同等重要。

2020 年技能提升:企业开发运维技能报告调查将于 2019 年 12 月 20 日结束，第二份年度报告将于 2020 年初发布。新的调查建立在 2019 年报告的基础上，通过用一系列一般知识补充特定知识的深度，帮助 IT 专业人员了解他们变得更加 T 型所需的技能。此外，它有助于 it 专业人员理解最适合他们的特定学科。

若要访问 2019 年技能提升:企业开发运维技能报告，[请在此免费下载](https://info.devopsinstitute.com/upskilling-report-2019)。

CloudBees 营销副总裁 Andre Pino 表示:“去年的技能提升报告有助于确定成功业务转型最需要的 IT 技能。“今年的报告对 IT 社区来说是无价的，因为它提供了对技能和实践状态的见解，这将确保开发人员与技术一起进步。我们很荣幸与 DevOps Institute 合作，连续第二年赞助该报告，以突出对各种规模的企业 IT 组织至关重要的人类 DevOps 技能。”

“DevOps 社区对知识的渴望和对持续学习的承诺是无与伦比的，”CloudBees 开源社区主任兼 CD 基金会董事会成员 Tracy Miranda 说。“CD 基金会很高兴赞助 DevOps Institute 的技能提升报告，并致力于推动行业的发展，尤其是人的因素。对于希望了解自己所需技能以便在业务转型中扮演不可替代角色的新兴 DevOps 专业人士而言，该报告是一个重要资源。今年的报告尤为重要，因为它将与去年的基准进行比较，以衡量全球 DevOps 技能的进步。”

**关于 DevOps 学院**
DevOps 学院致力于推进 DevOps 成功的人文因素。作为一个全球性的会员制协会，DevOps Institute 是连接 IT 从业者、教育合作伙伴、顾问、人才获取和业务高管的首选学习中心，旨在为支持数字化转型和新 IT 铺平道路。

我们基于以人为本的 SKIL 框架，包括技能、知识、想法和学习，在 DevOps 社区内帮助促进职业发展并支持新兴实践。我们的所有工作，包括认证、研究、活动和持续学习计划，都专注于提供“人类知识”,以实现 IT 现代化，并使 DevOps 取得成功。
