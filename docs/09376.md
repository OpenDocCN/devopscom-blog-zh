# code 公证人将漏洞扫描添加到流行的开源供应链安全社区平台

> 原文:[https://devo PS . com/code 公证人-adds-vulnerability-scanning-to-popular-community-platform-for-open source-supply-chain-security/](https://devops.com/codenotary-adds-vulnerability-scanning-to-popular-community-platform-for-open-source-supply-chain-security/)

*社区证明服务满足了开发人员对集成、持续安全性的需求*

**KubeCon Europe，VALENCIA，Spain，2022 年 5 月 16 日—**[code 公证人](https://codenotary.com/)，软件供应链安全的领导者，今天宣布加入有史以来第一个免费的后台漏洞扫描服务，结合免费和开源的[社区认证服务](https://cas.codenotary.com) (CAS)代码签名和认证服务，以进一步保护开源供应链。

[Gartner 预测](https://www.gartner.com/en/newsroom/press-releases/2022-03-07-gartner-identifies-top-security-and-risk-management-trends-for-2022)到 2025 年，全球 45%的组织将经历过对其软件供应链的攻击，比 2021 年增加了三倍。CAS 增加了一个免费的漏洞服务，允许云原生和开源项目更好地保护他们的项目。这项额外的服务扫描资产(基于上传的哈希值)以发现任何已知的安全漏洞，如果在堆栈中发现有问题的包，则提供警报。CAS 也可以用来“不信任”任何有问题的工件。

code 公证人联合创始人兼首席技术官 Dennis Zimmer 表示:“这是一项特别独特的完全免费的代码完整性服务，它集成了自动化、持续和自我更新的漏洞扫描，在发现问题时发出警报。“开源软件的用户——几乎是每个人——都有一种免费而简单的方法来确保他们的软件供应链的安全性，这解决了一个越来越大的问题”

code 公证人是 [immudb](https://codenotary.com/technologies/immudb/) 的主要维护者，immudb 是第一个也是唯一一个开源的企业级不可变数据库，为要求苛刻的应用程序提供大规模数据持久性，每天多达数十亿次交易。code 公证人使用 immudb 来支持其公证和验证产品。immudb 的下载量已经超过 1200 万次。

任何人都可以从今天开始使用 CAS 来确保他们的开源软件对他们自己和他们的用户是安全的。code 公证人将于 5 月 16 日至 20 日在 [KubeCon Europe](https://events.linuxfoundation.org/kubecon-cloudnativecon-europe/) 的 S92 展位。

code 公证人为软件开发生命周期的编目和信任组件提供了工具，有助于证明代码的来源和安全性。该公司通过提供一个额外的防篡改层来进一步增强这一核心功能，该层每秒处理和存储数百万笔交易，在内部或作为云服务，并具有加密验证。它为开发人员提供了一种为开发工件附加[软件材料清单](https://en.wikipedia.org/wiki/Software_bill_of_materials#:~:text=A%20software%20bill%20of%20materials,the%20components%20in%20a%20product.) (SBOM)的方式，这些开发工件包括源代码、构建、存储库等等，以及他们软件的 Docker 和 Kubernetes 容器映像。

**关于公证人**

code 公证人拥有 100 多家客户，其中包括美国和欧洲的三大银行，它通过为所有工件、操作和依赖关系提供端到端的可加密验证的跟踪和出处，将易于使用的信任和完整性带入软件生命周期。code 公证人可以在几分钟内建立，并可以与现代 CI/CD 平台完全集成。它是唯一不变的、客户端可验证的解决方案，能够每秒处理数百万个事务。借助 code 公证人防篡改材料清单，用户可以立即识别其软件版本中不可信的组件。欲了解更多信息，请访问[https://www.codenotary.com](https://www.codenotary.com)。
