# CloudBees 任命布拉德·约翰逊为产品营销副总裁

> 原文:[https://devo PS . com/cloud bees-appoints-Brad-Johnson-as-副总裁-产品-营销/](https://devops.com/cloudbees-appoints-brad-johnson-as-vice-president-product-marketing/)

加利福尼亚州圣何塞—(美国商业资讯)—[cloud bees](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.cloudbees.com&esheet=51244200&newsitemid=20151216005344&lan=en-US&anchor=CloudBees&index=1&md5=6bbbd432940d1e2cd7d56c859f866b7e)，企业 Jenkins 公司和连续交付(CD)的领导者，今天宣布任命布拉德·约翰逊为产品营销副总裁。Johnson 为 CloudBees 带来了超过 16 年的产品营销经验，以及深厚的技术专业知识和对新兴技术的热情。在他的职位上，Johnson 将负责 CloudBees Jenkins 平台的产品信息和定位，并执行整体上市战略。

在加入 CloudBees 之前，Johnson 在 SOASTA 担任了五年的产品营销和业务开发副总裁。在担任这一职务之前，Johnson 担任了两年的战略联盟副总裁，在那里他建立了 SOASTA 的主要分销和技术合作伙伴关系。Johnson 还担任过 Borland 的产品营销高级总监和 Mercury Interactive 的产品营销总监。

CloudBees 创始人兼首席执行官 Sacha Labourey 表示:“Brad 在将产品推向市场、发现新机会和利用竞争差距方面的成功将为 CloudBees 和我们的客户带来巨大的好处。“随着我们继续快速发展基于 Jenkins 的业务并巩固我们在 DevOps 市场的强势地位，Brad 是管理团队的一个宝贵成员。”

“CloudBees 已经在持续交付市场建立了领导地位，我有动力帮助我们将成功提升到一个新的水平，”Johnson 说。“我很高兴成为 CloudBees 团队的一员，并对摆在我们面前的机会感到兴奋。”

**关于 cloud bees**
cloud bees([www.cloudbees.com](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.cloudbees.com&esheet=51244200&newsitemid=20151216005344&lan=en-US&anchor=www.cloudbees.com&index=2&md5=71dc17855d4d1df6af64ab459cc7df7a))，企业 Jenkins Company，是连续交货(CD)的领导者。CloudBees 提供的解决方案使 DevOps 团队能够快速响应业务的软件交付需求。CloudBees Jenkins 平台建立在世界上最受欢迎的开源持续交付中心和生态系统 Jenkins 的基础上，提供了广泛的 CD 解决方案，可满足企业独特的安全性、可扩展性和可管理性需求。CloudBees Jenkins 平台支持许多世界上最大、最关键的业务部署。
