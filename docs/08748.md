# Splunk 扩大了观察范围，同时降低了成本

> 原文:[https://devo PS . com/splunk-扩展可观察范围-同时降低成本/](https://devops.com/splunk-expands-scope-of-observability-while-lowering-costs/)

在其在线 [.conf21 会议](https://conf.splunk.com/)期间，Splunk Inc .今天公布了一系列更新，这些更新提高了可观察性，同时降低了部署其平台的总成本。

作为这一努力的一部分，Splunk 还宣布，它现在将向所有客户提供基于工作负载的定价。
以前，特定数量的小客户可以获得基于工作负载的定价，而其他客户则根据加载到 Splunk 平台的数据量获得许可。然而，这种方法往往会阻碍 IT 团队加载尽可能多的数据。

首席执行官道格·梅里特表示，工作负载定价是为了实现客户可以采用的以价值为导向的定价模型，而不是以数据为中心的模型，这种模型会阻止 IT 团队分析尽可能多的数据来改善 IT 运营。

为了进一步降低总成本，该公司还为选择基于工作负载定价的客户预览了一个 [Flex Index](https://www.splunk.com/en_us/newsroom/press-releases/2021/splunk-cloud-platform-ushers-in-new-era-of-data-driven-transformation-at-conf21.html) 选项，以便以更低的成本延长低价值数据的保留时间。Splunk 还在扩展其云存储，以包括 SmartStore 的预览版，该版本允许在 Microsoft Azure Cloud 上更经济高效地存储大量数据，最初支持亚马逊网络服务(AWS)和谷歌云平台(GCP)。

该产品组合的其他增强功能包括数据管理器的预览、针对存储在亚马逊网络服务(AWS)和微软 Office 365 中的数据的改进 onboarding 工具，以及摄取操作，这使 IT 团队能够对存储在 Splunk 或 AWS 的 S3 云服务中的静态数据之外的动态数据进行操作。

该公司现在还通过联合搜索工具和 Splunk Dashboard Studio 产品(通过 Splunk Mobile 提供)使查找和浏览数据变得更加容易，从而使创建仪表板变得更加容易。

与此同时，该公司还为 Kubernetes 提供 Splunk Operator，以便更容易在云原生环境中安装其平台。现在还有 Splunkbase，这是一个管理目录，使查找或开发 Splunk 应用程序变得更加简单。

在可观察性方面，该公司正在为其应用性能管理(APM)平台(称为 Splunk APM )提供 AlwaysOn 概要分析的预览。AlwaysOn Profiling 扩展了现有的跟踪功能，以提供连续的代码级可见性和分析。

此外，现在还有 Splunk APM 数据库可见性增强版的预览版，它可以自动发现导致事务延迟的缓慢查询，而无需检测数据库。

该公司还宣布，其利用开源技术的移动应用真实用户监控(RUM)现已与 Splunk Observability Cloud for Mobile platform 一起推出。该公司还透露，Splunk Log Observer 与 Splunk Enterprise 和 Splunk 基础架构监控 AutoDetect(一种自动发现基础架构异常的工具)之间存在集成的预览。现在还有用于管理 Microsoft365、第三方 APM 工具和综合监控的附加内容包。

最后， [Splunk 扩展了其安全产品组合](https://www.splunk.com/en_us/newsroom/press-releases/2021/splunk-security-accelerates-detection-and-response-with-analytics-fueled-automation-driven-and-cloud-delivered-solutions.html)以包括 Splunk SOAR 应用程序编辑器，该编辑器提供了编辑、测试和创建安全协调、自动化和补救(SOAR)应用程序的方法。现在 Splunkbase 上有超过 350 个 SOAR 应用程序。该公司还推出了 Splunk Intelligence Management，它基于今年早些时候收购 TruSTAR 时获得的一个平台，并通过一个名为 SURGe 的专家团队提供安全服务。

目前还不清楚这些新增功能将在多大程度上使 Splunk 能够抵御已经成为众多竞争对手的[可观察性](https://devops.com/?s=observability)平台提供商。然而，作为企业中使用最广泛的 IT 运营管理平台之一，Splunk 显然享有其决心保持的现任领导者地位。
