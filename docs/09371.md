# Deepfactor 与 Synopsys 合作，帮助开发者解决云原生供应链安全风险

> 原文：<https://devops.com/deepfactor-partners-with-synopsys-to-help-developers-resolve-cloud-native-supply-chain-security-risks/>

**西班牙巴伦西亚 kube con+CloudNativeCon EU 2022–2022 年 5 月 16 日—**[领先的开发者安全平台 Deepfactor](https://www.deepfactor.io) 今天宣布，它将与 Synopsys Black Duck 软件组合分析工具集成，以帮助开发者在开发和测试早期优先考虑和修复 Kubernetes 和云原生应用中的应用漏洞和许可风险。

根据 Gartner 的报告《如何使用软件组合分析管理开源软件风险》，“由于开源软件的安全漏洞和许可风险，应用程序领导者陷入了升级的困境，”Gartner 著名副总裁兼分析师 Anne Thomas 写道。“他们应该强制使用软件组合分析工具，以便在整个软件开发生命周期中尽早并持续地检测和补救风险。”

随着工程团队采用云原生开发并快速采用开源软件，发布的频率和复杂性都在增加。Deepfactor Developer Security 与 Synopsys Black Duck 工具相集成，可减少 SCA 警报量，并为开发人员提供上下文安全洞察，以优先处理和修复开发和测试期间发现的应用漏洞和许可证风险。

“我们 Synopsys 期待与 Deepfactor 合作，进一步支持我们的使命，为云原生和 Kubernetes 工作负载改善软件供应链中的安全机制，”软件完整性小组的 Synopsys 业务开发副总裁 Vatsal Sonecha 表示。“Deepfactor 解决方案为使用 Black Duck 的工程团队提供了有价值的上下文、优先级和补救信息，可以加速降低应用程序风险。”

Deepfactor 现在是 Synopsys 技术联盟计划的成员，该计划帮助合作伙伴与 Synopsys 应用安全解决方案进行协作、集成和互操作，以转变软件的构建、部署和操作方式。

Deepfactor 首席执行官兼创始人 Kiran Kamity 表示:“我们很荣幸能够与 Synopsys 合作，将 Deepfactor Developer Security 与 Black Duck 集成在一起，帮助云原生应用的开发者更快地交付安全、合规的代码。“这种集成为工程团队提供了独特的安全见解，使他们能够更快、更轻松地解决关键的安全风险。”

**可用性**

Deepfactor 与 Synopsys Black Duck 的集成立即可用。
