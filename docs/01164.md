# VictorOps 推出了重要的移动应用增强功能，以缩短响应时间并减少随叫随到的专业人员的疲劳

> 原文:[https://devo PS . com/victorops-unveils-significant-mobile-application-enhancements-cut-response-times-reduce-burnout-call-professionals/](https://devops.com/victorops-unveils-significant-mobile-application-enhancements-cut-response-times-reduce-burnout-call-professionals/)

*用户可以使用 iOS 和 Android 设备更快地准备、应对和恢复事故*

科罗拉多州博尔德市–2016 年 3 月 30 日–[为开发团队提供实时、敏捷事件管理的公司 VictorOps](http://try.victorops.com/l/44432/2016-03-17/9p67ff) 今天宣布了对 VictorOps 移动应用程序的重大更新。自 2012 年以来，VictorOps 的沉浸式移动应用程序会向待命团队发出事故警报，并允许用户快速无缝地解决问题。VictorOps 全功能的本机移动应用程序允许团队修复问题，不受时间和物理位置的限制。这种增加的移动性意味着随叫随到的团队不再局限于他们的办公桌，从而将员工的倦怠保持在最低水平。

升级版 VictorOps 移动应用程序的功能包括:

*   **双向聊天集成**与 [Slack](https://slack.com/) 或 [HipChat](https://hipchat.com/) ，允许用户在一个应用程序中查看系统信息和聊天补救数据，无需在应用程序之间切换
*   **团队的新应用内组织**允许用户快速搜索大型组织，以便与特定用户协调，协助事件补救
*   **新的“快速过滤器”**允许应用程序用户区分当前在线或正在使用应用程序的人和那些只是随叫随到的人
*   **能够查看图表或操作手册等警报注释**,帮助待命专业人员确定警报的严重程度
*   **高度可定制的时间线**,允许开发运维工程师全面了解他们的所有系统，或者快速关注与他们相关的最新对话或警报
*   **能够在移动应用程序中启动会议桥**,并根据需要邀请尽可能多的团队成员
*   **改进的个人摘要视图**,允许用户查看他们下次通话的时间、任何错过的消息和其他与他们相关的项目

VictorOps 产品副总裁 Joni Klippert 表示:“每天都有成千上万的客户从口袋里掏出移动设备，以便实时响应和解决问题。“这一最新版本将为随叫随到的团队节省数小时的时间，为他们提供调查、分类以及通过移动设备与其他团队成员交流所需的一切。”

面向企业和中小型企业的最新版本的 VictorOps 移动应用程序现可在 [iTunes 应用商店](https://itunes.apple.com/us/app/victorops/id696974262)或 [Google Play](https://play.google.com/store/apps/details?id=com.victorops.androidclient) 下载。有关移动应用的更多信息，请点击[此处](http://try.victorops.com/l/44432/2016-03-17/9p5rdf)。

**关于 VictorOps**

VictorOps 是一个实时、灵活的事件管理平台，专注于 IT 和 DevOps 团队的事件生命周期管理和协作。该解决方案结合了人和数据的力量来实时解决 IT 问题。VictorOps 平台可与团队成员无缝协调团队态势感知、事件创建、上报、通知和补救，不受物理位置或时间的限制。VictorOps 是一家私人控股公司，由 Costanoa 风险投资和铸造集团提供支持。欲了解更多信息，请访问[www.victorops.com](https://www.victorops.com/)并在 Twitter @VictorOps 上关注该公司。
