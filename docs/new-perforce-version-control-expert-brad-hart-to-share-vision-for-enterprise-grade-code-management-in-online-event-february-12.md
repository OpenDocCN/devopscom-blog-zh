# 新任 Perforce 版本控制专家 Brad Hart 将在 2 月 12 日的在线活动中分享企业级代码管理的愿景

> 原文：<https://devops.com/new-perforce-version-control-expert-brad-hart-to-share-vision-for-enterprise-grade-code-management-in-online-event-february-12/>

Hart 拥有 20 年为企业设计和实施版本控制解决方案的经验。

**明尼阿波利斯，2019 年 2 月 5 日**–Perforce Software 是一家为在开发生命周期中需要生产力、可见性和规模的企业团队提供解决方案的供应商，它很高兴地宣布 Brad Hart 成为新的版本控制首席技术官。有兴趣与 Hart 会面并了解他对企业级版本控制和代码管理的愿景的客户和潜在客户可以参加 2 月 12 日的现场网络研讨会。

Hart 拥有 20 多年与大型技术组织合作优化开发管道的经验。他专门研究软件工程过程、设计和实现。Hart 是 accu rev(2013 年被 Micro Focus 收购)的联合创始人，他之前在 IBM 担任 ClearCase 顾问。

对于 Perforce 总裁兼首席执行官马克·蒂丝(Mark Ties)来说，让哈特加入是一项战略决策，旨在加强公司阵容强大的专家团队。“Hart 在版本控制方面的丰富经验使他非常适合这个职位。我们的客户依赖我们来解决大规模产品开发带来的可伸缩性和性能挑战。Ties 说:“我们之所以能够做到这一点，是因为像 Brad Hart 这样的专家——这些领导者不仅在版本控制方面，而且在整个开发生命周期中都拥有深厚的技术专长。

Perforce 一直为大型开发和 DevOps 团队提供复杂的版本控制需求，但是许多大型团队仍然面临着挑战，这些挑战可以通过正确的方法来克服。对布拉德来说，这是这个职位最令人兴奋的地方之一。“我特地带着解决这些以前无法解决的问题的目标来到 Perforce，并解决世界上最复杂的开发组织的需求，”Brad 说。

在 [Perforce 网站](https://info.perforce.com/webinar-meet-brad-overview.html)上注册网上研讨会。
