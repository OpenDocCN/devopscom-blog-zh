# 新冠肺炎·疫情对科技的持久影响

> 原文：<https://devops.com/the-covid-19-pandemics-lasting-impact-on-tech/>

新冠肺炎疫情创造了重大的创新机会，并带来了新的技术解决方案，但即使危险消退，这项技术仍然有其用途。

无论是面罩检测、体温检测、面部识别还是人工智能[监视](https://devops.com/?s=surveillance)，对抗新冠肺炎的技术将会一直存在。这可能会带来一些好处，但也有一些道德问题持续存在。

在整个疫情，各学区都在想方设法降低疫情爆发的风险，并确保学生遵守安全法规。[例如，人工智能摄像头](https://www.the74million.org/article/from-face-mask-detection-to-temperature-checks-districts-bought-ai-surveillance-cameras-to-fight-covid-why-critics-call-them-smoke-and-mirrors/)是首批实施的技术之一，允许学校监控哪些学生没有戴口罩。

亚特兰大富尔顿县学区 16 岁的大三学生诺亚·杨(Noa Young)说:“我认为这对新冠肺炎的东西很有帮助，但似乎有点碍事。”。"我觉得很奇怪，我们竟然没有意识到这一点。"

虽然学生们知道摄像头在监视他们，但他们并不知道监视的范围。这些摄像头能够扫描体温，还能确定学生是否正确佩戴了口罩。

除了监控合规性，这些摄像机还能够协助追踪接触者。他们可以扫描学生的面部，确定他们与谁有过接触，使学校能够追溯感染新冠肺炎病毒的人的足迹。这只是疫情理工大学持续发展的一个例子，但随着采用和使用扩展到其他领域，并用于其他目的，安全和隐私问题开始堆积。

31 岁的 Courtenay Brown 在亚马逊的一家仓库工作，她解释了自己在整个工作日所受到的审查。亚马逊使用类似的人工智能监控来跟踪工人的生产率，如果有人长时间偏离分配给他们的任务，就会通知管理层。许多人认为这种级别的监控造成了不人道的工作环境。

布朗说:“他们基本上可以看到你做的每一件事，这都是为了他们的利益。”。“他们不把你当人看。这是贬低身份。”

这项技术的实施是推动亚马逊工人通过工会努力表明立场的因素之一。

“我们绝不会停下来，也不会让这件事把我们拖垮，”T4 亚马逊工会的联合创始人 德里克·帕尔默说。“这将会产生完全相反的效果。我们要加倍努力。”

疫情的另一个副产品是全国各行各业的组织转向远程工作。这种转变产生了对远程访问组织的工具、信息、数据和资产的需求，这种需求必然会带来固有的安全风险，尤其是因为这种访问通常需要在一夜之间完成。

公司进行调整的一种方式是采用身份和访问管理(IAM)解决方案。IAM 可用于在云中识别、记录和授权访问，这是混合或远程工作场所中安全功能的一个组成部分。

“在网络犯罪的猫捉老鼠游戏中，人们可能会说没有资源过剩这回事。一个重大的漏洞可能会拖垮一家公司， World Wide Worx 首席执行官亚瑟·戈尔兹塔克说。

尽管 IAM 被证明是应对远程访问管理挑战的可行解决方案，但网络安全威胁千变万化，现在网络攻击者已经转移了注意力——他们可能正在攻击 IAM 解决方案。

“复杂的威胁参与者正在积极地瞄准身份和访问管理基础设施，凭据滥用现在是主要的攻击媒介，”Gartner 在其 [顶级安全和风险管理趋势 2022](https://www.gartner.com/en/newsroom/press-releases/2022-03-07-gartner-identifies-top-security-and-risk-management-trends-for-2022) 报告中描述道。

为了保证数据安全，企业必须准备好应对更广泛的网络攻击，包括对 IAM 的威胁。

这只是两个技术突破的例子，它们最初被视为创新的解决方案，但后来演变成了自己的一系列问题。
