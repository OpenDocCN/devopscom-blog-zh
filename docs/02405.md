# Cloudbees 加入 Linux 基金会的汽车级 Linux 项目，帮助汽车制造商缩短开发周期，提高软件质量

> 原文:[https://devo PS . com/cloud bees-joins-Linux-foundations-automotive-grade-Linux-project-to-help-car-manufacturers-drive-shorter-development-cycles-accelerate-software-quality/](https://devops.com/cloudbees-joins-linux-foundations-automotive-grade-linux-project-to-help-car-manufacturers-drive-shorter-development-cycles-accelerate-software-quality/)

*全球首家端到端连续软件交付管理提供商(SDM ),帮助汽车技术组织为实验、质量和安全创造更多时间*

**2019 年 4 月 30 日—**[cloud bees](https://www.cloudbees.com/)，为持续经济提供动力的企业 DevOps 领导者，今天宣布加入 [汽车级 Linux](https://automotive.linuxfoundation.org/) (AGL)，这是一项改变汽车制造商构建软件方式的跨行业合作努力。AGL 是 Linux 基金会的一个开源项目，这是一个通过开源技术实现大规模创新的非营利组织。

随着更多智能汽车和其他设备的开发，汽车和硬件制造商在软件层面的竞争更加激烈。CloudBees 通过加快构建时间和缩短功能、性能和安全测试周期，帮助这些组织创造更多的时间进行实验和创新。例如，仅 25 家 CloudBees 客户使用其构建和测试加速平台[cloud bees Accelerator](https://pages.cloudbees.com/l/272242/2019-04-17/jzzjk/272242/83128/cloudbees_accelerator.pdf)，仅在 3 月份就从全球 459，331 个构建工作中节省了超过 300 万个开发人员小时**–**，在过去 10 个月中总共节省了超过 2000 万个开发人员小时。这 2000 万个小时代表了 50 万个每周工作 40 个小时的工作周，也就是 9615 年，这些组织中的开发人员能够投入到新的创新中。

Linux 基金会汽车级 Linux 执行董事 Dan Cauchy 表示:“随着生态系统的不断增强和发展，成为汽车级 Linux 社区的一员是一件令人兴奋的事情。“我们很高兴欢迎 CloudBees 成为新成员，我们期待利用他们在持续交付和应用发布自动化方面的专业知识。”

今天，130 多个 AGL 成员正在共同开发一个通用的开源平台，该平台可以作为信息娱乐、远程信息处理和仪表组应用的事实上的行业标准。在整个行业采用开放平台使汽车制造商和供应商能够共享和重用相同的代码库，从而降低开发成本，缩短新产品的上市时间，并减少整个行业的碎片化。

CloudBees 技术战略副总裁 Anders Wallgren 表示:“对于汽车技术组织来说，让新软件更快地从装配线上下来并上路至关重要，但在漫长的构建和测试周期中，更快的创新周期是不可能的。“CloudBees 是一个引擎，通过加速构建和优化测试，帮助这些开发和工程团队更快地创建更安全的汽车技术，从而创造更多时间进行迭代和实验。”

**关于 CloudBees**
CloudBees 通过提供世界上第一个端到端连续软件交付管理系统(SDM)为连续经济提供动力。对于数百万推动各种规模企业创新的开发人员和产品团队来说，SDM 建立在持续集成(CI)和持续交付(CD)的基础上，使软件交付组织内部和周围的所有功能和团队能够最好地合作，以扩大价值创造。

CloudBees 是 CI、CD 和应用程序发布流程编排(ARO)的核心，建立在其产品的商业成功及其开源领先地位之上。CloudBees 是[詹金斯](https://jenkins.io/)和[詹金斯 X](https://jenkins-x.io/) 的最大贡献者，也是[持续交付基金会](https://cd.foundation/) (CDF)的创始成员。从全栈开发人员实践 NoOps 的创业公司到大型财富 100 强公司，CloudBees 使所有软件驱动的组织能够在正确的时间智能地部署正确的功能。

超过 3，500 个世界知名品牌和超过 50%的财富 500 强企业投资 CloudBees，因为它能够跨任何云、在任何开发环境中工作，并平衡公司治理和控制与开发人员的灵活性和自由度。CloudBees 是世界领先的 DevOps 专家的家园，帮助数千家公司利用“持续一切”的力量，将他们置于从伟大的想法到伟大的软件，再到伟大的商业价值的最快路径上。
