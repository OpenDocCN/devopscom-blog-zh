# 大多数软件工程师想要远程工作选项

> 原文：<https://devops.com/majority-of-software-engineers-want-remote-work-options/>

疫情给数字生态系统带来了前所未有的变化，对用户、公司和软件工程师都产生了影响。新冠肺炎加速了正在进行的数字化转型，迫使许多组织快速创新。新发现的对数字体验的依赖增加了对软件程序员的需求，加剧了已经缺乏人才的[市场的需求。在这个时代，在家工作(WFH)变得很普遍，大多数工程师转向全职远程工作岗位。](https://devops.com/low-code-an-oasis-during-developer-drought/)

体验了远离公司环境的自由后，大多数开发人员现在希望在大流行后的未来选择远程工作。相当多的人还希望 100%远程工作。因此，许多公司制定了永久性的在家工作政策。然而，他们仍在努力应对完全远程办公的影响。例如，对于是否采用基于地点的薪资调整，以及是否应该给予这些远程员工独特的员工福利，存在不同意见。

由 Terminal 执行的 [2021 年远程工程状况报告](https://terminal.io/state-of-remote-engineering-2021)调查了北美的 1，100 名软件工程师，并探讨了疫情如何影响远程开发人员；展示远程工作对工程师的积极和消极影响。有趣的是，该报告揭示了随着国内条件恶化和远程工作变得更加全球化，国际人才对移居美国的兴趣日益下降。下面，我们将探索该研究的关键点，以假设这种新常态如何影响北美软件工程师的生产力、士气和招聘。

## 新常态:远程工作

新常态极大地改变了现状。令人印象深刻的是，调查结果显示，86%的工程师目前 100%远程工作。这与疫情之前的数字形成鲜明对比，后者显示 19%的人 100%的时间都在远程工作。这些员工中的许多人更愿意在未来保持这些新的工作安排和标准。研究发现，80%接受调查的工程师希望在家和办公室工作，三分之一的人希望 100%的时间都远程工作。

与传统工作场所相比，工程师更喜欢远程工作的灵活性，原因有很多。首先，不需要通勤，因此节省了时间和精力。工程师们还提到了工作与生活平衡的改善，并对自主权的增加表示赞赏。许多组织似乎也有效地处理了向远程工作的转变——74%的工程师说他们的公司很好地适应了远程工作。由于这些积极的结果，76%的工程师愿意在 60%或更多的时间里完全远程工作。

远程工作不仅有利于软件工程师，还可以提高生产力，帮助实现业务目标。没有工作场所的干扰，70%接受调查的工程师称他们在家工作时效率更高。然而，这种新常态并非没有缺点。随着传统企业防火墙的减少，[ciso 们在两难的境地中挣扎:](https://securityboulevard.com/2020/11/the-cisos-dilemma-balancing-security-productivity-with-a-housebound-workforce/)在授予新的 IT 自由与锁定系统和限制访问之间寻求理想的平衡，以采取[零信任](https://devops.com/how-service-mesh-enables-a-zero-trust-network/)的心态。

## 疫情和远程工作的问题

当然，全职远程工作还有其他一些缺点。在这项研究中，58%的参与者认为缺乏与同事的日常互动是一个挑战。对于 39%的工程师来说，在这种新的动态中，更难合作或感觉自己是团队的一部分。倦怠或缺乏动力以及在家分心也可能成为远程工作工程师的障碍。在家工作的孤独感会带来负面影响；38%的人说他们的精神健康在疫情变得更糟。

在这种新模式下，公司在招聘方面也面临挑战。对于许多试图快速交付新软件的公司来说，人才短缺仍然是一个大问题；64%的工程师说软件工程师短缺，65%的工程师说这个缺口阻碍了技术发展。尽管存在明显的技能差距，但大多数公司都没有提供足够的福利来吸引全职员工。在这些福利中，23%的工程师希望获得家庭办公室津贴，用于公用事业、互联网和其他费用；22%的人还希望有更灵活的工作时间，11%的人希望有新的技术和生产力工具。26%的工程师还将对心理健康的支持列为他们追求的三大好处之一。

该报告还指出了一些与远程职位聘用工程师相关的低效率问题。对于 59%的受访工程师来说，招聘涉及太多的面试。我们许多在 IT 行业工作的人可能会证明，LinkedIn 的球探曾向我们提出过不相关的邀请。因此，40%的工程师说他们经常被不懂技术的人面试。其他招聘问题包括一般性面试和长时间拖延，这两者都可能使候选人感到沮丧。看来公司必须修正他们的招聘方式，否则就有失去潜在人才的风险。

## 员工选择的时代

如果有什么不同的话，远程工程报告的状态驱动家庭的必要性，考虑全职远程工作作为一个可行的选择，为未来的雇员。为了填补工程劳动力的短缺，一些总部位于美国的公司历来都会引进国际人才。然而，移居美国工作的愿望正在下降。事实上，39%的工程师没有兴趣去美国找工作。工程师们列举了疯狂的政治(56%)、枪支暴力和安全问题(55%)、种族主义(47%)和对冠状病毒的不当处理(51%)作为不搬到美国的原因。员工必须留在公司总部附近的想法正在发生变化；因此，公司将不得不去现有人才已经居住的地方。

此外，今天，工程师比过去有更多的讨价还价的权利。“员工选择的时代已经到来，”Terminal 首席执行官 Clay Kellogg 表示。如果我们处于卖方市场，工程师是获利者。谈判权利的增加可能意味着远程工程师可能会要求同工同酬，无论他们在哪里。研究发现，总体而言，工程师不赞成基于位置的薪酬；49%的人表示，只有生活成本下降 20%，他们才会接受基于地理位置的薪酬。

除了提供有竞争力的工资，团队正在寻找创造性的方法来保持文化和避免倦怠。例如，55%的工程师认为像快乐时光和社交活动这样的虚拟聚会有助于避免疲劳。考虑到所有这些发现，公司应该考虑尽可能实行在任何地方工作的政策，并为远程员工提供额外的福利和支持，以吸引更多的工程人才。
