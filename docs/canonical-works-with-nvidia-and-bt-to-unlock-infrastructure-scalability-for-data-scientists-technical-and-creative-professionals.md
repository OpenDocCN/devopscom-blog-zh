# Canonical 与英伟达和英国电信合作，为数据科学家、技术和创意专业人士提供基础设施可扩展性

> 原文：<https://devops.com/canonical-works-with-nvidia-and-bt-to-unlock-infrastructure-scalability-for-data-scientists-technical-and-creative-professionals/>

**2022 年 10 月 11 日——**[Ubuntu KVM](https://ubuntu.com/server)——行业领先的虚拟机管理程序——通过对[英伟达虚拟 GPU (vGPU)软件产品](https://docs.nvidia.com/grid/14.0/grid-vgpu-release-notes-ubuntu/index.html)的原生支持，将其触角延伸至 AI/ML 应用和图形密集型应用，包括英伟达虚拟计算服务器(vCS)和英伟达 RTX 虚拟工作站(vWS)。Canonical 一直与 NVIDIA 密切合作，以确保无摩擦集成和一流的用户体验。这使得所有 Ubuntu 用户，从数据科学领域的用户到从事高端专业图形工作的用户，都可以从整个云基础设施领域的强大 GPU 性能中受益。

"*通过在更靠近终端设备的地方处理数据，边缘计算有可能最大限度地减少必须通过核心网络传输的数据量。英国电信*公司软件网络研究高级经理保罗·维奇说。* " *但是，我们必须避免因运营复杂性增加而抵消这些网络成本节约。Ubuntu 中引入 NVIDIA 虚拟 GPU 支持意味着我们不仅可以在核心和边缘部署中一致且经济高效地利用相同的监控和维护工具，还可以为我们的客户提供差异化服务*”。*

Ubuntu KVM 中对 NVIDIA vGPU 软件产品的支持是在 2021 年 10 月发布的 [Ubuntu 21.10 公告](https://ubuntu.com/blog/ubuntu-21-10-has-landed)之后，Canonical 在 Ubuntu Server 中引入了对 NVIDIA vGPU 软件 13.1 的支持。在上一个 Ubuntu 开发周期中，Canonical 与 NVIDIA 合作，将 vGPU 功能扩展到 KVM 之外，允许开发人员通过自助式云门户提供虚拟化 GPU 资源。因此，所有拥有当前 NVIDIA vGPU 软件许可证的 Canonical 客户现在都可以在常规[Ubuntu Advantage for infra structure(UA-I)](https://ubuntu.com/pricing/infra)条件下获得商业支持。

" *NVIDIA 长期以来一直是数据科学和计算机图形领域的领导者，提供性能优化的芯片、系统和软件，显著加速所有类型的计算和可视化工作负载。Canonical 硅联盟副总裁 Cindy Goldberg 表示。"*通过与英伟达在工程方面的合作，Canonical 提供了一个面向未来的云平台，作为下一代人工智能/人工智能和可视化的基础。**

Goldberg 提到的面向未来的云平台基于[Charmed OpenStack](https://ubuntu.com/openstack)——Canonical 的企业级 open stack 分发。对 NVIDIA vGPU 软件产品的支持包括平台上提供的丰富的生命周期管理功能。组织可以快速部署经济高效的企业级基础设施，并将其与 NVIDIA vGPU 软件集成，受益于 Charmed OpenStack 中的完全自动化。这消除了技术障碍，并允许更多的组织更快地进入人工智能/人工智能和可视化空间。

NVIDIA 企业产品营销高级总监 Anne Hecht 表示:“电信、制造和研究机构需要可靠的基础设施，以高性能和可承受的成本运行现代计算和可视化工作负载 — *无论是在数据中心还是在边缘，*。“*魅力 OpenStack 凭借其性价比优化的架构和对 NVIDIA vGPU 软件产品的内置支持，满足了这一需求。*

NVIDIA vGPU 软件使数据科学家以及技术和创意专业人员能够突破硬件可扩展性的限制，为所有云用户按需提供高性能 GPU 资源。虽然物理限制规定了虚拟机管理程序上可以安装的最大 GPU 数量，但虚拟化层将这些限制民主化，允许共享资源。这提高了平台的能效，并有助于减少数据中心的负面气候影响。

“近 30 年来，英国电信一直是气候和可持续发展行动的领导者，我们承诺到 2030 年，我们的运营将实现净零排放，”英国电信软件网络研究经理 Christian Lafferty 说。“*在边缘复制现有的数据中心技术是不够的。我们需要新的、精简的算法，能够充分利用虚拟化和集装箱化的好处，*”他补充道*。* " *由于数据科学、机器学习和视频分析的快速增长，目前对 GPU 技术有着巨大的需求。我们需要满足这一需求，同时提高能源效率，将我们对环境的影响降至最低。Canonical Ubuntu 上的 NVIDIA GPU 虚拟化为我们的可持续研究提供了基础。*”
