# InfluxData 与谷歌云物联网核心集成，通过扩展的数据收集和分析来改善用户的物联网环境

> 原文：<https://devops.com/influxdata-integrates-with-google-cloud-iot-core-to-improve-users-iot-environments-with-expanded-data-collection-and-analysis/>

*谷歌云物联网核心现在包括强大的时间序列平台，帮助用户提高运营效率，并根据物联网传感器数据进行实时决策*

**伦敦(Google Cloud NEXT LONDON)——2018 年 10 月 10 日——**influx data，[现代开源平台](https://www.influxdata.com/products/)专为指标、事件和其他时间序列数据而构建，旨在支持开发人员构建下一代监控、分析和物联网应用，今天宣布推出一款适用于 Google Cloud 物联网核心的新 Telegraf 代理。现在，谷歌云物联网核心用户可以通过使用领先的时间序列数据库解决方案，从他们的物联网环境中获得更好的洞察力和分析，以提供持续的洞察力，提高运营效率，并通过实时决策和控制更好地优化他们的业务。

谷歌云物联网核心是一套完整的工具，用于在边缘和云中连接、处理、存储和分析数据。InfluxData 的 Telegraf 是一个插件驱动的服务器代理，用于收集和报告指标和数据。有了这个新的 Telegraf 插件，组织可以通过监控和跟踪传感器数据来获取价值，使用传感器的历史数据来获得可以应用于当前情况的洞察力(预测性维护、优化的交通路线)，并在没有人工干预的情况下提供自动化的操作和控制(例如关闭泄漏的水泵)。Google Cloud IoT Core 可以通过实时捕获、理解和处理从物联网设备和传感器收集的数据，帮助客户最大限度地提高其物联网应用的投资回报率。

“InfluxData 与谷歌云物联网核心的集成能力使我们能够满足物联网客户的时间序列数据收集和分析需求，我们很高兴成为谷歌云物联网生态系统的新成员，”InfluxData 平台与合作伙伴高级总监 Sharjeel Noor 表示。“这一努力符合我们的物联网战略，我们专注于成为首选的数据服务平台，并作为更大生态系统(如谷歌云)的一部分提供这些优势。”

物联网的投资回报率取决于收集的实时数据，以及可用于实时决策分析的数据。物联网数据是时间序列数据，InfluxData 提供了领先的时间序列平台，可确保实时收集和分析数据。InfluxData 为谷歌云物联网核心提供数据服务层，允许用户收集时间序列数据，用于任何系统的仪器化、观察、学习和自动化。

自 Telegraf 项目开始以来，今天可用的 160 个插件中的大部分都是由社区而不是 InfluxData 工程师构建的，这表明了社区的积极贡献。主题专家轻松创建新的 Telegraf 插件，帮助社区快速收集更广泛的指标。因此，客户(包括谷歌云物联网核心客户)可以实时获得现有插件的任何更新或新插件的引入，以快速收集关于其工作负载的自定义指标。谷歌云物联网核心客户可以在几个小时内启动并运行，轻松收集他们的物联网数据，并将其存储在谷歌云物联网生态系统中专门构建的时间序列数据库中，用于警报、监控和实时决策。

InfluxData 提供了一个全新构建的现代物联网数据平台，以支持组织最大限度地利用其设备和传感器提供的大量物联网数据，为其客户提供优势。物联网数据的独特之处在于，传感器提供带时间戳或时间序列的数据来衡量随时间的变化，数据从传感器源源不断地流出，传感器实时捕获和发射数据。借助 InfluxData，公司始终可以直接查看其每个设备和传感器的状态，以便进行更改来保持服务高效运行。根据 [DB-Engines 的最新结果](https://db-engines.com/en/ranking/time+series+dbms)，其 InfluxDB 数据库在[时间序列数据库管理系统](https://www.influxdata.com/time-series-database/)中遥遥领先。

InfluxData 通过提供增长最快的开源平台，使客户能够获得更好的业务洞察力、数据驱动的实时操作以及整个基础设施(从应用程序到微服务，从系统到传感器)的统一视图，迅速建立了跨行业的开发人员和客户基础，包括制造、金融服务、能源和电信。包括思科系统、Coupa Software、IBM、Houghton Mifflin Harcourt、Nordstrom 和 Tesla 在内的 420 多家客户选择 InfluxData 作为他们衡量指标和事件的现代数据平台。InfluxData 是在现代指标和事件平台中向时间序列转变的先锋，并使客户有可能成为数据驱动型并采取数字化转型举措。
