# CA Technologies 的企业自动化平台提供智能，以提高速度和效率

> 原文:[https://devo PS . com/enterprise-automation-platform-ca-technologies-delivers-intelligence-greater-speed-efficiency/](https://devops.com/enterprise-automation-platform-ca-technologies-delivers-intelligence-greater-speed-efficiency/)

*新版本为当今的现代软件工厂提供统一的产品套件*

**纽约—2017 年 10 月 10 日—**CA Technologies(NASDAQ:CA)今天宣布了 [CA Automic One 自动化平台](https://automic.com/business-automation-platform)的重大新发布，这是一套统一的自动化产品，能够为现代软件工厂提供[智能自动化。最新版本针对的是一个快速增长的市场，该市场到 2020 年有望超过 60 亿美元，其中云、大数据和 DevOps 等关键增长领域推动着数据中心自动化。](https://automic.com/modern-software-factory-overview)

CA Technologies 总裁兼首席产品官 Ayman Sayed 表示:“企业继续利用定义明确、高度自动化的流程，以更短的周期开发软件，并以更高的速度将其部署到生产中。“智能自动化提供了增强企业敏捷性的途径。通过使企业从传统流程自动化转向智能业务自动化，CA Automic 兑现了现代软件工厂的承诺。”

**用 V12.1 自动化现代软件工厂**

与市场上结合了不同用户界面、报告工具、分析数据库和引擎的其他产品不同，[CA Automic One Automation Platform](https://automic.com/business-automation-platform)独特地运行在单一、统一、通用的平台上，提供全面的业务驱动型解决方案。

o **对于需要跟上现代开发团队和业务发展速度的 IT 运营人员来说**:

o CA Automic V12.1 提供了独特的智能自动化产品，以简化流程并在现代软件工厂中提供零接触自助服务功能。

o **对于开发领导者来说，他们必须解放开发团队，专注于现代软件工厂中的创新**:

o CA Automic V12.1 提供了新的左移功能，以消除脚本和手动工作，并帮助开发团队以卓越的零接触自助方式在 DevOps 工具链内实现自动化并协调整个工具链。

o **对于要求开发& IT 转向敏捷和现代应用交付的企业主**:

o CA Automic V12.1 通过为全自动敏捷企业提供独特的智能自动化以及在 DevOps 工具链内实现自动化和协调的能力，帮助企业推动收入增长和客户体验。

[CA Automic One Automation](https://automic.com/business-automation-platform) 平台增强了每个公司的转型之旅，以交付灵活的业务、更快地部署应用程序、增强安全性并实现应用程序性能的自动修复。

**致力于智能自动化的现代软件工厂**

根据 IDC 和 Forrester 的研究，CA Technologies 是工作负载自动化和连续交付和发布自动化的全球顶级提供商之一。2017 年 9 月，CA Technologies 在 Gartner 应用发布自动化幻方图中被评为领导者。4

今天发布的新平台是 CA Technologies 持续致力于为企业实现智能自动化的一部分。CA Technologies 致力于为现代软件工厂注入活力以加速业务转型，它将包括 Automic 在内的全公司自动化产品系列整合在一起，以支持现代应用交付。

**配套行情**

“开发运维/敏捷领域的快速变化推动了对从业务需求、开发和测试到运营的整个流程的无缝控制的更大需求。现在，发布自动化必须在编排等领域具备额外的智能，并且能够支持云和混合云架构，而用户不必过度关注平台的实际物理方面。组织应该寻找这样的系统，它不仅能满足他们现在的需求，而且供应商已经能够提供系统将如何支持未来不断变化的需求的指示。面对如此快速的变化，如果因为供应商缺乏灵活性而需要从现在选择的解决方案转向新的解决方案，这对于组织来说可能是一场灾难。”–Clive long bottom，Quocirca 分析师兼创始人。

“CA Technologies 将其开发操作系统和应用程序发布自动化工具与 Automic Release Automation (RA)相结合，成为其核心。CA Automic 的业务流程方法是为具有重型发布需求的大型企业设计的，并对新的开发范例(如容器化)提供了出色的支持。”–欧文公司首席分析师迈克尔·阿佐夫。

作为 CA Automic One Automation 平台的一部分，CA Automic Workload Automation v 12.1、CA Automic Release Automation v 12.1 和 CA Automic Service Orchestration v 12.1 可立即正式上市。所有插件都可以在[汽车市场](https://marketplace.automic.com/)上立即获得。非 CA 汽车客户按需定价。
