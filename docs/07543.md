# Deque 研究显示，其自动化测试识别了 57%的数字可访问性问题，超过了公认的行业基准

> 原文：<https://devops.com/deque-study-shows-its-automated-testing-identifies-57-percent-of-digital-accessibility-issues-surpassing-accepted-industry-benchmarks/>

***大量数据集的结果重新定义了自动化“可及性覆盖”的限制***

<time datetime="2021-03-10T16:00:00Z">March 10, 2021 11:00 AM Eastern Standard Time</time>

弗吉尼亚州赫恩登([商业资讯](https://www.businesswire.com/))–数字无障碍领域值得信赖的领导者 Deque Systems 今天宣布了一项大型研究的结果，该研究显示通过其自动化技术识别的数字无障碍问题总数为 57 %,远远高于人们普遍认为的自动化无障碍测试仅提供 20-30%的覆盖率。

**研究简介:** Deque 的研究使用了来自不同行业的 2，000 多项审计的匿名数据，涵盖了 13，000 多页(全部为首次页面评估)，涵盖了近 300，000 个问题。该数据集中的自动化测试是使用 Deque 的 axe 工具套件进行的，该工具套件由开源的 axe-core 规则库提供支持。Deque 发现，平均来说，57%的可访问性问题被自动化测试完全覆盖。

Deque Systems 首席技术官 Dylan Barrell 表示:“当我们将‘可访问性覆盖率’的定义从 WCAG 成功标准的数量转移到问题总量时，自动化测试的真正影响就变得很明显了。“这项研究扩展了自动化的可能性领域，并可以加快无障碍测试专业人员的使命:确保他们的数字资产对所有人都是无障碍的，包括残疾人。”

**重新定义“自动化覆盖”:**自动化测试覆盖的度量通常基于可以使用自动化可访问性工具测试的 WCAG 成功标准的百分比。Deque 的分析表明，这种测量方法淡化了自动化测试的有效性。通过严重性和影响来衡量检测到的问题总量是一个更有意义和更准确的指标。

在实践中，某些类型的问题比其他问题发生得更频繁，这可能导致使用自动化工具发现的总可访问性问题的百分比更高。应该注意的是，axe-core 规则库非常强调不要报告误报或错误问题。
