# Automic 在 2016 年 Gartner 应用发布自动化幻方图中被定位为领导者

> 原文：<https://devops.com/automic-positioned-leader-2016-gartner-magic-quadrant-application-release-automation/>

**VIENNA&SEATTLE–2016 年 8 月 2 日—** [Automic Software](http://automic.com/) ，商业自动化领域的领导者，今天宣布领先的 IT 研究和咨询公司 Gartner，Inc .将 Automic Software 定位为其 2016 年首次应用发布自动化幻方图的领导者。

包括象限图在内的完整报告可在[此处](http://offers.automic.com/gartner-magic-quadrant-for-application-release-automation-solutions-report)获得。

Automic 认为，报告中的排名验证了我们对这个不断发展的市场以及企业客户需求的承诺，因为他们希望采用数字化转型计划来提高敏捷性、速度、可靠性和可扩展性。

根据 Gartner 的报告，“应用程序发布自动化(ARA)工具支持在应用程序生命周期内将与应用程序相关的构件、应用程序、配置甚至数据移动到一起的最佳实践”。在报告中，Gartner 估计市场规模“目前预计到 2020 年复合年增长率约为 20%(CAGR)。”[【1】](#_ftn1)

Automic 首席执行官 Todd DeLaughter 表示:“Gartner 将我们的[发布自动化](http://automic.com/products/application-release-automation)产品评为领导者，我们深感荣幸。“我们认为 Gartner 报告中的内容证明了我们对提供高效且经济的解决方案的重视，我们的客户需要这些解决方案来推动数字化转型时代所需的敏捷性和速度。Automic 通过 Automic Release Automation 为客户提供所需的解决方案，以确保可扩展的[发布自动化](http://automic.com/products/application-release-automation)，以及对所有应用的合规性和控制。”

[Automic Release Automation](http://automic.com/products/application-release-automation) 是市场上最灵活的产品，不受环境影响，这使得它能够在这个数字时代帮助任何企业实现转型。Automic Release Automation 是市场独有的，旨在促进敏捷开发中的快速反馈和持续交付，同时提供生产中所需的安全性、审计跟踪、版本控制和批准。

本报告提供了 ARA 的市场概况，并考虑了供应商工具如何帮助客户实现其目标。据 Gartner 称，“到 2020 年，50%的全球企业将实施至少一个应用发布自动化解决方案，而目前这一比例还不到 10%”。[【2】](#_ftn2)
