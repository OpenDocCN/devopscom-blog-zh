# Datical 宣布支持 Liquibase 工作流，使 Liquibase 用户能够使用企业数据库发布自动化功能

> 原文：<https://devops.com/datical-announces-support-for-liquibase-workflow-making-enterprise-database-release-automation-capabilities-accessible-to-liquibase-users/>

***市场领导者的举动使得数百万人可以随时访问关键的企业级数据库发布功能***

**得克萨斯州奥斯汀——2018 年 11 月 28 日——**[领先的数据库发布自动化解决方案提供商 Datical](https://www.datical.com) 今天宣布推出一项新功能，允许 [Liquibase](https://www.liquibase.org/index.html) 用户更轻松地访问其 [企业级平台](https://www.datical.com/liquibase) 以实现数据库发布自动化。使用 Liquibase 开源项目的数百万软件专业人员现在可以无缝升级到企业级解决方案，为所有数据库部署提供更高的效率、速度和更低的风险，而不必改变他们现有的流程或工作流。

Liquibase 下载量超过 500 万次，是领先的开源解决方案，可帮助应用团队进行版本管理和跟踪数据库变化。虽然 Liquibase 提供的版本控制和跟踪对于小型团队和项目来说已经足够了，但是随着组织的成长和寻找企业级的数据库发布自动化解决方案，Liquibase 还不够。组织有两个选择:1)投入资源来扩展、定制、集成和维护 Liquibase，或 2)转向企业、商业产品，如 Datical。

“无论规模大小，数据库发布流程都是整个企业敏捷性的关键部分。Liquibase 是很好的第一步。然而，随着我们企业需求的增长，我们需要更多。Zions Bancorporation 开发和测试服务高级经理 Russell Webster 表示:“当时，我们面临两个困难的选择，要么将大量时间、金钱和资源从面向客户的应用程序转移到扩展和定制 Liquibase，要么用商业解决方案中断我们的开发工作流程。“现在，我们的团队可以灵活地保留他们习惯于使用 Liquibase 的开发工作流，同时利用 Datical 的企业级数据库发布自动化解决方案，并获得他们所需的商业支持和功能。这是双赢的局面。”

Datical 构建于 Liquibase 之上，是开源项目的主要维护者。Datical 为 Liquibase 带来了重要的增强功能，包括基于管道的工作流、与所有主要应用程序发布自动化解决方案的现成集成，以及强大的规则引擎和更改模拟器，使应用程序团队能够通过软件管道安全地自动化数据库代码，就像应用程序代码一样。

世界上最大的组织依靠 Datical 来高效管理他们的数据库版本。许多客户，包括 Athene、MedImpact 和 Zions Bancorporation，最初都是从 Liquibase 开始的，后来随着需求的变化和团队的扩大，他们转向了 Datical。

Liquibase 开源项目创始人兼 Datical 架构师 Nathan Voxland 表示:“越来越多的应用程序开发人员需要修改数据库代码，他们更倾向于使用对象模型来修改数据库，而不是编写 SQL。“随着最新版本的发布，依赖 Liquibase 提供的 XML 变更集模型的数百万开发人员可以继续有效地使用相同的模型，同时在需求扩展时利用 Datical。”

Liquibase 用户可以通过迁移到 Datical 而受益于许多功能:

**企业级支持**

通过升级到 Datical，Liquibase 用户可以从:一个公共在线论坛到专门的电话和电子邮件支持；阅读在线文档，组建专门的实施和培训团队；从提出一次性的拉取请求或内部派生和维护一个开源项目，到依赖 Datical 作为持续订阅的一部分来获得关键特性和修复。

**企业规模**

随着组织的扩展，他们的需求也在增长。随着团队和环境的增长，使用 Liquibase 手动检查已经部署到给定环境中的内容可能会成为一场噩梦。通过迁移到 Datical，用户可以从自动生成的报告中受益，并可以方便地访问一个集中的 web 界面，该界面可以跟踪在所有环境中部署了哪些数据库更改。另外，Datical 的 web 界面具有丰富的功能，可以搜索和过滤各种参数，因此任何利益相关者都可以快速跟踪或解决数据库部署问题。

**企业自动化**

虽然 Liquibase 可以从开发人员那里收集更改，并将更改应用到数据库以获得下一版本，但它缺乏严格的验证，随着团队和环境的扩展，这可能会导致大量返工和重大部署问题。Datical 拥有强大的规则引擎，支持开发人员自助服务。如果开发人员的变更需要返工或与其他工作相协调，他们会立即得到通知。此外，Datical 模拟了提议的数据库更改的影响，因此部署更加成功，因为任何错误都可以在部署之前被捕获并解决。
