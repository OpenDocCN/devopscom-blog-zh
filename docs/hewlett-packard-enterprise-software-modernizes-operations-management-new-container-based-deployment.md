*April 12, 2017 07:45 ET* | **Source:** Hewlett Packard Enterprise

加利福尼亚州 PALO 阿尔托，2017 年 4 月 12 日(环球新闻网)——惠普企业软件公司今天宣布推出其 HPE IT 运营管理(ITOM)产品的容器化版本。这些套件—[混合云管理](https://www.globenewswire.com/Tracker?data=7GTZ9NfwSD4Onu80Eq-m7TSL2uD7DWOzF6nwgWxwzVdH8U77cdROqDaaXl7TNOASP0lq9Us7Jwl5LUApsvHVewlYmhCdD3dphL8CMZOn7NiVKj5sja4lAM4Mu8H34zXG7Csr2qTYDRS4foI-FJSBHD1N7SCkj-fwB_7I7xzUy-s= "Hybrid Cloud Management")、[数据中心自动化](https://www.globenewswire.com/Tracker?data=YAWBiWKVZO0dGdWxBWxuCenRQmorDbFYt7tIc4YJ6iMO5_5WDtBcO8nvsFaDuoADobPtE6aYPl9IoFjWfvj02j0en-QuQEK1gYEpiwKOvgeJFpdGG1jZXLHGlrgTvBzvIQFGKBe1rD9Eg4ZYrQN71jmiDL3KzwDdygZgETqov_k= "Data Center Automation")、[运营桥](https://www.globenewswire.com/Tracker?data=JGQmPYBysYHvEi8w5uACMF0umM5sKyA-_wFtFO6xZwDnIX5aZlfVS7P6d8HheK8eoFsh40fOpjGaJtx22xqg6rm0vf9Qs7KNuXEXbL22XN-WOnhX-6MOalb_hZyRxdQBUQ6-Inf3gV-3GSvTIGg_4w== "Operations Bridge")和 [IT 服务管理自动化](https://www.globenewswire.com/Tracker?data=EeYj9v5Lm2gAPFxqk9YmEzQEINie_9h5HN0UUmJ-eg0lI3tE26QkMR1PtoX95jNvHdApNEtrhrIfkcoqA8C-CsiDoi4g5BnPbElSSEZ9w0uopDDIrDF99FDDL1Q2FBLjHjwy9KOvTO6rRC7bPqifSX4cdzwNpteKSPRfz5U7X8M= "IT Service Management Automation")—帮助企业 IT 客户管理其私有云和公共云基础架构，并大规模交付新服务，以应对业务合作伙伴和开发运维团队快速变化的需求。部署了 Docker 和 Kubernetes 的内置开源容器技术，这四个套件预计将获得季度增强，为客户实时提供最新的创新，以大幅缩短价值实现时间。

套件中内置的新容器部署基础为 IT 团队提供了行业领先的部署、更新和横向扩展功能。客户将能够在几分钟内，在各种平台(包括裸机、虚拟化、私有云和公共云基础架构)上快速部署和升级到最新的分析驱动型 ITOM 套件。因此，客户可以最大限度地降低升级风险和停机时间，同时通过按需扩展最大限度地利用 IT 资源。

*“IT 运营领域正经历着前所未有的增长和复杂性，”* *说* *Tom Goguen，惠普企业副总裁兼 IT 运营管理总经理。“IT 组织需要能够快速响应业务需求并交付成果，同时管理法规和运营合规性。在过去的一年中，我们通过从单点产品转向集成解决方案来改进我们的产品组合，并按季度提供新的创新功能，如分析和 ChatOps。我们新的集装箱化套件简化了日常操作，实现了前所未有的价值，使 ITOps 能够以 DevOps 的速度运行。”*

**借助 HPE 的集装箱生命周期管理，IT 运营以 DevOps 的速度发展**

除了将容器技术构建到新产品的 DNA 中，HPE IT 运营管理套件还增加了许多功能，以帮助组织使用相同的熟悉工具大规模部署和管理基于容器的应用程序。

ITOM 的四个套件和关键的集装箱生命周期能力包括:

*   **混合云管理**—利用应用程序管道自动化和合规性功能管理各种混合云和多云环境。新版本支持新的基于容器的应用程序的服务设计和部署，包括现成的开发测试环境和快速应用程序发布。
*   **数据中心自动化**–自动化任务并协调配置、修补和合规性功能的流程。新版本为容器、硬件和主机操作系统提供了合规性功能。
*   **运营桥梁**–与现有工具集成，提供自动化事件减少、根本原因识别和业务服务建模。新版本通过对集装箱基础设施和应用程序进行监控和故障排除，提高了运营团队的速度和响应能力。
*   **IT 服务管理自动化**–简化 IT 运营的订购、管理和维护。新版本提供分析驱动的服务管理，以及自动化发现和配置管理，以支持基于容器的应用程序。

沃达丰集团服务部高级工程经理 Luke Bradley 表示:“借助新的微服务和集装箱化架构，HPE 运营桥正朝着正确的方向前进。“凭借更简单的部署、更快的版本控制、简化的补丁安装以及更高的可扩展性，这是一次激动人心的变革，将为我们带来真正的价值。”

GreenLight 首席顾问 Torrey Jones 表示:“新的 IT 运营部署通常是一个广泛而复杂的过程，可能需要几个月的时间来管理和完成。“HPE 的 IT 运营软件交付新方法通过将其套件容器化，极大地简化了设置，使我们能够专注于加快客户的价值实现。”
