# DevOps 聊天:与 ForgeRock 的 Peter Barker 进行身份管理

> 原文：<https://devops.com/devops-chat-identity-management-forgerocks-peter-barker/>

2019 年及以后的两个大趋势是规模化和自动化。身份管理空间就是一个完美的例子。考虑到即将上线的所有支持 IP 的设备(机器)以及数十亿人和他们的身份，管理这些数十亿人需要重新定义规模的系统，而且根据定义，几乎必须自动化。

进入锻石。该公司是消费品牌、物联网设备等身份管理领域的领导者。在本次 DevOps 聊天中，我们与首席产品官 Peter Barker 讨论了在这一级别进行扩展所面临的挑战以及身份管理的重要性。
