IT Revolution 创始人、《凤凰计划》(The Phoenix Project)和《DevOps 手册》(The DevOps Handbook)的合著者吉恩·金(Gene Kim)表示:“我们非常兴奋能够回到伦敦举办第二届欧洲 DevOps 企业峰会。”“从各方面来看，去年的活动都非常成功，今年会更好。我们还安排了为期两天的精彩演讲、经验报告、主题专家和网络论坛，以便与会者能够获得他们需要的见解和实用信息。您将听到大型复杂组织中的同行们分享他们的 DevOps 计划带来的改进和转型的具体例子。

DevOps 企业峰会是面向在全球范围内实施 DevOps 原则和实践的大型复杂组织的领导者的顶级行业会议。该活动展示了领先的全球品牌，这些品牌利用 DevOps 在日益由软件驱动的经济中蓬勃发展。DevOps Enterprise 提供主题演讲、小组讨论、分组讨论、研讨会和展厅。会议由在各行各业的大型复杂组织中率先实践这些实践的领导者领导。主题演讲包括行业名人，以及来自知名公司的演讲者，他们分享了从企业 DevOps 计划中获得的经验。

**关于 IT 革命**

IT 革命通过出版、活动和研究聚集了技术领导者和实践者。我们的目标是提升技术工作的状态，量化与次优 IT 性能相关的经济和人力成本，并改善 IT 专业人员的生活。
