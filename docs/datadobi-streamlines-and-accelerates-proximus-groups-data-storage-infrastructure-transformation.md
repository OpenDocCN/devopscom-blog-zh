# Datadobi 简化并加速 Proximus 集团的数据存储基础设施转型

> 原文：<https://devops.com/datadobi-streamlines-and-accelerates-proximus-groups-data-storage-infrastructure-transformation/>

电话、互联网、电视和基于网络的 ICT 服务的主要提供商部署 DobiMigrate，将从主目录到业务运营和客户关系数据的所有内容都移动到戴尔 EMC Isilon 上

比利时鲁汶，2021 年 5 月 4 日—[全球非结构化数据管理软件的领导者 Datadobi](http://www.datadobi.com) 今天宣布，比利时电话、互联网、电视和基于网络的 ICT 服务的主要提供商 [Proximus Group](https://www.proximus.com/) 已经部署 [DobiMigrate](https://datadobi.com/migrate/) ，以加速并帮助确保其数据存储基础设施转型的成功。

为了巩固其作为领先数字服务提供商的地位并保持卓越的客户体验，Proximus 最近启动了其 n as 数据存储基础架构的更新计划。它认识到，为了帮助客户获得数字化转型的好处，它也需要转型。为了避免通常会延迟和干扰数据迁移的障碍，并顺利过渡到新系统，Proximus 向其存储供应商戴尔 EMC 寻求指导。戴尔 EMC 建议他们评估 DobiMigrate 企业级软件，并测试其在内部或云中的任何存储平台之间安全、快速、轻松且经济高效地迁移文件或对象数据的能力。

“去年年底，我们安装了 DobiMigrate 来测试它在新旧存储平台之间迁移和同步的能力。我们还想看看它是否能够支持主动-主动存储环境，因为我们知道在迁移过程中，我们希望两个控制器都可用，以保持关键运营和业务服务的连续性，”Proximus 团队经理 Pascal De Cock 说。“Datadobi 超出了我们的预期。特别是，我们对其 100%的复制准确性印象深刻，这对我们的高可用性(HA)和灾难恢复(DR)非常重要。DobiMigrate 处理权限和访问控制列表的方式也与其他解决方案截然不同，这是我们管理访问和认证的一大优势。当然，它在移动客户数据时几乎消除任何损坏发生的可能性的能力也很关键。”

De Cock 总结道:“现在正式选择了 DobiMigrate，我们正在利用它将生产和非生产数据(从所有主目录到与业务运营和客户关系管理相关的所有数据)移动到我们新的戴尔 EMC Isilon 存储上。”

“Proximus 将客户放在一切工作的中心。这就是他们继续投资变革性技术的原因，因为这使他们能够为客户提供一个充满新可能性的世界，”data dobi EMEA 地区区域经理 Matthias Nijs 说道。“有了 Datadobi，Proximus 将能够加快向新数据管理环境的迁移，实现更快的投资回报和更低的总拥有成本。然而，最重要的是，他们将能够告别耗时的迁移任务，转而专注于其核心竞争力——提供适合每位客户需求的产品和服务，以便他们能够更好地生活和更智能地工作。”
