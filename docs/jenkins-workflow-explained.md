# 网上研讨会:詹金斯工作流程讲解

> 原文：<https://devops.com/jenkins-workflow-explained/>

各地的团队都在寻找工具来定义他们将代码投入生产的过程，并将其表示为代码。

了解如何将 Jenkins 与 Docker 配合使用，以便:

将平台扩展到多个团队和项目
–构建 docker 映像
–测试并发布 docker 映像
–跟踪映像直至容器使用
–从发布到 Docker 注册表的更改中触发构建
–使用 Jenkins 工作流构建一个管道，将上述内容整合在一起

在本次讲座中，我们将展示如何使用新的 Jenkins Docker 插件，并为您提供可在您自己的环境中进一步探索的配置。
