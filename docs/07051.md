# 敏捷和 DevOps 时代的 ITIL 4，有 Axelos

> 原文：<https://devops.com/itil-4-in-the-age-of-agile-and-devops-with-axelos/>

ITIL 是全球众多 IT 组织服务管理的流程、认证和信息资源库。随着最近对 ITIL 4 的更新，来自敏捷和开发运维的影响正在帮助 ITIL 扩展和适应创建软件和构建交付流程的方式的变化。

在本次 [DevOps 聊天](https://devops.com/category/devops-chat/)中，AXELOS 全球最佳实践产品大使阿克谢·阿南德和 BMC Software 首席产品经理乔恩·斯蒂文斯-霍尔与加速战略集团首席执行官米奇·阿什利一起讨论了 [ITIL 4](https://wiki.en.it-processmaps.com/index.php/ITIL_4) 中的新创新。作为 ITIL 4 的主要贡献者，阿克谢和乔恩分享了他们在 ITIL 4 中采用的方法，融合了敏捷和开发团队的想法。
