# DBmaestro 扩展其新的数据库自动化平台以支持 Microsoft SQL

> 原文:[https://devo PS . com/db maestro-expands-its-new-database-automation-platform-to-support-Microsoft-SQL/](https://devops.com/dbmaestro-expands-its-new-database-automation-platform-to-support-microsoft-sql/)

*调查显示，数据库可能成为持续集成的拦路虎*

**马萨诸塞州波士顿，2018 年 5 月 16 日**–[领先的数据库开发运维解决方案提供商 DBmaestro](https://www.dbmaestro.com/) 今天宣布，该公司已扩展其数据库自动化平台，通过版本控制和微软 SQL 的发布自动化支持 CI/CD(持续集成/持续开发)。现在，DBmaestro 的数据库自动化平台加快了发布速度，提高了生产率，最大限度地减少了错误，并提高了 SQL 和 ORACLE 数据库的安全性。

DBmaestro 的数据库自动化平台旨在快速、轻松、自信地更改数据库，从而成功实现端到端 CI/CD。世界各地的许多企业都在努力将他们的数据库引入他们的敏捷工作流程。最近对近 250 家公司进行的一项[全球 DBmaestro 调查](https://www3.dbmaestro.com/devops-for-database-survey-2018-results)显示，48%的受访者在过去 3 个月中遇到了严重的数据库问题(崩溃、停机等)，47%的受访者在大多数时间里害怕推动数据库更改。

DBmaestro 首席技术官 Yaniv Yehuda 表示:“许多企业投资 DevOps 工具进行应用程序开发和测试，却发现数据库不够灵活，阻碍了他们的发展。“DBmaestro 是一个 DevOps 平台，它为数据库提供了灵活性和安全性，使 IT 能够更快、更频繁地简化发布，同时管理风险。DBmaestro 帮助数据库跟上业务变化的步伐。”

主要平台特性包括:

*   角色管理和访问控制可增强安全性和治理
*   在数据库中实施可定制的组织政策，并对所有活动进行全面审计跟踪
*   配置漂移检测和预防
*   强大的强制签入/签出版本控制系统

DBmaestro 本月将与 SQL Server 专业人士的全球中心 [MSSQLTips](https://www.mssqltips.com/) 联合举办一场名为 ***SQL 持续集成恐怖故事*** 的网络研讨会。该网络研讨会将包括交付更加一致和无错误的 SQL 软件部署的技巧，以及从现实生活中的数据库恐怖故事中学到的一些经验教训。网络研讨会将于 5 月 24 日世界协调时下午 7:00 |美国东部时间下午 3:00 现场举行。[点击此处](https://www.mssqltips.com/sql-server-webcast-signup/?id=702&src=WebcastList)了解更多信息。

**关于 DBmaestro**

DBmaestro 为数据库带来了 DevOps 最佳实践，为数据库提供了新水平的效率、速度、安全性和流程集成。DBmaestro 的平台使组织能够安全有序地运行数据库部署，提高开发团队的工作效率，并显著缩短上市时间。该解决方案使组织能够针对数据库活动实施 CI/CD 实践，并具有可重复的管道发布自动化和自动漂移预防机制。该平台结合了数据库的几个关键特性，包括:管道发布自动化、数据库版本控制、治理和安全模块以及业务活动监视器。欲了解更多信息，请访问[www.dbmaestro.com](http://www.dbmaestro.com)。
