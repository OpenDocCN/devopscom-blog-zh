# 随着 Datical 获得 1000 万美元的 C 轮融资，数据库发布自动化市场继续增长

> 原文:[https://devo PS . com/database-release-automation-market-growth-continues-as-datical-secured-1000 万-in-series-c-funding/](https://devops.com/database-release-automation-market-growth-continues-as-datical-secures-10-million-in-series-c-funding/)

*来自 River Cities Capital Funds、S3 风险投资公司和 Mercury Fund 的投资凸显了对数据库发布自动化日益增长的需求*

**得克萨斯州奥斯汀——2019 年 2 月 28 日——**[领先的数据库发布自动化解决方案提供商 Datical](http://www.datical.com) 今天宣布获得由新投资者 River Cities Capital Funds 领投的 1000 万美元 C 轮融资，现有投资者 S3 风险投资公司和水星基金也参与其中。此外，River Cities 的副总裁 Adam Midkiff 也加入了 Datical 的董事会。

该公司将利用这笔资金通过扩大其研发、客户成功能力以及销售和营销职能来维持其巨大的增长。迄今为止，最新一轮投资使该公司的总投资额达到 2700 万美元。

“我们是市场领导者，在过去的三年里取得了巨大的增长。这笔资金为进一步的创新和扩张提供了动力，因为 Datical 的非凡势头已经证明，数据库发布流程的现代化是任何数字化转型之旅的必要步骤，”Datical 首席执行官 Derek Hutson 表示。“我们很荣幸有 River Cities 加入，因为我们进入了下一个发展阶段，开始了软件开发转型的激动人心的新篇章。”

随着企业继续采用敏捷、DevOps 和持续交付实践来更快地将应用程序推向市场，部署数据库更改的手动过程为开发团队带来了瓶颈。Datical 使企业能够通过自动化数据库发布来跟上快速加速的应用程序发布周期，同时消除导致应用程序停机和数据安全漏洞的风险。

River Cities 的副总裁 Adam Midkiff 说:“Datical 正在解决开发团队今天面临的一个关键但经常被忽视的问题。“该公司在改变组织通过软件改善客户体验的方式方面具有独特的定位，我们很高兴与这样一个创新团队合作实现这一使命。”

Datical [结束了 2018 年](https://www.datical.com/press-release/datical-concludes-2018-with-94-percent-increase-annual-subscription-revenue/) 年订阅收入增长 94%，客户群同比增长 30%，包括 MedImpact、Freddie Mac、Anthem、Colonial Life 和 Zions Bancorporation 等企业。此外，2018 年，该公司搬迁了其奥斯汀总部，以适应积极的员工增长轨迹。
