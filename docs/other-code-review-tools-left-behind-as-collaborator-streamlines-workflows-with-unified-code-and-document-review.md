具有过程意识的开发团队对可能将缺陷引入他们的代码库、需求、设计文档、测试用例以及用户故事的变更进行同行评审。目前，大多数团队的那些评审分散在电子邮件线程、聊天工具、文档评审工具和代码评审工具中——这降低了生产率，并给评审过程带来了风险。Collaborator 是唯一的同行评审工具，具有推动有意义的流程改进所必需的标准化功能，例如自定义模板、清单和详细报告，所有这些都在一个中央同行评审平台中。随着 v11.3 的发布，Collaborator 将 PowerPoint 和 Visio 添加到其本机 Microsoft 集成列表中，加入了 Word、Excel、Visual Studio 和 Team Foundation Server。
