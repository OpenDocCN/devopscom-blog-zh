# AsyncAPI:作为一个值的透明度

> 原文：<https://devops.com/asyncapi-transparency-as-a-value/>

历史的特点是不透明和极权的权力体系。在监狱和遮挡光线的墙的帮助下。权力集中在少数人手中标志着人类的进化。从封建主义到法西斯主义，个人权利的取消都是基于蒙昧主义和无知。信息总是由少数人在少数几个地方锁起来保护着。黑暗总是被理解为知识和科学的对立面。柏拉图提到了洞穴，从外面几乎看不到影子，影子是扭曲的，可以操纵的。受制于暗箱的不透明，我们生活在打开第一个灯泡的快门的希望中。

### 对抗一维

面对控制和审查的危险，面对单一方向的危险，面对马尔库塞最终称之为“单向度的人”的危险，20 世纪末诞生的新工具打破了权力赖以存在的传统基础。互联网的发展为世界打开了一扇窗户，增加了道路的多样性。光以一种看似无限的信息洪流的形式进入每个家庭，无法审查，深不可测。

社会 2.0 的到来标志着政府系统走向必要民主化的决定性转折点。在其前提下，没有什么可以或应该被隐藏；一切都是潜在可见的。此外，不再有观看表演的观众，但用户似乎准备在集体打赌中加入他们的沙粒，打开他们的快门，让更多的光进来。

### 聚光灯

用户的参与和激活是集体智慧的力量。用户上传的图片越来越多，使得操作变得更加困难，扩大了对比观点和意见的选择范围。他们给一个夸大的故事添加了几行文字。

用户因此成为避免盲点的聚光灯。为了公众的利益再多一个手电筒。对于投影到墙上的透明图像，例如幻灯片，我们需要一个指向醋酸盐的光源，从而使我们能够看到图像的复杂性，直到那一刻为止，图像一直处于黑暗中。

### 公开

近年来，透明度可能已经成为政府、机构和公司民主化进程的基本概念。最终，在理想状态下，民主只不过是一种允许社区成员从不同角度分析现实以最终做出决策的方法。

由此产生了开源政策，一个让信息免费，同时允许用户看到其管理和决策过程，并邀请每个人合作的系统。社区因此成为工作和交流的重要引擎。

### 光线

AsyncAPI 就是在这个基础上产生的。作为一个开放的政府项目，它捍卫其治理和组织系统的透明度和民主化。这种组织系统旨在基于决策的水平结构，授权给组成项目的人员。

所有相关的项目数据都是公开的。正在制定的任务是社区可以完成的，并开放给社区参与。这是为了培养一种信任和尊重用户的环境，激励他们参与到共同目标的追求中。这使得项目越来越具有协作性和参与性。数据和元数据。代码和图像。正如数码摄影所做的那样，它不再可能只关注最终状态，关注被展示、编辑、合成和处理的图像的表面。图像变成了一个开放、透明的过程，我们可以比较它的属性和内部结构。

最后，光。
