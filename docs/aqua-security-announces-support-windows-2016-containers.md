# Aqua Security 宣布支持 Windows 2016 容器

> 原文：<https://devops.com/aqua-security-announces-support-windows-2016-containers/>

*与微软 Visual Studio Team Services 集成，实现容器映像漏洞扫描自动化*

**以色列特拉维夫——2016 年 10 月 27 日**，领先的虚拟容器应用安全平台提供商 Aqua Security 今天宣布支持 Windows Server 2016 容器以及与微软 Visual Studio Team Services (VSTS)的集成。Aqua 的容器安全平台的这些最新添加使其成为第一个真正的多操作系统容器安全解决方案。

Aqua Security 首席技术官兼联合创始人 Amir Jerbi 表示:“Aqua 对 Windows Server 2016 的支持以及与 VSTS 的集成，使我们的客户能够原生开发和运行 Windows 容器，以部署内在安全的应用程序，从而以云的速度发展他们的业务。“随着微软用户开始大规模采用容器，我们将继续提供原生跨平台支持。”

Amir Jerbi 上个月在微软 Ignite 的一次[演讲中预演了 Aqua Container Security Platform(CSP ),现在它为在 Windows 上本地运行的容器提供了与 Linux 容器相同的自动化安全控制。主要功能包括:](https://lookingglasspr-dot-yamm-track.appspot.com/Redirect?ukey=1C54CF8tVNnhZOrKyl9RJwvQJgo44wg5ASL-zbgCi1sc-987368591&key=YAMMID-62656507&link=https%3A%2F%2Fwww.youtube.com%2Fwatch%3Fv%3D4BJqSN49Ye0%26t%3D35m55s)

*   防止运行易受攻击的映像并在整个生命周期中跟踪映像使用情况的映像保证。
*   利用多层防御的运行时安全性，从最低权限默认值开始，然后使用行为分析将良好行为列入白名单并防止异常行为，从而增强运行时安全性。
*   针对各种攻击媒介提供保护的网络情报
*   防止用户未经授权访问的精细访问控制

Aqua 今天还宣布，它的图像漏洞扫描现在可以在 Visual Studio Marketplace 中使用，并与 Visual Studio Team Services 集成。VSTS 用户可以在构建过程中使用 Aqua 来扫描容器图像中的漏洞。用户可以创建策略来确定如何对扫描结果采取行动，例如，他们可以为构建失败的漏洞分数设置阈值。通过这种集成，VSTS 用户可以轻松地将安全性自动化到他们的映像构建流程中。

Aqua 的 VSTS 图像扫描仪在 https://marketplace.visualstudio.com/items?有售 itemName=aquasec.aquasec )。

Aqua 的安全平台现在为 Windows 容器提供了运行时安全性，并通过 VSTS 提供了安全的开发流程，这使得依赖微软 Windows 基础设施的企业能够加快采用容器技术，同时提高应用程序的安全性和合规性。

**关于 Aqua Security**
