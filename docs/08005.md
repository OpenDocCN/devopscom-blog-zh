# 它如何利用人工智能为不确定性做准备

> 原文:[https://devo PS . com/how-it-can-use-ai-to-prepare-for-uncertainty/](https://devops.com/how-it-can-use-ai-to-prepare-for-uncertainty/)

对于企业来说，过去的一年是不可预测的——从迁移到远程工作，到计划重返办公室，再到跟上不断变化的政府指导方针。随着[如此多的变化](https://www.wsj.com/articles/if-you-thought-working-from-home-was-messy-here-comes-hybrid-work-11621935000)，这是一个很好的提醒，组织必须为意外做好准备，并有应急计划到位。

从 IT 的角度来看，这意味着无论未来一年的工作看起来如何，都要确保员工能够高效、安全地工作。幸运的是，[人工智能](https://devops.com/?s=AI) (AI)技术正在快速发展，使企业能够更好地管理不断变化的混合劳动力。

以下是组织应该实施的三个人工智能战略，以帮助为不确定时期做好准备:

### 投资自动驾驶网络

去年，当公司出人意料地迅速将员工转移到远程工作时，网络在多个方面面临着越来越大的压力:使用、质量和问题补救。除了这些因素之外，在过去几年中，随着工作负载越来越多地分布在公共云和私有云之间，网络变得越来越复杂。即使在最好的情况下，构建和维护传统网络也需要一个专门的专家团队，而且可能非常耗时。由于其复杂性和不断增加的数据量，网络变得越来越难以通过命令行界面(CLI)进行管理，并且需要适应不断变化的工作负载和员工连接性——这就是人工智能发挥作用的地方。

自动驾驶网络通过使用人工智能和自动化来运行网络的端到端用户体验，以最少的人工干预来降低复杂性。这包括监控网络中的问题，如坏电缆和容量热点，并在它们影响最终用户之前解决它们。通过确保网络在基础层面上平稳运行，自动驾驶网络允许 IT 团队将宝贵的时间用于解决更复杂的问题和推动创新。

### 用人工智能助手为 IT 赋能

另一种释放 IT 团队时间的方法是招聘新员工——不是传统员工，而是人工智能助理。更具体地说，一个可以回答问题和管理网络的人工智能助理可以与网络领域专家相提并论——一个可以在工作中学习和提高的人工智能助理。随着人工智能真正交付与人类同等的完成功能，这就是人工智能和机器学习正在颠覆行业的地方。

今天的人工智能助手通过对话界面，不仅仅是对 IT 问题给出简单的“是”和“否”的答案，它们还能够回答复杂的开放式问题，例如，“为什么 Bob 的缩放体验很差？”他们甚至在学习解决更复杂的问题，帮助 IT 团队最大限度地减少沮丧的远程员工发出的支持请求。随着企业认识到人工智能助理给他们的团队带来的价值，他们现在将他们视为 IT 团队的成员，帮助他们简化日益复杂的 IT 网络。

### 让员工了解人工智能的好处

2021 年，我们预计人工智能在网络中的应用将会加速。确保正确的人工智能技术得到正确执行始于信任。因此，让团队更多地将人工智能助手视为队友，而不是威胁工作的技术，是成功实现人工智能的重要一步。终端用户体验比以往任何时候都更加重要，人工智能对于企业管理这种体验变得越来越重要。

对许多员工来说，人工智能的概念可能很可怕。一个常见的误解是，AI 会接管人类的工作。然而，人们开始通过看到 AI 解决实际问题来理解它的重要性。当公司将人工智能技术实施到他们的解决方案中时，他们应该首先优先考虑关于人工智能商业优势的公开对话。通过确保每个人都在同一页上，团队可以一起努力建立一个更好的过程。毕竟，目标是让人工智能最终让他们的工作变得更容易。

过去的一年是艰难的一年，但它提醒我们未雨绸缪的重要性——这是继续高效运营企业和远程支持员工所必需的。尽管未来肯定会有更多不确定的时期，但我们有工具尽可能在今天做好准备。

通过采取措施建立自动驾驶网络，投资人工智能助手并对员工进行人工智能教育，组织可以安心地为自己、员工和利益相关者建设一个稳定的未来，为他们面临的任何不确定性做好准备。
