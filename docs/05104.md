# DevOps 聊天:Automic + CA + Broadcom，发布自动化之旅

> 原文:[https://devo PS . com/devo PS-chats-automic-ca-Broadcom-a-release-automation-journey/](https://devops.com/devops-chats-automic-ca-broadcom-a-release-automation-journey/)

被收购公司的生活充满了意想不到的曲折和变化，这对 Automic software 来说也是如此。斯科特·王绍博是播客的常客，也是 DevOps 社区的长期成员，他再次加入了我们的 DevOps 聊天。他将自己的收购之旅描述为类似于“被大白鲨吃掉的梭鱼，被鲸鱼吃掉的梭鱼。”Scott 是 Broadcom 公司 CA Technologies(Automic+CA+Broadcom 的新专有名称)的产品营销总监。

Scott 分享了作为 Broadcom 内部的一家软件开发公司的感受，该公司的强大根基来自芯片制造业务。当然，尽管软件与芯片有许多不同之处，但 Scott 分享了 DevOps 的原则和精益制造的遗产是如何让这种转变变得更加容易的。这对我们所有人都是有价值的教训。

在所有这些变化中，Automic 将其产品转向免费增值模式。automic Continuous Delivery Director 现在可以免费用于多达 10 个活动版本。我们谈论即将到来的发布，其中包括机器学习。加入我们，让我们一起穿梭于收购故事和连续自动化交付的世界。
