# DigiCert 推出 CT 日志监控，现已在 Secure Site Pro 中提供

> 原文:[https://devo PS . com/digicert-launchs-CT-log-monitoring-now-available-in-secure-site-pro/](https://devops.com/digicert-launches-ct-log-monitoring-now-available-in-secure-site-pro/)

*通过访问 DigiCert CertCentral 以简化证书管理，Secure Site Pro 提供了高级品牌和网站威胁防护，以及新的 PQC 测试套件*

**犹他州 LEHI(2019 年 12 月 10 日)——**[全球领先的 TLS/SSL、物联网和 PKI 解决方案提供商 DigiCert，Inc.](https://www.globenewswire.com/Tracker?data=YGypWQzHjycEQU_RyphASsU5WkvGhIdAGFrRdlcV_6G2zRTyjuM6YDM_iVgWKNNYCfKlLQxaFOeHSnlAhB1DLw==) 今天发布了一项全新的 CT 日志监控服务，用于高级品牌保护。该服务在 DigiCert Secure Site Pro 中提供，这是第一款提供 ct 日志监控以及其他行业领先功能的 TLS 产品，如 DigiCert 最近推出的[后量子计算(PQC)测试套件](https://www.globenewswire.com/Tracker?data=NiFXHlUcQ011HDa5nUoIECqye2mDv6nJgXvO1MWvc7yGjupUsv1MGPsdiOx-KCU4ASZKRGjWK4dnBqzortFu0gdaHDEIauTnyBe98YIDGasVS3ZDT9MCynUyXWWsW4HAGDwHWzukk1KMB0OYjJYzlm_EXNsuaWTmJVHkUXl1UkIVEkT-CwIcyaRir2Ie0lkI)。

Secure Site Pro 的 CT 日志监控服务将帮助公司保护其品牌免受在线欺诈。DigiCert 杰雷米·罗利产品总监表示，为了保护政府机构和大型组织，通过 CT 日志监控来记录和监控 TLS 证书的颁发正迅速成为强制性的。

Rowley 说:“没有任何其他认证机构产品包括对 ct 日志监控服务的默认访问，以实现快速、轻松的可见性。”“DigiCert Secure Site Pro 正在提供一项基于 API 云的服务，该服务可持续监控 CT 日志，升级公司批准帐户之外颁发的证书，并通知他们可疑的证书颁发情况。”

CT 日志监控帮助企业

*   通过针对多个 CT 日志的全局监控和跟踪，获得每个域的可见性，
*   通过对不符合要求的证书进行自动检查，节省时间和精力，以及
*   确保每个域都是可信的，并完全监督每个证书由哪个机构颁发。

所有 Secure Site Pro 证书均可访问屡获殊荣的 DigiCert 证书中心管理平台，该平台具有 ACME、SCEP、EST 和 GraphQL 协议等自动化解决方案，可处理证书颁发、安装、发现、续订、撤销、补救和其他任务。CertCentral 还支持多个编排层。

安全站点专业证书的其他独特的高级功能包括优先支持和验证、200 万美元的保修以及世界上最受认可的信任标志:诺顿印章和 DigiCert 印章。

Rowley 说:“对于 IT 团队来说，管理和维护多个网站安全解决方案非常耗时、复杂且昂贵。“DigiCert Secure Site Pro 是一款一体化网站安全解决方案，可为公司节省时间和精力。Secure Site Pro 是保护重要网站的行业最佳 TLS 解决方案，在这些网站中，完全的加密意识是必要的。”

此外，DigiCert 对后量子加密技术进行了广泛的测试，并于最近为客户发布了 PQC 测试套件。购买 Secure Site Pro 证书的组织可以使用该套件测试其系统的 PQC 兼容性，并为其长期应用提供未来保障。客户可以创建他们自己的私有混合证书，该证书与经典加密算法一起实现后量子加密，以保持向后兼容性。

罗利说:“量子计算对今天的加密协议的威胁可能需要几年时间，但公司需要现在就制定计划，特别是针对今天构建的、预计在 10 年或更短时间内安全的物联网设备和应用。”“我们与技术合作伙伴的合作让这一切成为可能。”
