# DevOps 聊天:与雨林的 Derek Choy 的众包持续测试

> 原文：<https://devops.com/devops-chat-crowdsourced-continuous-testing-with-rainforests-derek-choy/>

扩展测试是许多软件团队的灾难。虽然测试自动化和连续测试已经取得了长足的进步，但是仍然很难自动化每一个 QA 测试。此外，很难找到能够编写这些测试的工程资源。进入雨林。

雨林有一个独特的持续测试众包解决方案。它将众包人才与自动化相结合，持续提供高水平的质量保证。

在本次 DevOps 聊天中，我们采访了首席信息官 Derek Choy，他向我们解释了为什么雨林在某些方面可以成为 QA 的优步。像往常一样，下面是流媒体音频，然后是我们的谈话记录。
