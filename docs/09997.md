## *2.说到这个…*

*不和谐刚刚被罚款 80 万€打破 GDPR。法国监管机构 CNIL 率先从网络电话/聊天设备中提取现金。*

### *分析:从别人的错误中学习*

*给出的理由是你的应用程序可能犯的错误。看看 GDPR 现在是如何被解读的**。***

***[Marie-Laure Denis](https://www.cnil.fr/en/discord-inc-fined-800-000-euros "read the full text") *及其朋友:***disagree Inc .罚款 80 万欧元*****

***[我们]认为该公司未能遵守《一般数据保护条例》(GDPR)规定的几项义务:
……
**未能定义和遵守数据保留期:**……CNIL 证实，DISCORD 数据库中有 2，474，000 个法国用户账户已超过三年未使用。…
**未能确保数据保护:** …当登录到语音室的用户通过点击微软 Windows 窗口右上角的“X”图标关闭 DISCORD 应用程序窗口时，他们实际上只是将该应用程序放在后台，并保持登录到语音室[这]可能导致用户在他们认为他们已经离开时被语音室中的其他成员听到。……
**未能确保个人资料的安全:**……创建账户时……接受了包括字母和数字在内的六个字符的密码。[我们]认为，DISCORD 的密码管理政策不够有力和严格，不足以确保用户账户的安全。…
**未能进行数据保护影响评估:** …鉴于公司处理的数据量和未成年人使用其服务的情况，受限委员会认为公司本应进行此类影响评估。***

***那是很大的损失。 [femboy](https://news.ycombinator.com/item?id=33638553 "read the full text") 欢呼:***

***支持欧盟让数据巨头负起责任。令人遗憾的是，数据保护当局只有资源来处理这么多公司，不幸的是，许多公司在对用户隐私造成更多伤害的情况下逃脱了惩罚。…虽然不是直接的安全违规调查，但这些调查对用户数据有很大影响。***

***“你就是产品。”[u/马克 _ 福克斯](https://www.reddit.com/r/privacy/comments/yxob5a/comment/iwqe00u/ "read the full text")同意:***

***百分百正确。底线是，这些产品和服务是由大公司制造和提供的。他们最关心的不是你的隐私，甚至不是你的用户体验。…他们最关心的是美元。***

## ***3.亚马逊网络服务解雇表现不佳的员工***

***亚马逊的长期招聘冻结已经影响到了 AWS。有消息称，经理们被“鼓励”解雇任何不尽职的员工。***

### ***分析:这是新闻？***

***虽然我不喜欢按曲线评分或自动失去垫底的同事，但如果表现不佳的员工无法改进，为什么*不会*让他们走呢？尤其是在亚马逊这样的文化中。无论 T2 的经济环境如何，这都是事实。***

***凯莉·罗宾森 : **网络服务业务延长招聘冻结*****

***[AWS]正在要求经理们淘汰发展过快的团队中表现不佳的员工，并将把整个组织的招聘冻结延长至 2023 年第一季度。…亚马逊的云业务在第三季度创下有史以来最慢的增长，并且没有达到分析师的预期。
…
一些 AWS 团队…目前超出了他们的人员目标。…此类团队的经理应在 Q1 会议结束前“解决”这种情况…通过绩效改进计划、自然减员和其他方式管理表现不佳的员工。
……
鉴于严峻的经济背景，一些 AWS 内部人士已经做好了更糟的准备。…云业务对底线的重要性可能会为 AWS 员工提供一些保护，使其免受亚马逊正在进行的更大规模成本削减的影响。***

***“管理表现不佳的员工。”这个*真的与亚马逊的正常运营方式*不同吗？ [vaidhy](https://news.ycombinator.com/item?id=33624692 "read the full text") 称之为“压力和无情”:***

***你可以被要求挖洞，然后再把它们合上[即]完全无用的工作。你可以得到越来越多的配额去做这件事，你挖的一个洞可能会有你的身体在里面的威胁。亚马逊之外的大多数人没有意识到的挑战是微服务架构对团队协作造成了严重的影响，尤其是。随着软件的成熟。很大一部分核心服务已经构建了几年(甚至几十年),它们的架构是以牺牲其他操作为代价来优化某些操作的。您可以将此想象为如何设计一个 DB 模式。这是写速度、查询速度和可用性之间的权衡。现在，想象它大规模发生。作为一名工程师，在交付的压力下，你会花很多时间在合作伙伴团队上工作/等待，以确定你的变更的优先级。许多团队最终将这些部分构建到他们自己的服务中，导致了更大的维护和操作负载。……保持 KTLO 的正常运转是我在那里时制定的每个年度计划中的一大项。***

***“绩效改进计划、损耗和其他手段”有一种奇妙的奥威尔式的氛围。 [King_TJ](https://tech.slashdot.org/comments.pl?sid=22401430&cid=63056440 "read the full text") 解释道:***

***通常发生的情况是，有人达不到一些新的或升级的期望(因为他们不断改变他们对人们期望的目标)。…一旦你被[绩效改进计划]击中，你就等于被解雇了。在那一点上，你在显微镜下。…我在那里认识的大多数人都乐于辞职，去找一个压力更小、更通情达理的雇主，而不是在老板显然认为他们是可牺牲者之一的情况下努力留在那里。***
