# 云计算基金会欢迎比特币基地成为黄金会员

> 原文：<https://devops.com/cloud-native-computing-foundation-welcomes-coinbase-as-gold-member/>

*密码经济中的创新者申明对云原生生态系统的承诺*

**西班牙瓦伦西亚，KubeCon + CloudNativeCon 欧洲——2022 年 5 月 18 日——**[为云原生软件构建可持续生态系统的云原生计算基金会](https://www.cncf.io/) (CNCF)，今天宣布比特币基地作为黄金会员加入该基金会。

[【比特币基地】](https://www.coinbase.com/) 为接入更广泛的密码经济提供了一个可信且易于使用的平台。100 多个国家的 9800 多万认证用户、13，000 家机构和 230，000 家生态系统合作伙伴使用比特币基地轻松安全地投资、消费、储蓄、赚取和使用加密。该公司的使命是通过加密经济增加世界的经济自由。为了实现这一目标，有必要开发透明、安全、可靠并惠及所有参与者的公共基础设施。

“比特币基地的基础设施在云 本地技术上蓬勃发展，随着公司的发展，Kubernetes 将继续在我们的技术堆栈中发挥越来越重要的作用，”比特币基地 的开发者关系工程师 Jordan Harband 说。“我们正在大力投资云原生技术，以确保开发者生态系统云服务的持续成功和可持续性。我们期待与 CNCF 合作，推动更多计划，并为社区提供和改进更多工具。”

比特币基地云在很大程度上依赖 Kubernetes 提供一个平台，使公司能够跨多家云提供商部署和维护区块链基础设施。出于这个原因，雇佣了 Kubernetes 的核心贡献者，而云已经为几个项目做出了贡献，包括 Kong、kOps、Kyverno 和 Flux。

“比特币基地一直是加密货币经济的开拓者，”云计算原生计算基金会的执行董事 Priyanka Sharma 说。“在一个新生但正在改变世界的行业中，他们面临着一系列特殊的挑战，我们很高兴比特币基地决定与 CNCF 生态系统联手，并期待他们将带来的独特见解。”

**关于云原生计算基础**

云原生计算使组织能够在公共、私有和混合云中使用开源软件堆栈构建和运行可扩展的应用。云本地计算基金会(CNCF)托管全球技术基础设施的关键组件，包括 Kubernetes、Prometheus 和 Envoy。CNCF 汇集了业界顶尖的开发者、最终用户和供应商，并举办世界上最大的开源开发者大会。由 500 多个成员支持，包括世界上最大的云计算和软件公司，以及 200 多个创新创业公司，CNCF 是非营利 Linux 基金会的一部分。欲了解更多信息，请访问 www.cncf.io.
