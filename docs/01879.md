# CloudBees 任命 Susan Lally 为工程副总裁

> 原文：<https://devops.com/cloudbees-appoints-susan-lally-as-vice-president-of-engineering/>

October 16, 2019 09:00 AM Eastern Daylight Time

加利福尼亚州圣何塞([商业资讯](https://www.businesswire.com/))–[CloudBees，Inc.](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.cloudbees.com&esheet=52108306&newsitemid=20191016005004&lan=en-US&anchor=CloudBees%2C+Inc.&index=1&md5=e8f2f9547d3bfb6f5db41f73acbee340) ，推动持续经济发展的企业开发运维领导者，今天宣布 Susan Lally 加入 cloud bees，担任工程副总裁。在她的领导角色中，Lally 将为快速增长的持续集成、持续交付和应用程序发布协调中心规划、实施和监督工程策略。她从数字学习公司 McGraw-Hill Education 来到 CloudBees，担任工程和产品开发副总裁。

Lally 在构建和领导各种领域的高性能软件工程团队方面拥有 20 多年的经验。她擅长管理科技创业公司的小团队以及更大、更分散的企业团队。Lally 在将敏捷方法与业务目标联系起来方面有着丰富的经验，并且热衷于在开发人员工具领域工作。

CloudBees 首席执行官兼联合创始人 Sacha Labourey 表示:“随着 CloudBees 继续实施其企业开发运维战略，我们致力于确保我们的解决方案继续引领行业发展，并满足客户的企业需求。“Susan 深厚的技术经验和在高增长环境中管理团队的广泛技能使她非常适合。”

在 McGraw-Hill Education 工作的三年时间里，Lally 帮助团队交付数字学习产品，这些产品使用数据和见解来提高教师的效率和学生的学习体验。在加入 McGraw-Hill 之前，她在 GitHub 工作了两年，领导一个由 100 多名工程师、经理和质量专家组成的团队为 GitHub 应用程序和平台提供新功能和改进。她还在 Nook Media LLC 工作，担任 NOOK content services 的工程副总裁，在职业生涯早期，她在 Adobe Systems 担任了 10 年的高级工程职位，介绍并完善了 Adobe Flex 产品线。

“我喜欢在像 CloudBees 这样真正致力于卓越的合作和创新公司中推动工程团队的挑战，”拉里说。“我期待确保 CloudBees 的解决方案继续响应客户需求，并支持公司的高增长计划。"

**关于 CloudBees**

CloudBees 通过提供世界上第一个端到端连续软件交付管理系统(SDM)为连续经济提供动力。对于数百万推动各种规模企业创新的开发人员和产品团队来说，SDM 建立在持续集成(CI)和持续交付(CD)的基础上，使软件交付组织内部和周围的所有功能和团队能够最好地合作，以扩大价值创造。

CloudBees 是 CI、CD 和应用程序发布流程编排(ARO)的核心，建立在其产品的商业成功及其开源领先地位之上。CloudBees 是[詹金斯](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fjenkins.io%2F&esheet=52108306&newsitemid=20191016005004&lan=en-US&anchor=Jenkins&index=2&md5=b13a297b6ee8a12de0374b97becaf096)和[詹金斯 X](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fjenkins-x.io%2F&esheet=52108306&newsitemid=20191016005004&lan=en-US&anchor=Jenkins+X&index=3&md5=3f4d946942a9faba5b56384e80f75e5e) 的最大贡献者，也是[持续交付基金会](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fcd.foundation%2F&esheet=52108306&newsitemid=20191016005004&lan=en-US&anchor=Continuous+Delivery+Foundation&index=4&md5=3222dc441c1f8f9b62023d42c93fcb16) (CDF)的创始成员。从全栈开发人员实践 NoOps 的创业公司到大型财富 100 强公司，CloudBees 使所有软件驱动的组织能够在正确的时间智能地部署正确的功能。

超过 3，500 个世界知名品牌和超过 50%的财富 500 强企业依赖 CloudBees，因为它能够在任何开发环境中跨任何云工作，并平衡公司治理和控制与开发人员的灵活性和自由度。CloudBees 是世界领先的 DevOps 专家的家园，帮助数千家公司利用“持续一切”的力量，将他们置于从伟大的想法到伟大的软件，再到伟大的商业价值的最快路径上。

CloudBees 由 Matrix Partners、Lightspeed Venture Partners、威瑞森风险投资公司、Delta-v Capital、Golub Capital 和 unregular Ventures 支持，于 2010 年由前 JBoss CTO Sacha Labourey 和一个由持续集成、持续交付和 DevOps 专业人士组成的精英团队创立。在 [Twitter](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ftwitter.com%2FCloudBees&esheet=52108306&newsitemid=20191016005004&lan=en-US&anchor=Twitter&index=5&md5=5ea46faf84e4a8d601071f0ee25c9ab9) 、[脸书](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.facebook.com%2FCloudBees&esheet=52108306&newsitemid=20191016005004&lan=en-US&anchor=Facebook&index=6&md5=4f1e057fa6ce2b63f365128598816ddd)和 [LinkedIn](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.linkedin.com%2Fcompany%2Fcloudbees&esheet=52108306&newsitemid=20191016005004&lan=en-US&anchor=LinkedIn&index=7&md5=71fd9f0e7d74539d442713faf0a0a542) 上关注 CloudBees。
