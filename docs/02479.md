# DevOps 聊天:F5 的 NGINX 收购，与 F5 的 Lori MacVittie

> 原文：<https://devops.com/devops-chat-f5s-nginx-acquisition-with-f5s-lori-macvittie/>

F5 Networks 收购了领先的开源 web 服务器厂商 NGINX。此举显然将 F5 置于 DevOps 领域以及开源市场。

在本次 DevOps 聊天中，我们采访了 F5 的 Lori MacVittie 和 NGINX 的 Sidney Rabsatt。一些关于为什么这次收购对双方都有意义的深刻见解。
