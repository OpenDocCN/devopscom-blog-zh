# 《科学》杂志将 AIOps 评为主要行业分析领域的领导者

> 原文：<https://devops.com/sciencelogic-named-aiops-leader-in-premier-industry-analysis/>

*旗舰 SL1 平台在不断发展的 AIOps 领域引领大规模自动化根本原因分析*

**弗吉尼亚州雷斯顿——2020 年 11 月 3 日**—[ScienceLogic](https://jecoven14-dot-yamm-track.appspot.com/Redirect?ukey=1HPuFCJaoEi1CpfC82o-AKGtHlLFlU_4WINaimOU8TLA-1264390480&key=YAMMID-30370687&link=https%3A%2F%2Fsciencelogic.com%2F)今天宣布，在今年的 Forrester Wave 中，备受期待的报告发现 science logic 成为 AIOps 解决方案的领导者，并明确标志着 IT 运营管理前进道路上的全行业转变。

Forrester 深思熟虑的详尽分析巩固了该公司在 AIOps 市场的权威地位。[Forrester Wave:IT 运营的人工智能，2020 年第 4 季度](https://jecoven14-dot-yamm-track.appspot.com/Redirect?ukey=1HPuFCJaoEi1CpfC82o-AKGtHlLFlU_4WINaimOU8TLA-1264390480&key=YAMMID-30370687&link=https%3A%2F%2Fsciencelogic.com%2Fproduct%2Fresources%2Fleader-in-the-2020-forrester-wave)建立在之前评估 AIOps 的标准之上，概述了与 AIOps 的未来最相关的驱动趋势、解决方案和竞争优势。

ScienceLogic 被认可为今年浪潮中的领导者，这对该公司来说是一个有意义的基准。ScienceLogic 认为，这证实了它作为一个成熟参与者的地位，它在过去三年中不断扩张，继续塑造 ITOps 的故事，并为客户和最终用户提供强大的工具。

Hughes Network Systems，LLC 网络管理和架构高级总监 Karl Fosburg 表示:“在 Hughes，我们正在采用人工智能来优化我们的网络并增强客户体验，包括通过利用 AIOps 进行事故分类和缓解，将我们的网络运营提升到下一个卓越水平。"在支持我们采用 AIOps 技术方面，ScienceLogic 一直是一个积极的合作伙伴."

Wave 的发现尤其有意义，因为 AIOps 市场本身正在经历一场重大变革。哪家提供商领先将取决于分析和自动化的改进。AIOps 客户应该寻找通过数据关联支持跨团队协作的解决方案；提供深度、端到端的数字体验监控；并无缝集成到整个 ITOM 工具链中。

Forrester 的 Rich Lane 在报告中写道:“科学逻辑在规模和自动化 RCA 方面领先，”或根本原因分析。“SL1 应用一种算法方法来将实时数据湖置于上下文中并进行搜索，这可以加快运营团队确定根本原因和补救事故的速度。…[客户说]由于其先进的自动化功能，包括运行手册自动化、预测容量管理、自动化事件管理和 CMDB 准确性，该解决方案允许他们在不增加员工的情况下发展业务。”

鉴于其快速发展，AIOps 正变得越来越具有竞争力，其动态不断变化，基本类别和标准的定义也不断变化。由于需求、成熟度、目标最终用户和视角不同，没有哪两个客户会以相同的方式看待 AIOps 或 AIOps 提供的体验。

Wave 是组织与最适合其特定需求的供应商保持一致的重要指南——根据其成熟度级别、发展轨迹和目标，无论是现在还是未来。

ScienceLogic 首席执行官戴夫·林克(Dave Link)表示:“迈向 [AIOps](https://jecoven14-dot-yamm-track.appspot.com/Redirect?ukey=1HPuFCJaoEi1CpfC82o-AKGtHlLFlU_4WINaimOU8TLA-1264390480&key=YAMMID-30370687&link=https%3A%2F%2Fsciencelogic.com%2Fsolutions%2Faiops) 的旅程始于基础构建模块，并利用以服务为中心的管理，这种管理避开了设备级监控的限制，提高了应用程序的运行状况、解决问题的时间和用户体验。“面向服务的视角提供了对 IT 基础架构中的关系和依赖关系的上下文和理解，最终会影响服务和体验，全面地可视化风险和缓解策略。这些是连接基本原则和高级功能的分层路线图的结果。”

认识到竞争转型格局的破坏性以及 ScienceLogic 在塑造 ITOps 未来中的明确角色，该公司将继续致力于改善用户体验和增强对基于指标的智能的访问，并推动作为关键市场竞争优势的新优势的边界。
