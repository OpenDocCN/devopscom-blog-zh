# DevOps 聊天:JFrog 收购 Shippable

> 原文:[https://devops.com/devops-chat-jfrog-acquires-shippable/](https://devops.com/devops-chat-jfrog-acquires-shippable/)

今天 DevOps 的大新闻是 JFrog 收购 Shippable。这为 JFrog Enterprise+平台添加了一个 CI/CD 解决方案，并进一步扩展了 JFrog 的端到端 DevOps 产品，超越了 Artifactory 等产品。

在本次 DevOps 聊天中，我们与 Shippable 首席执行官阿维·卡瓦勒、JFrog 首席执行官施洛米·本哈姆和 JFrog 营销副总裁阿亚利·戈尔德施米特讨论了此次收购背后的想法以及 JFrog 的未来计划。
