# AppDynamics 和 ServiceNow 主动保持您的关键应用运行

> 原文：<https://devops.com/appdynamics-servicenow-proactively-keep-critical-apps-running/>

**app sphere—LAS VEGAS—2016 年 11 月 15 日—** [领先的应用程序智能公司 AppDynamics](https://www.appdynamics.com/) 和企业云公司 [ServiceNow](https://www.servicenow.com/) 今天宣布推出 AppDynamics for ServiceNow，这是一款全新的集成解决方案，可帮助全球 2000 强企业主动管理企业应用程序，以实现卓越的业务正常运行时间。

随着越来越多的商业服务通过应用程序提供，每个行业的企业都面临着提供快速无缝的数字体验的压力，这种体验曾被亚马逊、谷歌和脸书视为主流。世界各地的 IT 部门都需要解决如何实现业务数字化转型以满足这些现代消费者需求的问题，但他们却越来越落后，因为他们不了解如何解释应用体验对业务的影响的指标。

AppDynamics for ServiceNow 是一种以当今数字业务所需的速度提供应用洞察和服务健康感知的现代方式。该解决方案连接市场领先的平台，通过利用 [AppDynamics 应用性能管理](https://www.appdynamics.com/product/application-performance-management/)和[service now service watch Insight](https://www.servicenow.com/products/it-operations-management.html)，帮助现代 IT 组织从被动管理应用和业务服务性能转变为主动管理应用和业务服务性能。借助 AppDynamics for ServiceNow，企业现在可以:

*   **获得更好的可见性**–通过这种集成，AppDynamics 业务事务图被导入到 ServiceNow CMDB 中，提供了对性能的高级可见性。
*   **了解全面的服务运行状况**–通过将 AppDynamics 应用洞察与 ServiceNow 中来自其他基础设施监控源的事件进行组合、过滤和关联，IT 可以呈现服务运行状况的整体情况。这使 IT 团队能够大幅减少他们管理的事件数量，并几乎立即发现可能影响最终用户体验的问题。
*   **减少停机时间**-利用这些综合信息，ServiceNow 可以智能地触发自动补救措施，以防止停机或触发事件管理流程，从而缩短恢复时间。
