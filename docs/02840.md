# DevOps 儿童节日礼物指南:给予代码的礼物

> 原文:[https://devo PS . com/devo PS-kids-holiday-gift-guide-giving-the-giving-the-gift-of-code/](https://devops.com/devops-kids-holiday-gift-guide-giving-the-gift-of-code/)

任何一个称职的 DevOps 专业人士都知道未来的工作与编码、脚本和自动化有着千丝万缕的联系。这已经是一个以发展为导向的世界，并且正在日益发展。因此，如果今天的孩子想在以后的生活中获得成功，他们至少需要了解编程的基本原理——即使他们自己没有兴趣成为成熟的开发人员。这种知识就是教育专业人士所说的“计算思维”

然而，学习这种思维模式并不一定要做苦工！幸运的是，有大量令人敬畏的 STEM 玩具和游戏可以让孩子们从很小的时候就进入编码思维模式——同时在这个过程中获得乐趣。我们不厌其烦地筛选礼物，让 DevOps 社区在他们的节日礼物赠送上有一个良好的开端，无论他们希望带进这个圈子的孩子的年龄如何。

# **初中和高中**

![](../Images/59c5ecc9e29c7826b069c4ad74f3b1dc.png)

[赛车场](https://www.amazon.com/Circuit-Playground-Express-Developer-Base/dp/B0764MCBNR/ref=cm_cr_arp_d_product_top?ie=UTF8)

价格:34 美元

这款方便的入门套件内置了彩色 led、环境传感器等，可以让孩子们进入实验性嵌入式编程。该平台运行 MakeCode、Circuit Python、Adruino 和 Code.org CS discovery，为创建独特的项目提供了大量的跑道。游乐场的创造者有一个非常广泛的在线知识库和创造者社区，给青少年一些开始的想法。

![](../Images/2af3f3dacf0b49b165410006d4af8355.png)

[太空化学](https://store.steampowered.com/app/92800/SpaceChem/)

价格:2 美元

在 Steam 上以便宜的价格购买这款基于设计的益智游戏，让年龄较大的孩子以有趣和引人入胜的方式进行编程。该游戏让玩家在一个科幻太空殖民地中扮演“反应堆工程师”，建造化学合成器，将原材料转化为化学产品。当他们解决难题时，他们开始学习像排序、循环、分支和子程序这样的基础知识。

![](../Images/689acb564eff0ce03e776c160f260c17.png)

[MakeBlock Ultimate 2.0](https://www.amazon.com/Makeblock-DIY-Ultimate-Robot-Kit/dp/B00W6Y0Z4E/)

价格:350 美元

这是市场上最通用的青少年机器人套件之一，它有 160 多个部件，可以组合成 10 多种形式，然后通过多种方法进行编程。初学者可以使用拖放功能块，随着学习的深入，孩子们可以使用 Python、Arduin IDE 和 NodeJS 进行编程。
