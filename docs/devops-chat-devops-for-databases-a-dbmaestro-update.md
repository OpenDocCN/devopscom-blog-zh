# DevOps 聊天:数据库的 DevOps，DBmaestro 更新

> 原文：<https://devops.com/devops-chat-devops-for-databases-a-dbmaestro-update/>

DevOps 正在向数据库进军。虽然在过去几年中有几家公司致力于此，但随着越来越多的数据库管理员和组织意识到数据库开发运维的好处，所有人都认为数据库开发运维已经取得了长足的进步。
