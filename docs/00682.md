# 网络研讨会:使用 NGINX 负载平衡和缓存扩展 Web 应用

> 原文:[https://devo PS . com/webinar-scaling-web-applications-with-nginx-load-balancing-and-caching/](https://devops.com/webinar-scaling-web-applications-with-nginx-load-balancing-and-caching/)

构建一个 web 服务器很容易，但是当您添加负载平衡器和缓存服务器时，事情会变得复杂得多。这些设备的最佳配置需要该领域的大量专业知识，而且随着站点的增长，确保高性能变得越来越复杂。在本次网络研讨会中，Datadog 和 NGINX 将向您展示:

*   如何配置最佳负载平衡器和缓存服务器
*   确保最佳性能的关键指标是什么
*   当你需要帮助时，到哪里去找

与 NGINX 的技术解决方案架构师里克·尼尔森和 Datadog 的布道者马特·威廉姆斯一起，学习如何创建一个健壮的 web 环境来保持全速运行。
