*借助公司 15 年来在低代码方面的专业知识，新的统一开发平台使 IT 团队和业务用户能够构建复杂、现代和合规的解决方案，以获得安全、易用和定制的低代码体验来解决业务挑战*

**德克萨斯州奥斯汀，2022 年 3 月 3 日—** 今天，[全球领先的技术公司 Zoho Corporation](https://www.zoho.com/) 发布了其低代码产品的最新版本——Zoho Creator Platform，这是一款一体化解决方案，通过提供低代码解决方案构建体验，而无需考虑技术诀窍，从而在业务用户和 IT 团队之间架起了一座桥梁。组织目前面临着对新业务软件需求的飙升，但现有的面向业务用户的低代码平台只能促进基本的应用程序构建，缺乏开发可扩展、企业就绪的端到端工具所需的复杂性。市场上专门为业务用户设计的现有解决方案只能促进基本的应用程序构建，缺乏开发端到端工具所需的复杂性，这些工具允许组织进行扩展并确保集中治理。

Zoho Creator Platform 满足了业务用户和 IT 团队的需求，支持所有级别的构建、管理、集成和分析，以获得完整的低代码体验，实现比市场上任何其他解决方案快 10 倍的部署速度。对低代码开发工具的民主化访问允许 IT 和业务用户协同工作，创建和部署高度定制的应用程序，同时最小化常见的互操作性、权限和治理挑战。

Zoho Corporation 首席宣传员 Raju Vegesna 表示:“目前，市场上还没有一种低代码解决方案可以让业务用户和 IT 部门真正构建端到端的业务解决方案。“通过此次更新，Zoho 的 Creator 平台将应用程序开发、集成、流程自动化、商业智能和分析整合到一个平台中，同时使 IT 团队能够轻松应对安全性、合规性和治理挑战。组织现在可以利用一个统一的解决方案来满足他们所有的低代码需求，将创新能力扩展到每个员工。”

Zoho Creator 平台整合了数字数据和流程，以便为业务用户和 IT 团队创建真正敏捷的解决方案。业务开发人员现在可以轻松构建可扩展的低代码解决方案，包括应用、集成、分析和流程自动化，而无需严重依赖 IT 团队或耗时的培训。对于 IT 利益相关方而言，该平台现在提供了更深入的治理和管理控制，而不会影响自动化、数据驱动的洞察、定制以及影响收入和增长的能力。业务用户和 IT 部门可以一起快速部署安全、动态的自动扩展解决方案，以便组织可以专注于解决业务问题和寻求新的机会。

“我们需要一个完整的解决方案来解决具体的运营困难，包括缺乏简化的沟通和无法有效监督员工。Zoho Creator 平台使我们能够构建新的上下文工具，统一分散的沟通渠道，自动化流程，并无缝集成到我们现有的系统中，”阿联酋物流公司系统流程经理 Gaurav Kakkar 说。“此外，新平台为特定管理员提供了更大的灵活性，使他们能够通过沙盒和演示用户选项来实施新流程，从而确保我们的数据安全，同时提供重要的业务见解。通过合作，我们能够更好地支持员工提高工作效率，使他们能够协作并发现机会，从而更好地为自己和客户服务。”

“有了这个版本的 Creator，Zoho 正在将其低代码重点从应用程序转移到商业解决方案，”行业研究公司 Intellyx 的总裁杰森·彭博说。“Creator 帮助业务利益相关者和 IT 人员协作，将应用程序、商业智能和分析以及流程自动化整合到端到端业务解决方案中。Creator 对于非技术人员来说足够简单，同时也足够强大，能够在安全、可扩展的环境中处理复杂的工作流场景。”

**新 Creator 平台的主要创新领域包括:**

将应用构建扩展到解决方案构建

**统一解决方案构建器—**从单一控制面板创建、集成、分析和管理大规模定制业务解决方案。借助各种功能之间的深度互操作性和内置护栏，IT 团队可以在不同元素之间无缝切换，从而推动企业范围的公民发展计划。

**人工智能(AI)—**Zoho 的智能人工智能助手(Zia)现在可以让开发人员从任何地方导入数据，一键清理和转换，建立数据模型，并通过智能导入自动检测数据关系。预测分析、关键词提取、情感分析、OCR 和对象检测等人工智能任务内置于拖放式用户界面中，便于部署。

**流程蓝图—**这种直观的拖放式构建器通过指定每个步骤、在利益相关者之间分配职责，以及基于预设条件自动执行操作，以图形化方式构建业务流程的在线副本。用户现在可以识别瓶颈，并清楚地促进可扩展性，以满足不断发展的业务需求。

**增强的无服务器功能—**开发人员现在可以在 Zoho Creator 平台中使用 daulus、Java 或 Node.js 编写、存储和执行可重用的代码块。这些功能鼓励 IT 和业务团队合作，更快地构建可扩展且易于维护的企业级应用。

构建解决方案的互联生态系统，实现无缝体验

**统一数据建模(UDM)–**未经检查的数据点、多样的数据格式和不断发展的应用环境为数据集成增加了难以解决的复杂性，导致业务成效不佳。现在包括通用云建模，通过统一数据架构和 600 多个即插即用连接器自动化数据集成协议，帮助开发人员建立安全的数据传输，使用户更容易进行数据集成。

**集成状态仪表板—**用户可以通过新的集成状态仪表板全面了解每个集成的运行情况。状态控制面板提供了有关最近执行情况、成功和失败统计数据以及顶级连接的重要发现，有助于早期错误检测和解决，使组织能够更快地发现错误并解决问题。

**嵌入式商业智能和分析引擎—**Zoho Creator 平台中现已包含业内首个成熟的商业智能和分析引擎，使用户能够从驻留在其应用程序中的数据中获得商业见解，而无需担心会带来新的订阅。预测分析、数据警报、假设分析、数据混合和对话分析等高端 BI 功能现在都在低代码平台上得到支持。

通过端到端 ADLM 增强对 IT 的控制

**环境—**开发人员现在可以根据解决方案的准备情况，通过单击鼠标，在不同的环境中启动解决方案，包括开发、试运行(测试)或生产环境。这使得用户可以不断添加功能或进行更改，而不会干扰生产中的现有应用程序。这些环境还支持基于角色的访问，以帮助业务开发人员模拟特定于用户的体验验证。

**集中治理—**为了帮助 IT 部门管理低代码开发，低代码平台中现已推出各种功能，如基于角色的细粒度访问控制、全面的审计跟踪和备份选项、应用程序 IDE、用于查看和管理用户的仪表板、计费仪表板、本地化、品牌定制等个性化选项。

**集成的沟通和协作帮助—**通常，业务开发人员会因为一些小障碍而陷入构建阶段，尤其是在对平台不熟悉的时候。为了增强通信和协作，集成通信支持消息传递、发送文件、共享屏幕以及进行音频和视频通话。
