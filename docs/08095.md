# 使用 Menlo Security 调整安全性

> 原文:[https://devops.com/adapting-security-with-menlo-security/](https://devops.com/adapting-security-with-menlo-security/)

云数据安全是 CISOs 最关心的问题。用户数据正在跨云、平台和应用程序移动，如果我们不转向以数据为中心的方法，我们将无法保护这些数据。

杰克·米勒(Jack Miller)是门罗安全公司(Menlo Security)的专业服务负责人，曾担任 CISO 超过 16 年，他致力于解决数据、安全性和最终用户便利性之间的平衡问题。将安全性放在靠近用户和数据的位置，这是使安全性适应数据不断扩展的基本策略。
