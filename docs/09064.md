# 大量辞职激发了人们对技术认证的兴趣

> 原文：<https://devops.com/great-resignation-spurs-interest-in-tech-certifications/>

被称为“大辞职”的现象开始出现在全国性新闻中已经将近一年了。这些报告显示，每个月都有数百万美国工人离职，去寻找更好的工作。这种趋势，根据 [最新消息](https://www.marketwatch.com/story/people-quit-jobs-at-slightly-slower-rate-in-october-11638976546) ，是不会消失的。

虽然劳动专家对人们离职的原因提供了一系列猜测，从对薪酬的不满到对新冠肺炎的恐惧，再到寻找更有意义的职位的愿望，但他们尚未就那些现在失业的人在做什么达成共识。

在 IT 和技术领域，似乎那些加入[大辞职](https://devops.com/?s=great+resignation)的人正在利用业余时间通过追求技术认证来增长技能和完善简历。

## 科技课程在 2021 年大受欢迎

根据免费在线课程的搜索引擎和评论网站 Class Central 的说法，2021 年最热门的[](https://www.classcentral.com/report/most-popular-courses-2022/)在线课程是科技课程，其中许多是由谷歌提供的，并提供在简历上看起来很棒的认证类型。以下是一些顶级课程的例子:

*   用户体验基础(UX)设计，这是关于这个主题的一系列七门课程中的第一门。
*   使用 R 编程进行数据分析，提供如何“以新的更强大的方式清理、组织、分析、可视化和报告数据”的培训
*   AWS Cloud Technical Essentials，介绍 AWS 的基础知识。
*   AWS 云从业者基础知识，探索“基本的 AWS 云概念，帮助您获得信心，为贵组织的云计划做出贡献。”

## 支付红利的认证

总的来说，证书为求职者提供了动力。它们传达了对职业发展的承诺，并提供了一个人将带到某个职位的技能的可靠目录。对于那些想要脱颖而出的人来说，证书肯定能增加你获得面试的机会。

然而，考虑个人时间的机会成本是至关重要的。虽然我们喜欢免费认证，但许多免费认证不太可能帮助人们开始他们的职业生涯。在许多情况下，在人们花费数月时间获得这些免费认证后，他们将需要花费额外的时间获得新的认证，以确保他们拥有雇主想要的特定认证。当你考虑到这些技术职位的薪水时，花几个月时间在错误的免费认证上可能会让某人损失数万美元的收入。

虽然以下认证可能更难获得，并且可能不是免费提供的，但它们在帮助 IT 员工做好最理想工作的准备方面绝对有所作为:

*   网络–思科认证互联网专家(CCIE)和思科认证网络专家
*   云计算——Azure 专家、AWS 认证解决方案架构师专家和 Google 专业云架构师
*   安全——认证信息系统安全专家、认证信息安全经理、风险和信息安全控制认证或 CRISC(注意:这些认证需要有记录的经验)
*   Cloud devo PS–AWS devo PS 工程师专业版

## 将软技能添加到认证中

认证过程可以提供许多 IT 雇主正在寻找的技术技能。在简历中增加认证可以显著增加你被面试的机会。然而，以我的经验来看，找到工作是一回事；it 行业的成功需要一系列不同的技能，这些技能不属于 IT 认证的范畴。

软技能不是像编码或工程那样能让你胜任工作的技能。它们是决定你如何工作的技能。它们包括人际交往、沟通和解决冲突等技能，以及创造性思维和时间管理。当应用到 IT 领域时，我喜欢把它们看作是“认证加”

虽然 IT 工作者经常忽视软技能的价值，但我可以向你保证，IT 雇主绝对不会。随着 IT 工作的激烈竞争，IT 提供商需要尽可能利用一切优势来获得和留住客户。如果员工不具备与技术技能相匹配的人际技能，公司就会失去这种优势。

对于那些希望通过获得认证来增加自己对雇主吸引力的人来说，花些时间去获得软技能认证。这些课程是存在的，在你找到工作之前和之后，它们都会给你带来丰厚的回报。

## 结束大辞职

对于 IT 公司来说，认证课程越来越受欢迎，这可能会为如何结束大辞职提供一些见解，至少在 IT 行业是如此。

最近的 [研究](https://www.talentlms.com/tech-employees-great-resignation-statistics) 显示，IT“辞职者”正与其他行业的“辞职者”一样，面临许多相同的就业问题，包括对薪水的不满和对更灵活工作的渴望。然而，技术人员也提到了一些问题，表明在员工辞职前提供的认证培训可能足以让他们留在办公桌前。

例如，科技、IT 和软件开发公司的员工想离职的首要原因是职业发展有限，而不是薪水和福利。此外，90%的人表示他们“希望从公司获得更多的学习和发展机会。”

对于找工作的科技工作者来说，“技能发展”是他们在选择公司时考虑的第二大标准，仅次于工资和福利。超过 60%的人还表示，“更多的学习和培训机会会让他们在工作中更有动力”，75%的人认为他们的公司更注重吸引新员工，而不是投资现有员工。

留住 it 员工并填补大辞职留下的职位空缺，可能就像向员工提供认证培训一样简单吗？
