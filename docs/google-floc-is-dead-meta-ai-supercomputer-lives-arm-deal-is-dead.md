 *## 1.忘记“群体的联合学习”——万岁“主题”

本周第一条新闻:谷歌向不可避免的事情低头。第三方 cookies 很快就会消失，因为人们已经受够了被跟踪。但是谷歌的 FLoC 提议并不是答案。

### 分析:FLoC 是一个翻牌

谷歌不想侵犯你的隐私。但是谷歌确实想销售目标明确的广告。FLoC 和 Topics 的主要区别在于，FLoC 桶是自动生成的，因此是不透明的，有可能与“受保护的类别”重叠，例如种族或性别。但是主题 API 中的主题分类是经过管理的，并且具有用户可见的描述性名称。

弗雷德里克·拉迪诺瓦 : **谷歌消灭了 FLoC，代之以话题**

谷歌为基于兴趣的广告取代 cookies 的有争议的项目……已经死了。取而代之的是，谷歌今天宣布了一项新提议。…当您在网上浏览时，您的浏览器会了解您的兴趣。
…
当你偶然发现一个支持 Topics API 用于广告目的的网站时，浏览器会分享你感兴趣的三个主题。…然后，网站可以与广告合作伙伴分享这些信息，以决定向您展示哪些广告。…用户将能够查看和删除列表中的主题[或]关闭整个主题 API。
……
其他浏览器厂商是否有兴趣加入主题 API 还有待观察。[但是]他们都很快对絮状物表示冷淡。…计划在本季度末开始试用主题 API。

一个小的调整和重命名？“是也不是，”mananaysiempre 模棱两可地说:

是的，这仍然相当于在用户代理中构建用户跟踪功能。不，这不是同一种方法。
…
FLoC 正在对用户进行无监督聚类，而 Topics 则将用户分配到预先确定的聚类中。这使得它潜在地更加透明，并且似乎是为了解决“受保护类别”异议而设计的。

但是谷歌要说服 T2 还有很长的路要走:

我很高兴 FLoC 被抛弃了，但作为一个最终用户，我仍然不要求这个话题废话。这解决不了我的任何问题。它解决了广告业的问题。我会选择使用不支持此功能的浏览器。

## 2.元宇宙规模的机器学习是昂贵的

扎克伯格的新玩具是 *RSC:* 一台价值 4 亿美元的基于英伟达的人工智能超级计算机。这是**的第一阶段**。

### 分析:每秒 3.8×10 ^(18) 整数运算……很多

如果你的开发依赖于开放计算项目系统，你要知道它不再是黑客之路。梅塔/脸书公司正在向定制配置的**现成硬件**投入令人眼红的资金。

[蒂莫西·普里克特·摩根](https://www.nextplatform.com/2022/01/24/meta-buys-rather-than-builds-and-opens-its-massive-ai-supercomputer/ "read the full text") : **Meta 购买，而不是建造和开放它的大型人工智能超级计算机**

如果你认为建立脸书的社交网络需要大量的计算和存储，那你还什么都没看到。…建造元宇宙…将需要绝对巨大的超级计算能力。但是令人惊讶的是，考虑到脸书十多年来一直致力于通过开放计算项目开放其硬件设计，超级研究计算机[RSC]系统是由商用服务器、存储设备和交换机构建而成的。[这]可能归结为…与一些想要脸书知名度并愿意在价格和零件供应上做文章的供应商达成交易。
…
【所以 Meta 购买了】英伟达的 760 台 DGX A100 系统——每台系统……都有一对 AMD“罗马”64 核 Epyc 7742 处理器……1tb 主内存……八个 A100 GPU 加速器[和]八个 200 Gb/秒 HDR InfiniBand 控制器 Pure Storage 的全闪存存储，以及 Penguin Computing 的缓存服务器。…按照标价，这台机器将花费大约 4 亿美元 RSC 机器的第一阶段额定为 59 petaFLOPS … (FP64)和 118.6 petaFLOPS FP64，使用张量核心矩阵引擎。…在张量核上进行 INT8 处理，这是人工智能推理所需的，phase one 机器的额定性能为 3.79 亿次运算。

*怎么*多了？？？对克莱巴来说，这似乎不是什么新闻:

我曾经在一所大学工作，我的教授已经研究自动语音识别很长时间了，但大约 10 年前基本上放弃了这一研究方向，因为他认为大学根本无法在预算上与大型行业公司竞争。我想，对于大多数与 ML 相关的研究领域来说，迟早也会出现同样的情况。近年来，大公司已经在[自然语言处理]和[计算机视觉]领域进行了大量的研究创新。当然，对于这对整个社会意味着什么，还有一场讨论。

但是 3.79 exoOPS？根据桑格泽的说法，风就是这样吹的:

再过 15 年左右，这种水平的计算将会出现在你的桌面上。如果桌面还存在的话。
…
或书桌。

## 3.手臂出价:没有腿

说到英伟达，正如我在之前[说过的，Arm 合并真的不应该通过。而且看起来越来越不可能了。ARM 芯片在数据中心越来越重要，尤其是那些看重**“性能功耗比”的芯片**](https://devops.com/aws-outage-outrage-rusty-linux-arm-latest/)

### 分析:ARM M&A DOA—IPO PDQ

一群名副其实的彭博记者在一篇匿名报道上签下了他们的名字。这篇报道认为，Nvidia 从软银集团收购 Arm 的提议已经失败。**下一站:IPO** 。

[伊恩·金、贾尔斯·特纳、彼得·埃尔斯特伦、迪娜·巴斯、大卫·麦克劳林、鲁思·大卫和迪内什·奈尔](https://www.bloomberg.com/news/articles/2022-01-25/nvidia-is-said-to-quietly-prepare-to-abandon-takeover-of-arm "read the full text") : **英伟达悄悄准备放弃……Arm 出价**

据知情人士透露，英伟达预计交易不会完成。…与此同时，软银正在加紧准备 Arm 的首次公开募股。如果英伟达成功达成交易，这将是一次巨大的成功。…但这将是一场艰苦的战斗。…世界上最大的科技公司依赖 Arm 技术，他们担心在英伟达的领导下，他们可能会失去不受限制的访问权。
……
在软银内部，也有派系。这场磨难已经在英伟达内部造成了分歧。…“我们仍然对交易获得批准抱有希望，”软银发言人在一份电子邮件声明中表示。

我想 Arm 已经在英国——主要是剑桥——对其员工进行了⅔培训，他们正在那里扩建校园。该大学卡文迪什实验室的半导体物理小组目前拥有 18 个博士项目，在全球范围内处于领先地位。猜猜 Arm 的许多顶尖科学家是从哪里招募来的？Arm 和大学紧密相连。那家公司哪儿也不会去。
