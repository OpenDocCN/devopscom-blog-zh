# 网络研讨会:用 NoSQL 驱动下一代应用

> 原文:[https://devo PS . com/webinar-power-next-generation-applications-with no SQL/](https://devops.com/webinar-powering-next-generation-applications-with-nosql/)

**当今的企业应用需要极高的数据库性能和规模，以满足不断增长的业务和用户需求。**

**过去，扩展数据库意味着向集群添加更多节点，而不管这些集群支持什么服务。现在，借助多维扩展等功能，企业可以以无与伦比的精度优化其数据库基础设施，从而支持下一代业务应用。**

 **加入 Couchbase 的资深产品营销经理尚恩·约翰逊，他将讨论 ****向下一代企业应用的转变，以及像 Couchbase 这样的 NoSQL 数据库如何推动它们。**T3】**

主题包括:

*   不断变化的业务模式如何推动对下一代应用的需求
*   让这些新应用与众不同的重要特征
*   贵公司下一代应用的融资、构建和成功战略
