# 傀儡名字艾比·科恩斯首席技术官

> 原文:[https://devops.com/puppet-names-abby-kearns-cto/](https://devops.com/puppet-names-abby-kearns-cto/)

### 卡恩斯将监督企业平台开发的下一阶段，扩展商业产品

俄勒冈州波特兰市。，2020 年 4 月 29 日(环球新闻通讯社)——[木偶](https://www.globenewswire.com/Tracker?data=GKK8exM4KRCL4YBipiAgzXROSh0rboA-oHy2Iu3MWvghEzyOgPBtS4wpPXUJy3DuuRb3TTjKcPF8dCMRSOeNig==) ，自动化交付和操作软件的标准，为我们周围的一切提供动力，今天宣布 Abby Kearns 为其首席技术官。Kearns 在云计算和不断发展的开源社区方面拥有 20 年的深厚专业知识，曾在 Cloud Foundry Foundation、Pivotal Software 和威瑞森担任重要领导和产品管理职务。在 Kearns 的领导下，Puppet 将继续增长和发展其产品组合，以实现帮助企业现代化和管理其基础设施的承诺，保持高效、安全和合规，并快速提供更好的软件和服务。

“Puppet 是 DevOps 运动的先驱，引入了基础设施即代码(IaC)的概念，并通过基础设施自动化为 80%的财富 5000 强企业提高了效率。加入这样一个令人惊叹的团队，并在一个瞬息万变的市场中进一步挖掘我们的潜力，我感到无比兴奋，”卡恩斯说。“作为 Puppet 的首席技术官，我期待着与领先企业合作，在一个日益混合的世界中有效地改变他们管理和扩展基础设施的方式；随着新冠肺炎从根本上改变我们的工作和运营方式，这一点从未像现在这样重要。”

“随着我们继续在市场中推动加速影响，我们需要合适的人来实现我们的愿景。这就是为什么我很高兴欢迎 Abby Kearns 成为我们的首席技术官。我很幸运地称艾比为同龄人，并一直关注她的职业生涯多年，”Puppet 首席执行官伊冯娜·瓦塞纳尔说。“Abby 非常清楚 Puppet 的使命，去年她在我们的产品顾问委员会任职，本质上是一名技术专家。她为我们的职位带来了巨大的平衡——她在企业中有丰富的经验，知道如何大规模解决问题。”

Kearns 是技术行业的资深人士，他的职业生涯涵盖了财富 500 强公司和初创公司的产品营销、产品管理和咨询。最近，Kearns 担任了 Cloud Foundry Foundation 的执行董事，她主要负责推动基金会的愿景和开源项目的发展。在加入 Cloud Foundry 之前，她推动了 Pivotal Cloud Foundry 的采用，在威瑞森，她领导的产品管理团队致力于云服务的早期发展。她的背景还包括 Totality、EDS 和 Sabre 等公司的运营。

在 Puppet，Kearns 将领导公司当前和未来产品组合的愿景和方向，确保客户拥有通过 Puppet 的商业产品获得成功所需的正确工具和平台。
