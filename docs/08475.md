# Jelastic 与 Tech Equity 合作，为云应用提供先进的 DevOps 平台

> 原文：<https://devops.com/jelastic-partners-tech-equity-deliver-advanced-devops-platform-cloud-applications/>

*je elastic Cloud Union 通过技术资产进行扩展，提供了一个易于应用生命周期开发和管理的交钥匙云平台*

加州帕洛阿尔托(PRWEB)2016 年 2 月 16 日

[Jelastic](https://jelastic.com/) ，Inc .，云协调和 DevOps 领域公认的领导者，宣布与 [Tech Equity](http://www.techequity.company/) 建立战略合作伙伴关系，为应用程序提供终极、经济高效的云托管平台。

Jelastic 使 DevOps 团队能够开发和运行他们的应用程序，并从资源自动伸缩中获得全部好处，同时提供成本优化。用户可以专注于他们的核心任务，而不必关心维护和管理问题。Jelastic platform 默认支持多种软件堆栈、Docker 容器、NetBeans IDEs 插件、Eclipse 和其他技术，提供了一个包含复杂解决方案的交钥匙云。je last`s highly advanced user-friendly platform simplifies greatly applications`开发，并因其现收现付的特性而提供真正灵活的定价。

“全球越来越多的公司对容器和 DevOps 集成到他们的业务表现出浓厚的兴趣。为了满足这些需求，我们不断扩大 Jelastic Cloud Union，现在与 Tech Equity 合作。”Jelastic 首席执行官 Ruslan Synytsky 说，

Tech Equity 目前提供两个云领域。入门级 Tech Equity DevOps 云提供了开发、测试和集成具有开箱即用交易处理功能的应用程序的机会。Tech Equity 的金融技术总监、伦敦金融城的前首席投资官 Linus Etube 博士领导了 Swifin 交易交易所的设计，该交易所使任何 ISV 和任何开发者能够将其应用程序与 Swifin 集成，为 ISV 和开发者提供了即时获得新客户和收入的机会。

Tech Equity 的创始人 Shiyghan Emmanuel Navti 博士评论道:“通过与 Jelastic 的合作，我们能够提供一个具有集成交易处理能力的高级企业云平台，使广泛的用户能够快速采用云服务，包括那些无法访问银行帐户和信用卡的用户。”他是一名技术行业资深人士，曾领导 IBM 欧洲私有云平台技术销售团队，并负责开发 IBM 非洲扩张战略的关键要素。

Linus Etube 博士也是 Sedel Capital LLP 公司的创始人，该公司最近获得了英国金融行为监管局对 eMoney 发行和提供相关支付服务的监管批准。他评论说，Jelastic 云平台提供的灵活性使得使用云技术和 Swifin 交易服务交易所确保向数百万人(包括无银行账户者)安全交付金融服务成为可能，从而使任何人或任何企业都可以使用任何移动电话或网络从任何地方访问云技术服务并为其付款。这将为实体经济中所有人更大程度的参与和更多地使用技术来完善和优化许多企业的关键业务流程以支持更大的财富创造带来机遇。

Natural Enterprises Finance and Marketing Agency Ltd 首席执行官 Callistus Mubangizi 在评论 Tech Equity cloud 的推出时表示，Swifin 云平台的灵活性和效率创造了前所未有的机会，可以为其遍布乌干达的 900 多个贷款和储蓄社区提供一系列服务。

Tech Equity 选择 Jelastic 作为合作伙伴，是因为它能够满足 ISV、DevOps 团队、企业和金融服务组织向云迁移的关键需求。Jelastic 提供了一个生产就绪的解决方案，具有一系列好处，如高级 Docker 容器支持、智能分发、自动垂直和水平扩展、高可用性、安全性和完全隔离。与 Jelastic 的合作使 Tech Equity 能够支持最流行的编程语言——PHP、Java、Python、Ruby 和 Node.js。

关于科技股

Tech Equity 是一家托管云平台服务提供商，致力于通过为以前无法支付服务费用的客户提供服务，使应用程序开发人员和服务提供商获得新的收入机会。科技股票云是与 Jelastic 合作设计的，并得到了微软和 IBM 的支持。Tech Equity 使 ISV 和云应用程序开发人员能够通过使用与交易处理服务集成的高效应用程序托管平台来扩大客户群并创造更多收入。Tech Equity 总部位于英国伦敦，是欧洲最大的金融科技加速器。有关科技股票的更多信息，请访问。欲了解瑞士金融服务公司的信息，请访问 http://swifin.com。
