# Zammo.ai 现已在 Microsoft Azure Marketplace 中提供

> 原文：<https://devops.com/zammo-ai-now-available-in-the-microsoft-azure-marketplace/>

*微软 Azure 全球客户现在可以访问 Zammo.ai，利用 Azure 的可扩展性、可靠性和敏捷性来推动应用程序开发和制定业务战略。*

**加利福尼亚州三藩市，****2021 年 5 月 26 日** — Zammo.ai **，**这家彻底改变对话式人工智能格局并加速向语音经济转型的公司，今天宣布其 SaaS 人工智能解决方案在微软 Azure Marketplace 上市，这是一家提供 Azure 上使用的应用程序和服务的在线商店。Zammo 客户现在可以利用高效、可信的 Azure 云平台，简化部署和管理。

Zammo 的 AI 语音平台使组织能够设计和部署跨所有渠道的对话式 AI，而无需编码，包括聊天机器人、交互式语音响应(IVR)/电话、亚马逊 Alexa 和谷歌助手等语音助手、短信/文本、电子邮件、来自 web 和移动的聊天机器人、Facebook Messenger 等。该平台通过加速向对话式人工智能经济的过渡，帮助组织创造卓越的客户体验。Zammo 还提供跨所有渠道的统一数据分析，以便公司可以利用用户参与的洞察力进行业务导向的学习和持续改进。

“我们很自豪能够在微软 Azure Marketplace 上拥有 Zammo 平台，因为它代表了我们增长目标的关键一步，”Zammo.ai 首席执行官亚历克斯·法尔说。“任何希望通过对话式人工智能快速转变用户交互的企业现在都可以访问 Zammo 的无代码技术，该技术适用于所有参与渠道。”

“通过微软 Azure Marketplace，世界各地的客户可以轻松地找到、购买和部署他们可以信任的合作伙伴解决方案，所有这些解决方案都经过认证和优化，可以在 Azure 上运行，”微软公司微软 Azure 平台总经理 Jake Zborowski 说，“我们很高兴地欢迎 Zammo 的解决方案加入不断增长的 Azure Marketplace 生态系统。"

Azure Marketplace 是一个在线市场，用于买卖经认证可在 Azure 上运行的云解决方案。Azure Marketplace 帮助寻求创新的、基于云的解决方案的公司与已经开发出可以使用的解决方案的合作伙伴联系起来。

在 Azure Marketplace 的[页面](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/zammo.zammo_ai_saas?ocid=GTMRewards_PR_zammo_ai_saas_72ba)了解更多关于 Zammo SaaS 解决方案的信息。

**关于 Zammo.ai**

Zammo.ai 是对话式 ai 的创新者。我们正迅速成为引领各种组织通过语音、聊天机器人和电话/IVR(交互式语音识别)实现数字化转型的领导者。我们正在彻底改变企业与客户沟通和交易的方式。随着技术的发展和语音成为我们日常生活的一部分，组织将需要通过客户选择的沟通渠道与客户互动。我们相信所有的企业都应该有这样的机会，而不仅仅是商业巨头。通过利用 Zammo 的用户友好、价格合理且跨平台的解决方案，各种规模的企业都可以在语音和聊天平台上拥有品牌形象。语音人工智能为残疾人打开了一个全新的能力世界。我们热衷于建立一个平台，让每个人都可以获得信息，让所有人都可以与企业进行交易。
