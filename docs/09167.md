# OneLayer 以 820 万美元的种子资金走出秘密，以保护私有 5G 网络

> 原文:[https://devo PS . com/one layer-steps-out-of-stealth-with-8-2m-seed-round-to-protect-private-5g-networks/](https://devops.com/onelayer-steps-out-of-stealth-with-8-2m-seed-round-to-protect-private-5g-networks/)

*随着越来越多的公司实施自己的 LTE/5G 网络，为这些网络提供强大的网络防御正成为企业安全的基础*

**以色列特拉维夫，2022 年 3 月 15 日** — [为企业提供私有 LTE/5G 网络安全的先驱 OneLayer](https://one-layer.com/) 今天宣布推出隐形产品，并获得由 Grove Ventures 和 Viola Ventures 领投的 820 万美元种子资金。其他投资来自领先的网络和云公司的天使投资人，包括 Orca Security 的创始人兼首席执行官 Avi Shua、Guardicore 的创始人 Ariel Zeitlin、Palo Alto Networks 以色列&研发中心的负责人戈宁·芬克以及 Epsagon 的创始人、首席执行官 Nitzan Shapira 和首席技术官 Ran Ribenzaft 等人。OneLayer 将利用这笔资金构建其产品套件，以实现理想的企业级安全性，并为全球客户部署该产品套件。此外，该公司将与一级参与者合作，创建一个基于私有 LTE 和 5G 技术的网络风险评估和验证实验室。

对于企业而言，私有 LTE/5G 网络的建立实现了真正的连接革命，提供了显著的优势，如连接可靠性的跃升、具有容量和范围的专用带宽以及跨广阔区域的物联网设备的连接。在过去几年中，越来越多的企业开始实施私有 LTE/5G 网络，预计这一数字在未来几年将会显著增长。然而，这些网络的安全性明显落后。虽然传统的企业安全工具在过去十年中已经发展到实现其期望的标准，但是这些工具中的大多数并不适用于蜂窝领域，因为它们往往忽略了不同蜂窝网络所需的相关标准和协议的变化和进步。

OneLayer 旨在为组织提供与 IT 和 OT 网络可访问的保护标准相当的企业级安全性，但不要求他们成为蜂窝专家。该公司的技术使网络管理员能够通过探索和列出企业域的现有安全工具，将它们与企业的授权、认证和计费工具(AAA)集成，并将它们的功能扩展到组织的蜂窝域，来限制设备对网络上注册的那些人的暴露。通过这样做，OneLayer 提供了一种简单的、类似局域网的方式来控制他们的专用 5G 网络。

OneLayer 的联合创始人兼首席执行官戴夫·莫尔(Dave Mor)表示:“OneLayer 结合了我们在网络安全和蜂窝技术方面的深厚知识，旨在帮助企业公司消除其网络中的重大漏洞。“我们的解决方案高效且有效地解决了企业蜂窝安全面临的独特挑战，使我们的客户能够构建扩展的私有网络，具有为其特定需求量身定制的最高安全级别，并且超越了 IT 和 OT 网络的当前标准和能力。”

Grove Ventures 的普通合伙人 Renana Ashkenazi 表示:“在私有 LTE/5G 网络出现之前，企业安全和蜂窝安全一直并行存在，但轨道不同。“这一现实现在已经改变，需要新的解决方案。OneLayer 在解决企业和蜂窝安全的差距方面面临着难以置信的挑战，并正在为一个非常大的潜在市场构建一个解决方案。通过从网络角度以及团队的集体蜂窝专业知识来应对安全挑战，OneLayer 提供了一种解决方案，可以保护组织免受攻击，同时允许他们在其 LTE/5G 网络中运营。OneLayer 使组织能够安全地利用专用 5G 蜂窝网络的优势。”

Viola Ventures 普通合伙人 Zvika Orron 表示:“我们坚信，专用蜂窝网络的出现将在未来企业连接无线技术中发挥重要作用。除了为企业提供设计最适合其特定业务需求的网络的灵活性和控制力，私有蜂窝网络还将保护这些网络的责任转移到了企业自身。这需要一种独特的安全解决方案来应对新的攻击面，这也是我们非常乐意与 OneLayer 合作的原因。该公司正在应对一个不断增长的市场，同时凭借对网络安全和蜂窝行业的深刻理解应对这一挑战，完美地将它定位为构建私有网络的企业公司的合作伙伴。”

**关于一层**

OneLayer 为私有 LTE/5G 网络提供企业级安全性。它的平台和物联网安全工具包可以在私有蜂窝网络中实施，以提供更好的可见性并保护组织免受安全攻击，创建一个身份、一个策略规则和一个涵盖组织中每台物联网设备的安全方法。该公司由 Dave Mor 和/或 Turgeman 创建，他们是世界级的网络安全专家，对蜂窝协议和物联网安全需求有深刻的理解，并且是 IDF 81 情报部门的退伍军人。OneLayer 的创始团队包括来自以色列 8200 和 81 单位的强大网络专家团队，并得到来自网络安全和电信行业的行业领先顾问的支持。要了解更多关于 OneLayer 的信息，请访问[one-layer.com](https://one-layer.com/)。
