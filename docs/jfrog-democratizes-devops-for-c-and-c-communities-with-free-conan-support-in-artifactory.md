# JFrog 使 C 和 C++社区的开发程序民主化，在 Artifactory 中有免费的 Conan 支持

> 原文：<https://devops.com/jfrog-democratizes-devops-for-c-and-c-communities-with-free-conan-support-in-artifactory/>

JFrog 宣布推出免费的 Artifactory 社区版，通过为 C/C++社区带来最可靠的软件库来帮助加速 DevOps 革命

加利福尼亚州森尼维尔，2018 年 3 月 28 日—[devo PS 加速器 JFrog](https://jfrog.com/) 宣布，该公司的通用软件库 [JFrog Artifactory](https://jfrog.com/artifactory/) 现在提供免费的社区版，通过柯南包管理器为 C 和 C++社区服务。新的企业级产品是 C 和 C++社区第一个可以免费管理软件包的存储库。

每个企业公司都有 C 和 C++遗产，但直到今天，这项投资还没有 DevOps 就绪。数以百万计的程序员使用 C 和 C++语言，有数十亿行代码使用这些语言。该代码已被用于构建从嵌入式系统到物联网解决方案的各种类型的应用。使用 C/C++工件存储库是自动化 C 和 C++代码以及在 DevOps 流程中应用持续集成所必需的。

C 和 C++社区中领先的开源打包管理器柯南(Conan)提供了一个模型，在支持所有主要系统的同时减少了二进制兼容性问题。Conan 帮助 C 和 C++开发人员构建应用程序，并跨平台自动化他们的 C++依赖关系。C++开发人员已经在生产中使用 Conan，并在通用二进制发行版中使用 JFrog Bintray。作为其社区方法的一部分，JFrog 正在拥抱世界上的 C 和 C++社区，为数百万开发人员提供 Artifactory 的免费社区版本，该版本提供柯南支持，以便帮助连续和自动发布二进制文件。

“Artifactory Community Edition for C 和 C++对柯南用户来说是一个巨大的进步。从现在开始，他们可以轻松地在内部托管私有的柯南库，享受用户界面、高级权限和认证，以及增强的可扩展性。而且全部免费，”柯南联合创始人迭戈·罗德里格斯·洛萨达说。“这是为全球 C 和 C++开发人员实现 DevOps 民主化的终极资源，以可扩展的方式加速软件交付，向流动软件又迈进了一步。”

今年在 JFrog 年度用户大会 [swampUP](https://swampup.jfrog.com/) 上，JFrog 将特别关注柯南和 C++开发，包括一整天的培训课程、专门的柯南专题讲座，以及与柯南创始人和社区思想领袖交流的大量机会。沼泽将于 2018 年 5 月 16 日至 18 日在纳帕谷举行。现在可以在 https://swampup.jfrog.com 的[网上买票。](https://swampup.jfrog.com/)

要了解更多关于这个版本的技术细节，请访问 JFrog 的博客。

### 关于

JFrog 的 universal binaries hub 拥有 4，000 多名客户，每月下载量超过 20 亿次，是管理和分发软件二进制文件的领先通用解决方案。JFrog 的四款产品，JFrog Artifactory，通用神器库；JFrog Bintray，通用分发平台；JFrog 任务控制，用于通用 DevOps 流量管理:和 JFrog Xray(通用组件分析器)被世界各地的开发人员和开发人员工程师使用，并作为开源、内部部署和 SaaS 云解决方案提供。客户包括一些世界顶级品牌，如亚马逊、谷歌、脸书、优步、网飞、Twitter、Adobe、VMware、Slack 和 Spotify。该公司是私有的，在加利福尼亚、西雅图、以色列、印度、西班牙、中国和法国运营。更多信息可在[jfrog.com](https://www.jfrog.com/)找到。
