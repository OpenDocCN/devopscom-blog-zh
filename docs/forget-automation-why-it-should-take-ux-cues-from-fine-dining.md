# 忘记自动化:为什么它应该从美食中汲取 UX 的灵感

> 原文：<https://devops.com/forget-automation-why-it-should-take-ux-cues-from-fine-dining/>

作为一名大学生，我在一家五星级意大利餐厅工作，在那里我学到了客户服务的原则和实践。总经理清楚地知道在服务期间站在哪个角落可以看到每一张桌子。他们会根据用餐者的互动方式、他们的微信号、盘子里还剩多少食物以及他们扫视餐厅的方式来评估用餐者的体验。这种对细节和个人客户体验的强烈关注与运营面向服务的技术业务有什么关系？一切。

科技行业因以牺牲个性化为代价实现流程自动化而臭名昭著，这种厌恶可以延伸到建立必要的关系以生产成功的最终产品。可以理解的是，一些公司和他们的开发者更喜欢与他们的客户保持严格的功能关系，因为这些关系是混乱的，人们是不可预测的，没有什么事情会完全按照计划进行。这是我们必须为每一个新客户和项目反复跳的一支舞。

对于我们这些希望成为服务型科技公司的人来说，掌握客户和顾客体验要求我们保持专注，积极倾听。就像经验丰富的餐厅经理一样，我们必须不断审视我们的环境，寻找满意和不满意的线索。

## 服务不仅仅是最终产品

每天晚上开业前，总经理会让员工拿着长绳子测量桌子之间的角度和距离，以确保桌子是直的，边缘完全对齐。然后，在告诉员工晚上的特色菜后，他会随机询问他们关于菜单的问题。*蚕豆*是什么颜色？

看到这种老派标准的价值给我留下了深刻的印象，并帮助我将对细节的关注作为全面的基础实践，从设计良好的 UX 到培养关系。创造一个令人满意的环境是关于成功地精心安排一百万个微小的细节，并总是寻找调整以使客户体验更好。

从我最早在餐饮业的经历来看，这位似乎无所不知的意大利人也教会了我，移情与经营服务型企业的务实现实密切相关。他们教我如何照顾他人，如何让他们感觉良好，以及如何读懂肢体语言。

人们可能很容易将科技公司定型为没有人情味的产品分配者(将它们的开发者视为机器中的齿轮)，但我们不希望只是接受订单。我们的工作是从 [到](https://www.forbes.com/sites/forbesbusinesscouncil/2021/08/13/how-entrepreneurs-can-create-a-service-or-product-from-the-ground-up/?sh=846eaa83fbf6) 成为战略合作伙伴，以取得长期成功，这需要持续的关注和不断的适应。

## 映射客户体验

我们必须愿意一次又一次地重新设计我们与客户和最终用户的合作方式——它总是在变化，因为期望总是在变化，市场规范也总是在变化。当我们停止试图通过小的增量[](https://www.nytimes.com/guides/business/how-to-grow-your-business)改进来变得更好的时候，就是我们停止面向服务的时候。

就像一家餐馆有一个[](https://hoteltalk.app/restaurant-sequence-of-service-or-service-cycle/)的服务序列，它描绘了用餐者从进门的那一刻起的旅程，我们可以从每个接触点设计客户体验。

在销售过程中，将不同的人分层以在整个组织中创造更多的关系接触点是有价值的，因此，在我们的案例中，新客户会见了我们运营团队的一名成员，然后是我们工程团队的某个人，以确保他们感觉得到了充分的支持。然后，我们在沟通方面设定期望，作为脚本化入职体验的一部分。在我们的评审会议中，我们寻求反馈，以确定我们是否表现不佳，并在必要时进行纠正*。*

我从那家意大利餐馆学到的另一件事是，没有什么错误是不可挽回的。第一次做对还是做错并不重要，重要的是当意外发生时你如何应对。在犯了一个可怕的错误后扭转局面——甚至送一份冷牛排——可以培养更多的忠诚顾客，因为这表明你真的在乎。

## 服务的创新之心

尽管技术的形象是冷漠和遥远的，但重要的是要认识到[编码和开发](https://devops.com/?s=software+development)本身是创造性的，我们应该努力营造一个健康的环境，以便人们能够发挥出最佳水平。为此，把你的责任想象成贯穿整个链条，从员工到客户，最后到最终用户，是有意义的。在每个环节的工作中发现人性是建立成功伙伴关系的关键。

在构建产品的背景下，我们必须 [有创造力](https://hoteltalk.app/restaurant-sequence-of-service-or-service-cycle/) 为最终用户设计无缝、精心策划的体验。有大量的应用程序在最终被删除之前一直闲置在某人的手机上。抓住有共同兴趣和问题的人的注意力的唯一方法是理解他们的需求。

为此，我们必须经常问我们的客户:对用户来说，最容易的入口是什么？创造一个低门槛是一种快速的胜利，这本身就是一种入职体验和建立信任的机会。

同样，可能让一个客户满意的东西实际上可能不会在更广泛的市场中产生任何共鸣。如果客户只是雇佣我们写代码，我们只会对满足他们的期望感兴趣。我们的目标不仅仅是制造产品，而是*成功的*产品，建立或巩固客户在市场上的良好声誉。

## 通过关系增加价值

当我们站在房间的隐喻角落里观察客户体验时，我们正在整合来自拥有数百万用户的产品的数据，并收获洞察力。我们对快速解决方案不感兴趣。客户服务和对最终用户体验的关注意味着以市场需求为导向，而不是在 [真空](https://hbr.org/2020/01/why-the-best-developers-keep-customers-front-of-mind) 中进行设计。然后，随着时间的推移，我们可以继续创造价值。

在餐饮业，晚餐结束时的小惊喜会增加价值，因为它是意想不到的。一杯免费的餐后利口酒和一个简洁的起源故事是细节的最佳体现。创造更丰富、更全面的客户体验可以让一家科技企业在[](https://www.techtarget.com/searchcustomerexperience/feature/The-future-of-customer-service-12-trends-to-watch)爆炸式增长的市场中脱颖而出。对于我的老导师们来说，从头到尾关注每一个细微之处就是五星级服务的定义。
