# Perforce Software 收购 Methodics，扩展 DevOps 产品组合

*Perforce 继续发展顶级半导体公司已经使用的平台*

**明尼阿波利斯、旧金山和圣塔莫尼卡，2020 年 7 月 20 日**–Perforce Software，Inc .(“Perforce”)，一家为需要开发生命周期中的生产力、可见性和规模的企业团队提供解决方案的公司，由 Clearlake Capital Group，L.P .(及其附属公司“Clearlake”)和 Francisco Partners 支持，今天宣布收购[Methodics Inc .](https://methodics.com/)。(“Methodics”)，领先的企业知识产权生命周期管理和可追溯性解决方案提供商。Methodics 的加入进一步凸显了 Perforce 作为解决方案提供商在 DevOps 市场中的独特地位，该解决方案解决了世界上一些面临极端规模、上市时间和质量要求的最大技术团队的需求。条款没有披露。

Perforce 和 Methodics 之前有一个战略合作伙伴关系,提供一个完全集成的解决方案来管理半导体行业从设计到生产和再利用的所有知识产权。许多世界顶级半导体公司已经利用 Perforce 和 Methodics 的电子设计和数据管理解决方案。

Perforce 首席执行官 Mark Ties 表示:“半导体和嵌入式软件设计市场继续扩大，特别是在它们服务于不断增长的人工智能、汽车、云和物联网市场的时候。“与 Methodics 一起，Perforce 将使这些市场能够推动更快的上市时间、更高的质量、更高的运营效率以及更大规模的安全性和合规性。我们也很高兴有机会将这一联合解决方案扩展到游戏、汽车和其他行业。”

“多地点和多地域设计团队之间的合作越来越重要。Methodics 解决方案支持关键协作，同时确保重要设计资产及其相关元数据的完全可追溯性和重用。“我们很高兴加入 Perforce 团队，我们相信此次收购将使我们能够满足半导体和嵌入式系统市场以及游戏、汽车等领域的需求。”

Maxlinear SoC 技术副总裁 Paolo Miliozzi 表示:“在过去几年中，Maxlinear 一直在使用 Perforce Helix 和 Methodics 知识产权生命周期管理解决方案来提高我们的生产力，并在 Maxlinear 引入知识产权再利用方法。“随着 MaxLinear 的持续增长，我们期待着合并后的公司能够向我们提供解决方案。”
