# DevOps 聊天:厨师全力以赴开源

> 原文：<https://devops.com/devops-chat-chef-goes-all-in-on-open-source/>

在下个月于西雅图举行的 ChefConf 之前，Chef 宣布了对其商业模式的[重大改进。Chef 确认并阐明了其对开源商业模式的承诺，现在和将来都会将其所有软件作为开源软件发布。](https://devops.com/chef-commits-to-100-percent-open-source-code/)

在本次 DevOps 聊天中，我们采访了大厨公司产品和工程部门的 SVP·科里·斯科比。他解释说，Chef 将为开源软件增加服务、培训和支持，并将所有这些打包在一起。
