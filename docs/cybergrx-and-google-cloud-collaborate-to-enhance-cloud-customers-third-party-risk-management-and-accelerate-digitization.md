# CyberGRX 和 Google Cloud 合作增强云客户的第三方风险管理并加速数字化

> 原文：<https://devops.com/cybergrx-and-google-cloud-collaborate-to-enhance-cloud-customers-third-party-risk-management-and-accelerate-digitization/>

*谷歌云满足广大客户群的风险和采购需求，并提供更高效的入职流程*

**科罗拉多州丹佛市，2022 年 6 月 7 日—**[cyber grx](https://www.cybergrx.com/)，世界上第一个也是最大的全球网络风险交易所的提供商，今天宣布与 Google Cloud 联合合作，以满足 Google Cloud 客户和潜在客户的风险管理和法规遵从性需求。Google Cloud 客户可以利用 CyberGRX 的第三方验证网络风险评估来深入了解 Google Cloud 的风险概况，其中详细介绍了其基础设施内置的行业标准和安全协议的合规性。

随着越来越多的组织实施数字化转型计划并迁移到云，关键业务工作负载被委托给云服务提供商，如 Google Cloud。为了帮助客户和潜在客户更好地了解和管理与外包业务关键工作负载相关的风险，Google Cloud 正在与 CyberGRX 合作，以提供其安全态势的透明度，并支持客户的第三方网络风险管理之旅。因此，Google Cloud 的当前或潜在客户可以轻松地[请求](https://info.cybergrx.com/access-google-cloud-assessment)并立即在 CyberGRX Exchange 上访问经过验证的 Google Cloud 风险简介。这将是 CyberGRX 连续第四年对谷歌云服务进行独立验证。

“我们与 CyberGRX 的合作是谷歌云展示其致力于成为业界最值得信赖的云提供商的另一种方式，”谷歌云企业信任全球负责人 Rani Urbas 表示。“CyberGRX 的平台提供了一个有效而灵活的解决方案，帮助组织满足其尽职调查、风险管理和法规合规性需求。”

CyberGRX 平台产生标准化和可操作的数据，远远超过传统安全评估的结果。CyberGRX 考虑了固有风险、行业特定的威胁情报和真实的攻击场景。这使谷歌云客户能够用由外而内的证据来验证谷歌的姿态，以便产生专注于风险的结果，而不是简单的合规性。客户和第三方(如云提供商)都从 CyberGRX 交换模式中受益。CyberGRX 减轻了第三方的评估负担，并为客户及其第三方提供了全面的网络风险数据。

“从历史上看，组织采用过时的方法来处理第三方风险管理，包括静态电子表格和定制评估。在等待可能永远不会回来的评估时，公司仍然很脆弱。而且，即使收集了这些评估，数据也不是标准化的，这意味着从分析的角度来看，几乎无法对其进行处理，”CyberGRX 首席执行官 Fred Kneip 说。“通过与谷歌云的合作，我们能够推动组织从追逐风险评估转向即时了解各种分析和网络风险数据，这些数据可以根据通用和定制的框架进行映射。”
