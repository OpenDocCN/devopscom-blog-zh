# CloudBees 被评为 Google Cloud 2019 年度应用开发技术合作伙伴

> 原文:[https://devo PS . com/cloud bees-named-Google-cloud-2019-年度应用开发技术合作伙伴/](https://devops.com/cloudbees-named-google-cloud-2019-technology-partner-of-the-year-for-application-development/)

加利福尼亚州圣何塞—(美国商业资讯)—[Cloud bees，Inc.](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.cloudbees.com&esheet=52198919&newsitemid=20200406005550&lan=en-US&anchor=CloudBees%2C+Inc.&index=1&md5=ffdea3779e6f16d77e4619328390545d) 今天宣布，该公司获得了 2019 年度谷歌云技术应用开发合作伙伴。

CloudBees 因其在谷歌云生态系统中的软件开发成就而获得认可。联合客户能够通过 CloudBees 的持续集成和持续交付(CI/CD)产品优化软件交付流程，包括首个端到端持续软件交付管理(SDM)系统和首个由 Jenkins X 提供支持的云原生 CD 即服务解决方案

SDM 是一种新兴的 IT 软件类别，它将软件交付定义为核心业务流程。CloudBees】概述了其对类别的愿景，为其 SDM 平台引入了一系列功能，将组织的 DevOps 工具链中的工件、数据和事件连接到一个统一的记录系统中。CloudBees SDM 平台是在谷歌云基础上从头开始构建的。

[由 Jenkins X 提供支持的 Cloud bees CI/CD](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.cloudbees.com%2Fproducts%2Fcloudbees-ci-cd%2Foverview&esheet=52198919&newsitemid=20200406005550&lan=en-US&anchor=CloudBees+CI%2FCD+powered+by+Jenkins+X&index=3&md5=15f9bce9d73c9a259d233ef1616b76dc)通过在 Google Cloud 上为 Jenkins X 提供 SaaS 环境，扩展了企业的价值。使用该解决方案，开发人员可以获得构建-测试-部署生命周期的完整云原生体验，而无需花费时间管理云实施和 Kubernetes 的复杂性。CloudBees CI/CD 由 Jenkins X 提供支持，与 Google Cloud 的平台基础设施紧密集成。

CloudBees 首席战略官 Francois Dechery 表示:“我们很荣幸被评为谷歌云 2019 年度应用开发技术合作伙伴。“该奖项认可了我们在 2019 年取得的突出成功，在多个技术领域使用谷歌云技术的成功组合，提供创新的软件开发解决方案和客户满意度。”

“我们很高兴认识到 CloudBees 在谷歌云的应用和软件开发方面的工作，”谷歌云全球生态系统公司副总裁凯文·伊奇普拉尼说。CloudBees 在软件交付管理方面的能力建立在 Google Cloud 之上，可以帮助企业 IT 简化整个组织中新应用程序的开发和交付。我们期待继续合作，帮助客户取得成功。"

CloudBees Core 和 CloudBees Accelerator 都可以在 Google Cloud Marketplace 上获得。

**关于 CloudBees**

企业软件交付公司 CloudBees 提供业界领先的 DevOps 技术平台。CloudBees 使开发人员能够专注于他们最擅长的事情:构建重要的东西，同时通过强大的风险缓解、合规性和治理工具让管理层高枕无忧。《财富 100 强》中有 50%的公司使用 CloudBees，它正在帮助数千家公司利用持续一切的力量，让他们以最快的速度从伟大的想法到伟大的软件，到令人惊叹的客户体验，再到成为改变生活的企业。

CloudBees 由 Matrix Partners、Lightspeed Venture Partners、威瑞森风险投资公司、Delta-v Capital、Golub Capital 和 unregular Ventures 支持，于 2010 年由前 JBoss CTO Sacha Labourey 和一个由持续集成、持续交付和 DevOps 专业人士组成的精英团队创立。在 [Twitter](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ftwitter.com%2FCloudBees&esheet=52198919&newsitemid=20200406005550&lan=en-US&anchor=Twitter&index=8&md5=e83bbba6e6cde15d53978919e924bb7a) 、[脸书](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.facebook.com%2FCloudBees&esheet=52198919&newsitemid=20200406005550&lan=en-US&anchor=Facebook&index=9&md5=53a1fe7c01df45324f9a3be607ff330b)和 [LinkedIn](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.linkedin.com%2Fcompany%2Fcloudbees%2F&esheet=52198919&newsitemid=20200406005550&lan=en-US&anchor=LinkedIn&index=10&md5=d0aa3755e53b4e27a7acf6023564bc43) 上关注 CloudBees。
