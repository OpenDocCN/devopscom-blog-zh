# 趋势科技通过集成的技术、渠道和多元化计划展示了与 Amazon Web Services 的关系优势

> 原文:[https://devo PS . com/trend-micro-democrates-strength of-relationship-with-Amazon-web-services-through-integrated-technology-channel-and-diversity-programs/](https://devops.com/trend-micro-demonstrates-strength-of-relationship-with-amazon-web-services-through-integrated-technology-channel-and-diversity-programs/)

**达拉斯–2018 年 11 月 29 日**–[趋势科技](https://www.trendmicro.com/us/index.html)股份有限公司([TYO:4704](https://www.google.com/finance?q=TYO:4704)；全球网络安全解决方案的领导者 TSE: 4704 ，今天展示了其与亚马逊网络服务公司(Amazon Web Services，Inc .)通过技术、渠道和员工多样性的整合所建立的关系。

趋势科技首席运营官 Kevin Simzer 表示:“与 AWS 的紧密结合促使企业越来越多地选择趋势科技来保护混合云环境。“我们自豪地报告，上一季度我们所有平台的云业务同比增长 69%，这让我们能够实现保护超过 300 万个混合云工作负载的目标。”

“趋势科技迅速利用 AWS 市场中的新功能来简化和加速安全的客户应用程序迁移。亚马逊网络服务公司 AWS Marketplace 服务目录和迁移服务副总裁 Dave McCann 表示:“我们有一个共同的愿景，即通过 AWS Marketplace 为容器创新软件采购自动化。AWS Marketplace 上有超过 200，000 名活跃买家，我们将继续致力于让组织更容易找到可以帮助他们在 AWS 上创新并满足其不断增长的业务需求的解决方案。”

趋势科技是 AWS 合作伙伴网络(APN)在 AWS re:Invent 2018 上启动的以下计划的启动合作伙伴:

**技术方案**

*   **AWS 容器能力:**趋势科技已达到 AWS 容器能力状态。标准包括拥有两个强有力的客户证明资料，满足一系列严格的要求，共同巩固趋势科技在集装箱安全领域的领导地位，并证明在 AWS 客户中的成功。趋势科技 20%的企业混合云安全客户已经使用该公司的技术保护了他们的容器，这一点得到了验证。
*   **AWS 容器市场:** AWS 宣布支持基于 Docker 容器的软件产品。作为 APN 高级技术合作伙伴，趋势科技正在通过其自动构建时间和容器映像注册扫描来扩展其在 AWS 市场中的解决方案。这将在部署之前检测漏洞和恶意软件，帮助客户在 CI/CD 管道中尽早降低威胁和风险，实现无摩擦安全。
*   **AWS 安全中心:**趋势科技现在可以向新的 AWS 安全中心发送关键安全事件。AWS Security Hub 旨在通过对来自多个 AWS 服务(如 Amazon GuardDuty、Amazon Inspector 和 Amazon Macie)以及其他 APN 安全解决方案的警报或发现进行聚合、组织和优先排序，为用户提供高优先级安全警报和合规状态的全面视图。然后，调查结果会在集成的仪表盘上用可操作的图形和表格进行直观总结。这将允许趋势科技用户获得更强的安全态势，同时事半功倍。

**频道节目**

*   **AWS Marketplace 咨询合作伙伴私人优惠:**趋势科技是一项新计划的 APN 启动合作伙伴，该计划将使其渠道合作伙伴受益。该计划允许 AWS Marketplace 客户和 AWS Marketplace ISV 协商软件购买的定制价格和最终用户许可协议(EULA)条款。结果将有助于渠道加速云迁移，并简化技术采购和部署。

**多元化计划**

*   **AWS 支持的趋势科技性别多元化计划:**趋势科技通过发起一项活动来缩小 IT 和网络安全人员中的性别差距，从而扩大了其多元化计划。这个*缩小差距*的倡议得到了 AWS 的支持，AWS 共同主办了 30 名年轻女性的技术女孩，否则她们可能无法参加 re:Invent 2018。趋势科技赞助了许多其他活动，包括 AWS 多样性跟踪招待会。

自 2012 年以来，趋势科技一直致力于保护客户的云环境。这种领导地位导致了与 AWS 的密切关系，趋势科技成为 APN 高级技术合作伙伴、 [AWS 安全能力合作伙伴](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Faws.amazon.com%2Fsecurity%2Fpartner-solutions%2F&esheet=51723412&newsitemid=20171130006034&lan=en-US&anchor=AWS+Security+Competency+Partner&index=4&md5=6c79b7127b72b7cfd1cbd0c83ca5955f)、AWS 软件即服务(SaaS)合作伙伴、AWS 政府能力合作伙伴以及 [AWS 市场销售商](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Faws.amazon.com%2Fmarketplace%2Fseller-profile%3Fid%3D934af31d-efde-4b34-bc44-2d477620a0c8&esheet=51723412&newsitemid=20171130006034&lan=en-US&anchor=AWS+Marketplace+seller&index=5&md5=89b9dac478c99af74b6485b0a6515539)就证明了这一点。
