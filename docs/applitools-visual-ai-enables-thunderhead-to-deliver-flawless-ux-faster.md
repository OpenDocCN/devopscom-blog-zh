了解客户的旅程对成功至关重要，但管理起来极其复杂和具有挑战性。从橱窗购物者到移动浏览器，有几十个甚至更多的客户接触点可以积累，这些接触点中的任何一个都是转化潜在客户的机会。这就是 Thunderhead 的 ONE Engagement Hub 为用户提供业务蓬勃发展所需的客户行为洞察和背景的地方。至关重要的是，Thunderhead 持续向用户提供完美的体验，使他们不会错过转换的机会。

Thunderhead 质量负责人 Walt Harris 表示:“Applitools 让我们整个团队高枕无忧。“我们知道我们代码的视觉效果。使用 Applitools 让我们更加敏捷，让我们开发得更快。”

在 Applitools 之前，Thunderhead 的工程团队编写了 Selenium 测试来验证 UI 行为和端到端功能。他们没有这些测试的可视化组件的自动化。为了通过测试，对 UI 代码的小改动将需要大量的重写，并且对内部元素定位器的许多改动没有视觉影响。

通过检查视觉元素，Applitools Visual AI 可以检测实际的 bug，忽略微小的视觉差异。使 Applitools 在浏览器和操作系统变化时更加准确，而不会产生耗时的误报。在实现 Applitools 之后，Thunderhead 发现了他们以前从未体验过的测试开发的简易性。用于验证屏幕内容的代码被一个对 Applitools 的简单调用所取代，appli tools 捕获整个屏幕并将其与上一次批准的版本进行比较。通过人工智能，有意义的差异被突出显示，因此工程师可以轻松地修复或批准。

**Thunderhead 实现了以下结果:**

*   使用 Applitools Ultrafast 测试云，构建速度更快，测试执行时间缩短 48 倍，从 4 小时缩短到 5 分钟
*   节省时间，编写和维护的代码更少
*   通过更早的测试提高开发人员的生产力，消除耗时的上下文切换
*   测试覆盖率大幅增加，让您高枕无忧
*   通过让开发人员将时间集中在创建更多功能和为用户提供更多价值上，加快上市时间
