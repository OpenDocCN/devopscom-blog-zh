# Prometheus Associate 认证将证明监控基础设施的能力

> 原文:[https://devo PS . com/Prometheus-associate-certification-will-demo-capability-to-monitor-infra structure/](https://devops.com/prometheus-associate-certification-will-demonstrate-ability-to-monitor-infrastructure/)

*来自 CNCF 和 Linux 基金会的新助理认证考试将使用开源系统监控和警报工具包 Prometheus 测试基础知识和技能*

**西班牙巴伦西亚，kube con+CloudNativeCon Europe–2022 年 5 月 18 日—**[为云原生 软件构建可持续生态系统的云原生计算基金会(CNCF )](https://www.cncf.io/) ，以及[Linux 基金会](https://www.linuxfoundation.org/) 、 通过开源实现大众创新的非营利组织 ，今天宣布了新的普罗米修斯认证协会(PCAPCA 是专为对 Prometheus 生态系统中的可观察性和监控有特殊兴趣的工程师或应用程序开发人员设计的职前认证。

PCA 以朝鲜中央通讯社、CKA 或 CKAD 等 Kubernetes 认证为蓝本，面向通过这些认证或完成 Prometheus 特定培训或云工程师训练营的候选人。该考试目前正在测试中，将于 2022 年晚些时候正式推出。

据 CNCF[2021 年云原生调查](https://www.cncf.io/wp-content/uploads/2022/02/CNCF-AR_FINAL-edits-15.2.21.pdf) 显示，普罗米修斯在生产中的使用已经达到 65%，同比增长 43%。生产用户包括阿迪达斯、Slack、摩根大通、威瑞森和优步等组织。普罗米修斯是 2016 年加入 CNCF 的第二个项目，也是 2018 年毕业的第二个项目，仅次于 Kubernetes。

CNCF 首席技术官 Chris Aniszczyk 表示:“随着云原生堆栈的发展和成熟，可观察性对于应用指标、性能、警报和洞察力的可见性变得至关重要。“普罗米修斯是其中的一个关键因素，它使组织能够建立可观察性的基础。PCA 考试将允许开发人员展示他们在普罗米修斯生态系统和云原生可观测性概念方面的熟练程度。”

PCA 将包括一个多项选择的在线认证考试，测试围绕监控云原生应用和基础架构的最佳实践的入门级技能和知识，以及通过可观察性数据提高应用性能的能力。成功的候选人将能够识别实施中的问题，监控并洞察性能，调试问题，并将数据输入其他系统，包括 QA、安全和自动化。现有的“ [监控系统和服务同普罗米修斯的](https://training.linuxfoundation.org/training/monitoring-systems-and-services-with-prometheus-lfs241/?utm_source=lftraining&utm_medium=pr&utm_campaign=pca) ”培训课程可以对那些新到普罗米修斯的人有所帮助。

“云领域一直缺乏人才，解决这个问题的唯一方法是让更多的人接受这些新兴技术的培训和认证，”SVP 的 Clyde Seepersad 说道，他是 Linux 基金会培训认证总经理&。“我们已经看到人们对 Prometheus training 的极大兴趣，因为这种有影响力的技术越来越多地被采用，这是一个很好的起点。这项新的认证将允许掌握这项技术的个人向雇主证明他们的能力，帮助他们推进职业发展，同时开始解决这一人才缺口。”

该考试是与 Certiverse 合作开发的。正如 Linux 基金会认证 IT 助理(LFCA)和 Kubernetes 以及云原生助理(朝鲜中央通讯社)考试所示，该公司的在线考试开发平台以保持 IT 内容最新和相关所需的速度将高质量的认证推向市场。

更多关于考试的信息和考试主题可以在这里找到。有兴趣注册测试版的人可以在这里免费注册。关于考试可用性的更多细节将在未来几个月内公布。

**关于云原生计算基础**

云原生计算使组织能够在公共、私有和混合云中使用开源软件堆栈构建和运行可扩展的应用。云本地计算基金会(CNCF)托管全球技术基础设施的关键组件，包括 Kubernetes、Prometheus 和 Envoy。CNCF 汇集了业界顶尖的开发人员、最终用户和供应商，并举办世界上最大的开源开发人员会议。由 500 多个成员支持，包括世界上最大的云计算和软件公司，以及 200 多个创新创业公司，CNCF 是非营利 Linux 基金会的一部分。更多信息请访问[www . cncf . io](https://www.cncf.io/)。

**关于 Linux 基金会**

Linux 基金会成立于 2000 年，由 1000 多名成员支持，是世界领先的开源软件、开放标准、开放数据和开放硬件协作中心。Linux 基金会的项目对世界基础设施至关重要，包括 Linux、Kubernetes、Node.js 等等。Linux 基金会的方法侧重于利用最佳实践并满足贡献者、用户和解决方案提供商的需求，以创建开放协作的可持续模式。如需了解更多信息，请访问 linuxfoundation.org[](https://www.linuxfoundation.org/)。
