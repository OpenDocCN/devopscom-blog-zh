# CLOUDBEES 与 JENKINS 和 JENKINS X 项目、LINUX 基金会、GOOGLE 合作推出持续交付基金会

> 原文:[https://devo PS . com/cloud bees-works-with-Jenkins-and-Jenkins-x-projects-Linux-foundation-Google-to-launch-continuous-delivery-foundation/](https://devops.com/cloudbees-works-with-jenkins-and-jenkins-x-projects-linux-foundation-google-to-launch-continuous-delivery-foundation/)

*行业组织为开发和推广开源 CD 相关项目奠定基础*

**英国伦敦****–2019 年 3 月 12 日下午 4:05—**[cloud bees](https://www.cloudbees.com)，为持续经济提供动力的企业 DevOps 领导者，今天宣布启动 [持续交付基金会](https://cd.foundation) (CD 基金会或 CDF)，这是一个将在 Linux 基金会旗下运营的新基金会。CloudBees 与 Jenkins 社区、谷歌和 Linux 基金会合作，领导了发布活动。

CDF 将专注于开发、培育和推广与持续交付相关的开源项目、最佳实践和行业规范。CDF 将容纳各种开源项目，包括 [【詹金斯】](https://jenkins.io)[詹金斯 X](https://jenkins-x.io)[Spinnaker](https://www.spinnaker.io/)和 [Tekton](https://github.com/tektoncd) 。预计会有更多的项目加入进来，目标是将一个连续交付(CD)生态系统整合在一起，围绕可移植性和互操作性构建规范和项目。CloudBees 将作为创始成员参与，并将持续为该项目贡献技术和资源。

CloudBees 的首席技术官兼 Jenkins 的创始人 Kohsuke Kawaguchi 表示:“现在是一个强大的、供应商中立的组织致力于推进持续交付的时候了。“CDF 提供了一个机会来提高技术人员以外的人对 CD 的认识。对于像 Jenkins 和 Jenkins X 这样的项目，它代表了一个全新的成熟度级别。我们期待着帮助 CDF 发展 CD 生态系统，并促进顶级开发商、最终用户和供应商之间的合作。”

CDF 将为其他商业公司创造一种方式，投资于像 Jenkins 这样的开源项目的发展和成功，帮助扩展该项目及其周围的生态系统。它还将为顶级开发者、最终用户和供应商提供倡导开源 CD 解决方案、分享想法和在 DevOps 领域推广开放标准的能力。

CloudBees 首席执行官兼联合创始人 Sacha Labourey 表示，将 Jenkins 项目转移到 CDF 增强了 CloudBees 对 Jenkins 社区的长期支持。“我们相信这是詹金斯和詹金斯 X 项目以及整个行业的正确道路，”Labourey 说。 **"** 迄今为止，CloudBees 一直是 Jenkins 所有活动的主要支持者，而且这种支持将会毫不含糊地继续下去。然而，Jenkins 加入 CDF 意味着该项目将会发展得更快，有更多的贡献者支持它。CDF 是一个 CloudBees 驱动的计划，我们认为它对推进 CD 至关重要。这样做对我们大家都有好处。”

CDF 将由 Linux 基金会提供人员和运营。成员预计将包括来自许多不同行业的跨部门云基础架构提供商、DevOps 供应商、企业软件提供商、系统集成商和最终用户。

“随着市场转向容器化和云原生技术，CI/CD 系统、DevOps 和相关工具的生态系统发生了根本性的变化。Linux 基金会开发人员关系副总裁 Chris Aniszczyk 说:“可用工具的数量增加了，但没有关于管道和其他 CI/CD 工具的定义性行业规范。“CloudBees、Google 和其他 CDF 创始成员认识到需要一个中立的协作和整合之家来解决这个问题。CDF 将建立一个社区来推进围绕 CI/CD 的行业最佳实践和创新。”

**关于 CloudBees**

CloudBees 通过构建世界上第一个自动化软件交付的端到端系统 CloudBees Suite，为持续经济提供动力。CloudBees 套件基于新兴的 DevOps 实践以及持续集成(CI)和持续交付(CD)自动化，增加了实现最佳效率和控制新风险所需的治理、可见性和洞察力层。由于现在世界上的每个公司都是软件公司，这种新的自动化软件交付系统正在成为现代企业中最关键的业务系统。作为当今 CI/CD 领域的明确领导者，CloudBees 在定义和引领这一新类别方面具有独特的优势。CloudBees 让公司以最快的速度将伟大的想法转化为伟大的软件，并更快地为企业带来价值。
