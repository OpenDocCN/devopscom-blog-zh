# VictorOps 宣布推出新的企业和标准产品

> 原文:[https://devo PS . com/victorops-announces-new-enterprise-standard-product-offerings/](https://devops.com/victorops-announces-new-enterprise-standard-product-offerings/)

*任何规模的组织现在都可以从 VictorOps 的高级定制事件管理方法中受益*

科罗拉多州博尔德([商业资讯](http://www.businesswire.com/))–[针对开发团队的实时事件管理公司 VictorOps](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fvictorops.com%2F&esheet=51261342&newsitemid=20160119005396&lan=en-US&anchor=VictorOps&index=1&md5=5d3694195c4432d24413fb53b4ab2c52) 今天宣布推出 VictorOps Standard 和 VictorOps Enterprise，以支持各种规模的组织。新产品为随叫随到团队提供了一个虚拟环境，在这个环境中，他们可以快速、无缝地解决事件，而不受位置或设备的限制。与目前市场上的其他产品不同，这两种产品都支持整个 DevOps 事件生命周期，允许团队对每个事件进行准备、响应和恢复。

> “这两种产品都提供了基本的警报和协作解决问题的功能，使随叫随到的团队从第一次警报到最后的回顾整个过程变得更加容易。但是，我们的企业级专门满足拥有大型待命团队的公司的需求，这些团队需要高级事件路由和自动化文档记录。”

VictorOps 产品副总裁 Joni Klippert 表示:“我们的客户群正朝着大规模企业推广的方向发展，这通常涉及到他们软件团队的很大一部分。“这两种产品都提供了基本的警报和协作解决问题的功能，使随叫随到的团队从第一次警报到最后的回顾整个过程变得更加容易。但是，我们的企业级专门满足拥有大型待命团队的公司的需求，这些团队需要高级事件路由和自动化文档记录。”

除了 VictorOps Standard 中包含的所有功能之外，VictorOps Enterprise 的功能还包括:

*   **企业集成**—除了 VictorOps 标准中提供的 50 多种集成，VictorOps 企业用户还可以访问与 [StatusPage.io](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fvictorops.com%2Fblog%2Fstatuspage-integration%2F&esheet=51261342&newsitemid=20160119005396&lan=en-US&anchor=StatusPage.io&index=3&md5=0dd42513d512079604b1955fbcd6fd17) 和 [ServiceNow](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fvictorops.force.com%2Fknowledgebase%2Farticles%2FIntegration%2FServiceNow-Integration%2F&esheet=51261342&newsitemid=20160119005396&lan=en-US&anchor=ServiceNow&index=4&md5=6e93c1c0887cf000aa77fa8d8f2df964) 的集成，前者旨在向内部利益相关方通报 IT/Ops 问题，后者允许与传统事件管理和报告实践进行集成。
*   **单点登录(SSO)**–该功能可为大型组织提供更高效的用户管理。
*   **内置语音会议**–用户只需点击几下鼠标，就可以打开一座桥梁，与应用程序中所有合适的人交谈，同时捕获呼叫信息，用于 SLA 和事后报告。
*   **无限制的 API 调用**–用户可以无限制地访问 VictorOps 新发布的[公共 API](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fvictorops.com%2Fblog%2Fpublic-api-launch%2F&esheet=51261342&newsitemid=20160119005396&lan=en-US&anchor=public+API&index=5&md5=fe914564a771b33e69398e09829aa78f) 端点，为他们的组织构建定制解决方案。VictorOps Standard 包括每月 500 次 API 调用。
*   **高级报告**–除了 VictorOps 标准中包含的事后报告之外，VictorOps Enterprise 用户还可以访问高级用户指标和事故频率报告，详细说明他们最嘈杂的警报。
*   **卓越成功管理**–victor ops 致力于为所有客户提供卓越的支持。那些利用企业解决方案的客户将获得额外的支持，以确保产品高级功能的推出和利用是无缝的。

“在过去的一年里，我们的客户群在这些功能的开发和验证中发挥了重要作用，”Klippert 说。因此，我们决定免费将所有现有客户升级到我们的顶级服务 VictorOps Enterprise
