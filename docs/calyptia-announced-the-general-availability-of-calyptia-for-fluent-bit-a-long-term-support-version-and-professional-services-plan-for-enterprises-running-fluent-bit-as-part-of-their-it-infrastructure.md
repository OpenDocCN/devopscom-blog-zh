# Calyptia 宣布正式推出适用于 Fluent Bit 的 Calyptia，这是一个长期支持版本和专业服务计划，面向将 Fluent Bit 作为其 it 基础架构一部分运行的企业

> 原文：<https://devops.com/calyptia-announced-the-general-availability-of-calyptia-for-fluent-bit-a-long-term-support-version-and-professional-services-plan-for-enterprises-running-fluent-bit-as-part-of-their-it-infrastructure/>

**加利福尼亚州三藩市，2022 年 5 月 16 日—**[Calyptia]([https://calyptia.com](https://calyptia.com))今天宣布正式推出适用于 Fluent Bit 的 Calyptia，这是一个长期支持版本和专业服务计划，面向将 Fluent Bit 作为其 it 基础设施一部分运行的企业。Calyptia Fluent Bit 客户在运行标准化、安全强化、稳定版本的 Fluent Bit 的同时，可在规定的时间内获得安全更新、补丁和错误修复以及基本支持。

Calyptia 的独特定位是为 Fluent Bit 用户提供长期支持，因为它是 Fluent Bit 的创始人之一，并且该公司是 Fluent Bit 项目的主要维护者。

Fluent Bit 是用于日志和指标的企业可观察性的行业标准开源引擎，它包含在主要的 Kubernetes 发行版中，包括 Google Kubernetes 引擎(GKE)、Amazon Elastic Kubernetes 服务(EKS)和 Azure Kubernetes 服务(AKS)。它收集和处理来自任何来源的事件数据，并将这些数据发送到任何目的地(甚至多个目的地)进行存储和分析。它是云和容器化环境的首选，最近部署量超过 10 亿次。

由于其高性能和轻量级占用空间，Fluent Bit 在可观测性环境中越来越成为关键任务，允许其每天处理数 Pb 的数据，同时消耗很少的资源。Fluent Bit 还对与之交互的系统采取厂商中立的方法，帮助企业避免昂贵的厂商锁定解决方案。

开源软件提供创新、功能和快速；然而，这些同样的优势也给努力跟上时代的企业带来了挑战。Calyptia Fluent Bit 通过将创新与稳定性、支持和强化安全性相结合，为需要可预测升级和 SLA 的企业解决了这些挑战。

Calyptia 联合创始人、Fluent Bit 的创始人爱德华多·席尔瓦(Eduardo Silva)表示:“已经有几家公司与我们接洽，希望提供支持服务，所以肯定有这种需求。我们对他们的要求考虑得越多，我们就意识到 Calyptia Fluent Bit 是我们当前和计划中的产品的合理扩展。随着组织采用分布式、动态、云原生服务，面临的挑战是如何快速高效地从整个企业中创建的大量日志和事件数据中获得洞察力。Calyptia 致力于为这一挑战提供解决方案，而 Calypia Fluent Bit 是一些组织的理想选择。”

上个月，领先的技术研究和咨询公司 Gartner 将 Calyptia 评为监控和可观测性解决方案的“酷供应商”，这是一个快速增长的行业，促使 Gartner 分析师写道，“这一趋势可能会导致 21 世纪 20 年代被称为‘可观测性的十年’。”"
