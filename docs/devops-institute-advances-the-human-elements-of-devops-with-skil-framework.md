# DevOps 研究所用 SKIL 框架推进 DevOps 的人的因素

> 原文：<https://devops.com/devops-institute-advances-the-human-elements-of-devops-with-skil-framework/>

*DevOps Institute 提供所需的技能、知识、想法和学习，以确保 DevOps 的人力要素与新兴流程和技术一起进步*

DEVOPS 企业峰会–伦敦–2019 年 6 月 25 日–DevOps 学院([https://devopsinstitute.com/](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=I%2BqMhXDuIS2RujNbavWSOKShOr7Ezi73JeGvxlkJ09Qb0kDpZZQFeERc9Ftx%2Fbg4ZpvJNGn%2Fi5y4vLBcRmoJI2KD%2F3dSZo2S1P%2FzHqS2Uzisip4yZCmmoA%3D%3D&G=0&R=https%3A%2F%2Fc212.net%2Fc%2Flink%2F%3Ft%3D0%26l%3Den%26o%3D2426491-1%26h%3D3734902455%26u%3Dhttps%253A%252F%252Fdevopsinstitute.com%252F%26a%3Dhttps%253A%252F%252Fdevopsinstitute.com%252F&I=20190625110031.000000a827da%40mail6-46-ussnn1&X=MHwxMDQ2NzU4OjVkMTE1ODVjOTUyZjZhNDQ4YmExODg2Yjs%3D&S=V8mxWIMe48JhC0rayBYvcGN2mXCujpW607XFgqmJi-s))今天宣布了推进 devo PS 人力要素的 SKIL 框架。SKIL 框架包括 1) **技能**来发展职业成长；2) **知识**为 DevOps 理解；3) **激发创新的创意**，以及 4) **基于 DevOps 专业知识的学习**。“SKIL”的支柱代表了一个知识、持续学习和认证的整体框架，以推进职业生涯、支持 IT 转型并满足 DevOps 社区的需求。

DevOps Institute 的首席执行官 Jayne Groll 解释说:“我们相信，DevOps 的成功，对于个人、组织和整个行业来说，不仅是基于技术，也是基于人类技能。

**“SKIL****框架**代表了我们知识和学习的整体生态系统，随着我们开发和策划行业最佳思想并将其提供给 DevOps 社区，该生态系统将继续发展。对 DevOps 技能和知识的渴求非常强烈。”

DevOps Institute 与从业者、行业领导者和合作伙伴一起，建立了一个蓬勃发展的社区，成员可以访问最具创新性、启发性和变革性的 DevOps 内容和认证。它专注于基础和新兴的 DevOps 实践和原则，提供了促进职业生涯的内容和学习，并激发了 DevOps 成功方程式中人的因素的讨论、协作和转变。

作为一个成立于 2015 年的全球会员制协会，DevOps Institute 是一个开放且不断发展的持续学习社区的统一力量，该社区由 IT 从业者、教育合作伙伴、顾问、人才获取和业务高管组成。它通过认证、研究、活动和持续学习计划帮助发展职业生涯，并支持 DevOps 社区内的新兴实践。它的主要焦点是提供“人类知识”来使其现代化。

**《SKIL 框架》包括四个关键要素，旨在促进人类发展:**

**技能** : DevOps Institute 通过行业思想领袖策划的企业级和基于角色的认证和认证，帮助数千名 IT 从业者和专业人士提升了技能。它严格研究和策划开源价值观，以帮助成员建立和导航职业道路，并利用独家工作板和机会。

认证包括:DevOps 基金会、认证敏捷服务经理(CASM)、认证敏捷流程负责人(CAPO)、DevOps 测试工程(DTE)℠)、持续交付架构(CDA)℠)、DevOps Leader (DOL)℠)和 DevSecOps 工程(DSOE)℠，是 DevOps Institute，Inc .的商标。

**知识** : DevOps Institute 通过研究和高端思想领导内容、视频、网络研讨会、播客、电子书、博客和发表的文章来帮助指导组织和成员。对于所有对利用基于社区的原则来提高个人和组织绩效感兴趣的 It 专业人员来说，这是一个值得信赖的资源。DevOps Institute 领导了“2019 年技能提升:企业 DevOps 技能报告”(【https://info.devopsinstitute.com/upskilling-report-2019】T4)的研究和分析，并将推动 DevOps Institute 未来 2020 年技能提升报告的整体研究战略。要访问资源，请访问([https://devopsinstitute.com/resources/](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=I%2BqMhXDuIS2RujNbavWSOKShOr7Ezi73JeGvxlkJ09Qb0kDpZZQFeERc9Ftx%2Fbg4ZpvJNGn%2Fi5y4vLBcRmoJI2KD%2F3dSZo2S1P%2FzHqS2Uzisip4yZCmmoA%3D%3D&G=0&R=https%3A%2F%2Fdevopsinstitute.com%2Fresources%2F&I=20190625110031.000000a827da%40mail6-46-ussnn1&X=MHwxMDQ2NzU4OjVkMTE1ODVjOTUyZjZhNDQ4YmExODg2Yjs%3D&S=YUhHJ1vdbdRJTTrOKpkYI6mRhM4_tgkLlySB-tGRC20))。

**创意** : DevOps 学院促进对话和探索活动，阐明应用 DevOps 技能的好处。它组织创意黑客马拉松、点对点活动和众包，以挖掘 DevOps 中的下一个“大创意”。会员可以访问相关论坛、现场&虚拟活动和本地聚会，建立关系网，培养成功理念。DevOps Institute 正在通过其全球大使计划推出当地的 SKIL 活动，并将于 2019 年 10 月举办全球 SKIL 日。更多信息，请访问(【https://devopsinstitute.com/devops-institute-skilup/】T4)。
