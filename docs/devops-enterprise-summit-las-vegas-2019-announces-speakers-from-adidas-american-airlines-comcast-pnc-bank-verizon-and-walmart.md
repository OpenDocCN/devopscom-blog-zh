# DevOps 企业峰会 2019 宣布来自阿迪达斯、美国航空公司、康卡斯特、PNC 银行、威瑞森和沃尔玛的演讲人

DevOps 企业峰会于 10 月 28 日至 30 日在拉斯维加斯的 Cosmopolitan 举办，是一个面向实施 DevOps 原则和实践的大型复杂组织的业务和技术领导者的会议。它的目标是为这些领导者提供他们需要的工具和实践，以更快地开发和部署软件，并在市场中取胜。

“我参加 DevOps 企业峰会，因为这是唯一一次超越工具和技术，深入探讨人员、文化、领导力、协作和整体数字化转型的 DevOps 会议，”Capital One 高级总监兼高级杰出工程师 Tapabrata Pal 表示。"我可以听到关于成功和失败的真实故事来学习."

“多年来，我们的主要编程目标之一是收集经验报告，展示商业和技术领导之间的最佳工作关系，”Gene Kim 说，他是 IT Revolution 的创始人，也是即将出版的书籍 *The Unicorn Project* 和 *The Phoenix Project* 、 *The DevOps Handbook* 和 *Accelerate* 的合著者。“通过让这些领导者分享他们的经验报告，我们希望其他人能够向他们学习并效仿他们，以取得类似的成果；这有助于他们自己的组织赢得市场。”

DevOps 企业峰会在美国和欧洲举办活动，提供主题演讲、小组讨论、分组讨论、DevOps 研讨会、演讲人问答会议、图书签售会和展厅。会议将由在各行各业的大型复杂组织中开拓这些实践的领导者领导。主题演讲将邀请行业名人以及来自知名公司的演讲者，他们将分享从企业开发运维计划中获得的经验。
