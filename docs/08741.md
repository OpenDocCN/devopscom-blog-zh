# DevOps 的未来是什么样子的？

> 原文:[https://devo PS . com/what-the-future-of-devo PS-looking/](https://devops.com/what-does-the-future-of-devops-look-like/)

在过去几年中，我们目睹了数字化转型的快速发展。许多企业一直在利用 DevOps 和云的优势来加速他们的数字化转型之旅。随着行业中如此多的变化，是时候展望一下 DevOps 正在成为什么，以及 DevOps 团队如何在整个生命周期中管理变化了。

10 月 27 日，DeployHub 首席执行官兼 CDF 董事会成员特雷西·拉冈将在她的“明天的 DevOps 管道-彻底的转变”会议期间，在 [DevOps 体验:咆哮的 2020s 虚拟峰会](https://www.techstrongevents.com/devopsexperience)上讨论 DevOps 的未来和 GitOps 的融合、基于事件的连续交付以及微服务目录的未来使用

我们与特雷西·拉冈坐下来讨论了 DevOps 社区一直面临的一些挑战，如何克服这些挑战以及 DevOps 的角色如何变化。观看下面的采访，先睹为快她即将在 DevOps 体验会议。

今年的 [DevOps 体验](https://www.techstrongevents.com/devopsexperience/home)探索了 DevOps 和云如何重塑数字化转型之路。为期两天的虚拟活动的议程包括各种主题，包括技术会议、深入探讨和案例研究，涵盖数字化转型、DevSecOps、云原生、Kubernetes、AppSec、CI/CD、开源、价值流管理等。虚拟环境的特色是行业领袖的 50 多场会议和一个展厅，展厅里摆满了可供下载的资源。
