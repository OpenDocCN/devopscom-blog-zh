# 调查发现替代云服务提供商正在崛起

> 原文:[https://devo PS . com/survey-sees-alternative-cloud-service-providers-gaing-ground/](https://devops.com/survey-sees-alternative-cloud-service-providers-gaining-ground/)

DevOps.com 的姐妹实体 [Techstrong Research](https://techstrongresearch.com) 对 458 名开发专业人员、经理和高级领导进行的[调查](https://resources.techstrongresearch.com/devops-and-the-public-cloud-new-ways-to-pay-for-the-public-cloud)发现，43%的人正在考虑在未来 12 个月内增加更多的[云服务提供商](https://devops.com/?s=cloud+service+providers)。总体而言，近三分之二的受访者表示，他们至少在考虑、评估或准备从值得信赖的替代云供应商处购买，20%的受访者表示他们已经签约。

这项调查是代表提供云服务的 Akamai Technologies 旗下的 Linode 进行的，调查发现，整整 93%的受访者已经在使用亚马逊网络服务(AWS)、微软 Azure 或谷歌云计算平台(GCP)。近三分之二(65%)的受访者表示他们已经使用了不止一家云服务提供商。超过四分之一(28%)的人表示，他们正在使用替代提供商来增加他们从 AWS、微软、谷歌、甲骨文、IBM Rackspace 或阿里巴巴消费的服务。19%的人表示他们正在使用超大规模和替代云服务提供商。

受访者提到的使用替代提供商的最大驱动因素是减少对单一提供商的依赖(55%)、提高性价比(38%)和最近的云中断(31%)。

受访者还表示，他们担心大型供应商有一天会在他们将业务扩展到其他行业领域时与他们竞争(24%)，或者可能参与会造成道德冲突的商业行为(19%)。四分之一(25%)的受访者表示，他们已经将云服务提供商视为竞争对手。

Techstrong Research 的首席分析师丹·基尔希(Dan Kirsch)表示，随着云计算的不断成熟，越来越多的组织开始考虑他们在消费商用计算、存储和网络服务方面的选择。他指出，许多组织根本不需要访问复杂的云服务。共有 42%的受访者表示，提供核心基础设施 primatives 的云服务提供商可以处理 90%的典型工作负载。Kirsch 指出，在许多情况下，替代的云服务提供商被用来扩大而不是取代超大规模的云服务提供商。

总体而言，近四分之三的受访者(74%)表示，他们预计到今年年底，他们的基础架构将基于云。近四分之三的受访者直接从基础设施提供商处购买云服务，其余的受访者雇佣托管服务提供商或其他类型的第三方云服务经销商。

根据这项研究，超过三分之一的受访者还报告说[使用了一种支付机制](https://devops.com/webinars/getting-paid-why-cloud-providers-are-changing-how-customers-use-and-pay-for-services/)而不是合同或招标书(RFP)。在替代云提供商的客户中，超过三分之二(67%)的人使用其他支付机制，四分之一的人使用加密货币支付。Kirsch 说，这种选择正变得越来越重要，因为越来越多的组织正在寻找通过采用替代支付来降低成本的方法。

他补充说，在其他情况下，IT 专业人员正在寻求绕过繁琐的购买流程的最终途径，或者只是出于个人偏好而选择使用替代支付方式。Kirsch 指出，一般来说，替代供应商对处理这些类型的支付更加开放。

自新冠肺炎疫情开始以来，向云的转变显然一直在加速。但是，随着形势的不断发展，随着云成为部署应用程序的首选平台，IT 组织现在有了前所未有的更多选择。
