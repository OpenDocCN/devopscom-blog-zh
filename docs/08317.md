# HCL 推出了云原生解决方案工厂 SoFy 和云原生即服务产品 HCL Now

> 原文:[https://devo PS . com/HCl-launchs-sofy-the-cloud-native-solution-factory-and-HCl-now-a-cloud-native-as-a-service-offering/](https://devops.com/hcl-launches-sofy-the-cloud-native-solution-factory-and-hcl-now-a-cloud-native-as-a-service-offering/)

**美国纽约/印度诺伊达，2021 年 7 月 20 日:**全球领先的技术公司 HCL Technologies(HCL)今天发布了云原生解决方案工厂 HCL SoFy 和云原生即服务产品 HCL Now。这两项新功能进一步增强了该公司的#HCLCloudSmart 产品。

SoFy 和 HCL Now 产品是 HCL 对其产品组合的发展和现代化进行重大投资的结果，也是其长期云原生承诺的结果。仅在去年，HCL 软件部门就发布了 22 个主要产品版本、10 个新产品模块、390 个产品版本和 900 多个增强功能。

与#HCLCloudSmart 之旅一致，HCL Software 的云原生战略将植根于最广泛采用的云原生技术的技术平台与允许每个客户以最符合其特定业务目标的方式采用云原生的选择和自由相结合。

HCL SoFy 是基础的云原生平台，允许客户在几分钟内将企业软件部署到他们选择的云上，没有供应商锁定，并可根据企业需求进行扩展。SoFy 的主要特性包括:

*   **触手可及的 HCL 软件:**轻松访问 50 个产品组件、2，000 多个 REST API 端点和云原生演示。
*   **一键式部署**:借助创新的 SoFy 平台，能够在几分钟内部署 HCL 软件。
*   **多种云、多种选择**:支持实施多种云策略，在任何云(公共云或私有云)上运行。

HCL 现在在客户选择的云平台上释放了 HCL 产品和服务能力的全部潜力。云原生即服务提供:

*   **云原生现在:**企业级可用性、无限的可扩展性和灵活性，以云原生的方式全新构建，并在选择的公共云上部署和管理专用的安全环境。
*   **按需专家:**在过渡到由我们的云和 HCL 产品专家提供支持和帮助的原生云时，降低风险和成本。
*   **事半功倍，无束缚:**软件和定制投资可移植到任何云，公共云或私有云和共享初始投资，以启动您的转型。

HCL 软件公司高级副总裁兼负责人 Darren Oberst 表示:“我们架构战略的基础——云原生和 API First——使我们的客户能够在我们的产品中快速创新新功能和组件——从自动化、开发运维、商业和营销、数字解决方案到安全性。“它为我们的客户提供了战略选择，以避免‘锁定’,并具有控制其数据、在不同工具之间混搭以及构建自己的解决方案架构的灵活性”，他补充道。

HCL SoFy 和 HCL Now 的云原生功能仅依赖于 Kubernetes，并且完全不受云的限制。这种灵活性允许在云提供商之间轻松迁移，提高业务灵活性，降低风险，并允许客户实施多云战略。

HCL 是云原生计算基金会(CNCF.io)和云原生基金会(CNF)的黄金会员，也是云铸造基金会(CFF)的董事会成员。

**关于 HCL 技术**

HCL Technologies (HCL)为全球企业提供未来十年的技术支持。HCL 的模式 1- 2-3 战略基于其深厚的行业专业知识、以客户为中心和 Ideapreneurship 的企业文化，使企业能够转型为下一代企业。

HCL 通过三个业务部门提供服务和产品:IT 和业务服务(ITBS)、工程和 R&D 服务(ers)以及产品和平台(P&P)。ITBS 通过在应用、基础设施、数字流程运营和下一代数字化转型解决方案领域的产品，帮助全球企业实现业务转型。ERS 在产品开发和平台工程的所有方面提供工程服务和解决方案。P&P 为全球客户提供现代化的软件产品，满足他们的技术和行业特定需求。通过其尖端的联合创新实验室、全球交付能力和广泛的全球网络，HCL 在各个行业垂直领域提供整体服务，分类为金融服务、制造、技术和服务、电信和媒体、零售和 CPG、生命科学和医疗保健以及公共服务。

作为一家领先的全球技术公司，HCL 以其多元化、社会责任、可持续发展和教育计划而自豪。在截至 2021 年 6 月 30 日的 12 个月中，HCL 的综合收入为 105.4 亿美元。其 176，499 名创意企业家在 50 个国家开展业务。
