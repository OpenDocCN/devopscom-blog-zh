# Applitools 将人工智能驱动的视觉测试引入 Selenium IDE

> 原文:[https://devo PS . com/appli tools-brings-ai-powered-visual-testing-to-selenium-ide/](https://devops.com/applitools-brings-ai-powered-visual-testing-to-selenium-ide/)

*人工智能增强了广泛使用的记录和剧本测试平台的视觉测试能力，帮助开发团队以无与伦比的速度和质量发布视觉上完美的 web 应用*

加州圣马特奥，2019 年 2 月 26 日—应用可视化管理的领导者 Applitools，今天宣布了 Selenium IDE([seleniumhq.org/selenium-ide](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=5eXH0qSKdBRphS6WO4YlQwwSTi8wtQOqejFjZKiddbCzZ5Bbl4XDYaFzCBlKjmMmr%2BznpL6dRwXOn3Kvl6joPQjhH9Iiu0KO5SQICz7DEajbk7VMODyXiw%3D%3D&G=0&R=https%3A%2F%2Fwww.seleniumhq.org%2Fselenium-ide%2F&I=20190226130740.000000045ac4%40mail6-43-usnbn1&X=MHwxMDQ2NzU4OjVjNzQzOWQ2MWNmNjEwYTZmNDM4OTZhNDs%3D&S=IMtl6uQrVCl38aMEXMJUAlh7a5nByHLJ8tH4RW-RYME))、 一款流行的免费开源记录和回放测试工具，及其 AI 驱动的可视化测试和监控平台 Applitools Eyes 。 这种集成使 QA 和开发团队能够在数十种浏览器和设备上使用可视化人工智能检查响应软件应用。Applitools Eyes 还提供了根本原因分析([【applitools.com/root-cause-analysis】](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=5eXH0qSKdBRphS6WO4YlQwwSTi8wtQOqejFjZKiddbCzZ5Bbl4XDYaFzCBlKjmMmr%2BznpL6dRwXOn3Kvl6joPQjhH9Iiu0KO5SQICz7DEajbk7VMODyXiw%3D%3D&G=0&R=https%3A%2F%2Fapplitools.com%2Froot-cause-analysis%3Futm_term%3Dcat%26utm_source%3Dpartner%26utm_medium%3Dpress-release%26utm_content%3Drca-page%26utm_campaign%3Dselenium-ide-campaign%26utm_subgroup%3D&I=20190226130740.000000045ac4%40mail6-43-usnbn1&X=MHwxMDQ2NzU4OjVjNzQzOWQ2MWNmNjEwYTZmNDM4OTZhNDs%3D&S=SnnRvKeXGMkEC0vGUQ0OeV9NV0BC_ztD8kaE0zO2Izw))来帮助开发者指出代码中错误的确切位置，并立即修复它们，而无需花费宝贵的时间进行调试工作。

Applitools Visual AI 分析每张截图，只找到人眼可检测到的视觉差异——消除了困扰传统基于像素的视觉测试工具并让前端开发人员感到沮丧的误报。通过根本原因分析，Applitools Eyes 识别出支撑每个视觉差异的少量 DOM 和 CSS 行，使开发人员能够将错误修复时间从几小时减少到几分钟。

Applitools 的联合创始人兼首席技术官 Adam Carmi 表示:“通过集成 Selenium IDE 和 Applitools Eyes，QA 团队(包括手动测试人员)现在有了一种简单而智能的方法来更快地发现更多的错误，并使与前端开发团队合作查明和调试代码行变得更加容易。“开发团队现在拥有 Selenium WebDriver 的大部分功能，而无需学习如何编写测试自动化代码。”

**关于 appli toolsT3】**

Applitools 是应用可视化管理(AVM)的创造者，从其可视化测试和监控解决方案开始，帮助公司以完全自动化的方式交付视觉完美的 web 和原生移动应用。Applitools 成立于 2013 年，使用复杂的人工智能图像处理技术来确保应用程序在所有设备、浏览器、操作系统和屏幕尺寸上正确显示和正常运行。

Applitools Visual AI 模仿人类的眼睛和大脑，以避免报告无法检测的视觉错误，最大限度地减少使传统视觉测试工具无法使用的误报。Applitools Eyes 可以即时验证整个应用程序页面，甚至检测最复杂和动态页面的布局问题。当开发团队确信他们可以更快地修复视觉缺陷时，他们就会被释放出来，提供比以前更多的面向客户的特性。
