# Island 在 B 轮融资中筹集了 1.15 亿美元，以支持企业浏览器的大规模创新

> 原文：<https://devops.com/sland-raises-115-million-in-series-b-funding-to-enable-massive-innovation-on-enterprise-browser/>

*由 Insight Partners 牵头，Stripes 和 Sequoia 参与，new capital 证实了 Island 的网络安全和员工生产力游戏规则改变者的成功和客户接受度*

**德克萨斯州达拉斯，2022 年 3 月 23 日—** [企业浏览器的创造者 Island](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.island.io%2F&esheet=52615840&newsitemid=20220323005242&lan=en-US&anchor=Island&index=1&md5=b46d21c4a29435173ebd5bc55f23b6c0) 今天宣布，它已在 B 轮融资中筹集了 1.15 亿美元，估值为 13 亿美元。这轮融资由总部位于纽约的全球风险投资和私募股权公司 Insight Partners 牵头，该公司主导了 Island 的上一轮融资。现有投资者 Stripes 和红杉资本(Sequoia Capital)也参与了这轮融资，使 Island 的总融资额超过 2 亿美元。

 这一轮是在 Island 于 2 月 1 日推出世界上第一款企业浏览器后开始的。通过将企业的核心需求构建到浏览器本身，Island Enterprise Browser 为组织提供了完全的可见性和最后一英里的控制，在安全性、IT 和生产力方面产生了前所未有的改进，同时提供了最终用户期望的基于 Chromium 的体验。

“Island 创造了一种全新的企业工作思维方式。通过从根本上将工作环境转变为设计安全，Island Enterprise Browser 使组织能够实现全新级别的安全性、生产力和 IT 效率。我们的客户和 IT 社区的热情非常高。”Island 首席执行官兼联合创始人迈克·菲说。“来自 Insight Partners 的新投资和来自原始投资合作伙伴的更多投资验证了我们的产品与市场的契合度，加速了我们的发展势头，并凸显了摆在我们面前的巨大机遇。”

“当我们开始构建企业浏览器时，我们的目标是从第一天起就为安全专业人士提供巨大的价值，”Island 的联合创始人兼首席技术官 Dan Amiga 说。“看到我们已经实现了这一目标，令人兴奋。但我们现在注意到的是，不仅在安全领域，而且在生产力和 IT 领域，都存在解决以前无法克服的问题的巨大机会。这笔新资金确保我们能够利用更多这种惊人的潜力。”

**企业浏览器**

企业浏览器是未来的桌面，使组织能够在用户和数据与 SaaS 和内部 web 应用程序交互时保护用户和数据。通过使用 Island Enterprise Browser，安全团队可以完全控制最后一英里，从复制、粘贴、下载、上传和屏幕截图等基本的渗透保护，到智能网络路由和多因素身份认证插入等更高级的安全需求。这为越来越多的企业使用案例带来了前所未有的机遇，包括保护关键 SaaS 和内部 web 应用免遭数据泄露、承包商和自带设备员工的安全访问，以及对特权用户帐户的完全监管。与成本高昂、性能不佳的虚拟桌面基础架构(VDI)相比，它还为混合工作者提供了本机用户体验，同时支持内置的安全浏览、网页过滤、网页隔离、漏洞防御和零信任网络访问，成本低得多，用户体验也显著改善。

**我们的投资伙伴**

Insight Partners 的联合创始人兼董事总经理 Jeff Horing 表示:“当一款革命性的产品满足真正的市场需求时，利用这一势头并继续扩大机会至关重要。“Island 有能力显著改变行业，此次资本注入将有助于推动新兴企业浏览器类别的持续领导和创新。”

Stripes 创始人兼合伙人 Ken Fox 表示:“Stripes 认为，令人惊叹的产品是每一个成功企业的基础，我们以此为核心寻求投资机会。“我们对这一领域的市场研究表明，孤岛企业浏览器类别有着真实的需求和强劲的客户反应，这自然会引导我们在这一重大机遇中大幅增加投资。”

“孤岛企业浏览器颠覆并从根本上改善网络安全和 IT 环境的潜力是巨大的。”红杉资本(Sequoia Capital)全球管理合伙人道格·莱昂(Doug Leone)说。
