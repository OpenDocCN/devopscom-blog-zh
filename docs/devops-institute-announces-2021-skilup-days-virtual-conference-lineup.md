# DevOps Institute 宣布 2021 SKILup Days 虚拟会议阵容

> 原文：<https://devops.com/devops-institute-announces-2021-skilup-days-virtual-conference-lineup/>

*DevOps Institute 扩展了其为期一天的成功会议计划，该计划重点关注由 Harness、GitLab、Epsagon、Instana、DataStax、Datadog 等组织赞助的 DevOps 实践“方法”*

佛罗里达州 BOCA 拉顿——2021 年 2 月 4 日——DevOps Institute 是一家致力于推进 devo PS 人性化元素的全球专业协会，今天宣布了面向 IT 社区所有成员(包括学生、从业者和决策者)的每月补充 SKILup 日阵容。SKILup Days 是一个为期一天的虚拟微型会议，专注于特定的行业实践，研究 IT 人员、流程和技术方面的“如何”观点。

SKILup Days 包含了会议中常见的所有元素，包括带有虚拟赞助商展位的展厅、抽奖和寻宝游戏、与会者聊天以及与演讲者互动的网络休息室。
