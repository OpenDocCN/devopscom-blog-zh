# InfluxData 向公众开放 InfluxDB Cloud 2.0 测试版

> 原文:[https://devo PS . com/influx data-opens-influx db-cloud-2-0-to-public-beta/](https://devops.com/influxdata-opens-influxdb-cloud-2-0-to-public-beta/)

*新的统一平台实现了更深入的数据洞察、更高的灵活性、更快的入职速度和费率受限的自由层*

**旧金山-2019 年 5 月 7 日-**时间序列数据库 InfluxDB 的创建者 InfluxData 今天宣布了 InfluxDB Cloud 2.0 的公开测试版，这是该项目自 2013 年启动以来该平台最重大的演变。根据开源用户和客户的反馈，InfluxData 加强并简化了整个 InfluxDB 平台，以改善开发者和公司从时间序列数据集中处理和获取深刻见解的整体体验。InfluxData 邀请富有冒险精神的开发人员[加入测试版](https://www.influxdata.com/influxdb-cloud-2-0-beta/)并提供反馈，这将有助于塑造 InfluxDB 云的未来。

InfluxData 的联合创始人兼首席技术官 [Paul Dix](https://www.influxdata.com/blog/announcing-influxdb-cloud-2-0-beta-free-tier) 表示:“了解数据如何随着时间的推移而变化，对于创造可提供更大商业价值的可行见解至关重要。“这使得时间序列成为理解数据的最佳抽象，这是我们专门构建平台的动机。我们希望开发者社区能够加入我们的旅程，加入 InfluxDB Cloud 2.0 测试版计划。”

InfluxDB Cloud 2.0 的主要新功能包括:

*   **限速免费层:**专为入门者和业余爱好者设计，比下载开源产品更快入门。
*   **对新 Flux 语言支持的深入了解:** Flux 是一种功能强大的新脚本和查询语言，是第一种专门为时序数据构建的函数式语言。现在可以跨测量进行复杂的分析和数学计算。
*   **单一、统一的 API:**InfluxDB 中的一切(摄取、查询、存储和可视化)现在都可以通过统一的 API 访问，从而实现开源和云之间的无缝迁移。
*   **集成可视化和仪表板:**chrono graf 项目的开创性工作现已成为 InfluxDB 的一部分；实现增强的入门和用户体验，而无需依赖额外的安装。
*   **基于使用的定价:**新的定价模式提供了更大的灵活性，并确保客户只为他们需要的东西付费；它将根据数据需求自动调整项目。

InfluxData 拥有 500 多家客户和 200，000 多台运行 InfluxDB 的服务器，涵盖三个主要用例: [DevOps 监控](https://www.influxdata.com/customers/infrastructure-and-application-monitoring/)、[实时分析](https://www.influxdata.com/customers/real-time-analytics/)和[物联网监控](https://www.influxdata.com/customers/iot-data-platform/)。任何需要跟踪服务器、虚拟机、容器、应用程序、传感器、用户或事件的组织都可以从使用 InfluxDB 中受益。
