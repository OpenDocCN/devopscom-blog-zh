听说过[潘祖拉](http://www.panzura.com)吗？如果没有，你可能并不孤单。该公司在 DevOps 市场上相对较新。但这并不是一些新的创业公司通过使用 DevOps 作为流行语来获得一些风投资金:Panzura 已经存在多年，完善了其基于云的文件系统，可以在加快文件请求以及你必须传输的数据量方面提供令人瞠目结舌的结果。对于 DevOps 来说，这种速度节省太好了，不能错过。

我有机会与 Panzura 的 Rich Weber 和 Barry Phillips 坐下来讨论为什么 DevOps 团队发现 Panzura 的产品如此诱人。Barry 是 Panzura 的营销副总裁，但是不要被这个头衔欺骗了——Barry 是一个高度技术性的人。Rich 是 Panzura 的首席产品官，他也非常博学。我们一起就这个有趣的产品进行了很好的交谈，并讨论了为什么 DevOps 市场实际上是 Panzura 而不是 Panzura 来的。
