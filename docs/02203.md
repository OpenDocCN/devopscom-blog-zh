# IBM 和 DBmaestro 宣布在 DevOps 和云解决方案上建立全球合作伙伴关系

> 原文：<https://devops.com/ibm-and-dbmaestro-announce-on-a-global-partnership-on-devops-and-cloud-solutions/>

**IBM 和 DBmaestro 联手向全球企业客户提供数据库自动化功能**

*IBM 选择 DBmaestro 的 DevOps 平台作为 IBM UrbanCode Velocity 客户的补充产品*

马萨诸塞州波士顿——2019 年 7 月 7 日:数据库自动化领域的全球 DevOps 领导者 DBmaestro 今天宣布与 IBM 建立全球合作伙伴关系，提供旨在优化企业 DevOps 环境的数据库发布自动化功能。作为交易的一部分，IBM 将向其全球企业客户群提供 DBmaestro 平台。

为了保持竞争力，所有行业的公司都应该通过频繁而快速的发布周期来提供卓越的客户体验和持续创新。对于运行混合云环境的企业来说，在一个期望每天甚至每小时发布一次的市场中，后端数据库一直是一个重大挑战。DBmaestro 的客户群包括 ADP、ING Bank、加拿大国家银行、LFG、Anthem，长期以来一直为财富 1000 强企业提供最强大的 DevOps 平台。其产品自动化了数据库的持续集成和持续交付(CI\CD)流程，对现有流程没有任何中断，结合了版本控制、安全性和治理、发布自动化和业务活动监控。

该平台允许用户定义和实施对数据库进行更改的流程，在源代码控制中对所有内容进行版本控制(数据库 as-code)，提供以数据库为中心的配置管理，并为所有组成部分建立数据库策略，从而通过防止应用程序停机来提高生产率和效率并降低风险。

今天的发布反映了 DBmaestro 通过与互补的敏捷和 DevOps 解决方案集成来支持企业客户需求的战略，其中包括 Atlassian 的吉拉、Git、Chef、Puppet 和 Jenkins 等。IBM 长期以来一直是合作伙伴，DBmaestro 提供了与 IBM UrbanCode Velocity 的无缝集成。

DBmaestro 首席执行官兼联合创始人 Yariv Tabac 表示:“这一战略联盟建立在我们利用彼此产品的现有关系基础上，并支持我们的共同使命，即让企业变得更加敏捷，更安全、更快速地构建、*部署和发布代码*。“IBM 选择 DBmaestro 为其‘全球企业客户群’带来 DevOps for Database automation，我们对此感到非常自豪和兴奋。"

[IBM devo PS 解决方案产品经理 Eric Minick](https://twitter.com/EricMinick) 补充道:“IBM 早就了解其全球企业客户群对快速发布数字产品和服务的需求。我们已经与 DBmaestro 团队密切合作了一段时间，我们的许多 UrbanCode 客户已经在使用他们领先的 DevOps 解决方案，因此这种战略合作伙伴关系是非常自然的下一步。我们期待加深与 DBmaestro 的合作伙伴关系，以加速我们全球企业客户的 DevOps 部署。"

要了解有关联合解决方案的更多信息，请访问 DBmaestro 网站。

**关于 DBmaestro**

DBmaestro 为数据库带来了 DevOps 最佳实践，为数据库提供了新水平的效率、速度、安全性和流程集成。DBmaestro 的平台使组织能够安全有序地运行数据库部署，提高开发团队的工作效率，并显著缩短上市时间。该解决方案使组织能够针对数据库活动实施 CI/CD 实践，并具有可重复的管道发布自动化和自动漂移预防机制。欲了解更多信息，请访问[www.dbmaestro.com](http://www.dbmaestro.com/)。
