# 高德纳 2020 年 DevOps 价值流管理平台市场指南中认可的 Plutora

> 原文：<https://devops.com/plutora-recognized-in-gartners-2020-market-guide-for-devops-value-stream-management-platforms/>

*Plutora 宣布将其纳入 Gartner 的 2020 年 DevOps 价值流管理平台市场指南*

企业 IT 价值流管理(VSM)解决方案的领先提供商 Plutora 今天宣布，它已被确定为 Gartner devo PS 价值流管理平台市场指南的代表厂商。Gartner 的市场指南旨在帮助负责开发运维计划的基础架构和运营领导者，重点关注市场定义、市场原理和市场动态。

根据 Gartner 的报告，“组织缺乏对产品交付的端到端可见性，并努力改善其价值流。I&O 领导者和应用领导者必须实施 DevOps 价值流管理平台，并分析价值流指标，以优化产品交付的整体状况。”因此，这些平台“将成为关键投资，使他们能够评估和监控交付渠道，并消除持续改善业务成果的制约因素。”

“对我们来说，被 Gartner 评为 DevOps 价值流管理平台 2020 年市场指南中的代表性供应商是对我们在 Plutora 所做工作的证明，也是对我们为客户提供最大价值所做工作的认可，”Plutora 联合创始人兼首席执行官 Dalibor Siroky 说道。“为了在当今不断变化的环境中保持竞争力，组织必须专注于推动成功结果和增加业务价值。我们期待继续帮助企业实现数字化转型，成为软件驱动型企业。”

DevOps 价值流管理平台的市场指南指出，“负责 DevOps 计划的 I&O 领导者与应用领导者合作，通过投资 DevOps 价值流管理平台(VSMP)来消除产品交付中的障碍，该平台提供对产品性能、质量和价值的综合见解。”据 Gartner 称，“到 2023 年，70%的组织将使用价值流管理来改善开发运维管道中的流程，从而更快地交付客户价值。”

Plutora 平台为客户提供了一个独立于方法、自动化或工具供应商的控制塔，可监控其软件交付人员、流程和工具的整体情况。利用该平台的 Plutora 客户体验到了更快的速度、更高的质量和优化的价值。

Plutora 将向任何想要访问的组织或个人免费提供一份 Gartner 年 9 月 DevOps 价值流管理平台市场指南:[在此阅读](https://www.gartner.com/doc/reprints?id=1-24A15YMQ&ct=200930&st=sb "Read it here")。
