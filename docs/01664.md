# 网络研讨会:2016 年入职和离职员工 IT 指南

> 原文：<https://devops.com/it-onboarding-offboarding/>

一步一步的指南，帮助简化您的流程，同时确保您公司的安全。

你坐下来点午餐，你的手机亮了，无数的电话和短信飞了进来。一个工程师被解雇了，事情进行得不太顺利。他/她可以访问所有源代码、一些生产机器，并且知道关键密码。您是否有合适的系统来快速安全地切断访问？

在本次网络研讨会中，JumpCloud 首席执行官 Rajat Bhargava 将带您一步一步地了解入职和离职流程，以确保您为员工的成功做好准备，并确保您公司的安全。

您将学到的内容:

*   如何以准确、高效和安全的方式让员工入职

*   解雇员工的四个步骤

*   与不当离船相关的风险
