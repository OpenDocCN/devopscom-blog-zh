# MediaOps 宣布 DevOps 十二个 2020 奖项的决赛选手

> 原文：<https://devops.com/mediaops-announces-the-finalists-for-the-devops-dozen%c2%b2-2020-awards/>

*MediaOps 宣布了 DevOps 领域中有资格进入 DevOps 十二大奖提名流程决赛的顶级领导者、公司和工具的名称。*

佛罗里达州 BOCA 拉顿，2020 年 12 月 2 日——以最有力的方式讲述你的故事的地方 MediaOps 公布了第六届年度 DevOps 十二大奖 24 个类别的入围名单。DevOps 十几个奖项旨在表彰在 DevOps 社区中表现出领导力和奉献精神的最具创新性和最杰出的个人和公司。

今年的奖项包括的类别数量是往年的两倍，分为两个主要部分:DevOps 十几个工具和服务奖以及 DevOps 十几个社区奖。

晋级决赛的社区领袖、公司、工具和服务是每个类别中投票数最高的。在过去的一年中，那些进入最后阶段的人表现出了非凡的表现和对社区的服务。

“我们的决赛选手肯定是 DevOps 中最有影响力的领导者，”MediaOps 的首席执行官 Alan Shimel 说。“祝贺每一位决赛选手。能成为如此令人印象深刻的名单中的一员，是一项不小的成就。”
