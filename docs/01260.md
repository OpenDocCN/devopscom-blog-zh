# XebiaLabs 创新了渠道编排，增加了新的发布控制面板，并推出了 DevOps 聊天机器人

> 原文:[https://devo PS . com/xebialabs-innovates-pipeline-orchestration-adds-new-release-dashboard-introduces-devo PS-chatbot/](https://devops.com/xebialabs-innovates-pipeline-orchestration-adds-new-release-dashboards-introduces-devops-chatbot/)

**XL Release 的新版本有助于企业协调和控制大型复杂的版本，因此他们可以更快地生产软件，减少错误**

马萨诸塞州波士顿—(market wired—2016 年 5 月 3 日)—[XebiaLabs](http://www.xebialabs.com/)，DevOps 和持续交付软件工具领域公认的领导者，今天宣布推出 XL Release 版和 XL Deploy 版以及一款新的 DevOps 聊天机器人。

XL Release 允许公司在企业范围内自动化、协调并获得对其软件发布管道的可见性。XL Release 5.0 简化了涉众洞察软件交付过程的方式——从 IT 主管到项目经理再到开发人员。新的可配置仪表板和生产力特性旨在帮助管理复杂、多层次和相互依赖的软件发布渠道。

“随着发布速度加快，渠道变得更加复杂，企业知道他们需要做的不仅仅是自动化发布流程。同样重要的是，他们需要一种方法来理解、组织、管理和监控大量相互依赖的发布信息。“我们设计了新的功能来满足专注于实现业务成果的大型企业的需求，无论他们是使用现有的中间件和大数据系统，还是实施新的容器或物联网计划。”

**XL 5.0 版亮点**

*   *可定制的发布仪表板:*发布团队现在有了新的易于使用的仪表板，可以一目了然地显示每个版本中的内容。仪表板可以自定义配置，以显示团队最重要的状态指示器和指标，如整体发布进度、合规性和审核详细信息、交付的功能、发现的问题和解决的未结票据。它们还可以显示来自其他工具的信息，这些工具在软件交付管道中发挥了作用，如 JIRA、詹金斯、XL Deploy 或 ServiceNow。XL Release 的新仪表板为与该版本有利害关系的每个人提供了新的可见性，从 CIO 到风险管理团队，再到发布经理或 DevOps 团队。
*   *高级发布管道协调:* XL Release 5.0 提供了将互连管道作为一个整体单元进行管理的独特能力，因此很容易确定一个管道中的延迟何时会影响整个业务功能的交付。新版本现在可以从另一个版本中自动启动，版本自动链接，团队可以快速看到所有版本是如何组合在一起的。这种能力有助于组织管理更复杂的发布模式:他们可以自动编排几十个相关的发布，管理发布之间的依赖关系，对相关的发布进行分组，并轻松地在团队之间划分工作。XL Release 5.0 还增加了一个新的扩展表格视图，让团队更好地组织和管理复杂的、相互依赖的版本。

**发布协调交付业务成果** 与其他发布协调工具不同，XL Release 5.0 自动跟踪相关发布对当前发布的影响，因此可以快速发现并修复流程中任何级别的延迟。

通过自动化更多的发布过程和自动管理依赖性，企业可以更快地发布软件，减少错误。借助涵盖第三方工具的统一视图，组织可以获得所需的详细可见性，从而在整个软件开发生命周期中监控流程并跟踪其功能。随着 XL 版本增加的可见性和控制，开发运维及持续交付计划变得更加有效和高效。

**XL Deploy 的增强云配置选项** XebiaLabs 在发布 5.0 版的同时，还发布了其广受欢迎的 XL Deploy 产品的 5.5 版，该产品可为从 Docker 到大型机的任何环境提供企业级部署自动化。新版本将云环境的供应集成为部署流程的一部分，无需等待数天甚至数周才能获得团队所需的云资源。开发人员现在可以更容易地“自助服务”:他们可以启动和关闭云环境，作为其自动化部署的一部分，系统可以确保满足必要的控制。该功能通过最有效地利用云资源，帮助团队节省资金。它还让他们能够更深入地了解并更好地控制应用程序和环境的部署。

为了帮助 DevOps 团队更好地交流，XebiaLabs 最近推出了一个新的 ChatOps 功能。许多 DevOps 团队已经广泛使用聊天工具来更有效地工作，并将分散的团队聚集在一起。XL Deploy 的新插件通过让团队从他们选择的聊天工具(如 HipChat 或 Slack)中监控部署活动、触发部署和故障排除事件，促进了通信并提高了生产率。更多详情请访问 XebiaLabs 博客:[http://blog . XebiaLabs . com/2016/04/22/XebiaLabs-launchs-chatops-automation-software/](http://blog.xebialabs.com/2016/04/22/xebialabs-launches-chatops-automation-software/)

XebiaLabs 客户渴望使用新产品功能。“XL Release 的新版本提供了有趣的新仪表板功能，这将帮助我们了解每个版本中的内容，并更好地了解版本状态。此外，高级版本编排将使我们能够自动管理不同版本之间的父/子关系。“这些新功能在像我们这样复杂的环境中提供了很多价值。”
