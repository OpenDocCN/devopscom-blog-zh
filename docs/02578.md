# F5 收购 NGINX 以连接 NetOps 和 DevOps，为客户提供跨各种环境的一致应用服务

> 原文：<https://devops.com/f5-acquires-nginx-to-bridge-netops-devops-providing-customers-with-consistent-application-services-across-every-environment/>

*   全球云计算应用服务领导者 F5 宣布收购应用交付领域的开源领导者 NGINX。
*   战略收购和有机投资，以确保长期收入和每股收益增长。
*   F5 和 NGINX 将携手支持跨所有环境的多云应用服务，提供开发人员所需的易用性和灵活性，同时提供网络运营团队所需的规模、安全性、可靠性和企业就绪性。
*   F5 致力于持续创新并增加对 NGINX 开源项目的投资，以支持 NGINX 广泛的用户群体。
*   F5 将保持该品牌，现任 NGINX 首席执行官格斯·罗伯逊和创始人伊戈尔·塞索耶夫和马克西姆·科诺瓦洛夫将加入 F5，继续领导 NGINX。

<time datetime="2019-03-11T20:05:00Z">March 11, 2019 04:05 PM Eastern Daylight Time</time>

西雅图和旧金山—([商业资讯](https://www.businesswire.com/))—F5 Networks，Inc .(纳斯达克股票代码:FFIV)和 NGINX 今天宣布了一项最终协议，根据该协议，F5 将收购私人持有的 NGINX 的所有已发行和已发行股票，总企业价值约为 6.7 亿美元，并可进行某些调整。

F5 总裁兼首席执行官 Fran ois Locoh-Donou 表示:“F5 收购 NGINX 通过加速我们的软件和云计算转型，强化了我们的增长轨迹通过将 F5 世界一流的应用安全性和丰富的应用服务组合(用于提高性能、可用性和管理)与 NGINX 领先的软件应用交付和 API 管理解决方案、DevOps 社区中无与伦比的可信度和品牌认知度以及庞大的开源用户群相结合，我们可以在企业的云计算环境中提供一致的应用服务，从而弥合 NetOps 和 DevOps 之间的鸿沟。"

Locoh-Donou 继续说道:“我们相信每个组织都可以从现代技术带来的敏捷性和灵活性中受益，而不会影响安全性、可管理性和可靠性。“合并后的公司将为每个客户(从应用程序开发人员到网络工程师到安全专家)提供所需的工具，以确保他们的应用程序在从企业数据中心到私有云和公共云的每个平台上都可用且安全。”

F5 将利用 F5 安全解决方案增强 NGINX 的现有产品，并将 F5 云原生创新与 NGINX 的软件负载平衡技术相集成，加快 F5 面向现代容器化应用的应用服务上市时间。F5 还将利用其全球销售队伍、渠道基础设施和合作伙伴生态系统，扩大 NGINX 在企业中的销售机会。

“NGINX 和 F5 有着共同的使命和愿景。我们都相信应用是推动数字化转型的核心。NGINX，Inc .首席执行官 Gus Robertson 表示:“我们都认为，需要端到端的应用基础设施，即从代码到客户的应用基础设施，来交付跨云计算环境的应用。我很高兴能够继续这一旅程，将 NGINX 开源创新的力量融入 F5 的 ADC 领导地位和企业影响力。F5 通过专为 DevOps 设计的解决方案获得深度，而 NGINX 通过接触成千上万的客户和合作伙伴获得广度。"

NGINX 蓬勃发展的开源社区是这一组合最具吸引力的元素之一，F5 认识到用户群体对 NGINX 技术的信任。开源是 F5 多重云战略的核心部分，也是 F5 下一阶段创新的驱动力。因此，F5 致力于持续创新并增加对 NGINX 开源项目的投资，以支持 NGINX 广泛的用户群体。F5 预计，与 NGINX 的结合将加速其与领先开源项目的产品集成，并将增强其与开源厂商的强大技术合作伙伴关系。

收购完成后，F5 将保留 NGINX 品牌。格斯·罗伯逊和 NGINX 创始人伊戈尔·塞索耶夫和马克西姆·科诺瓦洛夫将加入 F5，继续领导 NGINX。Robertson 将加入 F5 的高级管理团队，向 Fran ois Locoh-Donou 汇报。F5 将维持 NGINX 在旧金山、加州和全球其他地方的运营。

**GAAP 与非 GAAP 对账**

F5 的管理层使用各种运营措施评估并做出运营决策。这些措施通常基于其产品、服务业务的收入以及这些业务的某些成本，如收入成本、研发、销售和营销以及一般和行政费用。其中一个衡量标准是净收入，不包括基于股票的薪酬、购买的无形资产的摊销、收购相关费用、净税收以及某些非经常性税收费用和福利，这是 1934 年《证券交易法》(修订版)G 条例第 101 节规定的非 GAAP 财务衡量标准。该指标包括 GAAP 净收入，不包括基于股票的报酬、购买的无形资产的摊销、诉讼费用、重组费用、设施退出费用、出售专利的收益、非经常性税收费用和福利以及与收购相关的费用。如果公司使用非 GAAP 结果而不是 GAAP 结果来计算公司的纳税义务，非 GAAP 净收入的这一衡量标准将根据公司应计的额外税收或税收优惠的金额进行调整。基于股票的薪酬是一项非现金费用，自 2005 年 7 月 1 日起，F5 已根据财务会计准则委员会(“FASB”)会计准则编码(“ASC”)主题 718 薪酬-股票薪酬(“FASB ASC 主题 718”)的公允价值确认规定进行了核算。无形资产摊销是非现金费用。投资者应注意，无形资产的使用有助于在所列期间赚取收入，并将有助于未来期间的收入。收购相关费用包括与收购相关的专业服务费用。此外，与公司总部搬迁相关的非经常性成本已从 GAAP 净收入中排除，以衡量 2019 财年第一季度的非 GAAP 收益和每股收益。

管理层认为，非 GAAP 每股净收入为管理层和投资者提供了有关公司核心业务运营绩效的有用补充信息，并有助于与公司的历史经营业绩进行比较。尽管 F5 的管理层认为这种非 GAAP 衡量标准有助于评估核心业务的绩效，但管理层对这种衡量标准的依赖是有限的，因为这种衡量标准中排除的项目可能会对 F5 的收益和根据 GAAP 计算的每股收益产生重大影响。因此，F5 的管理层将使用其非 GAAP 收益和每股收益指标，结合 GAAP 收益和每股收益指标，在评估公司核心业务的绩效时解决这些限制。投资者应将这些非 GAAP 指标视为 GAAP 财务绩效指标的补充，而不是替代。

F5 认为，呈现其非 GAAP 收益和每股收益指标为投资者提供了一个评估公司核心业务绩效的额外工具，并被管理层用于评估公司绩效。我们鼓励投资者将 GAAP 结果视为财务表现的最佳衡量标准。然而，虽然 GAAP 结果更加完整，但公司为投资者提供了这些补充措施，因为与 GAAP 一致，它可以提供对公司经营业绩和财务结果的额外见解。
