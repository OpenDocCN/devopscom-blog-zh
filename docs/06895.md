# 新的培训课程有助于获得 Jenkins CI/CD 方面的专业知识

> 原文:[https://devo PS . com/new-training-course-helps-gain-expertise-with-Jenkins-ci-CD/](https://devops.com/new-training-course-helps-gain-expertise-with-jenkins-ci-cd/)

*帮助 sre、软件开发人员、软件架构师和其他 DevOps 专业人员使用 Jenkins 改进其 CI/CD 实践的课程*

**三藩市，2020 年 10 月 6 日**–[通过开源实现大众创新的非营利组织 Linux 基金会](https://www.linuxfoundation.org/)今天宣布推出一门新的培训课程[lfs 267–Jenkins Essentials](https://training.linuxfoundation.org/training/jenkins-essentials-lfs267-2/?utm_source=lftraining&utm_medium=pr&utm_campaign=jenkins)。

LFS267 是与[持续交付基金会](https://cd.foundation/)联合开发的，旨在为开发运维工程师、质量保证人员、sre 以及希望通过 Jenkins 获得持续集成(CI)和持续交付(CD)活动专业知识的软件开发人员和架构师而设计。

Jenkins 是一个开源自动化服务器，它提供了数百个插件来支持任何项目的构建、部署和自动化。自 2004 年成立以来，Jenkins 一直是世界上最受欢迎的 CI 服务器。

在本课程中，学员将学习如何在 Kubernetes 集群中设置 Jenkins，如何使用 IaC 概念自动安装 Jenkins，如何设置高度可用的 Jenkins 服务器，如何利用静态和动态构建代理来扩展 Jenkins 基础架构，如何使用 CLI 和 RestAPI 自动执行各种常规 Jenkins 任务，如何通过实施精细的安全控制来保护 Jenkins 作业，如何设置有效的 CI/CD 反馈循环，如何设置模拟真实场景的复杂多分支管道，以及如何使用 GitOps 原则自动执行拉请求工作流。这些技能将使你为 SRE、开发人员和发布工程师的角色做好准备。

在入学前，学生应该对 Jenkins、Kubernetes 和 Docker 有一个基本的了解。那些不熟悉这些项目的人应该首先考虑报名参加 Linux 基金会的免费培训课程，包括[詹金斯简介](https://training.linuxfoundation.org/resources/free-courses/introduction-to-jenkins/?utm_source=lftraining&utm_medium=pr&utm_campaign=jenkins)、[云基础设施技术简介](https://training.linuxfoundation.org/resources/free-courses/introduction-to-cloud-infrastructure-technologies/?utm_source=lftraining&utm_medium=pr&utm_campaign=jenkins)和[Kubernetes 简介](https://training.linuxfoundation.org/resources/free-courses/introduction-to-kubernetes/?utm_source=lftraining&utm_medium=pr&utm_campaign=jenkins)。

新课程由 Himanshu Gautam 和 Deepika Gautam 开发。Himanshu 是一位作家、培训师、演讲者和问题解决者，在软件工程方面有 20 多年的经验。他也是一个热心的读者和专注于创建优雅的云原生架构的 DevOps 布道者。他还与詹金斯 X (LFS268)合作撰写了 [CI/CD 课程。Deepika 是一名作家、演讲人、培训师和 DevOps 传播者，在软件行业拥有近二十年的经验，擅长在多云环境中实现 DevOps 工具链。她还撰写了](https://training.linuxfoundation.org/training/ci-cd-with-jenkins-x-lfs268/?utm_source=lftraining&utm_medium=pr&utm_campaign=jenkins)[詹金斯简介](https://training.linuxfoundation.org/training/introduction-to-jenkins-lfs167/?utm_source=lftraining&utm_medium=pr&utm_campaign=jenkins) (LFS167x)课程。他们一起创建了 T4 的一家 CI/CD 解决方案提供商 Aplima 公司。

课程可以立即开始。299 美元的课程费用提供了一年内对课程所有内容和实验的无限制访问。感兴趣的个人可以在这里注册。

**关于 Linux 基金会**

Linux 基金会成立于 2000 年，由 1，000 多名成员支持，是世界领先的开源软件、开放标准、开放数据和开放硬件协作之家。Linux 基金会的项目对世界基础设施至关重要，包括 Linux、Kubernetes、Node.js 等等。Linux 基金会的方法侧重于利用最佳实践并满足贡献者、用户和解决方案提供商的需求，以创建开放协作的可持续模式。如需了解更多信息，请访问我们的网站[linuxfoundation.org](https://www.linuxfoundation.org/)。

**关于 CD 基金会**

持续交付基金会(CDF)致力于提高世界安全和快速交付软件的能力。CDF 是一个厂商中立的组织，它正在建立软件交付自动化的最佳实践，推动 CD 工具的教育和采用，并促进新兴技术的交叉融合。CDF 是许多发展最快的 CD 项目的所在地，包括 Jenkins、Jenkins X、Tekton 和 Spinnaker。CDF 是非营利组织 Linux 基金会的一部分。有关 CDF 的更多信息，请访问 [https://cd.foundation](https://cd.foundation/) 。
