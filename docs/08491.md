# 詹金斯用户会议和光盘峰会:呼吁所有伦敦开发者

> 原文：<https://devops.com/jenkins-user-conference-and-cd-summit-calling-all-london-devops/>

我想给一个大的，充满活力和支持的伦敦 DevOps 社区一个快速的呼喊。本月晚些时候(6 月 23 日和 24 日)，将在伦敦希尔顿大都会酒店举行联合会议。结合詹金斯用户大会和 Cloudbees CD 峰会，是两天的 CD/CI/DevOps 学习。

演讲者包括:

弗雷斯特公司首席分析师库尔特·比特纳

埃森哲 DevOps 董事总经理 Martin Croker ，

**Sanjeev Sharma** ，DevOps 技术销售&采用 CTO，IBM 杰出工程师，IBM，

XebiaLabs 产品经理 Viktor Clerc

还有更多。

Jenkins 用户大会(JUC)的另一大特色是演讲者阵容。DevOps.com 和许多其他国家赞助了这次活动。

在与我在 Cloudbees 的朋友交谈时，他们提到还有票。我发现这很难相信，因为我知道伦敦的社区是什么样的。因此，如果您打算在 23 日和 24 日进城，不要错过这个活动！23 号我也会在那里主持一个很棒的座谈会。

詹金斯社区是我见过的最活跃的社区之一。我期待着 23 日和 24 日在伦敦见到你们中的许多人。

仍然有对裁谈会峰会的赞助。如果感兴趣，您可以联系 Cloudbees 获取信息。

同样是本周四[devo PS Connect:Rugged devo PS](http://www.devopsconnect.com/events/infosecurity-europe-london/)@ info security Europe 全天。注册是免费的，所以下来一个伟大的一天 DevOps 轨道！
