 *## 调查显示:开发者远离办公室

本周首先:GitHub 对开发者的年度调查有一些关于工作场所(咳咳)“新常态”的有趣数据。疫情彻底改变了开发人员对工作地点的期望。

### 分析:WFH 是 SOP

与其他角色一样，开发人员希望继续在家工作。但这对你的 DevOps 文化构成威胁了吗？确保你能以**同等的灵活性**完成*操作*。

保罗·克里尔 : **发展生产力回到了疫情之前的水平**

GitHub 的“2021 年 Octoverse 状态”研究……看到了工作节奏回到疫情之前水平的迹象。GitHub 还发现，46%的与队友共处一地的开发人员现在希望完全远程工作或在混合环境中工作。
…
根据从 400 多万个存储库中收集的数据和对 12，000 多名开发人员的调查，JavaScript 和 Python 仍然是最受欢迎的语言，其次是 Java 和 TypeScript。GitHub 发现，当重用代码时，开发团队的性能可以提高 87 %,当使用… CI/CD 时，可以提高 43%。

对准 WFH 角，是[理查德·斯毕德](https://www.theregister.com/2021/11/17/githubs_state_of_the_octoverse/ "read the full text"):

全职或兼职在办公室的受访者从疫情之前的 42%下降到现在的 10.7%。…有很多空桌子。ctrl-c、ctrl-v 组合的粉丝们会高兴地注意到，GitHub 发现，当代码重用在工作中变得“无摩擦”时，开发人员的性能提高了 87%。…这些都是极好的东西。

但是要小心统计上的谬误，亨里克·英戈—[@ h _ 英戈](https://twitter.com/h_ingo/status/1460873914809204738 "read the full text")认为:

#在家工作可能比任何人意识到的都要普遍。[但是 GitHub 用户/回答者的人群是否在某种程度上倾向于先驱者/早期采用者？…这可能是一个相关而非因果的例子。

## 国防部订购新云

国防部发布了最新的云合同机会通知，标题很简洁，叫做“联合作战云能力”(JWCC)。从通知的字里行间来看，这一切听起来非常 **DevOps- *暨*-敏捷**。

五角大楼已经放弃了之前对单一供应商的坚持，现在希望至少有两家云供应商。

### 分析:绝地归来，因为 AWS 想要重来

JWCC 是流产的绝地采购的后续。亚马逊网络服务“失望”地被微软忽略了。现在，国防部似乎希望他们共享 T1。

你能从这种**多厂商开发运维战略中学到什么？**

[乔丹·诺维特](https://www.cnbc.com/2021/11/19/pentagon-asks-amazon-google-microsoft-oracle-for-cloud-bids.html "read the full text") : **五角大楼向亚马逊、谷歌、微软和甲骨文招标**

亚马逊和微软是单一联合企业防御基础设施的最终入围者。…微软在 2019 年赢得了它，亚马逊提出了抗议，五角大楼在 7 月取消了合同。这项新的努力被称为联合作战云能力。……JWCC 不同于绝地，因为它允许五角大楼依赖多个云提供商。
……
美国总务管理局表示，只有两家美国云基础设施提供商，亚马逊和微软，似乎能够满足五角大楼的所有要求。……新合同的价值尚不清楚，但国防部估计可能高达数十亿美元。

总之，留意这个[匿名懦夫](https://forums.theregister.com/forum/all/2021/11/19/us_defense_jedi/#c_4370303 "read the full text"):

竞争供应商会比非竞争供应商更加努力。国防部在 F-35 这样的“一环定天下”系统上有很多问题。…绝地也会一样。
…
测试/使用时升级的步骤越小，回报越高。如果做得好，他们可以通过让这些公司在确保产品实际有用方面相互竞争来获得更好的交易。

但是仔细看看国防部的合同机会，[巴斯汀布雷恩斯](https://yro.slashdot.org/comments.pl?sid=20258145&cid=62003187 "read the full text")说:

他们都会得到合同。然后国防部会简单地使用他们最喜欢的一个。没有人可以抱怨，因为它是开放式的，没有任何保证。

## 联邦贸易委员会关注的 Arm/Nvidia 交易

上周，我们谈到了英伟达从软银收购 Arm Ltd 的目标，以及它带来的监管挑战——特别是在芯片设计师的家乡英国。我们去按后，**另一只鞋掉了**。

### 分析:已经放弃了

正如我们上周所说，不能允许英伟达这样做。ARM 芯片不仅出现在 99%的智能手机中，而且越来越多地出现在数据中心——尤其是重视“性能功耗比”的地方

理查德·沃特斯 : **混乱的收购 Arm 的下一步是什么？**

英伟达(Nvidia)从软银(SoftBank)手中收购英国芯片设计公司 Arm 的交易，在大西洋两岸引发了严重反对。……美国美国联邦贸易委员会有自己的担忧。与此同时，中国正严阵以待。这笔交易凸显了 Arm 在芯片行业的独特地位。(英伟达)保证不会阻止任何其他公司许可 Arm 的设计。这一提议被置若罔闻，欧盟和英国都认为它不够。……后英国退出欧盟时代的政治也发挥了作用。
……
英伟达和软银会在什么时候收手？…股市上市是最有可能的选择，英国是最受青睐的上市地点。

从内部来看如何？这是来自 T2 的一个提示:

软银收购他们的时候我在 ARM 这对每个人来说都是一个惊喜，因为我们绝对相信我们是不可战胜的，因为任何对我们感兴趣的人都会想利用差别许可作为一种武器来对付他们的竞争对手，而我们…相信监管者不会接受这一点。许多人带着一年的薪水回家，拿着免费的钱。而且不少人…带着 10 年的工资卡回家了。

越来越多的 DevOps 工作负载在 ARM 芯片上运行。这是生物光子:

ARM 的低功耗专业技术在数据中心(DC)市场上是一个优势。…另外，Linux 在 ARM 上可用，因此英特尔的架构锁定可能会在 DC 市场受到侵蚀。
