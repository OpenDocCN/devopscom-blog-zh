# 与 StackRox 一起开创 Kubernetes 的新时代

> 原文:[https://devops.com/the-new-era-of-kubernetes-with-stackrox/](https://devops.com/the-new-era-of-kubernetes-with-stackrox/)

Orchestrators 正在成为未来的云操作系统。你认为哪一个成为了主导？StackRox 的联合创始人兼首席技术官 Ali Golshan 从一开始就把赌注押在了 Kubernetes 身上。

[StackRox](https://www.stackrox.com/) 是一个 Kubernetes-native 安全平台，帮助开发人员和安全团队实施安全，提供更丰富的上下文、本机实施和持续强化。

在与 MediaOps 创始人兼主编 Alan Shimel 的访谈中，Ali 讨论了 Kubernetes 的采用及其在今天的流行程度。越来越多的公司意识到他们不能使用传统的安全措施。工具必须是他们正在使用的基础设施的原生工具，它必须集成到构建和部署中。他说，Kubernetes 是现有的最佳解决方案。
