Ahana 今天还宣布了 Ahana Cloud for Presto Community Edition，旨在简化 Presto 的部署、管理和集成，Presto 是一个用于 Open Data Lakehouse 的开源分布式 SQL 查询引擎。Ahana 社区版立即对所有人开放，包括在 DockerHub 上下载 Ahana 的 PrestoDB 沙盒超过 100，000 次的用户。它提供简单的分布式 Presto 集群配置和经过调整的开箱即用配置，免费为各种规模的数据团队带来 Presto 的强大功能。不用下载和安装开源的 Presto 软件，数据团队可以快速了解 Presto 并在云中部署初始的 SQL 数据 lakehouse 用例。Community Edition 用户可以轻松升级到 Ahana Cloud for Presto 的完整版本，这增加了安全性，包括与 Apache Ranger 和 AWS Lake Formation 的集成，性价比优势，包括多级缓存和企业级支持。

“在过去的一年里，我们专注于将 Presto 最简单的托管服务推向市场，今天我们很高兴地宣布推出一个永远免费的社区版，以推动更广泛的开源用户社区更多地采用 Presto。Ahana 联合创始人兼首席执行官 Steven Mih 表示:“我们与 Liberty Global 建立了新的合作关系，这更加坚定了我们对 Presto 作为 Open Data Lakehouse 最佳 SQL 查询引擎的信念。“有了 Community Edition，数据平台团队可以在很大规模上无限制地使用 Presto，以闪电般的速度洞察他们的数据。”

“今天，我们看到越来越多的公司采用基于云的技术来提供卓越的客户体验。Liberty Global Ventures 的合作伙伴 Ankur Prakash 表示:“一个基本的架构模式是利用一个开放的数据湖库，这是一个更灵活的堆栈，可以解决传统数据仓库的高成本、锁定和限制。“Ahana 以其行业领先的方法创新地解决了这些挑战，为数据平台团队带来了最高性能、最具成本效益的 SQL 查询引擎。我们对 Ahana 的投资反映了我们为企业创造更多价值的承诺，特别是在数据仓库的下一次发展中，以开放数据湖库。”

Ahana Cloud for Presto Community Edition 的详情包括:

*   永远免费使用
*   在开放数据湖中使用 Presto，开放文件格式如 Apache Parquet，高级湖数据管理如 Apache 胡迪
*   一个预存储集群，包含除 Graviton 之外的所有受支持的实例类型
*   预配置集成到多个数据源，包括用于亚马逊 S3 的 Hive Metastore、亚马逊 OpenSearch、用于 MySQL 的亚马逊 RDS、用于 PostgreSQL 的亚马逊 RDS 和亚马逊 Redshift
*   通过公共 Ahana 社区 Slack 频道提供社区支持，外加与 Ahana Presto 工程师进行的 45 分钟免费入职培训
*   无缝升级到完整版本，包括数据访问控制、自动扩展、多级缓存和基于 SLA 的支持等企业功能

“企业继续接受‘湖屋’平台，这些平台将 SQL 结构和查询功能应用到云原生对象存储，”埃克森集团研究副总裁凯文·佩特里说。“Ahana 的新社区版 Presto 提供了一个 SQL 查询引擎，可以帮助推动市场对湖边小屋的采用。”
