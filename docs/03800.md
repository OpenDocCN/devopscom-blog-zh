# Datical 以创纪录的客户增长、产品创新和合作伙伴扩张结束了 2017 年

> 原文：<https://devops.com/datical-wraps-2017-record-customer-growth-product-innovation-partner-expansion/>

*数据库发布自动化领域的领导者继续迅猛发展*

**得克萨斯州奥斯汀–2018 年 1 月 18 日**–[领先的数据库发布自动化解决方案提供商 Datical](http://www.datical.com/) 宣布其在 2017 年取得了巨大的增长，其旗舰产品 Datical DB 的年订阅量几乎翻了一番，客户增长了 30%。Datical 不断看到大型企业部署 Datical DB，以帮助加快他们的数字化转型和开发运维计划。

根据 Forrester 的首席分析师 Robert Stroud 在最近的博客中的说法，2017 年被预测为 DevOps 年。Forrester 指出，“我们的数据证实 50%的组织正在实施开发运维，开发运维已经达到‘逃逸速度’。”"

虽然组织已经意识到在整个企业中部署 DevOps 的好处，并开始采用自动化来加快软件交付的步伐，但仍然存在一个巨大的障碍——将*数据*引入[连续交付流程](https://www.datical.com/why-datical/)。在过去的一年里，Datical 通过扩展对新数据库平台的支持，发布促进应用团队之间沟通和协作的新功能，以及通过 Datical 合作伙伴计划构建 DevOps 生态系统，帮助其客户消除了这一障碍。

"我们很高兴地宣布，Datical 又迎来了惊人增长的一年."Datical 的首席执行官德里克·赫特森说。“随着 2018 年的开始，我们将继续为我们的客户提供创新，通过消除数据库的障碍来帮助他们提高软件发布的速度。随着我们的客户继续追求数字化转型、向云迁移和开发运维，我们将确保他们的数据库发布流程快速、安全、可靠。

**Datical 2017 公司、产品、客户和合作伙伴里程碑**

*企业势头*

*   领先企业的客户群增长了 30%，如 AIG、LabCorp 和美国财政部
*   通过将在 R&D 的投资增加 70 %,支持新创新的交付
*   启动 [Datical Partner Program](https://www.datical.com/press-release/datical-launches-partner-program-to-unite-and-empower-the-devops-ecosystem/) 联合并赋能 DevOps 生态系统，联合系统集成商和全球最受欢迎的 DevOps 厂商如 Delphix、IBM、XebiaLab、CA 等，为数据库带来持续交付。
*   开发了与 [Pivotal](https://www.datical.com/press-release/datical-teams-up-with-pivotal-to-deliver-database-automation/) 、CA 和 [Jenkins 2.6](https://www.datical.com/press-release/datical-brings-continuous-delivery-to-the-database-with-jenkins/) 的集成，将 DevOps 引入数据库
*   获得行业荣誉，包括入围“顶级 DevOps 传播者”和“最佳 DevOps 商业解决方案”类别的 [DevOps 十几个](https://devops.com/devops-dozen-2017-finalists-announced/)，以及“最佳数据库开发解决方案”类别的[数据库趋势和应用读者选择奖](http://www.dbta.com/Editorial/Trends-and-Applications/Best-Database-Development-Solution-119269.aspx)Datical 还被《数据库趋势和应用》评为 2017 年 [DBTA 100 强](http://www.dbta.com/Editorial/Trends-and-Applications/DBTA-100-2017---The-Companies-That-Matter-Most-in-Data-118003.aspx)中“对数据最重要的公司之一”

*产品创新*

*   向[数据平台](https://www.datical.com/solution/)、[数据部署监控控制台](https://www.datical.com/solution/deployment-monitoring-console/) (DMC)推出了[新功能](https://www.datical.com/press-release/delployment-monitoring-console-to-further-expand-devops-to-the-database/)，该控制台可自动监控整个企业中每个数据库部署的状态，并提供对所有数据库部署结果的前所未有的可见性。
*   宣布[支持 Db2 for z/OS](https://www.datical.com/press-release/datical-announces-support-for-applications-on-ibm-db2-for-z-os/) ，这是存储和利用任务关键型数据的无可争议的领导者，随着企业继续进行数字化转型，这一举措为大型机带来了自动化。
*   启动了对 Oracle 数据库云服务器的[支持，这是最强大的数据库平台，旨在实现组织数据库部署流程的现代化，缩短应用发布时间，提高安全性，并提高应用开发人员、测试人员和数据库管理员的工作效率。](https://www.datical.com/press-release/datical-announces-support-for-oracle-exadata/)
