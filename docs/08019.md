# 为什么 AIOps 对疫情商业复苏至关重要

> 原文：<https://devops.com/why-aiops-is-critical-for-pandemic-business-recovery/>

过去一年，随着企业努力维持运营，许多领域的创新都落后了。企业领导人难以理解新冠肺炎疫情法案将对他们的业务产生的短期和长期影响，而员工担心失去工作，客户争相重新安排预算。当我们接近终点，终于看到隧道尽头的曙光时，商业领袖们正在寻找回到正轨的方法。

面向 IT 运营的人工智能( [AIOps](https://devops.com/?s=AIOps) )是许多企业需要的解决方案，以释放资源，将重点放在创新上。数据说明了一切——2020 年，AIOps 市场的价值为 135.1 亿美元，预计[将在未来五年增长超过 200%](https://www.researchandmarkets.com/reports/4833395/aiops-market-growth-trends-covid-19-impact) 至 405.1 亿美元。AIOps 为企业带来了几个好处，比如自动化日常任务、提供可操作的报告以及改进风险管理。

但是，在 IT 领导者一头扎进采用 AIOps 之前，他们仍然必须解决一个常见的误解:AIOps 不是一个解决方案，而是一个长期战略。AIOps 根本不能被视为复苏之路上的一个权宜之计。为了真正实现利益最大化，并将创新带回后疫情时代的业务中心，团队必须将 AIOps 视为一项长期战略投资。

对于希望采用 AIOps 策略的 IT 团队来说，可以考虑以下三个要点来启动这一旅程并长期保持下去。

### 寻找合作伙伴提供市场洞察力

在复苏的道路上，IT 领导者可能没有时间或带宽来真正投资实施稳健的 AIOps 投资战略，更不用说从哪里开始了。但是，与一个值得信赖、经验丰富、有时间和资源制定计划并密切关注市场趋势的合作伙伴合作，可以帮助缓解 IT 领导者可能面临的任何困难。

战略合作伙伴还可以帮助团队确定 IT 团队已经准备好了哪些工具来支持 AIOps 战略，哪些工具会阻碍它。通过全面的工具合理化流程，IT 团队可以更好地了解他们拥有哪些工具，哪些工具执行重叠的功能(换句话说，增加了不必要的成本)，以及他们仍然需要哪些工具来支持他们的 AIOps 之旅。

合适的合作伙伴可以为 IT 领导者规划前进的道路，并让他们放心，因为他们知道自己的背后始终有一位专家。

### 识别用例

在过去的一年中，AIOps 的用例可能已经发生了变化，这取决于公司在市场中的地位、产品重点和未来的增长计划。但是随着公司将创新带回前沿，他们必须重新评估他们的 AIOps 用例。如果你是一名希望投资 AIOps 的 IT 主管，问问自己“为什么？”并精确地确定 AIOps 如何支持最终目标。您值得信赖的合作伙伴可以提供不同的视角，这可能是 IT 领导者所不具备的，因为他们日复一日都处于困境之中。

### 制定计划并执行

有了合适的工具和您身边的合作伙伴，现在是时候制定您的完整 AIOps 投资计划，并实施可行的步骤了。考虑您确定的用例以及您对 AIOps 的短期和长期目标作为起点。请记住，这不是一个有明确终点的计划，而是一个有明确里程碑的战略，可以帮助保持业务正常运行和客户满意，同时最大限度地优化内部流程。

当 IT 领导者展望未来时，他们可以重拾信心，并感受到将创新带回业务的力量。他们不应该再觉得自己淹没在事故中，不断灭火。有了长期的 AIOps 计划，IT 团队就能在大流行后茁壮成长，并比以往更加强大。
