# 茄子在其数字自动化智能套件中增加了预测洞察力，以最大限度地降低产品发布风险

> 原文：<https://devops.com/eggplant-adds-predictive-insights-to-its-digital-automation-intelligence-suite-to-minimize-product-release-risks/>

#### *提供直观的见解，将业务风险可视化，提高产品质量并优化客户体验*

**伦敦，&科罗拉多州博尔德 8 月 9 日^(日，) 2018 [茄子](https://eggplant.io/)** [，](https://eggplant.io/) 以用户为中心的智能测试和性能解决方案提供商，今天宣布将茄子 Release Insights 加入其数字自动化智能(DAI)套件。该技术增强了茄子市场领先的自动化能力，提供直观的分析和见解，帮助团队最终了解如果他们现在发布另一个版本的产品，对用户满意度和业务成果的影响。这有助于加速应用程序和软件的持续交付，并为组织提供竞争优势。

发布新的应用程序和软件为开发团队提供了大量的技术数据。但是，组织由于没有整理这些信息，根据经验而不是数据做出决策，因此错过了一个重要的业务机会。茄子 Release Insights 通过提供关键信息的聚合、可视化摘要来解决这个问题，使团队能够立即了解业务影响并采取行动。这使得团队能够做出数据支持的决策，从而提高每个版本的质量并增强客户体验。

茄子 Release Insights 与茄子 AI 协同工作，提供了一个不断扩展的综合预测器集，这些预测器决定了整体的发布质量。这创建了对发布评级的直观、实时的理解，使公司能够在发布之前微调他们的测试并解决任何问题。

#### 预测因素包括:

*   #### **Bug Content Predictor** : Predict the expected number of undetected bugs and evaluate all aspects of the build process.

*   #### **Development quality predictor** : Identify the biggest risk factors in the release and evaluate their impact. The result is to evaluate the quality of the whole development process and determine the key risks affecting the quality.

*   #### Test Coverage Predictor: using coverage data from regression and exploratory tests, it provides an evaluation of the current product coverage level.

*   #### **Usability Quality Predictor** : Analyze the developing UI and generate visual metrics of usability and consistency of the current version.
