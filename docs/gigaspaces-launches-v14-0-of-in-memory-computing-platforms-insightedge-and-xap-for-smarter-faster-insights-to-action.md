# GigaSpaces 推出内存计算平台 InsightEdge 和 XAP 的 v14.0 版本，以更智能、更快速地洞察行动

*全面支持面向云、内部和混合环境的 Kubernetes 简化容器部署，以及英特尔 Optane DC 永久内存*

**纽约州纽约市——2018 年 12 月 5 日** — [领先的即时洞察行动内存实时分析平台 InsightEdge 的提供商 GigaSpaces](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgMbw%2B1PN%2BlPv2yMzr4OGY2dXT3PxLZYM1A%3D%3D&G=0&R=https%3A%2F%2Fwww.gigaspaces.com%2F&I=20181203161631.0000009df04d%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjVjMDU1NmM3M2RhNmNjZWYyNTRkMTMzZjs%3D&S=IgJcIpaaKsjML8z69EX7HzGxnmapO_sVE45jnVSIHxg) 今天宣布 [InsightEdge](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgMbw%2B1PN%2BlPv2yMzr4OGY2dXT3PxLZYM1A%3D%3D&G=0&R=https%3A%2F%2Fwww.gigaspaces.com%2Fproduct%2Finsightedge-platform&I=20181203161631.0000009df04d%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjVjMDU1NmM3M2RhNmNjZWYyNTRkMTMzZjs%3D&S=v0niV63WOAleT4qpb7KDbmqupx17CgMauGmXK7OAK9Q) 平台 [v14.0 和 XAP v14.0](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgMbw%2B1PN%2BlPv2yMzr4OGY2dXT3PxLZYM1A%3D%3D&G=0&R=https%3A%2F%2Fdocs.gigaspaces.com%2Fxap%2F14.0%2Foverview%2Foverview.html&I=20181203161631.0000009df04d%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjVjMDU1NmM3M2RhNmNjZWYyNTRkMTMzZjs%3D&S=Hb3zZABV-NjkX0I9PY7bJdsxHKL49HmF7DpG_ykBfz4) 正式上市，以简化客户开发和部署时间敏感和任务关键型服务所需的应用程序。

14.0 版中的新特性和功能包括:

*   Kubernetes 环境支持；用于简化云、内部和混合环境中的部署
*   英特尔 Optane DC 永久内存集成利用 GigaSpaces 内存扩展模块显著降低总拥有成本(TCO ),并以接近内存的性能增加容量
*   定制的重新平衡策略可在运行时执行热交换，而无需重新加载数据
*   数据可视化，只有 InsightEdge 用户可以更快地访问 Apache Zeppelin

这一最新版本旨在帮助企业以优化的方式利用和分析其数据的实时影响，以实现和保持竞争优势。

Gartner 分析师 [Massimo Pezzini](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgMbw%2B1PN%2BlPv2yMzr4OGY2dXT3PxLZYM1A%3D%3D&G=0&R=https%3A%2F%2Fwww.gartner.com%2Fanalyst%2F9152&I=20181203161631.0000009df04d%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjVjMDU1NmM3M2RhNmNjZWYyNTRkMTMzZjs%3D&S=6NcNFr0dC-MsH0L7vd7Vg4wiO0Y2mXzh01o0S1NYcYE) 在[预测 2018:随着采用率的增长，内存计算技术仍将无处不在】的报告中表示:“业务的数字化产生了对更快性能、更大可扩展性和更深入实时洞察力的无尽需求，这正在推动 IMC 技术的创新。”。](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgMbw%2B1PN%2BlPv2yMzr4OGY2dXT3PxLZYM1A%3D%3D&G=0&R=https%3A%2F%2Fwww.gartner.com%2Fdoc%2F3840569%2Fpredicts--inmemory-computing-technologies&I=20181203161631.0000009df04d%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjVjMDU1NmM3M2RhNmNjZWYyNTRkMTMzZjs%3D&S=nZJN8wlh-R9ucM0M0rOJ6hDjBZu0nNJlyc2-oMZ0ffY)

GigaSpaces 是一组精选的独立软件开发商(包括亚马逊、谷歌和微软)之一，他们在 2019 年上半年广泛可用之前，集成了英特尔的 [Optane DC 持久性存储器](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgMbw%2B1PN%2BlPv2yMzr4OGY2dXT3PxLZYM1A%3D%3D&G=0&R=https%3A%2F%2Fnewsroom.intel.com%2Feditorials%2Fre-architecting-data-center-memory-storage-hierarchy%2F&I=20181203161631.0000009df04d%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjVjMDU1NmM3M2RhNmNjZWYyNTRkMTMzZjs%3D&S=6MET2-e2WXOhjoO5npqIQjEs7j05QIqXHVZ1LSlIhwA)技术。GigaSpaces 的集成以独特高效的方式利用了英特尔的突破性技术。存储在内存中的索引根据客户定义的业务逻辑，而不仅仅是根据更简单的 LRU 方法，从这个新的数据存储层访问所需的数据。与固态硬盘技术相比，英特尔 Optane 永久内存具有更高的数据容量和更快的速度，通过支持实时机器学习以亚秒级延迟在更大的数据集上运行，可以加速数据处理，并提供更快、更准确的分析。此外，所需服务器的数量也显著减少，导致占地面积更小，功耗、维护、网络和其他开销成本更低。

InsightEdge 平台和 XAP 的这一最新版本还支持 Kubernetes 环境，用于为主要云提供商(包括 AWS、微软 Azure、GCP 等)的云、内部和混合环境大规模自动化部署、扩展和管理容器化应用。Kubernetes 在 GitHub 上的活跃程度排名第一，有超过 1500 名贡献者，并且已经被认为是云原生编排的事实上的标准。Kubernetes 的有状态集用于实现自我修复、负载平衡和自动回滚，以最大限度地减少停机时间和零数据丢失。支持自动部署数据服务、可视化工具和机器学习框架，如 Apache Spark 和 Apache Zeppelin。

GigaSpaces 产品副总裁 Yoav Einav 表示:“对于许多企业来说，获得智能的实时洞察并在关键时间框架内采取行动是一项挑战，尤其是在大规模情况下。“随着 InsightEdge 平台和 XAP 的发布，我们将继续创新和简化更智能、更快速的应用和服务的开发和部署，跨云、内部和混合云，并优化总拥有成本，帮助我们的客户加快他们的数据驱动计划。”

关于千兆空间

GigaSpaces 提供领先的内存计算平台，用于实时洞察行动和极端事务处理。通过 GigaSpaces，企业可以实施机器学习和交易处理，以获得对其数据的实时洞察，并立即采取行动。全球金融服务、零售、运输、电信、医疗保健等行业的数百家一级和《财富》杂志列出的组织都在利用跨云、内部或混合云的任务关键型应用程序的不间断平台。GigaSpaces 的办公室位于美国、欧洲和亚洲。
