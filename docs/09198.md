# 随处工作如何影响开发工作

> 原文：<https://devops.com/how-work-from-anywhere-impacts-devops/>

Ciena 的 Brian lavalée 与 Mike Vizard 讨论了[随处工作](https://devops.com/?s=remote+work)运动对开发、网络和安全的影响。下面是视频，后面是对话的文字记录。

*迈克·维扎德:* 你们刚刚公布了这项研究，讨论在家工作对科技的影响。很多人都在访问云资源。他们的家庭网络也许能胜任，也许不能胜任。当我们开始在任何地方工作时，我们会面临某种网络危机吗？人们是否明白带宽将成为一个问题？

*Brian lavallée:*我不会说这是一场危机，但这是一项正在进行的工作。就在去年，一年半，全球疫情，我们的家已经变成了新的分支机构，新的教室。人们在家学习、娱乐和工作，这给网络带来了额外的压力。一个主要的挑战是速度，因为现在你有两个、三个或四个人同时从家里访问同一个宽带连接，然后你有整个安全方面，如果你的家是新的分支机构，你经常需要额外的安全来工作，不仅是与你的总部、你的同事，而且与你的云应用程序。

*迈克·维扎德:* 我发现自己试图与一个 11 岁的孩子争夺带宽，所以我知道问题是什么，问题可能是什么。你认为在我们前进的过程中，公司会回头对他们的员工说，“这是我们希望你使用的路由器”吗？他们会给他们那个路由器，还是家里的接入点？他们能在多大程度上实现这一切，并提供额外的安全保障？

*Brian lavallée:*是的，我认为，毫无疑问，增加额外的安全性，甚至可用性，一个更具弹性的连接是向前发展所必需的。公司或企业将以不同的方式实现这一点。一些公司可能在云中完成大量的安全功能。这允许您在一个集中的位置执行所有的安全任务，您知道，谈论 SASE ，_ _ _ _ _ _连接 ，因为它是集中的，所以更容易实施和控制，而不是在每个人的笔记本电脑上安装专用设备或专用软件。如果你是一个大企业，你可能有成千上万的人在家工作，所以用高速连接集中你的安全方面可能是前进的方向。

*迈克·维扎德:* 这对核心网络有什么影响？我的意思是，如果每个人都突然在边缘获得了更多的带宽，这是否会对网络基础设施的其余部分产生级联效应？我需要额外的核心路由器或网络吗？你会如何看待这件事？

*布莱恩·拉瓦雷:* 耶，_____。内容是在边缘创建和消费的，这些内容来自人类或机器，比如传感器、摄像头等等。毫无疑问，随着我们开放接入坡道，5G 是开放无线接入坡道的一种方式，此外，您可以为家庭甚至办公室提供各种有线宽带技术。可以肯定的是，它对核心网络具有连锁效应，因为如果你仔细想想，你访问的大多数内容和应用程序，无论是在你的智能手机、笔记本电脑、平板电脑或你正在使用的任何设备上，都位于某个数据中心。这可能是一个离您很近的数据中心，也可能是一个与您位于同一城市的边缘数据中心。它可能是位于世界另一端的挪威、瑞典或芬兰的数据中心。

*迈克·维扎德:* 我觉得人们开始构建、消费和分析数据的许多应用程序，更多的是在边缘产生的。对延迟是否更加敏感？网络物理定律会很快回来咬我们吗？我们会看到新的应用类别吗？

*Brian lavallée:*是的，这是一个很好的问题，因为它与讨论的主题有关，是当今最热门的话题之一，即边缘计算或边缘云。通过将存储和计算访问放在更靠近终端用户的地方，这些终端用户可以是人也可以是机器，它确实内在地减少了延迟，只是因为您的物理距离更短。但是，我们必须记住，延迟不仅仅与距离有关，还与您和您的内容之间的网络节点数量有关，因此，您使用的网络设备(交换机、路由器等)越多，产生和积累的延迟就越多。通过高速连接将存储和计算推向边缘是减少延迟的一种非常好的方式，它还开辟了一系列新的应用程序，在这些应用程序中，延迟是除了提高速度之外的一个关键因素。

*Mike Vizard:* 我感觉就像在疫情之后，我们把每个人都送回家，我们给他们一个小小的无线网络，我们用 VPN 连接他们，他们可以访问办公室里的应用程序。然后，他们转向云，他们有这种网络流量的回传效应。你提到了塞丝。我们是否会改变我们对应用程序的部署方式和访问方式的看法，无论它们是否在云中？我们多年来所熟知的路由概念是否正在改变？你认为我们将如何继续前进？

*Brian lavallée:*是的，所以我认为云服务的日益普及是一股不可阻挡的力量。它正在发生。存在安全隐患。我们所做的调查和研究确实表明，越来越多的人转向云计算存在安全问题，但云提供商、电信运营商在过去几年中做了大量出色的工作来解决安全问题，为他们的最终用户提高安全性。基于云的服务提供商意识到安全是重中之重，所以我认为这一趋势将会继续。随着越来越多的最终用户访问基于云的应用和内容，网络成为一个关键因素，因为网络现在位于您和您试图访问的内容之间。

*Mike Vizard:* 你认为坏人已经想通了这一切，他们现在把目标锁定在端点的人身上，看着这些稍微配置错误的网络？有时我们会使用 欺诈 RDP 协议。他们明白了吗？他们在追我们吗？

*Brian lavallée:*他们将永远存在，邪恶的行动者，坏的行动者，无论你怎么称呼他们，他们都将永远存在，但是云提供商，电信运营商，他们知道这一点，所以他们一直在升级，现代化，并在他们的网络中增加安全功能。想要利用网络和内容做不太好的事情的人与实际运营这些数据中心和网络的人之间的斗争将一直持续下去。我认为这将继续向前发展，这是人类的天性，但我们作为一个行业已经取得了很大的进步，在安全性方面跨越了 it、电信和基于云的服务。随着越来越多的人在家工作，我的意思是现在我们生活在可能是我们这个物种中最大的在家工作的实验中，如果你想想，网络，它在一些地方弯曲了，对吗？

大多数网络运行得越来越热，但它并没有坏掉。互联网没有断。许多人早期认为，如果每个人都搬回家，并不是你有更多的能力来家里，而是交通模式发生了变化。白天，当很多人去上学和工作时，交通是到学校和总部，但现在整个交通模式已经进入家庭。虽然很多人担心，但我认为电信运营商值得肯定。网络在一些地方弯曲了，但它并没有崩溃，他们只是在过去的一年中学到了很多，网络只会变得更加可靠。

*迈克·维扎德:* 在疫情之前，运营商试图鼓励人们卸载网络，使用 WiFi。现在我们有了 5G，人们在谈论 6G，谁知道之后会发生什么。我们是否已经到了可以让 5G 进入家庭的地步？这是否会在某个时候变得具有成本效益，我们将不会有所有这些 WiFi 路由器，或者它是否会一直是 5G，6G 和 WiFi 之间的某种平衡，我们必须找到它？

Brian Lavallée: 是的，这是我个人的观点，我认为它将永远是各种技术的结合。我在这个行业呆的时间够长了，知道一个技术网络 完全 消失，至少在我的职业生涯中不会。它总有用武之地。与 5G 相比，4G LTE 和以前的技术，甚至 WiFi 和前几代移动技术的巨大差异，带来了很多新的好处，对吗？如果我们看看更快的速度，这是得到最多媒体报道的，因为这是人们马上就会看到的，但它也带来了更低的延迟。我们谈论的是数量级更低的延迟。它连接到更广泛的连接设备，如传感器和智能设备，然后我们有了网络切片的整个概念，它可以保证端到端的性能。

您可以保证速度、延迟和弹性。我们在过去的任何一代移动网络中都没有过这种情况，而且在很大程度上，我们甚至在有线技术上也没有过这种情况。因此，5G 将直接进入你的家庭，如果你想一想，如果你是一个网络运营商，发送 5G 热点要容易得多，它可以是一个路由器，一个通往某人家庭的住宅网关，他们打开它，把它插在墙上，它会发现移动网络，然后嘣，你就可以在你家获得每秒 1gb 的容量。我认为，在许多情况下，这是未来的发展方向，但仍有一些技术挑战需要克服。但是，大多数运营商都在考虑将 5G 固定无线接入接入住宅，甚至是企业，作为其工具包中的一部分，以与有线技术竞争。

迈克·维扎德: 你认为在使用这种东西的最终用户中会出现反叛吗？当你听到他们抱怨工作经历时，你认为他们会在某个时候开始根据他们能获得的在家工作、在任何地方工作的经历来选择他们想去的公司吗？这一因素会比过去更多地影响招聘和保留吗？

耶。我的意思是，网络是当今至关重要的基础设施，尤其是对于出生和成长在不间断连接环境中的年轻一代。我认为这将继续向前发展。但是，随着越来越多的人在家工作，我知道我们听到了很多关于新常态的谈论，例如，工作场所会是什么样子，或者疫情事件平息后学校系统会是什么样子？我们真的不知道。但是，我认为我们知道的是，更多的人将在家工作。我在我的个人生活中看到它，我在我的同事中看到它，我在朋友和家人中看到它。他们的公司正在考虑每周至少在家工作几天。这样做对公司也有很多好处，所以网络会变得更加重要。

我认为，如果他们拥有的网络提供商不能提供他们需要的性能，人们会货比三家，这将成为一个竞争激烈的市场。我的意思是，现在竞争非常激烈，但我认为未来在速度、延迟和可用性方面会更激烈，这是高速连接到您家所需的三个主要因素。

迈克·维扎德: 我甚至可能有一天会把网络成本纳入我的薪酬包，但我们会看到会发生什么。

*Mike Vizard:* 你对 IT 人员的最佳建议是什么，他们是网络经理，他们是安全人员。他们应该在食物链的更高层与董事会进行什么样的对话，以真正实现下一波工作？

*Brian lavallée:*是的，我的意思是，回家的更高速度是很多人谈论的事情。我们之前刚刚谈到过，越来越多的人在家工作、娱乐和学习，因此容量将始终是讨论的焦点，也是一家运营商与另一家运营商竞争的焦点。但是，我认为增加安全功能，这可以通过宽带连接连接到您的家庭，因此，他们不仅可以购买高速管道连接到您的服务提供商，还可以为您提供 SASE 类型的功能，如安全、防火墙等。我认为考虑捆绑销售时，IT 服务与连接绝对是一条可行之路。这是一个与众不同的地方，也是人们想要的，因为如果你是一个普通的家庭用户，无论是为了玩、学习还是工作，你都不会在乎你买的是什么技术，只要它经济实惠并且能满足你的需求。

我想我给运营商的建议是，看看最近有什么可用的技术，一些较新的技术，虚拟化和 SASE 技术以及 SD _____，还有一大堆其他技术，并开始将这些技术打包成一个包，以解决用户的大部分需求，如果不是全部需求的话，然后您就可以在更高的速度之外实现差异化。
