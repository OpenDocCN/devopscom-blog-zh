# ASG 技术公司推进 DevOps 价值流的自动化、流程编排和安全性

> 原文：<https://devops.com/asg-technologies-advances-automation-orchestration-and-security-for-devops-value-streams/>

*ASG-企业协调器更新进一步使组织能够简化向客户的价值流，并加强从大型机到云的价值流中的开发合作*

[信息驱动型企业解决方案的领先提供商 ASG 科技](https://nam12.safelinks.protection.outlook.com/?url=https%3A%2F%2Fcts.businesswire.com%2Fct%2FCT%3Fid%3Dsmartlink%26url%3Dhttp%253A%252F%252Fwww.asg.com%252F%26esheet%3D52402766%26newsitemid%3D20210330005088%26lan%3Den-US%26anchor%3DASG%2BTechnologies%26index%3D1%26md5%3Db5e6caaf8b34d6bcaef03be902bd866b&data=04%7C01%7C%7Cdb1649938f184c964f3e08d8f37d8fab%7Cbf0b48c768944eb6bf538621676ccee4%7C0%7C0%7C637527067915668098%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C1000&sdata=afgeuiG0iVvXAB%2FkmENGBAhPL7yvx5vNYvul26mzjGQ%3D&reserved=0)今天发布了其 DevOps 价值流管理平台——ASG 企业流程管理器 4.2 的新功能。该版本进一步支持基础设施&运营(I & O)和开发运维团队可视化、编排和自动化从大型机到云的开发和运营价值流。4.2 版引入了新的功能，增强了 DevSecOps 并自动化了手动、耗时的流程，以实施价值流中的变更。

随着组织不断接受数字化转型和价值流变得更加复杂，ASG 企业流程管理器的进步使 I&O 和 DevOps 团队能够优化业务流程，提高资源效率，并通过以下方式加快向客户交付创新:

*   **多平台价值流编排:**从单一控制台到大型机的云
*   **DevOps 工具链自动化:**在整个软件开发生命周期中为流程编排提供动力
*   **价值流可视化和设计:**现代白板，降低复杂性

新的 ASG 企业流程编排功能增强了开发人员的工作能力，并实现了变更管理流程的自动化，包括:

*   **安全提升**:使开发人员能够创建价值流的变更包，这些变更包可以被具有适当安全性的人员接受到 QA 或生产中，从而提升端到端价值流中的开发人员
*   **批量更新:**允许自动更新价值流中的多个项目，消除了更新单独项目的手动工作，减少了时间和成本，提高了生产率

Mainline 合作伙伴开发主管 Ken Gross 表示:“为了应对当今的 IT 挑战，我们的客户正在寻找解决方案，帮助他们快速、轻松、安全地管理从大型机到云的混合 IT 基础架构中复杂的 DevOps 价值流。“通过与 ASG 的合作以及 ASG 企业流程编排的这些最新进展，我们能够满足这一需求，并帮助客户更快地创新和提供价值。”

ASG 产品管理总监 Anna Murry 表示:“去年的事件表明，it 领导者需要打破工作流程中存在的孤岛，以便更高效、更有效地管理端到端价值流。“ASG 企业协调器使我们的客户能够自动化和协调他们从大型机到云的价值流，并确保数据在整个工作流程中保持安全，从而将他们解放出来，专注于为其最终用户和客户提供正确的价值。”
