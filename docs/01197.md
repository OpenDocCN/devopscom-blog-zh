# New Node.js Foundation 调查报告了企业开发人员对新“全栈”的需求

> 原文:[https://devo PS . com/new-node-js-foundation-survey-reports-new-full-stack-demand-in-enterprise-developers/](https://devops.com/new-node-js-foundation-survey-reports-new-full-stack-demand-among-enterprise-developers/)

*近 50%接受调查的 Node.js 开发者使用容器技术，在云、前端、移动和设备领域出现强劲增长*

旧金山，2016 年 4 月 12 日—[Node.js 基金会](http://ctt.marketwire.com/?release=11G082331-001&id=8448115&type=0&url=https%3a%2f%2fnodejs.org%2fen%2ffoundation%2f)，一个由社区领导和行业支持的财团，旨在推进 Node.js 平台的发展，今天公布了其有史以来第一份 node . js 用户调查报告。

Node.js 基金会的调查发现，Node.js 拥有超过 350 万用户，年增长率为 100%，正在成为用于 web 应用程序、物联网和企业的通用平台/语言。它展示了对微服务架构、实时 web 应用、物联网(IoT)等新兴趋势的见解，并为前端、后端和物联网开发人员描绘了 Node.js 在生产和语言偏好(当前和未来)中使用的技术的详细图片。

Node.js 和容器一起起飞

Node.js 和 containers 都是高效开发和部署微服务架构的良好匹配。此外，尽管容器使用的激增相对较新，但参与调查的 45%的开发人员将 Node.js 与该技术结合使用。其他容器相关数据点:

*   58%被认为是物联网开发人员的受访者将 Node.js 与 Docker 结合使用。

*   39%被认为是后端开发人员的受访者使用 Node.js 和 Docker。

*   37%被认为是前端开发人员的受访者使用 Node.js 和 Docker。

node . js——驱动物联网的引擎

JavaScript 和 Node.js 已经成为物联网的首选语言和平台，因为它们都适合需要无中断并行编程的数据密集型环境。JavaScript，包括 Node.js 和 React 等框架，已经成为在这些互联、设备驱动的环境中工作的开发人员事实上的选择，96%的物联网受访者表示他们使用 JavaScript/Node.js 进行开发。

这些环境极具挑战性，调查显示，平均而言，使用 Node.js 的物联网开发人员比他们的前端和后端同行更有经验，超过 40%的受访物联网开发人员拥有超过 10 年的开发经验。

此外，尽管 Docker 是一种服务器技术，但许多物联网开发人员(58%)正在使用 Node.js 和 Docker，相比之下，只有 39%的后端开发人员使用 node . js。这一指标非常重要，因为它意味着新的物联网世界也在快速采用容器和微服务。

Node.js 成为通用语言

整个堆栈不再是“前端和后端”，而是“前端、后端和连接的设备”，这是从浏览器到烤面包机的所有东西的组合，所有这些都在 JavaScript 中运行并由 Node.js 支持。调查显示，62%的受访者在前端和后端开发中使用 Node.js，近 10%的受访者在前端、后端和物联网开发中使用 Node.js。

企业中普遍存在的 Node.js

Node.js 越来越多地在企业中使用，并在 PayPal、Go Daddy、Capital One 和 Intel 等大型企业中使用。Node.js 基金会调查的受访者备份 Node.js 在企业中日益增长的作用:

*   超过 45%的用户已经在使用 Node.js 长期支持版本(v4 ),该版本面向需要稳定性和高性能的大中型企业用户。

*   在尚未升级的受访者中，80%的人表示明确计划升级到 v4，一半的受访者计划今年升级。

*   34%的技术领导者对企业工具感兴趣。

全“均值”堆栈爆炸

实时、社交网络和互动游戏应用的流行正在开发者中推动一个新的堆栈。MEAN stack 能够处理大量并发连接和极高的可伸缩性，这正是这些应用程序所需要的。Node.js 与 MongoDB、Express、Angular.js 相结合，允许开发人员处理前端和后端开发的需求。毫不奇怪，所有这些技术通常都与 Node.js. Express 一起使用，被引用最多，平均有 83%的开发人员使用。

JavaScript 和 Node.js 的流行

JavaScript 和 Node.js 在后端、前端和物联网开发人员中很受欢迎。这三种开发人员都喜欢的其他语言包括 PHP、Python 和 Java。然而，展望未来，后端、前端和物联网开发人员计划减少对 Java 的使用。Net 和 PHP (PHP 平均下降 15%)并增加 Python 和 C++的使用。

关于调查

本次调查为期 15 天，从 2016 年 1 月 13 日至 1 月 28 日。在此期间，来自世界各地的 1760 人完成了调查。70%是开发人员，22%是技术管理人员，64%在生产中运行 Node.js。调查覆盖的地理代表:35%来自美国，22%来自欧洲大陆，6%来自印度，6%来自英国，其余的受访者来自亚洲、拉丁美洲、非洲、俄罗斯和中东。
