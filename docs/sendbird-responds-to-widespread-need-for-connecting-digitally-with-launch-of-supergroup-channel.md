# Sendbird 推出超群频道，响应数字连接的广泛需求

> 原文：<https://devops.com/sendbird-responds-to-widespread-need-for-connecting-digitally-with-launch-of-supergroup-channel/>

*现在，多达 20，000 人可以在群聊中互动，为大型在线社区和直播流重新创造了可能性*

*新的高级分析提供对 Sendbird 应用程序中的用户、渠道和消息的洞察*

**加利福尼亚州圣马特奥——2020 年 10 月 6 日——**[send bird](https://sendbird.com/)，一家领先的移动聊天、语音和视频以及 web 应用交互 API 公司，今天宣布了打破聊天界限的强大新功能。Sendbird 的 Supergroup 频道专为世界上最大、最活跃的社区设计，现在可以让多达 20，000 名用户参与实时私人群聊，这在行业内尚属首次。该公司还推出了开放频道的动态分区和高级分析，分析用户行为，并提供对 Sendbird 应用程序内频道和消息的详细见解。

**超群频道支持为新世界设计的大规模连接**

虽然具有有限聊天功能的开放频道已经存在了一段时间，但创建一个拥有聊天组所有丰富和参与功能的私人组从未达到过这种规模。这一行业新的工程壮举发生在世界各地的人们在新冠肺炎疫情中寻求与他人联系、分享想法和观点的方式之际。超群频道已经被世界上最多产的社区用来促进大规模的动态和真实的对话。

Sendbird 首席执行官兼创始人约翰·金表示:“规模一直是阻碍聊天和信息新应用的最大问题之一，我们很高兴能够消除这一障碍，特别是在当前大型面对面聚会受到限制的环境下。”。“超群频道是我们美丽新世界的完美功能，我们很高兴看到社区如何利用它。”

随着大型社区越来越多地寻求提供新型的人类体验，无论是直播音乐会还是其他类型的虚拟活动，超级团体首次亮相。超群频道提供了一种独特的选择，使与成千上万人的联系像在现代信使应用程序中与朋友交流一样引人注目。超级群组已经被成千上万的人用作目的地，他们从大型在线社区的影响者和专家那里寻求建议和技巧，而游戏社区正在使用超级群组来聚集用户，寻找玩家和交流想法，以私下推动游戏的发展。

印度最大的梦幻运动平台 Dream11 的首席产品官 Rahul Mirchandani 表示:“通过超级群组，Dream11 可以在同一个私人群组聊天中吸引成千上万的用户，让大型梦幻运动爱好者社区在一起玩耍的时刻建立联系，并推动 Dream11 应用程序的实时参与。”

超群的力量

超级群组支持标准的群组聊天功能，包括在成员之间发送和接收消息，共享媒体或文件消息，显示在线状态，显示发送或接收的最后一条消息，以及统计多达 100 条未读消息。此外，超群频道管理功能允许冻结频道，禁止或静音用户，并保护组免受垃圾邮件。组织也可以将 Sendbird 行业领先的审核工具应用到超级群体中。

Sendbird 继续创新，推出一种新的推送通知，称为智能推送，用于超群规模。这些通知让用户保持活跃或让他们回到对话中，并帮助超级群体在不影响参与度的情况下实现大规模对话。

超级群组进一步扩展了 Sendbird 最近推出的 [Announcements API](https://www.prweb.com/releases/sendbird_launches_announcements_api_to_help_companies_engage_users_amid_new_covid_19_behaviors/prweb17356585.htm) 的功能，它提供了一种机制，可以同时向异常庞大的活跃用户群营销或传递信息。超级团体和公告结合在一起，提供了一个强大的工具包，用于吸引和营销大量受众，无论是社区、直播流还是远程工作人员。

“由于 Sendbird 旨在以行业第一的规模支持嵌入式聊天，我们有能力继续推动行业向前发展，”Kim 补充道。“没有人能够与 Sendbird 竞争，它能够持续提供引人注目的新聊天体验和建立联系的机会，也无法提供同等水平的用户行为洞察。”

除了超群，Sendbird 还宣布了开放频道的动态分区。动态分区可以动态地将大型开放式频道群聊分成较小的虚拟群组，这样用户就可以感觉他们是大型群聊的一部分，但接收消息的速度会带来愉快的体验。动态分区在后台工作，对确保大规模的实时流媒体群聊体验有着巨大的影响。

**高级分析帮助 Sendbird 客户了解并更好地服务他们的用户**

Sendbird 还推出了高级分析(AA ),这是一项超越之前仪表板分析的高级功能。通过高级分析，可以即时访问自与 Sendbird Chat 集成的应用程序发布以来汇总的所有指标。这使得能够从用户数据中收集到更相关、更有意义和更务实的见解。

Sendbird 的高级分析通过消息数量、每个用户的消息、新用户、已删除用户、已停用用户、新频道、活跃频道、消息发送者和消息查看者等指标来描述用户行为。有了这些更先进、更准确、更及时的指标，就可以轻松分析用户行为。然后，可以将结果应用于决策制定，从而改善整体用户体验。
