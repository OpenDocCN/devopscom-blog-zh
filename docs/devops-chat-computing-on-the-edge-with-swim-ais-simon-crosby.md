> 原文：<https://devops.com/devops-chat-computing-on-the-edge-with-swim-ais-simon-crosby/>

西蒙·克罗斯比的职业生涯一直处于技术的边缘。从 Xen 虚拟机管理程序到 Bromium 微管理器以及更多其他产品，Crosby 领导了业内一些最具创新性的技术开发人员。

He is now the CTO of [Swim.ai](https://www.swim.ai/) which is literally bringing computing to the edge (almost an anti-cloud). This approach promises some revolutionary capabilities.

如果西蒙参与其中，它也很有可能成功。听听西蒙现在是怎么回事。
