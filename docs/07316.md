# Logi Analytics 宣布为 BlueVolt 平台提供新的分析仪表板体验

> 原文：<https://devops.com/logi-analytics-announces-new-analytics-dashboard-experience-for-bluevolt-platform/>

*借助 Logi Info，BlueVolt 为平台配备了新的分析和报告仪表板功能，以支持专注于渠道的合作伙伴*

**弗吉尼亚州麦克林—2021 年 1 月 26 日** —今天，[领先的软件团队嵌入式分析解决方案提供商 Logi Analytics](https://www.logianalytics.com/) 和屡获殊荣的销售和渠道营销在线学习解决方案权威机构 [BlueVolt](https://www.bluevolt.com/) 宣布建立新的合作伙伴关系，利用 Logi Info 为所有 BlueVolt 客户创建先进的分析仪表板体验。借助即将推出的 BlueVolt analytics 仪表盘，客户将比以往更好地了解他们的渠道合作伙伴教育和培训战略。对于专注于渠道的公司来说，持续的参与、培训和伙伴关系发展对长期业务成功至关重要。但是并不是所有的培训和营销策略都是一样的。事实上，许多企业继续将时间、金钱和人员投入到外联战略中，这些战略不能提供及时的数据或足够的 ROI。这就是为什么 BlueVolt 投资 Logi Analytics，为 BlueVolt 平台客户创建一套新的动态且易于理解的嵌入式分析仪表板。这些仪表板为用户提供了快速利用实时更新和定位公司走向成功所需的工具和洞察力。

Logi Analytics 产品管理副总裁 Charles Caldwell 表示:“BlueVolt 的平台是一个完美的例子，说明了为什么在应用程序中嵌入强大、可操作的见解是在现代市场中实现更好业务增长的关键。“我们的解决方案使最终用户能够汇总、分析数据并根据数据洞察采取行动，而无需离开他们的应用程序工作流。借助 Logi Info，BlueVolt 可以为其合作伙伴定制和创建全新的分析和报告体验。”

基于 Logi Symphony 解决方案套件的定制化和灵活性，BlueVolt 最初选择 Logi Analytics 来满足其分析需求。现在，BlueVolt 可以访问完整的 Logi Analytics 产品组合，特别是包括当前平台更新的 Logi 信息。事实证明，这些解决方案对 BlueVolt 的合作伙伴至关重要，他们都需要快速分析和响应实时市场和客户数据，以获得竞争优势。

“我们的合作伙伴需要做出数据驱动的决策，他们需要一种在整个过程中可视化数据的方法，”BlueVolt 战略和业务发展副总裁 Gaven Singh 说。“德勤最近的研究表明，在商业智能上投入的每一美元可以带来近 30 美元的生产率增长，证明更聪明的企业更强大。借助 BlueVolt 的新 BI 功能，我们的合作伙伴将获得实时查看所有数据的好处，而分析对于实现这一点至关重要。”
