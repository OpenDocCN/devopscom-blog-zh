# 酱实验室深化对 OSS 社区的投资；建立新的开源项目办公室

> 原文：<https://devops.com/sauce-labs-deepens-investment-in-oss-community-establishes-new-open-source-program-office/>

*新部门推动开源计划支持的显著增长*

**旧金山-2020 年 2 月 27 日-**[](https://www.saucelabs.com/)**[酱料实验室有限公司](https://www.saucelabs.com/)，提供[连续测试](https://saucelabs.com/solutions/continuous-testing)解决方案的领先供应商，今天宣布成立其首个开源项目办公室，进一步加强其在更广泛的开源社区的重大投资。新部门将开发、定义和实施公司的长期开源战略，同时为增加对高价值社区项目的贡献铺平道路。该办公室将由该公司最近任命的首席技术官约翰·凯利领导，并将包括著名的开源专家和社区贡献者，如克里斯蒂安·布罗曼、迭戈·莫利纳和艾萨克·穆奇。**

 **“这个办公室的建立是对开源是我们公司灵魂的认可，”Kelly 说。“我们是 Selenium 的创始人之一，我们在开源领域的领先地位是如此之多的客户信任我们来建立数字信心的原因。作为一家公司和一个社区，我们只是触及了开源的皮毛，不仅是像 Selenium、Appium 和 WebDriverIO 这样的成熟项目，还有像 Puppeteer 和剧作家这样的新项目。机会是无限的，我们很高兴能够增加我们的贡献。”

除了为自己的解决方案产品建立和执行公司的开源战略之外，新部门将直接与已建立和新出现的开源项目的领导者合作，以推动所有 Sauce 实验室之间的合作。该集团还将继续发展和培育公司广受欢迎的长期“ [Open Sauce](https://saucelabs.com/solutions/open-source) 计划，该计划为合格的开源项目提供免费访问其持续测试平台的机会。

**酱油专家被任命为硒领导委员会成员**

就在新的开源项目办公室成立的几天前，多名 Sauce Labs 专家被任命为 Selenium 项目的关键领导委员会成员，Selenium 项目是 web 应用程序自动化测试的事实上的标准开源工具。Sauce Labs 现场服务高级总监兼 Selenium 社区的长期领导者 Marcus Merrell 被任命为 Selenium 项目领导委员会成员，而高级解决方案架构师 Molina 和 Titus Fortner 被任命为新成立的 Selenium 技术领导委员会成员。这些任命是上周在基辅举行的硒营会议上宣布的。

该公司与开源社区的深厚联系也超越了 Selenium。Murchie 目前是 Appium 的项目负责人，app ium 是移动应用程序自动化测试的事实上的标准，Bromann 是 WebDriverIO 的项目负责人，WebDriverIO 是基于 JavaScript 的跨浏览器测试自动化框架。此外，Bromann、Fortner、Merrell 和 Molina 是最近 W3C 浏览器测试和工具工作组中仅有的供应商代表，该工作组认可 WebDriver 为“推荐”标准。该公司还密切参与了许多维护开源项目的基金会，包括 OpenJS 基金会，Bromann 是 Cross Project Council 的成员，帮助促进项目和更大社区之间的合作。

该公司在开源社区扩大投资正值一个显著增长的时期。现在，每天在其持续测试云平台上运行近 300 万次测试，该平台涵盖数千个浏览器/操作系统、移动仿真器、模拟器和真实设备组合。

**关于酱实验室**

Sauce Labs 是持续测试解决方案的领先提供商，提供数字信心。Sauce Labs 持续测试云提供了客户应用体验的 360 度视图，确保 web 和移动应用在每一个浏览器、操作系统和设备上的外观、功能和性能始终如一。酱油实验室是一家由多巴资本、Salesforce Ventures、Centerview Capital Technology、IVP、Adams Street Partners 和 Riverwood Capital 资助的私人控股公司。更多信息，请访问[https://saucelabs.com](https://saucelabs.com/)。**
