## 我被耍了吗？有 38%的几率

本周，特洛伊·亨特本已庞大的被盗证件数据库出现了巨大的增长。如果你还没看过的话，HaveIBeenPwned.com 是一个惊人的免费服务，它让应用程序安全地检查用户的密码是否已经在一个**受损网站**的其他地方使用过。

### 分析:尽快将 API 构建到您的授权流中

英国相当于联邦调查局的机构已经捐赠了一个巨大的凭证宝库——在重复数据删除之后，它已经向数据库添加了超过 2.25 亿个新密码。坦率地说，如果你仍然允许你的用户依赖 HIBP 中 8 . 47 亿个密码中的任何一个，it **将会疏忽**。

[西蒙·夏伍德](https://www.theregister.com/2021/12/21/nca_finds_255m_fresh_stolen_passwords/ "read the full text") : **国家犯罪署发现 2.25 亿个此前未被曝光的密码**

特洛伊·亨特(Troy Hunt)的《我被赋予了(HIBP)名声》昨天宣布……NCA 与 HIBP 分享了 585，570，857 个[凭证],亨特说是 225，665，425 个……在 HIBP 已经存储的 6.13 亿个凭证中，他以前从未见过。[HIBP]允许任何人对被盗密码进行安全搜索，以检查他们的凭据是否已被暴露。NCA 在给亨特的声明中没有透露密码宝库的来源[但表示他们]“能够在一个受损的云存储设施中识别出大量的……密码。…很明显，这些凭据是已知和未知的被破坏数据集的累积。”
…
【The】的发布使密码总数达到 847，223，402，增长了 38%。

这与 DevOps 有什么关系？[特洛伊·亨特](https://www.troyhunt.com/open-source-pwned-passwords-with-fbi-feed-and-225m-new-nca-passwords-is-now-live/ "read the full text")解释道:
