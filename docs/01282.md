维也纳和西雅图—([商业资讯](http://www.businesswire.com/))—商业自动化软件的领导者 Automic 今天宣布，它已被任命为新的“Ovum 决策矩阵:选择 DevOps 发布管理解决方案，2016–17”[研究报告](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Foffers.automic.com%2Fovum-decision-matrix-selecting-a-devops-release-management-solution&esheet=51340091&newsitemid=20160511005859&lan=en-US&anchor=research+report&index=1&md5=0b040df5b673807221a216f4190e4558)的“领导者”。根据该组织提供“企业级版本管理和自动化功能”的能力，Automic 被评为关键供应商。

[报告](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Foffers.automic.com%2Fovum-decision-matrix-selecting-a-devops-release-management-solution&esheet=51340091&newsitemid=20160511005859&lan=en-US&anchor=The+report&index=2&md5=aa65e16a67a7e9c9b1e6e370b94a76a9)强调了 DevOps 版本管理解决方案为希望通过减少手动错误和打破 IT 孤岛之间的壁垒来加快应用交付的组织提供的价值。Ovum 观察到，DevOps 生态系统中使用的工具，如发布自动化，通过提高发布频率和确保面向客户的新应用和服务的质量，使企业受益。

该报告强调了[Automic Release Automation](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fautomic.com%2Fdevops-ara&esheet=51340091&newsitemid=20160511005859&lan=en-US&anchor=Automic+Release+Automation&index=3&md5=d1ee308f5d198f4d44bdc88739dcaa98)(ARA)的能力和优势:

*   使用户能够从使用通用 UI、通用集成集以及统一平台中受益，从而提供对所有自动化处理的端到端可见性和控制。
*   提供可扩展性、弹性和安全性，并对活动进行完全可审计的跟踪和报告。
*   提供强大的部署撤消功能，将环境恢复到部署前的状态。
*   提供无代理或基于代理的部署技术选择。

Automic Software 首席执行官 Todd DeLaughter 表示:“我们很高兴在 Ovum 最新的 DevOps 发布管理报告中被认可为领导者。"[应用发布自动化](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fautomic.com%2Fdevops-ara&esheet=51340091&newsitemid=20160511005859&lan=en-US&anchor=Application+Release+Automation&index=4&md5=01b194e69fab72906d1cd6124aaf8c61)为经历数字化转型的客户提供所需的敏捷性和速度。该报告证实了我们在提供一流的发布自动化技术以满足他们需求方面的方法。”

Ovum 分析师 Michael Azoff 评论道:“Automic 解决方案超越了 DevOps 版本管理的覆盖范围，使用户能够从使用通用 UI、一组通用集成以及底层平台中受益，并在 it 运营领域实现全面自动化”。
