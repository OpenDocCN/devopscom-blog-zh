# 网络研讨会:最重要的 Redis 指标

> 原文:[https://devo PS . com/webinar-optimizing-your-docker-registry-with-redis-and-data dog/](https://devops.com/webinar-optimizing-your-docker-registry-with-redis-and-datadog/)

Redis 提供了一个快速持久的内存存储系统、高级缓存和支持多种数据类型的能力。Redis 的功能不同于 IT 团队过去使用的其他键值存储或数据库。这些差异可能会导致内存交换、处理命令的延迟和按键删除等问题。如果不加以解决，所有这些都会影响应用程序的性能。

识别和解决源自 Redis 的问题需要理解这个键值存储是如何工作的，以及对其性能指标的可见性。在本次网络研讨会中，Datadog 的宣传员马特·威廉姆斯以及 Datadog 的宣传员马特·威廉姆斯和 Redis Labs 的开发者倡导者戴夫·尼尔森将向您展示:

*   如何以及在哪里访问关键的 Redis 数据
*   如何解释 Redis 指标来检测问题
*   如何解决这些问题，或者一起避免它们
