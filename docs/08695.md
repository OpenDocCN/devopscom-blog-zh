# 关于 DOI

> 原文：<https://devops.com/about-doi/>

DevOps Institute (DOI)成立于 2015 年 3 月，是新兴 DevOps 实践的全球学习社区。DOI 的目标是根据来自最受尊敬的资源的最新指导，在企业范围内建立 DevOps 培训和认证的标准。我们由来自 DevOps、敏捷和 ITSM 空间的思想领袖组成的董事会指导。我们正在与 PEOPLECERT 和 DevOps.com 等全球领先企业建立强大的联盟。

DOI 不是一个培训机构；所有教育和认证都是通过我们的全球注册教育合作伙伴(代表)网络和作为我们考试机构的 PEOPLECERT 提供的。自我们推出以来，数百名候选人参加了 DevOps Foundation 和认证敏捷服务经理课程和考试，并获得了积极的反馈和结果。

为什么我们需要 DevOps 学院？

DevOps 没有单一的知识体系。它几乎触及了 It 的每个方面，从文化到自动化到指标等等。虽然缺乏单一的知识体系可能会让一些人感到沮丧，但包容性和集体知识体系的发展使 DevOps 能够继续发展和有机适应。

在更高的层面上，那太好了。然而，希望将其环境转变为更快、更协作和更自动化的企业正在寻求更正式的培训和认证方法，以确保他们的团队完全具备全面的知识、技能和能力。这就是土井的用武之地。

DOI 努力将新兴的 DevOps 指南转化为有意义且可持续的培训和认证。我们通过不懈地研究广泛的 DevOps 集体知识体系、聘请主题专家、参加会议、采访案例研究以及关注大规模 DevOps 的发展来做到这一点。我们跟踪和引领那些在真实组织中带来真实价值的创新。然后，我们的工作小组和主题专家将某个主题的所有知识封装到实际和有形的培训中。在适当的情况下，培训将附带认证考试，以展示个人在特定主题领域的技能和资格。成功的候选人将成为 DevOps 研究所的成员。

正规的培训和认证课程有什么好处？

*   创建一个共同的心态和词汇
*   鼓励对话、分享和协作
*   激发想法和创新
*   提供可持续和可操作的步骤
*   教授一些可以帮助团队、组织和个人提高的东西
*   创造全面的专业知识
*   获得认证后，给个人一种自豪感和成就感

您是否注意到 DOI 更关注“新兴实践”而不是“最佳实践”？我们相信 DevOps 是一个新兴的领域，新的实践在这里被频繁地发现、验证和推广。我们也相信 DevOps 的分享和协作精神，永远不会把候选人指定为“大师”或“专家”相反，DOI 的培训和资格将支持更现代的角色方法，包括 DevOps 对适应性、透明度和组织敏捷性的文化关注。

在 DevOps 学院附近，我们希望提供:

*   来自我们的代表和其他人的即时知识文章，重点关注新兴的 DevOps 实践或相关框架
*   关于当前和未来培训课程和资格的信息
*   访问全球注册教育合作伙伴提供的课程日历
*   视频访谈、模板和其他资产，帮助您完成开发运维之旅

所以，欢迎来到我们的社区。我们希望您在我们的分享中发现价值，并经常回来。
