# DevOps 聊天:应用程序注意力指数和数字反射时代，与 AppDynamics

> 原文：<https://devops.com/devops-chat-app-attention-index-and-the-era-of-digital-reflex-with-appdynamics/>

应用性能公司 App dynamics(2017 年被思科收购)近日发布了其 2019 年版 App 关注指数报告。2019 年的报告副标题为数字反射时代。AppDynamics 区域首席技术官兼技术战略 Steve Long 将与我一起参加本期 DevOps 聊天。

数字连接几乎是不变的。通过数字应用和交易与我们的世界互动无处不在:使用 OpenTable 预订餐厅，使用应用或我们的智能手机数字钱包进行支付，检查我们的智能手机以计划和安排我们的一天。因此这份报告的名字叫做《数字反射时代》。

除了报告中的见解之外，Steve 和我还讨论了衡量客户体验、将绩效与期望的业务成果联系起来、发现问题和减少解决问题所需时间的策略。加入我们这场关于应用性能、客户体验和业务成果的精彩讨论。
