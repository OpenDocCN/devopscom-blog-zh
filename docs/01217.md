# 威胁堆栈获得 1530 万美元的 B 轮融资，以提高云安全团队的效率

*规模创投合伙人领投一轮；M* *市场需求和客户快速增长是 B 轮融资的关键*

**BOSTON—2016 年 4 月 19 日**—[威胁堆栈](http://www.threatstack.com/)，云安全和合规管理领域的领导者，今天宣布获得 1530 万美元的 B 轮融资，由 Scale Venture Partners (Scale)领投，所有现有投资者参与，包括帮凶和. 406 Ventures。Threat Stack 今天还宣布，Scale 的合伙人 Ariel Tseitlin 将加入公司董事会。最新一轮融资使威胁堆栈的总投资超过 2600 万美元。

威胁堆栈 B 轮融资紧随该公司 2015 年[创纪录的客户增长年](https://www.threatstack.com/press-releases/threat-stack-reports-100-percent-bookings-growth-brings-on-strategic-market)和 2016 年初[威胁堆栈云安全平台(CSP)](https://www.threatstack.com/platform) 的推出。新资本将用于在下一年将员工人数增加一倍，支持销售、营销和工程，以进一步增强产品能力和用户体验。未来几个月，Threat Stack 还将利用资金支持向波士顿新总部的扩张。

“我们很高兴将威胁堆栈纳入我们强大的云业务组合。Threat Stack 快速的季度环比增长和业界领先的云安全方法使这一投资对我们来说极具吸引力。Tseitlin 说:“云原生应用的安全需求和市场上安全工具的功能之间存在巨大差距，而 Threat Stack 通过一种易于使用和扩展的独特产品填补了这一差距。“随着各种规模的企业继续在云中快速安全地构建，我期待着 Threat Stack 在市场上的进一步扩张。”

Interactive Intelligence 的高级安全工程师 Jarrod Sexton 表示:“Threat Stack 的云安全平台通过替换单点工具和更高效地使用专用于我们云环境的安全资源，使我们能够更快地发展。“通过使我们能够更快、更经济地识别和解决安全问题，威胁堆栈已被证明是一个非常有价值的合作伙伴。”

云原生、完全集成的威胁堆栈云安全平台支持跨所有环境、云安全提供商和容器的安全洞察，消除了对碎片化解决方案的需求，并降低了 IT 领导者和 CISOs 面临的安全问题的复杂性。借助来自云环境各个方面的集成安全数据，客户可以更快地解决问题，并降低总体拥有成本。Threat Stack 的 CSP 被医疗保健、金融科技和 SaaS 等行业的公司使用。

“自从我一年前加入 Threat Stack 以来，我们经历了令人难以置信的业务发展势头。Threat Stack 董事长兼首席执行官 Brian M. Ahern 表示:“我们的增长不仅体现在我们的客户群中，还体现在我们作为一个团队积极努力构建安全和合规性解决方案，以满足现代云基础设施不断增长的需求。“最新一轮融资验证了我们的市场策略，并将推动威胁堆栈云安全平台的进一步增强，以提高运营效率，同时解决所有行业客户的安全和合规性挑战。”

当今发展最快的公司正依靠现代云基础设施来加快上市速度、创新和规模。资源非常宝贵，而安全风险比以往任何时候都大。根据最近企业战略集团(ESG)对网络安全和 IT 专业人员的调查,“34%的人说他们的组织发现跨云监控工作负载很有挑战性。这可能与混合云中的工作负载有关，也可能表明跨异构私有云或公共云基础架构的工作负载面临监控挑战。”
