# 塔塔咨询服务公司与 PagerDuty 合作，加速关键行业的数字化转型

> 原文：<https://devops.com/tata-consultancy-services-partners-with-pagerduty-to-accelerate-digital-transformation-across-key-industries/>

*新的全球战略联盟将帮助企业实现运营现代化，通过特定行业的解决方案和服务实现数字化商业成功*

旧金山—[全球领先的数字运营管理公司 PagerDuty，Inc.](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.pagerduty.com%2F&esheet=52336519&newsitemid=20201201005031&lan=en-US&anchor=PagerDuty%2C+Inc.&index=1&md5=65cbbda85ecdf848e252df5755a0d903) (NYSE:PD)今天宣布与全球领先的 IT 服务、咨询和业务解决方案组织 [Tata Consulting Services](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.tcs.com%2F&esheet=52336519&newsitemid=20201201005031&lan=en-US&anchor=Tata+Consulting+Services&index=2&md5=ff8e6a71043232b0f969e7bb08f90bfd) 结成全球联盟，共同开发基于人工智能和自动化的新一代 IT 解决方案，旨在加速数字化转型计划，实现关键垂直行业运营的现代化。

PagerDuty 将与 TCS 合作建立“PagerDuty-TCS 解决方案服务中心”，专注于为基础设施管理服务、云迁移、ITOps 和 DevOps 现代化、客户服务、安全运营和行业特定运营的共同客户开发和实施数字解决方案。

依靠数字服务和应用来生活、工作、学习和联系，各行各业正在经历快速的数字化转型。例如，美国电子商务在短短三个月内经历了 [10 年的创新增长](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.mckinsey.com%2Fbusiness-functions%2Fmarketing-and-sales%2Fsolutions%2Fperiscope%2Four-insights%2Fsurveys%2Freinventing-retail%3Fcid%3Dother-soc-fce-mip-mck-oth---%26sid%3D3564267558%26linkId%3D96442478&esheet=52336519&newsitemid=20201201005031&lan=en-US&anchor=10+years+of+innovation+growth+in+just+three+months&index=3&md5=6651640b828fcbac0d49ad68a9d90d83)。为了满足全球公司的新需求和加速增长路径，TCS 成立了一个技术小组，与 PagerDuty 等公司合作，设计和实施跨金融、零售、技术、电信、媒体、制造等领域的现代化创新 IT 解决方案。

“TCS 的人工智能和自动化解决方案和服务正在帮助客户实现数字化运营转型，以实现业务现代化，”TCS HiTech 和联盟与技术部全球负责人 Raman Venkatraman 表示。我们很高兴能与 PagerDuty 合作开发尖端的企业解决方案，在这种新的数字化和分布式工作方式中蓬勃发展。"

TCS 和 page duty 将通过为全球企业带来 page duty 的数字运营管理平台，提供事件管理、事件响应和 AIOps 的新功能，以及高级见解和分析，帮助客户实现数字运营转型。PagerDuty 使团队能够使用其人工智能和[自动化](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.pagerduty.com%2Fnewsroom%2Fnew-digital-ops-platform-machine-learning-automation%2F&esheet=52336519&newsitemid=20201201005031&lan=en-US&anchor=automation&index=4&md5=d29204502ebdc2e011554b6ce24c9dd3)自信地扩展服务并加速数字计划，使组织能够将解决事件的速度提高 70%。两家公司将携手合作，帮助客户将运营从缓慢、手动和被动转变为实时、自动化和智能，这对于在当今永不停机的数字世界中取得成功至关重要。

“数字优先是组织的现实。然而，推动无缝数字体验的技术非常复杂，许多组织没有资源和内部专业知识来有效地推动这一转变，”PagerDuty 首席收入官 Dave Justice 说。“TCS 专注于为企业解决数字化转型挑战。随着整个世界都转移到网上，新冠肺炎带来了前所未有的需求。我们很高兴能让 PagerDuty 在 TCS 的新一代解决方案中发挥关键作用，实现数字化运营现代化和创新。"

作为这一新合作伙伴关系的一部分，TCS 将加入面向解决方案合作伙伴和托管服务提供商的 page duty[合作伙伴计划](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.pagerduty.com%2Fpartner-with-pagerduty%2F&esheet=52336519&newsitemid=20201201005031&lan=en-US&anchor=Partner+Program&index=5&md5=31566dfec31a5722057fd193886f34be)，在该计划中，他们可以在全球范围内提供分销、转售、专业服务和托管服务，并根据客户需求创建新的解决方案。
