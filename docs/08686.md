# DBmaestro DevOps 平台 7.5 发布，支持 PostgreSQL

> 原文：<https://devops.com/dbmaestro-devops-platform-7-5-released-with-postgresql-support/>

*为 DB2、SQL、Oracle 和 PostgreSQL 管理多数据库环境*

**马萨诸塞州波士顿2018 年 10 月 11 日–[db maestro](https://www.dbmaestro.com/)，数据库解决方案提供商的先驱和领先的 DevOps 今天宣布推出 DBmaestro DevOps Platform 7.5，该平台支持 PostgreSQL 的 CI/CD 和发布自动化。PostgreSQL 正成为越来越多的企业和小型组织的首选数据库。它目前在全球数百个数据库中排名第四(更多信息见 [DB-Engines 排名](https://db-engines.com/en/ranking))。**

DBmaestro 的 DevOps Platform 7.5 包括数据库发布自动化解决方案，可自信、可控地简化数据库变更，同时帮助公司缩短应用上市时间。快速采用连续交付和部署方法来加速应用程序开发，产生了对数据库发布自动化工具的强烈需求，以防止错误和停机。根据 [2018 DevOps for Database 报告](https://www3.dbmaestro.com/devops-for-database-survey-2018-results)，83%的公司已经采用了 DevOps，而数据库的持续交付则落后了，采用率只有 37%。

DBmaestro 的 DevOps Platform 7.5 零摩擦解决方案使 DevOps 团队和数据库专业人员能够更轻松地管理多个数据库，而无需更改现有流程。DBmaestro 为跨平台和操作系统(包括 Oracle、DB2、Microsoft SQL Server 和现在的 PostgreSQL)的 CI/CD 采用提供了一个总体解决方案，满足了各行业和跨国企业不断发展的数据库发布自动化需求。

DBmaestro 首席技术官 Yaniv Yehuda 表示:“我们的 DevOps 平台让 DevOps 和数据库团队能够实施 DevOps 最佳实践，以高度的信心、最小的风险和最高的效率加快上市时间，从而释放 PostgreSQL 数据库的强大功能。

DBmaestro 将于 10 月 22 日至 24 日在举办的 [DOES18](https://events.itrevolution.com/us/sponsors/) 展会上展示 DBmaestro DevOps 平台 7.5 解决方案，展位号为 811。

**关于 DBmaestro**

DBmaestro 为数据库带来了 DevOps 最佳实践，为数据库提供了新水平的效率、速度、安全性和流程集成。DBmaestro 的平台使组织能够安全有序地运行数据库部署，提高开发团队的工作效率，并显著缩短上市时间。该解决方案使组织能够针对数据库活动实施 CI/CD 实践，并具有可重复的管道发布自动化和自动漂移预防机制。欲了解更多信息，请访问[www.dbmaestro.com](http://www.dbmaestro.com/)。
