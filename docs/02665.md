 **各行各业越来越多的公司认识到软件交付的战略重要性，他们在数字化转型上下大赌注，以增加他们的竞争优势。Electric Cloud 首席执行官 Carmine Napolitano 表示:“我们在全球企业中看到的强大牵引力是我们为客户提供的技术创新和屡获殊荣的解决方案的直接结果，以支持他们的数字化转型努力。“我们的客户生动地证明了大型复杂组织可以在数字化转型中取得巨大进步。这种规模的高性能不再只是独角兽的专利。”

**devo PS 和连续交付的事实标准**

金融、保险、高科技制造和酒店业的众多知名客户为电云令人兴奋的一年画上了句号，2018 年新业务增长了 95%，同时创下了该公司有史以来最高的订阅净保留率，接近 115%。市场正在用行动投票，因为现有客户和新客户都在大幅增加对电云产品的投资。

Electric Cloud 的 ElectricFlow 已经成为包括 AdvancedMD、AGCO、E*TRADE、三星、TopDesk 和高通在内的许多组织的应用发布流程编排的事实标准，因为它能够:

*   快速实施和共享安全、可重复和适应性强的管道，交付大型机、单片和微服务应用
*   对企业版本的运行状况和状态提供前所未有的、可操作的洞察，而无需手动收集和分析数据
*   随着开发运维流程在整个组织中的推广，实现可靠的扩展

**支持向现代技术的转变**

2018 年，电云加入了云原生计算基金会，目标是通过下一代技术帮助企业实现遗留应用和基础设施的现代化，包括 PaaS、Multicloud、微服务、容器、CI/CD 和 DevOps。

“我们很高兴地欢迎电云成为 CNCF 的一员，”云本地计算基金会的执行董事 Dan Kohn 说。“随着企业越来越多地采用云原生技术，我们认为持续交付和版本编排对于利用这些技术至关重要。我们很高兴拥有像 Electric Cloud 这样的 CNCF 成员，他们可以帮助企业企业实现云原生之旅，特别是在确保从开发到生产的安全性和合规性方面。”
