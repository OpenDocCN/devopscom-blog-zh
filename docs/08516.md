# Moogsoft 宣布 Observe 扩展其 AIOps 平台，在源头进行实时分析和 IT 问题检测

> 原文：<https://devops.com/moogsoft-announces-observe-expanding-its-aiops-platform-with-real-time-analysis-and-it-problem-detection-at-the-source/>

*突破性的新技术让 IT 团队能够全面观察影响客户的问题，无论这些问题发生在何处，无论是在内部还是在云环境中*

**旧金山—2018 年 8 月 30 日—** 面向 IT 运营的人工智能(AIOps)的先驱和领先提供商 Moogsoft 今天宣布推出 Moogsoft Observe。这项突破性的技术将 Moogsoft 的核心 AIOps 平台功能从集中式分析扩展到数据源，为 IT 团队提供了对影响客户的问题的绝对可见性和完全可观察性，无论这些问题发生在何处。

Moogsoft Observe 是世界上唯一一项实时吸收时间序列和指标数据并应用人工智能从问题源头检测事件的技术。Observe 仅存储异常和上下文相关的数据，为 IT 团队提供高度先进的特定知识来改进他们的在线服务和应用程序，同时大幅降低数据传输、接收和存储成本。不需要数据湖或数据科学家来操作，包括站点可靠性工程师和开发人员在内的 IT 通才可以轻松使用 Observe。

Moogsoft 首席执行官 Phil Tee 表示:“通过 Observe，我们正在提升 AIOps 的能力，以帮助 IT 团队在面对数据的大规模扩散和分散时更好地管理他们的服务和应用。与其他工具不同，Observe 使 IT 和 DevOps 团队能够在客户所在的任何位置检测到问题，并让他们能够更快地确定服务中断的根本原因，而无需在一个集中位置收集和分析所有数据的成本、复杂性和延迟

Moogsoft Observe 由一套正在申请专利的新算法提供支持，这套算法是为全球最大的企业环境中最复杂的 IT 用例而开发的。通过在源位置接收和分析数据，并仅集中存储异常数据和上下文数据，Observe 能够对相关的、有意义的数据进行趋势和历史分析，但成本更低，延迟显著减少，检测问题的时间减少高达 60%，每比特信息的存储和通信成本减少近 90%。

测试版 Observe 的首席技术官 Eli Chen 表示:“在我们的纯云和敏捷环境中，我们需要一种能够执行异常检测并主动通知我们有问题的解决方案——让人们站在仪表盘前不再是一种选择。”。“Moogsoft 以人工智能为中心的方法来解决这个问题，并通过其 Observe 产品将其扩展到时间序列数据分析，这正是解决这个问题的正确方法。”

Observe 专为企业打造，是一种单代理技术，可以跨所有环境运行，包括云实例、工作负载和容器，以及内部系统和应用程序。

Observe 是 Moogsoft AIOps 平台的扩展，但可以独立部署。早期访问测试程序将于今年 11 月推出。欲了解更多关于观察和注册测试版的信息，[点击这里](https://www.moogsoft.com/product/observe?utm_source=PR&utm_medium=referral&utm_campaign=Observe)。
