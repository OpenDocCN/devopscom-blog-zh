# Platform.sh 向 Magento Commerce 提供 PaaS 技术

> 原文:[https://devo PS . com/platform-sh-provides-PAAs-technology-magento-commerce/](https://devops.com/platform-sh-provides-paas-technology-magento-commerce/)

*全球商务提供商将 Platform.sh 集成到其新的企业云产品中*

法国巴黎(PRWEB)2016 年 4 月 12 日

今天，持续交付平台即服务(PaaS)云托管公司 Platform.sh 宣布，Magento Commerce 为其新推出的 [Magento](https://magento.com/products/enterprise-edition "Enterprise Cloud eCommerce from Magento") 企业云版产品选择了该公司的第二代 PaaS。旗舰开源 Magento 平台已经是全球电子商务的领导者，为全球 250，000 多个网站提供支持，每年支持超过 500 亿美元的商品交易总额。

Magento 希望为他们的云产品找到一个战略 PaaS 合作伙伴，并找到一个解决方案来帮助消除在开发和发布管理周期中管理多个环境的传统负担。这些功能使企业能够降低总体开发成本，加快网站开发速度，同时消除生产服务更高变更率的风险。

Platform.sh 首席执行官 Fred Plais 表示:“看到全球电子商务行业中如此强大的力量对其产品组合进行如此急剧的改变，以向其客户提供更大的业务优势，这是对我们技术的令人难以置信的验证。这正是我们设计 Platform.sh 要做的事情，远远超出了公共平台即服务的范围，我们很高兴这一影响将如此深远。”

在 Platform.sh 吸引 Magento 的显著特性中，有一项是动态克隆技术，该技术消除了系统管理杂务，能够对整个实时网站进行逐字节复制，使代码、文件、服务和数据保持同步。超越 DevOps，这种“NoOps”方法在分支环境之间保持完美的一致性，这本身对于实时站点支持是有价值的，但对于开发团队来说更是如此，他们现在可以按需访问生产环境的完美一致的副本。这意味着每一项新功能都可以在自己的真实系统副本中并行、独立地进行开发和测试，消除冲刺阶段的瓶颈，让首席技术官和 CMO 相信，在投入生产时，它会完全按照预期运行。

“有了 Magento Cloud 旗下的 Platform.sh 和我们的持续部署能力，CTO 和 CMO 全年都可以发挥作用，尤其是在销售旺季，当你真正需要这种竞争优势时，其他人都在为同样的季节性价格而战。”Platform.sh 的首席商务官 Kieron Sambrook-Smith 说，“谁会想到 PaaS 提供商能做到这一点！”

Platform.sh 加速了许多常见的开发和部署流程，从而实现了:

*   支持票证数量和 SysAdmin / DevOps 减少高达 90%,
*   更高的开发效率(超过 20%)，
*   极大地提高了批准和用户接受度测试(UAT)——因为每个功能分支都有自己的 URL 要发布(速度提高 40%)。
*   通过端到端流程自动化实现更高级别的持续集成(CI ),因为许多常见的测试和跟踪工具现在能够在任何 Platform.sh 项目中启动活动，通过 API 请求用于各种目的的环境的即时副本。

Magento Commerce 战略主管 Peter Sheldon 表示:“我们评估了 Magento Enterprise Cloud Edition 的许多 PaaS 提供商，但 Platform.sh 是我们的明显选择，因为他们专注于 PHP 社区、电子商务背景以及他们的 PaaS 提供的创新的持续云集成工具。“您能够针对新版本的通用服务测试功能甚至整个应用程序的速度和轻松程度正在改变游戏规则，组件越多，项目越大越复杂，Platform.sh 带来的价值就越大。”

Platform.sh PaaS 的设计考虑了连续交付和 99.99%的高可用性。向上或向下扩展计算和存储资源既快速又无缝，同时对堆栈和核心应用程序组件进行关键更改不会被购物客户注意到。事实上，在去年 11 月为期 4 天的黑色星期五周末，我们能够宣布没有一个 Platform.sh 客户经历过任何应用程序宕机，尽管存在所有常见问题(其中 70%是在线商家)。这往往显示了 Platform.sh 提供的价值:阻止收入损失的卓越控制，以及在流量高峰期进行实时收入优化变化的能力。这种精细的控制在在线零售商需要的时候为他们提供了巨大的竞争优势。

关于 Platform.sh

Platform.sh 是一个开创性的 web 应用程序托管和开发工具。它将分支-合并工作流扩展到整个基础设施，这样每个新功能都可以像在生产中一样进行测试。实时网站可以毫不费力地扩展，以满足最苛刻的流量需求。全天候支持与基于微容器的独特三冗余架构和创新的云控制技术相结合，使该服务即使在最复杂的使用情形下也能提供 99.99%的 SLA。该公司既通过其公共平台 https://platform.sh 直接向最终客户提供服务，也通过第三方提供服务。基于开源技术，Platform.sh 是敏捷软件开发、持续集成(CI)和持续交付(CD)的理想选择。

该公司是一家总部位于法国巴黎的风投支持的初创公司，其技术、支持和销售人员遍布欧洲、美国和亚洲，并为包括时尚、出版、零售、旅游、工程和公共部门在内的各个领域的数千家公司提供云服务。
