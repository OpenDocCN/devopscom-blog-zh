# Argo con 21:通过 Kubernetes 完成更多工作

> 原文：<https://devops.com/argocon-21-get-more-done-with-kubernetes/>

Argo 是一个开源项目套件，帮助 DevOps 团队更快速、更安全地交付软件。Argo Workflows、Argo CD、Argo Rollouts 和 Argo Events 是 Kubernetes 运行工作流、管理集群和正确执行 GitOps 的开源工具。12 月 8 日，Argo 社区将聚集在 [ArgoCon '21 虚拟活动](https://www.techstrongevents.com/argocon21)上，分享他们的 Argo 之旅，一些关于该项目的鲜为人知的细节，并提供对未来可能发生的事情的一瞥。

[Argo con’21](https://www.techstrongevents.com/argocon21)的议程包括技术会议、演示、深度探索和案例研究，涵盖各种主题，包括:

*   GitOps 工作流如何与 DevOps 运动相关联，以及它如何为成功采用 DevOps 铺平道路
*   如何使用 Argo 工作流、事件、CD 和展示为 CI/CD 创建灵活、可定制的开发人员体验
*   Argo CD 制作最佳实践以及如何定制您的 Argo CD 设置
*   使用 Argo 工作流构建医疗级人工智能
*   通过云原生分布式 ML 管道融入 Python 生态系统
*   如何使团队能够跨多个多租户集群管理他们自己的基于 Argo 的 CI/CD 管道
*   Argo 的可扩展性如何
*   GitOps-用 Argo CD 和 Codefresh 丰富您的 K8s 管理
