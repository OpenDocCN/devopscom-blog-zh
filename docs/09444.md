我们 Appstellar 非常激动地宣布，我们现在是 CNCF 银会员、Kubernetes 认证服务提供商(KCSP)和 Linux 基金会银会员！在 [CNCF 新成员公告](https://www.cncf.io/announcements/2022/05/18/cloud-native-computing-foundation-hits-800-member-milestone-at-kubecon-cloudnativecon-europe-2022/)上了解更多信息。

这是一个重要的里程碑，展示了 Appstellar 作为云和 DevOps 解决方案提供商的专业知识，帮助客户踏上他们的 Kubernetes 和云原生之旅。为了来到这里，我们做了很多工作，所以我们非常自豪和激动，终于获得了 KCSP 徽章。

由于我们的服务严格专注于云软件开发和 DevOps 解决方案，我们已经为金融科技、健康科技等行业的多个客户实施了 Kubernetes 实践。话虽如此，KCSP 认证将有助于展示 Appstellar 在支持组织迁移到云原生平台方面的丰富专业知识。

作为拥有[云原生计算基础](https://www.cncf.io/)的 Kubernetes 认证服务提供商，Appstellar 将继续为希望迁移到云技术或改善其现有云基础设施的企业和组织提供 Kubernetes 支持、咨询和专业服务。

另一方面，作为 CNCF 和 [Linux 基金会](https://linuxfoundation.org/)的成员，Appstellar 致力于扩展 Kubernetes 和云原生生态系统，并弥合开发者和运营之间的差距。

Appstellar 的联合创始人 Rozafe Llalloshi 和 Mehmetali Shaqiri 说:“我们很荣幸成为科索沃和西巴尔干地区首家 Kubernetes 认证服务提供商。

至于 Appstellar，我们目前正处于高速增长模式。在过去的 6 个月里，我们的团队增长了 122%！此外，我们已经签署了多个新的客户交易，在新的运营行业中进行了扩展，并且还搬到了新的办公场所。

除了这些合作关系，Appstellar 还成为了 AWS 的合作伙伴。有了这些伙伴关系，我们期待着进一步加速我们的增长，并接受我们喜欢的项目。
