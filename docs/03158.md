**Electric Cloud 首席执行官 Carmine Napolitano 表示:“我们相信，连续第三年在 Gartner 的研究中被评为领导者，并在愿景完整性方面被定位为最具竞争力，这突显了 Electric Cloud 对市场领先创新的承诺。“ElectricFlow 为复杂的企业管道和发布提供了改变游戏规则的控制和可见性，涵盖基于单片、大型机和微服务的应用的任何组合。随着我们的客户继续在 DevOps 上下大赌注，我们帮助他们安全地适应并利用正在进行的大规模和持续的技术变革。我们每天都在努力工作，让我们的客户能够将软件发布从繁琐的工作转变为真正的竞争优势。”**

****从自动化到流程编排****

**去年的魔力象限关注的是应用发布**自动化**。今年，Gartner 将关注范围从自动化扩展到**编排**。**

**Gartner 的 2018 年魔力象限和应用发布协调的相关关键能力根据未来市场需求能力评估了供应商，包括 1)部署自动化，2)管道和环境管理，3)发布协调，4)可扩展性和性能，以及 5)可用性和访问。**

**根据这些标准，Electric Cloud 因其“愿景的完整性”而被定位在最远的位置。**

**“为了保持相关性，组织在 DevOps 上下了很大的赌注——不仅是在一个团队中，而是在许多团队中。随着应用架构和工具链的转变，更频繁的开发周期使基础设施和运营团队不堪重负，数字化转型面临风险，”Napolitano 补充道。“我们很荣幸被这么多行业领先的公司选中来协调他们的软件交付管道，以实现更快、更好、更安全的大规模软件交付。我们的用户信任我们的核心业务，我们将持续提供服务。”**

****将发布从琐事转变为竞争优势****

**如今，软件创新的步伐等同于市场成功，安全高效地发布高质量的软件是所有公司的关键需求。有很多方法可以实现 ARO，但是客户越来越多地选择 ElectricFlow 来实现他们的发布。电气云实现了 220%的年同比新业务增长，净客户保持率超过 100%，表明客户满意度和扩张势头强劲。**

**我们最近的一个客户，一家财富 100 强金融服务公司，解释了他们如何通过 ElectricFlow 快速提升其 DevOps 转型。DevOps 技术负责人表示:“我们能够很快开始实施几个有用的管道。“我们发现它具有很强的适应性，我们创建的管道易于扩展，因此我们能够很好地整合开发人员推荐的新架构、技术和工具，而不用担心下游出现任何问题。这种不断适应的能力让我们在市场上获得了不公平的优势——我们将利用这一优势为任何规模的最终用户提供可靠、安全的服务。”**

*   **对企业版本运行状况和状态的前所未有、可操作的洞察，无需手动收集和分析数据**
*   **该解决方案将随着他们在整个组织中推广开发运维流程而可靠地扩展**
