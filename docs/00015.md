# ***F5 的参考架构和解决方案支持服务提供商优化、保护和盈利其移动网络***

西雅图—(美国商业资讯)—[F5 Networks](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ff5.com%2F&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=F5+Networks&index=1&md5=4f04c6f55356320bf72d8a1eda156c22)(纳斯达克: [FFIV](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.google.com%2Ffinance%3Fq%3DFFIV&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=FFIV&index=2&md5=49ac895b0a0d01df853762ce1e145374) )今天宣布推出产品和参考架构，扩展其[F5 Synthesis](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fsynthesis.f5.com%2F&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=F5+Synthesis&index=3&md5=7c7ec42e237dcfddaf263226de1877ff)^(框架，为服务提供商带来新的优势。F5 将在下周于巴塞罗那举行的[世界移动通信大会](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.mobileworldcongress.com%2F&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=Mobile+World+Congress&index=4&md5=78a7260e459830da465cdbbbc557a9ac)上展示这些参考架构，并展示其服务提供商产品的优势。)

F5 Synthesis 提供了一个弹性服务平台，通过交付[软件定义的应用服务](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fdevcentral.f5.com%2Farticles%2Ff5-synthesis-software-defined-application-services&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=Software+Defined+Application+Services&index=6&md5=d9a36566cae4032e80de5069f0fc57d7)^((SDAS^()来帮助服务提供商有效扩展其网络并实现新兴技术的优势，例如[网络功能虚拟化](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.f5.com%2Fpdf%2Fwhite-papers%2Fservice-provider-nfv-white-paper.pdf&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=Network+Functions+Virtualization&index=5&md5=7f9ec3d3a317a9cbb2a68e88734eda9f) (NFV)。SDAS 通过高效、智能地结合虚拟化、可扩展性、可编程性、流程编排和其他功能，充分利用 F5 基于结构的方法。))

F5 Synthesis 愿景利用核心 F5 技术，包括本周早些时候发布的技术，以及专注于数据、信令和应用层面的增强型服务提供商解决方案，进一步实现安全、优化和协调的应用服务交付。F5 帮助服务提供商采用服务驱动的业务实践，通过政策、用户身份、Diameter 信令和其他环境和应用感知解决方案来实现。此外，通过以高效、统一的方式提供服务，创新的 [服务提供商产品](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ff5.com%2Fproducts%2Fservice-provider-products&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=service+provider+products&index=9&md5=5311d0f75e4d1688bd35a6e60c12b87b) 帮助组织管理和货币化服务进入市场的方式。

作为架构框架，F5 合成:

*   将物理、虚拟和基于云的部署相结合，形成统一、灵活的高性能服务结构。这为服务提供商实现了更高效的网络架构，以及向高级 LTE 服务和 NFV 等新兴技术的优化迁移路径。
*   通过开放的基于 REST 的 API，借助 SDAS 的动态可编程性，提供敏捷、协调的应用服务创建和自动化功能，允许与第三方管理和协调系统集成。
*   为 F5 应用服务提供灵活的许可模式，使服务提供商能够经济高效地按需为所有应用扩展有价值的服务，并根据需要扩展敏捷技术结构。
*   支持跨网络、应用和设备的动态、多层安全性。
*   通过显著降低关键服务的总拥有成本并加快创收增值服务的交付，为服务提供商提供先进的“应用驱动”功能、服务体验和更高的经济效益。 ¹
*   符合当前和发展中的行业惯例。F5 支持影响服务提供商的标准和指南，包括 IETF、3GPP、GSMA、ETSI NFV 以及 ONF 等 SDN 相关标准。
*   提供以服务提供商为中心的参考体系结构，就如何应对不断发展的业务挑战提供规范性指导。

**主要优势**

新的 F5 技术和解决方案继续扩大公司的流量和信令管理解决方案组合，使服务提供商能够经济高效地构建和扩展其网络，为客户实现高服务速度和高质量的体验，并通过引入创新服务和业务模式进一步实现网络盈利。统一的 F5 Synthesis 架构框架通过以下方式为服务提供商转变应用服务的经济性、体验和支持:

*   **释放宽带服务，实现最佳性能**–**F5 是领先的供应商，提供智能服务链、动态策略执行、Diameter 路由和[互通](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ff5.com%2Fglossary%2Finterworking-function-iwf&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=interworking&index=10&md5=0ebaa1d33e4bc0c1e65e5fb39ac62682)以及 DNS 功能，以支持新的服务和业务模式，并增强移动宽带能力。**
*   ****支持多层安全以获得最大保护**–**为了支持更灵活的功能，LTE 网络天生不如其 3G 和 2G 前身安全。这可能会使移动网络面临更多重大威胁，同时人们越来越期望移动运营商能够为用户、网络和应用提供更好的保护。F5 的安全解决方案旨在以行业领先的性能和规模保护终端用户设备、网络和云部署。****
*   ******为最终价值提供经得起未来考验的规模和可扩展性**–F5 为组织提供了一个多服务平台，可提供重要的网络服务，如安全性、策略实施、本地 DNS、IPv6 迁移和内容过滤，并通过可编程的 [iRules](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fdevcentral.f5.com%2Firules%2Fgetting-started&esheet=50806984&newsitemid=20140219005492&lan=en-US&anchor=iRules&index=11&md5=58d06f4759ef980555d48032bd0dae13) 功能对流量策略和导向进行额外的精细控制。F5 提供无与伦比的规模和可扩展性，为未来提供进一步的投资保护。****
*   ******提供与技术合作伙伴生态系统的集成点—**F5 全面的可编程性和 API 为技术合作伙伴在联合产品和互操作性方面的合作提供了更多机会。F5 解决方案经过了广泛的评估，旨在促进与下一代服务提供商生态系统中技术合作伙伴的无缝集成。****
