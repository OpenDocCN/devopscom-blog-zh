**旧金山，2019 年 3 月 27 日**——连续交付即服务平台 Harness 今天宣布[现可在亚马逊网络服务(AWS)市场](https://c212.net/c/link/?t=0&l=en&o=2415731-1&h=1142118730&u=https%3A%2F%2Faws.amazon.com%2Fmarketplace%2Fpp%2FB07PZY3369&a=now+available+for+purchase+on+Amazon+Web+Services+(AWS)+Marketplace)购买——让 AWS 客户能够访问 Harness 的解决方案，以自动化连续交付流程，并快速、安全地将工件交付给生产，并重复使用机器学习(ML)和模板化管道。Harness 最近还获得了 AWS 合作伙伴网络(APN)的标准技术合作伙伴资格。

Harness 为开发人员和工程团队带来了企业连续交付的力量，使他们能够快速移动和交付代码，而不用担心部署失败。凭借其智能自动化技术，Harness 有助于自动化整个连续交付流程。通过将无监督的机器学习应用于该过程(一种称为持续验证的新技术)，该平台可以了解应用的基线环境，并可以在检测到不规则活动时启动自动回滚，从而避免应用停机或大范围故障。

Linedata 是一家上市公司，使用 Harness 以及亚马逊弹性计算云(亚马逊 EC2)、亚马逊弹性容器服务(亚马逊 ECS)的 AWS Fargate、亚马逊关系数据库服务(亚马逊 RDS)、亚马逊 Route 53、亚马逊简单存储服务(亚马逊 S3)、亚马逊弹性容器注册(亚马逊 ECR)和 AWS Lambda。line data DevOps 总监 Andrey Budzar 表示:“通过为我们的团队提供自助服务，我们能够向非技术业务部门介绍 devo PS 方法的优势，为速度和质量指明方向，并支持 BizDevOps (DevOps 2.0)。“通过利用 Harness 和 AWS 的强大功能，我们将部署时间缩短了 80%，验证时间缩短了 80%，并加强了工程和业务之间的反馈回路，从而节省了大量成本，同时提高了创新速度，并为我们的客户实现了价值最大化。”

Harness 联合创始人兼首席执行官 Jyoti Bansal 表示:“Harness 客户正在以闪电般的速度向 AWS 部署应用程序，因此，随着我们继续快速发展，并向全球资源紧张的工程和开发团队提供我们的平台，与 AWS 合作并加入 AWS Marketplace 是 Harness 的天然选择。“我们认为持续交付是一个非常复杂的问题，今天，人们正在使用过时的工具来解决这个问题:Jenkins 管道、脚本和大量的手动监督。Harness Continuous Delivery platform 专为复杂的云原生应用程序而设计，这些应用程序是 AWS 客户的生命线。”

线束连续交付平台的主要功能包括:

*   **管道建造商。**devo PS 团队无需花费数小时来构建管道，只需几分钟就能构建和执行完整的连续交付管道，并在其应用、服务和环境中使用串行或并行工作流。
*   **工作流向导。**亚马逊 EKS、亚马逊 ECS、AWS Lambda 和 AWS Auto Scale 组上对金丝雀部署、蓝/绿部署和滚动更新的现成支持使复杂的部署变得简单。
*   **持续验证。**通过利用无人监管的机器学习来自动验证生产中的应用部署，发布团队最终可以确定部署是成功还是失败。他们可以从 Amazon CloudWatch、AppDynamics、New Relic、Splunk、Elastic 和 Sumo Logic 等工具中检测错误质量和回归，并使用 Harness 24×7 ServiceGuard 功能无限期监控发布的性能。
*   **自动回滚。**与痛苦的手动回滚不同，Harness 可以自动回滚到最后一个工作工件版本和运行时配置，而无需脚本或代码。
*   **持续安全。**在部署流程的每一步中，线束都具有安全性。Harness 使用其基于 AWS 密钥管理服务(AWS KMS)或 deep HashiCorp Vault 集成构建的 SecretStore，在所有部署工作流和管道中无缝引用机密。Harness 还保留每个部署的完整审计跟踪，因此团队知道每个操作背后的人员、内容、地点和时间。
*   **实时交付分析。**开发运维团队和工程团队领导现在对每个应用、环境、版本和部署都有了深入的了解。团队可以在几秒钟内调试部署，基于角色的访问融入到平台的每个部分。

Harness 可以免费试用两周，并且可以在 https://aws.amazon.com/marketplace/pp/B07PZY3369 的 AWS 市场上直接购买。
