# SmartBear 收购泽法，以扩大其市场领先的测试管理产品组合，并加强 Atlassian 市场联盟

> 原文：<https://devops.com/smartbear-acquiring-zephyr-to-expand-its-market-leading-test-management-portfolio-and-strengthen-atlassian-marketplace-alliance/>

将 Atlassian Marketplace 上票房第一的泽法吉拉应用程序添加到 SmartBear 产品组合中，创建了最全面的测试管理解决方案集

马萨诸塞州萨默维尔。加利福尼亚州圣何塞——2018 年 8 月 28 日——团队软件质量工具的领导者 SmartBear 将收购泽法，以创建市场上最全面的测试管理解决方案。通过这次收购，SmartBear 加强了其承诺，为软件开发团队提供最具创新性的工具，以更快地构建、测试和监控优秀的软件。泽法在吉拉的受欢迎程度已经使其成为 Atlassian Marketplace 上票房第一的应用程序，有 18000 个团队使用其工具执行了超过 4000 万次测试。SmartBear 将继续扩展和深化泽法产品线，并通过利用 SmartBear 工具为 Atlassian 市场的用户带来更多价值，加强与 Atlassian 的关系。

“我们很高兴泽法加入智能熊团队，”智能熊首席执行官贾斯汀·蒂格说。“对泽法的收购将确立 SmartBear 在测试管理领域的领导地位，并扩大我们高影响力、易于使用的工具组合，包括 SoapUI、TestComplete、SwaggerHub、CrossBrowserTesting、Collaborator 和 AlertSite。”

测试管理是能够以数字业务需求的速度开发、测试和部署高质量软件的关键组成部分。然而，每个软件开发团队的测试管理需求都是独特的，这取决于他们的开发方法、组织结构和过程。通过将泽法加入产品组合，SmartBear 现在提供了市场上最全面的测试管理产品。吉拉的泽法支持希望在吉拉进行本地测试管理的 Atlassian 客户，Hiptest 支持采用行为驱动开发(BDD)和持续测试的敏捷和 DevOps
团队，泽法企业支持寻找惠普质量中心的现代替代品的企业。

泽法首席执行官 Scott Johnson 表示:“我们创建了泽法来帮助组织不断提高软件测试人员和开发人员的效率和协作。“通过利用行业专业知识和一系列 SmartBear 解决方案，我们的客户将继续受益于他们熟悉和喜爱的工具，同时现在能够解决与跨 UI 和 API 层构建、测试和监控软件应用程序相关的其他软件开发挑战。”

Atlassian 首席执行官 Scott Farquhar 表示:“Atlassian Marketplace 使 Atlassian 的 125，000 多家客户能够无缝扩展 Atlassian 工具的功能。“我们很高兴能与 SmartBear 更紧密地合作，推进泽法产品线，并将更多 SmartBear 行业领先的软件开发和测试工具引入 Atlassian 市场。”451 Research 的首席分析师 Carl Lehmann 说:“人们越来越需要集成的生命周期管理工具来支持各种软件开发环境。“SmartBear 对泽法的收购代表了我们在现代数字企业中看到的加速趋势，即加强对多样化开发和测试工具链的支持。”

关于 smart bear Software
smart bear 在 194 个国家支持超过 600 万名软件专业人员和超过 22，000 家公司，是团队软件质量工具的领导者。该公司的产品有助于提供最高质量和最好的 SmartBear 收购泽法，以扩大其市场领先的测试管理产品组合，并加强 Atlassian Market Alliance 执行软件的可能性，同时帮助团队以几乎不可能的速度运送代码。SmartBear 的产品用于移动、web 和桌面应用的 API 测试、UI 测试、代码审查和性能监控，为每个开发、测试和运营团队成员提供工具，以确保 Twitter 或脸书每个阶段的质量。
