# Linux 学院调查显示，云技术人才和经验存在巨大的技能缺口

> 原文：<https://devops.com/linux-academy-survey-shows-vast-skills-gap-cloud-technology-talent-experience/>

**德克萨斯州达拉斯****–2017 年 1 月**–根据最重要的在线 Linux 和云培训平台和社区 Linux Academy 最近的一项调查，大多数企业认为劳动力中缺乏熟练的 Linux 和云技术专业人员，并且当前的专业人员没有得到他们需要的培训。

当 Linux Academy 就 Linux 和云技术培训趋势对近 900 名 IT 专业人员进行调查时，76%的人表示全球缺乏具备适当 Linux 和云技术技能的人才；大多数受访者表示，他们的公司要求他们定期接受不同级别的 Linux 和云技术培训；超过三分之一的受访者认为，他们共事过的 IT 专业人员对这些技术知之甚少或毫无经验。

“公司应该注意这些调查结果，特别是随着开源和云技术的持续扩散，”Linux 学院创始人安东尼·詹姆斯说。“这些技术的进步显然超过了能够服务和管理它们的专业人员的水平。当专业人员接受他们需要的培训时，技术已经进步，使他们的培训变得过时。这不仅强调了获得及时且负担得起的培训的必要性，还强调了公司进一步投资员工技能的必要性。”

接受 Linux 和云技术培训的两个最大障碍是缺乏时间和资金，38%的人支付一门课程的费用高达 500 美元。总共有 77%的受访者表示，他们的公司每年在员工的 Linux 和云培训上仅花费 25，000 美元。60%的公司有 1 到 20 名员工管理这些技术。

当谈到招聘 Linux 和云专业人员时，总共有 72%的受访者表示，他们所在的领域很难找到人才，而且很难找到人才；67%的人表示，他们总是或有时很难招聘到这些专业人士。公司招聘面临的最大挑战是普遍缺乏技能人才，以及缺乏找到和吸引他们的资源。大多数公司需要四到六周的时间来填补 Linux 和云相关的职位。

IT 专业人员认为最有益的培训主题是 Linux、AWS 和 DevOps。虽然答案因行业而异，包括技术、医疗保健、金融、政府、制造、教育、商业/企业、零售和其他，但大多数受访者认为 RHCSA 和 RHCE 等正式认证对其垂直行业有些重要。

“调查结果表明，特定的技术技能和相关认证在职业发展中发挥着重要作用。詹姆斯补充说:“在线培训提供商可以帮助专业人士在实际、真实的环境中以更快、更经济的速度学习这些技能，是帮助缩小人才差距的最佳选择。”
