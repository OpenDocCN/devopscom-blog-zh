# Harness 向其持续集成开源社区提交新的开发体验

> 原文:[https://devo PS . com/harness-commits-new-dev-experience-to-its-continuous-integration-open-source-community/](https://devops.com/harness-commits-new-dev-experience-to-its-continuous-integration-open-source-community/)

Harness CI Community Edition(又名 Drone)现在包括管道执行可视化、实时调试功能以及企业级安全性和治理

旧金山——2021 年 3 月 18 日——[领先的软件交付平台 Harness](https://harness.io/) ，今天宣布了其开源模块 Harness 持续集成(CI)社区版(前身为 Drone.io)的重大更新。除了旨在反映 Harness 行业领先的软件交付平台的新开发体验和用户界面，今天的版本还为开源贡献者提供了新的管道执行可视化、高级治理和安全控制以及实时 CI 管道调试功能，以便在创纪录的时间内构建和测试代码。

今天的软件开发和工程团队浪费了大量宝贵的时间和资源来等待管道构建，只是为了在一个管道崩溃时匆忙寻找解决方案。为了缓解这种令人沮丧且耗时的循环，Harness 增强了其 marquee 开源 CI 模块，通过一系列更新，允许开发人员快速应对故障，并创建一条更短的创新之路。更名为 Harness CI Community Edition，如果构建确实中断，开发人员现在会看到一个到故障 CI 服务器的链接，使他们可以轻松地进行调试。

“收购 Drone 后，Harness 将为 OSS 社区提供简化和加速软件交付所需的投资和工具作为我们的使命，”Drone.io 的首席执行官兼创始人 Brad Rydzewski 说。“我们很自豪不仅能够传递这一信息并满足当今负担过重的工程师的需求，而且能够提供超越专有供应商的功能。”

今天的新闻标志着 Harness 在 2020 年 8 月收购无人机后对开源软件社区的第一个重大贡献。自 2013 年成立以来，Drone 已经从 DockerHub 获得了超过 1 亿次下载，拥有 50，000 名月活跃用户，是 GitHub 上最受欢迎的 CI/CD 项目，有 21，319 个赞，比 Jenkins 多 5，500 个，比 Spinnaker 多 13，000 个，比 Jenkins-X 多 18，000 个。Harness CI Community Edition 的其他更新包括:

*   新的开发体验:一个圆滑、现代的用户界面，设计用于匹配其他 Harness 产品模块，为在 Harness 模块之间工作的开发人员创建了一个平滑的过渡。

*   管道执行可视化:开发人员可以通过可视化地图准确了解管道步骤的执行时间和顺序。与在日志文件中以文本形式查看管道相比，这是一种更简单、更快速的跟踪复杂管道的方法。

*   实时 CI 管道调试:调试在开发人员的计算机上运行而不是在 CI 服务器上运行的构建可能是一项耗时且困难的任务。通过实时 CI 管道调试，开发人员可以快速解决 CI 实例上发生的构建失败。

*   增强的治理和安全性:管理人员不再需要通过命令行或 API 添加用户和控制对管道的访问。CI 系统在一个更快、更简单的用户界面中进行管理，该界面也是可扩展的。

Harness 将继续投资、创新和支持开源社区，计划在 2021 年发布更多的 OSS。要了解更多关于 Harness CI 社区版的信息，请访问[https://harness.io/](https://harness.io/)。
