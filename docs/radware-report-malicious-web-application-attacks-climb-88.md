# Radware 报告:恶意 Web 应用程序攻击攀升 88%

> 原文：<https://devops.com/radware-report-malicious-web-application-attacks-climb-88/>

*其他发现:企业面临越来越多负责云环境的“专业”攻击者*

**美国新泽西州马赫沃 2021 年 3 月 10 日** [**—**](https://bit.ly/3ki5zZS) [领先的网络安全和应用交付解决方案提供商 Radware](https://bit.ly/3ki5zZS) [](https://bit.ly/3ki5zZS)(纳斯达克股票代码:RDWR)，今天发布了其[2021](https://www.radware.com/pleaseregister.aspx/?returnurl=1d0f012c-ab3c-4277-a9e5-00566d3c63f3)[—](https://www.radware.com/pleaseregister.aspx/?returnurl=1d0f012c-ab3c-4277-a9e5-00566d3c63f3)[2022 年全球威胁分析报告](https://www.radware.com/pleaseregister.aspx/?returnurl=1d0f012c-ab3c-4277-a9e5-00566d3c63f3) [。](https://www.radware.com/pleaseregister.aspx/?returnurl=1d0f012c-ab3c-4277-a9e5-00566d3c63f3)报告结果强调 2021 年是网络应用攻击年。2020 年至 2021 年间，恶意 web 应用程序请求的数量攀升了 88%，是分布式拒绝服务(DDoS)攻击年增长率的两倍多，2020 年的年增长率为 37%。

然而，web 应用攻击前所未有的增长并没有阻止 DDoS 在 2021 年成名。该报告详细介绍了去年多个破纪录的 DDoS 攻击和拒绝赎金服务(RDoS)是如何在威胁格局中赢得一席之地的。在大型攻击成为头条新闻的同时，经常未被发现的微洪水攻击的数量与 2020 年相比上升了近 80%。

“统计数据讲述了一个关于糟糕演员的故事。他们变得越来越聪明，越来越有组织，越来越有针对性地追求他们的目标——无论是为了金钱、名誉还是政治事业，”Radware 威胁情报总监帕斯卡尔·吉内恩斯说。“此外，网络犯罪分子正在改变他们的攻击模式，从利用更大的攻击媒介转向在更复杂的防范活动中结合多种媒介。勒索软件运营商及其附属机构，现在包括 DDoS 雇佣演员，正在以全新的专业和纪律水平开展工作，这是我们以前从未见过的。”

Radware 的《2021-2022 年全球威胁分析报告》回顾了 2021 年最重要的网络安全事件，并提供了对 DDoS 和 web 应用攻击发展以及主动网络扫描趋势的详细见解。该报告的关键要点包括:

*   **云规模的 DDoS 攻击在预测中:**随着越来越多的企业将关键资源和应用迁移到公共云，攻击者正在调整他们的战术和技术，以适应公共云提供商的规模。虽然企业不应该立即对大规模攻击的报告感到震惊，但他们确实需要意识到 DDoS 攻击是其威胁形势的一部分，无论其地理位置或行业如何。在公共云中托管服务的公司需要为云规模的攻击做好准备。
*   **赎金拒绝服务(RDoS)团伙掌权:**2020 年，针对没有按时支付赎金的组织的 DDoS 攻击有所增加。2021 年，RDoS 通过几次活动证实了其在 DDoS 威胁领域的普遍存在。这包括针对全球 VoIP 提供商的攻击，引发了对关键基础设施的担忧。
*   **勒索软件运营商转向三重勒索:**2021 年，更老练、组织更好的运营商改进了他们的战术，为他们的武器库增加了更多勒索能力。为了让不情愿的受害者回到谈判桌前，他们发起了三重勒索活动，不仅结合了密码锁定和数据泄露，还结合了 DDoS 攻击。因此，由勒索软件运营商支持的蓬勃发展的地下经济正在出现对 DDoS 出租服务的新需求。
*   **微洪水大放异彩:**虽然 2020 年至 2021 年间大型攻击媒介(10Gbps 以上)的数量下降了 5%，但微洪水(1Gbps 以下)和应用程序级攻击却上升了近 80%。通过在更长的时间内精明地组合大量微洪水，攻击者使组织面临更大的风险，他们必须不断增加基础设施资源，如带宽、网络和服务器处理，直到服务变得成本过高。

《2021-2022 年全球威胁分析报告》的其他主要结果包括:

**DDoS 攻击**

与 2020 年相比，2021 年每个客户的恶意 DDoS 事件数量增加了 37%。欧洲、中东、非洲(EMEA)和美洲各占 2021 年攻击量的 40%，而亚太地区占 20%。

与 2020 年相比，2021 年每个客户的平均 DDoS 攻击量增长了 26%。

2021 年受攻击最多的行业是游戏和零售，在标准化的基础上，这两个行业各占攻击量的 22%。这两个行业依次是政府(13%)、医疗保健(12%)、技术(9%)和金融(6%)。

**网络应用攻击**

从 2020 年到 2021 年，恶意 web 应用程序请求的数量增长了 88%。破坏访问控制和注入攻击占 web 应用程序攻击的 75%以上。

2021 年受攻击最多的行业是银行和金融，以及 SaaS 提供商，共占 web 应用程序攻击的 28%以上。零售和高科技行业排名第三和第四，各占网络安全事件的近 12%，其次是制造业(9%)、政府(6%)、运营商(6%)和运输业(5%)。
