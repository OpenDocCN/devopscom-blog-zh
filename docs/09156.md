# Sumo Logic 提高了自动化程度，为现代安全运营提供了更多的协作、更短的调查和响应时间

> 原文:[https://devo PS . com/sumo-logic-boosts-automation-to-delivery-increase-collaboration-shorter-investigation-and-response-times-for-modern-security-operations/](https://devops.com/sumo-logic-boosts-automation-to-deliver-increased-collaboration-shorter-investigation-and-response-times-for-modern-security-operations/)

*Sumo Logic 将作战室和应用程序中心功能引入行业领先的 Cloud SOAR*

**加利福尼亚州雷德伍德城，2022 年 2 月 22 日**—[连续智能](https://www.sumologic.com/continuous-intelligence-platform/)的先驱 Sumo Logic(纳斯达克代码:Sumo)今天宣布推出新产品，进一步推进其屡获殊荣的 [Sumo Logic Cloud SOAR](https://www.sumologic.com/solutions/cloud-soar/) 的作战室和应用程序中心功能。作战室为安全团队提供事件的详细信息，以加快通常需要几分钟才能完成的手动流程，现在只需几秒钟即可完成。在 App Central 中，包括使用案例、集成和行动手册在内的关键资源汇集在一起，以促进必要的自动化，从而使安全团队能够构建标准操作程序并更快地响应事件。

“企业领导很早就认识到在组织内部的系统之间共享关键数据的必要性。这在安全运营中心(SOC)也很重要，SOAR 解决方案已成为缩小调查差距的必要工具。“安全自动化和响应的使用案例将继续发展。像作战室这样的工具将为网络安全团队提供灵活性，使他们能够专注于重要的任务，而将其余的任务留给我们强大的自动化、开放集成功能和高级分析，以便满怀信心地进行调查和响应。”

阅读博客: [*不知所措:为什么 SOAR Solutions 是游戏规则改变者*](https://www.sumologic.com/blog/overwhelmed-why-soar-solutions-are-a-game-changer)

Sumo Logic Cloud SOAR 作战室和 App Central 提高了效率，甚至实现了更高的自动化，以推动最重要的安全运营活动。

*   **快速决策:**作战室通过一系列卡片，按时间顺序详细查看特定事件中发生的每个相关事件，以及从开始到结束的逐步流程。每张卡都显示事故的最关键信息，一目了然，并提供可操作的见解和对执行结果的可见性。借助新的图表视图，分析人员可以及时了解事件，并根据以前操作的可视化结果做出决策，从而以结构化的方式分析数据。
*   **通过开放集成框架提高自动化程度(OIF):** Sumo

Logic Cloud SOAR 可轻松与现有安全堆栈集成，包括针对 EDR、NDR、IAM、威胁情报、SIEM、漏洞管理等的解决方案。没有对第三方 API 集成的限制，团队可以轻松集成现有的自动化工具和技术。OIF 不仅允许网络团队创建可以插入剧本的动作，还具有增加灵活性的附加功能，如守护程序、触发器和自定义动作。

*   **访问最新行动手册、行动和使用案例库:**App Central 中提供了预构建、可定制的模板、行动和使用案例，以指导安全分析师根据其组织的需求应用解决方案。由相扑逻辑专家策划，Cloud SOAR 配备了行动，以结束对最新威胁的调查。
