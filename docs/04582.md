# Linux 基金会现在提供新的 Kubernetes 基础课程

> 原文：<https://devops.com/new-kubernetes-fundamentals-course-now-available-linux-foundation/>

**旧金山，2017 年 1 月 26 日**——推进[专业开源](https://www.linuxfoundation.org/)大规模协作管理的非营利组织 Linux 基金会今天宣布推出一门新的培训课程，[lfs 258——Kubernetes 基础知识](https://training.linuxfoundation.org/linux-courses/system-administration-training/kubernetes-fundamentals)。

LFS258 是与 Cloud Native Computing Foundation 联合开发的，面向想要开始使用 Kubernetes 的开发人员和管理员。它教学生如何使用 Kubernetes 来管理他们的应用基础设施。涵盖的主题包括 Kubernetes 架构、部署、如何访问集群、提示和技巧、配置图等。这个自定进度的在线课程提供了理解 Kubernetes 并快速上手所需的基础知识，以便开始构建可扩展、容错且易于管理的分布式应用程序。

Kubernetes Fundamentals 将提炼关键原则，如 pod、部署、副本集和服务，并为学生提供足够的信息来开始自己使用 Kubernetes。本课程设计用于广泛的 Linux 发行版，因此无论学生选择何种发行版，他们都能够应用所学的概念。

Kubernetes 正迅速成为在数据中心大规模运行容器化应用的事实上的标准。它建立在谷歌 15 年管理容器化应用的经验之上。随着开源社区的不断发展，Kubernetes 准备改变应用程序的构建和管理方式，并改变系统管理员的角色。来自 Linux 基金会和 Dice 开源专家的 [2016 开源工作报告](http://go.linuxfoundation.org/download-2016-open-source-jobs-report)称，容器是仅次于云技术的第二重要技能。然而，对于那些希望开始使用 Kubernetes 的人来说，缺乏正式的培训选择，这正是 LFS258 旨在解决的问题。

“Kubernetes 在开发测试和试验以及生产环境中正在迅速成熟，在过去的八个月中，它的使用量增加了近两倍，”云原生计算基金会执行董事 Dan Kohn 说。“我们很高兴与 Linux 基金会合作，提供一个培训、认证和推广 Kubernetes 托管服务提供商(KMSP)的计划。该计划将确保企业获得他们所寻求的支持，以更快、更高效地加快速度并推出新应用。”

LFS258 还将帮助那些计划参加 Kubernetes 认证考试的人做准备，该考试将于今年晚些时候推出。我们计划在认证考试开始前对课程进行更新，这是专门为帮助准备考试而设计的。

该课程从去年 11 月开始接受预注册，现在可以立即开始。199 美元的课程费用提供了一年内对课程所有内容和实验的无限制访问。感兴趣的个人可以在这里注册。

**关于 Linux 基础**

Linux 基金会是世界顶级开发者和公司的首选组织，旨在构建加速开放技术开发和商业采用的生态系统。它与全球开源社区一起，通过创造历史上最大的共享技术投资来解决最困难的技术问题。Linux 基金会成立于 2000 年，如今提供工具、培训和活动来扩展任何开源项目，这些共同带来了任何一家公司都无法实现的经济影响。更多信息可在[www.linuxfoundation.org](https://www.linuxfoundation.org/)找到。
