# 纯存储通过扩展 Portworx 产品组合并使 Kubernetes 的采用成为现实，提高了开发人员的工作效率

> 原文:[https://devo PS . com/pure-storage-boosts-developer-productivity-expanding-the-portworx-portfolio-and-making-kubernetes-adoption-a-reality/](https://devops.com/pure-storage-boosts-developer-productivity-by-expanding-the-portworx-portfolio-and-making-kubernetes-adoption-a-reality/)

*Portworx 产品组合的进步提供了比以往更多的数据服务，同时给予开发团队对其工作流更大的控制权*

**加利福尼亚州山景城，****2022 年 5 月 16 日—**提供全球最先进的数据存储技术和服务的 IT 先驱 Pure Storage(纽约证券交易所代码:PSTG)今天宣布对其 Portworx 产品组合进行一系列更新，包括:

*   业内首款 Kubernetes 数据库即服务(DBaaS)平台 **Portworx 数据服务** 全面上市
*   下一代标志性的**Portworx Enterprise**， 完整的多云就绪 Kubernetes 数据管理平台，用于运行生产中的任务关键型应用。

**Portworx Backup-as-a-Service 在 AWS** 上的全面上市，它还支持任何 Kubernetes 服务的备份和恢复，包括亚马逊 EKS、Red Hat OpenShift、谷歌 Kubernetes 引擎、VMware Tanzu 和微软 Azure Kubernetes 服务。
