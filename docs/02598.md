这是一种非常强大的技术，不仅可以排除 struts 和 spring 中的已知漏洞，还可以排除所有形式的表达式语言注入，甚至是还没有人发现的。所以，我认为这非常令人兴奋，让你在处理每周出现的新简历时处于一个非常独特的位置。
