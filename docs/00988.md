# F5 发布 2016 年“应用交付状态”客户调查结果

> 原文:[https://devo PS . com/F5-发布-2016-应用状态-交付-客户-调查-结果/](https://devops.com/f5-releases-2016-state-of-application-delivery-customer-survey-results/)

西雅图—([商业资讯](http://www.businesswire.com/))—[F5 Networks](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ff5.com&esheet=51259010&newsitemid=20160114005175&lan=en-US&anchor=F5+Networks&index=1&md5=c691e0d8923cd023bb6509280234b35e)(纳斯达克: [FFIV](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.google.com%2Ffinance%3Fq%3DFFIV&esheet=51259010&newsitemid=20160114005175&lan=en-US&anchor=FFIV&index=2&md5=5941fc91517fc678dfd6c7e2b15d9e85) )，今天发布了其第二次年度应用交付客户调查结果。今年的报告扩展到包括来自全球 3，000 多家客户的反馈，详细介绍了他们如何成功交付应用、如何保护数据和用户安全，以及混合云、软件定义网络(SDN)和 DevOps 如何改变 IT。此外，F5 高管将主持一系列现场小组会议和网络研讨会，为与会者提供更深入的数据见解。第一场现场小组会议将于 2016 年 1 月 28 日举行，由 F5 首席技术官兼产品开发副总裁 Karl Triebes 主持。报名参加小组讨论和网络研讨会，并在:[http://f5.com/SOAD](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Ff5.com%2FSOAD&esheet=51259010&newsitemid=20160114005175&lan=en-US&anchor=http%3A%2F%2Ff5.com%2FSOAD&index=3&md5=76fe44e569f34c7a0cbbbc2f94b6a2d4)下载完整报告。

**调查亮点**

今年的报告因美洲(AMER)F5 客户的投入而变得更大更广；欧洲、中东和非洲(EMEA)；和亚太(APAC)地区。各种行业都有代表，金融、技术、电信公司和政府机构的客户参与最为广泛。关键要点包括:

*   **应用服务必不可少且无处不在:**超过一半的受访者使用十种或十种以上的应用服务，他们认识到缓慢、无响应且不安全的应用会对收入和运营产生重大负面影响。
*   **混合云是一种新常态:**绝大多数受访者(81%)正在转向混合云环境，以利用它提供的灵活性和潜在的成本节约，尤其是对中小型组织而言。
*   **安全重点是保护用户、数据和应用:**对抵御攻击的能力最有信心的安全专业人员正在保护客户端、请求和响应，这些都是数据容易遭到破坏的关键点。
*   **DevOps 和 SDN 是提高运营效率的关键:**由于 DevOps 和 SDN 支持自动化和流程编排，因此它们都被视为降低运营成本和缩短上市时间的关键因素。

“应用对业务成功至关重要。每个员工都使用这些应用程序，它们有助于提高效率、降低成本和增加利润。“如果没有正确的服务来支持应用程序，它们就不可用、不安全或不够快，无法满足公司的需求。今年的应用交付状况调查结果为客户如何在动态复杂的 IT 环境中管理和实施应用服务提供了宝贵的见解。"
