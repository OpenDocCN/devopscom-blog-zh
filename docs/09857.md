 *## 1.密钥:添加了云的 FIDO2-WebAuthn

本周的第一条新闻是:谷歌宣布了支持密钥的计划。本质上是 WebAuthn 之上的一个可用性层，Passkeys 承诺**取消密码**。

### 分析:当心狗？

DevOps 的人应该密切关注这一点，并在适当的时候增加支持。但是，正如我今年早些时候概述的那样，许多技术负责人自然对锁定和让[“隐私侵犯大公司”](https://securityboulevard.com/2022/03/bad-dog-everyone-hates-this-fido-passwordless-idea/?utm_source=richisoc "Bad Dog—Everyone HATES This FIDO Passwordless Idea") **从用户的私人数据中获利**持怀疑态度。

[亚当·康威](https://www.xda-developers.com/google-passkeys-chrome-android/ "read the full text") : **谷歌将为安卓和谷歌浏览器提供密钥**

<big>**`WebAuthn API can be implemented`**</big>
密码虽然方便，但天生就有缺陷。…这就是为什么谷歌宣布将在 Android 和谷歌 Chrome 上推出的 passkeys 非常重要。…今年早些时候，苹果、谷歌和微软宣布，他们将在所有主要平台上采用 FIDO 标准作为万能密码。…这将使无密码登录变得更加普遍。在 Android 上，它会把你的智能手机变成一个可以用来登录网站的密钥。这些密钥然后通过谷歌密码管理器同步，以便在您的设备上轻松访问。…虽然它将开始只支持网站，但其想法是开发者将能够在他们的应用程序中实现它。[和]密钥是平台无关的，这意味着您甚至可以使用运行 Safari 的 Mac 登录支持密钥的服务。
…
如果你想尝试一下，开发者可以注册 Google Play 服务测试版，使用 Chrome Canary。WebAuthn API 可以在网站上实现，以便与 Google Chrome、Android 和其他平台中的密钥兼容。预计今年晚些时候还将发布针对原生 Android 应用的 API。

摇滚我，[罗恩·阿马迪奥](https://arstechnica.com/gadgets/2022/10/google-rolls-out-beta-passkey-support-for-chrome-and-android/ "read the full text") : **死亡密码**

大技术想取消密码。…万能密码得到了谷歌、苹果、微软和 FIDO 联盟的支持，所以预计很快就会随处可见。
…
密码管理器革命完全是一个黑客…建立在原来的文本框之上。我们真的不再需要文本框了。… Passkeys 只是直接与网站交换 WebAuthn 加密密钥。…您的手机将…通过某种生物特征进行认证，如指纹解锁。手机将通过蓝牙与客户端通信，浏览器解锁你的密钥，然后将其发送到网站。

但是许多人闻到了锁定的味道。例如， [wnevets](https://news.ycombinator.com/item?id=33179180 "read the full text") :

万能钥匙听起来像是谷歌和苹果这样的公司把你锁在他们有围墙的花园里的另一种方式。让每个围墙花园为每个域随机生成一个密钥，而不是使用实际的域名作为密钥的一部分，这是一种将普通人锁定在他们各自的生态系统中的好方法。这些围墙花园造成的人为问题是，每个域名都有自己随机生成的私钥。保持所有这些随机生成的密钥在多个设备间同步的唯一可行的方法是使用“云”相反，如果每个站点的密钥是使用私钥和域名生成的，用户只需将一个私钥传输到另一个设备，就可以免费同步，而不需要“云”
……
如果使用实际域名来生成密钥，也将完全消除网络钓鱼攻击的能力。Paypal.com 和 PaypaI.com 将生成两种完全不同的密钥，并消除对第三方云服务的需求。

其他批评指向缺乏信任。加入[次郎](https://tech.slashdot.org/comments.pl?sid=22208545&cid=62960871 "read the full text"):

我不信任谷歌。…有很多臭名昭著的案例，谷歌以虚假的理由关闭了人们的账户，让他们无法上诉。如果你有万能密码支持，然后谷歌把你锁在你的账户之外，会发生什么？。

被放逐到阿思星球的是第三个医生[:*【你被解雇了。]*](https://arstechnica.com/gadgets/2022/10/google-rolls-out-beta-passkey-support-for-chrome-and-android/?comments=1&post=41297916#comment-41297916 "read the full text")

作为一名安全人员，我不认为这是正确的方向。…我不认为密码已经过时，也不认为密码在概念上是错误的。…答案是多因素身份认证…结合基于安全认证器的 MFA。最终，我们想不出什么办法来阻止所有的身份冒充攻击。做不到。…但是我们可以极大地降低这种可能性,*不需要*使用蓝牙……而*不需要*重写现有的每个网站。…我不认为这需要放弃密码。

但是这和 WebAuthn 有什么不同呢？不是这样的，T2 说:

它*就是* webauthn。它现在可以同步，并且不需要额外的硬件来使用。这才是重点。“正常人”更有可能使用它，因为几乎没有人愿意仅仅为了登录网站而购买和携带一把钥匙。

## 2.四天工作制试点:调查显示…

你想每周多休息一天吗？如果您的工作量不变，会怎么样？这是“全球 4 天工作周”组织的承诺。

### 分析:到目前为止，一切顺利

全球各地的试点计划正在返回类似的结果:**它只是工作**。

史蒂文·j·沃恩-尼科尔斯 : **四天工作制**

<big>**`The writing is on the wall`**</big>
去年 6 月，在英国的一个试点项目中，数千名员工开始每周工作 4 天，工资不变。…四个月后，结果出来了。请敲鼓吧，一周工作四天的人和传统作息时间的人工作得一样好。… 95%的公司…表示他们的生产力水平保持不变或有所提高。
…
工人在正常工作时间的 80%内获得 100%的工资，同时期望获得 100%的生产率。…人们更有精神、更有活力地投入工作。…员工流动率…倦怠、病假和…悄悄离职都显著下降。……写在墙上的东西是给那些能看懂的人看的。但是，当亨利·福特在他的整个企业中推行五天工作制时，批评家们认为，如果他的工人们不是一周工作 48 小时，他们的生产率就不会那么高。那么他们错了。他们现在错了。

埃里克·约翰逊认为，这不是一个新想法，但它的时代已经到来:

在疫情之后工作是不同的。…四天工作周即将到来…尽管公司多年来一直反对四天工作周的想法。过去，老板们经常拒绝这种想法，但分析师认为他们最好习惯这种做法。…科技行业永远缺乏人才，在一个候选人主导的市场中，他们听取员工的需求是明智的。

此外，这项业务还有其他优势。 [@Wests040](https://twitter.com/wests040/status/1580166677740015617 "read the full text") 举了一个例子:

专业人士现在甚至更强…对于能源紧缩的欧洲！我们需要停止扶持恐怖主义政权。
