# 疫情理工大学转型的下一阶段

> 原文：<https://devops.com/the-next-phase-of-pandemic-tech-transformation/>

去年春天，由于疫情和向远程工作的转变，企业几乎在一夜之间经历了快速的数字化转型，我们被标题、推文和媒体帖子淹没了。这在当时是一个有趣的感觉良好的故事，但它并没有描绘出全貌。公司被扔进了深水区；许多公司通过实施他们认为只需要很短时间的快速修复、表面水平的变化来勉强维持运营。事实证明，被吹捧的“转变”只是冰山一角。

一年多过去了，员工队伍仍在不断变化。现实是，某种形式的分布式混合工作很可能会继续存在。员工将继续期待远程选项，客户将希望以无缝、数字化的方式与企业互动，无论是否在疫情。

因此，企业必须摆脱去年的临时权宜之计，让这些彻底的变革永久化。是时候推进真正的、可持续的变革了:技术上的、经验上的，最重要的是文化上的。每家公司都需要成为一家现代科技公司，能够为客户、合作伙伴和员工提供一流的数字解决方案，否则就有被淘汰的风险。

## 交易型变革与转型型变革

好消息是，几乎每个公司都理解利用技术的价值。最近的 [2021 年首席信息官状况调查](https://www.cio.com/article/3619279/top-cio-priorities-in-2021-include-process-transformation-and-automation.html)发现，82%的首席信息官表示他们在过去一年实施了新技术、新 IT 战略和/或新方法。问题是公司如何进行技术部署。

人们一直在谈论数字化转型，但它被用得太多了，已经成为一个无益的误称。通常情况下，公司正在实施的变革并不是真正的转型，而是交易性的或只是轻微的增量。

当公司有交易心态时，关注的是即时、简单和可预测的变化。目的——或者更重要的是，结果——仅仅是继续一切照旧。这让人想起去年许多公司转向技术只是为了保持正常运转。另一方面，转型思维旨在打破这种一切照旧的模式，并通过工作方式、技术和文化的使用，从根本上改变公司的运营模式。这是有目的的，旨在整体解决业务问题并刺激长期增长。

为了在当今的环境中取得成功，并获得与客户的竞争优势，公司必须努力实现转型。

## 如何走向真正的转变

这里有三个公司如何从交易型思维转变为转型型思维的例子。

### 云迁移与现代化

通常，当公司谈论他们对云的使用时，他们吹嘘他们已经“转移到了云”但是很多时候，它被视为一次性事件；要到达的目的地和要检查的方框。在疫情开始的时候，我们看到如此多的组织迅速动员起来支持他们的数字战略，并确保他们考虑在虚拟环境中开展业务的局限性，但他们的云战略并没有发展到重新思考这些范式。他们只是加快了他们的迁移策略，从物理数据中心迁移到云中。

利用云的机会远不止这些。它可以实现更高的效率、灵活性和增长，并巩固新的数字体验和收入机会，这正是大多数公司所设想的战略。例如，医疗保健领导者 InnoCare 正在与过时的诊所管理软件作斗争。他们不再只是将旧的部分转移到云提供商，而是采用了更广泛的数字化战略来实现交付能力的现代化，并重新想象患者和临床医生的体验，在此过程中设定新的行业标准。这就是我们所说的现代化，也是导致变革的原因。

### 软件工程与产品工程

随着软件的快速增长，软件工程师的需求量很大也就不足为奇了。美国劳工统计局预计，到 2029 年，软件工程职位空缺将增长 22%。虽然软件工程师将在帮助构建创新解决方案方面发挥不可替代的作用，但专注于转型的公司知道如何吸引更多人。通过将软件工程师与带来不同水平的艺术、设计和受众理解的其他跨职能专家结合起来，最终结果是一个更具吸引力、以客户为中心的产品，该产品在其整个有用的生命周期内不断发展壮大。

健身巨头鹦鹉螺正是这么做的。他们召集了一支由工程师、数据科学家和机器学习专家组成的团队，来帮助弥合他们的设备制造能力与他们的数字体验和软件开发目标之间的差距。总的来说，他们生产了一个集成硬件、软件和数据的总平台，以创建一个动态的锻炼方案，给用户留下深刻印象，并抵御竞争对手。

### 超越概念验证，迈向长期投资回报

概念验证(POC)是科技世界的一项仪式。这是一个公司或团队如何证明他们的想法是可行的，可以在现实世界中应用。但是仅有概念证明是不够的；他们以交易为中心，不专注于实现持久的业务成果或产生更高的投资回报率。相反，他们主要关注技术可行性。向转型升级的关键是关注过程，而不仅仅是最终产品。在真实环境中测试功能时，请确保您主动测量的不仅是它们的工作情况，还有它们对业务需求的贡献、满足客户期望和战略成果。如果他们没有，回到绘图板，并在头脑中优化这些更高层次的目标。

### 文化是关键

正如你所看到的，交易型和转型型思维模式之间的区别更多的是关于文化而不是技术。创新文化和现代工作方式带来了变革，从根本上改变了技术的概念化和创造方式。这不仅仅是为了提高效率或削减成本，而是为了让客户、员工和日常用户的生活更美好。专注于转型的公司相信技术进步的前景，但他们相信，随着时间的推移，当技能、纪律和组织买入的正确组合得到实施时，技术进步也可以使运营更顺畅，客户关系更有意义，并增加收入。正是这些公司将在这个新时代茁壮成长。
