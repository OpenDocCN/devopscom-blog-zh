# SmartBear 通过收购 Hiptest 加快了连续测试

> 原文:[https://devo PS . com/smart bear-accelerates-continuous-testing-with-acquisition-of-hip test/](https://devops.com/smartbear-accelerates-continuous-testing-with-acquisition-of-hiptest/)

Hiptest Empowers Agile and DevOps Teams to Collaborate on an Idea, Test Code Continuously, and Generate Living Documentation from Real-time Insights**SOMERVILLE, Mass. and BESANCON, France — May 09, 2018 —**[SmartBear](https://smartbear.com/), the leader in software quality tools for teams, has acquired continuous testing platform [Hiptest](https://hiptest.com/). Hiptest empowers Agile and DevOps teams to deliver high quality software, faster, by collaborating on an idea, testing code continuously, and generating [living documentation](https://hiptest.com/docs/living-documentation/). Hiptest was built by an Agile team, for Agile teams, to address the needs of modern software development – and is currently used by over 25,000 users in 140 counties. Hiptest directly complements existing SmartBear tools, and together, will help modern software development teams accelerate continuous testing.

Hiptest 的联合创始人兼首席执行官 Laurent Py 说:“我们创立 Hiptest 是为了让敏捷和开发团队能够更快地将他们实际想象的产品推向市场。“我们强烈地感觉到 SmartBear 是我们前进的理想合作伙伴，因为我们共同的愿景是让现代软件开发团队更快地从想法进入生产。”

随着组织努力拥抱数字化转型并更快进入市场，敏捷和 DevOps 实践正日益成为主流。然而，采用敏捷和 DevOps 的团队与传统的软件开发团队工作方式不同，因此需要工具来支持他们的开发工作流并实现持续交付。Hiptest 获得了快速的流行，因为它通过对[行为驱动开发](https://blog.hiptest.net/2016/04/28/getting-started-with-bdd-part-1/) (BDD)的本地支持促进了业务和技术团队之间的协作，无缝集成到开发工作流中，并支持连续测试。

“Hiptest 已经成为我们开发工具包中不可或缺的一部分，” [Flowbird](https://www.flowbird.group/) 的软件项目经理[raphal Citeau](https://www.linkedin.com/in/rciteau/)说。“吉拉插件和对 BDD 的本机支持使我们能够轻松地在工作流程中利用 Hiptest，并且实时文档的使用使我们能够实时了解我们的软件在生产中的行为。”

智能熊产品、测试和开发副总裁[瑞恩·洛伊德](https://smartbear.com/company/team/featured-people/ryan-lloyd/)说:“智能熊产品组合中 Hiptest 的加入补充了我们所有的测试自动化工具，这些工具如今受到开发团队的喜爱，包括 [TestLeft](https://smartbear.com/product/testleft/overview/) 、 [CrossBrowserTesting](https://crossbrowsertesting.com/) 、 [SoapUI Pro](https://smartbear.com/product/ready-api/soapui/overview/) 和 [Swagger Inspector](https://swagger.io/swagger-inspector/) 。Hiptest 是一项令人兴奋的快速增长的业务，与 SmartBear 的战略相一致，即提供易于使用和易于消费的工具来支持软件开发生命周期(SDLC)

所有 Hiptest 员工，包括联合创始人劳伦特·皮、[塞瓦琳·达洛特](https://www.linkedin.com/in/s%C3%A9verine-darlot-304ab5/)、[琼·拉塞内](https://www.linkedin.com/in/joanracenet/)、[文森特·普雷特](https://www.linkedin.com/in/vincentpretre/)和[克里斯托夫·布利亚尔](https://www.linkedin.com/in/christophebliard/)，都加入了 SmartBear，继续发展 Hiptest 产品和业务。SmartBear 将继续在法国贝桑松开发 Hiptest，并打算在那里扩大业务。Hiptest SaaS 平台提供免费的 [30 天试用](https://app.hiptest.com/users/sign_up)，并提供灵活的[定价选项](https://hiptest.com/pricing/)，从小型团队的免费计划到企业产品。

[451 Research](https://451research.com/) 的首席分析师 [Carl Lehmann](https://451research.com/analyst-team/analyst/Carl+Lehmann) 说:“随着敏捷开发、DevOps 项目和 CI/CD 自动化在企业中的加速，质量控制挑战也在增加。“这种趋势需要一个全面的自动化测试架构，就像 SmartBear 最近收购 Hiptest 时正在打造的那样。”
