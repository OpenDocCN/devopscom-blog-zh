# 茄子被评为持续功能测试自动化提供商的领导者

> 原文:[https://devo PS . com/guillenge-named-as-a-leader-in-continuous-functional-test-automation-providers/](https://devops.com/eggplant-named-as-a-leader-among-continuous-functional-test-automation-providers/)

*独立研究公司认可茄子的人工智能测试自动化平台*

[茄子，](https://www.eggplantsoftware.com/)智能自动化专家，今天宣布其被命名为“Forrester Wave:持续功能测试自动化套件，Q2 2020”的领导者茄子在 16 个标准中获得了最高分，包括云平台支持、全渠道测试、设计自动化和优化、自动化执行/持续测试、测试角色、报告、分析和质量洞察、TDD 和 BDD 以及创新路线图。

《Forrester Wave》一书的作者迭戈·洛·朱迪切在报告中指出:“茄子有一个强大的测试愿景，有清晰的执行和创新路线图。如果成功实施，它将在竞争中保持领先，证明自己是企业测试客户的有力选择，并且仍然非常适合敏捷加开发运维转型。”

“我们正在改变智能测试自动化的世界，”茄子公司首席执行官约翰·贝茨博士说。“我们的使命是消除世界上的不良软件，我们相信 Forrester 的这一认可强调了我们的人工智能驱动的自动化平台所提供的竞争优势。我们的客户正在受益，并可以通过我们对人工智能、深度学习和分析的突破性使用来优化客户体验和业务成果。我们将继续拓展智能自动化的边界，以支持和加快数字体验的速度和质量。”
