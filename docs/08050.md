# Ubuntu 支持 Oracle 云基础设施 Ampere A1 计算

> 原文：<https://devops.com/ubuntu-supports-oracle-cloud-infrastructure-ampere-a1-compute/>

**英国伦敦，2021 年 5 月 25 日—**Canonical 与甲骨文公司合作，宣布推出针对甲骨文云基础设施(OCI) Ampere A1 Compute 的优化 Ubuntu 映像。Oracle 云客户将受益于在安全、可扩展且极具成本效益的基础设施上运行最流行的云操作系统 Ubuntu。

“Oracle 云基础设施 Ampere A1 Compute 上的 Ubuntu 为所有类型的工作负载提供了高性能且经济高效的解决方案。Ubuntu 为开发人员提供了高度优化的云操作系统和内核，具有出色的启动速度、强大的安全性和出色的稳定性。

“Ampere 和 Canonical 合作将云原生解决方案推向市场。我们一起为 Ampere Altra 优化了从 Ubuntu OS 到 OpenStack 到 K8s 到 Anbox Cloud 的一切。Ampere Computing 首席产品官 Jeff Wittich 表示:“我们很高兴看到所有这些技术都可以在 OCI Ampere A1 平台上投入市场，该平台今天可以在 OCI 免费层使用。

同样安全的 Ubuntu，现在采用了 Oracle 云优化的内核和配置。结合 Ubuntu 的优势和 Oracle 新产品的性能，企业可以以成本优化的方式运行传统工作负载。基于 ARM 的平台也特别适合新的用例，如 Anbox Cloud，它允许组织在云中大规模安全地运行 Android。出于这个原因，监管市场的合规通信解决方案供应商乌莫尼将把 Anbox 云与新的 Ampere A1 计算实例集成在一起。

“我们需要一个在云中大规模运行 Android 的解决方案 Canonical 的 Anbox Cloud 让我们做到了这一点。它为我们构建以合规为重点的产品奠定了坚实的基础，借助新的 OCI 安培 A1 计算实例，我们可以显著降低每个 Android 实例的成本，因为它可以提供每个实例运行大量 Android 设备所需的密度，”乌莫尼首席执行官迪安·埃尔伍德说

[Anbox Cloud](https://anbox-cloud.io/) 是 Canonical 的解决方案，可在任何云上大规模运行 Android，受益于 OCI 安培 A1 Compute 提供的性能和密度。为自动化用例运行数百个 Android 容器变得简单，同时带来巨大的经济效益。

OCI 安培 A1 上的 Ubuntu 现在可以在 OCI 控制台上作为平台镜像使用。
