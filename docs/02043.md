# DevOps 企业峰会拉斯维加斯 2019 年会议议程发布，发言人来自 ING 银行、毕马威、微软、Ticketmaster、T-Mobile 和美国银行

事件编程强调在大型组织中引领广泛变革所需的不断发展的业务、技术和架构实践。

俄勒冈州波特兰市。，2019 年 8 月 27 日— [推动 DevOps 发展的行业领导者 IT Revolution](https://itrevolution.com/?utm_medium=affiliate&utm_source=pr&utm_term=us19-agenda-pr-1&utm_content=onsale&utm_campaign=does19us) ，在 2019 年拉斯维加斯 DevOps 企业峰会(【https://cvent.me/D49MZ?RefId=c11】T2)上发布了完整的会议议程和下一轮业务和技术领导者。DevOps 企业峰会于 10 月 28 日至 30 日在拉斯维加斯的 Cosmopolitan 举办，是商业和技术领导者学习如何更快地开发和部署软件并在市场中获胜的首要行业会议。

DevOps 企业峰会在美国已进入第六个年头，预计将有 2000 多名与会者、100 多名演讲人和 50 多家赞助商在拉斯维加斯举办。会议议程包括三整天的主题演讲、行业小组讨论、分组会议、DevOps 研讨会、演讲人问答会议、图书签售会和展览馆。

IBM 杰出工程师 Rosalind Radcliffe 表示:“请务必参加 DevOps 企业峰会，从真实的用户体验中学习，并与该领域的佼佼者交流。“这是一个你不能错过的 DevOps 活动。”

与会者将体验正在进行的业务和数字转型的纪录片，并与领先的 DevOps 从业者和技术行业的主题专家交流见解。
