# DevOps 聊天:你的数据到底值多少钱？使用 Splunk

> 原文：<https://devops.com/devops-chats-what-is-your-data-really-worth-with-splunk/>

获得真正的洞察力可以让公司在竞争中脱颖而出，但这说起来容易做起来难。Splunk 最近发布了新报告[您的数据真正有价值吗？产生了一些非常引人注目的结果。](https://www.splunk.com/en_us/form/whats-your-data-really-worth.html)

领先的数据创新者利用数据将毛利提高了 12.5%。97%的顶级客户达到或超过了他们的客户保持目标。成熟的组织从新的、创新的产品和服务中获取 20%以上收入的可能性几乎是其他组织的 10 倍。一组被归类为数据创新者的公司取得了令人印象深刻且可衡量的成果。

Splunk 首席技术倡导者 Andi Mann 与 Mitch Ashley 一起参加了 DevOps 聊天，分享了报告中的一些重要见解，并讨论了公司如何利用其数据来实现更高的收入增长、改善客户体验和节省成本。
