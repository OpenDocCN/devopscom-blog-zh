# Sauce Labs 和 Sumo Logic 联手帮助企业提高工程效率

> 原文:[https://devo PS . com/sauce-labs-and-sumo-logic-team-up-to-help-enterprises-drive-increased-engineering-efficiency/](https://devops.com/sauce-labs-and-sumo-logic-team-up-to-help-enterprises-drive-increased-engineering-efficiency/)

### 新的联合计划结合了持续测试和端到端管道可见性和分析，帮助客户更快、更安全地开发更高质量、更可靠的应用

旧金山，2021 年 2 月 9 日(环球新闻通讯社)— [Sauce Labs Inc.](https://www.globenewswire.com/Tracker?data=6FEM2hzk8wh6lICWfRQPdwHptFdHxTSCBubpWOVxUVkwHlG9uFCGpIBlOhcGBeLqpSHxVo-w8t_5b7J4ySGOnA==) ，提供数字信心的[连续测试](https://www.globenewswire.com/Tracker?data=ckV4LG4KTSSTMHpEc8cD8MSGNO_CRyUt19EkHa2gGPSlfMDQGaEaXBOkqL5NbK3iECRxnr4Ox6FXe6eYvX5PU-NSaPZhbx9D3lxzrbhQGIM5-n2l3Whl0aijNvGPTpQ4)解决方案的领先供应商，和[连续智能](https://www.globenewswire.com/Tracker?data=ckV4LG4KTSSTMHpEc8cD8HRe3Mclu0VbeNALczcBny1tPYCmRHvNea7d7aHlfsgx01LWoTkUMZv_JhhgVZ8PgO5fufeDi0eaB_OSnvbM_Knq-Q-bfaUT6YQ7CtGGVW0Jfj_S2gBmrx0jgjs-tgUtUA==)的先驱 [Sumo Logic](https://www.globenewswire.com/Tracker?data=vA_esZVw1kDuOPfO1hF7NDiO32gopULmY7eo2j9Hafjnpk0AxwIDORDH-DWl7OgNyWirD8vIss8u45_0sEQAmg==) (纳斯达克代码:Sumo)，今天宣布成立一个新的联合计划，旨在通过在整个开发生命周期中提供对应用程序健康和安全的更大可见性，帮助企业组织提高工程效率。这种结合将使客户能够利用来自 Sauce Labs 连续测试云的测试数据和 Sumo Logic 的软件开发优化解决方案，并将其与来自软件开发管道其他部分的数据相关联，以自动获得可操作的见解，帮助提高应用的质量和可靠性。

随着组织越来越重视质量，测试在持续交付和改进过程中发挥着关键作用，从测试数据中收集和分析信号的能力对于实现持续智能至关重要。这一新的联合倡议建立在两家公司长期的合作伙伴关系、协作历史以及各自平台之间业已存在的强大集成基础之上，使客户能够在 Sumo Logic 的云原生连续智能平台内查看来自 Sauce 实验室的丰富测试数据，以便就其工程和开发流程的功效做出更好、更明智的决策。

Sauce Labs 首席产品官 Matt Wyman 表示:“随着测试日益成为一项持续的举措，在部署之前发现和修复缺陷的能力在建立数字信心方面发挥着至关重要的作用。“Sauce Labs 和 Sumo Logic 的结合使客户能够在整个产品开发生命周期(包括生产前和生产后)中全面了解其应用程序的运行状况。我们很高兴能与 Sumo Logic 团队合作，帮助组织从他们的工程投资中获得更多回报。”

“随着公司从传统商业模式转向数字商业模式，它们成为软件公司，它们的成功取决于其数字服务的速度和可靠性。Sumo Logic 战略和解决方案创始副总裁 Bruno Kurtic 表示:“现代软件开发生命周期(SDLC)和相关工具生成的数据可用于优化和改进 SDLC，进而提高公司的敏捷性和创新速度，从而提高其竞争力。“我们与 Sauce Labs 的合作伙伴关系以及我们将测试结果与代码更改、环境或部署更改以及生产事件相关联的双向集成，将帮助软件开发团队提高发布速度、应用可靠性和客户体验，同时提供安全智能。”

这一新举措是在组织加速数字化转型以创造竞争优势的背景下推出的。Sauce Labs 连续测试云与 Sumo Logic 的可观察性解决方案套件的独特组合，包括[软件开发优化解决方案](https://www.globenewswire.com/Tracker?data=Aj5uGnZvOxFqQ5Dm7edIjq8VBmTpXT3mVjPeEpr-WhW_0_sFSj7DeizHq2mZvKP77TbbB4BuePXpdSEUt5EhJBVqEnXVk-u_3IIvUnnzhW28ife3YELWr-tzzr6wSw0ug9GevvC0hqd3VJf7dNSuuCQJi6KnKVlGaTpWCMGjjxAe0QK_ajwycTX_wSI-1FXS) 、，确保客户全面了解其整个 CI/CD 管道，以安全地开发和发布新应用程序或更新现有应用程序，比以往任何时候都更快、更高质量和更可靠。
