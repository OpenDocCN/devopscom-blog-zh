# JumpCloud 通过 Apple MDM 支持扩展云目录平台

> 原文：<https://devops.com/jumpcloud-extends-cloud-directory-platform-with-apple-mdm-support/>

*苹果 MDM 的加入深化了跨平台系统管理能力*

**路易斯维尔，科罗拉多州——2020 年 5 月 21 日——**全球首家目录即服务公司 JumpCloud 今天宣布全面支持苹果的移动设备管理(MDM)服务。最新版本扩展了 JumpCloud 的功能，通过 MDM 协议管理苹果 macOS 设备，并支持通过苹果设备注册计划(DEP)进行部署。Apple MDM 支持的增加进一步扩展了该平台同时管理 macOS、Windows 和 Linux 台式机、笔记本电脑和服务器基础设施的能力。凭借新的 MDM 功能，JumpCloud 继续向其全面的身份访问控制和设备管理平台添加强大的安全功能，IT 管理员无需部署不同的技术和昂贵的单点解决方案来全面管理远程工作人员。

*发这个:用苹果移动设备管理简化#IAM 和设备管理；现收录于@JumpCloud 云目录平台*[*http://ow.ly/vh3L50zGCSd*](http://ow.ly/vh3L50zGCSd)*#远程工作# MDM #安全*

Nimbus Therapeutics 的 IT 经理 Marie Selsky 表示:“作为一家必须同时管理 Mac 和 Windows 机器的公司，JumpCloud 帮助我们大幅减少了供应商的占用空间。“我们可以从一个平台完成所有的设备管理和访问控制。”

JumpCloud 引入了苹果的 MDM 集成，提供了新的功能来支持苹果 macOS 资产的部署和管理。该版本扩展了其屡获殊荣的产品的管理功能，支持对 macOS、Windows 和 Linux 台式机、笔记本电脑和服务器的快速控制。目前，全球有超过 100，000 家组织注册了 JumpCloud 平台。

JumpCloud 的远程设备管理产品通过其基于云的管理控制台为管理员提供跨操作系统的以下功能:

*   系统访问和权限控制
*   设备多重身份认证(MFA)和密码管理
*   设备安全策略配置
*   设备擦除、重启、锁定和关机命令
*   带安全恢复密钥托管的全磁盘加密
*   针对主机和应用程序资产清点和威胁检测的设备级洞察

JumpCloud 首席技术官 Greg Keller 表示:“JumpCloud 针对 Mac、Windows 和 Linux 的基于云的设备管理功能已被全球各地的组织成功用于远程管理、访问控制和安全性。“新冠肺炎放大了对 IT 和系统管理员的这种需求，立即迫使他们制定策略，如何在其实体领域之外安全地管理设备和所有用户访问控制。我们很自豪能够成为苹果认证的 MDM 供应商生态系统的一部分，并为 macOS 提供更深入的部署和管理功能，以及我们的 Windows 和 Linux 设备支持。”

企业战略集团(ESG)网络安全副总裁兼集团总监 Doug Cahill 表示:“虽然新冠肺炎加快了向远程工作的转变，但 IT 团队长期以来一直在评估如何以最佳方式通过任何设备在任何时间、任何地点访问任何应用程序和任何数据来开展业务。“随着分布式团队成为常态，管理员正在寻求从 JumpCloud 等单一平台整合设备管理和访问控制的解决方案，以便跨多个操作系统和设备简单、安全地管理用户。”

借助新冠肺炎，IT 管理员现在的任务是管理完全分散的员工队伍中的用户和设备，研究表明，这一行业转变可能是永久性的。来自 [Global Workplace Analytics](https://globalworkplaceanalytics.com/brags/news-releases) 的最近一项调查强调了这种转变的规模，发现近 89%的受访者报告说自从新冠肺炎开始后，他们在家工作；一旦疫情飓风过去，近 80%的劳动力希望继续在家工作。将基于云的目录服务和设备管理相结合，为 IT 管理员简化了这种转变，为用户、系统和设备管理提供了统一的解决方案。

“MDM 是 JumpCloud 的锦上添花，”Wisetail 的 IT 系统管理员 Evan Sheeks 说。“我更加自信地认为，只需使用一个工具点击几下鼠标，我就可以访问 100%的库存。”
