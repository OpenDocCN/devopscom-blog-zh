# DevOps World 2021:成功扩展的 3 场会议

> 原文:[https://devo PS . com/devo PS-world-2021-3-成功扩展会话/](https://devops.com/devops-world-2021-3-sessions-for-successful-scaling/)

您的企业已经从一个只有五个人的初创企业成长为一个拥有 100 人的成熟组织…祝贺您！但是，现在怎么办？在你刚开始成长时，适用于小公司的实践可能已经奏效，但现在是时候扩展业务以满足你的新需求了。就像一个孩子穿不下他们的衣服一样，您的组织需要一个新的“衣柜”，而[可扩展性](https://devops.com/?s=scalability)是长期成功的关键。

今年的虚拟会议(定于 9 月 28 日至 30 日举行)将提供一系列分组会议、小组讨论和主题演讲，重点讨论企业内部的可扩展性。如果你准备在你的企业内扩展，这里有今年会议的几个会议来帮助你开始。

### 从创业到企业—可持续扩展

随着发展，许多组织努力扩展 DevOps 文化。现在是时候告别创业阶段，向像企业一样思考问好了。

Campspot 软件交付总监 Alston Rousseau 和 Bonusly 人才招聘总监 Kailene Cloud 将向与会者介绍可持续扩展运营、发展和文化所需的投资。此外，他们还将分享在 DevOps 的资金投入、时间和多样性方面发现更多价值的技巧，以及这种方法如何应用于具有和不具有高法规遵从性要求的组织。

### 扩展汽车配置:从静态托管的前端到普遍提供的服务

在过去的一年里，科技公司 Code Orange 从传统的 API/SPA 分离过渡到技术上更具挑战性的服务器渲染 SPA，并实现了完全缓存移交。

在由 Code Orange 开发人员 Niels Hoiting 主持的分组会议中，与会者将了解更多关于 Code Orange 从 API/SPA 分离到服务器渲染 SPA 的过渡，同时分析过渡中可能出现的缺陷以及如何优化计算支出以最佳处理特定工作负载。

### 利用 CloudBees 在多层面虚拟环境中扩展卫星地面系统仿真

航空航天和国防技术制造商雷神技术公司已经成功实施了 CI/CD 管道，用于全球卫星地面系统的软件和硬件的测试和接近运行的集成。

在雷神公司 GPS 产品工厂副经理 Casey Jackman 主持的分组会议中，与会者将了解雷神公司如何在虚拟专用云中使用模拟卫星控制站进行 CI、测试和检验。此外，Casey 还将分享该团队如何应对 AWS 和内部数据中心的多个不同目标环境之间的初始管道生成挑战。
