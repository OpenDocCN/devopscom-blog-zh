# CloudHealth 与 Datadog 合作，为企业和快速发展的技术公司将云性能指标与云成本联系起来

> 原文：<https://devops.com/cloudhealth-partners-datadog-connecting-cloud-performance-metrics-cloud-costs-enterprise-fast-growing-technology-companies/>

***新的集成使客户能够合理调整复杂的云环境***

波士顿—([商业资讯](http://www.businesswire.com/))—[云服务管理的领导者 CloudHealth Technologies](http://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.cloudhealthtech.com%2F&esheet=51363523&newsitemid=20160616005726&lan=en-US&anchor=CloudHealth+Technologies&index=1&md5=ccc7119ac29935464809a3c322b4c00c) 今天宣布与领先的动态云应用监控服务 [Datadog](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.datadoghq.com%2F&esheet=51363523&newsitemid=20160616005726&lan=en-US&anchor=Datadog&index=2&md5=91b988f46e6c8838e299123224070474) 建立集成合作伙伴关系。CloudHealth 和 Datadog 平台的集成为客户提供了对其云环境各个方面的端到端可见性，为他们提供建议和分析，以便在云中做出分析驱动的合理规模决策。

虽然云资源的分散管理和动态配置极大地提高了应用程序开发的灵活性，但它也为组织带来了与成本分配、基础架构规模调整和治理相关的新挑战。CloudHealth-Datadog 合作伙伴关系解决了企业在采用敏捷流程和基于云的基础设施时面临的主要挑战:无法有效管理大规模云资源。

Datadog 提供了一个 SaaS 平台，可监控整个基础架构堆栈中的数据，并在简洁易懂的仪表盘中呈现这些数据，从而提供切实可行的见解。CloudHealth 平台简化了云管理，使企业能够整合、评估、分析、优化和自动化其云。

通过将云指标与成本联系起来，集成平台允许客户以更具战略性和整体性的方式管理资产。CloudHealth Technologies 和 Datadog 将精细的性能数据、端到端可见性和强大的分析功能结合在一起，帮助客户不断做出有关云资源调配和合理调整的最佳决策。CloudHealth-Datadog 合作伙伴关系为客户提供了以下优势:

*   **合理调整分析**:客户可以使用来自这两种解决方案的数据，获得关于在何处调整过度配置或配置不足的资源的建议，从而节省他们的时间和金钱。
*   **关键资产的可视化**:使用 CloudHealth，客户可以专注于他们关心的云数据——按部门、业务线或任何相关角度查看。使用 Datadog 标签，他们可以在一个地方集中查看所有资产以及其他工具和服务。
*   **每项资产的高级性能指标**:客户可以在各种工具之间切换，以真正了解他们的环境。他们可以跳转到 CloudHealth 来跟踪趋势或浏览性能数据，或者跳转到 Datadog 控制台来更详细地探索各个资产。

“为了优化您的云资源，您需要有一些工具来帮助您可视化您的数据，理解它并采取行动，”云健康技术公司的首席技术官兼创始人乔·金塞拉说。“将 Datadog 的同类最佳云监控资产与 CloudHealth 的云服务管理功能相结合，使客户能够真正解决云复杂性问题。”

Datadog 首席产品官 Amit Agarwal 表示:“对于我们的许多企业客户来说，云成本分配和优化是一个重大挑战，他们在各个业务部门中部署了数万个云实例。“我们期待与 CloudHealth 合作，为使用这两种产品的客户提供一种解决方案，帮助客户快速分配并最大限度地降低成本，同时确保他们的关键应用程序始终得到充分供应。”

许多共同客户已经在一起使用 CloudHealth 和 Datadog，包括 Elastera，一家为电子商务网站提供托管云托管环境的供应商。该公司转向云技术以创新和更快上市，并采用了 CloudHealth 和 Datadog 来为其 AWS 基础架构获得准确的合理规模建议。

“CloudHealth 获取从 Datadog 收集的信息，并将指标与资金联系起来。它向我们展示了一个实例是否未得到充分利用，以及通过改变我们可以节省多少，”Elastera 的联合创始人兼首席架构师 Zachary Stevens 说。“使用这套强大的数据分析工具，我们能够降级过度调配的资产，从而节省我们的时间和金钱。”
