# Graviti 推出改变游戏规则的数据平台，摆脱隐形模式

> 原文：<https://devops.com/graviti-launches-game-changing-data-platform-out-of-stealth-mode/>

*Graviti 数据平台将* *电力企业大规模管理非结构化数据*

纽约州纽约市，2022 年 1 月 27 日— [Graviti](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.graviti.com%2F&esheet=52569443&newsitemid=20220127005183&lan=en-US&anchor=Graviti&index=1&md5=d773ab7e9f204dafd6bc432576debd05) ，一家总部位于纽约的现代数据基础设施初创公司，过去三年一直处于隐形模式，今天推出了其首款产品—gravi ti 数据平台。

Graviti 数据平台旨在消除全球人工智能(AI)应用程序开发人员面临的最昂贵和最令人困惑的问题之一:处理大量非结构化数据。该产品不仅消除了管理和处理此类数据时面临的许多麻烦，还从根本上实现了开发人员处理的非结构化数据的可伸缩性和复杂性。企业可以更轻松地在云上更快地获取、管理和查询数据，从收集的数据中识别可操作的见解，并创建开创性的人工智能应用程序。

对于企业和人工智能开发者来说，数据平台的推出恰逢其时。由 Graviti 委托 Propeller Insights 在 2021 年末对 500 多名开发人员进行的调查显示，如今的开发人员被大量的非结构化数据淹没。调查的主要发现包括:

*   几乎所有参与调查的开发人员(94%)都表示，他们当前管理非结构化数据的方法存在不足。
*   四分之一的受访者(25%)表示，他们将一半到三分之二的时间花在与监管非结构化数据相关的耗时且令人沮丧的琐事上:数据收集、清理、选择和探索。

调查显示，花在数据监管上的时间与公司的规模成正比。几乎所有接受调查的开发人员(95%)都承认，他们的组织中存在数据管理方面的棘手问题，近一半(46%)的人表示，从不同且分散的来源中寻找数据是他们最头疼的问题。另有 39%的受访者表示，跟踪数据集是最大的难题，因为他们每天必须生成大量数据。近三分之一(32%)的人认为低质量的数据是最大的障碍。

Graviti 的创始人、前优步先进技术集团机器学习专家 Edward Cui 表示:“根据调查，公司规模越大，开发人员花在处理数据上的时间就越多。”。“在分析给开发人员带来最大麻烦的领域时，我们发现许多用户都面临着类似的挑战。好消息是 Graviti 数据平台消除了这些问题。”

该平台让开发人员能够在一个位置管理他们的原始数据、元数据和语义数据。借助该平台，他们可以利用自定义过滤器快速定位和可视化数据。最终结果是，Graviti 将开发人员从琐事中解放出来，他们现在可以花更多时间分析非结构化数据和训练模型。

Graviti 智能物流合作伙伴之一的算法和人工智能主管 Justin Cai 表示:“通过加入该平台并采用最佳实践，我们的目标是让我们的数据科学家和机器学习工程师能够以最简单、最快的方式访问和操作海量数据集。“我很高兴 Graviti 正式推出其数据平台，它不同于原始文件系统。云数据管理解决方案使开发人员能够看到人工智能的新未来。”

类似 Git 的数据版本控制、工作流自动化和数据可视化功能使工程师能够提高效率。Graviti 数据平台建立在具有弹性和可扩展架构的数据引擎上，为企业提供了基于消费的模型来降低成本。

“我们很高兴推出数据平台，”崔说。“我们给人工智能开发者的信息是，我们已经听到了你想要什么和你需要什么。如果人工智能的发展要取得进展，非结构化数据管理的挑战是最重要的，Graviti 能够帮助解决这个问题。”

**关于 Graviti**

Graviti 由 Edward Cui 于 2019 年创立，是开创性的数据平台，其使命是帮助企业管理和利用大规模非结构化数据。Graviti 在 Pre-Series-A 轮融资 1200 万美元，在全球范围内雇佣了包括优步、谷歌、Airbnb 在内的领域专家，解决了创建现代人工智能应用程序所面临的最严峻的挑战之一。更多信息可以在网上的 www.graviti.com 找到。
