# Digital.ai 深化价值流管理整合

> 原文:[https://devo PS . com/digital-ai-deepens-value-stream-management-integration/](https://devops.com/digital-ai-deepens-value-stream-management-integration/)

Digital.ai 今天宣布，它已经在其同名的[价值流管理](https://devops.com/?s=value+stream+management) (VSM)平台中扩展了机器学习算法的使用，除了增强其风险预测和流程优化模块外，还包括优化流程和提高应用质量的模块。

此外，Digital.ai 还扩大了它现在为数百个 DevOps 工具和平台提供的预构建集成的数量。

Digital.ai 的敏捷和 DevOps 总经理德里克·霍尔特(Derek Holt)表示，目标是扩大核心 Digital.ai 平台与该公司及其第三方 DevOps 合作伙伴提供的模块和工具之间的双向数据流。

Digital.ai 由 XebiaLabs 和 CollabNet VersionOne 合并而成。合并后的公司收购了人工智能分析工具提供商 Numerify、测试工具 Experitest 和应用安全平台提供商 Arxan。Holt 说，随着最新版本的发布，这些功能已经被集成到核心的 Digital.ai 平台中，远远超出了仅仅通过公共用户界面提供访问的范围。

Holt 指出，目标是为组织提供一个 VSM 平台，通过该平台可以做出更好的决策，以优化软件开发和交付，首先从所有使用的 DevOps 工具中收集数据，然后使用机器学习算法来提出优化建议。Holt 指出，随着应用开发环境变得越来越复杂，很明显，大多数 it 团队将需要嵌入 VSM 平台的人工智能能力来管理相互依赖的项目。

有了这些建议，在关键项目比计划晚完成时重新分配资源或转移资源以确保项目按时完成变得更加简单。VSM 平台还使 IT 组织能够评估这些决策将对其余业务产生的影响，而此时组织在财务上从未像现在这样依赖软件的及时交付。对于越来越依赖数字流程来吸引客户的组织来说，单个应用程序模块的延迟交付最终会对收入预测产生深远的影响。

Digital.ai 最近进行的一项调查发现，许多数字化转型计划在某种程度上举步维艰。近一半的受访者(49%)表示，他们没有从数字化转型工作中看到预期的结果，54%的受访者承认他们担心自己的竞争能力。超过 90%的人需要看到他们的数字化转型投资获得更好的回报。

Digital.ai 声称，目前它有超过 1200 家客户，包括安联、德国商业银行、易贝、骏利和李维斯，这些客户正在使用 Digital.ai 平台的元素来更好地管理软件开发项目。这些工具的许多用户现在包括业务主管和其他应用程序利益相关者，他们使用 VSM 平台来跟踪应用程序开发项目的进展。Holt 指出，特定于领域的 Digital.ai 分析镜头使不同类别的最终用户更容易访问预先构建的仪表板和指标。

市场研究公司 Gartner 预测，到 2023 年，70%的组织将采用价值流管理平台来改善其 DevOps 管道中的流量。然而，随着 DevOps 平台和项目管理应用程序的不断发展，许多人也在添加 VSM 平台中的许多分析功能。每个 IT 组织都需要决定哪种方法对他们最有意义，但 Digital.ai 显然认为，随着应用程序开发变得越来越具有挑战性，对集成 VSM 平台的需求将变得更加明显。
