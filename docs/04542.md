## 7 DevOps 趋势

1.  DevOps 将成为主流，并获得大量关注；因此，2017 年将是“开发年”
2.  与 DevOps 一起，三个 C——持续集成、持续部署和持续交付——将获得巨大的动力。
3.  将发布许多新的 DevOps 自动化工具，改变我们进行软件开发的方式。
4.  “集装箱化”也将成为焦点。
5.  许多软件公司将转向微服务架构。
6.  自动化和持续测试将变得更加流行——也更加重要。
7.  包括 Docker、AWS、GitHub 和 JIRA 在内的必备工具和平台将在开发者专区变得更加流行。
