# JAMstack Radio 播客:API 优先 CMS 的优势

> 原文：<https://devops.com/jamstack-radio-podcast-benefits-api-first-cms/>

欢迎来到 [JAMstack Radio](https://www.heavybit.com/library/podcasts/jamstack-radio/ep-1-introducing-jamstack-radio/) ，这是一个讨论 JAMstack 的播客，这是一种构建快速安全的应用程序或网站的新方法。JAMstack 电台由 Brian Douglas 主持，他是 T2 Netlify 的开发者代言人。在这一集里，斯蒂芬·朱迪斯和鲁文·威斯林将和布莱恩一起讨论 [Contentful](https://contentful.com) ，一个开发者友好的 API 优先的 CMS。

他们解释了将你的网站内容从它的表现中分离出来的好处，包括更好的跨平台支持。有了 Contentful，你可以完全自由地用你想要的语言编写你的网站，而该服务负责 CMS 后端，存储和管理你的数据，并备份一切。
