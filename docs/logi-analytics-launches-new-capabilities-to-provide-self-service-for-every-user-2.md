# Logi Analytics 推出新功能，为每个用户提供自助服务

> 原文：<https://devops.com/logi-analytics-launches-new-capabilities-to-provide-self-service-for-every-user-2/>

*Logi Composer 6 以上下文和安全性为中心，扩展了对用户体验的最终控制*

**北卡罗来纳州罗利，2021 年 5 月 19 日—**[insight software 公司和软件团队嵌入式分析解决方案的领先提供商 Logi Analytics](http://www.logianalytics.com/) ，今天在 [Logi Composer](https://www.logianalytics.com/logi-composer/) 中推出了新功能，这是第一款开箱即用的嵌入式分析开发体验。Logi Composer 6 通过事件监听器框架、具有仪表板交互性的自助服务功能以及每一层的企业级安全性，增强了对用户体验的控制。

Logi Analytics 和企业战略集团(ESG)最近的研究发现，49%的数据领导者正在投资新的分析供应商和[自助服务](https://www.logianalytics.com/news/investing-in-analytics/)，以帮助缩小技能差距。Logi Composer 超越了简单的可视化，通过控制自助服务体验(包括调整大小、布局、过滤器、链接和创建跨源连接)来个性化自助服务，使自助服务用户能够构建自己的仪表板，其复杂程度符合他们的需求。

Logi Analytics 产品管理副总裁 Charles Caldwell 表示:“终端用户希望在现有应用中获得分析体验，他们可以在其中定制、操作和查询数据，以获得洞察力并增强决策能力。“而且太多的自助服务解决方案不符合用户的需求和技能水平。在此版本中，Logi Composer 为软件团队提供了一个低代码、安全的解决方案，该解决方案将超越用户的个性化和自助服务要求，并为管理员提供粒度控制，以针对不同的角色和用户成熟度调整分析体验。"

Logi Composer 还优先考虑企业安全性，通过对行和列的精细数据安全控制，在多租户部署中保持精确控制。此外，Composer 通过控制数据访问和用户权限的仪表板和源的对象级安全性简化了协作和共享。

主要特性包括:

*   通过嵌入多个独立的仪表板，将正确的数据和相关上下文整合到您的应用程序体验中，改进了应用程序工作流程。

*   能够利用新的弧形和组合图表在您的应用程序中为分析仪表板构建基于范围的多维可视化。

*   借助新的深入查看、时间间隔和汇总汇总功能，以表格形式直观显示和报告，增强了自助服务体验。

*   改进了在多租户部署中运行复杂查询的功能，支持自定义查询中的变量。

*   OAuth 集成支持用户授权，可将分析无缝嵌入到您的应用中，实现顺畅的最终用户体验。
