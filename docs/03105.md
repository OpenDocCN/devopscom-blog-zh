# Aqua Security 为日立企业云容器平台提供核心安全组件

> 原文：<https://devops.com/aqua-security-provides-core-security-component-for-hitachi-enterprise-cloud-container-platform/>

*Aqua Security 扩展其 Aqua Link 合作伙伴计划，纳入托管服务提供商，为云原生工作负载提供保护*

**波士顿，马萨诸塞州–2018 年 9 月 26 日**–[Aqua Security](https://www.aquasec.com/)，领先的基于容器和云原生应用安全平台提供商，今天宣布 Hitachi Vantara 已将 Aqua Security 功能集成到日立企业云(HEC)容器平台中。HEC Container Platform 为在私有云或混合云中使用本地数据服务、开发运维及微服务架构的组织提供公共云体验。Hitachi Vantara 选择 Aqua 平台的部分原因是它能够为多租户云容器即服务环境提供运行时安全性。

[HEC Container Platform](https://www.hitachivantara.com/en-us/solutions/hybrid-cloud/enterprise-cloud.html)是一个完全托管的端到端解决方案，用于部署 Spark、Kafka、Cassandra 和 HDFS 等数据服务。Kubernetes 容器编排也是现成的，与 Docker 和本机 Marathon orchestrator 一起，为客户提供简化的单击部署 Marathon 和 Kubernetes。HEC Container Platform 用户可以按需运行容器，而无需配置或管理虚拟机实例。

Hitachi Vantara 全球解决方案副总裁 John Murphy 表示:“集成 Aqua 解决方案使我们能够在开发运维流程的每个阶段提供更全面、更精细的方式来保护客户数据、应用和工作负载。“此外，Aqua 的多租户解决方案允许我们自动隔离客户内容，并在各种环境中一致地应用基于策略的安全控制。Aqua 帮助我们为客户在 HEC Container 平台环境中移动和安全管理其工作负载提供了一种便捷的体验。”

Aqua 业务开发副总裁 Upesh Patel 表示:“我们很高兴与日立合作，让组织能够利用容器和云原生环境带来的优势，而不必管理复杂的基础设施或担心保护这些环境。“与 Aqua Link 合作伙伴一起，我们致力于将安全最佳实践融入整个容器环境，提供最佳的网络威胁防御，并弥合开发运维与 IT 安全之间的差距”。

**Aqua Link 合作伙伴计划扩大到包括 MSP**

作为容器和云原生应用安全领域的领导者，Aqua 与全球合作伙伴密切合作，这些合作伙伴了解保护和监控容器化应用的重要性，并努力利用企业级技术增强其解决方案组合。Aqua Link 合作伙伴计划涵盖广泛的合作伙伴模式，包括在云原生堆栈中提供集成产品的技术联盟合作伙伴、提供 Aqua 产品并提供技术服务以在客户环境中部署和管理 Aqua 的解决方案提供商，以及在全球范围内拥有专业知识和训练有素的员工来支持 Aqua 实施的系统集成商和顾问。

通过引入对管理多租户环境的支持，以及所需的策略、实施、监控和报告分离，Aqua 在通过 Hitachi Vantara 等托管服务提供商提供深度集成的解决方案方面具有独特的优势。
