# Lightbend 完成了由英特尔投资领导的 2000 万美元 C 轮融资

> 原文：<https://devops.com/lightbend-closes-20-million-series-c-funding-led-intel-capital/>

随着越来越多的全球 2000 强企业转向基于微服务的云优先业务模式，反应式系统领域的先锋加速了增长

加利福尼亚州旧金山—(market wired—2016 年 5 月 26 日)Lightbend(前身为 Typesafe)是一家开发反应平台的公司，该平台包括 Akka、Play 和 Lagom 框架以及 Scala 编程语言，该公司今天宣布完成了由英特尔投资领导的 2000 万美元的 C 轮融资。Blue Cloud Ventures 也作为新投资者加入了英特尔投资，之前的投资者贝恩资本风险投资公司、Polytech 生态系统风险投资公司和沙斯塔风险投资公司也全面参与。英特尔投资的企业软件总监 Vibhor Rastogi 将加入 Lightbend 的董事会。

包括沃尔玛、威瑞森、iHeartRadio、William Hill 和三星([参见案例研究](http://ctt.marketwire.com/?release=1245312&id=8602549&type=1&url=http%3a%2f%2fwww.lightbend.com%2fresources%2fcase-studies-and-stories%23filter%3amicroservices))在内的企业已经采用 Lightbend Reactive 平台来构建基于现代微服务架构的低延迟、快速数据应用。Lightbend 的平台和专业服务已经成为企业的热门选择，这些企业利用 Java 虚拟机(JVM)和 JVM 语言，如 Java 和 Scala，并寻求由[反应宣言](http://www.reactivemanifesto.org/)定义的应用弹性和极端规模的特征。

Lightbend 首席执行官 Mark Brewer 表示:“我们将继续为全球 2000 强企业提供技术、服务和专业知识，这些企业正在重构其应用架构，以实现敏捷开发周期和内部及云中的极端规模，英特尔投资和 Blue Cloud Ventures 的投资让 Lightbend 感到非常兴奋。“企业之所以选择 Lightbend，是因为我们的跟踪记录使团队能够更快地发布更好的代码，并且在运营模型中经得起最极端的规模。”

英特尔公司高级副总裁兼软件与服务事业部总经理道格·费舍尔表示:“移动和物联网用例正在推动企业实现处理大量数据的现代化。" Lightbend 为开发、部署和管理当今的大规模分布式应用程序提供了基础构件."

“这些变化正在发生，因为近年来应用程序的需求发生了巨大的变化，”Lightbend 的创始人兼首席技术官、Akka 的创始人 Jonas Bonér 说。“仅在几年前，一个大型应用程序拥有数十台服务器、数秒的响应时间、数小时的离线维护和数千兆字节的数据。如今，从移动设备到运行数千个多核处理器的基于云的集群，应用程序部署在各种设备上。用户期望毫秒级的响应时间和 100%的正常运行时间。过去的软件架构根本无法满足今天的需求。”

关于御光术

Lightbend (Twitter: @lightbend)致力于帮助开发者在 JVM 上构建反应式应用。借助 Lightbend Reactive Platform，开发人员可以使用 Lagom、Play Framework、Akka、Apache Spark、Scala 和 Java 等技术，创建可在多核和云计算架构上扩展的消息驱动型应用。为了帮助我们的客户取得成功，Lightbend 与 Databricks、IBM 和 Mesosphere 等技术先锋合作。Lightbend 是由英特尔投资、蓝云、贝恩资本和沙斯塔风险投资公司(Shasta Ventures)支持的风险投资公司，总部设在旧金山。

关于英特尔投资

英特尔投资是英特尔的战略投资和 M&A 组织，支持以计算和智能设备、云、数据中心、安全、物联网、可穿戴和机器人技术以及半导体制造为目标的创新创业公司。自 1991 年以来，英特尔投资已向 57 个国家的 1，445 家公司投资了 117 亿美元，595 家投资组合公司上市或被收购。通过其业务发展计划，英特尔投资每年在全球 2000 强企业中为其投资组合高管与英特尔的客户和合作伙伴之间策划数千次介绍。欲了解英特尔投资成为全球最强大风险投资公司的更多信息，请访问[www.intelcapital.com](http://www.intelcapital.com/)或关注[@英特尔资本](https://twitter.com/intelcapital)。
