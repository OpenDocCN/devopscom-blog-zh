# 如何建立以人为本的工程团队

> 原文：<https://devops.com/how-to-build-a-people-first-engineering-team/>

当我们想到成功的技术团队时，有一种倾向是主要关注正在创造的技术，而不是创造它的人。虽然这两种方法都可以导致成功的产品，但是当软件工程师成为经理而不一定拥有有效管理的正确工具时，技术优先的方法通常是默认的。成为一名工程经理需要一套全新的技能，包括不同的沟通方式、时间管理、组织、工具和软件、工作范围、学习如何有效地授权以及确定合适的人来完成[团队](https://devops.com/?s=team+culture)。

作为一名首席技术官，在经历了这一转变后，我注意到，经理越是能够与团队成员保持同步，他们就越不会感到疲惫，员工保留率就越高，员工的工作也会做得更好。

虽然大多数人不会完全以人或技术为中心，但这里有一个差异的快速概述:

*   以团队为导向的经理知道他们的员工正在经历的个人和职业斗争，并且有一种感觉，他们管理的团队正在为同一个目标共同努力。当你与这种类型的经理交谈时，他们自然会倾向于提供关于个人福祉的状态更新。
*   倾向于从更技术性的角度来看待团队的管理者将会识别项目的部分或者影响团队的技术或战略障碍。

这些不同类型的经理会以非常不同的方式处理同样的问题。当你谈论你的团队时，记录下你自己是有益的，这样可以确定你的重点在哪里，并评估这是否适合你。

### 与团队成员建立有效的一对一沟通

个人贡献者通常习惯于每周 1:1，与他们的经理一起解决他们可能遇到的任何问题。当他们成为经理时，除了建立信任之外，指导团队成员解决他们自己的问题也成了他们的职责。这种角色转变需要准备，如果你不确定如何做好，你可能会发现起草一份议程很有帮助，如果需要的话你可以遵循。你也可以通过与更有经验的经理交谈来了解这个过程。

你在一对一的会议中谈论什么？即使你是一个更注重技术的人，与你的团队的 1:1 会议也不应该是关于项目的状态，而应该更多地是关于你与之交谈的人是如何工作的，以及作为一名经理，你如何能提供帮助。我自己开始是一名软件工程师，现在是一名 CTO，我发现下面的问题对我如何最好地帮助我的团队在他们的角色中茁壮成长很有帮助。

1.  “你过得怎么样？”如果你和一个团队经理谈话，另一个很好的问题是，“你的团队做得怎么样？”除了工作的技术方面，你想真正了解个人的状况。
2.  “你有什么反馈给我吗？”我总是寻求反馈。一段时间，他们可能没有任何；但是我经常发现他们从一个小问题开始，比如“小吃的更好选择”这是一个通过改善工作体验来建立信任的绝佳机会。下一次，他们可能会分享更多，直到你收到真正重要的反馈，同时也改善了关系。这里要记住的重要事情是对反馈做出回应。
3.  "从 0 到 10 分，你在工作中得到多少乐趣？"我真的希望人们在工作中获得乐趣。你可以让他们给你一个 0 到 10 之间的数字。当他们用一个数字来回答时，比如说 6，一个很好的后续问题可以是问需要什么来改变这种情况，使这个数字变成 7。你如何帮助他们到达那里？对于这个问题，我听到了各种各样的答案，从简单到复杂；从允许一天一小时不回应到渴望学习做不同类型工作所需的技能。
4.  “你对未来两年和 10 年的自己有什么看法？”作为一名经理，帮助你的团队确定长期和短期目标是非常重要的一部分。特别是对于工程师来说，考虑一下你是否想继续在工程岗位上晋升，或者转换到管理岗位是否更合适，这是一个好主意。
5.  “我怎样才能帮助你在事业上更上一层楼？”在平衡日常工作和个人生活时，经常会偏离正题。您还可以利用一对一会议来帮助团队成员确定他们可以做些什么来实现他们的短期和长期目标，以及如何保持更好的工作和生活平衡。

### 支持同事

对大多数人来说，过去的一年在不同程度上是艰难的，这巩固了我们需要在困难时期支持团队成员的想法；这超越了疫情。这可能意味着通过快递送上他们最喜欢的食物，送上一个爱心包裹，或者简单地明确表示，当你团队中的某人需要外出或休假时，这很好——不要问任何问题。当你经历困难的时候，能够离线而不需要做长时间的交接是非常重要的。

在转移到远程工作的过程中，就像许多领导团队一样，我的公司违背了将生活和工作分开的文化规范。我们承认现在做这件事太难了，并且公开承认每个人都在经历一些事情。感觉精神带宽被工作之外的事情填满了，所以作为团队成员，最好确保工作不会增加压力。

在某种程度上，这个疫情时期已经打破了工作与“生活”之间的壁垒，这使得以人为本几乎成为一种义务。在我看来，这是我们如何开展业务的一个突破。通过思考团队中的每一个人，每个人都可以被提升和倾听，这可以成为更多创新的动力。如果你全力以赴，系统没有任何松懈，没有经理和队友的支持，你就会失去创造力。作为创新团队，这一经历提醒我们将“以人为本”的原则融入我们的文化，以促进健康的工作与生活平衡，从而带来创新的解决方案和更好的结果。
