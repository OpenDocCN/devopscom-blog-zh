# Zerto 加入 AWS ISV 工作负载迁移合作伙伴计划，以加快其云数据管理和保护解决方案的采用

> 原文：<https://devops.com/zerto-joins-aws-isv-workload-migration-partner-program-to-accelerate-adoption-of-its-cloud-data-management-and-protection-solutions/>

*确保虚拟机、亚马逊 EC2 和亚马逊 EKS 工作负载的企业级保护和恢复*

**波士顿，马萨诸塞州–2022 年 8 月 10 日**–[惠普企业公司 Zerto](https://www.zerto.com/) 今天宣布加入亚马逊网络服务 [(AWS) ISV 工作负载迁移合作伙伴计划](https://aws.amazon.com/partners/programs/isv-workload-migration/)，使其能够推动和交付 AWS 工作负载迁移，并加快其云数据管理和保护解决方案的采用。

Zerto 客户将从增强的技术支持中受益，该技术支持通过可重复和可扩展的方法、自动化和利用 AWS 最佳实践的工具来加速迁移。该计划还提供对 AWS 促销积分的访问，以抵消迁移成本，包括劳动力、培训和运行并行环境。此外，Zerto 将获得上市支持。

无论是迁移或保护内部的虚拟机(VM)或容器，还是迁移到云中或云中，Zerto 解决方案都可以利用 AWS 的灵活容量，同时最大限度地减少停机时间并帮助防止数据丢失。这使用户能够轻松实现他们的恢复点和恢复时间目标。

Zerto 的纯软件解决方案可确保虚拟机、亚马逊弹性计算云(亚马逊 EC2)和亚马逊弹性 Kubernetes 服务(亚马逊 EKS)工作负载的企业级保护和恢复。这包括易于扩展的基于云的设备，这些设备管理复制以实现以应用程序为中心的保护和恢复或工作负载迁移，并通过基于日志的复制实现连续数据保护，以实现内部到云或云到云。

Zerto 全球公共云联盟总监 Christina Rauscher 评论道:“Zerto 很荣幸能够与 AWS 合作，帮助客户保护他们的所有数据。“通过合作，我们可以帮助客户更好地利用 AWS，增加迁移、灾难恢复、勒索软件恢复、备份和长期保留功能。”

**关于 Zerto**

Zerto 是一家惠普企业公司，通过简化内部部署和云应用程序的保护、恢复和移动性，使客户能够运营永不停机的业务。Zerto 的云数据管理和保护平台消除了私有、公共和混合部署中现代化和云采用的风险和复杂性。这个简单的纯软件平台使用大规模连续数据保护来融合灾难恢复、备份和数据移动。Zerto 受到全球 9，500 多家客户的信任，并为 Microsoft Azure、IBM Cloud、Google Cloud、Oracle Cloud 和 350 多家托管服务提供商的产品提供支持。
