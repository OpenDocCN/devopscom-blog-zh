# DBmaestro 扩展其新的数据库 DevOps 平台以支持 DB2

> 原文：<https://devops.com/dbmaestro-expands-its-new-database-devops-platform-to-support-db2/>

使用 DB2、SQL 和 ORACLE 提供了管理多供应商环境的“零摩擦”

马萨诸塞州波士顿，2018 年 6 月 13 日——领先的数据库开发运维解决方案提供商 DBmaestro 今天宣布，该公司已扩展其数据库自动化平台，为 IBM DB2 实现 CI/CD 和发布自动化。由于 DB2 具有业界领先的性能、规模和可靠性，它是全球股票市场、投资银行、保险公司和养老基金的首选数据库。
DBmaestro 的数据库 DevOps 平台加快了发布速度，提高了生产率，最大限度地减少了错误，并提高了 MS-SQL、ORACLE 以及现在的 DB2 的安全性。DBmaestro 平台可以在 DB2 z/OS 和 DB2 LUW 上完全运行。

DBmaestro 正在实现成为数据库开发运维服务提供商的愿景，提供适用于多种数据库的解决方案。DBmaestro 为跨平台和操作系统的 CI/CD 采用提供了一个总体解决方案，满足了跨行业和跨国企业不断发展的需求。

DBmaestro 的零摩擦解决方案使系统管理员能够更轻松地管理多个数据库，而无需改变现有流程。多样化平台包括一个安全和治理模块，可保护数据库的完整性并防止数据丢失，这对于金融机构来说至关重要。通过跟踪所有环境中的数据库变化，DBmaestro 促进了无缝部署，同时使组织能够更快地构建和交付更好的产品，支持越来越短的产品发布周期。DBmaestro 首席技术官 Yaniv Yehuda 表示:“我们的银行和保险客户需要能够以高度的信心、最小的风险和最大的效率执行强大而灵活的数据库活动。“DBmaestro 是 DevOps 平台，可为数据库提供灵活性和安全性，使金融企业能够跟上业务变化的步伐，同时遵守法规并保护客户的个人数据。”

主要平台特性包括:
发布自动化功能，可在所有环境中自动部署版本
角色管理、访问控制和可定制的组织策略，可增强安全性和治理
业务活动监控，可满足关键绩效指标并衡量成功
通过 API 和适配器为 DevOps 生态系统中的工具(如詹金斯、JIRA、赛博方舟等)提供现成的连接

关于 DBmaestro
DBmaestro 为数据库带来了 DevOps 最佳实践，为数据库提供了全新水平的效率、速度、安全性和流程集成。DBmaestro 的平台使组织能够安全有序地运行数据库部署，提高开发团队的工作效率，并显著缩短上市时间。该解决方案使组织能够针对数据库活动实施 CI/CD 实践，并具有可重复的管道发布自动化和自动漂移预防机制。欲了解更多信息，请访问 www.dbmaestro.com。
