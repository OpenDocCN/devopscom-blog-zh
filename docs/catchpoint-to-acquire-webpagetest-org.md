# 收购 Webpagetest.org 的切入点

> 原文：<https://devops.com/catchpoint-to-acquire-webpagetest-org/>

*收购将大幅提升 Webpagetest.org 的核心能力；为网络性能社区提供卓越的性能监控*

**(美国纽约州)**—**2020 年 9 月 16 日—**[catch point](https://www.catchpoint.com/)，数字体验监测的领导者，今天宣布已签署协议收购 Webpagetest 和[Webpagetest.org](https://www.webpagetest.org/)，后者是一家开源的 web 性能测试软件和服务。该交易将允许 Catchpoint 加强和扩大 Webpagetest 的能力和 Webpagetest.org 的地理性能测试足迹，利用 Catchpoint 的一流基础设施，增加容量并提高分析的一致性和质量。

随着数十亿消费者和企业员工快速转向 SaaS 和基于网络的应用，企业一直在努力管理应用性能并为用户提供可靠的数字体验。此外，在数字经济中保持竞争优势不再是一件好事，而是一项重要的业务要务。在数字经济中，如果一家公司不能向其客户和员工提供快速、可靠和一致的 web 应用体验，他们的业务将受到负面影响，要么是客户流失，要么是员工生产力和士气低下。

Catchpoint 和 Webpagetest 从一开始就有着共同的使命；帮助 web 开发人员和运营团队向他们的用户提供最佳的数字和 web 体验。将 Catchpoint 的企业级全球监控网络基础设施与广受欢迎的 Webpagetest 的 web 性能测试功能相结合，将创建一个无与伦比的 web 性能测量和监控平台。

Webpagetest.org 服务每月支持 100 多万用户，允许他们从不同地点运行即时测试，并接收对其网页和 web 应用程序的详细性能审计。

作为网页测试的新托管人，Catchpoint 将投资大幅改善免费 Webpagetest.org 体验。Webpagetest.org 不仅将继续作为一项免费服务提供给社区，而且在 Catchpoint 下，这项服务将得到大大加强。web 性能社区将继续免费使用 Webpagetest.org，极大地提高测试结果的一致性、容量和质量。

[阅读 Patrick Meenan 给社区的个人留言](https://www.catchpoint.com/webpagetest-joins-catchpoint)

Catchpoint 工程研究员、Webpagetest.org 创始人帕特里克·米南(Patrick Meenan)表示:“如今，IT 团队正在努力满足网络和应用性能预期，并跟上数字化转型的需求和增长。“通过将 Catchpoint 著名的命令和控制架构和分布式足迹与 Webpagetest web 性能测试功能相结合，IT 团队将拥有快速监控、检测、识别和解决性能和可用性问题所需的洞察力。”

“这是我最近听到的最激动人心的行业新闻，也是 web 性能社区向前迈出的一大步。”Fastly 技术副总裁胡曼·贝赫什蒂(Hooman Beheshti)说。“我们是 Catchpoint 和 Webpagetest 的忠实粉丝，并在我们的监控实践中使用这两个平台。将这两者结合起来将使我们的生活变得更加轻松，并将节省隔离和应对大规模挑战的宝贵时间。”

“Patrick 是著名的 web 性能倡导者，也是几个关键 web 性能指标(如 SpeedIndex)背后的驱动力，这些指标可以更好地捕捉用户体验。他将在 Webpagetest 社区的未来发展中发挥关键作用，”Catchpoint 首席执行官 Mehdi Daoudi 说。“我们致力于为社区提供更好的免费工具体验，并将继续尊重 Webpagetest 社区的开源价值观。”
