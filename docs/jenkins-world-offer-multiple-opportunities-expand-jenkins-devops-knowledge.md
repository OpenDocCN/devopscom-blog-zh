**Jenkins 认证培训和免费认证考试:9 月 12 日至 13 日**
本课程由来自|opensourcearchitect|和 Nebulaworks 的专家讲授，旨在帮助学员准备 Jenkins 认证考试。它将涵盖所有的考试主题，还将有一系列的实验，涉及整个课程中的几个主题。与会者将:

*   **向 Jenkins 专家学习:【| opensourcearchitect |和 Nebulaworks 都经过认证，可以持续提供 Jenkins 培训和教授关于 Jenkins 的课程。**
*   **准备认证:**准备好参加认证詹金斯工程师(CJE)和认证 CloudBees 詹金斯平台工程师(CCJPE)考试。
*   **参加培训，然后参加会议:** Jenkins World 提供 50 场会议，涵盖各种主题和技能水平。
*   **参加免费的认证考试:**只要您在会议注册时报名参加，就可以在整个会议期间方便的时候免费参加认证考试。

**Jenkins Pipeline 和 Docker 培训基础:9 月 12 日**
本课程介绍 Pipeline 插件，与 Docker 结合使用。在对 Pipeline 和 Docker 进行概述后，参与者将:

*   **了解对管道的需求**:了解关键用例、结构/语法和执行控制。
*   **了解如何准备 Docker 环境**:接触关键的 Docker 工具，了解如何使用 Pipeline 和 Docker 创建部署生命周期。
*   **获得实践经验**:在与管道和码头专家一起参与两个实际项目的过程中，实时观察、学习。

**研讨会:9 月 13 日**

*   ***devo PS 2.0 工具包*** **研讨会**
    该研讨会将涵盖整个微服务开发生命周期，从架构设计到生产等。

    Jenkins Pipeline 是用 Jenkins 编写自动化作业的一种改变游戏规则的方式。来学习如何。

    评估您对詹金斯认证考试的准备情况。

**关于 Jenkins 和 Jenkins 社区**
Jenkins 是领先的开源自动化服务器，由开发人员、测试人员、设计人员和其他对持续集成、持续交付和其他现代软件交付实践感兴趣的人员组成的庞大且不断增长的社区提供支持。它建立在 Java 虚拟机(JVM)之上，提供了超过 1200 个插件，这些插件扩展了 Jenkins，使其能够自动处理软件交付团队使用的几乎任何技术。

Jenkins 社区倡导将 Jenkins 用于全球开发社区，为所有经验水平的 Jenkins 用户提供教程、论坛和其他有用的资源。通过对詹金斯项目的众多贡献者的认可，该项目为维护和进一步发展詹金斯创建并培养了一个社区供电的基础设施。你可以在 [Twitter (@JenkinsCI)](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Ftwitter.com%2FCloudBees&esheet=51402270&newsitemid=20160816005286&lan=en-US&anchor=Twitter%C2%A0%28%40JenkinsCI%29&index=9&md5=6dafb6585955aeb3bd13d5b22a131065) 上关注詹金斯社区。
