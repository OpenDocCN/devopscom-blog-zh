# Boca Code 奖“改变工程”奖学金

> 原文：<https://devops.com/mediaops-boca-code-award-engineer-the-change-scholarship/>

我非常自豪地宣布，MediaOps 与 Boca Code 合作，颁发了 2021 年秋季奖学金给 Boca Code 的 [软件工程职业课程](https://www.bocacode.com/courses/career/software-engineering) ，这是一个为期 10 周的严格、快节奏的编码训练营。 来自乌克兰的第一代移民 Alina Maliuk 是接受者。

Engineer The Change scholarship 由 MediaOps 发起，旨在促进当地科技行业的多元化，并帮助来自欠发达社区的个人发展在 IT 行业取得成功所需的技能。Boca Code 提供全面的软件开发培训，使课程参与者能够发展这些技能。

## STEM 领域对多样性的需求

研究显示，STEM 领域极度缺乏多样性，最明显的是计算机和工程职位。科技行业主要是白人和男性，这让我感到震惊(是的，尽管我是白人男性)，我想做一些事情来鼓励我们行业的多样性。性别和种族不平等是显而易见的劳动力差异，需要技术世界最高层的认可和责任，以创造更加多样化和公平的劳动力。

## 每年颁发两项全额奖学金

MediaOps 每年赞助两个 10，000 美元的奖学金，使学生能够参加 Boca Code 的旗舰软件工程职业课程。为期 10 周的全日制强化课程结合了理论和基于项目的实践学习，为学生从事软件工程职业做准备。作为他们课程的一部分，学生承担当地企业的发展项目，为学生提供真实项目的直接实践经验。

工程师改变奖学金结合了两个对我和 MediaOps 团队至关重要的目标。首先是尽我们的一份力量，让南佛罗里达成为世界级的科技产业中心。第二是为来自弱势群体的个人提供机会，让他们获得在科技领域取得成功所需的技能。我希望我们也能激励其他当地公司资助类似的项目来支持这些有价值的目标。

## 关于我们的冠军，艾莉娜·马柳克

![Alina Maliuk scholarship](img/e0604d6bb7e942ff514b36a4c451a670.png)

**Alina Maliuk, Engineer the Change scholarship winner**

Alina 来自乌克兰，她在那里获得了管理学学士学位。大学毕业后，她结婚并有了第一个儿子。2012 年她休产假期间，全家移民纽约。作为第一代移民，他们不得不在美国从头开始生活。最初几年，Alina 和她的丈夫从事初级工作。2018 年，Alina 现在是两个孩子的母亲，她参加了几门网络开发的在线课程，并重新发现了她对学习的热爱——特别是软件开发。Alina 现在是 Boca Raton 的居民，她发现了 Boca Code，并参加了一个 web 开发研讨会。

那次研讨会激发了 Alina 的活力，推动了她已经开始的继续教育。通过最初的课程，Boca 代码团队激励她走得更远。她准备继续她的下一个职业，她想从事软件工程。感谢工程师变革奖学金，她现在将有机会加速她的技术职业生涯。与此同时，她正在向佛罗里达州的其他女性移民证明，如果你能梦想并努力实现它，你就能实现它。

Alan Shimel

MediaOps 首席执行官
