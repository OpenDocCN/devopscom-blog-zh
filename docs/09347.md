# 软件成为网络威胁的 15 种方式

> 原文：<https://devops.com/15-ways-software-becomes-a-cyberthreat/>

软件是私人和商业生活不可分割的一部分；这是没有办法的。你需要软件来处理你的税务、预订机票或浏览互联网。软件在许多方面使我们的生活变得更加容易。然而，随着我们越来越依赖软件，我们也越来越容易受到网络攻击。

本文将探讨软件最终成为网络安全威胁的 15 种不同方式。通过了解这些风险，您可以采取措施保护自己和您的企业。

### 缺乏安全功能

许多流行的软件程序要么缺乏基本的安全功能，要么现有的功能 [失去功能](https://www.atera.com/blog/the-growing-problem-of-software-regression/) ，使用户容易受到攻击。例如，Adobe Reader 和 Microsoft Word 都被发现存在安全漏洞。如果您使用这些程序，请确保安装了最新的安全补丁。

如果您担心在安装新软件更新时必须处理停机时间，您可以将它们安排在您知道不会使用电脑的时间。这样，您可以确保您的软件始终是最新的和安全的。

### 密码安全性差

用户帐户被黑的最常见方式之一是使用弱密码或容易被猜到的密码。如果您使用需要密码的程序，请确保使用别人很难猜到的强密码。

此外，如果您对多个帐户使用相同的密码，攻击者只需找出一个密码就可以访问您的所有帐户。这就是为什么不同的服务使用不同的密码很重要。如果您不记得所有不同的密码，您可以使用密码管理器来帮助您跟踪它们。

考虑使用密码管理器——以利用这些应用程序通常具有的更强大的安全性——以及密码随机数发生器，它将为您生成长而随机的密码。许多密码管理器包括一个随机数发生器功能。

### 网络钓鱼攻击

钓鱼攻击可以影响任何类型的软件程序。在网络钓鱼攻击中，攻击者会伪装成合法的网站或程序，试图诱骗您向他们提供您的密码或其他敏感信息。对你收到的任何要求提供个人信息的电子邮件或信息要非常小心，除非你完全确定它们是合法的，否则永远不要点击这些信息中的链接。

通过软件程序进行的网络钓鱼也可能通过 Skype 或 Facebook Messenger 等消息程序或 Gmail 等电子邮件程序进行。一定要小心通过这些程序收到的任何链接或附件。

### 键盘记录器

键盘记录器是恶意程序，在你不知情的情况下安装在你的计算机上。一旦安装，他们可以跟踪你的每一次击键，这意味着他们可以很容易地窃取密码和其他敏感信息。键盘记录器可以通过电子邮件附件、恶意网站甚至受感染的 USB 驱动器安装。

为了保护自己免受击键记录程序的攻击，千万不要安装来源不可靠的软件。对你打开的邮件和访问的网站要非常小心。如果你的电脑上有键盘记录程序，确保尽快运行恶意软件扫描来删除它。

### 路过式下载

是另一种恶意软件，可以在你不知情的情况下安装在你的电脑上。当您访问恶意网站或点击恶意链接时，通常会下载这些类型的程序。一旦它们被安装，它们就可以窃取你的密码或跟踪你的网页浏览。

为了保护自己免受恶意下载，请小心您访问的网站和单击的链接。如果您认为您可能下载了恶意程序，请尽快运行恶意软件扫描。

### 恶意软件

恶意软件是一种恶意软件，它可以在你不知道的情况下感染你的计算机，一旦它出现，它可以做一些事情，如窃取你的密码或数据，或让攻击者远程访问你的机器。从那里，攻击者可以潜在地访问您组织的 IT 基础设施。要非常小心你打开的电子邮件附件和你访问的网站，因为这是恶意软件在你的电脑上最常见的两种方式。

您使用的软件也可以用来将 [勒索软件](https://devops.com/elasticsearch-ransomware-attacks-highlight-need-better-security/) 引入您的系统。恶意行为者可以创建一个流行软件程序的假版本，并在网上发布。当用户下载并安装假冒软件时，他们会在不知情的情况下允许攻击者访问他们的系统。或者，攻击者可以利用软件程序中的漏洞来访问用户的系统。一旦进入系统，攻击者就可以安装勒索软件并加密用户的文件。

### 不安全的 Wi-Fi 网络

如果您使用公共 Wi-Fi 网络连接到互联网，您可能会将自己和您的组织置于风险之中。这些网络通常是不安全的，这意味着网络上的任何人都可以窥探你的流量，看到你在做什么。如果您需要使用公共 Wi-Fi，请确保首先连接到安全的 VPN，以便您的流量得到加密和保密。

### 社会工程

社会工程是一种攻击，攻击者试图欺骗你做一些事情，让他们访问你的帐户或数据。例如，他们可能假装是您使用的程序的客户支持代理，并要求您输入密码，以便他们可以“修复”您帐户的问题。

或者，他们可能会向您发送一封看似来源可靠但实际上包含恶意链接的电子邮件。对您收到的任何通信都要非常小心，即使它们看起来来自合法来源。

### 不安全的浏览器扩展

如果你使用像谷歌 Chrome 或 Mozilla Firefox 这样的网络浏览器，你可能已经安装了一些扩展来增加额外的特性或功能。但是其中一些扩展实际上会带来安全风险。例如，一个扩展可能可以访问您访问的所有网站以及您在其中输入的数据。对于安装哪些扩展要非常有选择性，并且只安装来源可靠的扩展。

### WordPress 插件

如果你有一个 WordPress 网站，小心你安装的插件。 [一些插件](https://portswigger.net/daily-swig/unpatched-plugins-threaten-millions-of-wordpress-websites) 会带来安全风险，例如，让攻击者访问您的网站或数据库。仅安装受信任的插件，并确保它们保持最新。此外，在浏览新插件时，确保下载并安装定期更新的插件是很重要的。在决定安装之前，先看一下软件上次更新的时间。

### 广告软件

广告软件是一种在计算机上显示广告的软件，通常以弹出广告的形式出现。虽然并非所有的广告软件都是恶意的，但某些形式的广告软件可以跟踪您的在线活动，甚至收集您的密码或信用卡号等敏感信息。对你安装的程序要小心，在同意任何事情之前都要阅读 EULA。

### **谷歌文档**

Google Docs 是一个流行的基于云的文字处理应用程序，是 Google Workspace 的一部分。虽然一般来说使用它是安全的，但也有一些恶意行为者使用它来传播恶意软件或发起网络钓鱼攻击的报告。如果你使用谷歌文档，确保只打开来源可靠的文档，不要点击文档中的任何链接，除非你完全确定它们是安全的。

### **第三方应用商店**

如果你使用 Android 设备，你可能会从第三方应用商店而不是官方的 Google Play 商店下载应用程序。虽然有一些合法的应用程序商店，但其中许多都充满了恶意软件和其他恶意程序。为了避免这些风险，最好还是坚持使用你的设备的官方应用商店。

### 过时的软件

确保计算机安全的最重要的事情之一是确保所有软件都更新到最新版本。软件开发人员会定期发布修补安全漏洞的更新，因此在这些更新可用时尽快安装非常重要。您通常可以将软件设置为自动更新，也可以定期手动检查更新。

### **霸占循环**

霸占循环是一种攻击，攻击者通过命令行界面发送恶意命令来控制您的计算机。如果您不小心下载并运行了一个恶意程序，或者如果您访问了一个被攻击者破坏的网站，就会发生这种情况。为了保护自己免受这种类型的攻击，请小心您下载和运行的程序，并且只访问您信任的网站。

## 结论

软件，以其无数的形式，对于经营我们的业务和提高我们的个人生活是至关重要的。几乎没有办法避免每天使用几个甚至几十个不同的程序。但是正如我们所见，如果使用不当，即使是看起来最无害的软件也会带来严重的安全风险。但是提前了解攻击媒介有助于保护您和您的组织免受这些风险的影响，并保护您的数据安全。
