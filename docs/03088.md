# 企业管理协会将 ScienceLogic 评为 IT 运营领域的顶级人工智能/人工智能供应商

> 原文:[https://devo PS . com/enterprise-management-associates-names-science logic-as-top-ai-ml-vendor-for-it-operations/](https://devops.com/enterprise-management-associates-names-sciencelogic-as-top-ai-ml-vendor-for-it-operations/)

 **弗吉尼亚州雷斯顿([美国商业资讯](https://www.businesswire.com/))—[ScienceLogic](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fsciencelogic.com%2F&esheet=51876405&newsitemid=20181002005555&lan=en-US&anchor=ScienceLogic&index=1&md5=335586901b791e6d78f1f22f4380c245)，IT 运营(AIOps)领域注入情境的人工智能(AI ops)和使用数据为 IT 运营提供可操作见解的领导者，今天宣布分析公司 Enterprise Management Associates(EMA)将 science logic 评为其“ [EMA 人工智能(AI)和机器学习(ML)三大决策指南(用于优化开发运维、IT 运营和业务](http://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fsciencelogic.com%2Fproduct%2Fresources%2Fsciencelogic-receives-ema-top-3-award&esheet=51876405&newsitemid=20181002005555&lan=en-US&anchor=EMA+Top+3+Decision+Guide+for+Artificial+Intelligence+%28AI%29+and+Machine+Learning+%28ML%29+for+Optimizing+DevOps%2C+IT+Operations%2C+and+Business&index=2&md5=5c23b8548c5308966aeaff21cf3a3a0e)”中的顶级供应商这一认可是最终用户研究、市场分析、技术产品审查和专家访谈的结果，旨在为企业客户提供指导，以最大限度地受益于人工智能和人工智能。

EMA 认可了 ScienceLogic 的 SL1 平台，这是一个 AIOps 自动化引擎，旨在跨整个应用堆栈提供全面的业务服务可见性。在当今瞬息万变的技术环境下，开发人员花费大约 25%的时间来调查与混合 IT 基础架构相关的应用程序问题的根本原因。

Enterprise Management Associates 的行业分析师 Torsten Volk 表示:“迄今为止，跨数据中心和云连接、接收、上下文化、规范化、分发和管理全面 IT 运营数据的能力是 AIOps 最重要的成功因素。”。“science logic SL1 因展示了应用 ML 创建对 AIOps 至关重要的培训数据而获得 EMA Top 3 奖。SL1 可以接收来自任何来源的数据，并通过跨应用程序堆栈所有层的高级拓扑图建立上下文，这为及早检测异常、优化预测性维护和确定根本原因提供了必要的可操作性见解，而不会出现典型的“警报风暴”

**EMA 确定的 SL1 的主要优势包括:**

*   对全栈 IT 运营数据的全面、实时洞察
*   持续更新的拓扑图为 AI/ML 驱动的事件噪声抑制和异常检测提供了所需的环境
*   在其单独的应用程序和服务环境中自动发现几乎任何基础架构组件
*   API 可访问的标准化数据湖，用于通过 AI/ML 解决方案进行上下文数据分析或自动更新企业 CMDB

“警报洪水仍然是运营团队面临的最大难题，随着超过三分之一的企业平均使用四家公共云提供商，这一问题只会继续增长。随着服务交付期望的提高，企业以机器速度管理和自动化流程变得更加重要，尤其是当每个流程都在日益复杂的云计算环境中进行时。“EMA 的这一认可证明了业务服务健康的可见性和上下文对于通过利用人工智能和人工智能的力量解决行业中最紧迫的 IT 运营挑战至关重要。”

EMA 的报告为企业影响者和决策者提供了关于他们应该投资的 AI 和 ML 解决方案类型的指导，以优化 DevOps、数据中心运营和业务。它还表彰了 EMA 认为应该纳入产品评估的前三名供应商。
