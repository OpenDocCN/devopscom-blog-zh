# 温室数据和 Zerto 利用混合云的弹性和可移植性快速跟踪数字化转型

> 原文：<https://devops.com/green-house-data-and-zerto-fast-track-digital-transformation-with-hybrid-cloud-resilience-and-portability/>

***协调有助于 IT 经理确保混合 IT 环境中的连续可用性、工作负载移动性和多云灵活性***

2020 年 7 月 1 日东部夏令时上午 10:00

怀俄明州夏延市&波斯顿—([商业资讯](https://www.businesswire.com/))—[领先的数字化转型咨询和托管 IT 服务提供商 Green House Data](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.greenhousedata.com&esheet=52243826&newsitemid=20200701005157&lan=en-US&anchor=Green+House+Data&index=1&md5=e4998c50c5e242299461e4921c2b5452) 今天宣布与 [Zerto](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.zerto.com%2F&esheet=52243826&newsitemid=20200701005157&lan=en-US&anchor=Zerto&index=2&md5=2ac1a75254928dc99040633949bef2d1) 达成战略联盟，以加速围绕混合云弹性和多云应用迁移的[数字化转型](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.greenhousedata.com%2Fdigital-innovation&esheet=52243826&newsitemid=20200701005157&lan=en-US&anchor=digital+transformation&index=3&md5=04a583be3045eaee681b78ddd43dda11)计划。

Zerto 是业界领先的软件解决方案，它用单一平台取代了传统解决方案，以实现跨超大规模云、托管服务和内部数据中心的灾难恢复、数据保护和工作负载移动性。所有这些都降低了现代化和云采用的风险和复杂性。这两家公司共同帮助企业设计、测试、迁移和保护关键应用程序和数据，即使是在复杂的相互依赖的混合环境中。

“Zerto 是我们平台无关的云服务的基础解决方案，”Green House Data 首席信息官 Cortney Thompson 说。“这种合作关系将使我们的员工和解决方案与 Zerto 的专业知识紧密结合，以提供更强大的服务交付和弹性托管平台，从而有效地为我们的客户实现现代 IT 云计算灵活性。”

随着越来越多的 IT 环境跨越本地数据中心、服务提供商合作伙伴以及 Azure 和 AWS 等超大规模云平台，工作负载的可移植性和敏捷性变得至关重要。同时，企业技术面临着 100%连续可用性的期望。

“数字化转型工作的迁移阶段往往会变成棘手的问题，需要复杂的规划和系统测试，尤其是当我们处理面向客户的生产工作负载时，”绿色之家数据部数字化转型高级副总裁 Victor Tingler 表示。“客户对 Zerto 非常满意，我们将其归因于测试环境切换时的易用性，在您点击迁移或故障切换按钮之前，在性能和配置方面获得了对目标环境的高度信任。”

Green House Data 利用 Zerto IT 弹性平台在众多客户项目中促进数字化转型，包括零停机云迁移和具有近实时恢复时间目标(rto)要求的持续业务连续性。

“作为我们最灵活的供应商合作伙伴之一，Green House Data 已经证明了 Zerto 对于各行各业的真正混合云环境的多功能性和价值，”Zerto 的销售、云和联盟总监 Emily Weeks 表示。“凭借在灾难恢复方面的长期专业知识，他们的工程和支持团队在使用 Zerto 进行恢复和迁移方面非常精通。我们期待继续合作，帮助客户应对现代 IT 服务交付的挑战。”
