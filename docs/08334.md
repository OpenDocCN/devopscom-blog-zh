# MediaOps 和 DevOps Institute 获得 5000 英镑荣誉

> 原文:[https://devo PS . com/media ops-and-devo PS-institute-earn-Inc-5000-honors/](https://devops.com/mediaops-and-devops-institute-earn-inc-5000-honors/)

我今天非常自豪地宣布 [MediaOps](https://www.prnewswire.com/news-releases/2021-inc-5000-names-mediaops-one-of-the-fastest-growing-private-companies-in-america-301356431.html) 和 [DevOps Institute](https://devops.com/2021-inc-5000-names-devops-institute-one-of-the-fastest-growing-private-companies-in-america/) 都获得了 Inc .的荣誉，并被列入 Inc. 5000 美国发展最快的公司名单。

对于 MediaOps，这是我们连续第二年获此殊荣。对于 DevOps Institute 来说，这是他们第一年获得这一荣誉。首先，祝贺所有在 MediaOps 和 DevOps Institute 工作并使它们成为伟大公司的优秀员工。没有他们的才华、技能、努力工作和奉献精神，这些公司就不可能取得成功。感谢你们所有人。

其次，作为这两家公司的创始人/联合创始人，我无法告诉你我有多自豪，因为这两家公司不仅取得了成功(如这一荣誉所证明的)，而且在全球疫情中取得了成功。这真的让所有的辛苦工作和漫长的日日夜夜都变得值得。

对于任何一个曾经创业的人来说，你知道你把一定量的鲜血、汗水和泪水投入到你所参与的任何风险投资中。这里没有什么不同！自 2014 年以来，我竭尽全力将 MediaOps 发展成为今天的公司。对于 DevOps Institute，我很幸运有我的合作伙伴 Jayne Groll，DevOps Institute 的首席执行官，带头使 DevOps Institute 成为当今 DevOps 领域领先的成员组织/认证机构。

是的，你希望创办一家公司的金钱回报让一切都值得。但这种认可也是很可喜的。它给你的步伐带来额外的弹力，减轻负重。

这些奖项还凸显了其他一些东西。技术的优势。这两家公司都涉足数字化转型领域，并取得了如此成功，这绝非巧合。数字化转型正在推动世界前进。软件驱动数字化转型，DevOps、网络安全、云驱动软件。

最后，非常荣幸的是，不是一家，而是两家你创办的公司同时获得了这个奖项。我认为这充分说明了这两个组织的未来。这方面的更多消息即将发布，敬请关注来自 MediaOps 和 DevOps Institute 的更多精彩内容！
