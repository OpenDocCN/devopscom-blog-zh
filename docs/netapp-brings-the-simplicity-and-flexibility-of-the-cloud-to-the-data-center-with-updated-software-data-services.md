# NetApp 通过更新的软件数据服务将云的简单性和灵活性带到了数据中心

> 原文：<https://devops.com/netapp-brings-the-simplicity-and-flexibility-of-the-cloud-to-the-data-center-with-updated-software-data-services/>

更新使企业能够跨云和内部环境轻松构建和优化应用，从而加快数字化转型

加利福尼亚州森尼维尔([美国商业资讯](https://www.businesswire.com/))–以云为主导、以数据为中心的全球性软件公司 NetApp(纳斯达克股票代码:NTAP)今天宣布对行业领先的云连接 NetApp ONTAP 数据管理软件进行增强。该公司还宣布了更灵活的 NetApp Keystone Flex 订阅服务和新的 NetApp SolidFire 企业 SDS 解决方案。通过这些更新，NetApp 帮助各地的组织充分利用云。组织现在可以优化性能和安全性，降低成本，轻松地将数据管理从内部扩展到任何云，并使用混合云基础架构即服务。

NetApp 总裁 César Cernuda 表示:“数字化转型已经加速，过去需要数年才能完成的项目现在只需几个月甚至几周就能完成。“凭借丰富的以数据为中心的软件创新，NetApp 在帮助组织快速适应当今的混合云世界并实现可持续转型方面具有独特的优势。现在，我们让他们更容易在云中开发应用程序、将应用程序迁移到云中，或者在内部创建类似云的体验。”

NetApp 今天推出的新特性和功能包括:

*   增强的 **NetApp ONTAP 软件**功能提供了更好的整合、更深入的云集成和连续数据可用性，从而提高了业务关键型企业应用程序的简单性、效率和保护。
*   对 **NetApp Keystone Flex 订阅的更新，**通过“按需购买、渐进扩展”的订阅方式，提供了一条快速、灵活的云数据中心之路，实现云般的本地体验。Keystone Flex 订阅还通过 NetApp 合作伙伴生态系统提供公共云集成。
*   全新的**NetApp****SolidFire Enterprise SDS**解决方案，利用 NetApp 元素软件作为可在任何硬件上部署的独立软件定义的存储，为私有云提供了简单、自动化的基础。
