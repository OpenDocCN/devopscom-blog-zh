# Element 宣布 Element Unify 与 AWS IoT SiteWise 集成，为工业客户提供基于条件的监控

> 原文:[https://devo PS . com/element-announcements-element-unify-integration-with-AWS-IOT-site wise-enable-condition-based-monitoring-for-industrial-customers/](https://devops.com/element-announces-element-unify-integration-with-aws-iot-sitewise-to-enable-condition-based-monitoring-for-industrial-customers/)

*Element Unify 为实时 KPI 和指标提供丰富的工厂数据，帮助工业企业使用 ISO 27001 认证的解决方案做出更好的数据驱动型决策*

**加利福尼亚州三藩市，2021 年 6 月 9 日—** [工业企业 IT/OT 数据管理领域的领先软件提供商 Element](https://www.elementanalytics.com/) 今天宣布了一项新产品，该产品在其 Element Unify 产品和亚马逊网络服务公司(AWS)的托管服务 [AWS IoT SiteWise](https://aws.amazon.com/iot-sitewise/) 之间集成了一个 API，可以轻松收集、存储、组织和监控大规模工业设备的数据。API 集成旨在为客户提供集中工厂数据模型集成和元数据管理的能力，使数据能够被纳入 AWS 服务，包括 AWS 物联网 SiteWise 和亚马逊简单存储服务(S3)工业数据湖。

Element Unify AWS IoT site wise API integration 在 AWS Marketplace 中提供，旨在允许工程师和操作员跨设施监控操作，快速计算性能指标，创建分析工业设备数据的应用程序以防止代价高昂的设备问题，并缩小生产差距。

“我们期待使用 Element Unify 将我们为 OSIsoft PI 等系统构建的内部数据模型连接到 AWS，用于设备数据监控，”Covestro 能力中心数字服务负责人 Philipp Frenzel 表示。

Element 还宣布了其 ISO 27001 认证，证明安全控制可保护客户数据，信息安全管理系统(ISMS)可提供现代 SaaS 应用所需的治理、风险管理和控制。Element Unify 还支持 AWS PrivateLink，为客户提供更高级别的网络安全和控制。

“我们的客户正在寻找能够帮助他们提高设备正常运行时间、避免收入损失、削减 O&M 成本和提高安全性的解决方案，”AWS 能源物联网合作伙伴全球负责人 Prabal Acharyya 表示。“现在，AWS 计划中的工业机器连接(IMC)和 Element Unify 使来自多个系统的数据的实时和资产上下文的无缝 API 集成成为可能，成为 AWS 上的工业数据湖。”

工业客户需要能够进行数字化转型，以最大限度地提高生产率和资产可用性，并降低成本，从而保持竞争力。Element Unify 围绕情境化的工业物联网数据将 IT 和运营技术(OT)结合起来，为工业企业提供关键的企业集成和治理能力。这为他们提供了丰富的洞察力，实现了更智能、更高效的运营。通过将工业运营中传感器生成的先前孤立且关键的时间序列元数据与已建立的 IT 系统(如企业资产管理)相集成，IT 和 OT 团队现在可以轻松地使用其整个工业物联网环境的上下文数据进行合作。

LNS Research 首席分析师 Joe Perino 表示:“Element 与 AWS IoT SiteWise 的合作不仅增强了 AWS 在工业数据湖中共同管理 IT 和 oT 数据的能力，还进一步展示了 Element 为工业公司提供的领先数据运营基础设施解决方案。

通过 API 集成，AWS IoT SiteWise 客户现在可以:

*   将工厂数据模型集成和元数据管理集中到单一上下文服务中，供 AWS 物联网现场使用。

*   将数据直接接收到 AWS 服务中，包括 AWS 物联网 SiteWise 和亚马逊 S3 数据湖。

*   在单站点、多站点和多实例部署中集成来自 IT/OT 源的元数据并将其置于上下文环境中，并将数据模型部署到 AWS 物联网站点。

*   通过感应自动化点火和 PTC KEPServerEX 将元数据从客户系统导入 AWS 物联网站点，以创建上下文数据。

*   随着 Element Unify 模型适应底层数据的变化，保持 greenfield 和 brownfield AWS 物联网现场资产模型和资产层次结构的更新。

*   缩小原始、孤立和杂乱无章的 IT/OT 数据与丰富、情境化和结构化数据之间的差距，这些数据可以与商业智能(BI)、分析工具和条件监控工具(如 AWS IoT SiteWise)轻松配对。

*   使用户能够轻松构建复杂的资产数据模型。

*   在 AWS 物联网站点上构建和部署数据模型，并根据源系统中发生的变化更新数据模型。

*   轻松创建底层数据模型，为 AWS 物联网 SiteWise 和许多其他分析系统提供支持，包括 Amazon SageMaker、Amazon QuickSight 和 Amazon Lookout for Equipment。

“运营数据在工厂和会议室都必须易于使用和理解。Element 首席执行官安迪·贝恩(Andy Bane)表示:“将工厂数据转化为环境数据以用于 AWS 物联网 SiteWise 和其他 AWS 服务的工业组织，将推动更大的运营洞察力，并实现突破性的业务成果。

Element Unify 和 AWS IoT SiteWise 集成将在 [AWS Marketplace](https://aws.amazon.com/marketplace/pp/B08VZY2ZJB) 向 AWS IoT SiteWise 客户提供。

在 AWS 博客文章[这里](https://aws.amazon.com/blogs/industries/centralized-it-ot-data-management-element-unify-aws-industrial-machine-connectivity/)中了解更多关于 Element 和 AWS IoT SiteWise 之间集成的信息。
