# 移动应用和大数据大趋势推动应用性能管理

> 原文:[https://devo PS . com/mobile-apps-big-data-mega trends-drive-application-performance-management/](https://devops.com/mobile-apps-big-data-megatrends-drive-application-performance-management/)

随着大数据、移动性和云的大趋势继续重塑 IT 格局，许多公司正在通过重新调整其解决方案来应对，以便为这些技术提供更好的可见性和支持。一个很好的例子是昨天宣布的 AppDynamics 2014 年春季新版本(完全披露，app dynamics 是 DevOps.com 的赞助商，但这种赞助与编辑内容无关，只是横幅)。

[![Jyoti Final_1](../Images/9957694747d1cb4694051ab2162e7ca3.png)](https://devops.com/wp-content/uploads/2014/04/Jyoti-Final_1.jpg) 昨天，我有幸采访了 AppDynamics 的创始人兼首席执行官 Jyoti Bansal。Jyoti 非常清楚地表明，移动应用程序正在迅速打破他的客户要求支持的应用程序类型的界限。加上增强的 NoSQL 支持和对当今开发中使用的一些最热门语言(Node.js 和 Scala)的支持，在他公司的最新版本中处于领先地位。

Gartner 强调了监控移动应用性能的日益重要性。Gartner 的研究副总裁 Jonah Kowall 在最近的一份报告中指出:“由于移动应用程序端点和日益动态的 Web 技术的重要性日益增加，最终用户体验监控对企业来说将变得更加重要。事实上，它很可能会将其重心从监控单个应用程序的最终用户体验转移到监控和分析最终用户在任何给定端点可用的整个应用程序组合的最终用户行为。”

就我个人而言，我不认为我们中的任何人需要 Gartner 告诉我们，有一个应用程序可以处理任何事情，其中许多都是移动的。移动性正在改变我们许多人的工作方式。Jyoti 说，移动应用程序有望在今年的某个时候取代网络浏览。

如果说有什么比移动设备更重要的话，那就是大数据。似乎大数据无处不在。前几天我看到一个有趣的统计数据，世界上 90%的数据存储时间不到两年。我想这解释了为什么在此之前我们没有听到太多关于大数据的信息。无论如何，大量资金正被投入到大数据解决方案中。证据就是英特尔对 Cloudera 的投资，7 . 4 亿美元，占该公司 18%的股份。当然，Cloudera 是 Hadoop 发行商，AppDynamics 新的大数据支持的一部分是为了增强 Hadoop 功能。

NoSQL 公司也是大数据大趋势的一大受益者，当然还有风险投资。最大的受益者之一是 MongoDB(再次完全公开，DevOps.com 是 MongoDB World 的媒体赞助商)。Matt Asay 曾在 DevOps.com 撰文，他也是营销和业务发展副总裁，他对 AppDynamics 新 NoSQL 支持有如下看法。借助 AppDynamics 和 MongoDB，组织现在可以利用应用性能管理解决方案来进一步了解其基于 MongoDB 的应用。这种合作关系使用户能够获得端到端的可见性，从而在生产中获得最佳性能，这对于正在扩展其 MongoDB 部署的公司来说是一项重要功能。”

准确地说，AppDynamics have 年春季版本中的所有这些新功能实际上至少在几周或几个月前就已经可用了。在我与 Bansal 的谈话中，我问他，如果他没有看到像 AppDynamics 这样的 DevOps 推动者仍然做一些传统的软件发布，而不是某种连续交付，这是否有点讽刺。Bansal 说，这些每季度一次的发布实际上只是将前一季度连续发布的功能打包。这些季度发行版充当了一个检查点，如果你愿意，也可以称为版本控制。实际上，尽管 AppDynamic 的客户已经获得了这一功能。

应用程序性能管理似乎是一个非常热门的领域。这在很大程度上是因为大数据、移动性和云的大趋势。这些解决方案是否会成为更大的[应用开发生命周期管理(ALM)](https://en.wikipedia.org/wiki/Application_lifecycle_management) 市场的一部分(通过收购或合作)还有待观察。
