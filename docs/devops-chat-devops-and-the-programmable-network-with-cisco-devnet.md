> 原文：<https://devops.com/devops-chat-devops-and-the-programmable-network-with-cisco-devnet/>

随着云基础设施、我们创建软件的方式以及 DevOps 的采用，网络技术也经历了自身的转变。SDN、NFV、虚拟网络设备和不断扩展的 API 套件使网络和安全技术更容易为开发者所用。

网络工程师的世界也在迅速变化。从高技能到准水平，所有的网络工程师都面临着建立一个新的和必要的技能:软件开发。学习软件开发可能是一个令人畏惧的、令人生畏的命题，即使对于已经精通网络工程领域的人来说也是如此。

那么，我们如何提升网络工程师群体并帮助他们建立自己的开发能力呢？软件和 DevOps 工程师如何轻松访问所有配置、管理和控制网络的编程接口？

这些问题以及更多问题，我们与思科 DevNet 的创始人、高级副总裁兼首席技术官 Susie Wee 进行了探讨。DevNet 是一个在线资源，为应用程序开发人员、基础设施开发人员以及网络和 DevOps 工程师提供了丰富的信息。在 DevNet 上，你可以找到培训(Python、API 等)、沙箱实验、代码交换(包括 GitHub)、CI/CD 和新的思科软件认证。
