# 网络研讨会:介绍 CloudBees Jenkins 平台——您的 DevOps 和带有 Docker 容器的 CD 的基础

> 原文：<https://devops.com/webinar-introducing-cloudbees-jenkins-platform-your-foundation-for-devops-and-cd-with-docker-containers/>

应用程序的连续交付(CD)正迅速成为这种应用程序经济中的一个独特优势。企业 Jenkins 公司 CloudBees 最近宣布 CloudBees Jenkins 平台的可用性，以帮助组织采用 CD。CloudBees 的这一新企业产品通过新的 Team Edition 为大众带来了 CD，同时通过企业版实现了由 Jenkins 支持的企业 CD 部署。与此同时，Jenkins 开源社区与 CloudBees 联合发布了六个新插件，以支持在 Jenkins 支持的 CD 和 CI 流程中使用 Docker 容器。Docker 是一种强大的新技术，它允许专注于交付软件的团队标准化他们的开发和生产应用程序环境，以降低风险和加速应用程序交付。在 Jenkins 管理的 CD 过程中利用 Docker 容器是更快交付更好软件的最终方法。

加入 CloudBees 的 Tracy，她将讲述 Cloudbees 的新企业产品以及 Jenkins 社区发生的变化，为 Docker 创建 CD 管道提供一流的支持。在本次网络研讨会中，Tracy 将介绍 CloudBees Jenkins 平台最新版本中的功能，以
帮助管理整个组织中的 Jenkins，并与 Jenkins 和 Docker 一起了解六种常见的交付模式。然后，Tracy 将使用 Docker 进行端到端交付管道的演示，将这一切结合在一起。

Jenkins 的持续交付和 Docker 的应用容器化正迅速成为 DevOps 运动的两个基础。查看此网络研讨会，了解如何在您自己的 DevOps 转型中采用这些新工具和技术。
