# Perforce 为微软 Azure 上的游戏开发和 VFX 工作室发布免费的交钥匙解决方案

> 原文：<https://devops.com/perforce-releases-free-turnkey-solution-for-game-dev-and-vfx-studios-on-microsoft-azure/>

*交钥匙云解决方案简化了行业领先开发工具的部署，帮助工作室更快地创新、快速上市，并随着项目的增长而扩展。*

**明尼苏达州明尼阿波利斯2021 年 6 月 17 日—**[Perforce Software](https://c212.net/c/link/?t=0&l=en&o=3199945-1&h=2988379572&u=https%3A%2F%2Fwww.perforce.com%2F%3Futm_source%3Dpr-newswire%26utm_medium%3Dpress-release%26utm_campaign%3Dvcs-global-2021q2-ew-createlikeadesignervisionlikeadev%26utm_content%3Dperforce&a=Perforce+Software)，一家为开发生命周期中需要安全性、可见性和可伸缩性的团队提供解决方案的公司，宣布与微软合作推出[免费增强型 Studio Pack](https://c212.net/c/link/?t=0&l=en&o=3199945-1&h=3578099533&u=https%3A%2F%2Fwww.perforce.com%2Fperforce-and-cloud&a=free+Enhanced+Studio+Pack) ，在 Azure cloud 上以一键启动的模式提供业界最佳的开发工具。

通过在一个简单的预配置环境中提供企业工具——具有支持大文件、大量迭代和远程团队成员所需的基础设施——任何工作室都可以快速启动并运行。这使得工作室快速开发高质量、改变游戏的体验成为可能。

对于希望利用云优势并避免昂贵的硬件投资的工作室，增强工作室包将最大的 AAA 游戏开发工作室信任的世界级开发工具与 95%的财富 500 强公司信任的云提供商 Microsoft Azure 相结合，以实现创造力、协作和速度。为了帮助推动创新，Azure 向在 https://www.perforce.com/perforce-and-cloud[注册增强版 Studio Pack 的前 200 名客户提供 $500 积分。](https://c212.net/c/link/?t=0&l=en&o=3199945-1&h=2624497741&u=https%3A%2F%2Fwww.perforce.com%2Fperforce-and-cloud&a=https%3A%2F%2Fwww.perforce.com%2Fperforce-and-cloud)

“在微软，我们的使命是通过我们的工具和服务，帮助世界各地的游戏开发者获得技术、创意和经济机会。通过为各种规模的游戏工作室提供一流的 DevOps 解决方案，Perforce 是这一旅程中的重要合作伙伴。随着 Perforce Enhanced Studio Pack 现在在 Microsoft Azure 上运行，我们正在扩展我们的解决方案，以最值得信赖和最安全的企业云为后盾，通过久经考验的游戏服务满足游戏创作者的特定开发需求，”微软通用游戏云总经理 James Gwertzman 表示。

点击启动增强型 Studio Pack Azure 部署可用于寻求在竞争日益激烈的市场中加速和获胜的小型创意工作室，或需要快速启动新项目的大型娱乐公司。

为了帮助工作室快速有效地扩展，Perforce 还提供远程管理服务。Helix Remote Admin (HRA)允许工作室专注于他们的工作，同时 Perforce 专家团队优化他们的云部署并防范与停机相关的风险。

Perforce Software 继续扩展其云产品，为团队提供加速上市所需的灵活性，并轻松扩展资源以满足需求，没有任何限制。
