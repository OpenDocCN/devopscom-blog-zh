# 转换可观测性

> 原文:[https://devops.com/transforming-observability/](https://devops.com/transforming-observability/)

前段时间走进一个不熟悉的操作中心，我立即注意到数据库错误警报在主显示器上快速显示，速度比矩阵屏保改变字母还快。奇怪的是，似乎没有人对此特别兴奋。对我来说，这种情况令人头晕目眩，因为我来自“一切都必须平衡”的银行业。

一位高级运营技术人员很快解释说，航空预订系统在速度和容量方面进行了高度优化，几乎牺牲了所有其他东西来应对高峰。“没有烦恼；代理刚刚又按了回车键。重试逻辑只会让我们变慢，”他们告诉我。这里的教训是，消除约束可以深刻地改变一个系统，就像 Eliyahu Goldratt 博士在他的改变游戏规则的约束理论中描述的那样。

如今，云原生架构、多云和混合云平台、动态基础架构即代码(IaC)、DevOps 以及我们存储无限量数据的能力都以自己的方式改变或消除了限制。可观察性可能在这个列表中，但我认为可观察性的转变仍在我们面前发生。

## 可观测性和开放式遥测

通过开源的 OpenTelemetry (OTel ),高度竞争的供应商通过补充和替换曾经高度珍贵的专有代理、接口和数据规格，合作将他们的产品转移到价值链的上游。没有比让这些曾经的专有代理拥有一个将对整个行业产生深远影响的开放解决方案更好的消除限制的例子了。

OTel 是一个蓬勃发展的生态系统，由 11 家科技公司组成，参与各种董事会和委员会，并与 Jaeger、Kubernetes、Prometheus 和 OpenMetrics 等开源项目合作和集成。超过 20 家 observability 公司本身支持并提供 OTel 发行版。OTel 继续走向成熟，将于 2021 年发布稳定性 1.0 版本，并于 2022 年宣布其度量规范路线图。

OTel-native 支持代表了供应商的一项重要承诺，因为这意味着重新构建产品以通过 OpenTelemetry Protocol (OTLP)分发数据，而不是使用 OTel SDKs 和采集器前端内部接口。随着[网络安全管理软件产品 2022 年 10 月的公告](https://investors.solarwinds.com/news/news-details/2022/SolarWinds-Announces-Launch-of-SolarWinds-Observability-to-Provide-Comprehensive-Visibility-in-Hybrid-and-Multi-Cloud-Environments/default.aspx)，我怀疑他们也将加入在商业产品中支持 OTel 的公司行列。T3】

数字化转型、产品和技术领导者看到了可观察性的价值，因为它具有衡量数字化体验和衡量业务和数字化服务绩效的潜力。要做到这一点，需要满足三个重大挑战的可观测性。

首先，可观测性必须有效地跨越微服务、容器、云和传统应用、多个云提供商、数据库源、SaaS 服务、基础设施以及内部和外部 API 的复杂边界。今天的挑战远远超出了集中聚合大量日志数据和抑制不必要的警报。

大多数企业架构看起来与试验板布线项目极其相似，应用程序、系统和数据源相互交叉，代表了跨系统的各种路径和接口。事实上，这些元素中的任何一个都可能导致数字体验的退化，可观察性必须跨越这些元素，无论它们是位于我们严格控制的数据中心，还是分布在微服务、云服务或第三方界面中。

有了这种广度和深度的可见性，我们还需要上下文来匹配和关联看似不相关的信息和来源。开源爱好者 Chris Engelbert [很好地描述了这一挑战](https://www.instana.com/blog/the-role-of-context-and-dependencies-in-observability/):

在提供系统运行状况的确凿证据时，必须考虑数据相关性、基础架构和服务的部署知识，以及应用程序、服务和(最终)硬件的依赖关系树

## 跟随银线

有了上下文和依赖关系，可观察性可以让我们看到软件开发人员所说的“银线”，即收集涉及测量体验或对性能问题进行分类的系统组件的能力。然后我们可以沿着路径，或者说银线，穿过所有的组件，无论它们在哪里，找到约束或者瓶颈。例如，某个特定 API 的低性能可能是由于到达所需数据源所需的位置和遍历，而不是该 API 的微服务或应用程序中的问题。

总之，[可观察性](https://devops.com/?s=observability)对于现代应用和云基础设施的监控、操作和分类非常重要。OpenTelemetry 的采用和成熟将带来许多好处，包括传统边界约束的消除(或至少更大的透明度)。在理解和遵循银色主线的背景下，可观察性超越了运营，可以衡量数字化交付的业务服务和体验。
