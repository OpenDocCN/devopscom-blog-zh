# 如何让供应链恢复(好于)正常

> 原文:[https://devo PS . com/how-to-get-the-supply-chain-back-to-better-normal/](https://devops.com/how-to-get-the-supply-chain-back-to-better-than-normal/)

如果一条链的强度取决于它最薄弱的一环，那么今天的供应链将陷入困境。《纽约时报》最近的一张 [信息图](https://www.nytimes.com/interactive/2021/12/05/business/economy/supply-chain.html) 非常清楚地表明，国际供应链正处于动荡之中，这直接或间接归因于新冠肺炎的爆发。与此同时，供应链上产生的大量数据将被浪费掉。物联网(IoT)无法独自修复供应链，但物联网技术与云、人工智能和自动化相结合，有可能使供应链(和社会)回到类似正常的状态——实际上，比正常状态更好的状态。

事实上，即使在“正常”时期，供应链也因不断降低成本和提高利润，以及优化准时制( [JIT](https://devops.com/devsecops-can-jit-pam-bring-relief/) )库存的交付时间的挑战而捉襟见肘。话虽如此，过去两年尤其严峻，工厂、商店和餐馆因缺乏原料而关门大吉。然后是电脑[芯片短缺](https://devops.com/semiconductor-shortage-may-be-here-to-stay/)，根据白宫简报室文件[](https://www.whitehouse.gov/briefing-room/blog/2021/09/23/when-the-chips-are-down-preventing-and-addressing-supply-chain-disruptions/)称，这无异于“拖累经济”

现在问题是多方面的，所以解决方案不会像增加一个更好的监控机制那么简单。

至少，公司需要 更好地了解和透明他们的供应链。这让他们对整个制造过程有了更多的控制，从原材料的采购到最终产品的交付。物联网、AI/ML、自动化和边缘计算等技术可以通过提供可见性和指标来优化供应链管理，以便公司可以做出实时决策。

例如，在制造业，物联网可以为生产流程带来更大的可见性。来自机器的数字双胞胎的数据-实时分析并反馈给控制系统-可以提高运营和业务效率。当制成品准备装运时，物联网技术可以在货物从制造商流向最终用户的过程中指导货物的物流和运输。除了管理供应，企业还需要更好地预测客户需求，因此这是一个自我维持的反馈循环。

然而，采用这些技术的公司需要意识到，单靠这些技术是行不通的。与任何数字化转型计划一样，人员和流程与技术同样重要。为了缓解甚至克服当今的供应链挑战，企业不仅必须在整个供应链中就通用数据交换格式和 API 达成一致，还必须致力于持续的合作和激励，以实现简化和强化供应链的共同目标。为此，o pen source 对于持续开发通用的全行业标准至关重要，这些标准可让公司的解决方案经得起未来考验。

例如，特斯拉等汽车制造商正在通过使用能够执行多种功能的高端处理器来简化供应链，而不是为每种功能使用几十个专用微控制器，并通过无线方式发布更新。当谈到他们的物联网架构时，组织需要沿着类似的路线思考。开发人员过去一直以一种离散的方式考虑物联网应用，即创建定制框架来为低功耗设备创建应用。相反，开发人员应该将物联网应用程序视为其应用程序组合的扩展——对物联网应用程序使用与云应用程序相同的语言、框架和工具。

### 后疫情供应链问题

随着我们进入疫情的下一阶段，我们可能会很容易相信供应链世界一切都会好起来。然而，除了新冠肺炎，还有几个宏观趋势将影响供应链，包括气候变化、网络安全和全球贸易的动态变化。

例如，为了应对气候变化，企业需要减少第三类排放。 [EPA 将范围 3 排放](https://www.epa.gov/climateleadership/scope-3-inventory-guidance) 定义为非公司自身所有或控制的资产活动的结果，而是由供应链(或价值链)所有或控制的资产活动的结果。范围 3 的排放通常是公司温室气体排放的主要部分。为了达到可持续发展的目标，公司需要更好的供应链资源管理。  

供应链也越来越多地被用作网络攻击的媒介。这既包括物理实体，如建筑运营(如通过第三方 HVAC 供应商 对目标系统的 [)也包括软件供应链(如](https://www.npr.org/sections/thetwo-way/2014/02/05/272101928/u-s-hvac-firm-reportedly-linked-to-target-s-data-security-breach) [【网络安全管理软件产品】](https://www.wired.com/story/solarwinds-hack-supply-chain-threats-improvements/) )。其实，一个组织受到 攻击，已经不是 的问题，而是 当 的问题。

乌克兰战争放大了我们高度关联的全球经济的复杂性。美国政府及其盟友对俄罗斯实施了严厉制裁，此举将产生广泛影响。[纽约时报报道](https://www.nytimes.com/2022/01/29/us/politics/russia-sanctions-economy.html) “美国官员承诺的回应可能会扰乱主要经济体，尤其是欧洲的经济体，甚至威胁到全球金融体系的稳定。”如果俄罗斯做出回应，比如切断制造半导体所需的 [材料的运输，或者发动新的网络攻击，那么全球供应链也是如此。](https://venturebeat.com/2022/02/24/ukraine-supplies-90-percent-of-us-semiconductor-grade-neon-what-it-means-to-chip-supply-chain/)

### 结论

新冠肺炎，网络攻击、贸易壁垒、战争……这些因素都没有导致我们现在看到的物理和技术供应链的问题；相反，它们暴露了现有的弱点，让全世界都可以看到，也可以体验到。组织必须共同努力，通过使用物联网、AI/ML 和 edge 等关键技术，以及致力于开放、基于标准的解决方案，来提高灵活性和安全性。
