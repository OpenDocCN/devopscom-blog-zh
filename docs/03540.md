# ScienceLogic 推出行业首个注入上下文的 AIOps 平台

> 原文：<https://devops.com/sciencelogic-launches-industry-first-context-infused-aiops-platform/>

*新科学逻辑 SL1 使运营以发展的速度前进，因此企业可以以机遇的速度前进*

*   *SL1 平衡了开发人员创造服务的速度与运营人员部署和维护服务的速度*
*   *在 AI/ML 的帮助下，两项强大的新技术为运营数据带来实时环境，并以机器速度扩展自动化*
*   *创新使企业能够通过将应用程序连接到底层基础设施来实现端到端服务视图，从而实时管理业务服务状况*

**弗吉尼亚州雷斯顿——2018 年 5 月 1 日—**science logic，一家为 IT 运营提供 IT 运营数据的可操作洞察的领先公司，今天宣布推出其全新的行业定义 AIOps 平台 [ScienceLogic SL1](http://www.sciencelogic.com/SL1) 。

IT 运营是寻找弹性数字体验的最后一个重要领域。对于现代运营而言，在一个越来越瞬息万变的世界中保持步伐超出了人类的认知，在这个世界中，应用程序和基础架构工作负载跨越多云架构并快速转变技术环境。为了蓬勃发展，运营部必须重新装备自己，准备好支持对服务敏捷性、新业务线的引入和持续并购的需求

**进入 ScienceLogic SL1** ，它的设计和开发目标是在开发人员创造*卓越*数字体验的速度和运营人员创造*弹性体验的能力*之间建立公平的竞争环境。

“十年前，我们看到企业在速度和敏捷性的推动下，正在进入一种短暂的运营状态。它始于基础设施层，但迅速向强调服务开发和快速发布的 DevOps 运动发展。ScienceLogic 首席执行官 Dave Link 表示:**当今的绩效管理系统必须通过深入了解应用程序及其底层基础设施之间的依赖关系，提供实时的运营洞察。**

SL1 通过导出拓扑图了解任务关键型应用程序如何连接到底层基础架构，从而实现实时服务运行状况视图，以提供信息、进行分析和采取行动。这种环境使企业能够跨越各种数据孤岛带来意义，并产生洞察力，以前所未有的方式推动自动化操作。

IBM Services CTO 兼技术、创新&自动化副总裁 Bridget Karlin 表示:“随着 IBM 通过 Watson 继续发展我们围绕 IBM 服务平台的生态系统，我们与其他领先公司合作至关重要，这些公司可以帮助提供增强人类智能的解决方案，以帮助改善服务提供商的运营，并解放专业人员，使他们专注于推动更高价值的结果**。“通过与 ScienceLogic 合作，利用人工智能来帮助企业在业务服务受到影响之前预测性地确保 IT 成功，我们可以更好地帮助我们的客户创造有弹性的数字体验。”**

**科学逻辑 SL1 核心能力**

*   **参见**–自动化实时发现，包括跨 IT 孤岛和多云环境的应用和基础架构
*   **上下文化**–自动化拓扑图在不同数据集之间建立实时关系，为数据带来上下文
*   **行动**–跨多种技术(包括 CMDB(配置管理数据库)、DevOps 和 APM(应用性能管理))自动发现问题并进行后续补救

**SL1 工作中的创新技术**

*   **PowerMap**–如果说上下文是 AIOps 的王，那么 power map 就是提供上下文的工具。PowerMap 跨所有技术层实时创建所有组件之间依赖关系的多维拓扑图，提供底层原始数据的即时上下文。这成为机器学习引擎提供可操作的见解和自动化的关键输入源。
*   **PowerSync**–如果自动化是 AIOps 的终极目标，power sync 就是为其提供动力的引擎。通过提供一个通用的通信桥梁来获取数据、共享数据并保持数据同步，PowerSync 打破了目前限制自动化的数据孤岛。它在任意数量的平台之间异步并发地提供多方向的智能数据转换。PowerSync 以一个强大的自动化引擎来实现对环境的改变，从而完成了这个故事。
