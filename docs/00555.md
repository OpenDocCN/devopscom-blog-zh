# 物联网互联

> 原文:[https://devops.com/interconnecting-the-internet-of-things/](https://devops.com/interconnecting-the-internet-of-things/)

在所谓的物联网中，每个方向都有碎片。有许多编程语言、设备以及最近出现的一系列不同的物联网开发和管理平台都在争夺市场和开发者的注意力。他们希望在未来的岁月里成为物联网的标准。

广泛采用物联网的最大障碍之一是难以吸引开发人员使用大多数人使用的低级编程语言。RedMonk 高级分析师 Donnie Berkholz 表示:“长期以来，物联网一直很难使用，因为很多东西都需要使用 Arduino 工具集，例如，大量的 C++。“如今，许多开发人员对用低级语言工作不再感兴趣了，”他说。

进入 IBM 物联网基础的以太网物联网初学者工具包。芯片制造商 ARM、Freescale 和 IBM 的套件旨在开发工业物联网应用，并基于 ARM 的 mbed 操作系统、Freescale 设计的开发板和 IBM 的云服务 BlueMix。

据 ARMmbed 博客称，IBM 物联网基金会的入门套件提供了流畅的用户体验，并使得在几分钟内将数据从电路板传感器传输到 IBM BlueMix 云成为可能。“它特别适合没有嵌入式或 web 开发经验的开发人员，因为它提供了一个学习新概念和创建工作原型的平台，”该公告称。

“在最初的开箱即用体验之后，可以通过 IBM 的生产级 BlueMix 平台探索云应用的无限可能性，在该平台上，一台设备的部署和设备管理就像一百万台设备一样简单。根据公告，可以修改和扩展入门套件硬件，以探索设备设计空间，并使用 mbed SDK 和 HDK 将最终设计投入生产。

在本周的 IBM InterConnect 2015 大会上，还有更多物联网产品发布。企业云提供商 CloudOne 宣布推出他们的 [CloudOne 物联网(IoT)实验室](http://www.prweb.com/releases/2015/02/prweb12538509.htm)。CloudOne 物联网实验室希望成为一个实时协作的平台，以构建物联网解决方案。根据 CloudOne 的说法，它将很快在伊利诺伊州芝加哥建立一个技术中心，容纳公司的技术团队，并为客户提供一个空间，以获得 CloudOne 平台的实践经验，包括 IBM 连续工程、资产管理和分析技术。

“CloudOne 物联网实验室让我们的团队能够与客户实时讨论物联网解决方案。CloudOne 首席执行官 John McDonald 在一份声明中表示:“这将加快互联产品的交付，同时拥抱和扩展客户在合作伙伴技术、数据和流程方面的现有投资。

同样在本周的 InterConnect 2015 上，stream network PubNub [宣布，它将与 IBM](http://www.hostreview.com/news/150223-pubnub-delivers-secure-data-stream-network-on-ibms-bluemix-paas#ixzz3SivbwEch) 合作，为 BlueMix 带来实时数据流和消息传递功能。

PubNub 在会议上展示了它所谓的实时安全设备连接，由它易于使用的数据流 API 提供支持。根据 PubNub 的说法，IBM BlueMix 客户将能够利用 PubNub 安全可靠地交付大规模实时物联网、网络和移动解决方案。

最近几周，物联网领域出现了许多动向。本月早些时候，Raspberry Pi 2 发布，操作系统由微软和 Canonical 共同提供。Pi 2 增强的处理能力意味着可以在从机器人到无人机的更多智能设备中使用。

最近的许多公告都希望让更多的开发者更容易获得设备、数据和开发工具。如果物联网要尽快发挥其全部潜力，所有这些都是必不可少的。
