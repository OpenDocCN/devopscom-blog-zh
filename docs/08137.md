# Nylas 筹集了 1.2 亿美元用于提高工作自动化的生产率

> 原文：<https://devops.com/nylas-raises-120-million-to-accelerate-productivity-apis-for-work-automation/>

*由 Tiger Global 牵头，精英企业家投资，随着企业继续优先考虑 API 驱动的数字化转型，新的融资将强调开发者体验和创新*

**加利福尼亚州三藩市，2021 年 6 月 17 日—** [面向开发者和企业的现代电子邮件、日程安排和工作自动化工具的通信 API 平台 Nylas](https://www.nylas.com/?utm_source=PR&utm_medium=display) 今天宣布，它已经筹集了 1.2 亿美元的 C 轮资金。这项投资使 Nylas 的总资金达到 1.75 亿美元，进一步加速了 Nylas 和 API 经济的增长。

本轮由 [Tiger Global Management](https://www.tigerglobal.com/) 牵头，现有投资者[花旗风险投资](https://www.citi.com/ventures/)、 [Slack Fund](https://slack.com/fund) 、 [8VC](https://8vc.com/) 、 [Round13 Capital](https://round13.com/) 、 [Scale Asia Ventures](https://www.sav.vc/) 等参与，新投资者包括 [Owl Rock Capital、Blue Owl 的一个部门](https://www.blueowl.com/)、 [Blue Cloud Ventures](https://www.bluecloudventures.com/) 以及来自[帕特里克·科利森](https://www.linkedin.com/in/patrickcollison/)的个人捐款 还有他的兄弟[约翰·科利森](https://www.linkedin.com/in/johnbcollison/)，Stripe 的总裁兼联合创始人[，Klarna 的首席执行官](https://www.linkedin.com/in/sebastian-siemiatkowski-768977/)塞巴斯蒂安·西米亚考斯基[，Nest 的创始人](https://www.linkedin.com/in/tonyfadell/) [Future Shape 的负责人](https://www.futureshapellc.com/)。 老虎环球的合伙人约翰·柯蒂斯将加入 Nylas 董事会。

“向数字通信和连接的根本转变使得各行各业的公司越来越依赖开发人员来解决关键的业务挑战并构建引人入胜的产品体验。Nylas 联合创始人兼首席执行官格雷布·波利亚科夫说:“API 已经成为现代软件开发和数字化转型的核心。“通过我们强大的 API 套件，我们为开发人员提供了更快满足客户和市场需求所需的工具，通过强大的定制客户体验创造了竞争优势，并通过更高效、安全和智能自动化的开发周期节省了大量运营成本。我们很高兴能够推进我们的使命，让世界变得更有生产力，并且很荣幸能够得到杰出的世界级投资者和企业家的支持。”

Nylas 将利用这笔资金进一步投资和扩展其集成的、生产力驱动的开发人员体验，并通过扩展其上市团队、产品开发、工程和客户支持组织来支持使用 Nylas 的开发人员和公司的涌入。Nylas 还将寻求加快对颠覆性技术的持续投资，这些技术可以自动化工作流，并无缝集成对话式 AI/ML、情感分析、自然语言处理(NLP)、光学字符识别(OCR)和跨系统智能自动化等技术。

Tiger Global 和 Nylas 董事会成员 John Curtius 表示:“公司正在迅速采用 API 作为自动化生产的方式，并找到新的创新方法来支持现代工作和协作。“这一趋势对于创建无摩擦、有意义的数据驱动型通信至关重要，它将推动数字化转型。我们相信 Nylas 在引领 API 经济的未来方面具有独特的优势。”

此次融资紧随 Nylas 的显著增长而来，在过去 12 个月中，Nylas 的新收入增长了 3 倍以上，员工人数增加了一倍，同时还进一步扩展到 salestech 和 martech、医疗保健、金融服务、房地产、汽车技术、人力资源和招聘等行业。
