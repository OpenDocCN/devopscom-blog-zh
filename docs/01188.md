# 威胁堆栈云安全平台获得 AWS 安全认证

> 原文:[https://devo PS . com/threat-stack-cloud-security-platform-achieves-AWS-security-competency/](https://devops.com/threat-stack-cloud-security-platform-achieves-aws-security-competency/)

*威胁堆栈获得全面安全和无缝客户体验的能力*

**BOSTON—2016 年 4 月 7 日**—[云安全管理和合规领域的领导者 Threat Stack](http://www.threatstack.com/) 今天宣布，作为威胁堆栈云安全平台 ^(TM) (CSP)的 AWS 能力计划的一部分，该公司已获得亚马逊网络服务(AWS)安全能力认证。

Agari 工程总监 Chris Haag 表示:“安全始于可见性，威胁堆栈为我们的系统和云基础设施提供了无与伦比的可见性。“威胁堆栈云安全平台提供智能异常检测和高度灵活的规则引擎来检测可疑行为，如权限提升、数据泄露和异常网络活动。”

AWS 能力计划旨在突出 AWS 合作伙伴网络(APN)的成员，他们在专业解决方案领域(如安全)表现出熟练的技术并证明了客户的成功。威胁堆栈云安全平台**基于在 AWS 上构建产品和解决方案以在配置和漏洞分析领域支持客户的成功经验，获得了安全能力认证。**

**Threat Stack 董事长兼首席执行官 Brian Ahern 表示:“我们坚信分层的云安全方法是最好的，这使得 Threat Stack CSP 成为 AWS 的重要补充。“威胁堆栈提供了识别内部威胁和入侵、防止数据丢失以及在 AWS 上构建和运行时满足合规性法规所需的深度可见性。获得 AWS 安全能力合作伙伴认证表明我们致力于保护客户的工作负载和更广泛的环境免受威胁。"**

**威胁堆栈 CSP 是业界领先的完全集成的云安全套件。云原生且独立于平台的威胁堆栈 CSP 显著降低了识别威胁、保护数据和自动化合规性要求所需的费用和复杂性。威胁堆栈 CSP 提供即时可见性和深度关联上下文，使公司能够高效运营、防止数据丢失以及识别和验证威胁。**
