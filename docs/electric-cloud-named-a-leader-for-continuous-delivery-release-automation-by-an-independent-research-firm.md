# 电云被一家独立研究公司评为连续交付发布自动化的“领导者”

> 原文：<https://devops.com/electric-cloud-named-a-leader-for-continuous-delivery-release-automation-by-an-independent-research-firm/>

*公司在当前的产品和战略中获得最高分，因保持对支持 DevOps 社区的坚定承诺以及在将高级分析应用于持续交付方面领先于其他被评估的供应商而得到认可*

加利福尼亚州圣何塞，2018 年 11 月 26 日——自适应发布协调和持续交付领域的领导者 Electric Cloud([https://electric-cloud.com/](https://electric-cloud.com/))今天宣布，它被 Forrester Research Inc .评为 2018 年第四季度 Forrester Wave:持续交付和发布自动化的领导者。在其 CDRA Wave 中，Forrester 根据公司的“当前产品”、“战略”和“市场存在”的实力对公司进行了评估，Electric Cloud 在“当前产品”和“战略”方面获得了最高分。

**在此下载报告全文:****[福瑞斯特-波-连续-交付-发布-自动化/](https://electric-cloud.com/resources/whitepapers/forrester-wave-continuous-delivery-release-automation/)**

该报告旨在帮助基础设施和运营(I&O)专业人员在为其开发和运营(DevOps)自动化寻找 CDRA 解决方案时做出正确的选择。在该报告中，作者 Charles Betz(服务于基础设施和运营专业人员的首席分析师)领导了对 15 家供应商的评估，并对他们进行了研究、分析和评分。

Forrester 报告指出:“电云提供了从 RA 市场涌现的最深入的 CDRA 产品之一，并一直坚定地致力于支持 DevOps 社区。”。“Electric Cloud 跨各种当前和传统技术提供发布自动化，并通过其 ElectricFlow 产品提供更高级别的发布管理和端到端数字价值流可见性。它还有自己的构建管理器，适用于在数字管道中寻找单一供应商的组织。在将高级分析应用于持续交付(如风险评分)方面，电气云领先于其他 CDRA 产品。”

Electric Cloud 首席执行官 Carmine (CJ) Napolitano 表示:“我们相信，在 Forrester 的报告中被评为领导者，并在‘战略’和‘当前产品’类别中获得最高分，突显了 Electric Cloud 对市场领先创新的承诺。“ElectricFlow 为复杂的企业管道和发布提供了改变游戏规则的控制和可见性，涵盖基于单片、大型机和微服务的应用的任何组合。随着我们的客户继续在 DevOps 上下大赌注，我们帮助他们安全地适应并利用正在进行的大规模和持续的技术变革。我们每天都在努力工作，让我们的客户能够将软件发布从繁琐的工作转变为真正的竞争优势。”

正如报告中提到的，电云也非常重视支持 DevOps 社区。“DevOps 不是装在盒子里的，它需要熟练的人来让它工作，”纳波利塔诺继续说。“电云通过其 [#c9d9 教育视频播客](https://electric-cloud.com/c9d9)系列以及与 Gene Kim 和 it 革命共同创办 [DevOps 企业峰会](http://devopsenterprise.io/)，在支持和教育 DevOps 社区方面有着悠久的历史。”

基于这种对教育和社区的关注，Electric Cloud 在今年早些时候与 DevOps Institute 合作推出了一项独特的培训和认证计划，名为“ [Electric Cloud University](https://electric-cloud.com/products/ecu/) ”，旨在为 IT 和 DevOps 专业人员提供他们所需的技能和信心，以帮助推动数字化转型，而不是被他们压倒。

此外，电云还是即将发布的“2019 提升技能:企业开发技能报告”的创始白金赞助商，以帮助我们的社区确定和解决全球 IT 社区的关键技能差距。Forrester 前副总裁兼研究总监 Eveline Oehrlich 是该报告的首席研究分析师。
