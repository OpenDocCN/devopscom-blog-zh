# EP 20: DevOps 是低代码的朋友还是敌人？

> 原文:[https://devops.com/ep-20-devops-for-low-code-friend-or-foe/](https://devops.com/ep-20-devops-for-low-code-friend-or-foe/)

自从 2020 年新冠肺炎疫情开始以来，低代码开发变得越来越流行。低代码技术允许组织使用很少或不使用代码来构建和定制完全响应、易于部署的应用程序。它加速了数字化转型，增强了公民开发者的能力。在这一集的[devo PS unbounded](https://devopsunbound.com)中，主持人 Alan Shimel 和 Mitch Ashley 与 Derya Sousa (Kianda)、Andrew Manby (HCL)和 Brian van den Brink (Valori)一起讨论低代码的好处以及它如何改变软件开发。下面是视频，下面是对话的文字记录。
