# 复制加速了基于 Kubernetes 的应用程序的采用

> 原文:[https://devo PS . com/replicated-accelerated-adoption-of-kubernetes-based-applications/](https://devops.com/replicated-accelerates-adoption-of-kubernetes-based-applications/)

*新功能改善了软件供应商向企业客户交付 Kubernetes 应用的方式*

**加利福尼亚州三藩市，2022 年 5 月 17 日—**[复制](http://www.replicated.com/) ，使软件供应商能够在客户控制的环境中运营和扩展其应用的交付和管理，今天宣布了一系列新的平台特性和功能，使其客户能够加快其 Kubernetes 应用的企业采用。

“确保我们的客户能够提高采用率，运营他们的应用分发流程，并有效地扩展他们的上市战略，这是我们不断努力实现的目标，”Replicated 首席执行官兼联合创始人格兰特·米勒说。“当我们的客户取得成功时，我们也取得了成功，因此我们非常高兴能够推出大量新功能来帮助独立软件供应商简单高效地交付产品。”

**新发布的关键特性和更新加快了企业采用基于 Kubernetes 的应用程序的速度，并推动供应商取得成功:**

*   **社区许可**
    *   企业软件供应商努力将他们软件的试用版发布到自托管环境中。Replicated 的社区许可功能允许软件供应商为客户控制的实例分发无限制的试用许可证。随着企业从试用转向生产，软件供应商可以通过将社区许可证转换为商业许可证来提高采用率。
*   **飞行前检查**
    *   安装故障可能会成为企业销售的障碍。预检检查将验证容器环境、群集资源和配置，在安装问题出现之前将其标记出来，从而大大提高开发运维效率。新的可定制的主机和应用程序级预检提高了首次安装和升级的成功率。

*   **语义版本化和自动更新**
    *   企业软件供应商花费宝贵的资源为客户解决他们的应用程序的旧版本不受支持的问题。有了语义版本控制，供应商可以让所有客户控制的软件保持最新。供应商现在可以轻松地管理多个渠道，支持自动更新，并允许客户选择他们想要安装的版本，并在分步升级期间要求特定的主要版本。

重复的客户已经看到了这些新功能的价值。 [Wallaroo](https://www.wallaroo.ai/) ，一个构建来提供简单、安全和可扩展的方式在企业生产环境中部署、运行和观察机器学习的平台，传达了他们对 Replicated 的新功能的满意。“由于 Replicated 的社区许可，我们找到了一种很好的方法来加快我们对 [Wallaroo 社区版](https://www.wallaroo.ai/blog/wallaroo-community-edition-is-here) 的采用率，”Wallaroo 产品负责人 Younes Amar 说。“对于希望简化 ML 模型和管道的部署和管理的机器学习和数据科学社区中的任何人，我们增加了我们的可信度和透明度。”
