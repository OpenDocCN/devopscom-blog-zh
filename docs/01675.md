# 网络研讨会:用 Docker 和 Kubernetes 扩展 Jenkins

> 原文:[https://devo PS . com/webinar-scaling-Jenkins-with-docker-and-kubernetes/](https://devops.com/webinar-scaling-jenkins-with-docker-and-kubernetes/)

Docker 正在彻底改变人们思考应用程序和部署的方式。它为从轻量级虚拟机到复杂的分布式微服务架构的各种用例提供了一种运行和分发 Linux 容器的简单方法。Kubernetes 是一个开源项目，将 Linux 容器集群作为一个系统来管理，在多个 Docker 主机上管理和运行 Docker 容器，提供容器的协同定位、服务发现和复制控制。它由 Google 发起，现在得到了微软、RedHat、IBM 和 Docker Inc 等公司的支持。Jenkins 持续集成环境可以通过使用 Kubernetes 和 Docker 插件来动态扩展，使用容器来运行从设备和作业，还可以隔离作业执行。
