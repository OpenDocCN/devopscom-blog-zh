# 波音加入云计算原生计算基金会成为白金会员

> 原文：<https://devops.com/boeing-joins-cloud-native-computing-foundation-as-a-platinum-member/>

*全球领先的航空航天公司大举投资云原生生态系统*

**西班牙瓦伦西亚，KubeCon + CloudNativeCon 欧洲——2022 年 5 月 18 日——**[为云原生软件构建可持续生态系统的云原生计算基金会](https://www.cncf.io/) (CNCF)，今天宣布波音公司作为白金会员加入该基金会。作为白金会员的一部分，该公司还将在 CNCF 管理委员会中拥有一名代表。

[波音](https://www.boeing.com/) ，全球最大的航空航天制造商之一，拥有超过一个世纪的航空航天领导和创新传统。该公司已经建立了一个巨大的软件足迹和数千名软件工程师的全球社区，他们对计算、学习和扩展技术边界有着共同的热情。随着波音公司投资加速数字革命，它正在 寻找机会培养和贡献以及创建基于开源技术的新开源项目。

“在波音公司，我们正在努力利用公共云部署的灵活性和巨大的可扩展性来构建极其强大的航空航天产品，并向我们的客户提供快速迭代和创新的软件解决方案，”波音公司软件工程副总裁 Jinnah Hosein 说。“波音很高兴加入 CNCF，成为生态系统的一部分。我期待着这种合作关系，我们将一起通过极其强大的软件部署来改变世界。”

波音公司致力于增加其在开源项目中的参与，并于最近成立了首个开源项目办公室(OSPO)。作为一家从海底到太空保护、连接和探索我们世界的公司，波音致力于学习、知识共享，并为其全球团队成员提供成长和繁荣的机会。

“波音是一家经久不衰的全球性公司，影响着全世界人民的生活。该公司拥有丰富的经验和资源，将极大地造福于 CNCF 社区，”云原生计算基金会执行董事 Priyanka Sharma 说。“我们期待与他们合作d 欢迎瑞卡多·托瑞斯加入 Go verning Board。”

**关于云原生计算基础**

云原生计算使组织能够在公共、私有和混合云中使用开源软件堆栈构建和运行可扩展的应用。云本地计算基金会(CNCF)托管全球技术基础设施的关键组件，包括 Kubernetes、Prometheus 和 Envoy。CNCF 汇集了业界顶尖的开发者、最终用户和供应商，并举办世界上最大的开源开发者大会。由 500 多个成员支持，包括世界上最大的云计算和软件公司，以及 200 多个创新创业公司，CNCF 是非营利 Linux 基金会的一部分。欲了解更多信息，请访问 www.cncf.io.
