# InfluxData 视频系列:Chris Churilo 的时间序列数据

> 原文：<https://devops.com/influx-video-series-time-series-data-with-chris-churilo/>

时间序列数据可以在许多方面为组织带来价值，而不仅仅是帮助他们了解基础设施性能的指标。它还有助于阐明可能影响性能的其他问题，如客户使用情况。

在这个[视频](https://webinars.devops.com/time-series-data-more-than-just-metrics)(四个系列中的第一个)中，InfluxData 的产品营销总监 Chris Churilo 讲述了如何使用时间序列数据来解决其客户之一的电池耗尽问题，该客户是一家在非洲拥有最终用户客户的太阳能电池板提供商。

[https://www.youtube.com/embed/pUKRqMxhtFc](https://www.youtube.com/embed/pUKRqMxhtFc)

“这是一种很好的方式，可以查看指标，实际上可以让他们了解人们是如何使用它的，他们可以扩大对客户的服务，最终让客户对他们的服务更加满意，”她说。
