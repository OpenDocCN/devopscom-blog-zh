# 嵌入式分析如何改变未来的业务

> 原文:[https://devo PS . com/how-embedded-analytics-can-transform-future-business/](https://devops.com/how-embedded-analytics-can-transform-future-businesses/)

众所周知，谷歌、亚马逊和网飞等大型科技公司不仅利用数据，还将嵌入式分析和[洞察](https://devops.com/data-mapping-and-gdpr-how-are-they-related/)作为其软件、服务、应用和工具包的标准功能。这些分析以推荐、使用习惯和其他有用信息的形式呈现给消费者。这种见解不仅有助于消费者从他们使用的产品中获取最大价值，也有助于提供商了解他们如何提供更好的服务。嵌入分析以提供差异化用户体验的企业也会对[参与度和收入](https://www.thoughtspot.com/press-releases/new-research-finds-companies-that-embed-analytics-with-a-differentiated-user-experience-increase-engagement-and-revenue)产生直接影响。

您是否考虑过嵌入式分析如何区分贵公司自己的软件和硬件产品？让我们了解嵌入式分析如何帮助技术企业转型:

### 1.在你的产品中，分析是被期待和欢迎的

通过将分析嵌入到现有的企业软件和应用中，用户可以深入了解实时和环境感知数据(例如:一个位置在一天中的不同时间有多忙)，并可以受益于更快的数据驱动型决策，而无需担心应用兼容性。通常，软件应用程序提供应用内分析来取悦和服务他们的客户，而硬件公司可以在门户网站或应用程序上提供有用的统计数据。

### 2.如今，嵌入式分析比以往任何时候都更容易

云技术已经彻底颠覆了以技术为主导的行业。软件、应用和硬件比以往任何时候都更容易连接到数据存储库来保存日志，这些日志是丰富的分析来源。S3 桶和对象存储的发明使得保存日志数据变得相当便宜。最后，向平台即服务(PaaS)和软件即服务(SaaS)的过渡从根本上增加了一个跟踪消费模式和资源利用的“简单按钮”。

### 3.嵌入式分析有助于设计更好的产品和服务

嵌入式分析可以让企业更深入地了解应用遥测(客户如何使用数据，他们如何参与产品)，并从可用性的角度洞察他们所面临的挑战。提供商可以使用高级分析快速迭代、绘制模式和进行预测、解决客户挑战、发现新的盈利途径、检测和缓解安全威胁，或者创建全新的功能和应用，让客户更快地获得更好的应用体验和更好的决策。

### 4.嵌入式分析提高了运营效率

科技行业正在从应用程序自包含的传统整体架构过渡到微服务架构，在微服务架构中，不同的组件通过 API 调用按需启动。通过将分析直接嵌入到开发工具中，企业可以在不牺牲可用性或性能的情况下管理资源。公司可以动态地分配和优化资源，了解消费的高峰和低谷。企业必须能够利用历史和实时数据来构建预测模型，以便在客户遇到问题之前主动识别和缓解应用程序各个组件的问题。

## 在您的应用中嵌入分析

在您开始在应用中嵌入分析之前，选择正确的数据库基础非常重要，因为这将支持当前的需求和未来的目标。以下是应该成为分析数据库标准特性的顶级功能:

**1。处理海量数据:**数据库必须使用大规模并行处理，以最小的成本轻松高效地扩展。今天的规模可能是 TB，明天可能是 Pb，如此等等。

**2。支持快速数据分析:**速度是积极用户体验的关键因素。用户想要即时洞察，因为他们不喜欢等待结果。数据库应该能够优化负载和查询性能，以便分析既及时又相关。

**3。嵌入机器学习:**机器学习算法对于预测分析至关重要。一个可以嵌入 ML 并从新数据中迭代学习的平台可以带来很大的不同，并增加一个全新的智能水平。

**4。特性原生集成:**开发人员花在解决工具和集成上的时间越多，花在创新上的时间就越少。理想情况下，该平台必须与流行的应用程序、数据源(内部部署、云、多云或混合)、ETL 和可视化工具(例如，Microstrategy、Looker、Power BI、Qlik、Tableau)进行现成的集成。它应该足够灵活，能够为没有本机集成的东西支持用户定义的函数。

**5。鼓励自助服务:**数据库应该让数据科学家能够轻松利用他们已经熟悉的工具和编程语言，如 R、Java、C/C++、Python 和 SQL。数据科学家还会喜欢一个具有内置功能的平台，用于广泛的分析(如地理空间、时间序列、事件序列、实时、机器学习、文本分析、模式匹配和回归)，并提供工具来发现趋势、发现异常和预测意外。

**6。支持高度高级的分析:**许多平台可以分析单个表或进行简单的查找，但很少有平台能够分析几年来来自众多来源的数十个表、数百种数据类型、维度和属性。

**7。统一:**数据库应该能够简化从数据接收到勘探到生产直至交付的所有工作。用户应该能够在其位置分析数据，以始终有一个真实的来源。

不难想象，未来最具颠覆性的公司将是那些将数据作为核心资产并在其产品和服务中嵌入分析的公司。嵌入分析需要正确的数据库基础；为了找到正确的基础，IT 团队需要避开噪音，研究细节。有了合适的数据库，未来由你来创造。
