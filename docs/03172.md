# Redgate 数据屏蔽器全球企业采用数据屏蔽技术的趋势的一部分

> 原文：<https://devops.com/redgate-data-masker-part-of-adoption-trend-toward-global-enterprise-use-of-data-masking-techniques/>

英国剑桥，2018 年 9 月 5 日，Redgate 认为，Gartner 最新且及时的[数据屏蔽市场指南](https://www.gartner.com/doc/3883287)提供了关于公司日益增长的数据屏蔽需求和采用的宝贵见解，以应对威胁和合规性要求。它建议公司“审查您的数据安全和合规路线图。关注那些提供不止一种数据挖掘技术并支持跨数据挖掘场景、应用和平台的一致策略的供应商。” [Redgate](https://www.red-gate.com/products/dba/data-masker/) 被列为数据屏蔽技术的代表厂商。

该市场指南于 2018 年 7 月 20 日发布，预计到 2021 年，全球企业对数据屏蔽或类似去识别技术的使用将从 2017 年的 15%增加到 40%。

Redgate 认为，受日益增多的隐私法规、数据增长以及从数据中提取更多价值的分析应用不断扩大的驱动，数据屏蔽正成为公司满足合规性要求以及应对内部和外部威胁的一种重要方式。

需要数据屏蔽，因为许多公司在其开发、测试和业务分析环境中使用其生产数据库的副本。然而，这些数据库中的敏感数据必须通过用虚构数据替换来保护。该数据必须是真实的，真正代表原始数据，以保持其引用完整性和分布特征。

使用随机数据是行不通的，这正是 Redgate Data Masker 等第三方工具的用武之地。它们提供了多种方法来整理 SQL Server 和 Oracle 数据库中的数据，使用模板来保持与现有数据格式的一致，并包括替换数据集，如姓名、邮政编码和电子邮件地址。

微软数据平台 MVP 史蒂夫·琼斯对此并不感到意外。“大企业，尤其是金融行业的大企业，长期以来一直在使用定制的数据屏蔽解决方案，但成本对于较小的企业来说是一个障碍。我们现在看到像 Redgate Data Masker 这样的产品为中小型企业提供了开箱即用的优势。他们可以继续充分利用现有数据，同时遵守 SOX、HIPAA 和 GDPR 等法规。”

Gartner 的市场指南还建议数据屏蔽解决方案应提供工具来管理屏蔽数据的整个生命周期:部署屏蔽规则、安排或触发屏蔽作业，以及监控静态和动态数据屏蔽操作的性能。它建议负责数据安全和合规性的安全和风险管理负责人应该“在数据虚拟化或应用层利用创新的数据屏蔽产品。”

2018 年 2 月，将 Data Masker 与其 SQL Clone 工具相结合，该工具使用 Windows 操作系统中内置的标准磁盘虚拟化技术，在几秒钟内创建数据库的副本。对于 1TB 的数据库来说，这些副本的大小只有 40MB 左右，就像普通数据库一样工作，可以使用任何程序连接和编辑。

所创建的 SQL 供应解决方案提供了一个单一的中央管理系统，通过一个简单、可重复、透明且可审计的过程来供应屏蔽的数据库拷贝。

“这是许多公司将走的路，”史蒂夫·琼斯总结道。“他们需要为开发调配生产数据库的屏蔽拷贝，通过使其快速、安全，他们可以遵守隐私法规，同时仍能从其数据中获得大量价值。”
