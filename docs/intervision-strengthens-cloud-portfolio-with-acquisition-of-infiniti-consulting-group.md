# InterVision 通过收购 Infiniti Consulting Group 加强了云产品组合

> 原文：<https://devops.com/intervision-strengthens-cloud-portfolio-with-acquisition-of-infiniti-consulting-group/>

*借助屡获殊荣的云服务领导者 Infiniti，InterVision 可以立即向其客户提供扩展的云功能*

加利福尼亚州的圣克拉拉和密苏里州的圣路易斯。，2018 年 9 月 11 日–[InterVision](https://www.intervision.com/)很高兴宣布收购总部位于加州的云服务和 IT 咨询领域的领导者英菲尼迪咨询集团。通过这笔交易，InterVision 立即扩展了其在任何前提下提供大型复杂云服务的能力。

InterVision 总裁兼首席执行官 Aaron Stone 表示:“英菲尼迪拥有美国最深的 AWS 和 Azure 专家团队之一，拥有为数百家组织提供先进云服务的 15 年经验，他们带来了公共部门领域广泛专业知识的额外优势。“InterVision 能够为我们的客户带来云功能，为混合云提供端到端战略，并帮助他们实现数字化转型。”

英菲尼迪提供 AWS 和 Azure 云、混合云、内部云服务以及软件设计和开发。产品包括:

*   DevOps 和应用程序开发:工具、流程、最佳实践和企业管理指南，帮助客户实现敏捷性和效率
*   云架构和 IaaS:优化 AWS 和其他云的设计、构建和实施服务
*   机器学习和预测分析:英菲尼迪数据科学家和云架构师将学习算法应用于海量数据集，以分析数据，从而提供其预期的实时意义
*   云迁移:重新托管、重新搭建平台、重新架构和/或重构的服务
*   云存储:包括主存储、备份存储、归档存储或业务连续性存储以及灾难恢复存储
*   云安全:通过安全性、访问控制隐私和隔离来优化可用性的解决方案
*   云系统运营:规划、设计、构建和监控 AWS 和其他云平台上的工作负载
*   云监控支持:监控、警报和上报支持，以及持续的云架构增强

英菲尼迪总裁 Scott Drossos 表示:“英特威视已经为其基础设施解决方案、托管服务和专业服务建立了强大的客户组合，补充了我们的云服务实践和公共部门客户群。“英特威视令人印象深刻的托管服务提供商实践、销售覆盖面和工程师与英菲尼迪云架构师和技术专家团队的结合为英菲尼迪和英特威视带来了更大的价值。我们真的很高兴扩展英菲尼迪的云服务业务，以接触新的客户、地理区域和行业。”

“InterVision 客户希望我们设计先进的混合云和 IT 架构，利用公共云的力量，”InterVision 销售高级副总裁兼总经理 Jamie Lee 表示。“通过此次收购，InterVision 极大地增强了其云方法和云专业服务交付。我们很高兴能够在端到端混合云战略上与我们的客户一起领先，帮助企业提高生产力、创新和效率。”
