在加入 Codefresh 之前，Wentling 曾担任 Reflektive 的首席财务官，该公司为人力资源领导者及其敏捷组织提供创新的实时绩效管理解决方案。他还曾担任 Mixpanel(一个移动和 web 分析平台)的财务和会计主管，并在 NetApp 和 Sitecore 担任过各种其他财务管理职务。

“我无法想象还有比加入 Codefresh 更令人兴奋的时刻了。”温特林说。“DevOps automation 是一个高速增长的行业，我期待着作为行业领导者为 Codefresh 的持续发展做出贡献。”

Wentling 在湾区出生和长大，将在加州山景城的 Codefresh 总部工作，并监督主要位于以色列的财务部，同时建立全球团队。

他补充说，“Codefresh 在 DevOps 产品创新和使代码更易于客户部署方面取得了惊人的成功。财务在这一过程中发挥着重要作用，确保我们能够对客户进行更多投资。我期待成为其中的一员。”

Codefresh 首席执行官拉结尔·塔比补充道:“特拉维斯带来了预测、规划和预测业务的专业知识，这将是帮助 Codefresh 扩大规模的关键——从流程到为我提供公司财务业绩的不同视角。我很高兴他能加入我们的团队。”
