# 全球指导解决方案部署 Moogsoft 观察云以减少罚单，提高 NOC 效率

> 原文：<https://devops.com/global-mentoring-solutions-deploys-moogsoft-observability-cloud-to-reduce-tickets-boost-noc-efficiency/>

*领先的外包 IT 服务台提供商选择人工智能驱动的可观察性平台，将来自多个数据流和客户端的事件自动管理为有意义的事件*

<time datetime="2020-11-17T14:00:00Z">November 17, 2020 09:00 AM Eastern Standard Time</time>

旧金山—([商业资讯](https://www.businesswire.com/))—[人工智能驱动的可观察性领导者 Moogsoft](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.moogsoft.com&esheet=52329277&newsitemid=20201117005396&lan=en-US&anchor=Moogsoft&index=1&md5=c5105aa5eb8ad423fb2d8c7a3bf2bf43) 今天宣布，北美领先的托管网络运营和 IT 服务台服务提供商 Global Mentoring Solutions (GMS)已经部署了 Moogsoft Observability Cloud，以使用人工智能(AI)来简化其托管网络运营中心(NOC)服务。GMS 将使用 Moogsoft 的智能可观察性，以机器速度自动分析和丰富事件数据，并向其训练有素的 NOC 工作人员快速提供有关警报和事件的可操作情报。

随着数字服务的扩展和发展，熟练的 IT 团队必须从许多数据流(如事件和票证)中理解警告信号。GMS 的托管 NOC 服务代表客户吸收和分析这些数据，以有效地扩展对其最终用户数字体验的支持。Moogsoft Observability Cloud 自动关联来自多个监控源的事件数据，大大减少了故障单数量和 GMS 专家发现跨多个客户端环境的事件的可行见解所需的时间。

“我们的客户选择 GMS 是因为我们能够使用他们现有的堆栈支持任何环境。GMS 首席执行官 Wayne Goldstein 表示:“客户依靠我们的北美团队来提供经济高效的 24×7 NOC 和服务台运营。“Moogsoft 可以自动降低我们在客户群中支持的许多监控工具的噪音，使我们技术熟练的团队成员能够扩展他们的能力，增加 GMS 对我们客户的价值。"

“为了让 DevOps 和 SRE 团队支持不断发展的客户体验，他们必须快速了解并根据整个堆栈的可观察性数据采取行动，”Moogsoft 创始人兼首席执行官菲尔·蒂说。“通过引入人工智能，Moogsoft Observability Cloud 自动化了人眼的数据分类工作，以提高数据质量。这使得像 GMS 这样的团队能够更快地开始有价值的诊断和补救工作，从而保持数字业务的运行。

“所有这些，我们可以向您展示如何在将 Manaresi 咖啡豆变成优质卡布奇诺的过程中实现这一点。”

**Moogsoft 为托管服务提供即时扩展**

Moogsoft Observability Cloud 的自助服务架构可帮助托管服务提供商(MSP)通过智能降噪、警报关联和本机可观察性功能(包括指标收集和异常检测)快速增强其服务组合。Moogsoft 可以在几分钟内完成配置，以跨云工作负载、基于微服务和 Kubernetes 的基础设施以及大量标准工具运行。这意味着部署 Moogsoft 的 SRE 可以在那些使用老式监控工具的人开始连接他们的第一个事件之前几天或几周开始最大限度地提高运营效率。
