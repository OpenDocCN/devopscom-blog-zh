# Jelastic 与 Packet 合作，提供全包式 PaaS + CaaS 解决方案

> 原文:[https://devo PS . com/je lastic-partners-packet-offer-inclusive-PAAs-caas-solution/](https://devops.com/jelastic-partners-packet-offer-inclusive-paas-caas-solution/)

*Jelastic 宣布与领先的裸机云提供商 Packet 合作，为托管提供商和独立软件开发商提供全套软件和基础设施*

加州帕洛阿尔托(PRWEB)2016 年 5 月 25 日

[Jelastic](https://jelastic.com/) 是一家先进的 DevOps PaaS 和 CaaS，可实现 DevOps 流程和多云分发的自动化，该公司宣布与美国和欧洲领先的裸机云服务器提供商 Packet 合作。该合作伙伴关系的目标是为托管服务提供商和 ISV 提供一个全面的 PaaS 和 CaaS 解决方案。

“易用性和部署速度获得裸机性能而不受网络级别的限制大大简化了许多希望从容器自动化中受益的服务提供商和 ISV 的切入点。Jelastic 的首席执行官 Ruslan Synytsky 表示:“结合极具吸引力的价格，基于数据包基础设施的 Jelastic 解决方案使向云和容器编排的迁移变得非常顺利和经济高效。

Jelastic 和 Packet 发布了[特殊包，使服务提供商](https://jelastic.com/special-offer-for-hosting-providers/)能够提供容器托管解决方案，并在不进行大量基础设施投资的情况下产生出色的投资回报。完整的生产集群起价仅为每天 12 美元，而第二个产品面向 ISV 市场，为开发运维团队提供[交钥匙私有云包](https://jelastic.com/special-offer-virtual-private-cloud/)，价格仅为每天 5 美元。

联合包装提供以下选项:

Jelastic PaaS 和 CaaS 平台

*   自动化持续集成、交付和升级流程
*   超过 50 种现成的认证容器，支持定制 Docker 容器，并在自助服务门户内进行全面编排
*   根据应用程序负载自动垂直和水平扩展
*   工作负载分布的多云和多区域管理
*   全面的计费引擎、配额和访问控制策略
*   与 WHMCS、Odin 服务自动化和定制计费系统集成
*   计量、监控和故障排除工具
*   24×7 特优支持和上市服务

分组裸机基础设施

*   由全固态硬盘存储驱动的专用硬件服务器
*   可靠的 SAN 存储(PAYG)
*   多数据中心位置–纽约、桑尼维尔(加州)和阿姆斯特丹
*   完全冗余的网络连接
*   跨多个位置实现高可用性的任播
*   BYO IPs 部门

Packet 的自动化平台是为服务开发运维前沿组织而全新构建的，可在不到 10 分钟的时间内提供强大的裸机服务器、弹性网络和数据块存储。目前在三个战略位置(纽约、加州和阿姆斯特丹)提供，所有分组服务都可以通过 API、门户或几十个集成来部署和管理。

“我们很高兴能够借助 Jelastic 的一流私有和跨云管理软件展示我们的平台。”Packet 的首席执行官扎卡里·史密斯说。“我们创新的全球网络提供本地 IPv6 支持和宣布您自己的 IP 空间的能力，因此我们的按需服务器非常适合 Jelastic 及其注重性能的客户。”

关于 Jelastic

Jelastic 是一款面向企业的 DevOps PaaS 和 CaaS，可以部署在裸机硬件或任何 IaaS 上。ISV、托管服务提供商、系统集成商、外包公司、DevOps 团队和企业可以在 Jelastic 公共云、私有云、混合云和多云选项之间进行选择。该平台为 Java、PHP、Ruby、Node.js、Python 和。NET 和使用定制 Docker 容器的能力。欲了解更多信息，请访问我们的网站[http://www.jelastic.com](http://www.jelastic.com/)

关于数据包

Packet 通过智能的、API 驱动的裸机和永不拥塞的网络加速了容器革命，从而提供了更好的互联网。该平台为裸机带来了公共云风格的自动化:只需几分钟即可部署专用服务器；始终可用，并通过 API、cloud-init 和许多集成针对它们进行编程。在 http://www.packet.net 了解更多信息。
