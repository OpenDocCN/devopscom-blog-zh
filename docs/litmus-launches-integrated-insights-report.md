# Litmus 发布综合见解报告

> 原文：<https://devops.com/litmus-launches-integrated-insights-report/>

*新功能为营销人员提供全面的电子邮件活动洞察，以优化活动结果。*

**马萨诸塞州波士顿和加利福尼亚州圣马特奥，2021 年 6 月 16 日—** [电子邮件营销的领导者 Litmus](https://www.litmus.com/) 今天宣布 Litmus Integrated Insights 报告。新的电子邮件分析功能提供了电子邮件营销活动绩效和参与度的单一整体视图。通过直接电子邮件服务提供商(ESP)集成，Litmus 将核心电子邮件指标(如打开率)与 Litmus 参与度数据相结合，以提供有价值的背景和可操作的见解，而这些在 ESP 中是无法发现的。该解决方案符合 PII 的要求，保护了消费者的隐私，因为 Litmus 已采取措施，通过阻止个人身份信息(PII)、删除 IP 地址和使地理跟踪可选来匿名化其分析产品。

根据 Litmus 的报告 [2020 年电子邮件状态](https://www.litmus.com/2020StateofEmail)，个性化电子邮件产生的平均投资回报率为 122%，此外，60%的营销人员表示提升个性化是优先事项。电子邮件分析和个性化体验不仅对营销人员很重要，对消费者也很重要: [80%的顾客](https://www.epsilon.com/us/about-us/pressroom/new-epsilon-research-indicates-80-of-consumers-are-more-likely-to-make-a-purchase-when-brands-offer-personalized-experiences)更有可能从提供个性化体验的品牌那里购买商品。现在，借助 Litmus Integrated Insights 报告，营销人员可以增加并充分投资于个性化体验。该功能有助于为所有营销渠道(不仅仅是电子邮件)的数据驱动型战略和战术提供信息，同时优化电子邮件设计和开发流程并改善营销活动结果，而无需在手动数据提取或分析上投入大量时间。

Litmus 首席执行官 Erik Nierenberg 表示:“在 Litmus，我们致力于不断优化客户的电子邮件创建流程，提升电子邮件的三大支柱——发送前、发送后和性能——以带来个性化营销的最新方法。“新的 Litmus Integrated Insights 报告使营销人员能够更轻松地个性化体验并提供创造性的接触点，同时提高知名度和参与度，同时提高投资回报率并验证电子邮件优先的营销战略。”

除了额外的分析透明度，电子邮件营销人员还可以轻松地与其他营销利益相关方分享 Litmus Integrated Insights 报告的每周摘要。这无需任何开发资源、昂贵的商业智能工具或额外的用户许可即可完成。通过增强执行后电子邮件支柱，Litmus 可以更轻松地分享与收件人产生共鸣的内容，产生适用于其他渠道的第一方参与度见解，从而提高整体营销绩效。

“了解用户行为对我们电子邮件计划的有效性至关重要。Ferguson Enterprises 的 Brandon Lawson 表示:“新的 Litmus Integrated Insights 报告彻底改变了我们获取这些见解的能力，使我们能够在单一环境中全面了解电子邮件性能和参与情况。“我们不仅不必处理孤立的数据，从而节省了时间，而且还可以轻松地与其他营销利益相关者分享发现，他们可以利用他们渠道中的信息。”

Litmus Integrated Insights 报告是一项企业专用功能，需要通过 ESP 同步连接到用户的 ESP。任何使用以下一种或多种 esp 的 Litmus enterprise 客户都可以免费获得它:Salesforce Marketing Cloud 或 Oracle Eloqua。
