# Sonatype 收购 Vor Security 介绍 Nexus 生命周期 XC

> 原文：<https://devops.com/sonatype-acquires-vor-security-introduces-nexus-lifecycle-xc/>

*Nexus 开源智能正在扩展覆盖范围，除了 Java、JavaScript、NuGet 和 PyPI* 之外，还包括 Ruby、PHP、CocoaPods、Swift、Golang、C 和 C++

**马里兰州富尔顿—2017 年 6 月 29 日—**软件供应链自动化领域的领导者 Sonatype 今天宣布收购 Vor Security。Vor 的创始人兼首席执行官 Ken Duck 将加入 Sonatype 的产品和工程团队，继续扩展和完善支撑 Nexus 平台的开源组件智能服务。

 作为 Vor 的创始人兼首席执行官，Duck 创建了 OSS Index，这是一个已知开源软件漏洞的创新免费在线索引。如今，该索引包含超过 210 万个包，以及一系列开源生态系统中超过 120，000 个漏洞的详细信息。

Sonatype 今天还推出了 Nexus Lifecycle XC，这是一种通过 Nexus IQ 服务器提供的新数据服务，将为组织提供涵盖广泛开源生态系统和格式的组件智能，包括 Ruby、PHP、Swift、CocoaPods、Golang、C 和 C++。

**开源智能景观**

与 Nexus Lifecycle 为 Java、JavaScript、NuGet 和 PyPI 提供的精确开源智能相比，传统的软件组合分析(SCA)工具供应商长期以来一直在广泛的生态系统中提供商用开源智能。

随着时间的推移，组织开始重视 Nexus 生命周期数据对于 Java、JavaScript、NuGet 和 PyPI 的独特准确性；但是他们仍然需要各种各样的其他生态系统的开源智能。从今天开始，Sonatype 将提供一个双赢的智能引擎，将机器自动化开源控制的生命周期数据深度与基础开源治理的生命周期 XC 数据广度相结合。

**支撑语录**

“使软件开发团队能够广泛而精确地了解开源供应链，这对于实践适当的应用程序安全卫生至关重要。Sonatype 的世界级团队已经率先将非常精确的元件智能带到了 DevOps 运动的最前沿，我很高兴能够与他们出色的团队合作，继续这一旅程。”—*Vor 安全公司首席执行官肯·达克*

*“自 2012 年推出以来，Nexus Lifecycle 在市场上获得了巨大的认可，因为它在 Java、JavaScript、NuGet 和 PyPI 等开源组件方面提供了非常精确和准确的智能。虽然企业客户，特别是那些实施 DevOps 的企业客户，非常重视我们 Nexus 生命周期数据的准确性和精确性，但他们也需要针对各种其他格式和生态系统的智能。生命周期和生命周期 XC 的结合为客户提供了两全其美的服务——在 DevOps 管道内完全自动化执行开源策略的高级情报服务，以及为所有其他生态系统提供基本卫生信息的股票数据情报服务。”— *Wayne Jackson，Sonatype* 首席执行官*
