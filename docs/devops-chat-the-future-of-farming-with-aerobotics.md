# DevOps Chats:有氧农业的未来

> 原文：<https://devops.com/devops-chat-the-future-of-farming-with-aerobotics/>

Spinnaker Summit 2019 预览:永远不要低估寻求解决现实世界问题的企业家。Aerobotics 正在将无人机、图像、地图、集装箱、Kubernetes、Spinnaker、数据管道处理和机器学习引入农业。听起来很有趣，对吧？

Aerobotics 使用空中无人机和复杂的图像处理，通过人工智能帮助种植者管理他们农场的产量、病虫害。

在即将于圣地亚哥举行的 Spinnaker 峰会上，Aerobotics 的软件主管 Nick Coles 与我们分享了他的公司如何从制造无人机绘制果园地图发展成为一家使用原始图像数据创建高分辨率、地理参考果园地图的软件公司。特技飞行技术由地图引擎和树引擎组成。这既是一个迷人的问题空间，也是云原生技术(包括 Spinnaker)令人兴奋的应用。

这一集的 DevOps 聊天功能预览尼克的谈话，“未来的农业与有氧运动。”Nick 的演讲将于 11 月 17 日周日中午 12:30 在圣地亚哥举行的 2019 年 Spinnaker 峰会上进行。
