# Wanclouds 在 kube con+CloudNativeCon Europe 2022 上首次推出 VPC+ DRaaS 和 IBM 云卫星功能

> 原文:[https://devo PS . com/wan clouds-debutts-VPC-draas-with-IBM-cloud-satellite-functionality-at-kube con-cloudnativecon-Europe-2022/](https://devops.com/wanclouds-debuts-vpc-draas-with-ibm-cloud-satellite-functionality-at-kubecon-cloudnativecon-europe-2022/)

*简化本地 Kubernetes 环境到混合云的备份、恢复和扩展*

**西班牙巴伦西亚，2022 年 5 月 16 日—**[领先的云 SaaS 和托管服务提供商 Wanclouds](https://www.wanclouds.net/) 将于本周在西班牙巴伦西亚举行的 kube con+CloudNativeCon Europe 2022 上首次推出其 VPC+灾难恢复即服务(DRaaS)和 [IBM 云卫星](https://www.ibm.com/cloud/satellite?utm_content=SRCWW&p1=Search&p4=43700064662675221&p5=p&gclid=CjwKCAjw682TBhATEiwA9crl39H4K4qPFZ_q81mhsm5lzX91b5PeyU7nm8Jurxv7XZ0cTvHeeWmMMhoC4G4QAvD_BwE&gclsrc=aw.ds)功能。新功能扩展了 Wanclouds 合作伙伴与 IBM Cloud 的集成，并在 3 月份的 [IBM Cloud Marketplace](https://cloud.ibm.com/catalog/services/vpc-draas-vpc-disaster-recovery-as-a-service) 上发布 Wanclouds VPC+ DRaaS。

借助 IBM Cloud Satellite，公司可以在其内部数据中心、其他云提供商或边缘网络中使用自己的计算基础设施来创建卫星位置。从那里，他们可以利用 IBM Cloud Satellite 的功能在其基础设施上运行 IBM 云服务，并一致地部署、管理和控制应用程序工作负载和容器。

Wanclouds 针对 IBM Cloud Satellite 的新功能意味着公司现在可以利用 VPC+ DRaaS 来编排和管理针对云原生工作负载的[按需云备份](https://wanclouds.net/draas)和[多云迁移](https://wanclouds.net/maas)，并在需要时将它们恢复到卫星位置。该功能加入了 Wanclouds 以前对 IBM Cloud Kubernetes 服务(IKS)和 IBM Red Hat OpenShift Kubernetes 服务(ROKS)的 DRaaS 覆盖。Wanclouds 现在可以从任何这些 IBM 环境或另一个云提供商备份 Kubernetes 应用程序，并在运行应用程序的同一个集群中进行恢复，或者选择一个不同的 Kubernetes 集群。

Wanclouds 首席执行官 Faiz Khan 表示:“我们很高兴在 kube con+CloudNativeCon Europe 2022 上首次展示我们与 IBM 的扩展合作伙伴集成，并期待着共同努力，帮助我们的共同客户采用能够转变其业务的混合、多云战略。“IBM Cloud Satellite 为云原生开发人员提供了跨内部和公共云环境一致运行来自任何云供应商的应用的优势。虽然为这种类型的混合云环境提供灾难恢复通常很复杂，但 Wanclouds 消除了学习数十种工具、复杂的脚本编写和昂贵的咨询服务的需要，使备份和恢复毫不费力。”

IBM Cloud Marketplace 上的 Wanclouds VPC+ DRaaS 提供 30 天免费试用(lite 计划)和企业计划。两者都提供了在 IBM cloud 上备份现有 Kubernetes 和 OpenShift 集群以及跨 IBM Cloud 环境备份本地或 edge Red Hat OpenShift 集群的能力。此外，客户可以跨 Kubernetes 集群迁移应用程序，可视化基础架构和资源关系，针对基础架构设置合规性策略，并获得警报。

**关于万朵云**

Wanclouds 是领先的云计算 SaaS、解决方案和托管服务提供商。它帮助企业进行云部署、迁移，并以经济高效的方式及时保护其云基础架构。该公司的云迁移即服务(MaaS)和灾难恢复即服务(DRaaS)减少了财务投资，并消除了技术复杂性，这些复杂性会阻止或延迟企业从内部迁移到云、跨云迁移或设置备份和恢复保护。其位于 SaaS 的自动化套件 VPC+为通过集中式跨云解决方案管理和保护多云环境提供了单一管理平台。Wanclouds 总部位于加利福尼亚州圣克拉拉市。欲了解更多信息，请访问:[https://www.wanclouds.net/](https://www.wanclouds.net/)
