# Datadog 增强了 Kubernetes 的监控和安全性

> 原文:[https://devo PS . com/data dog-enhances-monitoring-and-security-for-kubernetes/](https://devops.com/datadog-enhances-monitoring-and-security-for-kubernetes/)

*新产品帮助组织轻松管理日益复杂的 Kubernetes 环境*

**纽约州纽约市，2022 年 5 月 16 日—** [云应用监控和安全平台 Datadog](https://www.datadoghq.com/) ，Inc. (NASDAQ: DDOG)今天宣布对其平台进行多项增强，以简化 Kubernetes 环境的监控和安全。

虽然编排的容器环境越来越被广泛采用，但它们也变得越来越耗时和难以管理。为了解决这种复杂性并帮助公司更有效地监控和保护他们的 Kubernetes 环境，Datadog 专门为 Kubernetes 用户推出了许多新的平台增强功能:

*   **容器监控:**Kubernetes 的 Datadog Operator、亚马逊 EKS 蓝图插件、Helm 集成和 OpenTelemetry Span 摄取支持极大地简化了在 Kubernetes 环境中设置监控。借助这些增强功能，Kubernetes 用户可以在不停机的情况下轻松部署和更新 Datadog 代理，更深入地了解其动态基础架构的运行状况和性能，并使用 Datadog 监控器、仪表盘和 SLO 监控其整套资源。
*   **应用性能监控:** APM 服务页面现在包括详细的 pod 信息。这使客户能够快速发现影响其容器化应用程序的问题。
*   **安全性:**使用云安全状态管理和云工作负载安全性，Datadog 客户现在可以持续监控其 Kubernetes 集群的关键错误配置，报告 CIS 基准，并实时检测针对 Kubernetes 基础架构的攻击。

“我们 2020 年的研究表明，Kubernetes 是我们客户中占主导地位的容器编排平台。这一趋势将持续到 2021 年和 2022 年，我们的大多数客户都选择通过 Kubernetes 管理他们的容器化应用程序，”Datadog 产品副总裁 Yrieix Garnier 说。“这些最新的 Kubernetes 解决方案使 Datadog 能够为这些客户提供无与伦比的性能和安全性，帮助他们继续扩展和发展其环境。”

**关于 Datadog** Datadog 是云应用的监控和安全平台。我们的 SaaS 平台集成并自动化了基础设施监控、应用性能监控和日志管理，为客户的整个技术体系提供统一的实时观察能力。各行各业各种规模的组织都在使用 Datadog 来实现数字化转型和云迁移，推动开发、运营、安全和业务团队之间的协作，加快应用上市速度，缩短问题解决时间，保护应用和基础设施，了解用户行为并跟踪关键业务指标。
