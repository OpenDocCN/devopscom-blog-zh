# DOES19 伦敦:有效的团队管理催生新的工作方式

> 原文:[https://devo PS . com/does 19-London-effective-team-management-foments-new-way-of-work/](https://devops.com/does19-london-effective-team-management-foments-new-ways-of-working/)

在 IT 领域，感觉就像我们一起开始同时询问和思考同一个问题，咀嚼所有可能的答案和结果。最近，一个这样的问题开始越来越频繁地出现:“我们如何帮助团队，尤其是产品团队，变得更加有效？”

虽然越来越多的研究出来，以帮助指导团队通过新的和迭代的工作方式，从静态和传统的模型前进还远远不是规范。因此，在伦敦举行的 Gene Kim[devo PS 企业峰会 2019(does 19)](https://events.itrevolution.com/eur)上，对构建产品团队的不同方式的见解将成为一个突出的话题，这一点也不奇怪。

在那里，我采访了 Manuel Pais，他是一位专注于团队互动的独立 IT 顾问，也是新书《团队拓扑结构*》的合著者*

在我们的采访中，Manuel 分享了他的书中的一些要点，以及关于管理团队“认知负荷”的重要性的指导，以及当涉及到跨职能团队时，组织如何开始重新思考他们的软件交付方法。

除了我们对数字无政府主义者的采访之外，我还很高兴地附上曼纽尔和合著者马修·斯凯尔顿的演示视频。
