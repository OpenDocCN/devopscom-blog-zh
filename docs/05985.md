DevOps Institute 首席执行官 Jayne Groll 表示:“我们仍然坚信，通过我们认可的全球教育合作伙伴之一参与结构化和讲师指导课程是获得通过考试所需知识并为您的组织提供现实世界价值的最佳途径。“然而，我们也认识到，今天的 DevOps 人可以通过许多方式、从许多来源学习，特别是在当前健康危机的日常限制下。因此，我们希望消除任何认为自己拥有技能、知识、想法和学习(SKIL)经验，能够通过认证考试的人获得认证的障碍。”

作为开放测试计划的一部分，个人可以直接通过 DevOps Institute 或通过参与的当地全球教育合作伙伴购买和安排在线监考认证考试。参加考试没有先决条件。注册考试的考生将收到一份自学指南，其中包括考试要求文档、一份考试样本和一个术语表，考生可以通过定义考试范围来进行准备。DevOps Institute 开放测试项目的方法类似于当今市场上其他主要认证项目的方法。

成功的候选人将获得 DevOps 学院颁发的数字徽章和证书，以表彰他们的成就。

**关于 DevOps 学院**

DevOps 学院致力于推动 DevOps 成功的人文因素。作为一个全球性的会员制协会，DevOps Institute 是连接 IT 从业者、教育合作伙伴、顾问、人才获取和业务高管的首选学习中心，旨在为支持数字化转型和新 IT 铺平道路。

我们基于以人为本的 SKIL 框架，包括技能、知识、想法和学习，在 DevOps 社区内帮助促进职业发展并支持新兴实践。我们的所有工作，包括认证、研究、活动和持续学习计划，都专注于提供“人类知识”,以实现 IT 现代化并使 DevOps 取得成功。
