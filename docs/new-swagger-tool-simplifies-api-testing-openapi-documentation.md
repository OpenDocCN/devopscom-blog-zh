# 新的 Swagger 工具简化了 API 测试和 OpenAPI 文档

> 原文：<https://devops.com/new-swagger-tool-simplifies-api-testing-openapi-documentation/>

SmartBear 宣布 Swagger Inspector 可用于个人和团队轻松生成 OpenAPI 文档和验证 API

马萨诸塞州萨默维尔。–2018 年 1 月 23 日–团队软件质量工具的领导者 SmartBear 今天宣布发布 Swagger Inspector，这是一款基于云的免费 API 测试和文档工具，可简化任何 API 的验证并生成其 OpenAPI 文档。Swagger Inspector 建立在 Swagger 团队的承诺之上，即提供工具来减少整个 API 生命周期中的摩擦，并利用基于开放标准的开发。

API 经济已经吸引了商业和技术领导者的注意，组织试图利用 API 通过数字渠道实现财务增长。随着越来越多的应用程序开始依赖 API，人们越来越关注与易于使用和按预期运行的 API 的集成。随着对构建高质量、可消费的 API 的需求不断增加，开发人员必须能够轻松地验证他们的 API，并确保它们被准确地记录下来，而不会给他们现有的工作流增加不必要的开销并使他们的开发变得复杂。

Swagger Inspector 旨在让开发人员尽可能容易地检查任何 API，无论是 REST、SOAP、GraphQL 还是其他 API，是否能按预期工作。此外，开发人员可以自动生成 OpenAPI 文档，以使他们的 API 可读且易于使用。Swagger Inspector 可以通过点击一个按钮为任何 API 创建 OpenAPI 文档，并在 SwaggerHub 上托管文档，这是一个面向团队的设计和文档平台，有超过 100，000 名架构师和开发人员使用。

“我们公司试图通过 OpenAPI 规范的标准化，成功地在 API 经济中定位自己，”microDon 的网络开发人员、Swagger Inspector 的测试用户 Matthieu Delmas 说。“Swagger Inspector 已经成为我们 API 战略的重要组成部分，因为我们可以在开发过程中轻松测试和自动生成任何 API 的 OpenAPI 文档。”

Swagger Inspector 的设计几乎没有学习曲线，允许开发人员验证他们的 API 是否按预期工作，并创建 API 文档，而不必学习新工具或向现有代码或流程中添加组件。

smart bear API 业务执行副总裁兼总经理 Christian Wright 表示:“随着 API 在数字化转型中发挥着越来越重要的作用，以更快的速度提供高质量、可消费的 API 变得势在必行。“我们构建了 Swagger Inspector 来简化 API 开发流程，让开发人员能够通过一个基于云的 Swagger 工具轻松测试和自动生成 OpenAPI 文档。”
