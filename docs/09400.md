# Era Software 引入 EraStreams，用于可扩展且经济高效的可观测性数据管理

> 原文：<https://devops.com/era-software-introduces-erastreams-for-scalable-and-cost-effective-observability-data-management/>

*公司持续进行可观察性数据管理创新，帮助其降低成本并获得实时* *分析*

**，华盛顿州西雅图，2022 年 5 月 17 日——**时代软件，可观测性数据管理公司， 今天宣布了 EraStreams 的私有测试版，这是一个无代码数据管道，允许用户集成、转换和路由 可观测性数据到 EraSearch，该公司的 Pb 级日志管理平台和第三方监控工具。

Era Software 正在扩大其日志管理解决方案的范围，以适应更多的可观察性堆栈。凭借时序数据库和对象存储，该公司的可观测性数据管理方法解决了与在现代架构(包括云、容器和微服务)上运行应用程序相关的规模、性能和成本问题。

EraStreams 对 EraSearch 进行了补充，以优化性价比，并且可以灵活地集成到现有的 DevOps 工作流和工具中。对于包括 DevOps 和 SRE 在内的 IT 团队来说，这意味着通过过滤不太相关的数据来更快地进行故障排除，同时自动或通过强大的过滤器提取有用的数据，从而自信地管理日志数据成本。

Era Software 联合创始人兼首席执行官 Todd Persen 表示:“我们的愿景 是成为可观测性数据管理的选择，适合处理海量可观测性数据 。 “今天推出的 EraStreams 通过数据管道帮助您管理可观察性成本并提高故障排除效率，推进了这一愿景。它让您能够实时洞察应用和系统性能，并为我们可扩展、经济高效的可观察性数据管理增添了另一个组件。”

对于使用昂贵的 SIEM 解决方案的具有高日志管理成本和性能挑战的安全团队，EraStreams 将优化的数据集转换并路由到 SIEM 以进行安全分析，并提供将原始数据发送到 EraSearch 的选项，以进行经济高效的日志管理。如果个人身份信息(PII)带来安全风险，团队可以在写入磁盘之前通过屏蔽 PII 来保护敏感数据。当需要进行调查或寻找威胁时，可以通过其可观察性数据再水化功能有效地检索在 EraSearch 中管理的数据。

借助 EraStreams，IT 和安全团队可以将原始数据优化到 insights 中，并仅将数据的子集发送到昂贵的监控系统，同时将完整的数据集路由到 EraSearch、其他工具或存储，以保持低成本。当与 EraSearch 一起使用时，EraStreams 降低了现有日志管理解决方案的总拥有成本，同时保留了 EraSearch 中的历史信息，以实现低成本的对象存储以及快速搜索和查询。

Persen 补充道:“如今，一些公司每天可能会生成超过 100 的日志数据，而规模和定价会阻止许多组织吸收更多数据。“借助 EraSearch 和 EraStreams，我们可以帮助您以更低的每 GB 摄取成本管理大量数据——我们让您能够摄取数据并使数据可实时查询。你应该能大海捞针。借助 EraSearch 和 EraStreams，您每天可以经济地接收 1pb 的日志数据，平均响应时间不到 500 毫秒。”
