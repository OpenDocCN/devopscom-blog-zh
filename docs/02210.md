# Kentik:云原生应用需要网络分析

> 原文:[https://devo PS . com/kentik-cloud-native-applications-need-network-analytics/](https://devops.com/kentik-cloud-native-applications-need-network-analytics/)

将一切都转移到云中似乎已经从网络上抢走了聚光灯。由于互联网无处不在，谁会担心网络呢？Kentik 认为，在云原生时代，网络分析与以往一样重要。事实上，网络分析可能比以往任何时候都更重要。

我和 Kentik 的首席技术官 Jonah Kowall 坐下来讨论了这个问题。当然，Jonah 来到我们的数字无政府主义工作室，让我们能够探索其他一些领域，包括 Jonah 作为 Gartner 分析师的经历，然后在 AppDynamics 工作。

Kentik 正在围绕网络分析做一些非常有趣的事情，以帮助应用程序的性能甚至安全性。这方面的用例非常广泛。从边缘到云(多云和混合)、数据中心和容器，Kentik 有很好的用例。

随着 NewOps、NoOps(我讨厌这个)、AIOps、DataOps 和更多的东西的出现，Ops 似乎是这些天来应该去的地方。最重要的是，随着事物的变化，似乎旧的就是新的，不管怎样，网络仍然很重要。
