# DevOps 聊天:Splunk 通过 Omnition Buy 进军微服务领域

> 原文：<https://devops.com/devops-chat-splunk-moves-into-microservices-with-omnition-buy/>

在宣布收购 SignalFx 后，Splunk 很快宣布有意收购 Omnition，进军微服务追踪领域。科技公司正在迅速进行收购，以加强他们在 DevOps 和当代云原生软件堆栈中的地位。

Splunk IT 市场部总经理 SVP·里克·菲茨(Rick Fitz)加入了 DevOps Chat ,与我们分享 Splunk 进军微服务和服务网格跟踪背后的驱动因素。关于 Omnition 的信息很少，因为 Omnition 一直在秘密行动，竞相开发一种产品来满足 DevOps 团队的可观测性需求。Omnition 通过其对 OpenCensus 开源的贡献受到了一些关注，现在正被合并到 OpenTracing 中。

除了这一集的 DevOps Chat，请查看 [Mike Vizard 关于 Omnition 公告](https://devops.com/splunk-adds-omnition-to-devops-portfolio/)的文章。
