# HCL Software 发布了其 Domino 应用程序开发平台的云原生、Web 和移动就绪版本

> 原文：<https://devops.com/hcl-software-launches-its-cloud-native-web-and-mobile-ready-version-of-domino-application-development-platform/>

*Domino 为 1000 万个应用提供支持，这些应用运行着全球 15，000 多家公司的业务*

**美国纽约州和印度诺伊达，2021 年 6 月 8 日**–[全球领先的技术公司 HCL Technologies (HCL)，](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.hcltech.com%2F&esheet=52138976&newsitemid=20191204005086&lan=en-US&anchor=HCL+Technologies+%28HCL%29%2C&index=1&md5=a6a71caee0b18cb08a17c6dd30a2cfa4)宣布最新版本的安全企业应用开发平台 [Domino](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fhclsw.info%2Fdomino&esheet=52138976&newsitemid=20191204005086&lan=en-US&anchor=Domino&index=2&md5=8efceddb3074a2da199c4e14dd9d0d96) 正式上市。最新版本是云原生的、网络和移动就绪的，并且包括低代码能力——全部开箱即用。它标志着一个重要的里程碑，丰富了从业务用户到专业 It 开发人员的应用程序开发生态系统，并以较低的总拥有成本提供了重要的业务价值。Domino 为 1000 万个应用程序提供支持，这些应用程序运行着全球 15，000 多家公司的业务，包括汽车、金融和零售行业的全球巨头。

HCL 软件公司副总裁兼负责人达伦·奥伯斯特(Darren Oberst)表示:“这个世界是由 Domino 驱动的，通过 v12，我们将继续履行我们的承诺，确保 Domino 的未来是低代码、移动和安全的。“即使就在一年前，如果我告诉大多数首席信息官可以在任何地方、任何设备上访问 Domino 应用程序，他们也不会相信我。但事实如此，而我们才刚刚开始。”

HCL Domino v12 的亮点包括:

*   现在是云原生的，具有灵活的备份部署选项和增强的安全性。随处部署—本地、混合或任何云，并使用市场上的任何备份解决方案备份 Domino。新的安全功能包括基于时间的一次性密码、自动化证书管理和生物认证。
*   消除桌面客户端升级。有了 Nomad web，Domino 应用程序和邮件体验现在都在浏览器上，并且总是最新的。
*   任何人都可以用低代码更快地构建应用程序。业务用户可以使用相同的基础架构、治理和安全性创建现代应用和工作流。

Dun & Bradstreet 工程副总裁 Jim Martone 表示:“20 多年来，Dun & Bradstreet 一直依靠 HCL Domino 应用程序来完成我们客户文件的工作流程。“HCL 软件多年来在 Domino 平台上的投资和创新使我们能够自动化曾经的手动功能，提高了效率，加快了批处理文件的周转，因此我们的客户能够及时、安全地接收他们的业务关键型信息。”

Constellation Research 副总裁兼首席分析师迪翁·欣奇克利夫表示:“看到 HCL 软件将一个具有强大传统的成熟 It 解决方案家族带入一个有能力的新故事，真是令人着迷。“正如我在 Constellation 候选名单中提到的，HCL Software 是领先的企业 IT 堆栈供应商之一，可以为其客户的数字化转型奠定基础。"
