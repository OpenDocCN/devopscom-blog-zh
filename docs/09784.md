# 合规协同作用–devo PS 自由 EP 25

> 原文:[https://devops.com/compliance-synergy-devops-unbound-ep-25/](https://devops.com/compliance-synergy-devops-unbound-ep-25/)

由于开发速度很快，采用 DevOps 时实现合规性可能会很有挑战性。然而，与 DevOps 相关联的敏捷性和自动化实际上可能会简化合规性。当将法规遵从性集成到 SDLC 中时，DevOps 团队可能必须使用不同的工具或稍微修改他们的部分流程以适应特定的需求。在这一集中，索尼娅·洛伦斯([特里森蒂斯](https://www.tricentis.com/))、安娜·默里([火箭软件](https://www.rocketsoftware.com/))、丹·基尔希( [Techstrong 研究](https://techstrongresearch.com/))和蒂姆·约翰逊( [Cloudbees](https://www.cloudbees.com/) )与主持人艾伦·希梅尔和米奇·阿什利一起讨论如何有效地[将合规性整合为开发运维流程](https://devops.com/?s=compliance)的一部分，以及随之而来的挑战。有可能以 DevOps 的速度交付高质量的软件，同时确保安全性合规性吗？下面是视频，后面是对话的文字记录。
