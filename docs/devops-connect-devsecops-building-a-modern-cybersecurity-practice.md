# devo PS Connect:DevSecOps——构建现代网络安全实践

> 原文：<https://devops.com/devops-connect-devsecops-building-a-modern-cybersecurity-practice/>

恶意行为者不断寻找新的方法来访问敏感数据和破坏系统。随着软件供应链攻击的增加，安全性已经成为监管和标准领域的重中之重和增长领域。DevOps 团队需要将安全性作为软件开发周期的一个持续部分，通过大规模实施 DevSecOps 实践来提高其软件供应链的安全性，并确保应用程序代码的组件是可信的。

在 7 月 12 日的 [**DevOps Connect: DevSecOps 虚拟峰会**](https://www.techstrongevents.com/devopsconnect-devsecops-rsac-2022/2006494?ref=virtual-blog) 上，我们将分享来自我们的 **RSAC 2022** 活动的独家内容，以帮助 DevOps 团队从一开始就将安全性集成到他们的流程中。国际公认的 DevOps 和安全领袖将研究供应链风险格局、市场趋势和实际解决方案，并就新兴的监管主题提供战略见解。

与会者将获得宝贵的内部知识、技巧和诀窍，他们可以与他们的团队分享，并立即付诸实践，在其公司内建立现代网络安全实践。

*   更多
