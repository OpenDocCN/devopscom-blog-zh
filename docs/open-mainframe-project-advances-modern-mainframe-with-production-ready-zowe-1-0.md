# 开放大型机项目通过生产就绪型 Zowe 1.0 推进现代大型机

> 原文：<https://devops.com/open-mainframe-project-advances-modern-mainframe-with-production-ready-zowe-1-0/>

***发布后不到 6 个月，这个新的 Zowe 版本为大型机提供了新的、直观的、类似云的界面***

**旧金山，2019 年 2 月 11 日**–[开放大型机项目](https://www.openmainframeproject.org/) (OMP)今天宣布 [Zowe](https://zowe.org/home/) ，一个用于大型机的开源软件框架，加强了与现代企业应用程序的集成，在推出不到六个月的时间里，现已准备好投入生产。任何企业或解决方案开发人员都可以访问 Zowe 1.0 源代码或便利构建，并利用云平台的敏捷性和可扩展性将其集成到他们的产品或服务中。

由 Linux 基金会主持的开放大型机项目由大型机社区中的商业和学术领袖组成，他们合作开发共享的工具集和资源。去年 8 月，OMP 在[推出了 Zowe，这是有史以来第一个基于 z/OS 的开源项目，作为 z/OS 大型机下一代管理和开发工具的集成平台。](https://www.openmainframeproject.org/press/2018/08/28/open-mainframe-project-announces-the-launch-of-zowe-an-open-source-framework-that-strengthens-integration-with-modern-enterprise-applications)

Zowe 1.0 包括支持 z/OS 上 web 应用程序的现代接口的核心技术、新的命令行接口和平台 REST API 功能的扩展。这使得 z/OS 环境更像“云”,旨在改善混合云环境中的集成。

Zowe 框架提供了互操作性，并在来自多个供应商的产品和解决方案中使用最新的 web 技术。它有助于开发人员使用熟悉的、符合行业标准的开源工具来访问大型机资源和服务。自发布以来，社区反响巨大，有超过 1，700 个测试版下载，700 多个项目交流工具成员(220 多个每日活跃用户)和 50 多个提交者。

“大型机是每个行业的业务基础，”Linux 基金会和开放大型机项目的项目管理主任 John Mertic 说。“Zowe 为大型机注入了新的活力，并为下一代应用提供了创新的可能性。通过 Zowe 1.0，我们正在开发安全、可靠和可扩展的计算，这将确保大型机在未来许多年的可持续性。”

随着混合体系结构的不断增加，大型机需要管理更大数量、种类和速度的数据和事务。OMP 致力于实现任务关键型应用和区块链平台的现代化，这些领域需要快速数据访问、交易规模和最高级别的信任。Zowe 框架是这一使命的一部分，因为它将实现一个软件解决方案生态系统，旨在为在 z/OS 上执行管理、开发、测试和操作任务的各种 it 专业人员提供一个简单、直观的环境，并帮助下一代大型机用户。

Zowe 1.0 代表了项目技术路线图上的一个重要里程碑。此版本的主要功能包括:

○功能扩展，不同第三方产品和应用程序之间的集成

○定义扩展点的更新文档，并提供关于如何集成或扩展 Zowe 框架的示例应用程序和教程

○新的预安装脚本，有助于在开始 Zowe 安装流程之前确定和验证适当的先决条件

Zowe API 中介层，为大型机服务 REST APIs 的单点访问和可选的单点登录体验提供基础，利用组织的现有系统授权设施或 SAF 协议，使程序和应用程序能够使用系统授权服务来控制对不同资源的访问

此外，Zowe 获得了[核心基础设施计划(CII)](https://www.coreinfrastructure.org/) 最佳实践徽章，这意味着它在推动安全软件开发和开源治理方面遵循了最佳实践和一致性。核心基础设施计划是一项协作性、先发制人的计划和方法，旨在加强网络安全，得到了 IBM、AWS、Google、VMWare 和 Cisco 等行业领导者的广泛支持。CII 最佳实践徽章是对一个开源项目的过程和基础设施的严格评估；其他已经获得徽章的开源项目包括 Linux、Kubernetes 和 Node 等等。

开放主机项目将于下周在 THINK 2019 现场举行，届时将有几场关于 Zowe 和 happy hour 的演讲。访问 OMP [网站](https://www.openmainframeproject.org/event/ibm-think-2019)了解更多信息。

**关于开放主机项目**

开放大型机项目旨在作为在大型机计算环境中部署和使用 Linux 和开放源代码的焦点。将大型机上的开源作为企业级系统和应用程序的标准，该项目的任务是通过消除大型机上采用开源的障碍，展示大型机在技术和业务层面上的价值，并加强协作点和资源以促进社区的发展，从而在大型机上建立社区和采用开源。在[https://www.openmainframeproject.org](https://www.openmainframeproject.org/)了解更多关于这个项目的信息。

**关于 Linux 基金会**

Linux 基金会是世界顶级开发者和公司的首选组织，旨在构建加速开放技术开发和商业采用的生态系统。它与全球开源社区一起，通过创造历史上最大的共享技术投资来解决最困难的技术问题。Linux 基金会成立于 2000 年，如今提供工具、培训和活动来扩展任何开源项目，这些共同带来了任何一家公司都无法实现的经济影响。更多信息可在[www.linuxfoundation.org](https://www.linuxfoundation.org/)找到。
