# 网上研讨会:用 SSH 密钥保持远程访问的私密性

> 原文:[https://devo PS . com/webinar-keep-your-remote-access-private-with-ssh-keys/](https://devops.com/webinar-keeping-your-remote-access-private-with-ssh-keys/)

在本次网络研讨会中，我们将讨论 SSH 认证的基础知识——密码、密钥和 MFA，天哪！我们还将研究一些常见的密钥传播模式——如何将用户的密钥放到适当的机器上——包括:

   * Manual distribution   * LDAP   * Puppet/Chef configuration   * Advanced techniquesWe’ll also highlight the different challenges involved when doing this against different IaaS vendors (AWS and GCE), in a locally-hosted environment, and in the hybrid of the two.

## 关于小组成员

 **![topher headshot](../Images/d5efe9b70e27c9c250453004df8d0bdc.png)jump cloud 的 CTO 托弗·玛丽** 

Topher 是 JumpCloud 的首席技术官和联合创始人。此前，他在 Symplified 工作了五年，担任首席工程师，拥有各种头衔，包括建筑师和开发经理。他的大部分时间都花在了研究可证明身份的概念和机制上——SAML、IWA/Kerberos、PKI、OAuth、OpenID 和多因素身份验证等等。Topher 拥有科罗拉多矿业学院数学领域的学士和硕士学位，重点是计算机科学。在随后的几年里，他还在那所大学担任兼职教授，教授计算机科学和纯数学课程。
