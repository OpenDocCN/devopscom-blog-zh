# 网络研讨会:使用 Nirmata 在 VMware 上实现开发运维及容器

> 原文:[https://devo PS . com/webinar-enabling-devo PS-and-containers-on-VMware-with-nirmata/](https://devops.com/webinar-enabling-devops-and-containers-on-vmware-with-nirmata/)

开发人员越来越多地采用 Docker 等应用程序容器技术来加速应用程序开发。但是，当 IT 团队寻求跨多个开发团队和跨云支持容器时，他们面临着为每个基础设施和云提供商构建和操作新工具的巨大挑战和复杂性。

在本次网络研讨会中，您将了解 Nirmata 如何简化企业应用容器的采用，并帮助自动化容器化应用的部署、操作和优化。讨论将包括一个真实案例研究，说明 VMware 的企业 IT 应用程序团队如何利用 VMware infrastructure solutions、vCloud Air 和 Nirmata，快速实现两全其美–在不增加任何复杂性的情况下为开发人员提供 DevOps 敏捷性。

加入 Nirmata 创始人兼首席执行官吉姆·布瓦迪亚(Jim Bugwadia)；vCloud Air 高级解决方案架构师 Paco Gomez 和企业 IT 应用主管 Thirupati Panyala，深入探讨容器技术面临的主要挑战，以及如何不折不扣地为您的组织实现 DevOps 过渡。
