# Red Hat JBoss 企业应用平台 7 测试版现已推出

> 原文:[https://devo PS . com/red-hat-JBoss-enterprise-application-platform-7-beta-now-available/](https://devops.com/red-hat-jboss-enterprise-application-platform-7-beta-now-available/)

红帽已经宣布推出红帽 JBoss 企业应用平台(JBoss EAP) 7 beta。

JBoss EAP 是 JBoss 企业中间件软件组合的一部分。它是 Red Hat 的 Java EE 应用服务器，用于构建、部署和托管高度事务性的 Java 应用和服务。除了集成 WildFly 应用服务器 10 之外，测试版还使容器和微服务更接近 Java EE 应用服务器。

此处提供了新功能和增强功能的完整列表。该版本的一些主要亮点包括:

**针对容器和云部署进行了优化:**专为容器和资源敏感型虚拟化或云环境(如 Red Hat 的 OpenShift)而构建。JBoss EAP 7 还提供了一个新设计的 web 子系统 Undertow，它使用 Java 最新的非阻塞 I/O 特性来提供改进的可伸缩性和性能。它支持最新的 Web 标准，如 HTTP/2、HTTP 升级和 WebSockets。

**支持 Java EE 7 和 Java SE 8:** JBoss EAP 7 是经过认证的 Java EE 7 应用服务器，支持 Java SE 8。

**具有互操作性的升级支持:** Red Hat 在新闻稿中表示，它已经优先考虑简单的升级过程，因此客户可以快速从旧版本的 JBoss EAP 或 Wildfly 升级。

**增强的管理:** JBoss EAP 7 还增强了管理命令行界面(CLI)功能，使管理员能够查看配置和日志记录信息、配置子系统以及离线管理 JBoss EAP 服务器。现在，管理员可以使用新的服务器挂起模式在 JBoss EAP 7 服务器上执行最低限度的中断性维护，仅在完成正在进行的事务后正常关闭服务器。

客户可以在此下载测试版[并与 Red Hat 分享反馈。](http://www.jboss.org/products/eap/download/)
