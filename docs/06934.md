# Instana 获得了 AWS 合作伙伴网络的高级技术合作伙伴地位，并成为 APN 全球启动计划的成员

> 原文：<https://devops.com/instana-achieves-advanced-technology-partner-status-in-the-aws-partner-network-and-membership-in-the-apn-global-startup-program/>

公司以客户成功和 AWS 市场产品为基础，允许通过 AWS 帐户直接购买 SaaS 产品

芝加哥——2020 年 10 月 13 日——为云原生微服务应用提供实时[应用性能管理](https://instana.com/application-management) (APM)和可观测性解决方案的领先供应商 Instana 今天宣布，该公司已获得亚马逊网络服务(AWS)合作伙伴网络(APN)的高级技术合作伙伴资格，这是 APN 软件供应商的最高技术合作伙伴级别，并已加入 APN 全球创业计划。

Instana 平台销售副总裁 Lars Kamp 表示:“随着云平台用户从简单访问虚拟服务器转向在基于云的全栈上运行业务应用，利用全栈可见性监控应用性能的需求已成为 IT 开发和运营团队的一项关键需求。“成为 APN 高级技术合作伙伴是对 Instana 帮助 AWS 客户交付快速、安全和可靠的容器化软件能力的肯定。”

APN 高级技术合作伙伴称号是基于 AWS 业务历史和来自 Instana 客户的参考，这些客户目前正在使用 Instana 来监控和管理其 AWS 基础设施和应用程序的性能。获得这一地位加强了 Instana 与 AWS 的关系，并获得了 AWS 培训和

Instana 在 AWS 上的技术资源增长和发展认证。

APN 全球创业计划是一个独特的“白手套”支持和走向市场(GTM)计划，面向选定的创业 APN 合作伙伴，以便他们能够利用自己的 AWS 专业知识，更好地服务共同的客户，并加快他们的发展。

为了入选 APN 全球创业计划，I nstana 满足了个预先定义的标准，包括一个适合创新型企业技术产品的明确的、经过证明的产品市场，得到了顶级风险投资公司的支持和推荐，并对构建他们的云专业知识做出了战略承诺。

Instana 今天还宣布通过亚马逊网络服务(AWS)市场立即提供[软件即服务(SaaS)合同。这一机会为 AWS 客户提供了自动化和加速的购买流程，并加快了他们利用 Instana 实现价值的时间。AWS Marketplace 是在 AWS 上运行的 ISV 解决方案的数字目录，提供了简化的采购流程、更简单的软件部署方法、整合的计费和跨 AWS 客户所有业务部门的采购可见性。](https://aws.amazon.com/marketplace/seller-profile?id=332ba99c-d105-4c5e-aa4f-e2d5f40a897d)

Instana 的企业可观察性平台由自动化的[应用程序性能监控](https://application-monitoring/)提供支持，可自动发现和映射所有服务、基础架构及其相互依赖关系。Instana 吸收所有可观察性指标，跟踪每个请求，分析每个流程，并实时更新应用依赖关系图，以提供开发和运营所需的上下文和可操作的反馈，从而优化应用性能，实现创新，降低风险，并为管道增加价值&效率。

Instana 与 AWS 上的容器服务紧密集成，如亚马逊弹性容器服务(亚马逊 ECS)、亚马逊弹性 Kubernetes 服务(亚马逊 EKS)、亚马逊 AWS Fargate 和 Lambda。在 https://instana.com 了解更多信息。
