# 为什么测试如此令人沮丧

> 原文：<https://devops.com/devops-unbound-ep-15-why-testing-is-so-frustrating/>

在这一集[devo PS unbounded](https://digitalanarchist.com/videos/devops-unbound/)中，主持人 Alan Shimel 和 Mitch Ashley 与 Tricentis 的 Adam Kalsey、Linux 基金会的 Brian Dawson 和 Cobalt 的 Caroline Wong 一起讨论为什么[测试](https://devops.com/?s=testing)被认为是开发人员和工程师最沮丧的瓶颈。小组成员将解释在 SDLC 早期进行测试的好处，如何通过自动化和持续测试来加快反馈周期，测试的利弊，左移测试的价值以及如何有效地平衡质量和速度。视频在下面，后面是文字记录。
