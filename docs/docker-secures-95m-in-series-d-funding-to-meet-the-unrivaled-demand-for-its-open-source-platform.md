# Docker 获得了 9500 万美元的 D 轮融资，以满足其开源平台无可匹敌的需求

> 原文：<https://devops.com/docker-secures-95m-in-series-d-funding-to-meet-the-unrivaled-demand-for-its-open-source-platform/>

> *新兴的生态系统在 Docker 上实现标准化，改变了应用在现代数据中心的构建、运输和运行方式*

**旧金山——2015 年 4 月 14 日——**[分布式应用开放平台背后的组织 Docker，Inc.](https://www.docker.com/) 今天宣布，它已经筹集了 9500 万美元的 D 轮资金。这一最新注入建立在 9 月份成功筹集的 C 轮融资的基础上，使 Docker 能够满足数百万生态系统用户和数千家已标准化 Docker 技术的企业的需求。此次融资将实现与 AWS、IBM 和微软等走向市场合作伙伴的更深入整合，这些合作伙伴已经进行了战略产品投资，以支持 Docker 的开源技术。它还将在应用程序开发生命周期中提供新的平台功能，巩固 Docker 作为新一代分布式应用程序的基础。

这轮融资由 Insight Venture Partners 牵头，Coatue、高盛和北方信托提供了新的投资。现有投资者 Benchmark、Greylock Partners、红杉资本(Sequoia Capital)、Trinity Ventures 和杨致远的 AME 云创投也参与了这轮融资。

Insight Venture Partners 的联合创始人杰夫·霍林(Jeff Horing)表示:“此次融资是对 Docker 管理的一次强有力的信任投票，加强了公司满足数百万开发者需求的能力，这些开发者承诺在日常工作中使用其突破性的产品和服务。“Docker 将能够继续进行额外的招聘，将更多急需的产品交到开发者手中。”

Docker 的模型使任何应用程序都能够作为一个一致的、低开销的 Docker 容器来构建和运行，从而将开发人员从基础设施的担忧中解放出来，并极大地增加创新。使用 Docker 的企业报告称，它将部署周期从数周缩短到了几分钟，并将计算资源效率提高了 20 倍。Docker 技术的超过 3 亿个实例已从其托管服务 Docker Hub 下载，使组织能够创建敏捷应用程序环境和敏捷组织。

Docker 的创始人兼首席技术官所罗门·海克斯(Solomon Hykes)说:“我们的责任是为人们提供他们需要的工具，以创建以前不可能实现的应用程序。“我们将继续履行对开发商和企业的承诺。我们认为他们仍然在寻找一个平台，帮助他们以真正标准化的方式构建和发布应用程序，而没有锁定或不必要的捆绑功能。这是我们着手建设的，我们对迄今取得的成就还不满意。我们从市场得到了一个明确的信息，他们喜欢我们正在建造的东西，我们计划继续建造它。这笔融资让我们能够兑现这一承诺。”

Dockerized 应用程序正在为 BBC、Gilt、高盛、Groupon、ing、Spotify 和 Yelp 等组织带来更高的生产力和持续的业务创新。像这样的公司通常最初由几个开发团队接触 Docker，这些团队将 Docker 集成到他们的工作流中，并能够立即在整个开发组织中展示该平台的生产力优势。这些基层的努力很快得到了管理层的关注，应用程序增强几乎一夜之间只需几分钟，而以前需要几个月。

高盛(Goldman Sachs)技术部门全球联席主管唐·杜特(Don Duet)表示:“随着金融行业数字化步伐的加快，我们软件和基础设施服务的广度、复杂性和多样性都有了实质性的增长。“在灵活、适应性强的生态系统中管理和运行业务关键型技术的能力已经成为我们公司的基础。当我们的工程师发现并开始使用 Docker 的开源平台时，他们立即被它提供的应用程序的可移植性所打动。它激励我们向基于 Docker 技术的包装、运输和运行应用程序的标准化基础设施迈进。”

【2015 年 4 月，Enterprise Technology Research 对 685 名企业首席信息官进行了调查，结果显示，Docker technology 的购买意向得分(即净支出意向得分或“净得分”)是该公司六年调查以来最高的，调查涵盖了企业技术 24 个领域的 274 家技术供应商。这与 Docker 在其旗舰商业产品 Docker Hub Enterprise 上收到的积极反馈相一致，该产品正在由许多财富 100 强公司进行 beta 测试，并将在本季度晚些时候广泛提供。

“强大的产品需求和快速扩张的生态系统相结合，使 Docker 的增长轨迹远远超过了大多数公司在这一发展阶段的经历，”Docker 首席执行官 Ben Golub 表示。“这种类型的资金支持，加上这群投资者的高素质，为 Docker 提供了所需的资源和指导，以满足我们社区各方面的需求——从开源贡献者到财富 100 强企业。在过去的六个月里，我们已经超越了为公司和社区设定的每一个增长里程碑。在我们的投资者、合作伙伴和社区的持续支持下，Docker 将继续打破组织惯例，将单一团队、流程和应用转变为更具生产力和灵活性的微服务环境。”

**关于 Docker，Inc.**
