# IT 运营趋势:云、容器和 COVID

> 原文:[https://devo PS . com/it-ops-trends-cloud-containers-and-covid/](https://devops.com/it-ops-trends-cloud-containers-and-covid/)

几乎所有公司都在对其 IT 基础架构进行现代化改造，以保持竞争力，并坚定地关注向云的迁移和动态基础架构的使用。在最近的[IT Ops 2021 趋势调查](https://checkmk.com/company/newsroom/trends-it-ops-2021)中，160 多名 IT 专业人员被问及 IT 战略的实施如何影响他们的日常工作。调查结果为系统管理员和 IT 运营人员当前的工作重点提供了深刻的见解。

根据调查，大多数管理员认为，监控云(79%)和容器(73%)环境将在未来扮演更重要的角色，或者说是更重要的角色。此外，59%的受访者认为，由于云计算和容器化，他们的工作角色已经发生了变化。

### 并非所有的 IT 团队都是平等的

然而，数据还显示，这些变化对每个 IT 团队的影响并不相同。例如，37%的受访者表示云解决方案与他们的日常生活不相关，或者不太相关。这种回应表明，一方面，[系统管理员](https://devops.com/?s=system+administrators)将云计算视为必不可少的话题。然而，另一方面，每个行政人员在实践中感受到的影响程度不同。这些回应表明，并非每个管理员都必须每天处理云平台，有些管理员被“孤立”在专门的团队中，可能不会与更广泛的企业云战略互动。此外，并不是每个组织都完全致力于云，有些组织发现他们的迁移受到不适合云迁移的应用程序的阻碍。

在调查中，45%的受访者还表示，他们认为监控物理基础设施的重要性在未来几年不会改变。然而，43%的人认为监控物理系统将变得更加重要或重要得多。相反，只有 1%的人认为物理基础设施监控不太重要。

综上所述，这些回答表明，尽管人们对云感到兴奋，但许多组织仍采用混合方法来实现基础架构。IT 运营和管理员认识到监控云系统和动态资产的重要性，同时，本地系统的监控仍然相关，甚至有望变得更加相关。

### 网络管理和自动化

云和容器能够加快 IT 环境的发展，促进数字化。然而，它们也导致数据流量的急剧增加和更复杂的系统联网。由于这些变化，对本地网络的需求也在增长。这种不断增长的需求证明了网络管理工具的重要性。81%的受访者表示，这方面的解决方案与他们的日常工作相关或非常相关。

今年，转向家庭办公室加入了向云、容器和网络监控迁移的趋势。在新冠肺炎疫情之后，许多 IT 部门不得不调整他们的基础设施，以允许员工在家办公。然而，大多数管理员相对来说没有压力，54%的人说他们在改造基础设施以满足远程工作需求方面没有问题。

自动化也是一个关键问题。例如，81%的受访者表示，在过去 12 个月中，交付部署和配置自动化的项目与其组织相关。此外，92%的人说自动化工具与他们的日常工作相关，或者非常相关。

### 云、容器和 COVID

《2021 年 IT 运营趋势》调查显示，系统管理员关心云计算和容器以及其他几个话题。此外，大多数管理员认识到监控云环境和容器越来越重要。也就是说，管理员仍然认为监控网络和其他物理基础设施资产至关重要，一些系统管理员认为监控这一重要领域越来越重要。

除了这里概述的主题，受访者还就虚拟化、应用性能监控和网络管理等主题提供了反馈。
