# Instaclustr 推出托管开源即服务平台

> 原文:[https://devo PS . com/instaclustr-launchs-managed-open-source-service-platform/](https://devops.com/instaclustr-launches-managed-open-source-service-platform/)

该平台提供管理和集成的 Apache Cassandra、Apache Spark、Elasticsearch、Kibana、Lucene 和 Zeppelin 作为 100%开源、零锁定的解决方案

加利福尼亚州雷德伍德城——2017 年 9 月 11 日——Instaclustr 提供作为完全托管解决方案交付的可扩展开源技术，今天宣布推出其开源即服务平台。这个全面的平台为各行各业的客户(从初创公司到企业)提供完全托管和安全管理的 Apache Cassandra、Apache Spark、Elasticsearch、Kibana、Lucene 和 Zeppelin。每个都以 100%开源的形式交付给客户，绝对没有供应商或技术限制。随着该公司继续实现超过 100%的年增长率，并达到 1000 万节点小时和 1pb 管理数据的里程碑，该平台应运而生。

Instaclustr 一直提供受管开源解决方案，运行一个自动化、成熟的管理环境，客户信赖该环境可提供大规模可靠性。现在，随着其新平台的推出，Instaclustr 通过致力于部署 100%开源技术，正式做出了这一承诺。在这个行业中，提供商往往会将开源解决方案重新包装成专有版本，以促进供应商锁定，Instaclustr 正在确保其提供的每个解决方案始终由完全可移植的开源代码组成。

在专有锁定情况下，企业发现自己受制于价格上涨，没有能力自己运行解决方案或修改代码以适应特定情况，并且面临错误和其他问题的解决速度较慢。相比之下，Instaclustr 的新平台让组织处于控制之中，并有助于降低开源技术投资的风险。客户能够利用和定制其托管开源解决方案，同时还提供了企业专注于构建其应用程序所需的支持和专业知识。这也意味着客户可以获得真正开源技术的所有好处，包括能够利用使用完全相同的开源代码库的开发人员和公司的巨大社区，并比专有产品更快地提供保证和改进。同时，Instaclustr 管理版本控制和补丁，以确保客户始终运行最新、最稳定的开源解决方案版本。

Instaclustr 的开源即服务平台提供数据库、分析、搜索和消息传递技术，这些技术因其提供可扩展性、可靠性和性能的能力而得到认可。每个解决方案都有 Instaclustr 的 100%正常运行时间保证，并提供全天候专家支持。Instaclustr 的开源即服务平台包括:

Apache Cassandra——在 AWS、Azure、GCP 和 Softlayer 上托管和完全管理 Apache Cassandra。具有连续可用性体系结构，该体系结构通过设计防止故障，并且能够轻松扩展，在几分钟而不是几小时内解决工作负载增加的问题。

Apache Spark——一个在内存中处理数据的集群计算框架，旨在提供速度和性能。

elastic search with Elassandra——一个快速、强大的索引和查询功能，利用 Cassandra 的弹性和分布式架构。

ki Bana——一个强大的用户界面，为开发人员提供了通过搜索来探索、分析和可视化数据的能力。

Lucene——一个高性能、多功能的文本搜索引擎，为需要全文搜索的应用程序而设计。

zeppelin–基于网络的笔记本电脑，提供强大的交互式数据分析。

Instaclustr 首席执行官 Peter Nichol 表示:“我们的开源即服务平台的开发是 Instaclustr 的自然发展。“我们一直致力于为客户赋权，提供急需的托管和管理解决方案，让客户掌控自己的命运和自己的核心技术。随着这个版本的发布，我们很自豪地正式承诺 100%开源解决方案。我们比以往任何时候都更容易从这些作为服务交付的强大技术中受益，并且没有专有解决方案或商业初创公司烘焙单个开源项目所带来的任何附加条件。”

IBM 数字战略和互动高级管理顾问 James Amos 表示:“我们依靠 Instaclustr 提供的开源技术一次又一次地证明了其巨大的优势，使我们能够将内部资源集中在产品开发和其他业务需求上。“Instaclustr 以完全透明的方式提供服务和解决方案的原则为我们带来了开源的所有优势，没有宕机、锁定或意外。”

关于 Instaclustr

Instaclustr 是一家开源即服务公司，提供大规模的可靠性。我们运营着一个自动化、成熟且值得信赖的托管环境，提供数据库、分析、搜索和消息传递。我们使公司能够将内部开发和运营资源集中在构建面向客户的尖端应用程序上。
