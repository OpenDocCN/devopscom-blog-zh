# 感恩节:你感谢什么？

> 原文:[https://devo PS . com/a-devo PS-thanksgiving-what-you-thanking-for/](https://devops.com/a-devops-thanksgiving-what-are-you-thankful-for/)

虽然看起来很疯狂，但我们正处于另一个感恩节的前夕。今年过得很快，随着我们进入假期和 2022 年底，我们这些 DevOps 社区的人有很多要感谢的。首先，我很高兴再次在世界各地的会议和活动中见到我的同事和朋友。我期待着不仅在美国，而且在世界其他地方恢复旅行，并看到 DevOps 爱好者分享他们的知识，他们的经验和进一步发展 DevOps 的状态。

虽然我们感谢恢复了更正常的生活方式，但我们也需要停下来承认和哀悼那些在这场可怕的疫情中失去的人。他们的生命和在过去这些年里从我们所有人身上夺走的时间是永远无法替代的。

他们说每一片乌云背后都有一线光明；如果说在新冠肺炎疫情期间 DevOps 有一线希望的话，那就是前所未有的数字化转型步伐。毫无疑问，数字化转型是由软件开发推动的，而 DevOps 是软件开发的动力。不仅仅是过去几年中开发和部署的应用程序或代码行的数量，而是软件供应链软件管道的加速成熟，这已成为软件开发和部署的事实上的标准；由于推动数字化转型，这一速度大大加快。

加速的不仅仅是软件开发，软件供应链攻击也带来了全新的安全子集。软件供应链安全和 SBOMs 现在是安全词典中的日常词汇。虽然我对这种全新类型的攻击并不满意，但我很感激白宫和美国政府对我们整个行业应对软件供应链攻击的回应。我也很感激围绕 DevOps 的工具和技术、人员和流程的持续发展。

虽然有些人希望通过宣布 DevOps 已死和其他类似的废话来赚钱，但事实是 DevOps 正在蓬勃发展。这就是今天软件开发的方式。事实上，结合云原生，我们似乎有了一个全新的堆栈和一种全新的软件创建方式。

展望未来，我们将继续看到人工智能的发展对可观察性、测试和应用管理产生真正的影响。我认为我们也即将迎来对软件开发产生深远影响的人工智能。与人工智能配对的低代码/无代码开发工具有望承担一些最普通的软件开发任务，并使它们自动化。这将解放超负荷工作的软件开发人员，使他们专注于更高价值的软件开发任务。

尽管宏观经济状况抑制了整个科技领域(如果不是整个世界经济的话)，我仍然对 DevOps 的持续增长和成熟充满热情，我很高兴看到未来几周、几个月和几年将会发生什么。我很高兴能在这个令人兴奋的时期加入我们的行业，我也很感谢你、我们的读者、支持者和社区对我们公司的持续支持。

为此，我想宣布我们今年在 Techstrong 集团尝试一些新的东西，我们希望你能加入我们。我们想知道你认为是什么让科技变得强大——我们要求人们用#Techstrong 标签录制一个简短的抖音视频或 LinkedIn 视频。我们将选出获胜者，并向评委选出的最佳两个参赛作品赠送两(2)台索尼 PlayStation5 游戏机。你可以在推特、T2、脸书和 T4 的社交媒体渠道了解所有细节。

所以，现在你知道我感激什么了——我们真的很想听听你感激什么，以及你认为是什么让科技变得强大！在整个假期和新的一年里，我们祝你健康快乐——并且知道技术将保持强大！
