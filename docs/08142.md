# Imply 以 7 亿美元的估值完成了 7000 万美元的 C 轮融资，扩大了作为动态分析基础平台的领导地位

> 原文:[https://devo PS . com/imply-closes-7000 万系列 c-at-7000 万-估值-扩展领导力-作为动态分析基础平台/](https://devops.com/imply-closes-70-million-series-c-at-700m-valuation-to-extend-leadership-as-foundational-platform-for-analytics-in-motion/)

*巨大的客户势头强调了从昂贵、缓慢和静态的分析解决方案向高度可扩展、经济高效和交互式分析解决方案的转变。*

**加利福尼亚州伯林盖姆，2021 年 6 月 16 日—** 由 Apache Druid 的原始创建者创建的动态分析先驱 Imply 今天宣布，由 Bessemer Venture Partners 牵头、Tiger Global Management 参与的 7000 万美元 C 轮融资结束，总融资额超过 1.16 亿美元。现有投资者 A16Z、Khosla Ventures 和 Geodesic Capital 也参与了此轮投资。这笔额外的资金是在市场对 Imply 和开源 Apache Druid 的采用不断增长的情况下取得的。Imply 现已超过 100 家企业和数字原生客户，使用 Imply 作为他们构建的基于分析的应用程序的引擎，和/或作为直接解决内部 BI 和运营分析用例的完整解决方案。额外的资金将扩大 Imply 在快速增长的动态分析领域的市场领导地位。

Bessemer Venture Partners 的合伙人 Elliott Robinson 表示:“分析世界正在被颠覆，因为现在等待数小时或数天来迭代见解、基于陈旧数据做出决策，或者利用要么崩溃要么大规模成本高昂的解决方案，都是一种竞争劣势。“每天都有越来越多的成熟企业和数字原生企业意识到动态分析的力量，并且在这场重新定义整个企业分析的竞赛中遥遥领先，在全球范围内拥有众多知名客户和成功的客户。”

动态分析是数字化转型的新要求，它的出现是由于现代业务的需求以及现有分析解决方案在满足这些需求方面的局限性。通过提供组织进行有效分析所需的所有四种属性，Imply 的独特技术消除了有效分析的障碍:

1.  **互动查询**:问题不需要预先确定，组织获得一个引擎和一个 app/UI，而不仅仅是一个数据库；可以即时创建查询，立即提供答案，并以思考的速度生成见解(亚秒级 OLAP 查询、自助数据分析、点击浏览集成 UI)
2.  **无限扩展:**专为扩展而构建(高查询并发性、PB+数据量、高接收吞吐量)
3.  **实时和历史数据加载**:洞察可以从动态数据中获得，现在可以与过去进行比较(Kafka/Kinesis 集成、数据湖集成、灵活和自适应模式)
4.  **最佳性价比**:以最大的成本效益进行扩展(可配置的性价比、可预测的性能和成本，根据使用情况进行微调)

这四项要求的结合实现了广泛的新客户成果，并将动态分析作为每个成功组织运营方式的定义特征。

**动态分析推动的新客户成果**

领先的组织利用动态分析作为:(1)以安全、运营和业务洞察为中心的分析驱动应用背后的分析引擎；以及(2)解决内部 BI 和运营分析用例的完整解决方案。
