适用于各种平台、公共云、网络和安全技术的现成可用、精心策划的自动化可帮助组织更轻松地开始使用最新的可信自动化

RALEIGH, N.C. — <time datetime="2020-06-23T00:00:00-04:00">June 23, 2020</time> —Red Hat, Inc., the world’s leading provider of open source solutions, today announced key enhancements to the Ansible Automation portfolio, including the latest version of **Red Hat Ansible Automation Platform **and **new Red Hat Certified Ansible Content Collections** available on Automation Hub. The latest release of Red Hat Ansible Automation Platform helps organizations expand automation to new domains while increasing productivity and cross-team collaboration. As a component of the latest platform release, new Ansible Content Collections developed, tested, and supported by Red Hat enable organizations to get the most up-to-date automation content.

Forrester Research 的一份新报告强调，“您的智能自动化路线图和成熟度将使您成为赢家(或输家)。” ¹ 随着全球形势的变化，自动化已经成为 IT 部门的首要任务，以帮助企业更快、更高效地发展。然而，为了充分利用自动化，IT 团队需要的不仅仅是自动化工具，他们需要更智能、更可靠的自动化技术。

red Hat Ansible Automation Platform 可帮助组织在需要时访问最新的 ansi ble 创新和技术，从而加快自动化工作，同时仍然提供生产就绪型工作负载所需的安全功能。除了 Ansible 内容集合，最新版本的 Red Hat ansi ble Automation Platform 还包括 Ansible Tower 3.7 的可用性、自动化服务目录的引入以及对 Automation Analytics 的更新。

**更快速地获得最新经过测试且值得信赖的自动化创新**
随着自动化需求的发展，组织需要能够扩展自动化功能，以帮助他们更快、更安全地扩展到新的领域。Ansible Content Collections 于 2019 年秋季与 Red Hat Ansible Automation Platform 一起首次发布，提供了一组持续更新的精选自动化内容，通过支持组织大规模重用 ansi ble 内容，帮助缩短价值实现时间，提高维护和使用自动化的效率。

随着 Red Hat 维护的 Ansible 内容集合的推出，Red Hat 正在发布 17 个新的集合，这些集合都是由 Red Hat 编写和支持的。集合可用于许多受欢迎的网络、安全和云合作伙伴，包括 AWS、Arista、Cisco、IBM、Juniper、Splunk、VyOS 平台等。该功能可以帮助组织更有信心地部署自动化，因为它已经过测试，并得到了 Red Hat 屡获殊荣的支持。集合是从 Ansible 产品版本异步发布的，这使得组织可以更容易地访问最新的可用集合。随着这些 Red Hat 编写的集合的增加，现在有 49 个认证集合可从 Red Hat 和 Automation Hub 中的合作伙伴获得。

在 cloud.redhat.com 可以找到受支持和认证的内容集合的完整列表。

**Red Hat 管理产品组合的深度集成**
除了扩展合作伙伴技术的自动化，Ansible Content Collections 还有助于进一步集成 Red Hat 的管理产品。新系列适用于:

> 面对不确定的全球环境，为了满足不断变化的业务需求，我们看到 IT 组织加快了自动化工作，以帮助推动更快的创新，同时管理远程工作人员带来的需求激增。最新发布的 Red Hat ansi ble Automation Platform 有助于进一步融合自动化创新和生产稳定性，为组织提供更大的信心，使他们能够采用新的自动化技术来扩大 IT 足迹，同时仍然支持任务关键型环境。

*   **Ansible Tower 内容收集**使组织能够大规模配置、管理和运营其 Ansible Tower 环境。通过该集合，用户可以创建和修改清单、凭据和 Ansible 塔作业模板，以及启动和监控 Ansible 塔作业。
*   **Red Hat Insights 内容集合**使组织能够通过使用 Red Hat Insights API 来自动化手动任务。这些常见任务包括安装和注册 Red Hat Insights 客户端，执行 Ansible 行动手册以进行问题补救，以及通过与 Ansible Tower 集成，自动执行合规性活动并充当动态库存来源。
*   **Red Hat Satellite Content Collection**使组织能够通过使用 Red Hat Satellite API 来自动化手动任务。用户可以从头到尾构建和配置卫星环境，通过 Ansible 提供完整的 Red Hat Satellite 端到端管理。

**更高级别的智能和更高的易用性**
除了 Ansible 内容集合，最新发布的 Red Hat Ansible 自动化平台还包括新的功能和增强功能，可帮助新用户和有经验的用户简化自动化，同时提高 Ansible 的速度和灵活性。这些更新包括:

*   **Red Hat Ansible Tower 3.7** 增加了新功能，帮助客户更快、更大规模地实现自动化。新的更新包括帮助消除作业处理中的瓶颈的调整，以及优化常见 API 查询的处理和重新设计用户界面通信协议以保持性能。
*   **Automation Analytics Automation calculator**是 Automation Analytics 的一个补充，它使用户能够评估自动化在其组织中的价值，并根据各自公司的数据进行定制。它可以衡量过去、当前和未来的投资回报，提供关于有效性的实时指标，以鼓励在更多领域更广泛地采用自动化。
*   **自动化服务目录**为自动化资源提供生命周期管理，涵盖物理、虚拟、云或基于容器的环境中的可用平台的配置、停用和日志记录。这提高了对自动化内容的可见性，并帮助组织维护其业务规定的控制和合规性。

Red Hat ansi ble Automation Platform 中的最新创新，包括 Ansible Content Collections，现已在 cloud.redhat.com 上推出。新系列将在滚动的基础上发布。
