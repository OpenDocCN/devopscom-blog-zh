全球超过 500，000 名开发人员使用 Appery.io ,使在低代码环境中创建混合移动应用、web 应用和渐进式 Web 应用(pwa)变得简单。最新版本 Ionic 4 为开发人员提供了选择在其上使用的工具和其他框架的灵活性，使其更易于管理。此外， [Appery.io](http://Appery.io) Ionic 4 框架支持最新的 Angular 版本 7 和 Cordova 8，并包括整体性能改进、现代 web 平台功能、CSS 变量等。 [Appery.io](http://Appery.io) 用户可以利用这些优势以及该平台提供的完整 MBaaS，并在 [Appery.io](http://Appery.io) 中构建一个 Ionic 应用。
