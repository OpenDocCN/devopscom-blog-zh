# VictorOps 宣布推出新的移动功能以改善通话体验

> 原文：<https://devops.com/victorops-announces-new-mobile-features-improve-call-experience/>

*新功能提升了可用性和性能，可随时随地提高工作效率*

科罗拉多州博尔德市–2017 年 11 月 15 日–[针对 DevOps 团队的实时事件管理公司 VictorOps](https://victorops.com/?utm_source=PR&utm_medium=Press%20Release&utm_vendor=Matter) 今天宣布为其 Android 和 iOS 应用程序提供新的移动功能，以帮助待命团队从任何地方访问解决事件所需的最相关的上下文。

“今天的员工队伍*是*移动的。不管一个人的角色是什么，在当今的竞争环境中，在任何地方工作的能力都是至关重要的。但对于随叫随到的团队来说，更迫切的需要是确保最短的问题解决时间和最大限度地减少停机时间，”VictorOps 产品副总裁 Joni Klippert 说。“我们希望使用 VictorOps 移动应用程序的随叫随到团队能够随时随地访问他们需要的信息，以便他们能够解决手头的事件，重新开始他们的生活。”

VictorOps 对移动应用程序进行了大量改进，以允许用户快速启动并运行。VictorOps 还在应用程序中实现了一个更直观的导航系统，旨在使用户能够更自然地与界面交互。

更多关键的移动发布功能包括:

*   **事件中的注释**–更快地访问与事件相关的元数据和图表意味着用户在半夜被呼叫时，对问题和应采取的步骤有了更多的了解
*   **集成图标**–包括集成徽标在内的改进视觉提示意味着用户一眼就能识别出行为不当的系统
*   **时间线**–随着数据源和通信在 VictorOps 应用程序中的增加，团队简化了时间线的外观，以漂亮的新格式呈现最具可操作性的数据
*   **聊天功能**–聊天已经过改进，图像处理更好，支持丰富的事后分析

“对于我们的客户来说，可达性和灵活性对于健康的工作生活平衡至关重要，”Klippert 补充道。“这些新的移动功能远远超出了寻呼，帮助待命团队更智能地工作，而不是更努力地工作，即使他们不在办公桌旁。”
