# 惠普企业为 HPE 软件公司推出新产品并建立合作伙伴关系，推动数据驱动的数字化转型

> 原文：<https://devops.com/hewlett-packard-enterprise-fuels-data-driven-digital-transformation-with-new-products-and-partnerships-for-hpe-ezmeral-software/>

*   HPE 将 HPE Ezmeral 数据结构作为独立产品推出
*   HPE 推出了新的 HPE 市场和技术生态系统计划，汇集了顶级的独立软件开发商和开源项目

## *HPE·埃兹迈拉公司为数据科学家、开发人员和 IT 人员提供边缘云解决方案，提供一个工业化的企业级平台，用于推动大规模人工智能、人工智能和数据密集型工作负载*

## 

**得克萨斯州休斯顿——2021 年 3 月 17 日**—惠普企业公司(纽约证券交易所代码:HPE)今天宣布对 HPE·埃兹迈拉软件组合进行改进，推出新产品和合作伙伴关系，旨在帮助组织统一从边缘到云的数据访问，以推动其数字化转型计划。总之，HPE Ezmeral 产品组合为客户提供了一个端到端的混合云分析平台，可提高其 AI/ML、分析和数据密集型工作负载的质量、可重复性、吞吐量和价值实现时间。今天的更新包括以下内容:

*   HPE 埃兹迈拉数据结构——除了作为 [HPE 埃兹迈拉容器平台](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.hpe.com%2Fus%2Fen%2Fnewsroom%2Fpress-release%2F2021%2F03%2Fnanya-technology-corporation-boosts-manufacturing-productivity-with-hpe-ezmeral-container-platform.html&data=04%7C01%7C%7Ce71182516c694d7b043508d8e948a4fb%7C84df9e7fe9f640afb435aaaaaaaaaaaa%7C1%7C0%7C637515845502250619%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C1000&sdata=eqTveRazDYHiDG%2Bj6LC3LrEhCkqePkcYqs4%2BPXEvzbw%3D&reserved=0)和 HPE 埃兹迈拉机器学习操作(ML Ops)的集成组件外，现在还可作为独立产品提供。HPE Ezmeral Data Fabric 带来了一个持久存储层，可简化内部、公共云、混合云和边缘环境中的整个机器学习模型生命周期。
*   一项新的 HPE Ezmeral 技术生态系统计划，其中包括行业领先的 ISV，如 Dataiku、MinIO、 [H2O。AI](http://H2O.AI) ，全神贯注。AI，Run:AI，Sysdig 和 unlauncher，它们已经在 HPE Ezmeral 上验证了它们的应用程序，允许客户更快速地创建简化的分析环境。
    面向客户的新 [HPE Ezmeral Marketplace](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.hpe.com%2Fus%2Fen%2Fsoftware%2Fmarketplace.html&data=04%7C01%7C%7Ce71182516c694d7b043508d8e948a4fb%7C84df9e7fe9f640afb435aaaaaaaaaaaa%7C1%7C0%7C637515845502250619%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C1000&sdata=WGzU%2BOW5oAQyNNOJIQj2JVDYbXF6rGG759dvS8ulHyY%3D&reserved=0) 将顶级 ISV 和开源项目(如 Apache Spark、Tensorflow 等)带给希望将其工作负载现代化为云原生的企业。

HPE 首席技术官兼软件负责人 Kumar Sreekanti 表示:“HPE·埃兹迈拉对我们的客户来说具有不可估量的价值，我们在新企业客户中的持续增长就是明证。“有效利用数据和人工智能的企业能够更好地在动态、不断变化的市场中快速发展。独立的 HPE Ezmeral Data Fabric 数据存储和新的 HPE Ezmeral Marketplace 为企业提供了他们选择的环境，并通过随处可见的开放、灵活的云体验提供了跨所有企业应用程序和数据的可见性和治理。”

企业正在寻求完善其数据分析能力和流程，以构建一个能够快速创建应用程序并推动计划的分析工厂，从而帮助他们的组织在市场中脱颖而出。为了实现这一目标，他们需要采用工业化的数据科学方法来支持许多用户、工具和步骤，以便提供端到端的自动化分析解决方案来提高质量和加快业务价值实现。

HPE Ezmeral Data Fabric 非常适合作为大规模定制应用程序的全局文件系统基础、分析应用程序的数据湖以及大规模容器环境的软件定义存储。它现在作为软件定义的横向扩展数据存储提供，有助于解决从边缘到云的数据密集型应用的挑战。此外，随着组织寻求实现其企业工作负载的现代化，HPE Ezmeral 生态系统将顶级开源项目和 ISV 聚集在一起，以访问企业喜欢的数据和工具，所有这些都在整个数据科学团队的一个控制平面中。

[HPE·埃兹迈拉](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.hpe.com%2Fus%2Fen%2Fezmeral.html&data=04%7C01%7C%7Ce71182516c694d7b043508d8e948a4fb%7C84df9e7fe9f640afb435aaaaaaaaaaaa%7C1%7C0%7C637515845502260613%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C1000&sdata=rSQoP4sPV2xbjMOYTjG5R3MyI056%2FB7dVOWLibYNatc%3D&reserved=0)软件产品组合是一个专门为数据科学和分析工作负载构建的混合云平台，可帮助企业大规模构建和加速其现代数据分析计划。它由一个完整协调的 Kubernetes 容器平台以及内置的持久存储层和用于数据科学工作流的 ML Ops 组成。HPE Ezmeral 产品组合在 HPE 边缘到云平台即服务战略中扮演着重要角色。结合 HPE 格林莱克云服务(可在任何地方提供灵活、低成本和一致的云体验)，客户可以使用 HPE Ezmeral 软件组合快速转变和更新其所有应用程序和数据，并简化分析工作负载，所有这些都在一个“即服务”模式中完成。[HPE Ezmeral Container Platform](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.hpe.com%2Fus%2Fen%2Fnewsroom%2Fpress-release%2F2021%2F03%2Forock-technologies-selects-hpe-ezmeral-software-and-hpe-greenlake-cloud-services-for-data-science-as-a-service-open-source-hybrid-cloud-and-big-data-storage-solutions.html&data=04%7C01%7C%7Ce71182516c694d7b043508d8e948a4fb%7C84df9e7fe9f640afb435aaaaaaaaaaaa%7C1%7C0%7C637515845502270608%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C1000&sdata=g65cewAZVodD8qt7GMqoPqFSwfA9X49GBYu%2BejHJRZE%3D&reserved=0)和 HPE Ezmeral ML Ops 现在可通过 HPE GreenLake 获得云服务，HPE 计划在未来将 HPE Ezmeral Data Fabric 作为 HPE GreenLake 服务提供。

HPE ezmaural Data Fabric 作为持久性存储设施运行，现在作为软件定义的横向扩展数据存储单独提供，并针对数据分析功能进行了优化。它可以跨越多个边缘站点、多个云以及混合环境，并通过可通过全局命名空间访问的一个实例进行管理。企业可以创建统一的数据存储库，供数据科学家、开发人员和 IT 人员访问和使用，并控制其使用和共享方式，从而为数字化转型奠定基础，实现新的数据源，并随着业务的增长而扩展。

[HPE Ezmeral Marketplace](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.hpe.com%2Fus%2Fen%2Fsoftware%2Fmarketplace.html&data=04%7C01%7C%7Ce71182516c694d7b043508d8e948a4fb%7C84df9e7fe9f640afb435aaaaaaaaaaaa%7C1%7C0%7C637515845502270608%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C1000&sdata=hw8lEhSRjBgtwJZTRyoMFb3qnQ%2B5HoPADDv4byTmP1s%3D&reserved=0) 提供了一套快速增长的经过 ISV 合作伙伴验证的全栈解决方案，使客户能够构建他们的分析引擎，包括 Dataiku 和 Run:AI，以及 Apache Spark 和 Tensorflow 等开源项目。HPE 计划每个季度都向 HPE Ezmeral 市场添加更多的 ISV 合作伙伴。该市场为客户的核心用例及工作负载提供了额外的生态系统堆栈支持，包括大数据和 AI/ML 用例。

Dataiku 首席执行官 Florian Douetteau 表示:“HPE ezmaural market place 让我们能够与着手实现数据分析和数据科学工作负载现代化的企业建立联系。“Dataiku 与 HPE Ezmeral 软件的结合将帮助我们的客户成功扩展和实施他们的机器学习项目，为他们的业务带来真正的影响。”
