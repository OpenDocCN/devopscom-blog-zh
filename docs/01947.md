Red Hat Linux 是大多数企业 IT 组织的软件食谱中的主食。当 IBM 收购 Red Hat 时，全世界的开发者和 IT 领导者都注意到了这一点。IBM 红帽会改变吗？Red Hat 会继续像今天这样运作吗？还是会被 IBM 的某个业务部门吸收并消失？IBM 红帽是否还会为 Kubernetes、JBoss、Fuse、OpenJDK、Eclipse IDE 等项目和工具做出贡献？这些都是重要的问题。

Brad Micklea 是 IBM 开发人员项目和工具的负责人，他加入了 DevOps Chat，谈论 Red Hat 作为 IBM 一部分的未来。Brad 传达了一个强烈的信息，即 Red Hat 将继续专注于 Linux、Kubernetes、Java、DevOps 等的开发人员社区和工具。

加入我们，与 Brad 一起探讨 Red Hat、DevOps 和开源开发工具的未来。
