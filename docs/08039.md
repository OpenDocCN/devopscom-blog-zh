# Qumulo 通过 AWS 上的 Studio Q 推出了云中的后期制作平台

> 原文：<https://devops.com/qumulo-unveils-post-production-platform-in-the-cloud-with-studio-q-on-aws/>

*Qumulo Studio Q 使使用 Adobe Creative Cloud 的创意团队能够在任何地方工作，以前所未有的速度快速创建、修改媒体内容并从中获利*

**华盛顿州西雅图市，2021 年 5 月 25 日—**qu mulo 是彻底简化混合云环境中企业文件数据管理的突破性领导者，今天宣布推出其新的 Studio Q Quick Start 产品，这是一个全面的后期制作平台，使创意团队能够在云中复制协作编辑制作流程。通过简单的 AWS CloudFormation 部署，工作室、广播公司和企业视频制作团队现在可以在 Amazon Web Services，Inc. (AWS)上运行 Adobe Premiere Pro 和 Adobe After Effects，消除内容创建工作流的阻塞，这在以前是次优的，或者在其他文件系统上是不可能的。

Qumulo Studio Q 将 Qumulo 文件数据平台的强大功能与 Teradici CAS PCoIP 技术相结合，为每个编辑器提供了一个独特的虚拟桌面，允许在共享媒体资产之间进行简单、轻松的协作，提供丰富、无损的用户体验，而不会造成中断。借助 Studio Q Quick Start，拥有高性能夹层编辑工作流的企业现在可以在云中快速部署一个完整的 Studio，用于协作式远程编辑团队。

“AWS 上的 Qumulo 为我们提供了超越对象存储所需的性能提升，以及在云中以任何规模使用我们高度活跃的文件数据的能力，”Cinesite 的首席运营官 Graham Peddie 说。“Qumulo 凭借其基于容量的消费选项，允许我们的业务线和 IT 团队在我们的数字项目上合作，而不会影响我们管理非结构化数据或控制成本的方式。”

随着制作工作室、企业视频编辑和广播公司将其创作工作流程转移到远程环境，Studio Q 通过使其变得极其简单来加速向云的过渡。客户现在可以利用 Studio Q 快速入门来选择预配置的亚马逊弹性计算云(Amazon EC2)实例，快速启动 AWS 上的整个 Studio Q 环境，将数百个手动程序减少到几个步骤，并允许视频编辑和艺术家立即开始运行。Teradici CAS 提供了在任何地方工作的灵活性，能够简单安全地访问 Studio Q，并提供出色的用户体验。

Qumulo 的产品副总裁 Ben Gitenstein 表示:“对视频制作的需求不断增长，再加上当今向远程工作的转移，这给媒体和娱乐行业的创意团队带来了前所未有的数据挑战。“Studio Q 是一个云解决方案，允许全球团队在 AWS 上运行 Adobe Premiere Pro，具有与本地工作站相同的性能。”

为了提供一条轻松的云之路，AWS 快速入门登录页面提供了两个平台服务选项。客户可以通过自己的 AWS 帐户选择 Studio Q 的自助服务模式，或者进行引导式销售体验，以测试编辑工作站。测试和开发路线将支持将当前工作站迁移到云的客户。希望参与引导式销售体验以测试其特定使用情形的客户可能有资格获得 AWS 积分作为测试的一部分。
