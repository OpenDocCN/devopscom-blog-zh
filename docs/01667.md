# 网络研讨会:sec DevOps:sec ops 和 devo PS 的联姻

> 原文:[https://devo PS . com/webinar-secdevo PS-the-marriage of-secops-and-devo PS/](https://devops.com/webinar-secdevops-the-marriage-of-secops-and-devops/)

我们现在看到了 SecOps 和 DevOps 的新“联姻”,这为推动组织内外的创新创造了全新的思维方式。开发运维团队和安全团队现在以同行的身份积极协作，而不是传统的请求者/批准者关系，并为安全专业人员提供一席之地。随着 DevSecOps 的兴起，我们开始真正重新定义如何将运营、工程和安全和谐地结合在一起，以实现无与伦比的成功。DevSecOps 通过做一些简单的事情来推动具有前瞻性思维的组织——促进看似矛盾的团队之间的协作，将他们不同的目标整合成单一的努力。

加入 DevOps.com 主编 Alan Shimel、获奖的 CTO 兼作家 Gene Kim、Intuit 的 DevSecOps 负责人 Shannon Lietz 和 Evident.io 联合创始人兼首席执行官 Tim Prendergast 的讨论，他们讨论如何加快安全智能以跟上不断更新的云环境，从而使团队能够更快地发现问题、更快地做出响应并更有效地保护他们的资源。
