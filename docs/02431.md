# 亚马逊 AWS 市场现已推出 Instana Automatic APM

> 原文:[https://devo PS . com/instana-automatic-APM-now-available-in-Amazon-AWS-market place/](https://devops.com/instana-automatic-apm-now-available-in-amazon-aws-marketplace/)

*AWS 用户可以无缝安装和购买自动应用程序监控和跟踪*

德国旧金山和索林根——2019 年 4 月 19 日

针对云原生应用的[自动应用监控(APM)](https://instana.com/application-management) 解决方案的领先提供商 Instana 今天宣布，其自动 APM 解决方案可在亚马逊网络服务(AWS)市场购买。AWS 用户现在可以通过他们的 AWS 帐户获得 Instana 来无缝管理他们的 AWS 应用程序和基础设施。

Instana 联合创始人兼首席运营官 Pete Abrams 表示:“无论 IT 组织是大是小，基于云的基础设施和应用程序都已成为部署战略的重要组成部分。允许这些组织为其云系统研究、购买和部署性能监控解决方案，可以提高监控启动流程的效率

只需几次点击，AWS 用户就可以访问 Instana 的自动 APM 和应用基础设施监控解决方案，将许可证添加到他们的 AWS 帐户，并开始监控性能。

与传统的监控解决方案不同，Instana 的自动 APM 解决方案可发现应用基础设施和服务组件，为应用的技术堆栈部署监控传感器，并跟踪所有应用请求，无需任何人工配置，甚至无需重启应用。该解决方案实时检测应用环境中的变化，调整自己的模型，并在几秒钟内可视化变化和对性能的影响。

新的 Instana AWS marketplace 条目包括一个免费试用全功能产品的选项，让用户有机会了解他们所有的 AWS 应用程序和基础架构的运行情况。Instana 的自动部署、发现和监控配置功能可在几分钟内实现应用程序依赖关系图和服务级别监控的可视化。

Instana 的自动 APM 包括对 100 多种不同语言、微服务和流行的亚马逊平台的监控和跟踪，包括 EC2、DynamoDB 和 AWS RDS 数据库、AWS SQS 消息传递、ELB 负载平衡和 AWS Kinesis 流。访问[http://www.instana.com](https://instana.com/)了解更多。
