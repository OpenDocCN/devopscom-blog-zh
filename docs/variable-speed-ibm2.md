# 网络研讨会:加拿大抵押住房公司打破 IT 孤岛，加快大型机开发

> 原文：<https://devops.com/variable-speed-ibm2/>

DevOps 鼓励采用精益和敏捷的开发实践，而不考虑平台。然而，在大多数组织中，大型机和移动/web 开发团队都在各自为政，导致了过多的技术平台和工具、分散的或手动的流程、导致不必要的延迟的技能短缺、团队之间的返工和摩擦，影响了组织响应业务需求的能力。

加入加拿大抵押和住房公司应用程序开发经理 Jeff Blackadar、Alan Shimel 和 IBM 专家的行列，探索各种 DevOps 策略、实践和解决方案，以增强协作、简化流程并利用现代工具吸引下一代大型机开发人员来加快大型机应用程序的开发和维护。
