# IBM 提高了沃森理解商业语言的能力

> 原文:[https://devo PS . com/IBM-advances-屈臣氏-理解商业语言的能力/](https://devops.com/ibm-advances-watsons-ability-to-understand-the-language-of-business/)

*–宣布 Project Debater 关键技术的首次商业可用性；*

–*集成到 IBM Watson 中的新功能有助于企业开始挖掘&分析人类语言中一些最具挑战性的方面*

纽约，2020 年 3 月 11 日 / [美通社](http://www.prnewswire.com/) / —商业人工智能领域的领导者 IBM(纽约证券交易所: [IBM](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=617918911&u=http%3A%2F%2Fwww.ibm.com%2Finvestor&a=IBM) )宣布推出几项新的 IBM Watson 技术，旨在帮助组织更清晰地识别、理解和分析英语中一些最具挑战性的方面，以获得更大的洞察力。

这些新技术代表了关键自然语言处理(NLP)能力的首次商业化，这些能力来自 IBM Research 的项目辩手，这是唯一一个能够就复杂主题与人类进行辩论的人工智能系统。例如，定义了一个新的高级情感分析功能，首次识别和分析[习语](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=316366685&u=https%3A%2F%2Fwww.merriam-webster.com%2Fdictionary%2Fidiom&a=idioms)和口语。像“几乎没有帮助”或“衣领下很热”这样的短语对人工智能系统来说一直具有挑战性，因为它们很难被算法发现。借助高级情感分析，企业可以开始使用 Watson APIs 分析此类语言数据，以便更全面地了解他们的运营。此外，IBM 正在从 IBM Research 引入理解业务文档(如 PDF 和合同)的技术，并将其添加到他们的 AI 模型中。

“语言是表达思想和观点的工具，也是信息的工具，”IBM 数据和人工智能总经理罗布·托马斯说。“这就是为什么我们从 Project Debater 获得技术，并将其集成到沃森中——使企业能够从人类语言中获取、分析和理解更多信息，并开始转变他们利用数据中编码的智力资本的方式。”

今天，IBM 宣布，它计划在全年将 Project Debater 技术集成到 Watson 中，重点是提高客户利用自然语言的能力:

A.分析—*高级情绪分析。* IBM 已经增强了情感分析，以便能够更好地识别和理解复杂的词汇方案，如习语(短语和表达)，所谓的情感转换器，即单词的组合，一起呈现新的含义，如“几乎没有帮助”这项技术将于本月集成到[沃森自然语言理解](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=1749126397&u=https%3A%2F%2Fwww.ibm.com%2Fcloud%2Fwatson-natural-language-understanding&a=Watson+Natural+Language+Understanding)中。此外，我们还宣布了一种新的分类技术，这种技术将使客户能够创建人工智能模型，可以更容易地对采购合同等商业文档中出现的条款进行分类。基于 Project Debater 基于深度学习的分类技术，新功能可以从数百个样本中学习，快速轻松地进行新的分类。它计划在今年晚些时候被添加到[沃森发现](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=3610854366&u=https%3A%2F%2Fwww.ibm.com%2Fcloud%2Fwatson-discovery&a=Watson+Discovery)中。

B.概要—*总结。* 这项技术从各种来源提取文本数据，为用户提供关于某个特定主题的言论和文字摘要。摘要的早期版本[在今年的格莱美](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=105949465&u=https%3A%2F%2Fdeveloper.ibm.com%2Farticles%2Ftimeless-ai-insights-at-the-grammys-changing-the-fan-experience%2F&a=leveraged+at+The+GRAMMYS)上被用来分析超过 1800 万篇文章、博客和简历，以产生对数百名格莱美艺术家和名人的微小见解。这些数据随后被整合到红毯直播流、点播视频和 www.grammy.com[的照片中，让粉丝们对当晚的主要话题有更深入的了解。计划在今年晚些时候将它添加到 IBM Watson 自然语言理解中。](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=3958938850&u=http%3A%2F%2Fwww.grammy.com%2F&a=www.grammy.com)

C.聚类—*高级主题聚类。* 基于从 Project Debater 获得的见解，新的主题聚类技术将使用户能够“聚类”输入的数据，以创建相关信息的有意义的“主题”，然后可以对其进行分析。这项技术计划在今年晚些时候集成到 Watson Discovery 中，它还允许主题专家定制和微调主题，以反映特定业务或行业的语言，如保险、医疗保健和制造业。

IBM 长期以来一直是 NLP 的领导者，开发了使计算机系统能够学习、分析和理解人类语言(包括情感、方言、语调等)的技术，准确性和速度不断提高。IBM 已经将其 NLP 技术通过沃森推向市场，该技术大部分诞生于 IBM 的研究中。诸如用于文档理解的[沃森发现](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=3610854366&u=https%3A%2F%2Fwww.ibm.com%2Fcloud%2Fwatson-discovery&a=Watson+Discovery)，用于虚拟代理的 IBM [沃森助手](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=1392883997&u=https%3A%2F%2Fwww.ibm.com%2Fcloud%2Fwatson-assistant%2F&a=Watson+Assistant)，以及用于高级情感分析的[沃森自然语言理解](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=1749126397&u=https%3A%2F%2Fwww.ibm.com%2Fcloud%2Fwatson-natural-language-understanding&a=Watson+Natural+Language+Understanding)等产品都注入了 NLP。

ESPN 梦幻足球使用沃森探索和[沃森知识工作室](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=1968109438&u=https%3A%2F%2Furldefense.proofpoint.com%2Fv2%2Furl%3Fu%3Dhttps-3A__nam04.safelinks.protection.outlook.com_-3Furl-3Dhttps-253A-252F-252Fwww.ibm.com-252Fwatson-252Fservices-252Fknowledge-2Dstudio-252F-26data-3D02-257C01-257CKevin.R.Ota-2540disney.com-257C4923d8eb55c94ed54c1e08d7c1190198-257C56b731a8a2ac4c32bf6b616810e913c6-257C1-257C0-257C637190185449091973-26sdata-3DMdC7nJ-252BKNOqRQ3KyLSheBUrok1-252F6-252FEfXCg-252FXX1AHqOE-253D-26reserved-3D0%26d%3DDwMGaQ%26c%3Djf_iaSHvJObTbx-siA1ZOg%26r%3DsabdztWu2x7hKgl9KSEnlXtfk42vo3MFpcQZfHCVcbs%26m%3D2aOLQTwfqIqtlICISGqEl_sRIg2lZl_c5sHo8kF8IRI%26s%3Da_ALVM7fYZN4MDQ2x4MQFQ-BchjCk_iFfEZl8uZOWNY%26e%3D&a=Watson+Knowledge+Studio)在赛季期间每天分析数百万足球数据源，为数百万[梦幻足球](https://c212.net/c/link/?t=0&l=en&o=2746414-1&h=3761097866&u=https%3A%2F%2Furldefense.proofpoint.com%2Fv2%2Furl%3Fu%3Dhttps-3A__nam04.safelinks.protection.outlook.com_-3Furl-3Dhttps-253A-252F-252Fwww.ibm.com-252Fsports-252Ffantasy-252F-26data-3D02-257C01-257CKevin.R.Ota-2540disney.com-257C4923d8eb55c94ed54c1e08d7c1190198-257C56b731a8a2ac4c32bf6b616810e913c6-257C1-257C0-257C637190185449091973-26sdata-3DO1dK12vgASXgrk-252F0XCQLPZe-252FbBe718wsbzwHk1Pnffw-253D-26reserved-3D0%26d%3DDwMGaQ%26c%3Djf_iaSHvJObTbx-siA1ZOg%26r%3DsabdztWu2x7hKgl9KSEnlXtfk42vo3MFpcQZfHCVcbs%26m%3D2aOLQTwfqIqtlICISGqEl_sRIg2lZl_c5sHo8kF8IRI%26s%3DXJ1mqW2OVN5funVAanLX93ik62xwFiB9jN3y09LedOs%26e%3D&a=fantasy+football)球员提供实时见解。通过处理自然语言，沃森识别新闻文章、博客、论坛、排名、预测、播客和推文的语气和情绪，涵盖了从更衣室见解到伤病分析的所有内容。《ESPN 幻想足球》在球员卡中展现了这些见解，这些卡片快照了每个球员的“繁荣”和“萧条”潜力，以及总结关于球员的正面或负面评论的“球员评论”部分。

跨国专业服务网络毕马威(KPMG)是四大会计机构之一，它与 IBM 合作创建了一个基于各种 Watson 服务的人工智能解决方案，包括 Watson 自然语言理解。这项技术使得公司更有效地识别、申请和保留潜在的 T4 所得税减免。该解决方案由毕马威(KPMG)开发，可以帮助客户增加他们获得的研发所得税抵免额，因为沃森技术能够快速审查更多文件，同时最大限度地减少对客户业务的干扰。

在过去的一年里，毕马威的客户看到了更多的 R&D 税收抵免的潜力，一些项目甚至看到审查的文件数量增加了 1000%以上。该解决方案帮助客户发现更多有资格获得额外所得税抵免的潜在活动，同时减少业务中断。因此，工程师和科学家可以在所得税合规活动上花费更少的时间，专注于创新的 R&D 工作。
