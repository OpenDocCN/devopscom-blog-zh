# 全球对以开发人员为中心的安全性的需求推动了 Secure Code Warrior 的持续增长

> 原文:[https://devo PS . com/global-demand-for-developer-centric-security-fuels-secure-code-warriors-continued-growth/](https://devops.com/global-demand-for-developer-centric-security-fuels-secure-code-warriors-continued-growth/)

**波士顿&悉尼&伦敦，2021 年 8 月 24 日—** 全球安全编码公司[Secure Code Warrior](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fsecurecodewarrior.com%2F&esheet=52480439&newsitemid=20210823005280&lan=en-US&anchor=Secure+Code+Warrior&index=1&md5=3d85beb2ad19e0cc352fc9388c498378)^(宣布与第 400 位客户签约，并取得了强劲的业务表现，以应对全球各行各业对安全意识和安全技能开发人员不断增长的需求。)

这一里程碑式的成就是在 2021 年上半年 30 个国家的 120 家新企业客户加入后取得的，其中包括主要的全球零售商、技术巨头和制造业领导者，同时授权用户数量增加了一倍，客户数量增加了 30%以上。今年 3 月，Secure Code Warrior 签署了公司历史上最大的一笔客户交易，使另外 33，000 名开发人员加入了定制的学习路径，旨在帮助他们更快地交付安全代码。

最近的客户收购意味着 Secure Code Warrior 正在帮助全球 400 家组织提高开发人员的软件安全技能，包括 Zoom Video、Reddit、Mastercard、BBC、DBS Bank、新加坡政府、TotalEnergies 和 Blue Prism。

Secure Code Warrior 首席执行官兼联合创始人 Pieter Danhieux 表示，面对日益增长的全球网络威胁，公司的增长是由优先考虑软件安全的组织推动的，并反映了世界显示出后 COVID 复苏的迹象。

“我们已经看到，像网络安全管理软件产品这样的软件相关违规行为的持续影响，以及对关键基础设施攻击的增加，影响着全球各地的政府政策和业务重点。Danhieux 说:“拜登的行政命令和澳大利亚基于州的网络安全战略凸显了增强开发人员安全技能的需求，而不是在违规‘补救’后实施权宜之计。

“培养具有安全意识的开发人员的必要性贯穿各个行业，我们正在与领先的组织合作，帮助保护在我们的世界中做出惊人事情的软件——从需要保护国家资产的政府机构到保护汽车和其他领域互联系统的汽车制造商。”

Secure Code Warrior 的旗舰[学习平台](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.securecodewarrior.com%2Fproducts%2Ftraining-ground&esheet=52480439&newsitemid=20210823005280&lan=en-US&anchor=Learning+Platform&index=3&md5=19f26f71f956e021e478d99dd93cf966)是其成功的关键，其 85%的客户群接受了 2020 年底推出的“任务”产品，该产品通过高度相关的沉浸式编码模拟现实世界的软件漏洞，培养开发人员安全第一的思维模式，他们需要在安全的环境中通过安全编码来解决这些漏洞。这种扩展一直专注于开发新的挑战，并加强了其平台，以支持近 60 种语言:框架，包括遗留系统、前沿系统和专业系统。

Secure Code Warrior 董事会非执行董事、硅谷资深人士吉姆·普弗拉格(Jim Pflaging)高度评价了该公司在全球疫情中的强劲增长。

“在过去的 18 个月里，企业迅速加快了数字化转型的步伐。现在，增长和盈利能力与通过引人注目的软件应用吸引客户和供应商密切相关。在同一时期，一系列高调的网络攻击暴露了软件开发实践中的弱点，拜登政府的行政命令正在通过声明安全编码应该是任何政府或商业实体的基础来打击这些弱点。因此，全球领先的组织正在与 Secure Code Warrior 合作，以“左移”并降低风险，同时通过提高软件开发过程中最高效阶段的安全性，使团队能够更快地交付高质量的代码，”Pflaging 说。

为了支持其下一章的增长和规模，Secure Code Warrior 做出了几项重要任命，以建立和加快其上市战略，包括 Junie Dinda 担任其首席营销官，Vikram Ghosh 担任新设立的战略联盟高级副总裁。

Dinda 在旧金山工作，在 Atlassian 担任了六年多的营销领导职务，最近担任的职务是企业和迁移营销副总裁。她将推动产品导向型增长的丰富知识带入了开发者生态系统，并在将高速增长的企业从 50 名员工扩展到拥有 5，000 多名员工的 10 亿美元级组织方面积累了丰富的专业知识。

Ghosh 在 Chef Software 担任业务开发副总裁四年后加入，此前曾在 Citrix 和微软担任高级领导职务。Ghosh 位于西雅图，负责引领 Secure Code Warrior 的[全球合作伙伴计划](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.securecodewarrior.com%2Fpress-releases%2Fsecure-code-warrior-unveils-enhanced-global-partner-program-adds-direct-integration-into-global-devsecops-ecosystem&esheet=52480439&newsitemid=20210823005280&lan=en-US&anchor=global+partner+program&index=4&md5=3ecd4bae109e2aca38e63462a0697130)的发展，该计划最近宣布了重大改进，并扩大了与领先 DevSecOps 供应商的集成。
