你将看到的几乎每一个 DevOps 演示都提到了她的[加速 DevOps 状态报告](https://cloud.google.com/devops/state-of-devops/)。而且理由很充分。它已经成为 DevOps 指标的“权威”。

我们和 Nicole 一起讨论了她认为今年报告中的一些关键发现。听一听，享受一下。
