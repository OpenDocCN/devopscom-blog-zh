# 西南航空:“可耻的”技术债务反咬一口

> 原文：<https://devops.com/southwest-technical-debt-richixbw/>

欢迎来到[![](img/cf2fef39ade11b1de79090d8fbbb753a.png)](https://devops.com/tag/the-long-view/)*——在这里我们细读本周新闻，并将其剥离至要点。让我们弄清楚**什么是真正重要的**。*

## *20 年的忽视导致了“崩溃”*

*上个月取消航班的灾难是由几十年的技术债务造成的。这是哥伦比亚大学教授 Zeynep Tufekci 的分析。*

 *### 分析:西南威尔士州需要一个云爆发

尽管有几个促成因素，但一个关键的机组人员调度系统缺乏可扩展性导致了几天近乎完全瘫痪的状态:在许多情况下，工作人员都在正确的地方驾驶和管理飞机，但 *SkySolver* 系统无法知道这一点。更糟糕的是，手动回退在工作量的**重压下崩溃了。**

**有什么故事？**Zeynep Tufekci 教授向主流观众解释技术债务——“[西南航空失败背后可耻的公开秘密](https://www.nytimes.com/2022/12/31/opinion/southwest-airlines-computers.html "read the full text")”:

**`Technical debt is real debt`**
一段时间以来，在西南航空公司内部，这已经是一个公开的秘密……该公司迫切需要将其调度系统现代化。…这个问题——依赖于需要更新的旧软件或有缺陷的软件——被称为招致技术债务,[它]似乎是西南航空公司在上个月的重大冬季风暴后无法像其他航空公司那样恢复正常业务的一个关键因素。
……
当打嗝或天气事件发生时，员工们不得不经历一个繁重、艰难的过程……因为西南航空公司还没有充分实现其机组人员调度系统的现代化。例如，如果……他们的航班被取消了……员工不得不手动打电话通知公司他们在哪里,[有时]为了让公司知道他们的下落，他们不得不在电话上等待……小时。…在线论坛上充斥着员工对这种痛苦的描述。
…
这很容易导致系统范围的停机。…此类由技术债务导致的故障通常是由天气等外部事件引发的。…那么，为什么西南航空不简单地更新其软件和系统呢？…更新软件既昂贵又困难。(但)我们不能继续让越来越多的基础设施和我们的生活依赖于过时的软件。…技术债务是真正的债务。最终会有人买单的。


**和**安德鲁·保罗知道该归咎于什么—“[过时的软件](https://www.popsci.com/technology/southwest-airlines-flight-cancel-holidays/ "read the full text")”:

据报道，许多疲惫的旅客仍在等待与他们丢失的行李团聚，而其他人则继续表示困惑，美国最受欢迎的航空供应商之一怎么会发生这样的物流噩梦。答案是:…与 Xbox 360 和 PlayStation 3 几乎同时推出的员工日程安排软件。…据报道，至少从 2015 年开始，西南航空公司的飞行员就恳求公司高管更新“过时”的系统。
…
西南航空公司长期依赖名为 SkySolver 的机组人员调度程序是主要原因。…这个近 20 年的项目无法[扩展]应对多次取消和延误。这使得西南航空公司的员工不得不手动尝试将空勤人员与可用的飞机进行匹配。
…
尽管西南航空前首席执行官加里·凯利在 2021 年称赞了该航空公司的“出色技术”，但由于其迟迟未能采用…基于云的数据系统，该公司已经经历了多次物流崩溃，该系统可以处理像冬季风暴一样大的危机。…除非像西南航空这样的公司真正开始投资将他们的软件带入现代时代，否则旅行者将继续面临未来的难题。


**你好，它。** Tobias Mann 问道，“[他们有没有尝试过关闭然后再打开？](https://www.theregister.com/2022/12/28/southwest_outdated_it/ "read the full text")”:

在过去的一年中，一些航空公司已经宣布对他们的 IT 基础设施进行全面升级，部分是为了避免像西南航空公司正在挖掘出路的陷阱。…达美航空宣布将实现工作负载的现代化，并将其迁移至亚马逊网络服务。
…
美国航空公司更进一步，宣布与微软建立类似的合作关系，将其数据仓库和传统应用程序迁移到 Azure 上的一个单一运营中心，以加快包裹跟踪，实现基于天气条件的抢先改道，并使用数字双胞胎模拟更大的变化。合作还包括开发一个移动应用程序，使航空公司员工无论在哪里都可以访问应用程序并进行交流。


但是我们真的在谈论债务吗？ [jgwil2](https://news.ycombinator.com/item?id=34221344 "read the full text") 试着拆包翻译:

技术债务在这里可以得到更好的解释。我认为问题是非常老的软件系统往往有很多技术债务，这可能导致增加新功能的阻力。在这种情况下，技术债务似乎导致了一个脆弱的系统，无法更新以处理超过基线水平的流量，迫使该公司在危机发生时诉诸手动解决方案。

我们需要更多的监管吗？ [skogs](https://tech.slashdot.org/comments.pl?sid=22657856&cid=63171194 "read the full text") 认为不是:

我不认为我们需要一些愚蠢的政府监督来解决这个问题。…比一千个愤怒的电话和政府的书面报告或罚款更有用的是，他们在一年中最忙的时候实际上赚了零美元。他们赔钱:
不动的飞机不赚钱。
退款。
酒店报销。
遗失行李递送。
…
这是行业在自我修正。如果你搞砸了，你的钱包就会受损。忽略它，以后会更痛。然后西南航空干脆破产，把他们所有的资产出售给一个合理的公司，这个公司可以…妥善管理。


**然而，** [格伦·里博斯基](https://nyti.ms/3CeKWaQ#permid=122308917 "read the full text")根本不同意:

任何自尊的经济体都不能仅仅依赖不受监管的自由放任资本主义。……需要对经济行为的所有方面进行法律制衡。即使它们偶尔会挫伤那些企业家的动物精神。

所以这是管理上的失误？西航飞行员[拉里·洛内罗](https://www.facebook.com/1315600950/posts/what-happened-to-southwest-airlinesive-been-a-pilot-for-southwest-airlines-for-o/10229509585826782/ "read the full text")不是粉丝:

我在西南航空公司当了 35 年的飞行员。…不幸的是，一线员工已经看着这场灾难像慢动作火车残骸一样降临了一段时间。我们一直恳求我们的领导层做出急需的改变。
……
我们沮丧而难以置信地看着我们曾经令人惊叹的航空公司变成了纸牌屋。在 2010 年中后期发生了六次小规模的熔毁…我们可以看到车轮快要从巴士上掉下来了。但是领导层中没有人会理会我们的请求。
…
【但是】2022 年初，鲍勃·乔丹被任命为 CEO。他是一个更注重操作的领导者。他用一个非常聪明的人取代了我们的首席运营官，他们宣布他们的首要任务是升级我们航空公司的技术，并为一线员工提供我们需要的运营工具来照顾我们的客户和员工。…但是二十年的忽视需要几年时间来克服。…这次崩溃不是他的失败，而是他的前任的失败。


那听起来是真的吗？是的，[说 u/untgraded](https://reddit.com/r/sysadmin/comments/zyetmo/35year_southwest_airlines_pilot_beancounter_ceo/j29eqkz/ "read the full text"):

大约 30 年后，我的父亲作为一名上尉从南航退休。…当我问他时，这也是他的看法。

我看过他们的竞价/调度软件。如果这就是涂了口红的系统的样子，我敢肯定后端是…值得一看的。


**从驾驶舱门的另一侧，**这个[匿名懦夫](https://forums.theregister.com/forum/all/2022/12/28/southwest_outdated_it/#c_4593217 "read the full text")同意道:

赫伯·凯勒赫在经营西南航空公司的几十年里，建立了世界上最好的航空公司，无人能出其右。…后来，赫伯退休了，不久后，这个名叫加里·凯利的会计接管了公司。…赫伯一死，凯利就像对待几乎所有其他大航空公司一样对待员工。…然后他在去年放弃了，把这个大烂摊子交给了…指定的替罪羊…鲍勃·乔丹。
……
我最同情的人是员工。…看着他们为之奉献一生、为之努力的公司被某个贪婪、卑鄙、毫无价值的会计彻底摧毁。

**与此同时，** [本地 ID10T](https://tech.slashdot.org/comments.pl?sid=22657856&cid=63171230 "read the full text") 提供了这个简洁的总结:

技术债务越积越多，因为花费开发人员的时间来解决问题是要花钱的。所以他们被忽视了。…每个企业都有技术债务的问题。大多数人只是运气好，不会被当众打脸。
