# 水母为工程领导者推出行业首个比较基准工具

> 原文：<https://devops.com/jellyfish-launches-industrys-first-comparative-benchmarking-tool-for-engineering-leaders/>

*水母基准添加匿名化第三方性能数据；允许工程领导根据对跨行业卓越标准的了解来设定生产力目标*

**马萨诸塞州波士顿，2022 年 5 月 18 日—** 工程管理平台(EMPs)的先驱水母今天宣布推出水母基准，这是一种通过引入比较方法来添加工程指标和性能背景的方法。凭借业内唯一的应用内基准测试功能，水母客户现在可以了解他们在组织、部门、小组或团队层面的众多指标上与同行的差距。

尽管比以往任何时候都更容易获得工程数据，但现代软件工程的复杂性意味着战略决策可能缺乏适当的背景。工程团队可能在历史上表现良好，但没有一个阈值来进行比较——代表不同行业中最优秀的工程团队——工程领导者缺乏对他们的决策是否能够使他们的组织实现精英绩效的洞察力。水母基准测试解决了这个问题，它为客户提供了从匿名的跨行业数据集进行比较的方法。

“在水母，我们的长期目标是支持客户建立尽可能高性能的工程团队，”水母首席执行官刘伟强说。“今天，我们实现了另一个里程碑，为工程领导者提供了一套值得信赖的、广泛的基准，这些基准来自水母无与伦比的客户群，提供了工程行业中最全面的衡量标准。”

**基于度量的工程管理就在这里**

选择加入的工程团队的数据将被匿名化，并添加到基准测试水母客户池中。对于每一个被定义为对工程组织重要的指标，客户都可以访问一个可视化界面，将他们在 给定指标下的绩效与他们的同行或他们自己的组织进行比较，这是真正的行业首创。

水母基准测试公布的关键指标包括那些关注分配、交付、生产力和协作的指标，包括 DevOps 方法中的关键指标。分配基准允许团队了解他们的资源投资与其他工程组织相比如何。交付可以使用周期时间等指标进行基准测试，生产率可以使用问题解决等指标进行基准测试，协作可以使用 PR 评审等指标进行基准测试。新的基准测试功能还使 DevOps 团队能够根据 DORA 指标进行跟踪，包括部署频率、交付时间、平均恢复时间和变更失败率。

“在 Acoustic，我们的首要目标是言行一致。我们始终如一地衡量整个工程团队的进步，”声学工程高级副总裁 John Riewerts 说。“随着水母的工程管理平台的到位，我们现在几乎完成了我们在敏捷文化中的所有承诺。我们已经看到我们这样做的能力提高了 35%。基准功能为我们提供了一个一致的晴雨表来衡量我们相对于其他行业团队的表现，为我们的文化带来了更像游戏的方法。从生产力的角度来看，这种文化是巨大的，使我们能够满足营销人员的需求，因为他们用高绩效的团队编排高度个性化的内容，而水母对我们实现这一点至关重要。"

水母产品负责人 Krishna Kannan 表示:“通过提供业内唯一的应用内基准测试，水母让工程领导者能够使用数据驱动的上下文来检查他们的工程策略和团队运营的有效性。“水母基准可用于跨团队或整个组织跟踪的每个指标，帮助客户了解他们的公司如何比较对其工程职能最重要的指标。”

水母基准测试还提供了数据驱动的洞察力，可以用来通知和影响非工程管理人员，从而推动更好的战略决策，并使工程与更广泛的业务更紧密地结合起来。水母基准的宣布增强了水母 EMP 的当前能力，以测量工程与战略业务优先级和工程团队的性能和操作的一致性。
