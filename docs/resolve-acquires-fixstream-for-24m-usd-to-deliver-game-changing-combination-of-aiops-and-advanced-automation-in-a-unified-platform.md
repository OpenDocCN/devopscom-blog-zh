# Resolve 以 2400 万美元收购 FixStream，在一个统一的平台上提供改变游戏规则的 AIOps 和高级自动化组合

> 原文：<https://devops.com/resolve-acquires-fixstream-for-24m-usd-to-deliver-game-changing-combination-of-aiops-and-advanced-automation-in-a-unified-platform/>

*收购使敏捷 IT 运营能够在几秒钟内预测、诊断和自动解决复杂的基础设施和应用程序问题。*

加利福尼亚州欧文市——2019 年 8 月 20 日**–**[领先的 IT 自动化和流程编排平台 Resolve Systems](https://resolve.io/) 今天宣布收购 AIOps 领域的先驱 [FixStream](https://fixstream.com/) 。此次收购预计将于 9 月底完成，将使 Resolve 能够提供市场上最强大的 IT 自动化平台，方法是将人工智能洞察与动态、混合 IT 环境相结合，并为现代企业的复杂性提供强大的自动化功能。

这个统一的平台将处理广泛的 IT 运营——从人工智能驱动的基础设施映射、运营数据关联和预测分析，到基于这些发现智能地自动化跨域操作。这将使客户能够显著提高基础架构性能，缩短解决问题的平均时间(MTTR)，提高 IT 运营效率，减少警报噪音，并主动、智能地为关键业务服务分配资源。

在过去 24 个月中，FixStream 的收入增长了 7 倍，是增长最快的 AIOps 平台之一。他们行之有效的解决方案提供了自动化的依赖关系映射，可动态跟踪应用程序和底层基础架构之间不断变化的关系，以帮助快速诊断性能问题和停机的根本原因。它还利用人工智能和机器学习来分析大量系统数据并将其置于上下文环境中，使该解决方案能够检测到预测整个 It 体系未来问题的模式。

最终，Resolve 和 FixStream 组合解决方案的长期愿景是帮助客户实现期待已久的“自我修复 IT”承诺将 FixStream 的多层可见性和预测性分析与 Resolve 的跨域、服务级自动化功能相结合，将为客户提供无与伦比的自动预测、预防和自主修复问题的能力。利用动态依赖映射和人工智能驱动的见解来通知、自动更新和触发智能自动化，Resolve 将能够提供一个发现、分析、检测、预测和自动化的闭环系统。

收购 FixStream 使 Resolve 能够进一步进入 AIOps 解决方案市场，随着 IT 团队在管理日益增加的复杂性(包括 IT 基础设施和应用程序产生的数据量的指数级增长)的同时应对降低 IT 成本的难题，ai ops 解决方案市场正在显著增长。Gartner 估计，到 2020 年，包括 AIOps、ITIM 和其他监控工具在内的性能分析市场细分市场的全球收入将达到 57 亿美元。*

Resolve 首席执行官约翰·费隆表示:“我们相信，FixStream 的 AIOps 和基础设施映射功能是 Resolve 企业自动化平台的完美结合，提供了改变游戏规则的功能，使客户能够在 IT 运营方面实现前所未有的敏捷性、速度和简单性。“通过将我们强大的跨领域自动化与 FixStream 人工智能的见解相结合，我们将能够帮助 IT 团队加快他们的数字化转型之旅。”

“Resolve 和 FixStream 共同为 IT 组织提供了他们一直在寻找的完整自动化平台。FixStream 首席执行官兼创始人 Sameer Padhye 表示:“我们一再听说需要一种解决方案，将一流的 AIOps 与成熟的跨领域自动化功能相结合，我们很高兴看到这一愿景成为现实。“除了我们产品中存在的协同作用，我们还有一个共同的目标，那就是帮助 IT 组织应对现代 IT 基础架构带来的挑战，并改善服务交付。”

451 Research 首席分析师 Carl Lehmann 表示:“我们的研究表明，IT 运营效率已成为首席信息官们的一个重要优先事项，因为他们团队的工作量持续增长，而由于熟练 IT 员工短缺和预算缩减，员工人数保持不变。将人工智能的洞察力与自动化技术相结合，使企业能够克服这些挑战，管理当今混合 IT 环境日益增长的复杂性，同时更接近自我修复 IT 的长期目标。"
