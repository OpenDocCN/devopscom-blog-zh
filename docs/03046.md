> 原文:[https://devo PS . com/devo PS-chat-perforce-acquisites-perfecto-mobile/](https://devops.com/devops-chat-perforce-acquires-perfecto-mobile/)

随着 DevOps 领域的两家知名公司——Perforce 和 Perfecto Mobile——联手，整合正在酝酿之中。Perforce 宣布有意收购 Perfecto Mobile，以获得其移动和 web 自动化测试技术，该技术与 Perforce 现有的产品组合相结合，将更好地支持组织实现大规模连续测试。这是一个很好的组合，我相信会在业内广受欢迎。

Perforce 首席执行官 Mark Ties 和 Perfecto Mobile COE Eran Yaniv 与我讨论了此次收购及其对整个测试领域的潜在影响，以及 Perforce 未来的更多收购计划。
