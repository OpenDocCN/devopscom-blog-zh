DevOps 企业峰会是一个面向技术和业务领导者的会议，这些领导者正在改变大型复杂组织交付软件的方式，以帮助他们的组织在市场中获胜。

在美国进入第六个年头，它的目标是加快 DevOps 的采用，增加这些计划成功的可能性，并帮助实现文化规范、技术实践和架构，以帮助组织实现最高水平的性能。

“当然，我们的目标是创造有史以来最好的编程——这对编程委员会来说是一个有价值的挑战，因为我们都认为伦敦 DevOps 企业峰会是迄今为止最好的，”IT Revolution 的创始人、《独角兽项目》的作者、*《凤凰项目》*、*《devo PS 手册》*和 *Accelerate* 的合著者吉恩·金说。“DevOps 企业峰会的标志之一是致力于突出经验报告，因为作为领导者，我们必须通过观察处于类似情况下的其他人如何解决问题来学习。通过这样做，我们帮助创建了一个积极互助的旅伴社区。”
