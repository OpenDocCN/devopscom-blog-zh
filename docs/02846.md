# Anodot 通过真正的自助式实时自主分析加快价值实现

> 原文：<https://devops.com/anodot-accelerates-time-to-value-with-true-self-service-autonomous-analytics-in-real-time/>

**纽约-2018 年 11 月 29 日-[阿诺特](http://www.anodot.com/) ，**自主分析公司今天宣布增强其 AI/ML 解决方案，增加了数十个入站数据收集器，使其客户生态系统能够受益于实时业务事件的真正自助检测。

数据中的事件检测和根本原因分析可以提供关键的业务信息，但只有在足够早地报告以便采取行动时，这种洞察力才有价值。 [Anodot 的专利算法](https://patents.google.com/?assignee=anodot&oq=anodot) 和机器学习确保更高的真阳性和更低的假阳性，同时实时准确地洞察季节性，即时准确地向客户发出警报。Anodot 的数十个入站数据收集器意味着分析师可以添加新的数据流和新的分析用例，并在同一天启动和运行。

该公司的集成合作伙伴生态系统正以每年 1000%的速度快速增长，集成合作伙伴包括 PagerDuty、Slack、mPartical、Google BigQuery、Amazon Redshift、Amazon Kinesis、MySQL、AWS S3、Google 云存储、Adobe Analytics 等。这一增长归功于 Anodot 不断扩大的客户群，包括通过 、 [、积压](https://www.overstock.com/) 和 [Openx](https://www.openx.com/) 的 [，这些客户群依靠其自主分析解决方案实时监控高粒度级别的业务数据，以更快地检测和解决业务问题，确保业务避免可能影响公司声誉和投资回报的问题。](https://ridewithvia.com/)

“Anodot 的工具易于集成，并在数据出现任何差异或峰值时立即向我们的分析师团队发出警报，”Rubicon 项目的分析总监 Rich Galan 说，“该技术也易于使用，我们在学习和使用 Anodot 的自主分析平台时从未遇到过问题。”

“PagerDuty 欢迎 Anodot 加入其集成合作伙伴计划。page duty 的实时运营管理平台和 Anodot 的实时异常检测、自主分析平台相结合，可以帮助客户更快地检测和解决业务事件，”page duty 的战略业务发展和联盟执行官 Steve Gross 说。

与需要用户手动上传或发送数据到 API 端点的工具不同，Anodot 的 [自动化了流程](https://www.anodot.com/blog/makes-automated-anomaly-detection-truly-automated/) ，实现了真正的自助服务。Anodot 的数据集成是安全的，完全受控的，对最终用户完全透明，提供了可见性和对数据和预算的更好控制。

Anodot 产品副总裁 Amit Levi 表示:“价值实现时间被定义为 Anodot 2018 年的一个关键支柱，这是我们持续致力于不断改善产品使用和客户利益的一部分。“在过去的几个月里，我们投入巨资开发最先进的收集器，以减少摩擦和价值，因此，我很自豪地宣布，现在的平均集成速度是几个月前的 30 倍。”
