# Axway 调查发现，消费者经常担心其数据的隐私和安全性

> 原文:[https://devo PS . com/ax way-survey-finds-consumers-frequency-have-concerns-concerns-about-the-privacy-security-of-their-data/](https://devops.com/axway-survey-finds-consumers-frequently-have-concerns-about-the-privacy-security-of-their-data/)

公司有机会在信任的基础上构建无缝的数字体验。

**亚利桑那州凤凰城，2021 年 6 月 3 日—**全球 API 管理领导者 Axway(泛欧交易所代码:AXW。PA)发布的一项消费者调查显示，大多数美国人表示，他们希望公司处理和跟踪数据的方式更加透明。2021 年的数据隐私研究询问了 1017 名美国成年人对公司如何使用、处理和跟踪他们的数据的总体理解和担忧。受访者广泛关注他们的数据是如何存储和保护的，并且绝大多数人支持对查看他们个人数据的人进行更多的控制。

研究发现，82%的人希望知道公司收集了他们的哪些具体数据。82%的人还担心他们的在线数据可能不安全，只有 39%的美国人表示他们知道他们的在线数据存储在哪里。与此同时，59%的人同意，如果能带来更好的用户体验，让公司访问他们的个人数据是值得的。

该研究还发现，四分之三的美国人(75%)选择与提供更安全的个人数据保护计划的企业合作，但对于该群体中的 36%来说，这仍然取决于他们对该公司的信任程度。与规模较小的独立或本地公司相比，只有三分之一的人更信任“大型科技公司”，而 49%的人对大型和小型公司的信任度大致相同。

“无论是在能源、医疗、金融还是零售领域，公司都在努力脱颖而出，提供出色的数字体验，同时还要遵守行业数据隐私和安全要求，这不是一个简单的命题，”Axway 副总裁兼首席催化剂 Brian Pagano 表示。

“消除客户体验中的摩擦是最大的竞争优势，这些调查结果表明，安全漏洞和隐私问题是人们期待的无缝体验的障碍。”

该调查还试图了解消费者对苹果应用程序跟踪透明度框架的态度，并发现[人压倒性地支持这一举措](https://blog.axway.com/news/apples-app-tracking-transparency-trust)。四分之三的美国人认为，其他操作系统应该效仿，如果广告商没有获得用户的许可，就阻止广告商在不同的应用程序上跟踪你的活动和偏好。

“经验告诉我们，消费者会去摩擦最少的地方，而不管你的数据的监管链，”Pagano 说。“如果人们不确定他们的数据是如何被处理的，公司就有机会建立信任，这样他们的普通消费者就不需要关心这些细节。企业可以依靠一套工具，如 Axway 的 [Amplify API 管理](https://www.axway.com/en/products/amplify-api-management-platform)平台，这将为他们提供可扩展性、隐私和安全审计跟踪以及监管和合规框架，以便他们可以专注于提供最佳和最安全的体验。”
