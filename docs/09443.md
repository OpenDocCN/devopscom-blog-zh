# Appstellar 现在是 AWS 咨询合作伙伴！

> 原文:[https://devo PS . com/app stellar-is-now-an-AWS-consulting-partner/](https://devops.com/appstellar-is-now-an-aws-consulting-partner/)

Appstellar 很高兴地宣布，我们已经作为 AWS 咨询合作伙伴加入了亚马逊合作伙伴网络。通过这种合作关系，我们的目标是继续在云上安全地构建软件产品！

我们很兴奋地宣布，我们正式[成为亚马逊网络服务合作伙伴网络(APN)](https://partners.amazonaws.com/partners/0010h00001jCcZLAA0/Appstellar) 的一部分！

此外，作为科索沃领先的云服务和 DevOps 服务提供商，我们 Appstellar 很高兴地告诉大家，我们是第一家在该国成立和总部设在该国的公司成为 AWS 合作伙伴。

APN 是亚马逊的全球合作伙伴社区，他们利用 AWS 来构建一流的软件解决方案。它由专业服务机构组成，拥有成功交付 AWS 解决方案的良好记录，并拥有大量经过认证、经验丰富的 AWS 技术顾问。

作为 AWS 合作伙伴，我们支持客户最大化 AWS 提供的所有价值，并加速他们的云计算之旅。APN 使我们能够作为合作伙伴提供创新的解决方案，解决技术挑战，并为我们和 AWS 的共同客户提供价值。

Appstellar 目前正处于高速增长模式。在过去的 6 个月里，我们的团队增长了 122%！此外，我们已经签署了多个新的客户协议，在新的运营行业(如 fintech 和 healthtech)进行了扩张，还搬到了新的办公场所。除了与 AWS 的合作关系， [Appstellar 还成为了 CNCF 会员和 Kubernetes 认证服务提供商](https://appstellar.io/stories/cncf-member-kcsp-announcement)。有了这些伙伴关系，我们期待着进一步加速我们的增长，并接受我们喜欢的项目。

为什么要与 Appstellar 这样的 AWS 合作伙伴公司合作？

亚马逊合作伙伴网络(APN)在帮助客户在云计算之旅中充分利用 AWS 产品方面具有独特的优势。作为 AWS 合作伙伴，我们帮助我们的客户在被称为最先进的云生态系统上发展和扩大业务。
