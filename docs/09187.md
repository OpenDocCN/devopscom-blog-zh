# Glasswall 推出桌面内容解除和重建应用程序的免费增值版，以抵御基于文件的网络安全威胁

> 原文：<https://devops.com/glasswall-launches-freemium-version-of-its-desktop-content-disarm-and-reconstruction-app-to-protect-against-file-based-cybersecurity-threats/>

*glass wall Desktop Freedom 提供无需等待检测的保护，使组织能够信任每个文件*

**英国伦敦，2022 年 3 月 17 日—** [内容解除和重建(CDR)技术的全球领导者 Glasswall](https://www.glasswallsolutions.com/) 今天宣布推出 Glasswall Desktop Freedom，这是其市场领先的桌面 CDR 应用程序的免费增值版本。它旨在帮助保护公共和私营部门的组织免受基于文件的威胁(如恶意软件和勒索软件)的风险，提供无需等待检测的保护，因此用户可以信任每个文件。免费增值版自首次下载之日起 12 个月内对用户开放。

Glasswall CDR 技术可以主动防御隐藏在当今最流行的文件格式中的基于文件的网络安全威胁。与基于检测的安全性(如必须应对新威胁的防病毒和沙盒)相比，Glasswall Desktop Freedom 可以即时清理和重建每个文件，以符合其“已知良好”的行业规范，从而自动消除潜在风险，并弥补基于检测的安全性可能长达 18 天的“保护差距”。

Glasswall Desktop Freedom 可作为 Microsoft Windows 10 和 11 的免费增值下载，使组织能够每天分析、清理和保护多达 20 个文件，最大文件大小为 5MB。清理可应用于一系列最广泛使用的文件和文档类型，包括 Microsoft Word、Excel 和 PowerPoint 以及 Adobe PDF。用户可以从 Glasswall Desktop Freedom 升级，通过购买 Glasswall 的一系列桌面 CDR 产品的许可证来解锁附加功能和保护容量。

该下载为组织提供了一种快速简便的方法来从用户计算机和安全网络基础设施的安全角度清理文件。旨在简化部署和使用，具有方便的拖放功能，以确保进入组织的每个文档都是安全的，而不会影响工作效率。该解决方案还为用户提供了离线文件处理和清理选项，使他们能够清理和重建文件，而不会将它们发送到已建立的信任边界之外。

Glasswall CDR 针对各种使用情形而开发，包括零日威胁防御、净化进入组织的新文件、作为遗留文件清理的解决方案，以及防范网站下载和上传中包含的威胁。处理完每个文件后，系统会向用户提供一份文件分析报告，详细说明原始文件的风险级别、已删除的活动内容以及已修复的对象和结构。

用户还可以为每种受支持的文件格式设置策略，允许他们指定是从文件中删除活动内容、允许活动内容(保留在文件中)还是不允许活动内容(不重建文件)。它还可以与 USB 驱动器或存储卡等可移动介质上的文件进行交互。

Glasswall 首席执行官 Danny Lopez 解释说:“在人们日益关注各个领域组织的网络防御的时候，Glasswall 提供了这种免费增值解决方案，以帮助更好地防范最常见和最成功的网络攻击策略之一——使用文件和文档。“它还将帮助 It 和安全团队评估和了解 CDR 对他们提供主动保护和解决零日漏洞的能力的主要影响。”
