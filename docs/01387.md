# 借助 API 驱动的混合云架构加速数字化转型

RALEIGH, N.C. — <time>June 22, 2016</time> —

全球领先的开源解决方案提供商 Red Hat，Inc. (NYSE: RHT)今天宣布，已签署最终协议，收购领先的应用程序编程接口(API)管理技术提供商 3scale。通过向其现有产品组合添加 3scale，包括 Red Hat JBoss 中间件、Red Hat OpenShift 和 Red Hat 移动应用程序平台，Red Hat 通过简化的云集成和基于微服务的架构加强了其对 API 经济的支持。

> 3scale 补充了我们现有的中间件产品组合和 Red Hat OpenShift，使公司能够使用 Red Hat JBoss Fuse 等工具创建和发布 API，并在这些 API 发布后管理和推动其采用。
> 
> CRAIG MUZILLASENIOR VICE PRESIDENT, APPLICATION PLATFORMS BUSINESS, RED HAT

在数字化转型、敏捷开发实践和混合技术的时代，API 经济已经成为常态。Forrester 估计，到 2020 年，40%的美国公司将拥有 API 管理解决方案，这一年将花费 6.6 亿美元。1 受管 API 在企业中变得越来越具有战略意义，因为它们使组织能够快速交付易于使用且安全、可扩展和可靠的新服务，并使服务(如 API)能够作为产品货币化。由移动、物联网(IoT)和其他趋势驱动的将功能分解为更细粒度组件的新兴架构——“微服务”，进一步提高了对 API 管理的需求。

3scale 成立于 2007 年，其目标是让各种规模的组织能够在 API 变得至关重要的网络世界中成功地创建、管理和使用 API。如今，3scale 提供了基于混合云的 API 管理平台，该平台将云管理层与 API 网关分离，为客户提供了灵活性、性能和规模。3scale 以自助服务模式为多个行业的 700 多家组织提供 API 管理平台，包括金宝汤公司；卡特拉商业公司；CrunchBase 江森自控；牛津大学出版社；SITA 和加州大学伯克利分校。3scale 在巴塞罗那和旧金山设有办事处。

3scale 和 Red Hat 于 2015 年初开始合作，为基于 API 的应用开发提供一个全面的平台。该联合解决方案将 3scale API 管理平台与来自 Red Hat JBoss 中间件、Red Hat OpenShift 和 Red Hat 移动应用平台的技术相结合，提供以 API 为中心的开发、部署和管理产品，旨在加快应用上市时间。Red Hat JBoss 中间件客户将从现代架构的 3scale 混合云功能中受益。特别是移动应用程序开发高度以 API 为中心，3scale 的平台与 Red Hat 移动应用程序平台相结合，可以简化大规模开发和管理。

最近，3scale 为 Red Hat OpenShift 推出了其 API 网关的容器化版本，使用户能够通过分布在不同混合环境中的微服务轻松、一致地创建应用程序。Red Hat 的容器应用程序平台和 3scale 的 API 管理功能相结合，使客户能够简化和自动化任务，帮助减少错误，同时提高灵活性和控制力。

Red Hat 为企业提供领先的开源平台，用于构建现代应用程序、集成和自动化解决方案。通过添加 3scale 的 API 管理解决方案及其现有的中间件、容器应用平台和移动产品，Red Hat 可以进一步简化和加速客户的数字化战略，加快 API 计划的上市时间，并使企业能够以更智能的方式更快地创新。

此次收购预计不会对红帽公司截至 2016 年 8 月 31 日的第二财季(“第二季度”)或截至 2017 年 2 月 28 日的财年(“2017 财年”)的收入产生重大影响。管理层预计，由于此次交易，第二季度 GAAP 运营费用将增加约 300 万美元，或每股 0.01 美元，2017 财年将增加约 700 万美元，或每股 0.03 美元。管理层预计，由于此次交易，第二季度非 GAAP 运营费用将增加约 150 万美元，或每股约 0.01 美元，2017 财年将增加约 500 万美元，或每股约 0.03 美元。Red Hat 通过从 GAAP 运营费用中减去非现金股份薪酬费用的估计影响来计算非 GAAP 运营费用，预计第二季度将增加约 80 万美元，无形资产摊销预计在 2017 财年剩余的每个季度增加约 20 万美元，以及与业务合并相关的交易成本，预计第二季度将低于 100 万美元。对于红帽的完整预测，包括交易对 GAAP 和非 GAAP 营业利润率以及 GAAP 和非 GAAP 每股收益的预期影响，请参考其 2016 年 6 月 22 日财报新闻稿中提供的指导。

根据惯例成交条件，交易预计将于 2016 年 6 月完成。
