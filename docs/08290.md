# 让基于区块链的应用开发变得更加容易

> 原文：<https://devops.com/making-blockchain-based-app-dev-more-accessible/>

区块链在加密货币之外还有很多潜在的用例。金融、汽车、供应链和移动等成熟行业可能很快就会采用分散式应用程序和[分布式账本](https://devops.com/unifying-partner-ecosystems-with-a-distributed-ledger/)。这项技术甚至被誉为人类星际旅行后保持互联经济的理想基础。

最近，旁观者注意到对分散的、对等的、自主的开发方法的兴趣日益增长。盯着围绕加密货币和区块链的兴奋，许多开发人员渴望开始试验他们自己的区块链应用程序。区块链和加密货币企业家 Max Kordek 表示，这种即将到来的破坏可能会消除中间人，将权力还给人们。

然而，从零开始构建你自己的区块链是非常困难的。因此，我最近会见了 Lisk 的联合创始人 Max Kordek，以了解如何使基于区块链的应用程序开发更容易实现。根据 Kordek 的说法，创建自己的区块链有很多好处，但简化创建过程对于授权新的创业公司并使这种新模式蓬勃发展是必要的。

## 潜在的使用案例

区块链的去中心化和防篡改特性具有颠覆性的潜力，可以消除中央权威，将更多的数据控制权交给集体蜂巢。“对我来说，这一切都是为了消除中间人，”Kordek 说。“它把权力还给人民。”加密货币显然是最广泛的使用案例；比特币证明了一个位于区块链的系统可以被大规模采用。但是在加密货币之外，还有许多其他的用例，类似的记录保存也很重要。

例如，[汽车行业](https://www2.deloitte.com/content/dam/Deloitte/cn/Documents/consumer-business/deloitte-cn-consumer-blockchain-in-the-automotive-industry-en-180809.pdf)可以链接制造商之间的数据，或者在车辆之间共享数据。 [NFTs](https://www.forbes.com/advisor/investing/nft-non-fungible-token/) 用区块链来表现独一无二的数字艺术品。[航空业](https://www.bcg.com/publications/2019/what-could-blockchain-do-airlines)期待区块链同步合作伙伴生态系统。区块链也正在通过充实游戏中的价值体系来改变在线游戏。

从保险到供应链到运输，大多数行业都有潜在的区块链用例。但是区块链并不局限于地球上的场景。信息系统工程师预见区块链将推动未来的太空经济。随着大量资金投入小行星采矿和火星任务，星际人类似乎不可避免。与比特币相比，“以太坊是星际经济的更好选择”，[大卫·范德沃特](https://singulardtv.medium.com/introduction-to-interplanetary-cryptocurrency-fc6755cd2be3)说。以区块为单位收集交易并对卫星发出信号，可以实现行星间的货币交易。“星际金融系统必须建立在区块链技术的基础上，”Kordek 说。

无数领域都有成熟的潜力。但是谁将领导这一转变呢？根据 Kordek 的说法，我们可能会看到区块链公司扩大现有的结构。然而，老牌企业会发现摆脱遗留系统的束缚是一项挑战。相反，新的创业公司更有可能在区块链革命中拥有更大的发言权。“一个全新的创业公司采用区块链，并围绕区块链构建他们的整个业务，这要现实得多，”科尔德克说。

## 区块链发展状况

比特币成立后，其他开发者分叉项目，改变参数引入新的加密货币，如莱特币和 Dogecoin。最终，以太坊出现了，这是独一无二的，因为它使开发人员能够使用智能合约来构建应用程序。然而，“开发者生态系统在一开始是可怕的，”Kordek 解释道。"这是西部开发的蛮荒之地。"直到今天，在以太坊上开发仍然非常复杂，他补充道。另外，由于它是开源的和不可变的，所以容易受到安全威胁。“这就像是世界上最大的赏金项目——如果出现一个漏洞，合同就会泡汤。”以太坊还有[可扩展性问题](https://coingeek.com/a-ethereum-2-0-still-not-scalable/)——每秒只能运行 30 个事务，相当于每天 100 万个事务。

出于这些原因，建立自己的区块链可能更好。然而，这样做是一项艰巨的任务。从研究的角度来看，这需要大量的理论知识，Kordek 解释道。区块链还引入了新的维护问题——因为基础设施一直在运行，所以它是不可阻挡的，这使得修补漏洞或推动更新充其量是脆弱的。如果数亿美元依赖于一个稳定的网络，一个错误就可能造成巨大的影响。高开发和质量保证的障碍使得从零开始构建基于区块链的系统原型具有挑战性。Kordek 说:“实验对于推动一个行业的发展至关重要。“重要的是我们要简化区块链的开发，这样才能进行更多的实验。”

## 开源 SDK 让区块链更可用

那么，我们如何让区块链变得更容易接近，以便进行实验呢？嗯，通过现成的开源语言专用 SDK 已经取得了很多进展。例如，有[Polkadot](https://github.com/paritytech/polkadot)(Rust)[Cosmos](https://github.com/cosmos/cosmos-sdk)(Go)和 [Lisk](https://lisk.com/) (JavaScript)。Kordek 是适合团队编程偏好的特定语言库的倡导者。除此之外，他更喜欢 JavaScript，因为它通常更容易学习，并且已经支持许多基于网络的界面。

这些项目与智能合同本身无关；它们是关于改进你自己的区块链创作过程。根据 Kordek 的说法，沿着这条路走下去，在早期会获得更多的控制。如果有东西坏了，你可以修复，推送，更新。他说，如果代币遭到黑客攻击，你可以回滚并进行调整。

随着公司变得越来越大，这项技术可能会产生集中效应。“我们如何才能以某种方式将权力还给人们？”科德克问道。区块链是一个深刻的答案，它让新进入者能够缓解跨行业的问题——它甚至可以帮助我们*大胆地去以前没有人去过的地方*。
