# 腾讯以 Linux 基金会白金会员身份支持开源社区

> 原文：<https://devops.com/tencent-supports-open-source-community-with-linux-foundation-platinum-membership/>

**三藩市和北京(Linux con China)——2018 年 6 月 25 日**–[通过开源实现大众创新的非营利组织 Linux 基金会](https://www.linuxfoundation.org/) 宣布 [腾讯](https://www.tencent.com/en-us/index.html) 成为该基金会最新的白金会员。腾讯是中国领先的互联网增值服务提供商，提供一些中国最受欢迎的网站、应用和服务，包括 QQ、Qzone、腾讯云和微信。

近年来，腾讯对开源越来越投入。它已经广泛使用 Linux 多年，是第一批利用 OpenDaylight SDN 控制器的公司之一，也是中国开放 SDN (COS)委员会的创始人之一。腾讯在 GitHub 上赞助了数十个开源项目，是云原生计算基金会、Hyperledger、LF Networking 和开放网络基金会的成员。此外，腾讯是最近成立的 LF 深度学习基金会的创始成员，该基金会专注于支持人工智能领域的各种技术努力。腾讯此前宣布，它打算向 LF 深度学习基金会捐赠其天使项目，这是一个针对大数据/模型调整的高性能分布式机器学习平台。

腾讯的刘鑫将加入 Linux 基金会董事会。除了在董事会的席位，作为白金会员，腾讯将能够利用基金会在开源治理、法律和合规、活动、营销等领域的专业知识。作为白金会员，腾讯将能够为各种各样的开源项目及其社区提供进一步的支持和资源。

“成为 Linux 基金会的白金会员显然增强了开源对腾讯的重要性，以及它对开源社区的承诺，”Linux 基金会执行董事吉姆·泽姆林说。“随着腾讯对开源的贡献不断增加，Linux 基金会很高兴能与该公司更紧密地合作，为社区提供更多的资源和支持。我们期待着与腾讯合作，在未来的日子里实现更多的创新和发展，以刺激新的和令人兴奋的技术的创造。”

“我们很荣幸成为 Linux 基金会的白金会员。开源是腾讯技术战略的核心，”腾讯移动互联网事业群总经理刘鑫表示。“我们期待加强与国际开源社区的关系，促进尖端技术的创新。作为第一步，我们向 Linux 基金会捐赠了开源微服务项目 TARS 和开源名称服务项目 Tseer。此外，我们计划将我们的开源 AI 项目 Angel 捐赠给 LF 深度学习基金会。我们将共同帮助建立一个受欢迎的、充满活力的全球开源生态系统。”

超过 800 个组织是 Linux 基金会及其主持的开源项目的成员。美国电话电报公司、思科、富士通、日立、华为、IBM、英特尔、微软、NEC、甲骨文、高通、三星和 VMware 也是白金会员。要了解更多关于 Linux 基金会会员资格或加入该组织的信息，请访问[https://www.linuxfoundation.org/membership/](https://www.linuxfoundation.org/membership/)。

**关于 Linux 基金会**

Linux 基金会是世界顶级开发者和公司的首选组织，旨在构建加速开放技术开发和行业采用的生态系统。它与全球开源社区一起，通过创造历史上最大的共享技术投资来解决最困难的技术问题。Linux 基金会成立于 2000 年，如今提供工具、培训和活动来扩展任何开源项目，这些共同带来了任何一家公司都无法实现的经济影响。更多信息可在找到 www.linuxfoundation.org。
