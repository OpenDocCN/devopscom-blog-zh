# Humio 提名 2020 年 Gartner 酷派厂商

> 原文：<https://devops.com/humio-named-a-2020-gartner-cool-vendor/>

*Humio 在* [*2020 年 Gartner 业绩分析报告中认可的酷派厂商*](https://www.humio.com/cool-vendor-2020)

华盛顿州西雅图 10 月 7 日消息， [Humio](https://www.humio.com/) ，唯一一个支持实时、大规模、自托管或云中的所有流日志的完整可观察性的日志管理平台，今天宣布 Gartner 根据 2020 年 10 月 5 日由 Padraig Byrne、Federico De Silva 等人撰写的题为[*【性能分析中的酷供应商】****的报告，将其评为酷供应商。](https://www.humio.com/cool-vendor-2020)

Gartner 写道:“性能分析供应商越来越多地提供高级分析、AIOps 和改进的可扩展性，以应对 IT 环境中速度和数据复杂性的增长。I&O 领导人应该考虑创新的供应商方法如何能够增强对其 IT 环境的洞察。”

Humio 首席执行官吉塔·施密特(Geeta Schmidt)表示:“我们认为，Humio 入选 Gartner 发布的 Cool Vendor 报告是对我们现代化创新架构和日志管理方法的肯定，也是我们越来越多地被企业客户采用的标志。“随着组织转向高度动态的体系结构，监控和处理越来越多的数据的需求也在急剧变化。客户收集的数据量永远不应该是有限的。由于我们最低的总拥有成本、行业领先的无限计划、最少的维护和培训成本以及极低的计算和存储要求，Humio 让客户能够收集他们的所有数据，从而改变了游戏规则。”

随着威胁和攻击(如最近宣布的 Zerologon)的增加，收集和查询所有日志数据的重要性比以往任何时候都更加迫切。这个漏洞至少从 Windows Server 2008 开始就存在了，它允许攻击者在几秒钟内控制企业 Windows 基础架构的任何部分或整个网络。使用 Humio query，客户可以轻松地查询他们的所有 Windows 日志，也可以在几秒钟内确定他们是否受到 Zerologon 漏洞的影响。Humio 的内存状态机支持数千个并发实时查询的亚秒级搜索结果，允许分析师、开发人员和安全人员在数据实时到达时与其进行交互。

Humio 的客户是各行各业的大型企业和组织，包括金融服务、教育、医疗保健、通信和基于云的服务提供商，如阿鲁巴 HPE、彭博、Decisiv、密歇根州立大学、Netlify、SpareBank 1 和 Vijilan。为了进一步提高互操作性和性能，该公司扩展了其生态系统，包括 AWS、Corelight、Instana、Kloia、Netic、Pondurance、SOC Prime、Tines 和许多其他产品。
