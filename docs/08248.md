# Attivo Networks 推出 CIEM 解决方案，扩展其身份检测和响应(IDR)产品组合

> 原文：<https://devops.com/attivo-networks-launches-ciem-solution-expanding-its-identity-detection-and-response-idr-portfolio/>

*新的 IDEntitleX 解决方案提供了前所未有的跨终端、活动目录环境和云基础设施的身份风险可见性*

**加利福尼亚州弗里蒙特，2021 年 7 月 14 日—** 身份检测和响应领域的领导者 Attivo Networks 今天宣布推出全新的云基础设施授权管理(CIEM)解决方案 IDEntitleX，旨在提供可见性并减少云中身份和授权的攻击面。IDEntitleX 扩大了公司在提供前所未有的可见性和防止身份特权升级和横向移动威胁活动方面的领导地位。随着这一新产品的推出，Attivo 成为同类解决方案中唯一一个能够对终端、活动目录(AD)和云上的身份和权限暴露和风险提供端到端分析的解决方案。

组织正以创纪录的数量向公共云迁移，但随着这种增长，用户身份管理和“非人类”身份(如应用程序、数据库和数据存储)的激增带来了意想不到的安全挑战。Gartner，Inc .估计“到 2023 年，75%的安全故障将由身份、访问和权限管理不当造成，而 2020 年这一比例为 50%”¹。不幸的是，传统的安全工具不能很好地处理这种资源管理的爆炸式增长，结果是过度配置访问并加剧了安全风险。

新的 IDEntitleX 解决方案通过为安全团队提供跨组织的身份和暴露的统一视图来解决配置管理挑战，同时保持运营效率，从而降低了这种风险。这包括对 AWS 和 Azure 的多云支持，并为用户、应用程序、虚拟机、容器、无服务器功能和攻击者试图锁定的其他对象提供详细的权限可见性。

Attivo Networks 研究副总裁 Venu Vissamsetty 表示:“随着组织不断加快采用云基础设施的速度，人类和非人类身份也在不断扩大，CISOs 必须专注于身份优先的安全解决方案，以跟上这种增长的步伐。“这款 CIEM 解决方案为安全团队提供了持续监控身份、权限以及跨标准云服务的相关性所需的最佳灵活性和可见性。”

Omdia 的首席分析师里克·特纳(Rik Turner)说，“在今天加速应用交付的令人兴奋的氛围中，安全性往往是事后才想到的。”。“企业对云计算的快速采用揭示了‘权限蔓延’的潜在威胁，这一挑战正在推动一个新兴的技术领域，该领域优先考虑可见性并减少云中身份和权限的攻击面。随着 IDEntitleX 的推出，Attivo Networks 将成为 CPM 或 CIEM 市场的主要参与者。”

Attivo Networks IDEntitleX 解决方案可应对身份和权限管理的挑战，并具有以下优势:

*   **最广泛的云可见性:**通过跨所有平台的身份和权限摘要了解暴露情况。
*   **特定于平台的云可见性:**监控关键云服务的授权，如 AWS S3 和 Azure Key Vault。
*   **端到端可见性:**从集中管理控制台查看端点、活动目录和云暴露。
*   **特定于对象的风险评估:**根据风险隔离对象，并分析其权限的详细信息。
