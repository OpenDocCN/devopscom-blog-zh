# Pega 推出 Pega Process Fabric——简化跨企业孤岛工作的“平台对平台”

> 原文:[https://devo PS . com/pega-introduces-pega-process-fabric-the-platform-for-the-streamlined-work-over-enterprise-silos/](https://devops.com/pega-introduces-pega-process-fabric-the-platform-for-platforms-that-streamlines-work-across-enterprise-siloes/)

新的云软件将不同的技术编织在一起，以改善客户、员工和合作伙伴的体验

马萨诸塞州剑桥。–2020 年 6 月 2 日–[Pega systems Inc .](https://www.pega.com/)(纳斯达克股票代码:PEGA)，一家为全球领先企业提供数字化转型支持的软件公司，今天宣布推出[Pega Process Fabric](http://www.pega.com/process-fabric)，这是一种基于云的新软件架构，旨在从根本上简化组织如何跨分布式企业技术推动工作。通过最少的部署工作，这种“平台对平台”打破了技术孤岛，统一了整个企业的工作，并有助于改善员工、客户和合作伙伴的用户体验。

即使在今天这个超级互联的世界里，组织仍然受到互不相连的软件系统的困扰，这些软件系统阻碍了他们发挥全部潜力。随着每年在企业软件上花费数万亿美元，组织通常会留下臃肿的技术堆栈或充满不兼容点解决方案的脱节云系统。因此，沮丧的员工从一个应用程序跳到另一个应用程序，只是为了完成基本任务，而同样沮丧的客户则面临缓慢且不一致的服务。

可通过[Pega Platform](https://www.pega.com/products/pega-platform)获得，Pega Process Fabric 结合了**动态 API**、一个**开放 UX 框架**、 **UI 组件**、**机器人自动化**和**数据虚拟化**的强大功能，以弥合端到端的连接差距。这种打破藩篱的软件将来自不同厂商的企业技术编织成一个统一的“平台对平台”虽然各个组件仍然在幕后独立运行，但它们对用户和客户来说是一个紧密结合的整体。

这种方法通过以下方式改变了整个组织的工作方式:

*   **让员工更智能地工作:**借助*交织体验*，员工可以在单个用户界面内跨 Pega 和非 Pega 应用工作，这有助于消除低效的屏幕切换。一个*交织的工作列表*将每个员工的任务整合在一个地方，并使用人工智能来帮助确保最重要的工作获得最高优先级。
*   **让领导更好地了解工作:**经理和高管可以看到整个企业的工作是如何进行的，以及在哪里进行的。 *Process insight analytics* 让领导者全面了解跨系统和业务部门正在进行和已完成的工作。
*   **将合作伙伴连接到工作流:**随着组织不断扩大其生态系统，Pega Process Fabric 帮助他们和他们的合作伙伴将他们的系统联系在一起，作为一个统一的团队运营。这使得每个人都可以跨组织边界跟踪进度、共享数据和自动化工作。
*   **将 IT 团队从低效的开发工作中解放出来:**借助 Pega 的低代码模型驱动架构和软件重用，业务人员可以在几分钟内轻松地将系统连接到 Pega Process Fabric。这减轻了 IT 团队不断将新软件连接到结构以满足最新市场需求的压力。

工作原理:集中力量
不同之处在于 Pega 独特的业务架构方法。Pega Process Fabric 从[中心向外](https://www.pega.com/insights/resources/center-out-explained)削减 IT 复杂性，而不是从*自上而下*重新编码每个孤立的渠道，或者从*自下而上*重新设计复杂的后端。Pega 在这个新的、更加敏捷的中间层中应用业务逻辑和智能，在不中断现有基础设施的情况下将工作流编织在一起。这有效地创建了一种新的平台类型，帮助其他平台更好地一起运行。现在，组织可以联合和优化他们的企业堆栈，为他们的利益相关者做更好的工作。

**可用性**
Pega Process Fabric 功能将通过 Pega 平台分阶段推出。现在有很多可用的，包括动态 API 和数据虚拟化。预计今年第三季度将推出交织工作列表和流程洞察分析。包括 interwoven experiences 在内的其他更新计划于明年年初推出。某些 Pega Process Fabric 元素可能需要额外的产品许可。更多信息，请访问[www.pega.com/process-fabric](http://www.pega.com/process-fabric)。

此外，预建适配器将很快在 [Pega Marketplace](https://community.pega.com/marketplace) 上提供下载。这些适配器将 Pega Process Fabric 与特定应用程序连接起来，包括来自 Salesforce、ServiceNow、Microsoft 和 Google 的企业应用程序，以及来自 UiPath 和 Blue Prism 的开放式机器人自动化 API。
