# McAfee 凭借丰富的 SASE 威胁防护突破 XDR 市场，主动阻止目标攻击

> 原文：<https://devops.com/mcafee-breaks-through-xdr-market-with-sase-enriched-threat-protection-to-proactively-stop-targeted-attacks/>

*MVISION XDR 利用从深度集成的云数据源中获得的可行威胁洞察，自动执行安全调查和响应流程*

**加利福尼亚州圣何塞，2021 年 5 月 18 日—** 云网络安全公司 McAfee Corp .(纳斯达克股票代码:MCFE)今天宣布，通过关联 McAfee 终端安全解决方案、安全访问服务边缘(SASE)解决方案和由 MVISION Insights 提供支持的威胁情报解决方案的广泛遥测技术，大幅扩展其 MVISION 扩展检测和响应( [XDR](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.mcafee.com%2Fenterprise%2Fen-us%2Fsecurity-awareness%2Fendpoint%2Fwhat-is-xdr.html&esheet=52430850&newsitemid=20210517005883&lan=en-US&anchor=XDR&index=1&md5=b287cde442fbdb56bcb56fae0d9263e2) )解决方案。这些集成保护组织免受最高级的威胁，同时通过从设备到云的统一控制和可见性简化安全运营。

这一时机至关重要，因为安全运营中心(SOC)正在应对针对远程员工和云服务的日益复杂的威胁因素，这些威胁因素在不断扩大的数字攻击面上使用更多规避技术，这使得传统安全控制更难发现对手。Enterprise Strategy Group 最近对 IT 安全专业人员进行的一项调查发现，云对大多数组织的威胁检测和响应能力构成了最大的差距。不出所料，根据[安永&扬](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fassets.ey.com%2Fcontent%2Fdam%2Fey-sites%2Fey-com%2Fen_gl%2Ftopics%2Fadvisory%2Fey-global-information-security-survey-2020-single-pages.pdf&esheet=52430850&newsitemid=20210517005883&lan=en-US&anchor=Ernst+%26amp%3B+Young&index=3&md5=8f24df878a7312ff58cb534d56d8d0dd)的调查，大约十分之六的公司在过去 12 个月中遇到过重大或重大事故，然而，只有 26%的公司表示他们的 SOC 发现了他们最重大的违规行为。

McAfee MVISION XDR 是首个主动式、数据感知型开放式 XDR 平台，旨在通过连接和融合不同终端、网络和云数据源的统一威胁检测和响应，帮助组织阻止这些复杂的多向量攻击。从今天开始，XDR 事件因 McAfee 的 SASE 解决方案提供的可操作的威胁洞察而更加丰富，该解决方案可检测 web 和 SaaS 环境中发生的云威胁。它提高了态势感知能力，推动了更好、更快的决策，并将 SOC 的效率和有效性提升到了一个新的水平。

McAfee 企业业务首席产品官 Shishir Singh 表示:“SOC 流程涉及孤立的监控和检测工具，这些工具会产生大量的安全警报，通常需要手动整理，并迫使分析师采取被动的姿态。“人工智能引导的调查就像催化剂一样，让分析师可以更轻松地编排智能、高效的工作流程。MVISION XDR 通过自动化来简化操作，提供跨所有攻击面的端到端威胁可见性、更强的上下文和态势感知能力，以便组织能够抢先应对攻击，而不是争先恐后地遏制违规。”

MVISION XDR 功能包括:

*   **高级威胁检测**:自动关联来自多个数据源的攻击遥测数据，包括端点检测和响应(EDR)、云访问安全代理(CASB)、数据丢失防护(DLP)和安全 web 网关(SWG)，并与主动威胁活动相融合，以揭示对手在整个攻击生命周期中的工作全貌。
*   **自动化威胁管理任务**:通过将最新的机器学习技术与人工分析相结合，MVISION XDR 通过人工智能指导的调查和米特 ATT & CK 映射，简化了复杂威胁活动中的分析师工作流程，从而加快调查并更快地解决问题。
*   **主动威胁搜索和优化响应**:m vision Insights 与 MVISION Cloud Security Advisor 的集成，通过对从终端到云的所有向量进行相关的安全态势评分，为安全团队提供可操作的情报，帮助他们加强安全卫生，并利用威胁组的关键上下文推进调查和分析。

Banco Delta(巴拿马)的 Ariel 表示:“MVISION XDR 的设计考虑了 SOC 体验。“它可以让分析师获得更大的控制权，以及对终端之外的威胁环境的更全面的了解，从而节省时间，并使我们能够在威胁发生或造成损害之前，通过更好地了解威胁来更审慎地采取行动。”

“威胁检测不会在真空中发生。IDC 研究总监 Chris Kissel 表示:“如果不将来自端点和非端点来源的取证数据整合在一起，描绘出更全面的杀伤链，就很难看到攻击者遍历您的环境并回答对 SOC 团队至关重要的调查问题。“XDR 是 EDR 的下一个合乎逻辑的步骤。McAfee 的 XDR 具有巨大的潜力，通过将更多类型的遥测技术与威胁英特尔原生集成到单一用户体验中进行检测和响应，实现了安全分析工具在很大程度上无法提供的功能。”
