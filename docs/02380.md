根据总裁兼首席运营官 Neelan Choksi 的说法，“由于我们的客户都试图通过软件加速商业价值的交付，测试自动化已经成为交付高质量软件竞赛中的一个重要组成部分。扩展敏捷和开发运维给组织带来了挑战，要将所有不同测试工具的必要数据整合在一起，以创建产品质量的综合图。凭借十多年来在价值流集成方面的领导地位和 Tasktop 在高质量软件交付方面的专业知识，我们很高兴地宣布测试管理支持的可用性，这是我们的客户最需要的功能之一。现在，全球 5000 强和最大的政府机构中的质量保证领导可以实现测试工具链的现代化，而不会失去对覆盖范围、质量和成本的可见性。”

Tasktop 最初为使用 Tricentis Tosca 和 Micro Focus ALM 的组织或使用多个版本和变体 ALM 的组织引入了此功能。Tasktop 在工具之间执行测试用例、测试实例和测试运行的双向数据同步。它最终将所有的测试设计和执行数据集中到一个单一的 ALM 实例中，利用 ALM 强大的仪表板功能进行全面的报告。

Tasktop 与 Tricentis 合作设计了新的集成。Tasktop Integration Hub 是将测试结果从 Tosca 传输到 ALM 的唯一解决方案，可即时创建必要的多层次测试工件集(测试、测试步骤、测试运行、测试结果和缺陷),从而将这些信息从 Tosca 传输到 ALM 进行报告。

无论组织处于测试现代化之旅的哪个阶段，新产品都是他们的理想选择。它适合那些引入测试自动化新工具的人；使用一个工具进行测试管理和测试自动化，另一个工具进行缺陷管理和报告；或者在单个组织内或跨供应商和分包商使用多个工具实例。

新的测试管理优势包括:

*   **现代化的自由:**组织可以通过向其现有的手动测试基础设施中添加新的、专门的测试自动化工具(Tricentis Tosca)来实现测试管理的现代化，从而确保产品质量的可见性和可控性。
*   **全包式报告:**他们可以使用熟悉且首选的微焦点 ALM 报告和仪表盘来报告多源数据。
*   **消除报告延迟:**他们现在可以接收近乎实时的报告。这消除了手动数据输入或多源数据操作(导出/导入、数据对齐、手动映射)的繁重工作。

Tasktop 的测试管理功能在 task top Integration Hub Ultimate Edition 中立即可用。
