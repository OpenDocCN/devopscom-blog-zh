# InfluxData 和 Sensu 提供联合解决方案来处理事件监控管道中的时间序列数据

> 原文:[https://devo PS . com/influx data-and-sensu-offer-joint-solution-to-handle-time-series-data-in-event-monitoring-pipeline/](https://devops.com/influxdata-and-sensu-offer-joint-solution-to-handle-time-series-data-in-event-monitoring-pipeline/)

*InfluxDB 企业版和 InfluxCloud 现已集成为 Sensu Core 中的首选时序数据存储，用于应用和基础设施事件监控*

**三藩市—2018 年 11 月 27 日—** 专门为指标、事件和其他时间序列数据打造的[现代开源平台](https://www.influxdata.com/products/)influx data 今天宣布与 Sensu 达成战略合作伙伴关系。两家公司合作开发了一个联合解决方案，为 Sensu 市场领先的关键应用和企业基础设施监控软件增加了处理大型时序数据工作负载的能力。

Sensu 的产品现已被金融和技术领域的大型组织使用，部署在客户自己的基础设施上，允许 IT、开发和运营团队轻松监控和收集整个基础设施堆栈的指标。然后，客户可以通过特殊的控制面板分析这些数据，以修复 IT 性能问题。

随着 InfluxData 平台的加入，Sensu 核心产品线现在可以处理大量时间序列数据工作负载，这些数据来自任何向 Sensu 发送数据的基础设施源。使用 Sensu Core 的开发人员现在可以使用 InfluxData 的任何产品来收集、处理、分析和存储更大的数据工作负载，包括 InfluxDB 开源、InfluxDB 企业版和 InfluxCloud，InfluxData 的托管数据库即服务产品。

Sensu 首席执行官 Caleb Hailey 表示:“随着越来越多的公司使用软件为客户提供价值，软件的可用性变得越来越重要。“监控已经成为 IT 组织的一项核心能力，我们的目标是提供一种可根据这些组织的需求进行扩展的产品，帮助他们消除停机时间和由此带来的收入损失。”

InfluxData 业务开发副总裁 Brian Mullen 表示:“我们很高兴与 Sensu 合作，将我们的时序数据平台带给运行多云部署的开发人员。“多云分发是现实；随之而来的是数据的指数级增长，尤其是时间序列数据。通过 Sensu，我们将两种专门构建的工具结合在一起，以优化开发人员监控多云部署的体验。”

两家公司已达成合作协议，Sensu 将把 InfluxData 产品作为其首选时间序列平台，分别纳入 Sensu Core 和 Sensu Core Enterprise 产品，向全球客户销售。更多信息，请参见[https://monitoringlove.sensu.io/influx](https://monitoringlove.sensu.io/influx)。
