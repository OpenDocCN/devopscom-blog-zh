# Instana 添加了基于 ARM 的主机支持，用于在 AWS Graviton2 上监视和跟踪 PHP 服务

> 原文:[https://devo PS . com/instana-adds-arm-based-host-support-for-monitoring-and-tracing-PHP-services/](https://devops.com/instana-adds-arm-based-host-support-for-monitoring-and-tracing-php-services/)

*首款应用监控和可观察性解决方案，可在基于 ARM 的处理器上自动进行本地跟踪和分析*

**芝加哥，2020 年 12 月 15 日—**针对云原生微服务应用的自动化[应用监控](https://instana.com/application-monitoring)和可观测性解决方案的领先提供商 Instana 今天宣布支持在亚马逊的 AWS Graviton2 上监控和跟踪 PHP 应用。第一个支持基于 ARM 的处理器的 Instana 版本，今天宣布的基于 ARM 的监控传感器使 Instana 成为第一个支持跨运行在 AWS Graviton2 上的 PHP 工作负载自动分布式跟踪的可观测性和监控解决方案。

Instana 总监兼 APM 策略师 Chris Farrell 表示:“云基础设施和平台的持续发展需要开发和运营团队的工具来跟上。“不仅仅是性能指标，Instana 针对 AWS Graviton2 的新 PHP 监控和跟踪传感器还提供了 DevOps 管理任务关键型应用程序所需的可观察性。”

与其他托管工作负载一样，Instana 的 PHP sensor for AWS Graviton2 监控基础设施和应用程序性能(包括深度代码可见性)，捕捉每个请求的踪迹，并分析每个进程。用于其他基于 ARM 的环境的 Instana 传感器预计将在未来上线。

Instana 的企业可观察性平台由自动化应用程序性能监控提供支持，可自动发现和映射所有服务、基础架构及其相互依赖关系。Instana 吸收所有可观察性指标，跟踪每个请求，分析每个流程，并实时更新应用依赖关系图，以提供开发和运营所需的上下文和可操作的反馈，从而优化应用性能，实现创新并降低风险，帮助他们提高管道的价值和效率。

今天的公告恰逢 AWS re:Invest 的最后一周，是亚马逊 AWS 生态系统应用程序监控和可观测性系列公告的第三次。上周，Instana 成为第一个原生跟踪运行在 AWS Lambda 上的 Java、Go、Python 和 Node.js 应用程序的解决方案。一周前，Instana 宣布推出新的 AWS Marketplace 产品。
