# DBmaestro 将通过任命前微软高管进行全球扩张

> 原文:[https://devo PS . com/db maestro-expand-globally-appointment-former-Microsoft-executive/](https://devops.com/dbmaestro-expand-globally-appointment-former-microsoft-executive/)

*DBmaestro 扩展到欧洲、中东、非洲、APAC 和日本*

德克萨斯州休斯顿，2016 年 4 月 21 日——数据库解决方案 DevOps 的先驱和领先提供商 db maestro([http://www.dbmaestro.com/](http://www.dbmaestro.com/))今天宣布，将扩大其全球业务重点，任命 Danny Neuberger 为欧洲、中东、非洲、亚太和日本销售高级副总裁，以进一步扩大其客户群和全球战略合作伙伴网络。DBmaestro 目前与 IBM、微软、甲骨文、CA Technologies、Perforce、Xebialabs、Serena、Clarive 和许多其他顶级技术供应商合作。

Neuberger 在过去五年中一直担任 EMC 的董事总经理和国家经理，并在微软担任过许多高级职位，包括中东和非洲地区的 SMS&P 高级总监。此次招聘紧随 DBmaestro 的 300 万美元首轮融资之后。

DBmaestro 的 DevOps for Database 解决方案为数据库开发生命周期提供了一种全面的方法，满足了企业更快交付变更的需求。DBmaestro 一直处于数据库持续交付的最前沿，提供安全数据库自动化的创新方法，并允许公司降低开发和部署成本。

DBmaestro 的联合创始人兼首席执行官 Yariv Tabac 说:“我们的解决方案目前部署在北美一些最大的企业中，是时候将重点转向欧洲和亚洲市场了。Danny 丰富的经验和行业知识已经使他成为 DBmaestro 管理团队的重要成员。他的任命标志着我们致力于将 DevOps 解决方案引入数据库。非常幸运的是，我们能够找到像 Danny 这样有能力的人来担任这一角色，我相信 Danny 将带领 DBmaestro 在全球范围内达到新的高度。”

Danny Neuberger 评论道:“我真的很高兴加入 DBmaestro 团队，并期待着与他们合作，进一步发展他们的全球客户群和合作伙伴网络。在过去的 20 年里，我一直从事软件解决方案的销售和营销，我亲眼目睹了数据库在开发运维及持续交付方面是如何被抛在后面的。DBmaestro 将这些最新的方法引入数据库，确保它不是链条中最薄弱的一环”

**关于 DBmaestro**

[DBmaestro](http://www.dbmaestro.com/) 是数据库解决方案提供商 DevOps 的先驱和领导者。其旗舰产品 DBmaestro TeamWork 支持数据库的敏捷开发、持续集成和交付。

团队合作支持开发过程管理的流线化和执行变更策略实践。该解决方案支持敏捷团队协作，同时促进法规遵从性和治理。借助 DBmaestro，组织可以通过执行部署自动化、增强和巩固安全性以及降低风险来促进数据库开发运维。

DBmaestro 的解决方案被许多大型国际公司采用，包括 Barclays、Visa、Isracard (MasterCard)、Frost Bank、Thomson Holidays、Bank Leumi 等。
