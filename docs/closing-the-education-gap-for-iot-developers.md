# 缩小物联网开发人员的教育差距

> 原文：<https://devops.com/closing-the-education-gap-for-iot-developers/>

我从正在构建 i 物联网 ( 物联网)解决方案的开发者那里听到的最常见的话是:这并不容易。事实并非如此。不是因为缺乏技能——我接触的开发人员非常有才华。在部署物联网解决方案方面，也不缺乏可供选择的技术。当处于开发阶段时，挑战往往归结为可用教育材料和学习支持的差距。

我们已经脱离了物联网的早期阶段，当时有很多谈论，但没有太多的 T2 应用。现在，我们正在进入的“现实阶段，在这个阶段，更便宜、大规模的物联网解决方案是可行的。在这个阶段，开发人员的任务是识别、构建和部署他们可能不得不解决的技术。期望今天的开发人员了解物联网解决方案所需的所有移动部件的来龙去脉，包括帮助实现这些部件的无线技术，这是一个很高的要求。

虽然开发人员在独立学习方面卓有成效，但在物联网技术方面，他们仍然需要更好地获得教育资源和工具。对于那些你在教科书或大学课程中找不到的新技术来说尤其如此。

这种更新的物联网技术的一个例子是低功耗广域网(LP wan)——具体使用 [LoRaWAN 标准](https://lora-alliance.org/about-lorawan/) 。我预计这一领域不仅会与物联网开发人员越来越相关，还会从更强大的教育资源中受益匪浅。

## 物联网发展相对较新

现代 LPWAN 技术只是在过去十年才出现，当 2015 年 LoRa 联盟成立时，LoRaWAN 成为了一个开放标准。总的来说，这是相当新的。这不是一个广泛包含在高等教育课程中的概念，尽管我们每年都看到更多的内容，但开发人员很难找到有 LoRa 设备开发经验的同行。

ABI 研究预测，到 2026 年， [LoRa 将成为领先的非蜂窝 LPWAN 技术](https://www.yahoo.com/now/abi-research-white-paper-highlights-130000340.html) 。这意味着在接下来的五年里，越来越多的开发人员需要了解 LoRaWAN 的基础知识，但可能没有基础知识或经验。

据我所见，这种差距会导致团队贪多嚼不烂。如果没有足够的培训或支持，开发人员可能会冒险构建覆盖范围不足、超出范围或预算、或者根本不符合其愿景的 LoRaWAN 部署。

## 物联网开发的复杂性

与 LoRaWAN 合作的物联网技术需要更强大的教育支持的另一个主要原因是它们的复杂性。开发一个物联网解决方案需要几种不同的技能，装备良好的团队要么必须在人员配备上加以考虑，要么——正如在许多情况下发生的那样——依靠一个开发人员来解决，并身兼数职。

构建和部署 LoRaWAN 解决方案有许多方面；例如:硬件工程、设备堆栈选择和应用适合度、网关部署和网络覆盖、云考虑事项以及部署中涉及的应用开发。有很多活动部件。

因此，如果企业和物联网供应商希望开发人员实现大规模物联网部署，我们必须提供适当的资源和培训。

## **面向 LoRa 开发者的资源和社区**

随着业界致力于更好地教育开发人员关于 LoRa 技术的方法，对于想要立即投入使用的开发人员来说，有一些好地方可以开始。这里有一些资源，您可以从中学习基础知识、构建原型、从其他开发人员那里获得建议以及测试和认证解决方案。

*   [LoRa 开发者门户](https://lora-developers.semtech.com/)—一个物联网开发者可以学习、与同行联系并通过教程、开发套件和免费网络服务器等资源加速构建的中心。对每个专业水平都有帮助，从学习 LoRaWAN 的基础知识到创建网络，再到推出全面的商业解决方案。
*   [LoRa Alliance](https://lora-alliance.org/lorawan-for-developers/)—一个非盈利性协会，其成员合作并分享使用 LoRaWAN 标准的经验。他们提供活动、网络研讨会、资源中心等。
*   [LoRa Basics Station](https://github.com/lorabasics/basicstation)—一个 GitHub 库，用于运行在符合 LoRaWAN 标准的网关上的开源软件。
*   [AWS 物联网lora wan 核心](https://docs.aws.amazon.com/iot/latest/developerguide/connect-iot-lorawan.html)—AWS 物联网开发者指南，涵盖从连接网关到监控和记录的覆盖开发和部署。
*   [AzureIoTHub](https://azure.microsoft.com/en-us/services/iot-hub/)—使用 LoRaWAN 的 ChirpStack 开源网络服务器连接到 Azure IoT Hub 的指南。

我希望高等教育机构在接下来的几年里开始将 LoRaWAN 进一步整合到他们的课程中，这将有助于填补下一波开发人员的教育缺口。与此同时，物联网供应商可以站出来，为首次学习如何使用 LoRa 等技术进行构建的开发人员提供额外的支持。这对于大规模部署的成功至关重要，我们所有物联网信徒都希望看到跨企业环境、城市、地区和其他地区的大规模部署。
