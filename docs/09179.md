 *## 1.CSS-把戏不再是印度

本周第一件事:克里斯·科伊尔的网站和时事通讯被数字海洋收购。双方承诺维护被广泛引用的前端开发**知识库**。

### 分析:企业协同作用，利用知识

撇开愚蠢的 MBA 术语不谈，这应该与 DigitalOcean 自己收集的*后端*提示、技巧和操作方法很好地融合在一起。但是，和以往一样，在一年或三年后的 T2，CSS 把戏是否还会受到尊重还不得而知。

克里斯·科伊尔 : **CSS-Tricks 加入数字海洋**

我要在这里宣布一件大事。(我的锣呢？)
…
你可以在数字海洋基础设施上建造任何东西。…我认为特别酷的一点是他们的新应用程序平台，我觉得它与像我这样的前端开发人员非常一致。…此外，他们的整个 Droplets 概念(可快速启动的超级简单服务器)在行业中具有变革性。
……
当我在 2007 年开始 CSS-Tricks 的时候，我无法想象它会增长多少。…现在，这是一项比任何一个人都要大得多的工作。…这就是 DigitalOcean 的用武之地:…他们拥有支持 CSS-Tricks 的资源，也有这样做的动机。

如果你没听说过这个网站，让我们来看看为什么[坦恩豪斯](https://news.ycombinator.com/item?id=30685139 "read the full text")是他的粉丝:

css-tricks.com 一直是最好的 CSS 网站之一——以至于我会在去通用搜索引擎之前搜索某个特定的主题。…我认为人们低估了我们今天在网络上认为理所当然的东西有多少是由这些人开创的。

但是为什么是数字海洋呢？这是克拉克斯顿:

很般配。Digital Ocean 定期发布关于如何做各种超级有用的后端工作的文章。很高兴看到他们现在也将支持更多的前端。

## 2.Peloton 玩奇怪的游戏

新任首席执行官巴里·麦卡锡正在 Peloton Interactive 进行改组。在它存在的主要原因消失后，它需要一个营救计划。

### 分析:PTON 失败？

人们不太想在家锻炼。因此，Spotify 的前首席财务官正在尝试一种新的商业模式——出售订阅，而不是硬件。你可能认为它是“剃刀和刀片”，或者“喷墨”，甚至是“价值流管理”但是这有点绝望的味道。

德雷克·班尼特、马克·古尔曼和克里斯汀·v·布朗 : **珀洛东陷入了万亿美元的幻想**

这是一个熟悉的故事:初创公司创始人让位给了会计和市场研究人员。…即使是邪教也需要会计师。新任首席执行官巴里·麦卡锡有一个计划。麦卡锡谈到了大量的小赌注和快速失败，谈到了 A/B 测试以了解消费者真正想要的是什么。
…
【它】遵循了一个陈腐的硅谷剧本。…麦卡锡对物理机器的兴趣不如对公司内容的兴趣大。“神奇发生在平板电脑上，”他说。他沉思道，也许 Peloton 屏幕应该是一个开放的平台，第三方程序员可以在那里放置应用程序。或者，该公司可以尝试喷墨打印机的商业模式，提供廉价的机器，并通过更高的月订阅费来赚钱。

但是一个*邪教*？“成为一个邪教的一部分是更好的，”萨默维尔兰说:

我的朋友们一直在减肥，我注意到他们是多么兴奋地挤在这里和那里锻炼。我注意到他们狂热的崇拜。一开始我甚至嘲笑过。然后我意识到这就是我想要的。我想成为一个崇拜者的一部分，受到非理性的激励，做出更好的决定，做得更好。…这很容易让我在好的夜晚锻炼 20%，在坏的夜晚锻炼 5%。…是的，这让我很讨厌，就像任何邪教成员一样，但我认识的人中有 80%的人和我一样热衷于此。这是技术、游戏化、心理学、应用程序力量等的一个伟大应用。

然而，Rich Duprey 认为，这种新的喷墨业务模式是“一场等待发生的灾难”:

Peloton Interactive 希望颠覆其整个业务，以扭转销售崩溃的局面。…麦卡锡希望，通过提供自行车和按月付费的订阅，这将是消费者更实惠的选择。
…
Peloton Interactive 希望用户看到保持订阅和锻炼计划的动力，但相反情况发生的风险似乎更高。…这就有可能产生大量闲置的二手库存，而这些库存可能需要打折出售。

## 3.Arm 将失去 12%-15%的员工

现在我们知道 Nvidia 的 Arm 竞标失败，这家即将上市的公司正在裁员。正如我在之前[说过的，ARM 芯片在数据中心越来越重要——尤其是那些看重**“性能功耗比”的芯片**](https://devops.com/unreliable-server-scare-information-batteries-arm-ipo-pdq/#tlv3 "Confirmed: Arm will IPO, Not be Sold to Nvidia")

### 分析:前方可能会有麻烦

这只是对财务数据的标准“美化”吗？也许不是:可能这个行业的结构性问题比我们想象的要多。

[西蒙·夏伍德](https://www.theregister.com/2022/03/15/arm_job_cuts/ "read the full text") : **裁员 15%——约 1000 人**

Arm 首席执行官雷内哈斯(Rene Haas)发给员工的一封电子邮件称:“为了保持竞争力，我们需要消除重复工作；…停止对我们未来成功不再重要的工作；想想我们如何完成工作。”…“我写这些是因为我知道，尽管这对 Arm 的未来来说是正确的，但这并不容易，”他补充道。Arm 对投资者的吸引力很大程度上在于其深厚的技术专长。因此，大部分被裁员工很可能不会直接参与 Arm 的核心业务。
……
新上市股票的投资者通常会欣赏暗示未来利润丰厚的资产负债表。因此，在首次公开募股之前降低成本是一种常用的策略。(但)Arm 发言人不同意这种说法，他们更倾向于传递这样一种信息，即裁员是一家谨慎企业不时进行的常规裁员，绝不是任何不适或战略变化的迹象。

那个“战术”真的是个东西吗？不，说[damoregonian](https://hardware.slashdot.org/comments.pl?sid=20982949&cid=62361275 "read the full text"):

我们这里有一个可笑的似是而非的声称，这是常见的做法，试图欺骗性地歪曲你的财务状况。…一家银行不会购买 20 亿美元的证券…直到他们知道该知道的一切。…你真的认为为首次公开募股提供资金的银行会去“帮一帮！看啊！去年他们的工资支出下降了 30%!那家企业经营状况良好！”

来了****了。…自 2016 年以来，Arm 的收入一直持平。这就是(软银)试图出售的原因。他们做不到，所以现在他们向公众销售，削减开支以达到目标。

那么，为什么会出现持平的情况呢？这是 [ggreg84](https://news.ycombinator.com/item?id=30686210 "read the full text") 告诉它的:

出问题的是，人们了解了英伟达的薪酬，并发现他们的薪酬明显偏低。…从 Nvidia 交易看起来不会成功的那一刻起，顶尖人才开始跳槽。
