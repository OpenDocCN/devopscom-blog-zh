# MediaOps 帮助 GitLab 启动了第一个 GitLab Commit Virtual，这是一个与众不同的 24 小时用户活动

> 原文:[https://devo PS . com/mediaops-helps-git lab-power-the-first-git lab-commit-virtual-a-24-hour-user-event-like-any-other/](https://devops.com/mediaops-helps-gitlab-power-the-first-gitlab-commit-virtual-a-24-hour-user-event-unlike-any-other/)

*GitLab 和 MediaOps 合作，将 GitLab Commit 作为一项免费的全球虚拟活动，为每个人提供参与 DevOps 讨论和与来自微软、谷歌、HubSpot 和 Forrester 的思想领袖交流的机会*

**佛罗里达州 BOCA 拉顿——**MediaOps，一个以最有力的方式讲述其故事的地方，宣布它正与 GitLab 合作，帮助推动有史以来第一次虚拟 GitLab 提交事件。为期 24 小时的会议将于 8 月 26 日举行，汇集了来自全球各地的 DevOps 杰出人士，提供免费的交互式学习体验。

GitLab Commit Virtual 社区活动有八个不同的内容专题，80 多位演讲者，包括小组讨论、演示和思想领袖演示。演讲者将展示社区的力量，并通过战略和技术讨论、经验教训、开发生命周期的幕后观察等分享 DevOps 最佳实践。

GitLab 产品营销经理 John Jeremiah 表示:“GitLab Commit 旨在成为 DevOps 学习和协作的场所，并以最终用户为中心进行定制。“在 GitLab，我们相信任何人都可以做出贡献。这项活动将为世界各地的与会者提供参与和贡献的机会，无论他们身处哪个时区。”

虚拟环境的特点是拥有超过八个不同轨道的礼堂，一个展厅，一排排展台，提供互动聊天和可下载的资源，一个虚拟照片亭，通过缩放和幕后采访进行的演讲者网络会议。它在虚拟环境中提供了最佳的物理环境，并结合了虚拟环境提供的优势和便利。

“与 GitLab 合作令人振奋，因为 GitLab 团队和 GitLab 社区都充满热情，”MediaOps 首席执行官 Alan Shimel 说。“这种热情帮助我们走得更远、更高，以确保 GitLab Commit Virtual 成为我们迄今为止举办的最大、最好的虚拟活动。对于 DevOps 世界来说，这的确是一次不容错过的盛会。”

GitLab Commit Virtual 的议程包括一系列主题，包括技术会议、现场编码、深入探讨和案例研究，涵盖了 DevSecOps、容器、Kubernetes、CI/CD、开源、全远程等等。

今年的活动有来自 DevOps 和 20 多个行业的杰出会议，包括:

*   “GitLab 的力量”，GitLab 首席执行官 Sid Sibrandij 的开场主题演讲。
*   “DevOps 作品！为什么安保没跟上？”来自 StackHawk 的 Joni Klippert 的主题演讲。
*   “政府和高度管制行业的发展合作”，美国空军 Nicolas Chaillan 的主题会议。
*   《真实世界的开发》，作者 Abel Wang，微软。
*   《为什么你的组织应该接受社区模式》，作者卡尔·卡德尼亚斯，国营农场。
*   GitLab 的 David DeSantos 和 Sam Kerr 写的“变得模糊:模糊传统 QA 和安全测试的界限”。
*   Forrester 的 Amy DeMartine 写的“市场现状:让开发人员能够发现并修复安全缺陷”。
*   Bobbi Wenzler 和 Nicole Schultz Northwestern Mutual 发表的《利用 GitLab 数据引领 DevOps 转型》。
*   GNOME 基金会的 Molly DeBlanc 写的“超越代码的协作 GitLab”。
*   “美国陆军网络学校如何使用 GitLab 创建‘代码形式的课件’”，作者 Ben Allison，美国陆军网络学校。
*   谷歌 Nathen Harvey 和 Dina Graves Portman 的“测量 DevOps”。
*   HubSpot 的 Eimear Marrinan 和 Meaghan Williams 写的“如何创造一种超越实体墙的公平和包容的文化”。

除了会议之外，与会者还可以参观虚拟展厅，与合作伙伴赞助商互动，包括 Google Cloud、Oracle、Pulumi、VMWare Tanzu、Chef、Lightstep、RedHat 和 ServiceNow。世博大厅将在整个活动期间开放，让与会者有机会随时与赞助商聊天和下载信息。

注册与会者还将收到一个链接，可在现场活动结束后的 30 天内按需访问虚拟环境。要查看完整议程并进行注册，请访问 [GitLab Commit Virtual](https://gitlabcommitvirtual.com/) 网站。

**关于 GITLAB**

GitLab 是一个全新构建的 DevOps 平台，作为 DevOps 生命周期所有阶段的单一应用程序，支持产品、开发、QA、安全和运营团队在同一项目上同时工作。GitLab 在整个 DevOps 生命周期中提供了一个单一的数据存储、一个用户界面和一个权限模型。这使得团队可以通过更高效的协作和更集中的注意力来显著减少周期时间。

GitLab 建立在开源的基础上，与其不断增长的社区(由数千名开发人员和数百万用户组成)一起工作，不断提供新的 DevOps 创新。包括 Ticketmaster、捷豹路虎、NASDAQ、Dish Network 和 Comcast 在内的 100，000 多家从初创公司到全球企业的组织信任 GitLab 能够更快地交付优秀的软件。

GitLab 是全球最大的全远程公司，在 68 个国家和地区拥有 1300 多名团队成员。
