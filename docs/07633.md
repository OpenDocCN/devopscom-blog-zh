# Thoma Bravo 对 Applitools 进行战略增长投资

> 原文:[https://devo PS . com/thoma-Bravo-makes-strategic-growth-investment-in-appli tools/](https://devops.com/thoma-bravo-makes-strategic-growth-investment-in-applitools/)

**加利福尼亚州旧金山和圣马特奥，2021 年 3 月 30 日** —专注于软件和技术支持服务领域的领先私募股权投资公司 Thoma Bravo 今天宣布，对领先的视觉测试自动化软件提供商 Applitools 进行战略增长投资。联合创始人吉尔·塞弗、亚当·卡米和摩西·米尔曼将保留公司的重要所有权。财务细节没有披露。

Applitools 成立于 2013 年，总部位于加利福尼亚州圣马特奥，为软件工程师提供基于人工智能和云技术的解决方案，帮助他们对应用程序进行自动化功能和视觉测试，并确保无缝的用户体验。该公司的视觉 AI 平台使用计算机视觉算法，使工程团队能够快速编写、运行、分析和维护测试，最终以令人难以置信的速度和更低的成本发布高质量的应用。这项投资可以让 Applitools 利用 Thoma Bravo 在软件领域的运营能力和丰富经验，加速其已经令人印象深刻的增长轨迹，并进一步扩大其创新的市场领先平台。

Applitools 首席执行官 Gil Sever 表示:“随着数字化转型的加速，越来越多的软件开发人员转向 Applitools 的跨环境测试解决方案，以构建和验证用户可以信赖的高质量软件应用。在这个动态变化的时刻，企业正在寻求安心和可靠的人工智能增强工具包，以保持数字接触点的连续性。我们期待着与 Thoma Bravo 合作，加倍发展我们的重要业务势头，推动持续的产品创新，并将我们独特的技术推向新的高度。"

“我们经济中的几乎每个行业都在经历数字化转型，客户要求完美的用户体验，Applitools 是 UX 测试自动化市场的领先创新者，”Thoma Bravo 的负责人 Carl Press 说。“疫情只是加速了这种转变，因为企业正努力在各种消费设备上实现高水平的持续质量。我们非常高兴能与 Gil 和 Applitools 团队合作，继续提供一流的人工智能解决方案。”

Thoma Bravo 副总裁 Annie Wei 表示:“Applitools 正在解决开发人员、质量工程师和数字产品所有者每天面临的问题，我们很高兴与该团队合作，因为他们使用开创性的视觉人工智能和跨浏览器测试技术继续定义和扩展视觉和功能测试自动化市场。“我们期待利用 Thoma Bravo 的运营能力来帮助推动持续增长和市场领先地位。”

贝尔德担任托马斯布拉沃的财务顾问，柯克兰&埃利斯担任法律顾问。威廉·布莱尔担任 Applitools 的财务顾问，梅塔尔·利库尔尼克·杰瓦·莱舍姆·塔尔和辛德利·奥斯丁担任法律顾问。

**关于 Applitools**

Applitools 通过可视化人工智能和超快测试云提供了下一代测试自动化平台。Applitools 使工程团队能够以令人难以置信的速度和更低的成本发布高质量的 web 和移动应用。

Applitools Visual AI 使重要的测试自动化用例现代化——功能测试、可视化测试、Web 和移动 UI/UX 测试、跨浏览器测试、响应性 Web 设计测试、跨设备测试、PDF 测试、可访问性测试和合规性测试——以显著降低的总拥有成本(TCO)转变组织以 CI/CD 的速度交付创新的方式。

来自科技、银行、保险、零售、制药和出版等垂直行业的数百家公司(包括财富 100 强中的 50 家)使用 Applitools，在任何设备和浏览器上，跨各种屏幕尺寸和操作系统，为数百万客户提供最佳的数字体验。
