# 木偶将木偶企业带入 AWS 市场

> 原文:[https://devo PS . com/puppet-bring-puppet-enterprise-AWS-market place/](https://devops.com/puppet-brings-puppet-enterprise-aws-marketplace/)

*新的 AWS Marketplace 产品增加了采用、管理和保护 AWS 云的简单性*

**俄勒冈州波特兰——2016 年 11 月 17 日**—[木偶](https://puppet.com/),自动化软件交付和操作的标准，为我们周围的一切提供动力，今天宣布木偶企业[现已在 AWS 市场](https://aws.amazon.com/marketplace/pp/B01MRIIWAR)上市。新清单建立在 AWS 和 Puppet 现有的合作基础上，使任何人都可以直接从 AWS Marketplace 启动核心 Puppet 企业服务。

Puppet 首席信息官 Nigel Kersten 表示:“如今，组织正在迅速采用云计算和 DevOps 实践，以便能够比以往更快地为客户提供新价值。“通过在 AWS Marketplace 上列出 Puppet Enterprise，我们让世界各地的组织更容易在自动化云环境中采用 DevOps 实践，这样他们就可以花更少的时间管理基础架构，将更多的时间用于推动创新和交付优秀的软件。”

Amazon Web Services，Inc . AWS Marketplace and Catalog Services 总经理 Barry Russell 表示:“许多企业依靠 Puppet Enterprise 来为其云环境提供、运营和增强保护。Puppet Enterprise 帮助客户快速、轻松地交付其软件，现在他们还可以通过 AWS market place 以极其简单的方式购买和部署软件。”

**AWS 市场上的傀儡企业**

从今天开始，AWS 客户可以直接从 AWS Marketplace 找到、启动和运行木偶企业[。现在，AWS 客户可以采用 Puppet Enterprise 作为标准的集中式方式来管理 AWS 环境中的基础架构服务和应用程序，充分利用 AWS 云的敏捷性、效率、灵活性和规模。通过 Puppet 将基础设施作为代码的声明式方法，客户可以快速从物理基础设施转移到在云中部署生产工作负载，同时降低风险并提高可靠性。](https://aws.amazon.com/marketplace/pp/B01MRIIWAR)
