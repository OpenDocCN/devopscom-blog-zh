# SauceCon 2019 演讲嘉宾简介:戴夫·海夫纳

> 原文:[https://devo PS . com/sauce con-2019-speaker-profile-Dave-hae ffner/](https://devops.com/saucecon-2019-speaker-profile-dave-haeffner/)

*这是一系列文章中的第一篇，重点介绍 [SauceCon 2019](https://saucecon.com/) 的演讲嘉宾。SauceCon 将全球的 Sauce Labs 用户和持续测试专家聚集在一起，互相学习，提升他们的自动化测试和持续交付技能。今年的活动将于 4 月 23 日至 25 日在德克萨斯州奥斯汀举行。*

SauceCon 2019 发言人戴夫·海夫纳(Dave Haeffner)表示，他涉足硒——但更准确的说法是，他吃硒、睡觉和呼吸硒。事实上，Dave 是 Elemental Selenium tip 时事通讯的作者，该通讯被成千上万的测试专业人员阅读，他还是-internet(一个非常适合编写自动化测试的开源 web 应用程序)的创建者和维护者，是“Selenium guidance”的作者，也是 Selenium 项目的活跃成员。![](../Images/99c5b6eae7a4d54e1f8fcf04bbb334cd.png)

不管他的背景如何，信不信由你，Dave 在 2009 年左右偶然发现了 Selenium IDE 的自动化软件测试。他是一名网络工程师，没有编程或软件测试背景。他和他的新队友最终将他们的自动化实践发展成了用 Ruby 编写的测试工具(事实上，他们 [是酱实验室的第一个 Ruby 客户](https://saucelabs.com/blog/the-motley-fool-makes-a-wise-investment-in-sauce-labs-technology) ！).今天，Dave 认为自己足够幸运，可以全职从事开源软件——尤其是 Selenium IDE 的开发工作。

Sauce Labs 团队很幸运有 Dave 再次出席 SauceCon 2019。在今年的活动中，戴夫的使命是打破记录和回放是一件坏事的刻板印象。他过去也相信这一点，但他想帮助消除这一耻辱。在这次演讲中，Dave 将展示 Selenium IDE 如何可靠地增强和提升您的测试实践，而不管您的团队的测试自动化成熟度或您的技术经验水平如何。“如果人们从我的演讲中只拿走一件事，那就是用 Selenium IDE 记录和回放值得一试，”他谈到他计划的演示。

想试试吗？ [下载最新版本](https://www.seleniumhq.org/selenium-ide/) 。

工作之余，戴夫花时间陪伴妻子和女儿，探索他们称之为家的新城市——特拉维夫。他也喜欢学习和练习希伯来语。

在第三届年度 Sauce Labs 用户大会 SauceCon 2019 上，你可以听到戴夫和其他 30 多位专家的演讲。要查看完整的演示列表或进行注册，请访问 [SauceCon](http://saucecon.com/) 网站。
