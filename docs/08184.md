# z/OS 的开源语言和工具

> 原文：<https://devops.com/open-source-languages-and-tools-for-z-os/>

**z/OS 的开源语言和工具**

让您的开发人员能够使用他们喜欢的开源语言。我们提供许多开源语言和工具——包括 Git、R 和 Python——允许任何开发人员利用大型机的强大功能，而不必花费数月时间学习新技能。

**去**

Git 是一个版本控制系统(VCS ),用于跟踪计算机文件的变化，并协调多人对这些文件的工作。它主要用于软件开发，但也可以用来跟踪任何文件中的变化。git 的 z/OS 端口支持文本文件在 ISO8859-1 (ASCII)和 IBM-1047 (EBCDIC)字符编码之间的自动标记和转换。

**痛击**

Bash 是一个 sh 兼容的 shell，它为用户提供了一个命令行解释器，该解释器通过文本窗口和脚本执行功能提供了一个传统的界面。它是 Linux 和 Mac OS X 上的默认 shell。

**野牛**

Bison 是一个用于创建解析器的编程工具。它通常用于构建自动化和构建自动化生成，并将带注释的上下文无关语法转换成 C、C++或 Java 解析器。Bison 被 PHP、Bash、Ruby 和许多其他工具使用。

**bzip2**

bzip2 是一个压缩单个文件的数据压缩器(它不是一个文件归档器)。它通常将文件压缩到统计压缩器的最佳可用技术的 10%到 15%以内，同时压缩速度大约是两倍，解压缩速度是六倍。这个版本的 bzip2 不是为了利用 z Systems 架构而编写的；gzip 可能最适合这些需求。

**卷曲**

cURL 是一个[开源](https://devops.com/developers-guide-to-open-source-spring-cleaning/)工具，可以通过使用支持的协议之一(HTTP、HTTPS、FTP、FTPS、TFTP、DICT、TELNET、LDAP 或 FILE)在服务器之间传输数据。这个工具可以在没有用户交互的情况下工作。cURL 功能包括代理支持、用户认证、FTP 上传、HTTP post、SSL 连接、cookies、文件传输、恢复等等。

不方便

Diffutils 是一个由几个程序组成的包，用于查找文件之间的差异。例如，diff 命令显示两个文件之间的精确行差异。

**Gzip**

Gzip 是一个流行的数据压缩/解压缩程序(我们移植的工具已经使用这个工具进行了压缩)。这个版本的 Gzip 也经过了修改，可以利用 z/OS EDC 压缩硬件(如果您的系统上有)。

**libssh2**

libssh2 是一个软件库，它允许开发人员在一个易于使用的自包含包中使用 SSH-2 网络协议。这个包只包含供其他程序或实用程序使用的 LIBSSH2 库。

**m4**

M4 是一个通用宏处理器。大多数用户需要 M4 只是因为它是 Autoconf 或 Bison 的一个依赖项，尽管这种语言也用于其他环境，如文本模板重用、文本编辑和文本处理应用程序。

**制作**

Make 控制从程序的源文件生成程序可执行文件和其他非源文件。Make 允许用户在不知道过程细节的情况下构建、安装和更新包。

**Mktemp**

mktemp 实用程序获取一个给定的文件名模板，并覆盖它的一部分来创建一个唯一的文件名。Mktemp 还管理临时文件和目录的创建，这些文件和目录允许用户避免清理永久不需要的文件。

**OpenSSL** 的缩写形式

OpenSSL 是一个软件库，用于需要保护通信免受窃听或需要确定另一端的身份的应用程序中。

**Pkg-Config**

pkg-Config 是一个帮助编译应用程序和库的编程工具。它帮助您在命令行上插入正确的编译器选项，而不需要在哪里找到 glib(或其他库)的硬编码值。

**Sed**

sed(流编辑器)是一个用于解析和转换文本的工具。它通常用于使用模式匹配提取文件的一部分，或者替换文件中多次出现的字符串。

**须藤**

sudo 是一个命令行实用程序，允许管理员授权，让某些用户能够以超级用户或其他用户的身份运行命令，同时提供命令及其参数的审计跟踪。

**拉开拉链**

UnZip 是一个解压工具，用于解压在。zip 格式。Unzip 还与 MS-DOS 的 PKZIP 和 PKUNZIP 实用程序以及 Info-ZIP 兼容，并支持 bzip2 压缩方法。

**Vim**

Vim 是一个高度可配置的文本编辑器，可以非常高效地创建和更改任何类型的文本。

**Zip**

Zip 是一个压缩和文件打包/存档工具。Zip 将一个或多个压缩文件以及这些文件的相关信息放入一个 ZIP 存档中。

**zlib**

zlib 是一个通用的无损数据压缩库，提供内存压缩和解压缩功能，包括对未压缩数据的完整性检查。
