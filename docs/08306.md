# 实现云游戏需要什么？

博彩业正在蓬勃发展。美国有 2 . 11 亿人玩游戏，这是这个星球上增长最快的娱乐形式之一。在 Twitch 流媒体时代，游戏甚至抢了传统体育观众的风头。平均而言，18 至 25 岁的年轻游戏玩家花在在线游戏上的时间比看传统体育比赛的时间多一个小时(T2)。

游戏也成为了与他人联系的一种方式——65%的玩家表示他们会与他人在线或面对面玩游戏。一些软件开发团队甚至鼓励在工作场所玩[游戏来提高生产力。出于这个原因，在线合作游戏、RPG 和沙盒游戏(如《《我的世界》》)出现了巨大的转变。此外，游戏越来越基于云，购买、下载和游戏保存都在云中进行。尽管如此，许多人缺乏在本地运行复杂游戏所需的硬件。所有这些因素都预示着一个即将到来的远离 PC 和基于控制台的处理的转变。](https://devops.com/cooperative-gaming-software-teams-secret-to-success/)

我最近会见了产品经理 Noam Salinger，了解游戏的未来以及游戏行业的软件团队应该如何准备。对塞林格来说，“云游戏是下一步。”从控制台转向基于云的远程处理将会产生巨大的连锁反应。降低游戏门槛可以将数百万玩家引入生态系统，并进一步推动该行业的发展。然而，云游戏存在重大障碍。这种转变将需要大幅降低延迟、5G 网络和高效计算优化。

## 云游戏的障碍

“云是任何游戏公司不可或缺的一部分，”塞林格说。越来越多与游戏相关的功能被推到网上。已经有在线游戏市场、基于云的账户管理和远程游戏保存。这在手机游戏中尤为普遍。但是，计算密集型游戏的实际处理又如何呢——这些工作负载会很快转移到云中吗？

这种基于云的游戏可能很快就会实现。然而，塞林格预见了云游戏的一些主要障碍:

### 1.软件组件

完全转向云游戏需要有合适的软件基础设施，并得到供应商的支持。塞林格说，这可能是软件行业云游戏的最大障碍，“它仍然不够成熟。”

### 2.业务组件

此外，游戏平台公司可能会避免使用云游戏，因为担心它会蚕食他们现有的硬件销售渠道。另一方面，云游戏提供了一个很好的机会来采取更多的基于订阅的模式，例如允许用户从大量的游戏套件中玩任何他们想要的游戏。类似于网飞、Hulu 或 Peacock 等流媒体服务，我们可以看到类似的平台风格在游戏中出现。“他们将别无选择，只能在某个时候参与，”塞林格说。

### 3.网络组件

正如大多数游戏玩家可以证明的那样，在线游戏很容易受到网络问题的影响，从而对游戏产生负面影响。塞林格预测，5G 网络的普及将是推动云游戏市场向前发展的必要条件。5G 可能是一种通用方法，可以为更广泛的最终用户游戏玩家实现实时连接。

即使软件开发商优化了游戏处理速度，糟糕的网络也会给数据吞吐量增加足够的延迟，抵消这些收益。5G 可以加快这一过程，但这比预期的要长得多。一些人[估计](https://www.inc.com/magazine/202002/jennifer-alsever/5g-wireless-network-broadband-high-speed-gigabit-technology.html0) place 5G 最终将在 2023 年的某个时候达到全速，届时将会有更多的天线建成，现有的物理基础设施也将升级。

### 4.延迟组件

云游戏的另一个重大障碍是在游戏供应商方面解决实时远程计算。塞林格说:“这就是潜伏性真正发挥作用的地方。”由于游戏客户端将进行许多远程调用，优化云 CPU 利用率对于减少请求响应时间至关重要。

云游戏将需要预测用户行为的并行运行进程。与现代 web 应用程序同时触发多个后端认证、地理定位、用户数据库等请求的方式类似，实时云游戏将需要许多不同的进程并行运行。服务器可以在请求到来之前分配资源，使用使用模式的预测分析在用户行为之前触发组件。

为了实现智能资源分配，Salinger 描述了与操作系统无关的内核模块如何负责资源分配，不断适应应用程序工作负载的变化。他说，CPU 级别的低级资源分配、分配内存、I/O 和执行顺序必须对每个特定的应用程序都是最优的。

这种过程已经被一些广告技术公司应用于生产中，以实现实时广告竞价。例如，通过优化云 CPU 利用率，Perion 能够更快地执行、处理更多响应，并使用更少的机器来执行相同的任务，从而减少每个请求的往返时间。类似的原理可以用来推进云游戏。

渲染复杂的虚拟环境是高度资源密集型的，高性能显卡非常昂贵。随着新的 AR/VR 模式的出现，视频游戏的复杂性只会增加。云游戏可以帮助应对日益增长的计算需求，并取代对本地计算硬件的需求。在这个过程中，这一举措可能会带来更多的游戏玩家，扰乱游戏机大战，并消除由来已久的 PC 对 Mac 兼容性问题。

迁移到基于云的方法也更符合当今快节奏的软件发布节奏。推送更频繁的补丁有助于更快地解决问题。但它也可以通过不断引入新的水平和目标来帮助保持参与度。

游戏生态系统的崛起不容忽视。但就目前而言，云游戏的更广泛渗透仍面临重大挑战。减少延迟的云计算优化、5G 的主流采用和软件基础设施的重大平台迁移只是在真正基于云的游戏体验提升到下一个级别之前需要解决的许多问题中的一部分。
