# Docker 为开发者扩展可信内容产品

> 原文：<https://devops.com/docker-expands-trusted-content-offerings-for-developers/>

*与 AWS、Datadog、Mirantis、Red Hat、VMware 和其他行业领导者的新合作将可信应用构建模块的访问权限扩展至 800 多万注册 Docker 开发人员*

**加利福尼亚州帕洛阿尔托，2021 年 5 月 27 日**–[Docker，Inc .](https://www.docker.com/)今天宣布扩展该公司面向软件开发人员的可信内容产品。在今天的公告中，发布了 Docker Verified Publisher 计划，该计划提供对 Docker 差异化和可信内容的访问，开发人员可以将这些内容用作其应用程序的可靠构建模块。Datadog、Red Hat 和 VMware 是最近加入该计划的三家发行商，在过去的六个月中，已经有 100 家公司加入了该计划。除了通过 Docker 分发的可信内容，该公司还宣布即将从亚马逊网络服务公司(AWS)和 Mirantis 向公共和私人注册中心提供 Docker 官方图像。在另一份新闻稿中，Docker [宣布了对 Docker 协作应用开发平台](https://www.docker.com/press-release/docker-advances-collaborative-application-development-platform)的更新和增强。

Docker 首席执行官 Scott Johnston 表示:“我们非常高兴地宣布，Docker 验证发行商计划将面向更多发行商，并通过更多注册中心向更多开发者分发 Docker 官方图像。“这极大地扩展了开发人员补充 Docker 官方映像的选择，并巩固了 Docker 平台和 Docker Hub 作为可信、安全容器映像的事实上的标准。”

对于开发团队来说，安全的软件供应链正变得越来越重要，因为他们在构建应用程序时努力降低暴露于恶意内容的风险。在每个阶段使用可靠的内容可确保应用程序的安全性，并最大限度地减少解决安全问题所花费的时间和金钱。有了 Docker，开发者不再需要担心自己图片的来源。相反，他们可以在应用程序中快速、自信地发现和使用来自已知、可信来源的图像。Docker Verified Publisher 产品包括流行的开发人员组件，如 VMWare 的 Bitnami 和 Spring 软件、RedHat Universal Base Images (UBI)、Canonical Ubuntu 等。

利用 Docker 官方图像和现有的 [Docker 开源计划](https://www.docker.com/blog/expanded-support-for-open-source-software-projects/)的成功，Docker 验证发行商计划扩展到商业内容，为属于应用交付管道一部分的开发人员整合可信解决方案。借助 Docker Verified Publisher 计划，开发人员可以放心地快速构建、共享和运行应用程序，因为他们知道每个组件在软件供应链的每个阶段都是安全的。

数百万开发者依赖 Docker Hub 作为他们可信内容的主要来源，包括 Docker 官方图片和 Docker 验证的发布者图片。开发者很快就可以灵活地从他们选择的注册中心下载这些图像，包括[亚马逊弹性容器注册中心公共注册中心](https://gallery.ecr.aws/)(亚马逊 ECR 公共注册中心)和[米兰蒂斯安全注册中心](https://www.mirantis.com/software/mirantis-secure-registry/)。这为开发人员提供了极大的灵活性，使他们可以使用自己喜欢的注册表从 Docker 提供的可信内容中获益。

超过 200 家 ISV 使用 Docker Hub 向开发人员分发他们的软件，看到 Docker“经验证的发行商”徽章的用户知道他们提取的图像直接来自发行商，并得到发行商的支持。超过 1300 万开发人员每月从近 800 万个存储库中提取 130 亿个图像， [Docker Hub](https://hub.docker.com/) 是业界领先的公共容器注册中心。Docker Hub 为开发人员提供了最大广度和深度的容器映像，并在构建和共享云原生应用程序方面发挥着核心作用。

**Docker 支持的可信内容**

**亚马逊 ECR 总经理 Nick Coult&亚马逊 ECR Public，AWS**

“我们的客户期望他们的容器应用程序具有最高级别的可用性和性能，对于许多采用容器的客户来说，容器注册中心是性能和可用性的关键要素之一。有了 Amazon ECR 和 Amazon ECR Public，AWS 客户可以依赖一个高可用、高性能的容器工件注册中心，该注册中心与他们的容器应用程序一起工作。很快，使用 Docker 官方图像的 AWS 客户就可以利用亚马逊 ECR Public 的高可用性和快速下载。”
