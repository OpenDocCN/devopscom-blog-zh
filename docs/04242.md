DevOps 企业峰会是面向在全球范围内实施 DevOps 原则和实践的大型复杂组织的领导者的顶级行业会议。本次活动的主题是“深入了解 DevOps”，展示了全球领先品牌利用 DevOps 原则和实践在日益软件驱动的经济中蓬勃发展的演讲。会议由革命性的技术领导者领导，他们在各行各业的大型复杂组织中推动这些实践。主题演讲包括行业名人，以及来自知名公司的演讲者，他们分享了从企业 DevOps 计划中获得的经验。

**关于 IT 革命**

IT 革命通过出版、活动和研究聚集了技术领导者和实践者。我们的目标是提升技术工作的状态，量化与次优 IT 性能相关的经济和人力成本，并改善 IT 专业人员的生活。
