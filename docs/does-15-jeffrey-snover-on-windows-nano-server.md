# 在 Windows Nano 服务器上做 15–杰弗里斯诺弗

> 原文：<https://devops.com/does-15-jeffrey-snover-on-windows-nano-server/>

在最近的 2015 年 DevOps 企业峰会上，我最喜欢的演讲之一是微软杰出工程师杰弗里·斯诺弗，他讲述了他 16 年来为市场带来今天所谓的 Windows 纳米服务器的追求。有趣的是，当我与 Gene Kim 谈论他认为 doe 的一些最佳演示时，他也提到了斯诺弗的演示。

没有 Windows 的 Windows Server 曾经被微软的任何人视为异端。正如杰弗里在下面的采访中和台上向我转述的那样，他为此付出了传福音的代价。但是今天我们有一个版本的 Windows Server，它的占用空间只有普通 Windows Server 的 1/20。它运行在大型机、命令行上，并且几乎拥有(没有双关语的意思)云中服务器所需的所有功能，而没有 Windows 通常带来的 GUI 开销。

对我来说同样重要的是，Jeffrey 也让我们看到了在塞特亚·纳德拉的微软工作是什么样子。这听起来与史蒂夫·鲍尔默担任首席执行官时大不相同。听起来也很清新。
