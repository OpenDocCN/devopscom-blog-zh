# Akamai 宣布对全球最大、最值得信赖的云交付平台进行新的增强

> 原文：<https://devops.com/akamai-announces-new-enhancements-to-worlds-largest-and-most-trusted-cloud-delivery-platform/>

*战略平台更新，旨在支持安全的云采用、卓越的 web 和应用性能及可用性、* *高质量和可扩展的视频流以及简化的应用开发*

马萨诸塞州剑桥。–2018 年 4 月 16 日**–**全球最大、最值得信赖的云交付平台 Akamai Technologies，Inc .(纳斯达克股票代码:AKAM)今天宣布了重大改进，旨在提高网站、应用和网络安全性，同时不影响数字应用的性能。通过这些补充，Akamai 的云交付平台被设计为数字业务的重要基础，具有支持数字化转型、下一代网络架构和越来越受欢迎的 DevOps 工作流的安全性和敏捷部署能力。

当今的现代企业组织在充满风险和不确定性的数字环境中运营。为了保持生产力、安全性和竞争优势，组织必须采取措施来发展其总体网络架构。

Akamai 网络安全和性能高级副总裁兼总经理 Ashutosh Kulkarni 表示:“Akamai 在过去二十年中一直致力于构建和改进市场上最大、最可靠的云交付基础设施。“我们知道，我们的客户正在一个竞争日益激烈、快速发展的数字环境中运营。为了帮助他们推动客户的差异化和价值，Akamai 平台正在提供自动化、工具和技术，以安全地将更多应用程序迁移到云，简化无缝 web 和移动体验的交付，并提高开发人员的灵活性，而不会影响混合环境的安全性或质量。”

**对抗安全风险的新增强功能，包括来自机器人的安全风险**

随着企业将其数字运营转移到云，系统暴露在 DDoS 攻击、应用程序级攻击和日益增多的僵尸程序威胁的风险之下。Akamai 的 [2017 年互联网安全状况报告](https://www.akamai.com/us/en/about/our-thinking/state-of-the-internet-report/global-state-of-the-internet-security-ddos-attack-reports.jsp)发现，超过 40%的在线登录尝试是恶意的， [Ponemon Institute](https://content.akamai.com/us-en-pg10079-the-cost-of-credential-stuffing.html) 报告称，基于这种攻击载体的违规行为平均每年会给数字企业造成 6000 万美元的损失。

Akamai 平台的新增强旨在管理机器人带来的日益增长的挑战，并减轻凭据滥用带来的威胁。Akamai Bot Manager 增加了保护应用程序和 API 的功能，包括一个用于移动应用程序的客户端 SDK。对于有可能通过体积 DDoS 攻击淹没组织的应用和基础设施的较大僵尸网络，Akamai 正在为其清理中心部署额外的容量，以缓解威胁数字业务的不断增长的 DDoS 攻击量。“Akamai 通过 TLS 1.3 为 HTTPS 无处不在提供安全浏览和支持，并通过增强的安全分析对不断变化的威胁和威胁参与者提供无与伦比的可见性。

**零信任时代的安全访问**

随着传统企业安全边界的消失和企业应用程序向云的迁移，安全的统一访问和单点登录变得势在必行。Akamai 的企业应用程序访问旨在使 IT 和安全团队能够转向零信任安全架构，同时简化安全访问并减少与云迁移、解散企业边界和日益增加的移动工作人员相关的攻击面。

**快速、安全的网络和移动体验**

根据 [Akamai 的零售业绩](https://www.akamai.com/uk/en/about/news/press/2017-press/akamai-releases-spring-2017-state-of-online-retail-performance-report.jsp) [报告](https://www.akamai.com/uk/en/about/news/press/2017-press/akamai-releases-spring-2017-state-of-online-retail-performance-report.jsp)的数据，网页加载时间延迟两秒会增加 103%的跳出率，53%的移动浏览器会留下一个需要超过三秒才能加载的页面。

随着应用架构的发展和数字应用变得越来越复杂，企业确保始终如一的高客户体验变得越来越具有挑战性。Akamai 新部署的功能增强了其平台监控、优化和验证应用性能的能力，为其 mPulse 真实用户监控解决方案增加了感知性能、页面构造和应用交付指标。

一旦发现问题，一些解决方案的固有功能将被设计来确保应用参与和体验的完整性，包括:Ion acceleration 解决方案的脚本管理和资源优化功能、数字资产优化功能和对 Brotli 压缩算法的支持、图像管理器优化以及增强的移动应用性能 SDK 中的网络优化功能。

Skyscanner 服务经理 John Paris 表示:“客户体验对我们的业务至关重要，我们很高兴能与 Akamai 合作，了解网络性能如何影响和推动我们在全球众多站点的关键业务指标。“我们现在可以利用来自 mPulse RUM 的可行见解来监控和优化数字体验，这些见解有助于我们确定修复和增强的优先级，从而产生最大的业务影响。”

**开发运维与安全的交集:开发运维**

随着越来越多的企业致力于在加速开发周期和“安全是每个人的责任”这一理念之间找到平衡，额外的新平台功能满足了将安全性融入开发运维的需求。Akamai 使用其旗舰 web 应用程序防火墙 Kona Site Defender 增加了对脚本的支持，以自动化应用程序安全，并改进了对 API 保护的管理和版本支持。此外，Akamai API Gateway 提供了一个集成的解决方案，旨在加速、保护和管理 API，包括公开、验证和管理对 API 的访问，以最大限度地提高收入并防止误用和滥用。

**OTT 内容和广告的规模化质量**

Akamai 继续专注于将传统广播电视的质量、性能和可靠性引入互联网，同时保护付费电视和 OTT 服务提供商、广播公司和消费者免受越来越多的网络威胁。流媒体交付、货币化和安全性方面的最新创新包括:媒体服务直播，具有 liveOrigin 功能，可显著降低延迟，同时保持出色的视频质量；其广播运营控制中心提供实时监控和问题检测服务；动态广告插入(DAI)功能有助于扩展可寻址广告；以及旨在帮助确保服务和内容可用性的安全解决方案。

“建立信任是 Akamai 产品的核心。Akamai 网络部门总裁兼总经理 Rick McConnell 表示:“不仅要相信我们正在为网络或媒体内容提供尽可能高的性能，还要相信这些内容不会被窃取和破坏。“随着我们的客户对其业务进行数字化转型，并满足其用户对丰富、高质量内容的需求，Akamai 致力于帮助他们提供卓越、安全的体验。”
