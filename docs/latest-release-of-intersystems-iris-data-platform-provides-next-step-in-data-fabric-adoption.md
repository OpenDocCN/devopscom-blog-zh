# 最新发布的 InterSystems IRIS 数据平台为数据结构的采用迈出了新的一步

> 原文：<https://devops.com/latest-release-of-intersystems-iris-data-platform-provides-next-step-in-data-fabric-adoption/>

*自助服务分析功能与用于实施数据结构计划的扩展功能相结合，为企业用户提供了无与伦比的分布式企业数据访问和探索能力*

**马萨诸塞州剑桥，2021 年 6 月 17 日—** [InterSystems](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.intersystems.com%2F&esheet=52446734&newsitemid=20210617005538&lan=en-US&anchor=InterSystems&index=1&md5=087e36434266df84937838f6f3ad3fec) ，一家致力于帮助客户解决最关键的可扩展性、互操作性和速度问题的创造性数据技术提供商，今天宣布了其最新发布的 [InterSystems IRIS 数据平台](https://cts.businesswire.com/ct/CT?id=smartlink&url=http%3A%2F%2Fwww.intersystems.com%2Fproducts%2Fintersystems-iris%2F&esheet=52446734&newsitemid=20210617005538&lan=en-US&anchor=InterSystems+IRIS%26%23174%3B+data+platform&index=2&md5=d57166eabe5e258931bf8022ee228cb3)，该平台包括自适应分析功能和用于分析的附加 SQL 扩展，以改进企业数据结构架构的实施过程。

[去年通过与 AtScale](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.intersystems.com%2Fnews-events%2Fnews%2Fnews-item%2Fintroducing-intersystems-iris-adaptive-analytics-new-solution-expands-self-service-business-intelligence-capabilities-within-intersystems-iris-data-platform%2F&esheet=52446734&newsitemid=20210617005538&lan=en-US&anchor=Announced+last+year+through+its+partnership+with+AtScale&index=3&md5=fdfff1402a2bc563a1e36f917288229c) 的合作宣布，InterSystems IRIS 中的自适应分析为企业用户提供了卓越的易用性和自助服务分析功能，以一致的格式可视化、分析和查询来自多个来源的实时数据。这使用户能够获得他们需要的信息，以便做出及时而准确的业务决策，而不管数据的来源、格式、速度或年限，而无需成为数据设计或数据管理方面的专家。

InterSystems IRIS 的最新版本加快并降低了实施企业数据结构的复杂性，企业数据结构是一种现代化的数据管理架构方法，可集成和协调整个企业中不同的分布式数据，并使其按需可用。

**自助式自适应分析能力**

Adaptive Analytics 进一步扩展了这些功能，使 fabrics ' *更智能'*'支持业务用户自助服务，并使数据可用于大量应用程序，以满足各种关键业务需求，包括风险管理、业务管理报告、决策支持和法规遵从性。

作为[智能数据结构](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.intersystems.com%2Fsmart-data-fabrics%2F&esheet=52446734&newsitemid=20210617005538&lan=en-US&anchor=smart+data+fabrics&index=4&md5=e1266d058b429695e0106aeb61556b46)的核心组件，InterSystems IRIS 还能更快实现价值，并支持智能预测和规范服务。

InterSystems 数据平台副总裁 Scott Gnau 表示:“当今企业面临的最大数据挑战之一是整合来自多个系统和孤岛的数据，并将其转化为可操作的见解。“企业数据结构作为解决这一问题的解决方案应运而生，因为它们允许数据留在原处，使企业能够最大限度地发挥其先前技术投资(包括数据湖和数据仓库)的价值，而不必“淘汰和替换”任何现有技术。InterSystems IRIS 的最新更新整合了组织快速轻松地实施数据结构所需的关键功能，以满足其最具挑战性的业务需求并充分利用其数据。”

奥纬咨询(Oliver Wyman)旗下 Celent 公司资本市场主管 Monica Summerville 表示:“金融服务行业对数据结构等方法表现出了浓厚的兴趣，以帮助满足行业参与者快速变化的需求。通过将分析直接嵌入数据管理平台，智能数据结构将这些计划向前推进了一步，消除了将数据移动到不同环境的需要，并减少了支持实时和接近实时使用情形的延迟。Celent research 发现，金融服务公司正在寻求自助服务方法和先进的数据管理技术来提高效率，遵守风险和监管报告，并为客户提供价值。”

**用于分析的附加 SQL 文件扩展名**

作为最新版本的一部分，用于分析的额外 SQL 扩展使数据团队能够更快、更高效地访问用于[机器学习、BI 和报告](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwww.intersystems.com%2Fproducts%2Fintersystems-iris%2Fanalytics%2F&esheet=52446734&newsitemid=20210617005538&lan=en-US&anchor=machine+learning%2C+BI%2C+and+reporting&index=5&md5=90ac17ad3682e1c1a5bdf4c46049a48e)用例的数据，而 InterSystems 在强大的嵌入式分析功能方面的持续投资将使 InterSystems IRIS 用户的查询性能提高 30%。

作为下一代数据平台，InterSystems IRIS 提供强大的数据库管理、集成、分析、应用程序开发和 API 管理功能，所有这些功能都集成在一个全新构建的产品中，从而加快价值实现、简化架构并降低总拥有成本。
