如果知道其他云用户是否看到了与您相同或相似的攻击，难道不是很有帮助吗？关于云应用程序的安全情报不仅仅局限于您作为企业拥有和运营的云应用程序，它为针对大型云应用程序的攻击可见性开辟了一个新的领域。

在 AWS re:Inforce 2019 期间，Sumo Logic 宣布将扩展其机器分析和智能平台，以包括 AWS GuardDuty。这项新服务被称为亚马逊守卫职责的全球情报服务，它不仅仅是一个数据聚合和报告游戏。这项新服务通过使用 AWS GuardDuty 报告多个 Sumo Logic 客户的攻击信息，提供了关于 GuardDuty 数据的附加上下文。本质上，这是一种跨云报告威胁情报的“众包”方法。

在本期 DevOps Chat 中，Sumo Logic 的高级工程经理 David Andrejewski 将与我们一起讨论这种新的、更广泛的威胁情报服务。更多关于亚马逊 GuardDuty 全球情报服务的信息可在[新闻稿和网站](https://www.sumologic.com/application/aws-guardduty/)获得。
