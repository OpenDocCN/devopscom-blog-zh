# 调查显示开源、DevOps 技能需求之间的相关性

> 原文：<https://devops.com/survey-shows-correlation-between-open-source-devops-skills-demand/>

本周由 Linux 基金会发布的一项调查表明，企业 IT 组织现在更愿意雇佣既有开源软件又有 DevOps 经验的 IT 专业人员。

这项调查是与 2U，Inc .的全球领先在线学习平台 edX 合作进行的，调查了 1，672 名开源专业人员和 559 名负责招聘开源专业人员的受访者。结果显示，对该专业知识的需求与对 DevOps 的熟悉程度之间也有很高的相关性。

在最受欢迎的技能方面，69%的雇主表示他们正在寻找拥有云和[容器技术](https://containerjournal.com)专业知识的 IT 专业人员，其次是 Linux (61%)和 DevOps/GitOps/DevSecOps 专业知识(57%)。另有 40%的人在寻找具有网络安全技能的人。

超过四分之三的受访 IT 专业人员(76%)目前使用 Linux (76%)，而 60%的人使用过云/容器技术，53%的人有 DevOps/GitOps/DevSecOps 经验。超过四分之三(79%)的人认为熟悉 DevOps 非常重要(44%)或极其重要(35%)。

Linux 基金会负责培训和认证的高级副总裁兼总经理 Clyde Seepersad 表示，在工资和福利方面一切平等的情况下，更多的 IT 专业人员更愿意为那些显然被视为开源软件领导者的组织工作。想要招募最优秀人才的组织应该努力让自己被视为开源软件项目的重要贡献者，以吸引最优秀的 it 人才。

大约一半的受访者目前为 IT 供应商工作，但其余的代表了通常使用开源软件的某种类型的最终用户组织。他指出，这种分布表明，对开源软件专业知识的需求现在已经远远超出了构建商业应用程序和 IT 平台的 IT 供应商。

总体而言，调查发现 93%的雇主难以找到足够的技能人才，近一半的受访者(46%)计划在未来六个月内招聘更多具有开源软件专业知识的 IT 专业人员。整整 86%的人说雇佣具有开源软件专业知识的 IT 专业人员是优先考虑的事情。超过三分之二的雇主(69%)表示，他们更有可能雇佣有证书的开源专家。总共有 90%的雇主表示，他们将为员工获得认证付费，58%的雇主表示，开源 IT 专业人员的工资通常更高。

总共有 81%接受调查的开源软件专业人员也表示他们计划在今年增加认证。近四分之三(73%)的人还指出，他们很容易找到新工作。调查显示，大约三分之一的受访者在过去一年中要么离职，要么换了工作。三分之二的开源软件专业人士表示，更高的薪水会阻止他们离职。

超过四分之三的 IT 专业人员(77%)还指出，他们将从额外的网络安全培训中受益。

然而，只有 43%的雇主认为培训是缩小当前技能差距的一种手段。另有 41%的人表示，他们聘请了顾问来填补这些缺口。只有 16%的受访者表示，他们愿意因为技能差距而推迟项目。

很明显，It 供应商和企业 IT 组织在许多方面都在争夺相同的人才。企业组织面临的挑战是确保他们有足够高的知名度，被认为是 IT 专业人员可以做出有意义的开源软件贡献的地方。
