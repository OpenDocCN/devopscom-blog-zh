# DevOps 聊天:Cyxtera 如何使用 DevOps 让 AppGate SDP 变得更好

> 原文：<https://devops.com/devops-chat-how-cyxtera-uses-devops-to-make-appgate-sdp-better/>

在这次 DevOps 聊天中，我们采访了 Cyxtera 公司的 Kurt Glazemakers。Kurt 是 Cyxtera 安全和分析团队中软件定义的边界团队的首席技术官。Kurt 讨论了他的团队如何使用 DevOps 方法来使 AppGate SDP 更好，以及如何反过来使其更好地用于支持 DevSecOps 的组织。

如果你对 Cyxtera 不熟悉，那么它就是收购 57 个数据中心和 CenturyLink 的协同定位业务(century link 随后用这笔钱完成了对 Level 3 的收购)，以及收购 Cryptzone、Catbird、Easy Solutions 和 Brainspace 的公司。该公司将自己标榜为“安全基础设施公司”，现在是全球领先的主机托管公司之一。由 Terremark 的大部分前高级管理团队管理，terre mark 本身已被威瑞森收购(哦，我们在科技世界编织了一个多么复杂的网络)，Cyxtera 现在是一个价值 28 亿美元的企业。

在我们的聊天中，Kurt 就 DevOps、自动化和 CI/CD 如何帮助 Cyxtera 的产品变得更好给了我们一些深刻的见解。和往常一样，下面是我们对话的流媒体，后面是文字记录。
