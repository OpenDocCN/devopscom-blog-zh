# 能效和数据中心

> 原文:[https://devops.com/energy-efficiency-and-the-data-center/](https://devops.com/energy-efficiency-and-the-data-center/)

在减少二氧化碳排放的竞赛中，IT 行业一直处于领先地位。减少排放对环境和公司利润都有好处。众所周知，数据中心是一个贪婪的数据消费者。但是数据中心消耗的调查结果与趋势保持同步。即使面对数据中心容量和性能的显著增长，它们的能耗自 2010 年代以来也只增加了几个百分点。这种几乎持平的能源消耗率主要是向云计算的转变以及向太阳能和风能的转变的结果，这是有意义的。

对可再生能源的投资有助于压制对整体 IT 碳足迹的批评。众所周知，为数据中心供电需要消耗大量能源，因此所有者和运营商一直在寻找提高效率和减少碳排放的方法。如今，组织有了更新更好的方法来提供适当支持现代应用的性能，同时牢记世界要求的能效和碳减排。

## 最大限度地减少能源消耗和二氧化碳排放

我们是怎么到这里的？几十年来，半导体工艺的微型化与摩尔定律的实现最终导致了性能、存储容量和设备密度的不断提高。今天的手机是这种持续进步的一个很好的例子。手机革命的第二个因素是针对效率和灵活性进行优化的处理器架构。自第一代 iPhone 以来，这种 ARM 架构几乎被用于所有设备。这些年来，ARM 也成为服务器的现实平台；通常是比 x86 系统更好的选择。

能效是 ARM 系统胜过传统服务器架构的一个方面。但是直到最近，ARM 处理器才能够为数据中心提供所需的性能。今天的系统基于 Armv8 设计，以更节能的方式发挥了 x86 级别的功能。基于 ARM 的 AWS Graviton 处理器的出现，据估计已经在 AWS 上占据了 20%的实例份额，表明 ARM 可以运行生产工作负载，同时提供 AWS 所称的“对于各种工作负载，其性价比比当前基于 x86 的可比实例高出 40%。”

为了说明数据中心从基于 x86 的架构转向 ARM 服务器时对碳排放的影响，我们对数据中心能源使用的主要因素进行了分析。根据为不同地点的电网供电所使用的燃料组合，计算出节能量并转化为减少的 CO2 排放量。结果证明，ARM 驱动的数据中心减少了 74%的二氧化碳排放量，相当于近 50 万桶石油。

## 高效服务器降低了功耗

前提很简单-创新的服务器设计显著降低了数据中心的能耗。在此模型中，服务器的功耗和物理尺寸是关键因素。其他数据中心子系统所需的能源是通过使用从经验中得出的比率推断出来的，并通过其他数据中心运营商的估计进行验证。

数据中心的能源使用可分为五类设备和基础设施:服务器、网络通信、电源、转换和 UPS、冷却系统和存储。服务器和冷却设备约占总数的 75%。
计算基于一个拥有 750 个机架的传统 1U 服务器的中型数据中心所使用的能源(42 个服务器/机架；总共 31500 左右)。使用的假设包括服务器占三分之二的占地面积，设备占总面积的 65%，相当于一个 62，000 平方英尺的数据中心。相比之下，主要云供应商运营的设施可能有 100 万平方英尺或更多。知道服务器功耗和设备类别的比率后，可以使用代数来计算每种设备的能耗。

如我们所说，这个假设示例中的数据中心有 31，500 台服务器。使用戴尔 R640 的 912.5 瓦功耗作为典型的 1U x86 服务器，ARM 数据中心的服务器总使用量将为 251，795 MWh/年。使用该公式，可以计算出使用 ARM 服务器节省的能源量。每台服务器的耗电量大约是标准服务器的四分之一。在公式中，假设安装 ARM 服务器需要一半的设备机架(375 个而不是 750 个)和 52%的单位系统功耗，数据中心的耗电量(152，409 MWh/年)仅为基于 x86 系统的数据中心的 26%。

数学计算表明，将一个中型数据中心转换为 ARM 服务器每年可以节省相当于路上行驶的汽车减少 45，459 辆或使用的石油减少 486，749 桶。

二氧化碳减排的价值被广泛理解为保护环境。从经济角度来看，这也是有利的。事实上，在下一代云数据中心中采用高能效的 ARM 服务器有许多技术和经济原因。
