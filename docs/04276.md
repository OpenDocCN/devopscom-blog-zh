# Datical 宣布支持 Oracle 数据库云服务器

> 原文:[https://devo PS . com/datical-announces-support-Oracle-exa data/](https://devops.com/datical-announces-support-oracle-exadata/)

*Datical 提高了 Oracle 上关键业务应用的部署速度*

**得克萨斯州奥斯汀**–**2017 年 5 月 31 日**–[领先的数据库发布自动化解决方案提供商 Datical](http://www.datical.com/) 今天宣布支持 Oracle 数据库云服务器，这是一款功能最强大的数据库平台，旨在为要求最苛刻的业务关键型数据库工作负载提供快速、可靠的性能。

软件正在重塑各行各业——从医疗保健到金融再到零售。有鉴于此，变革的步伐正在加快，组织必须掌握自己的应用程序，才能在这个数字时代生存下来。随着公司采用自动化来帮助提高软件交付的速度，一个新的限制出现了——数据库。

Infolob 业务开发高级副总裁 Josh Turnbull 表示:“Oracle 数据库云服务器是一款强大的数据库机，它将工作负载整合并简化到一个组合单元中，这是一个独特的硬件和软件组合，用于优化运营和降低成本。”info lob 是 Oracle 白金级合作伙伴，在数据库云服务器方面拥有深厚的专业知识，在测试工作中为 Datical 提供了支持。“Datical 的解决方案非常适合数据库云服务器，为市场带来了一个期待已久的解决方案，解决了一个非常复杂的问题。”

行业分析公司 IDC [1]采访了八家使用 Oracle 数据库云服务器的组织，[预测](https://blogs.oracle.com/infrastructure/idc-reveals-business-results-from-customers-using-oracle-exadata)这些 Oracle 客户将“实现平均每年每 100 名用户 157，712 美元的业务收益，这将带来 429%的五年投资回报率。”

借助 Datical 对 Oracle 数据库云服务器的支持，DevOps 团队将能够消除数据库部署对交付新应用的障碍，从而提高应用发布的速度。数据库已经成为整体数字化转型计划中的瓶颈，并且长期以来一直是被忽视的学科，是工具链中最薄弱的环节。大多数公司——即使是那些使用最新 DevOps 自动化工具的公司——仍然通过使用几十年前的流程来管理和部署数据库变更，来锚定开发团队。

“组织采用 Oracle 数据库云服务器为任务关键型应用程序实现卓越的数据库性能和可用性。这也意味着，他们需要一个数据库发布自动化解决方案，通过确保数据库更改可以在更新关键任务应用程序的同时部署，来实现投资的最大化。对于采用敏捷和 DevOps 的公司来说，部署数据库模式和逻辑快速变化而不损害数据完整性和安全性的能力至关重要。“我们很高兴将 Datical 带到一个新的平台。使用 Oracle 数据库云服务器的组织现在将能够实现数据库部署流程的现代化，并获得更多优势——更短的应用发布时间、更好的数据安全性以及更高效的应用开发人员、测试人员和数据库管理员。”

十多年来，Datical 背后的团队一直是数据库和应用程序部署发展的先锋。Datical 战略性地指导一些世界上最受推崇的公司完成数据库部署现代化工作。Datical 数据库发布自动化解决方案旨在打破当今手动数据库部署流程造成的应用程序发布僵局。Datical 提高了生产率和性能，使开发、测试和 DBA 人员能够专注于更重要的项目和计划。
