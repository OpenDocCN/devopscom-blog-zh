DBmaestro 的 DevOps Platform 7.5 包括数据库发布自动化解决方案，可自信、可控地简化数据库变更，同时帮助公司缩短应用上市时间。快速采用连续交付和部署方法来加速应用程序开发，产生了对数据库发布自动化工具的强烈需求，以防止错误和停机。根据 [2018 DevOps for Database 报告](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=btYXC68syxnIlVIaW0qBweEUHWPFwuN5EgnqFgPj0oiC1OMl9zi1RmuHF7fB%2BLB%2FnRzDQb88SyTevL3igHolMvyIuL7fIm4obc9RLVDhZq8VDgRLCAIPTg%3D%3D&G=0&R=https%3A%2F%2Fwww3.dbmaestro.com%2Fdevops-for-database-survey-2018-results&I=20181011132100.0000000498d8%40mail6-113-ussnn1&X=MHwxMDQ2NzU4OjViYmY0MjQ1YmQ3OTdkYzNiZjc2YWI3Njs%3D&S=Fk8h2yPWAH6EVBDJXxE15cymFwyFXsBfPrHPFZPwqIQ)，83%的公司已经采用了 DevOps，而数据库的持续交付则落后了，采用率只有 37%。

DBmaestro 的 DevOps Platform 7.5 零摩擦解决方案使 DevOps 团队和数据库专业人员能够更轻松地管理多个数据库，而无需更改现有流程。DBmaestro 为跨平台和操作系统(包括 Oracle、DB2、Microsoft SQL Server 和现在的 PostgreSQL)的 CI/CD 采用提供了一个总体解决方案，满足了各行业和跨国企业不断发展的数据库发布自动化需求。

DBmaestro 首席技术官 Yaniv Yehuda 表示:“我们的 DevOps 平台让 DevOps 和数据库团队能够实施 DevOps 最佳实践，以高度的信心、最小的风险和最高的效率加快上市时间，从而释放 PostgreSQL 数据库的强大功能。
