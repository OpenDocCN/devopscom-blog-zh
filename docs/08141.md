# 在 Gartner 2021 年云安全态势管理领域的优秀供应商评选中，Sonrai 安全性获得认可

> 原文:[https://devo PS . com/son rai-security-recognized-in-Gartner-2021-cool-vendors-in-cloud-security-posture-management/](https://devops.com/sonrai-security-recognized-in-gartner-2021-cool-vendors-in-cloud-security-posture-management/)

*公共云安全创新者还获得三项行业大奖。*

**纽约州纽约市，2021 年 6 月 16 日—**公共云安全和治理领导者 Sonrai Security 今天宣布，在 Gartner 2021 年云安全态势管理(CSPM)报告中，该公司获得了认可。Gartner Cool Vendor research“旨在突出有趣的、新的和创新的供应商、产品和服务。”

这一认可强调了 Sonrai Security 的持续创新和快速发展，包括为其 Sonrai Dig 平台设计的[重要新功能](https://sonraisecurity.com/blog/first-cloud-database-protection/)，旨在自动防止公共云部署中的数据泄露，确保关键企业数据无论驻留在云环境中的何处都是安全的。

根据 Gartner 的报告，“现代基础设施即服务(IaaS)和平台即服务(PaaS)工作负载的复杂性和规模继续呈指数级增长。多云、容器化和无服务器的应用结合在一起形成了动态的、不可预测的交互，如果没有先进的技术，这些交互越来越难以保护。Gartner 预测分析:全球云工作负载保护平台表明，到 2023 年，CWPP 市场预计将增长到 24.4 亿美元，复合年增长率(CAGR)为 18.8%。”

此外，Gartner 建议“管理云基础架构安全的安全和风险管理领导者应投资创新的 CSPM 工具，通过根据已知用例不断评估当前能力，并评估您保护无服务器和基于容器的工作负载免受不断演变的威胁的能力，来弥补战略工具中的差距。”

Sonrai Dig 是一个面向 AWS、Azure 和 GCP 的企业安全平台，它通过公共云的身份和数据安全扩展了 CSPM，还解决了云基础设施授权管理(CIEM)、数据分类和配置管理。这使企业公司能够实施 CSPM，获得并维护最低特权，强制执行职责分离，并锁定关键数据。Sonrai Dig 平台构建在一个复杂的图表上，该图表识别并监控组织公共云中存在的身份和数据之间的每一种可能的关系。Dig 的治理自动化引擎跨云和安全团队自动执行工作流、补救和预防功能，以确保端到端的安全性。

Sonrai Security 首席执行官兼联合创始人 Brendan Hannigan 表示:“Sonrai Security 在过去一年取得了长足的发展，我们相信被 Gartner 评为优秀供应商是对我们保护公共云的独特能力和方法的有力认可。“我们将继续创新，用我们以身份为中心的云安全愿景推动我们的客户向前发展。”

在相关新闻中，Sonrai 继续增加其证书。该公司最近获得了以下行业奖项:美国 Stevies /Silver Stevie 医疗保健技术解决方案奖；TMC 云计算年度产品奖；《网络防御》杂志的全球信息安全奖授予云安全领域的下一代和身份管理领域的下一代。
