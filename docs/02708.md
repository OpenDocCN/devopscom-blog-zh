# WhiteHat Security 推出新的“Essentials”产品线，用于全面代码覆盖的高速全自动安全测试

> 原文:[https://devo PS . com/white hat-security-launchs-new-essentials-product-line-for-high-speed-fully-automated-security-testing-with-comprehensive-code-coverage/](https://devops.com/whitehat-security-launches-new-essentials-product-line-for-high-speed-fully-automated-security-testing-with-comprehensive-code-coverage/)

加利福尼亚州何塞([商业资讯](https://www.businesswire.com/))–[致力于保护数字业务安全的领先应用安全提供商 WhiteHat Security](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Fwhitehatsec.com&esheet=51928646&newsitemid=20190122005251&lan=en-US&anchor=WhiteHat+Security&index=1&md5=cb5de0ffc36360e67c20d2ed959e7600) 今天宣布其新的“Essentials”产品线正式上市(GA)，推出了两种解决方案:Sentinel Source Essentials Edition 和 Sentinel SCA Essentials Edition。简化的“Essentials”产品旨在通过提供全面代码覆盖的高速全自动安全测试，帮助组织满足开发运维构建/测试阶段的需求。“Essentials”高速解决方案减少了扫描时间，帮助开发人员更快地获得他们需要的漏洞评估结果。

“基本”产品线目前包括:

|  |  |  | 1) |  | **Sentinel SCA Essentials Edition**，这是一款独立的全自动软件组合分析(SCA)服务，可快速准确地识别组织应用中使用的第三方和开源组件，提供所谓的软件材料清单。对于这些组件中的每一个，Sentinel SCA Essentials Edition 都会识别任何公开的安全常见漏洞和暴露(CVE)、许可证和过期的库版本。 |
|  |  |  |  |  |  |
|  |  |  | 2) |  | **Sentinel Source Essentials Edition**，提供高速、全自动静态应用安全测试(SAST)。由于使用了怀特哈特的攻击媒介情报(AVI)技术，这些发现是全面和高度准确的。已发现的漏洞根据其严重性划分优先级，从而为应该首先修复的内容提供指导。 |

新的“Essentials”产品线将补充 WhiteHat 现有的旗舰 Sentinel Source Standard Edition 解决方案，该解决方案包括经过充分验证的研究结果和与 WhiteHat 威胁研究中心工程师的无限联系。

根据 WhiteHat 的 [2018 年应用安全统计报告](https://cts.businesswire.com/ct/CT?id=smartlink&url=https%3A%2F%2Finfo.whitehatsec.com%2FContent-2018StatsReport_LP.html%3Futm_source%3Dwebsite%26utm_medium%3DWebsite-Content-Ongoing-2018StatsReport&esheet=51928646&newsitemid=20190122005251&lan=en-US&anchor=2018+Application+Security+Statistics+Report&index=2&md5=107f3a966b6710b42f2bd63590af1258)，每个应用中高达 70%的应用由可重用的软件组件组成(例如第三方库、开源软件(OSS)等)。).这意味着应用程序也“继承”了这些软件组件中的漏洞。因此，保护软件需要能够检测代码和组件中的漏洞的技术。

“通过‘Essentials’系列产品，我们继续执行我们的战略，为每个 DevOps 角色提供特定、专用的应用安全测试和 SCA 产品:WhiteHat Scout 的程序员、Sentinel Source 和 Sentinel SCA Essentials Editions 的构建人员和测试人员，以及 Sentinel Source Standard Edition 的部署专家，”WhiteHat Security 首席战略官约瑟夫·费曼说。“随着越来越多的应用程序被推出,‘Essentials’产品线为组织提供了多种技术选项，使安全团队能够通过让开发人员快速检测和修复安全漏洞来获得立竿见影的效果——这是实现 DevSecOps 的重要一步。”
