# WideOrbit 发布带有新创收功能的 WO Network 2021

> 原文:[https://devo PS . com/wide orbit-releases-wo-network-2021-with-new-revenue-generating-features/](https://devops.com/wideorbit-releases-wo-network-2021-with-new-revenue-generating-features/)

*标志着物料指令和 ADU 规划自动化的重要里程碑*

**加利福尼亚州旧金山，2021 年 7 月 14 日—**广播电视库存和收入工作流管理领域的市场领导者 WideOrbit 今天宣布了该公司旗舰网络广告销售和商业运营平台 WO Network 的最新发布。2021 版本已经成为领先的国家、有线和广播电视网络业务运营的基石，它包括新的功能和增强功能，有助于网络创造新的收入、降低成本和提高效率。

在努力帮助客户在快速变化的行业环境中导航和创新的同时，WideOrbit 还致力于与客户密切合作，通过不断改进我们的核心产品来帮助他们改善现有的流程。这一承诺在 WO Network 的最新版本中得到了体现，它标志着行业中的一个重要里程碑，推出了电子材料说明模块，这是第一个自动化解决方案，旨在满足简化和精简传统上繁琐的材料说明流程的长期需求。

作为 WO Network 的授权模块，电子材料说明集成了几个领先的材料说明提供商，包括 ITN Networks、ECN 和 PremiumMedia360，其他集成即将推出，包括 Warren Lamb。这些集成通过简化买家和卖家之间新的和修订的材料说明的交换，帮助网络节省时间和金钱。

WO Network 2021 版本还为销售团队引入了新功能，包括主交易向导，这是一个查看和管理在单个提案下销售的所有交易的集中位置。ADU 预测工具现已广泛应用于 WO Network 2021，通过最大限度地减少过度交付和交付不足，使用预测分析来确定达成交易所需的受众不足单位(adu)数量，从而帮助网络增加收入和减少负债。

“随着我们 2021 年 WO Network 的发布，WideOrbit 很高兴能够继续我们不断创新和投资改善核心产品的传统，”wide orbit WO Network 产品管理高级总监阿登·王茂林说。“电子材料说明模块和 ADU 预测工具对于我们的广播和有线网络客户来说是改变游戏规则的工具，有望对收入增长、成本节约和效率提高产生长期的积极影响。”
