# 水上集装箱安全平台获得 CIS 基准认证

> 原文:[https://devo PS . com/aqua-container-security-platform-award-cis-benchmark-certification/](https://devops.com/aqua-container-security-platform-awarded-cis-benchmark-certification/)

业内首个通过 CIS Kubernetes 基准认证的商业解决方案

**华盛顿州西雅图-2018 年 12 月 10 日-**kube con/CloudNativeCon-Aqua Security 今天宣布，其 Aqua Container Security Platform(CSP)已通过 CIS Benchmarks^(认证，可以将 Kubernetes 集群的配置状态与 CIS Kubernetes Benchmark 中包含的基于共识的最佳实践标准进行比较。利用 Aqua CSP 的组织现在可以确保其关键资产的配置符合基于共识的 CIS 基准实践标准。)

Aqua 首席技术官兼联合创始人 Amir Jerbi 表示:“我们很高兴我们的平台获得了 Kubernetes 基准的 CIS 认证。“这一认证证明了我们的平台执行的严格安全测试，以及我们为企业客户提供解决方案的承诺，使他们能够满足 CIS 最佳实践标准，并最大限度地提高其 Kubernetes 集群的安全状况。”

Aqua 的平台被全球 1000 强企业中的 100 多家使用，保护其基于容器和云的本地和云中的应用程序，支持跨 Kubnernetes 和其他 orchestrators 的 Linux 和 Windows 运行时环境。Aqua 平台推动了 DevSecOps 自动化，并为云原生工作负载提供了可见性和运行时保护，包括主机级和网络级控制。

该认证由 CIS (互联网安全中心有限公司)颁发，反映了由经验丰富的 IT 专业人士组成的全球志愿者社区不断完善和验证的成熟准则。“网络安全挑战与日俱增，这使得对标准配置的需求势在必行。安全最佳实践&自动化集团 CIS 执行副总裁柯蒂斯·杜克斯(Curtis Dukes)表示:“Aqua Security 通过 CIS 认证其产品，展示了其积极解决基本问题的承诺，即确保标准配置在整个企业中得到使用。

为了使产品获得 CIS 基准认证，供应商必须调整其产品，以准确报告相关 CIS 基准配置文件中的安全建议。CIS 认证安全软件产品展示了供应商的坚定承诺，即根据基于共识的最佳实践标准，为客户提供确保其资产安全的能力。

CIS 基准计划是一个值得信赖的独立权威机构，旨在促进公共和私营行业专家的合作，以就切实可行的解决方案达成共识。CIS 基准被推荐为行业认可的系统强化标准，并被组织用来满足联邦信息安全管理法案、PCI、健康保险可移植性责任法案和其他安全要求的合规性要求。
