GigaSpaces 的 InsightEdge 智能 ODS(运营数据存储)是一个开箱即用的内存数字集成中心(DIH)，可推动企业数字化转型。

微服务平台通过一个低延迟、可扩展的数据结构将数字应用从传统的记录和数据源系统中分离出来，该数据结构通过统一的 API 进行共享，企业利用该 API 来:

*   快速引入新的现代数字应用。

*   将数字应用从记录系统中分离出来，为新的数字应用提供速度、规模和高并发性。

*   确保应用程序和数据访问始终在线。

*   对昂贵的传统基础设施进行现代化改造和卸载。

*   借助云原生平台，一次开发，随处部署—内部、云、混合和多种云。

GigaSpaces InsightEdge 智能 ODS 执行数据事务和分析的速度比 NoSQL 快 30 倍，比其他内存平台快 6 倍。
