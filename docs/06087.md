弗吉尼亚州雷斯顿——领先的云计算管理和混合 IT 基础设施监控解决方案提供商 ScienceLogic 今天发布了其在情景融合 AIOps 方面的最新创新。随着行为相关性的引入，ScienceLogic 正在**转变 IT 团队识别、故障排除和补救服务中断事件的方式**—甚至在最终用户影响被检测到之前。

现代 IT 环境通过应用程序和数据的激增滋生了复杂性，超出了人类的分析和响应能力。被迫追踪和修复影响客户的 IT 事件的运营团队经常受到基于以人为中心的分析的过时方法的限制。通过行为关联，ScienceLogic **引入了一种更加有效的方法，利用机器速度的检测和补救。**现在可以一目了然地了解实时服务运行状况和风险，使 IT 团队能够在问题发生时快速解决问题，甚至在问题发生前拦截潜在的服务影响问题。

ScienceLogic SL1 的行为关联是通过紧密集成的核心功能实现的:

*   一个全面的实时数据湖，捕获多模态数据类型及其关系
*   能够映射和可视化 IT 服务、其底层依赖关系以及相关的运行状况、可用性和风险
*   机器学习技术，对这些服务拓扑进行推理，以检测问题或异常行为的根本原因，并可以建议采取措施来解决

**最终结果是一个系统，它通过直观的、以服务为中心的视角提供对复杂 IT 资产的整体可见性**。决策者可以快速了解影响、根本原因和相对优先级，确保 IT 将时间花在对业务最重要的事情上。

“这是一项改变游戏规则的创新，它摒弃了传统的、反动的 IT 事件管理方法。ScienceLogic 首席执行官 Dave Link 表示:**我们现在可以即时提供整个短暂 IT 状态的实时画面，确定服务降级发生在哪里，哪些问题应该优先处理，以及潜在的业务影响。“紧密结合的服务级别视图减轻了 IT 团队从一个事故到另一个事故的压力，并使全球提供商能够提供灵活的客户体验。”**

随着 IT 团队从 20 世纪 90 年代的作战室和防御位置摆脱出来，他们可以采用新的形势感知标准来帮助加快根本原因分析和解决问题的速度。
