# DBmaestro 横扫金融行业，六个月内获得九个新客户

> 原文：<https://devops.com/dbmaestro-sweeps-financial-industry-landing-nine-new-customers-in-six-months/>

*五大洲的 21 家机构选择了 DBmaestro Database Automation 来最小化和控制风险，同时加速软件开发*

**马萨诸塞州波士顿，2018 年 8 月 29 日**–[数据库发布自动化解决方案提供商 DevSecOps platform 的先驱和领导者 DBmaestro](https://www.dbmaestro.com/) 今天宣布，在过去的六个月中，它已被九家组织和 21 家金融机构选中，负责安全高效地运行数据库部署，提高开发团队的工作效率，并显著缩短软件解决方案的上市时间。

DBmaestro 的解决方案已在九家领先的北美金融机构和六家欧洲机构实施，包括巴克莱银行和 ing 集团的个人和企业银行部门([阅读案例研究](https://www.dbmaestro.com/wp-content/uploads/2018/07/Case-Study_ING-Bank-Poland-2018.pdf))，此外还有澳大利亚、南美和亚洲的金融公司。

通过持续集成和持续交付(CI/CD)管理快速部署，同时强制实施组织策略以符合 GDPR 和其他法规，同时避免停机，这一挑战是采用数据库发布自动化的主要驱动因素。根据 IDC 的一份报告,计划外应用程序停机每年会给财富 1000 强公司造成 12.5 亿到 25 亿美元的损失，平均每停机一小时就要花费 10 万美元。

用于数据库的 DevOps 平台**、** DBmaestro 的发布自动化可强制执行组织政策并验证未来版本，以最大限度地减少可能导致停机和数据丢失的错误和冲突。通过简单直接的实施和快速的部署，DBmaestro 使金融机构每月为 Oracle、IBM DB2 和 Microsoft SQL server 数据库发布的应用程序数量翻了一番。

DBmaestro 首席执行官 Yariv Tabac 表示:“由于竞争激烈，金融服务公司比其他行业更快地采用 DevOps 方法和最佳实践。“我们致力于让金融机构在不牺牲质量和安全性的情况下实现自动化，以跟上快速的技术创新步伐。”
