# Magic Software 的 SaaS 集成平台利用 GigaSpaces InsightEdge 开发的人工智能和机器学习

*快速流和高级分析为神奇的客户提供实时洞察，帮助他们提高运营效率、竞争力和创新能力*

 ***纽约州纽约市——2018 年 10 月 31 日**—[giga spaces](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgHoKEAVhhdECuMjf%2FFI0vrn6R7bJnl43GA%3D%3D&G=0&R=https%3A%2F%2Fwww.gigaspaces.com%2F&I=20181030135911.000000077a47%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjViZDg2Mzk3ZWY0NDg1MGExZjZmZWEyNzs%3D&S=IgJcIpaaKsjML8z69EX7HzGxnmapO_sVE45jnVSIHxg),[InsightEdge](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgHoKEAVhhdECuMjf%2FFI0vrn6R7bJnl43GA%3D%3D&G=0&R=https%3A%2F%2Fwww.gigaspaces.com%2Fproduct%2Finsightedge-platform&I=20181030135911.000000077a47%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjViZDg2Mzk3ZWY0NDg1MGExZjZmZWEyNzs%3D&S=v0niV63WOAleT4qpb7KDbmqupx17CgMauGmXK7OAK9Q)的提供商，一个领先的即时洞察行动的内存实时分析平台，今天宣布 insight edge 已被 Magic Software Enterprises 选中，为其 [Magic xpi 端到端集成平台](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgHoKEAVhhdECuMjf%2FFI0vrn6R7bJnl43GA%3D%3D&G=0&R=https%3A%2F%2Fwww.magicsoftware.com%2Fintegration-platform%2Fxpi%2F&I=20181030135911.000000077a47%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjViZDg2Mzk3ZWY0NDg1MGExZjZmZWEyNzs%3D&S=Mly9B5nHyp2yFzVqg2wtbM6jpEo51wNHf6vl9AoKeHQ)提供支持。这种集成将使公司能够做出更快、更智能的数据驱动型决策，从而增加收入、降低成本、降低风险并超越竞争对手。

Magic xpi 提供领先的 ERP、CRM、财务、MES 和其他系统之间的自由数据流，现在利用统一实时分析和人工智能的 InsightEdge 来实现精益制造、执行预测性维护和自动化运营工作流。机器学习模型在热数据诞生时就以亚秒级的延迟运行，同时从数据湖中丰富历史背景，从而产生准确的实时可操作见解，以改进决策。

Magic Software 产品创新副总裁尤瓦尔·狮式战斗机表示:“InsightEdge 使我们的客户能够为首席执行官和业务线管理人员提供见解，并通过机器、产品、员工的传感器和 GPS 读数以及车间应用和后台系统的输入产生的前所未有的大量数据优化自动化流程。“来自多个来源的集成数据、极端数据处理和实时分析的结合提供了影响更精简运营和改善客户体验的见解。”

与 InsightEdge 一起运行的 Magic xpi 支持多种数据驱动的流程，例如，使用传感器监控设备以预测故障，执行预测分析以确定应该执行哪些质量测试以及执行多少次质量测试，以及与合作伙伴和客户共享供应商生产数据以确定交付延迟并相应地调整流程。

[麦肯锡](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgHoKEAVhhdECuMjf%2FFI0vrn6R7bJnl43GA%3D%3D&G=0&R=https%3A%2F%2Fwww.mckinsey.com%2F%7E%2Fmedia%2FMcKinsey%2FIndustries%2FSemiconductors%2FOur%2520Insights%2FSmartening%2520up%2520with%2520artificial%2520intelligence%2FSmartening-up-with-artificial-intelligence.ashx&I=20181030135911.000000077a47%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjViZDg2Mzk3ZWY0NDg1MGExZjZmZWEyNzs%3D&S=trgueII5ZLj-Hc_zXFz1lJGJ01xxBvrjpJyZUfgcCkE)已经注意到，预测性维护计划可以显示年度维护成本减少 10%,停机时间减少 20%,人工智能驱动的预测性维护模型的检查成本减少 25%。

这不是 Magic 和 GigaSpaces 的第一次合作。Magic 多年来一直使用 GigaSpaces 的 XAP 内存计算平台来提供快速数据流、聚合和计算，去年宣布了 InsightEdge 集成，将 InsightEdge 作为物联网中心。Magic xpi 客户现在可以选择运行 InsightEdge 来体验实时分析的好处。

GigaSpaces 产品副总裁 Yoav Einav 表示:“未来的工厂需要结合实时高级分析的数据集成。“通过将 InsightEdge 功能整合到 Magic xpi 中，我们将机器学习的力量带到了车间和后台办公室，以帮助公司优化流程，最大限度地提高效率并超越收入目标。”

GigaSpaces 和 Magic Software 将于 11 月 6 日上午 8:30 在德国慕尼黑的设计办公室展示“洞察驱动的组织:利用人工智能将数据转化为收入”。该活动将包括案例研究和关于市场挑战、趋势和现实世界最佳实践的讨论，让企业充满信心地进行创新，并成为洞察力驱动的企业。有关此次活动的更多信息，请点击[此处](http://icm-tracking.meltwater.com/link.php?DynEngagement=true&H=3ZUQjNycMu7D%2Fe%2Bm%2FOmi3Qi1eTNrfRb0HcFplK3KYerw%2B6SfjwwI9iczpalQd%2BJ%2FRUwMdhWt30NeEu3MWXBqgHoKEAVhhdECuMjf%2FFI0vrn6R7bJnl43GA%3D%3D&G=0&R=https%3A%2F%2Flp.gigaspaces.com%2Fevents%2Fgermany-insight-driven-event-nov-2018%2F&I=20181030135911.000000077a47%40mail6-53-ussnn1&X=MHwxMDQ2NzU4OjViZDg2Mzk3ZWY0NDg1MGExZjZmZWEyNzs%3D&S=WGLm2b4xqEU62qhcGDqGCOw7-DoAfAgl2YTY8yKL0rg)。
