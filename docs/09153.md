# Skuid 收购 InFlight，帮助企业支持人员运营

> 原文:[https://devo PS . com/SKU id-acquisites-inflight-to-help-enterprise-companies-champion-people-operations/](https://devops.com/skuid-acquires-inflight-to-help-enterprise-companies-champion-people-operations/)

**美国田纳西州查塔努加市，2022 年 2 月 3 日—**创造以人为中心的数字体验的领先工具包 Skuid 今天宣布收购英飞公司。机上平台帮助企业组织实现其业务应用的现代化，并改善人才和员工体验(EX)。Skuid 和 InFlight 将共同为寻求优化人员运营的企业提供最完整的解决方案，从人才获取到人才管理等，以前所未有的速度和便捷性创建与其记录系统相集成的现代用户体验。

数百家公司，包括贝克休斯、T2、美国退伍军人事务部、T4、GolfNow、John Hancock 和 TalkDesk，已经在使用 Skuid 来构建人力运营应用程序。这包括员工门户、审计、合规和工作场所风险应用程序、招聘和保留应用程序、入职应用程序、时间管理和报告应用程序，以及许多其他应用程序。收购 InFlight 扩展了 Skuid 已经令人印象深刻的人力资源重点，增加了与流行的人力资本管理(HCM)企业解决方案的更深入集成，包括 Oracle PeopleSoft、Workday、Taleo 和 SAP SuccessFactors。

“这是未来人员操作应用程序的游戏规则改变者。Skuid 和 InFlight 的组合功能为希望设计、创建和交付有影响力的应用的企业提供了一个急需的解决方案，”Skuid 首席执行官 Ryan Niemann 说。“随着 InFlight 的加入，企业公司现在可以利用 Skuid 独特的以人为中心的设计方法，与 HCM 应用程序进行集成。”

当大多数企业需要更好地支持员工、招募和留住顶尖人才时，EX 却显得力不从心。根据最近的 Gartner 调查，只有 13%的员工对他们的体验完全满意——日常工作中与人、流程和技术的互动。随着工作人员变得越来越数字化，业务优先级快速发展，拥有合适的技术不再是好事，而是势在必行。但是拥有合适的技术只是解决方案的一部分。企业需要合适的产品和合适的流程。Skuid 以人为中心的设计流程与新扩展的 Skuid + InFlight 产品相结合，为企业优化人才和员工应用提供了独特而完整的途径。

凭借 Skuid 轻松打造现代用户体验的能力，并以飞行中的集成为后盾，企业可以快速创建基于网络的、完全可访问的应用程序，包括人力资源软件(HRS)、人力资源信息系统(HRIS)、申请人跟踪系统(ATS)等。这导致对 EX 的更好控制，而不需要繁重的编码。

InFlight 创始人兼首席执行官 James La Brash 表示:“我们很高兴加入 Skuid 团队。InFlight 为使用 Skuid 的企业带来了独特的新集成功能，同时通过低代码 UI 构建功能增强了我们自己的客户产品。这标志着 HCM 市场的 EX 进入了一个新时代，企业可以支持公民开发人员和 IT 团队快速轻松地改进其现有应用的数字 EX，以应对不断变化的员工期望和市场条件。为企业提供更大的灵活性，Skuid + InFlight 将使他们获得竞争市场优势。”

新客户和现有客户可以使用 Skuid + InFlight 组合来显著释放现有系统的价值，轻松定制跨收入、客户和人员运营的人性化企业应用和体验。公司可以无缝集成多个应用程序，统一不同的数据和流程，并简化 UX 以推动采用和提高生产力。

交易条款不会被披露。JMP 证券有限责任公司担任 Skuid 的财务顾问。Catapult Advisors LLC 担任空中公司的财务顾问。
