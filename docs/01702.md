# 网上研讨会:全国保险公司如何采用 DevOps 实现变速 IT 系统

> 原文:[https://devo PS . com/nationwide-insurance-adopting-devo PS-variable-speed-it-systems/](https://devops.com/nationwide-insurance-adopting-devops-variable-speed-it-systems/)

为了推动更快的创新，同时确保核心业务的稳定性，需要出现所谓的“双速或变速 IT”创新的参与系统需要速度、灵活性和实验性，而传统的记录系统需要治理和可预测性。双速 IT 通过使开发运维功能适应交付的“速度”来支持开发运维功能的采用。它需要双速业务规划和流程，甚至双速组织结构和预算。加入 Nationwide Insurance 的应用程序开发总监 Carmen DeArdo、IBM 的首席 DevOps 技术宣传员 Sanjeev Sharma 和执行 IT 专家 Lee Reid，他们展示了 IBM 的解决方案如何帮助全国采用变速 IT 系统的 DevOps 实践。

本次网络研讨会是[变速 devo PS——深度潜水系列](https://devops.com/blogs/variable-speed-devops-a-deeper-dive/)的第一场。
