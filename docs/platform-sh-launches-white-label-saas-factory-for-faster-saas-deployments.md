# Platform.sh 启动白标 SaaS 工厂，加快 SaaS 部署

> 原文：<https://devops.com/platform-sh-launches-white-label-saas-factory-for-faster-saas-deployments/>

*新的快速启动功能和 API 使软件公司能够迁移到云，并在几周内创造新的经常性收入业务，而不是几年*

[October 24, San Francisco, CA] Platform.sh, the Idea-to-Cloud Application Platform that simplifies cloud infrastructure, today announced the availability of its white-label platform, which enables software companies and SaaS entrepreneurs to transform their existing on-premise software or build new SaaS businesses with unprecedented speed.

组织在基础架构的开发和维护上投入了越来越多的时间和预算，分散了对他们为客户提供的核心价值(惊人的应用程序和数字体验)的关注。Platform.sh 使他们能够将精力放在激发和吸引客户上，而不是建立团队和技术债务来管理工具、服务器、容器和云基础设施，并消除了推出新业务所需的资本支出。

在几天之内，Platform.sh 就可以为大规模快速部署 SaaS 软件、商务应用、移动后端和 web 应用提供完全品牌化的体验，并以多种语言或框架构建。此功能利用 Platform.sh 的 API 优先开发方法、其独特的即时克隆技术及其完整的“包括电池”的持续部署工作流，其中基础架构变得可组合，并使用与应用程序代码相同的工具进行管理。

合作伙伴只需要提供基本的品牌信息，将他们现有的应用程序代码作为模板加载，在 Platform.sh 提供的选项(AWS、Azure 和 Orange，或所有这些)中决定他们想要解决的云，然后利用 Platform.sh 强大的计费和供应 API 来集成他们现有的业务流程。
