# 布里奇特·克鲁姆胡特的《反人类的债务》走红网络

> 原文：<https://devops.com/bridget-kromhouts-devops-humanity-goes-viral/>

许多阅读这篇文章的人可能已经看过这条推文，或者明尼阿波利斯的布里奇特·克鲁姆胡特的“ [DevOps Against Humanity](https://github.com/bridgetkromhout/devops-against-humanity) ”推文。推文和附带的图片如下:

对于那些不熟悉的人来说，DevOps 反人类是对“[卡反人类](https://www.cardsagainsthumanity.com/)”的模仿，这是一种用黑白卡进行的团体聚会游戏。游戏真的很简单。一名玩家选择一张黑色卡片，在空白问题中有一个填充。其他玩家从他们的白色卡片上选择他们认为最有趣的答案。其中一个白色卡片答案被选为获胜者，提交该白色卡片的人获得一分。还有一些其他的选择，但这是基本的游戏。

《反人类之卡》的游戏和卡片是在知识共享许可下发布的，所以你可以打印自己的副本。你也可以通过购买预印的套装来支持创作者。此外，还鼓励玩家用新的问题和答案制作自己的卡片。

由 Kromhout 支持的 DevOps 反人类变种是一个与 DevOps 相关的问题和答案的版本，但仍然保留了原游戏以其邪恶的尖锐潜力而闻名。布丽奇特已经建立了一个 github 网站，你可以在那里下载她创作的 pdf 和图片，以及打印它们的说明。还有一个[链接，链接到一个谷歌文档](https://docs.google.com/spreadsheets/d/1OKgmjNz8l7skYfrbrOtYT6QmSf_y-BzZOWJWZC3tbUQ/edit#gid=0)，里面有问题和答案的想法。您可以将自己的添加到列表中。随着时间的推移，很高兴看到这个名单真的扩大了。

听起来像是一个很酷的游戏，可以在下一次 DevOps 会议或志同道合的 DevsOps 朋友聚会上玩。我想你可以和非 DevOps 的人一起玩，但是对一群人来说有趣的东西对另一群人来说可能就不那么有趣了。

只是提醒一下，反人类卡和反人类 DevOps 是在知识共享许可下发行的，所以你不能转售它们。
