# sauce labs 达成最终协议，收购 autonomic

> 原文：<https://devops.com/sauce-labs-enters-into-definitive-agreement-to-acquire-autonomiq/>

*   *计划收购通过领先的无脚本测试自动化解决方案支持连续测试平台*
*   *无脚本测试自动化的计划增加扩展和补充了公司现有的能力集，使持续测试领导者能够为更广泛的用户和用例服务*

**三藩市-2020 年 2 月 2 日-**[Sauce Labs Inc .](https://www.saucelabs.com/)，提供数字信心的[连续测试](https://saucelabs.com/solutions/continuous-testing)解决方案的领先供应商，今天宣布达成最终协议，收购无脚本测试自动化解决方案的领先供应商 [AutonomIQ](https://autonomiq.io/) 。AutonomIQ 利用强大的 AI 和 ML 功能，帮助用户快速、轻松地为现代 SaaS 和低代码应用程序创建和执行测试。计划中的收购进一步扩展和丰富了 Sauce Labs 产品组合的深度和广度，将无脚本测试自动化添加到一个不断增长的功能集，该功能集现在还包括 API 测试、可视化测试和前端性能测试，以及该公司以其闻名的功能 UI 测试。

“我们很高兴能够从 2020 年的起点开始 2021 年的工作，”酱油实验室总裁兼首席执行官艾德·迈尔斯说。“在我们最近宣布收购 API Fortress 之后，我们准备为测试协作平台添加另一个重要部分，这将帮助我们的客户继续加快他们的数字化计划。我们期待着欢迎 AutonomIQ 的人员和产品，并看到他们作为 Sauce Labs 团队的一部分成长壮大。”

对 AutonomIQ 的计划收购补充了该公司在 Selenium、Appium 和其他领先的脚本测试框架方面的现有专业知识，使 Sauce Labs 能够服务于更广泛的用户和用例。AutonomIQ 非常适合缺乏编写测试脚本所需的编码专业知识的新兴用户类别，如公民开发人员、人工测试人员、产品经理和业务分析师，以及越来越多的小型、不太复杂的业务应用程序，这些应用程序不需要脚本测试框架的全部功能。该解决方案也非常适合 Salesforce、Workday、ServiceNow、Oracle、SAP 和现代企业中常见的其他 SaaS 应用程序的管理员，他们只是寻求定义业务流程，并确保他们的定制 SaaS 应用程序按照预期遵循这些流程。

AutonomIQ 的创始人兼首席执行官 Ram Shanmugam 表示:“对于任何了解自动化测试在快速交付高质量应用方面所发挥的更大作用的组织来说，Sauce Labs 和 AutonomIQ 都是完美的组合。”他将在计划收购完成后加入 Sauce Labs 的领导团队。“我为我们在 AutonomIQ 取得的一切感到无比自豪，也为成为 Sauce Labs 的一部分给我们带来的持续增长的机会感到兴奋。我们建立 AutonomIQ 的基础是对创新和客户服务的坚定承诺，我毫不怀疑这将继续下去，因为 Sauce Labs 也有同样的激情。”

**现代商业应用不断变化的面貌**

尽管 Selenium、Appium 和其他强大的脚本测试框架仍然是大型和复杂应用程序自动化测试的理想之选，但在 SaaS 业务应用程序流行的推动下，较小的低代码应用程序在企业中越来越常见。这些应用程序不仅在功能和范围上比它们更复杂的对应物更受限制，使得描述它们能执行和不能执行的功能和过程比以往任何时候都更简单，而且它们通常由没有编码和测试创建专业知识的用户来管理。

Sauce Labs 首席技术官 John Kelly 表示:“随着人工智能和人工智能功能的不断成熟，现在既需要也有机会提供易于使用的无脚本测试自动化解决方案，以解决那些脚本化框架不是理想解决方案的用户和用例。“对 AutonomIQ 的收购计划降低了缺乏自动化测试脚本编码背景或熟练程度的 Sauce Labs 用户的准入门槛，同时也为传统的 Sauce Labs 用户提供了额外的价值，这些用户虽然精通脚本化框架，但仍然要负责许多测试案例，在这些案例中，简单易用的无脚本自动化可以节省他们的时间，并帮助他们提高工作效率。”

**使用 AutonomIQ 实现简单的低代码自动化**

AutonomIQ 为用户提供了一个简单而强大的无脚本测试自动化平台，极大地降低了进入门槛，这些门槛曾经阻碍了许多组织开展自动化测试计划。AutonomIQ 平台利用复杂的 AI/ML 来理解测试用例的自然语言描述，或捕捉到的鼠标点击，然后生成合成数据和强大的自动化测试，当应用程序发生变化时，这些测试可以“自我修复”。AutonomIQ 提供了多种专为测试现代 SaaS 业务应用而构建的解决方案，包括 Salesforce、ServiceNow、Workday、Oracle 和 SAP。
