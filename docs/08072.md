# Apifiny 聘请总裁兼首席财务官 Erez Simha 帮助带领公司上市

> 原文：<https://devops.com/apifiny-hires-president-and-cfo-erez-simha-to-help-lead-company-toward-public-offering/>

【Simha 先生将帮助加速 Apifiny 面向机构交易商、交易所和做市商的数字资产交易和流动性解决方案的发展

**纽约州纽约市 —** [全球领先的数字资产交易和采矿网络 Apifiny](https://www.apifiny.com/?utm_source=businesswire&utm_medium=referral&utm_campaign=banks&utm_term=Thomas_Trepanier) 今天宣布任命 Erez Simha 为总裁兼首席财务官，帮助加快公司的全球增长，并带领公司在 2021 年上市。Simha 先生在全球上市和私营科技公司的全球金融、运营管理和行政领导方面拥有超过 20 年的经验。他的经验包括在 Stratasys(纳斯达克股票代码:SSYS)、奥宝科技(纳斯达克股票代码:ORBK)、Mercurity Fintech Holdings(纳斯达克股票代码:MFH)和两家私营科技公司 Roo 和 [Ju.st](https://ju.st/) Inc 等公司担任执行财务和运营职务。

在他的职业生涯中，Simha 先生承担了全球财务、运营、M&A 活动、IT 基础设施、人力资源、SEC 报告、萨班斯-奥克斯利法案合规、风险管理以及全球高速增长环境中的法律和投资者关系等所有方面的广泛责任。在加入 Apifiny 之前，他曾担任 Mercurity Fintech holding (MFH)的总裁兼首席财务官，以及 Roo Inc. (2019 年至 2020 年)和[ju . ST](https://ju.st/)Inc .(2017 年至 2019 年)的首席财务官和首席运营官，在那里他指导了执行领导的所有方面。在此之前，Simha 先生曾在 Stratasys 公司担任全球首席运营官和首席财务官(2011 年至 2017 年)，该公司是他的上市公司，并在奥宝公司担任首席财务官(2004 年至 2011 年)。

Apifiny 首席执行官徐浩瀚表示:“Erez 有着让全球公司上市并在上市前后加速其增长的良好记录。”。“他在私营和上市全球科技公司的行政领导成功将为我们的客户、我们的团队和整个市场带来巨大的价值。”

Apifiny 总裁兼首席财务官 Erez Simha 表示:“Apifiny 处于有利地位，可以抓住全球数字资产交易的巨大市场机遇。“我们拥有正确的技术、正确的团队和正确的经验来执行我们的使命，即连接世界的全球交易市场，以便专业交易员和数字资产交易所拥有一个更简单、更快速、更具成本效益的全球交易平台。”

**关于 Apifiny**

Apifiny 是一家领先的全球数字资产交易和采矿网络。Apifiny 统一了分散的全球交易市场，为专业交易商和数字资产交易所提供更简单、更快速、更具成本效益的全球交易和流动性解决方案。该公司将集中的金融机构与分散的网络连接起来，重新构想全球流动性和交易价值链。Apifiny 的交易网络使专业交易商能够执行全球多交易策略，获得深度的全球流动性、卓越的价格发现、标准化的市场数据、更紧密的价差以及更高的填充率和资本利用率。欲了解更多信息，请访问[https://www.apifiny.com/](https://www.apifiny.com/)。
