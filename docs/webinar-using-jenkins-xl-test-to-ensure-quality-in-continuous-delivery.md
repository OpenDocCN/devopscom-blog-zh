# 网络研讨会:使用 Jenkins & XebiaLabs 软件确保连续交付的质量

> 原文：<https://devops.com/webinar-using-jenkins-xl-test-to-ensure-quality-in-continuous-delivery/>

随着快速发布软件和为客户增加价值的需求的增长，持续交付的趋势是不可否认的。同时，在软件交付管道中管理代码质量也存在挑战。您可能会在管道中进行大量测试以确保质量，但随着您加快软件交付的速度，理解各种工作产生的所有测试结果——并识别罪犯和惯犯——变得越来越困难。

XebiaLabs 和 CloudBees 可以帮助您更快地交付更好的软件。参加我们的现场直播，看看您如何:

*   通过利用 Jenkins 工作流插件和 XL TestView(以前称为 XL Test)的聚合、报告和鉴定功能，快速了解并提高代码质量。
*   理解您的测试数据——在一个中央仪表板中收集、可视化和分析来自所有测试工具的数据。
*   直接从多个 Jenkins 作业中提取测试结果，并以一致的方式可视化它们。
