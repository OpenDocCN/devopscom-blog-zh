# DBmaestro 推出团队合作版本 5.0

> 原文：<https://devops.com/dbmaestro-unveils-teamwork-version-5-0/>

*新版本包括同类产品中的第一个管道构建器，可显著简化自动化数据库部署管道*

德克萨斯州休斯顿，2016 年 2 月 3 日–db maestro[(www。DBmaestro.com)，](http://www2.dbmaestro.com/l/11742/2014-04-02/vrnkh)数据库解决方案 DevOps 的先驱和领先提供商，宣布发布[团队合作版本 5.0](http://www.dbmaestro.com/dbmaestro-unveils-teamwork-version-5-0/) ，其中包括首个同类产品*管道构建器，*使用户能够直观地描述数据库环境和部署流程。新版本为 DevOps for database 带来了成熟性和稳定性，大大简化了为持续交付流程建立自动化数据库部署管道的工作。

DBmaestro 还推出了新的增强的数据库合规性和审计功能。DBmaestro 防止对数据库结构及其包含的数据进行未经授权的更改。现在，企业能够定义下至数据库对象或内容的访问级别，并审查显示所有数据库修改或未经授权或进程外修改尝试的合规性报告。

DBmaestro 首席技术官 Yaniv Yehuda 表示:“随着整个市场转向持续交付、自动化和合规性，组织正在寻求任何可能的优势。“DBmaestro 一直致力于为市场提供同类最佳的数据库开发运维解决方案。新版本增强了数据库开发和部署自动化、数据库的合规性和审计能力，以及应用程序开发和数据库开发团队之间的协作。这些新增功能意味着更快、更安全、更容易、更便宜的部署。”

新版本的一些增值优势包括:

*   更快、更轻松、更安全地实现数据库持续交付流程和自动化
*   记录和报告安全漏洞
*   消除手动设置连续输送管道时可能出现的错误
*   数据库交付管道、环境和流程的清晰可视化
*   为发布过程增加了一个部署安全网。 *Pipeline Builder* 验证升级包是最新的，并且可以在不破坏目标环境的情况下安全执行

**关于 DBmaestro**

[DBmaestro](http://www.dbmaestro.com/) 是数据库解决方案提供商 DevOps 的先驱和领导者。其旗舰产品 DBmaestro TeamWork 支持数据库的敏捷开发、持续集成和交付。

团队合作支持开发过程管理的流线化和执行变更策略实践。该解决方案支持敏捷团队协作，同时促进法规遵从性和治理。借助 DBmaestro，组织可以通过执行部署自动化、增强和巩固安全性以及降低风险来促进数据库开发运维。

DBmaestro 的解决方案被许多大型国际公司采用，包括 Visa、Isracard(万事达卡)、Frost Bank、Thomson Holidays、Bank Leumi 等。
