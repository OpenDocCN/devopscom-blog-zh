# DevOps 领导力系列:监控容器和微服务

> 原文：<https://devops.com/devops-leadership-series-monitoring-containers-microservices/>

特雷弗·帕森斯([@特雷弗·帕森斯](https://twitter.com/trevparsons))是 SaaS 领先的日志管理和分析服务公司 Logentries 的联合创始人兼首席科学家。我在圣克拉拉的 Velocity 大会上采访了 Trevor，问他今年与会者有哪些共鸣的主题。

在这一集的 [DevOps 领导力系列](http://www.sonatype.org/nexus/resources/2015-devops-leadership-series/)中，Trevor 简要说明了微服务和基于容器的架构的一些当前问题，同时分享了对未来的积极展望。

[https://www.youtube.com/embed/zHRE4011WVk?feature=oembed](https://www.youtube.com/embed/zHRE4011WVk?feature=oembed)

Trevor 表示，在如何记录和监控集装箱化系统方面已经有了很大的发展，他看到 Docker 对日志记录提供了更好的支持，这表明 2015 年及以后对这一主题的关注越来越多。他预计明年在管理和监控这些新系统和架构方面会有巨大的改进。

在 [DevOps 领导力系列](http://www.sonatype.org/nexus/resources/2015-devops-leadership-series/)的下一站，我前往伦敦参加[devo PS Connect:Rugged DevOps](http://www.infosecurityeurope.com/education/education-programme/devops-connect/)活动，在那里我们将讨论有关 Rugged devo PS 实践的更多信息。在伦敦，我得以赶上[海伦·比尔(Ranger4)](http://www.ranger4.com/blog/author/helen-beal) 、[贾斯汀·阿巴克尔(Chef)](https://www.linkedin.com/profile/view?id=719595&authType=NAME_SEARCH&authToken=_eeB&locale=en_US&trk=tyah&trkInfo=clickedVertical%3Amynetwork%2Cidx%3A1-1-1%2CtarId%3A1433522600769%2Ctas%3Ajustin) 和[加雷思·拉什格罗夫(Puppet Labs)](https://www.linkedin.com/profile/view?id=4847410&authType=NAME_SEARCH&authToken=9e5u&locale=en_US&trk=tyah&trkInfo=clickedVertical%3Amynetwork%2Cidx%3A1-1-1%2CtarId%3A1433522562011%2Ctas%3Agareth) 。
