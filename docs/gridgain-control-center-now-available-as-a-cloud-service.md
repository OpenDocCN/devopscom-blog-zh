# GridGain 控制中心现已作为云服务提供

> 原文：<https://devops.com/gridgain-control-center-now-available-as-a-cloud-service/>

*GridGain Control Center 的新 SaaS 版本延续了 grid gain 的内存计算解决方案向云的迁移*

**加利福尼亚州福斯特市，2021 年 5 月 24 日—**[grid gain](https://www.globenewswire.com/Tracker?data=5wyS5w4jABfnt_etCUtXoY-bWxhCyxhZ9wb-1CH9JqmT2o7qruu2gkQIGX3LIYf6wEpYVBWK6QkExWKKNl3xIQ==)[Systems](https://www.globenewswire.com/Tracker?data=4EByzMcvASn3ag251GIktXc2OTZEoqB__Q2rKGa-J2wX6I_fvnlq6lE3yX5HW-3ZUxEJZvM7JUWMyChXoVmfrA==)，由 Apache Ignite 分布式数据库提供支持的企业级内存计算解决方案提供商，今天宣布 GridGain Control Center 的增强版作为一项基于订阅的云服务提供。GridGain SaaS 控制中心是 grid gain 将更多解决方案迁移到云以支持客户加速云迁移的又一举措。虽然去年发布的 [GridGain Nebula](https://www.globenewswire.com/Tracker?data=5wyS5w4jABfnt_etCUtXoa2Ion_ukQV9JfkYmx-BWPqwEj1fesZqXUkeGaKbIBWaHM5UapxW_tbHEiunoQBWcEcWGcJxPbxzd9Bg_3wLBOtCHRrJ5pZIbWU0bPlB8EbS4VqVt4NGlo44VNmVcl3hnQ==) 是针对 Ignite 和 GridGain 的完全托管服务，但 SaaS 控制中心是针对喜欢管理其生产环境的运营团队的云原生解决方案。SaaS 控制中心的 30 天免费试用可在 GridGain 网站上获得。

GridGain Control Center 是针对 GridGain 平台和 Apache Ignite 的综合故障排除、管理和监控解决方案，它们运行在分布式服务器集群上。对这些分布式集群进行故障排除和管理可能要求很高，控制中心简化了关键任务，以帮助开发人员和运营团队在这些平台上更高效地构建、管理和优化应用程序。grid gain Control Center as a cloud service 使客户能够管理部署在单个或多个云服务、内部和混合环境中的计算集群。

GridGain Systems 云产品管理总监 Greg Stachnick 表示:“随着基础设施向云迁移的加速，我们看到越来越多的客户在其云环境中部署 Apache Ignite 和 GridGain 平台，越来越多地利用 GridGain 控制中心和 GridGain Nebula 托管服务。“因此，我们继续投资于新的 GridGain 产品，如 SaaS 控制中心，以更好地帮助客户实现其最苛刻应用所需的速度和规模。”

控制中心极大地简化了 Ignite 和 GridGain 平台特定的故障排除和管理任务。其中包括调试 SQL 性能瓶颈、跟踪负载下的操作性能、简化数据备份和恢复，以及主动管理数据偏差以优化集群利用率。此外，控制中心还执行日常监控任务，例如通过可定制的仪表板跟踪集群指标和设置警报。

新的小组功能使一组用户能够组成一个小组并共享配置资源，从而允许控制中心用作独立的观察平台。此外，控制中心可以补充现有的可观察性解决方案，如 Zabbix 或 Prometheus，提供额外的上下文信息，帮助故障排除人员快速了解根本原因并修复潜在问题。

**功能总结**

*   可定制的直观仪表板
*   用户定义的生产警报
*   查询开发工具
*   主动跟踪和根本原因分析
*   灾难恢复和备份管理
*   监控滚动升级和集群重新平衡
*   Apache Ignite 集群的简单配置

**可用性**
SaaS grid gain 控制中心的 30 天免费试用可在 [GridGain 网站](https://www.globenewswire.com/Tracker?data=5wyS5w4jABfnt_etCUtXoT8ZAvN7whjP9oq0B2V-Oq0_gpJK54XosEap3l4GtNJauGB3sDCt6ex09km-VtWU1LtgT5u95256cla5GzfAN5FsGLgfjMf6hZxe0MsutuZe)上获得。免费试用后，该服务可在订阅的基础上获得，根据要管理和监控的集群节点的数量按月或按年计费。
