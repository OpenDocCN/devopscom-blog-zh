# Tasktop 与 Broadcom 联手，为价值流管理连接数据

> 原文:[https://devo PS . com/task top-joins-forces-with-Broadcom-to-connect-data-for-value-stream-management/](https://devops.com/tasktop-joins-forces-with-broadcom-to-connect-data-for-value-stream-management/)

*由 Tasktop 支持的 Broadcom ValueOps 连接器统一数据并消除孤岛，帮助组织在软件开发期间更好地衡量商业价值*

不列颠哥伦比亚省温哥华市。，2022 年 3 月 8 日— [领先的价值流管理(VSM)洞察提供商 Tasktop](http://tasktop.com/) 今天宣布与 Broadcom 合作，为 Broadcom 的 ValueOps Connectors 提供技术支持，该解决方案可在 Broadcom 的 ValueOps 价值流管理解决方案和数据驱动型企业使用的软件开发工具之间实现双向数据同步。该解决方案已被数十家 Broadcom 和 Tasktop 共享客户采用。

尽管企业在敏捷和开发运维方面进行了投资，但大多数企业仍然没有足够的响应能力或适应性——根据 Forrester 的调查，只有 16%的企业能够每月发布一次以上。Tasktop 的 VSM 平台通过打破孤岛、识别和消除瓶颈、消除浪费和加速软件交付，帮助企业弥合软件交付和业务成果之间的差距。

此外，用于制定业务决策的数据通常不可靠、难以找到，并且被封存在仓库中。由 Tasktop 支持的 Broadcom 的 ValueOps 连接器使企业能够收集和统一价值流中的数据；从整个企业的筒仓中移除数据，以创建跨所有工具和团队的完整数据集；增加在分布式环境中使用不同工具的团队之间的协作；并消除与手动数据收集相关的错误。

“每项业务都是软件业务，在软件交付过程中，需要可访问的可靠数据来获取价值。这需要解决方案将技术和业务与正确的指标联系起来，以实现更有效的企业数字化转型，”Tasktop 首席执行官 Mik Kersten 博士说。“由 Tasktop 支持的 Broadcom value ops 连接器有助于发现孤岛中丢失的数据，这是解锁有效价值流管理所需的所有信息的关键。”

“孤立的数据不能反映全貌。消除数据孤岛是做出明智的业务决策的第一步，以便更好地调整产品开发和支持整体业务战略，”ValueOps 产品集团负责人 Jean-Louis Vignaud 表示。“我们领先的 VSM 解决方案利用这些 ValueOps 连接器将数据整合在一起，揭示新的故事和动力价值流。”
