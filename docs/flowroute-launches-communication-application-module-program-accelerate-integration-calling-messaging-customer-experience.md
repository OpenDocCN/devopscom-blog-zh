# Flowroute 推出通信应用程序模块计划，以加速将通话和信息融入客户体验

> 原文：<https://devops.com/flowroute-launches-communication-application-module-program-accelerate-integration-calling-messaging-customer-experience/>

华盛顿州西雅图市和伊利诺伊州芝加哥市—2016 年 8 月 8 日—[为 SaaS 公司提供先进呼叫和消息服务的领先供应商 Flowroute Inc.](https://www.flowroute.com/) 今天推出了其通信应用模块(CAM)计划，旨在帮助开发人员加快上市时间、降低开发成本并消除在其应用和服务中提供电信级消息功能的障碍。用 Python 编写的前三个应用程序模块包括 SMS 代理、双因素身份验证和约会提醒。

创建 Flowroute 的目的是为组织提供对其客户通信的无与伦比的控制。CAM 计划旨在加速消息传递在应用程序中的实施，使开发人员能够专注于创造引人注目的用户体验。初始模块包括:

● [**短信代理**](https://developer.flowroute.com/docs/sms-proxy)**–**通过使用 Flowroute 电话号码，应用程序用户可以在发送短信时隐藏他们的电话号码，永远不会暴露任何一个接收者的实际电话号码。这在拼车应用等用例中至关重要，乘客需要通过短信匿名与司机交流。

● [**双因素认证**](https://developer.flowroute.com/docs/two-factor-authentication)**–**通过在 web 和移动应用程序中应用双因素认证，这为用户提供了额外的安全层。通过 SMS 发送和验证 PIN 来增强 web 会话的身份验证，从而确保交互的安全性。

● [**预约提醒**](https://developer.flowroute.com/docs/appointment-reminder)——开发者可以让应用通过短信向用户发送预约提醒和确认消息。

Flowroute 本周在芝加哥举行的 [ClueCon 2016](https://cluecon.com/) 上介绍了通信应用模块程序。该计划的首席开发人员凯西·麦克菲(Casey MacPhee)将在 8 月 11 日(星期四)下午4:30-5:00(美国东部时间 )展示利用这些模块的应用。

“作为一家公司，我们的目标是让高级通信对开发人员社区来说更容易、更容易。Flowroute 创始人兼首席产品官 Sean Hsieh 表示:“通信应用模块计划的成功是我们帮助客户解决这些有价值的用例的结果。“我们很高兴能够继续向该计划添加新模块，开发人员可以利用这些模块来增强他们的用户体验。”

此外，Hsieh 还将发表题为“让 Flowroute 的 SMS 为您服务”的演讲，届时他将演示如何将公司的 SMS API 集成到 FreeSWITCH 等通信平台中。

开发者可以从 [Flowroute 开发者门户访问应用模块。](http://developer.flowroute.com/)

**关于流程**
