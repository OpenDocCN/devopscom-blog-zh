# “2020 年美国石油学会邮递员状态”显示，美国石油学会的投资将保持强劲

> 原文:[https://devo PS . com/2020-postman-state-of-the-API-reveals-API-investments-will-remain-strong/](https://devops.com/2020-postman-state-of-the-api-reveals-api-investments-will-remain-strong/)

*超过 13，500 名受访者告诉 Postman，虽然疫情让世界停滞不前，但 API 开发仍在继续激增。*

旧金山(商业资讯)2020 年 9 月 24 日——领先的 API 开发协作平台 Postman 今天发布了其 [2020 年 API 报告](https://www.postman.com/state-of-api/)的结果。这份年度报告是 API 行业最大、最全面的调查，调查了 13，586 名 API 开发人员、测试人员、管理人员和其他人，以了解谁在使用 API，他们是如何完成工作的，以及他们对行业发展方向的看法。

主要调查结果:

*   **API 投资保持强劲:**近一半的受访者表示，在未来 12 个月内，对 API 的时间和资源投资将会增加，而另外三分之一的受访者表示，尽管经济环境充满挑战，但对 API 的投资将保持不变。
*   **疫情改变了世界，但它并没有阻止 API:**89%接受调查的行业成员表示，由于新冠肺炎，他们的组织提供了远程工作选项。近三分之一(30.6%)的受访者表示，通过促进客户沟通、支持远程工作选项以及快速响应法规变化和政府举措，API 在他们的组织响应新冠肺炎的能力中发挥了重要作用。
*   **API 是数字化转型的核心:**对于那些致力于数字化转型计划的人，84.5%的人表示 API 在这些计划中发挥着重要作用。

Postman 的联合创始人兼首席执行官 Abhinav Asthana 表示:“我们的年度调查显示，API 生态系统正在扩大，投资正在流动，API 优先理念的势头正受到越来越多的欢迎。“这些数据还向我们展示了 API 的重要性，以及它们在应对企业、政府和非营利组织运营方式的突然和结构性转变(包括流行病)方面的作用。”

超过 60%的调查受访者在接受“API 优先”理念方面给自己打了 5 分或更高的分数(满分 10 分)，这表明 API 优先正在兴起。API 行业也继续触及几乎每一个工作角色，从开发人员到管理人员到销售人员，甚至到高级管理人员。事实上，今天从事 API 工作的人中有十分之一是管理角色——从最高管理层到董事和经理。

“在对数据进行全面审查后，我们发现超过 70%的受访者表示，在集成 API 之前，他们考虑的最重要的因素是可靠性、安全性、性能和文档，”Postman 首席福音传道者 Kin Lane 说，他与许多集成 API 的公司合作。"然而，不到一半的受访者表示价格是一个重要因素."

其他调查结果:

*   缺乏时间(52.3%)是生产原料药的头号障碍。缺乏文档(54.3%)是使用 API 的头号障碍。
*   受访者认为，与他们在现实生活中的经历相比，他们应该花更多的时间来设计(11%的时间，在理想情况下)和自动化测试(15%)API，花更少的时间来调试 API(10%):只有 8.7%的时间花在设计 API 上，11%的时间花在自动化 API 测试上，而 17%的时间花在调试 API 上。
*   REST (93.4%)是占主导地位的架构风格，其次是 webhooks (34.4%)、SOAP (33.4%)和 GraphQL (22.5%)。JSON Schema (75.6%)是占主导地位的 API 规范，其次是 Swagger 2.0 (43.9%)和 OpenAPI 3.0 (27.8%)。
*   微服务(48.7%)是受访者最感兴趣的未来技术，其次是 Kubernetes (43.6%)和 containers (42.2%)。
