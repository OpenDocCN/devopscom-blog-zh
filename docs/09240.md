# Moogsoft 通过新的功能和集成推动了 AIOps 的发展

> 原文:[https://devo PS . com/moog soft-pusses-the-bounds-of-aiops-with-new-features-and-integrations/](https://devops.com/moogsoft-pushes-the-bounds-of-aiops-with-new-features-and-integrations/)

*AIOps 领导者推进技术进步，增强了事件响应速度以及新的报告和分析功能*

**加利福尼亚州三藩市，2022 年 4 月 5 日—**[ai ops 的先驱和领导者 Moogsoft](http://www.moogsoft.com/) 今天宣布推出新产品功能和增强功能，这些功能和增强功能可扩展集成能力，改善用户体验，并在实施后几天内为客户提供切实可行的见解。

根据 [Gartner](https://www.gartner.com/en/newsroom/press-releases/2019-01-21-gartner-survey-shows-37-percent-of-organizations-have?utm_campaign=AI_Weekly&utm_medium=email&utm_source=Revue%20newsletter) 的数据，超过一半(54%)的首席信息官发现技能短缺是实施人工智能解决方案的最大障碍。Moogsoft 增强了其用户体验，以缩小这一技能差距，并使 AIOps 解决方案适用于 ITOps、SREs、DevOps 和应用团队。

本季度，Moogsoft 庆祝进入商业领域 10 周年，标志着该公司引领 AIOps 进入主流技术类别的十年工作。除了公司和客户的显著增长，Moogsoft 平台的增强巩固了公司作为行业领导者和创新者的地位。

“客户在快速适应新的数字世界。但是我们知道，当您混合使用新旧产品时，这会带来复杂性。这就是为什么 Moogsoft 提供了一个与领域无关的 AIOps 平台，有助于保持战略转型计划的正常进行，同时最大限度地减少对组织的干扰。”产品和市场战略副总裁约翰·哈利说。

最新的功能优势和增强包括:

**服务协作的增强**

*   **Insights 仪表板:** Insights 是最新的分析和报告仪表板，通过历史数据和业务服务趋势为用户提供事故管理生命周期的高级视图。

*   **高级 ServiceNow 集成:**通过高级双向功能、维护窗口和相关配置项提高了 ServiceNow 的可用性。

**用户体验增强和自动化**

*   **Webhook 增强功能:**通过链接到 Webhook 来进一步自动化工作流和出站集成，增强了对事件 API 的定制。

*   **改进的事件和警报过滤器:**网格过滤通过允许基于用户优先级的视图来改进 UI，从应用程序管理到事件严重性。

*   **单点登录:**改进了 Azure Active Directory 的配置，为开发运维团队和 SRE 团队的中央 IT 团队以外的用户提供单点登录(SSO)身份验证。

*   **新用户入职改进:**为了快速向新用户展示 Moogsoft 的价值，我们展示了一个更新的直观流程，用于获取数据并将警报转化为可操作的见解。
