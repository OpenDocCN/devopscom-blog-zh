## 裁谈会基金会欢迎新成员

Codefresh、Fidelity Investments、Octopus Deploy 和 Shipa 作为新成员加入了 CD 基金会。他们与顶级会员 Capital One、CircleCI、Cloudbees、富士通、谷歌、华为、IBM、JFrog、网飞和 Salesforce 以及更广泛的开源 CI/CD 社区一起，帮助加强持续交付模式的发展和演变。

“我们欢迎我们的新成员，并期待着纳入他们的专业知识和指导，以帮助建立有利于整个行业的 CI/CD 流程。技术监督委员会(TOC)主席丹·洛伦茨(Dan Lorenc)说:“这不是一项简单的工作，但它非常重要，仅靠一家公司是无法完成的。“CDF 是帮助指导技术开发和治理结构的方式，以确保 CI/CD 和 DevOps 继续快速高效地发展。”

**code fresh**–code fresh 是一个针对容器和微服务的持续交付和协作平台。这是一个现代 CI/CD 平台，专为软件开发团队使用 Docker、Serverless 和 Kubernetes 构建和部署云原生应用而设计。基于容器的管道提供了更好的灵活性、更容易的管道创建和数十种速度优化。

“我们很高兴加入裁谈会基金会。Codefresh 专为微服务和云原生应用而构建，通过 CD 基金会为开源 CI/CD 开发的方向做出贡献，我们的客户现在和未来都将受益，”Codefresh 首席技术宣传员 Dan Garfield 表示。“没有一致的战略和正确的工具，任何团队都无法有效地实践微服务的开发运维，我们致力于让客户更容易获得正确的战略和信息丰富的工具，从而能够实施和信任。”

**富达投资**–富达投资提供一系列服务，包括基金分销和投资建议、财富管理、人寿保险、退休服务以及证券执行和清算。它为个人投资者、企业、金融顾问和机构投资者管理股票、固定收益和平衡共同基金。它是美国最大的投资管理公司之一，拥有 2600 多万客户，客户总资产 6.5 万亿美元，管理的全球资产超过 2.4 万亿美元。富达将帮助围绕管道、工作流程和其他 CI/CD 领域建立行业规范和指南，并为 CI/CD 工具提供基础支持。

**Octopus Deploy**–Octopus Deploy 通过本地或云解决方案自动部署应用程序。它接管 CI 服务器的终端，提供发布管理、部署自动化和操作工具。Octopus Deploy 帮助开发人员和运营团队自动化软件部署。它还涵盖了审计和合规性要求。

“应用程序部署与 CI/CD 密切相关，我们很高兴加入 CD 基金会来帮助支持和加强社区。成为会员使我们能够引导路线图，并继续推动 CI/CD 向前发展，这直接使八达通用户受益，”八达通部署公司首席执行官兼创始人保罗·斯托维尔说。“如果您想了解更多关于发布管理的信息，请过来打个招呼。”

**Shipa**–Shipa 为在 Kubernetes 上运行任务关键型应用提供了一个云原生应用管理框架。Shiga 提供了对配置的集中控制，以降低错误配置的风险，并提供了一种以应用程序为中心的方法来开发、部署和管理云原生应用程序，而无需 Kubernetes 的专业知识。

“我们 Shipa 相信 CD 基金会的使命，这是一个供应商中立的治理愿景，重点是效率和安全性。通过成为会员并积极参与，包括最近开源我们自己的部署引擎，我们不仅为客户改善了生态系统，还支持了更广泛的 CI/CD 社区。这对 Shipa 的客户来说是一个双赢的局面。“期待参加 CDCon！”

## 关于持续交付基金会
