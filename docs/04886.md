什么是 DevOps 企业峰会？

DevOps 企业峰会是为实施 DevOps 原则和实践的大型复杂组织的技术领导者举办的会议。活动规划强调不断发展的业务、技术和架构实践，以及在大型组织中成功领导广泛变革所需的方法。目标是为领导者提供他们需要的工具和实践，以更快地开发和部署软件，并在市场中取胜。

DevOps Enterprise 在欧洲进入第五个年头，在美国进入第七个年头，它提供主题演讲、小组讨论、分组讨论、自发聚会的社区空间和一个展厅。会议将由在各行各业的大型复杂组织中开拓这些实践的领导者领导。主题演讲将邀请行业名人以及来自知名公司的演讲者，他们将分享从企业开发运维计划中获得的经验。
