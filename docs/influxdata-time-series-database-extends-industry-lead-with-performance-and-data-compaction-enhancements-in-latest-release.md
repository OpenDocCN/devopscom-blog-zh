# InfluxData 时间序列数据库在最新版本中增强了性能和数据压缩，从而扩大了行业领先地位

> 原文：<https://devops.com/influxdata-time-series-database-extends-industry-lead-with-performance-and-data-compaction-enhancements-in-latest-release/>

*公司还推出了业内首个专门为时序数据构建的函数式查询语言的技术预览版，供开发人员社区试用和反馈*

**三藩市—2018 年 11 月 7 日 —** InfluxData，[现代开源平台](https://www.influxdata.com/products/)专门为指标、事件和其他时间序列数据而构建，使开发人员能够构建下一代监控、分析和物联网应用，今天宣布发布 InfluxDB 1.7，为领先的时间序列数据库提供重要的查询性能和数据压缩增强。这一新功能在 InfluxDays SF 2018 上宣布，为开发人员提供了他们一直要求的强大工具，以查询和处理用于监控物联网、DevOps、云、容器和实时分析数据的时序数据。

InfluxDB 是 InfluxData 的时间序列数据库，用作任何涉及大量时间戳数据的用例的数据存储，包括 DevOps 监控、应用程序指标、物联网传感器数据和实时分析。根据社区的建议，今天宣布的这些增强功能进一步扩大了 InfluxDB 的功能与竞争对手之间的差距，根据 [DB-Engines 的最新结果](https://db-engines.com/en/ranking/time+series+dbms)，influx DB 被公认为[时间序列数据库管理系统](https://www.influxdata.com/time-series-database/)中的压倒性领导者。

InfluxData 还宣布了 Flux 的技术预览版，这是业界第一个专门为时间序列数据构建的函数式查询语言。Flux 将数据查询和处理结合在一个强大的、可扩展的开源语言中，这种语言通过用户的贡献得到了丰富。鼓励开发人员使用 Flux 并提供反馈，以帮助塑造时序数据脚本的未来。

Grafana 实验室 ux 总监 David Kaltschmidt 表示:“InfluxDB 是 Grafana 最受欢迎的数据源之一，而 Flux 是一个强大的查询处理引擎，因此我们很高兴能够为 Flux 开发一流的集成。“Flux 插件提供了一个新的查询环境，包括语法高亮显示、标签完成、原始表格预览和内嵌函数文档，以帮助您充分利用 Flux 和 Grafana。”

当使用时序索引(TSI)时，开源的 InfluxDB 1.7 在查询性能和吞吐量方面进行了重大改进，TSI 支持大量短暂且存储在时序数据库中的唯一时序数据。

Flux 的技术预览版今天可供开发者试用和反馈。现在可以从[https://bit.ly/2qph9Xx](https://bit.ly/2qph9Xx)获得入门指南，它提供了具体的例子以及语言语义的参考。鼓励开发者使用 InfluxData 社区网站(【https://community.influxdata.com】T2)来提问和分享经验。为了发布 Flux 技术预览版，InfluxData 将于今天在 InfluxDays SF 2018 举办办公时间，继续努力最终确定语言，解决问题，并接受更多反馈。

“性能、数据压缩和时间序列指数的改进有助于 InfluxData 通过提供卓越的性能和满足开发人员的高级需求来保持其在该类别中的领导地位，”产品副总裁 Tim Hall 说。“InfluxData 在定义 Flux 语言方面一直与开发人员社区密切合作。我们期待开发者下载 Flux，使用它并为我们提供有用的反馈，以帮助语言取得成功。”

随着对分析当今复杂环境的需求不断增长，指标、事件和其他基于时间的数据正以指数级的速度生成。InfluxData 平台提供了一套全面的工具和服务来积累指标和事件数据，分析数据，并通过强大的可视化和通知对数据采取行动。

InfluxData 通过提供增长最快的开源平台，使客户能够获得更好的业务洞察力、数据驱动的实时操作以及整个基础设施(从应用程序到微服务，从系统到传感器)的统一视图，迅速建立了跨行业的开发人员和客户基础，包括制造、金融服务、能源和电信。包括思科系统、Coupa Software、IBM、Houghton Mifflin Harcourt、Nordstrom 和 Tesla 在内的 450 多家客户已经选择 InfluxData 作为他们衡量指标和事件的现代数据平台。InfluxData 是在现代指标和事件平台中向时间序列转变的先锋，并使客户有可能成为数据驱动型并采取数字化转型举措。
