# Delphix 给 Amazon 关系数据库服务带来了更高的速度和安全性

> 原文:[https://devo PS . com/Delphi x-给亚马逊关系数据库服务带来更快的安全性/](https://devops.com/delphix-brings-even-more-speed-security-to-amazon-relational-database-service/)

*领先的酒店公司 Choice Hotels 和其他全球企业依靠 Delphix DataOps 平台，利用亚马逊关系数据库服务快速、安全地将数据迁移到云中，推动关键任务数字化转型计划*

**内华达州拉斯韦加斯，@ AWS re:Invent 2018，2018 年 11 月 28 日**–[通过 DataOps 加速创新的德尔菲克斯](http://www.dephix.com)，今天宣布利用[亚马逊关系数据库服务](https://aws.amazon.com/rds/)(亚马逊 RDS)的德尔菲克斯动态数据平台(DDDP)的可用性。基于 Delphix 和 Amazon Web Services，Inc. (AWS)之间的长期合作关系，这一创新解决方案将为全球企业的云开发团队提供快速、安全的数据，帮助他们以在当今数字经济中取胜所需的速度将云原生应用程序推向市场。

根据普华永道 2018 年的“[创新基准](https://www.pwc.com/us/en/services/consulting/innovation-benchmark-findings.html)”报告，近四分之三的企业表示，他们正在被竞争对手的创新超越。应用程序正迅速变得越来越复杂、智能和数据密集型，这让公司疲于应对传统数据基础架构，无法利用它来提供创新，以跟上不断变化的消费者需求。

开发团队需要在按需云环境中轻松访问数据，但运营却陷入了数据库配置和数据编辑等手动任务的泥潭，迫使开发团队等待数天或数周才能获得数据。随着这些组织将更多的工作负载迁移到云，他们面临着一个痛苦的现实，即他们的数据无法以云支持的速度移动。支持数据驱动型企业的需求激增，与管理、分发和保护数据的不断增加的成本、复杂性和风险产生了激烈的冲突。

Amazon RDS 的创建就是为了简化这一过程，使团队能够根据需要，以可扩展和可管理的方式快速轻松地创建新的生产环境。创建 Delphix 动态数据平台是为了允许在非生产开发和测试环境中随时访问数据，这样团队可以以更低的成本更快地移动，而不会损害数据隐私。这种强大的组合为客户提供了一种解决方案，可以帮助他们以很少的时间和成本，快速、安全地向市场提供持续创新。

“我们的许多客户信任 Amazon RDS 来大规模管理他们的数据库，”Amazon Web Services，Inc .负责分析、Aurora 和 RDS 的副总裁 Anurag Gupta 说，“通过与 Delphix 合作，我们不仅可以为企业简化数据库管理，还可以为这些组织提供所需的性能增强，以加快关键任务计划并在各自的市场中获胜。”

作为今天发布的一部分，亚马逊 RDS 上的 Delphix 平台提供:

*   AWS 上生产数据库的高可伸缩性、可用性和可管理性；
*   按需亚马逊弹性计算云(Amazon EC2)在几分钟内提供生产数据库副本，为数据库带来云的速度和敏捷性；
*   用于刷新、分支、书签和共享的自助服务控件，因此开发人员可以将数据视为代码，与 DevOps 工具和 Delphix 自动化框架集成；
*   集成数据屏蔽，以识别和减轻下游环境中的风险；
*   支持 Oracle、PostgreSQL 和 MySQL 的可扩展插件——更多功能即将推出。

“企业正在实施数字化转型计划，以前所未有的速度向客户提供创新的全新体验。IDC 软件开发和开源部门副总裁 Al Gillen 表示:“应用更新率的提高给开发人员和开发流程带来了更大的压力。“借助亚马逊 RDS 上的 Delphix DataOps 平台，公司现在有了一个新的目标平台来托管用于开发和测试的数据，从而减少摩擦，帮助开发团队加快下一代应用程序的交付。”

Choice Hotels 首席系统工程师 Nick Suwyn 表示:“随着我们希望将越来越多的工作负载转移到云，从而为我们的业务带来更高的效率和灵活性，随着我们全面采用云，现代软件开发生命周期的治理和数据管理已成为我们的首要问题之一。“Delphix 平台最大限度地减少了安全问题，并在云中提供了自动化数据环境，每天都在以越来越快的速度推动软件创新。”

**数据操作转向云原生**

《财富》50 强中近一半的企业都在使用 Delphix，世界上一些最大的品牌都在使用它来推动和加速关键任务转型计划。通过开放数据流，Delphix 平台与 Amazon RDS 合作，帮助在几分钟内创建安全的数据环境。此外，自动化数据环境配置的能力是 DataOps 方法的核心，它加速了软件开发和云迁移项目。DataOps 在市场中发展势头迅猛，最近被确定为 2018 年三个 [Gartner“炒作周期”](https://www.gartner.com/en/newsroom/press-releases/2018-09-11-gartner-hype-cycle-for-data-management-positions-three-technologies-in-the-innovation-trigger-phase-in-2018)(数据管理、企业 IT 管理和数据安全)中的最新“创新触发器”，并有望在五年内实现主流采用。

Delphix 首席执行官 Chris Cook 表示:“数据是实现在当今数字经济中取胜所需速度的主要障碍。“基于我们与 AWS 的长期合作关系，亚马逊 RDS 上的 Delphix 有助于保护 AWS 上的数据，使更多的企业能够加速创新、迁移到云，并引领各自的市场。”
